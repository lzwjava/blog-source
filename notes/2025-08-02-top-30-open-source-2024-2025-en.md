### Top 30 Open Source Projects Trending in 2024-2025

Based on aggregated data from various sources tracking GitHub stars gained, project velocity, trending lists, and community impact, here is a compiled list of the top 30 open source projects that gained significant traction or were launched around 2024-2025. The focus is on projects with high growth in stars, contributions, or relevance in AI, developer tools, and other emerging areas. I've prioritized those with notable momentum in the specified timeframe, drawing from startup-led innovations, AI advancements, and contributor-friendly repositories. Projects are ranked roughly by popularity metrics like stars gained or mentions across sources.

1. **Ollama**  
   An open-source tool for running large language models (LLMs) like Meta’s Llama locally on desktops. Gained 76,000 stars in 2024.

2. **Zed**  
   A high-performance, collaborative code editor for humans and AI, designed for monorepos. Open-sourced in January 2024, gained over 50,000 stars.

3. **Dify (LangGenius)**  
   An open-source platform for building LLM-powered applications. Gained 43,000 stars in 2024.

4. **ComfyUI**  
   A node-based tool for generating images, videos, and audio with generative AI models. Grew to 61,900 stars with 195% increase.

5. **OpenHands (All Hands AI)**  
   A platform for building AI software development agents that can code and fix issues. Launched in March 2024, gained 39,600 stars.

6. **UV (Astral)**  
   A fast Python package and project manager written in Rust for high-performance developer workflows. Gained 32,800 stars.

7. **Stirling PDF**  
   A privacy-focused, self-hostable PDF manipulation tool with features like editing and conversion.

8. **Maybe Finance**  
   Modern financial planning and wealth management software, self-hostable and privacy-oriented. Gained 35,400 stars.

9. **RustDesk**  
   Open-source remote desktop software emphasizing security and self-hosting.

10. **Puter**  
    An extensible internet operating system for building feature-rich web-based apps. Gained 27,000 stars.

11. **RAGFlow (InfiniFlow)**  
    A Retrieval-Augmented Generation engine based on deep document understanding for AI applications. Gained 25,600 stars.

12. **Shardeum**  
    An EVM-compliant blockchain using dynamic state sharding for scalability. Gained 22,900 stars.

13. **Open-Sora (HPC-AI Tech)**  
    A text-to-video generation model for enterprise AI, focused on efficiency. Gained 17,300 stars.

14. **Firecrawl (Mendable)**  
    An API for crawling websites and converting them to clean markdown or structured data for AI use. Gained 20,100 stars.

15. **OpenUI (Weights and Biases)**  
    A tool for creating UI components via natural language descriptions, integrated with ML workflows. Gained 19,500 stars.

16. **Devika (Stition.ai)**  
    An AI agent that understands instructions, researches, and patches code vulnerabilities. Gained 18,600 stars.

17. **Flux (Black Forest Labs)**  
    Inference code for running image generation and editing with advanced visual AI models.

18. **Open WebUI MCP**  
    A proxy server turning MCP tools into OpenAPI-compatible servers for AI app integration.

19. **Unbody**  
    A modular backend for AI-native software, handling perception, memory, reasoning, and action.

20. **OWL**  
    A multi-agent collaboration framework using browsers, terminals, and MCP tools; tops GAIA benchmark.

21. **F/mcptools**  
    A CLI for MCP servers to discover, call tools, and manage prompts for AI development.

22. **self.so (Nutlope)**  
    AI-powered tool to build personal websites from résumés or LinkedIn profiles.

23. **VoiceStar**  
    Duration-controllable text-to-speech synthesis with CLI and pre-trained models.

24. **Second-Me**  
    A "digital twin" AI agent that mirrors user knowledge and preferences for tasks like account management.

25. **CSM (SesameAI Labs)**  
    A conversational speech model using Llama architecture for natural audio generation.

26. **Letta**  
    An open file format for packaging and porting AI agents across frameworks.

27. **Blender meets Claude**  
    Integrates Blender with Claude AI via MCP for natural language control of 3D workflows.

28. **Appwrite**  
    An open-source backend platform for authentication, databases, functions, and storage.

29. **Storybook**  
    A frontend workshop for building, testing, and documenting UI components in isolation.

30. **Shadcn UI**  
    Customizable, accessible UI components for React, no installation required.

These projects reflect the dominance of AI-driven tools, developer productivity enhancers, and privacy-focused solutions in 2024-2025. Many are beginner-friendly for contributions, with active communities on GitHub. For those interested in contributing, check each project's "good first issue" labels.
