1.  **Download Battle.net:** Go to the official Blizzard website and download the **Battle.net Desktop App for Mac**.

Since Blizzard does not offer a native Linux version, you will use a Windows compatibility layer like **Wine** or one of its wrappers (**Lutris** or **Steam/Proton**). This is a well-established method for gaming on Linux.

### Option 1: Using Lutris (Recommended for Battle.net)

**Lutris** is a dedicated game manager for Linux that simplifies the process of running Windows games.

1.  **Install Lutris:** Install the Lutris client using your Ubuntu package manager.
2.  **Use the StarCraft Script:** Search for **StarCraft: Remastered** on the Lutris website. Lutris provides pre-configured installation scripts that handle the Battle.net client and Wine setup automatically.
3.  **Install and Play:** Run the script, and you can install and launch the game directly from the Lutris client.

### Option 2: Using Steam/Proton (Simple for General Users)

If you already use Steam, this can also be very simple:

1.  **Install Steam:** Install Steam on your Ubuntu system.
2.  **Install Battle.net via Steam:**
    * Download the **Windows Battle.net-Setup.exe** file from Blizzard's website.
    * In the Steam client, go to **Games** > **Add a Non-Steam Game to My Library...** and select the `.exe` file.
    * Right-click the new Battle.net entry in your Steam Library, go to **Properties** > **Compatibility**, and check **"Force the use of a specific Steam Play compatibility tool,"** selecting **Proton Experimental**.
    * Run it, install Battle.net, and then install StarCraft: Remastered through the Battle.net app.
