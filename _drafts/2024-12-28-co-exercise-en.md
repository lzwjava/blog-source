* Computer contains cpu, memory, cache, peripheral devices, input/output system. 

* CPU contains ALU, arithmetic logic unit, and control unit. ALU performs basic arithmetic operations, like add, subtract, multiply or divide, or left shift, right shift. Control unit controls how to execute instructions.

* Memory can be divided into ROM (Read Only Memory), RAM (Random Access memory), vilotile memory or non vilotile memory.
