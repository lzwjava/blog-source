لمراقبة استخدام وحدة معالجة الرسومات (GPU) مع مرور الوقت باستخدام `nvidia-smi`، تحتاج فقط إلى وسيط الحلقة:

**التحديث الحي الأساسي**

```
nvidia-smi -l 1
```

هذا يقوم بالتحديث كل ثانية واحدة. يمكنك استبدال الرقم `1` بأي فترة زمنية بالثواني.

**إذا كنت تريد إحصائيات أكثر تفصيلاً (الاستخدام لكل عملية)**

```
nvidia-smi pmon -s um -d 1
```

هذا يعرض استخدام وحدة معالجة الرسومات (`u`) والذاكرة (`m`) لكل عملية كل ثانية.

**إذا كنت تريد إحصائيات وحدة معالجة الرسومات الكاملة بشكل متكرر**

```
nvidia-smi --query-gpu=timestamp,utilization.gpu,utilization.memory,memory.used,memory.total --format=csv -l 1
```

هذا يطبع سطراً مضغوطاً كل ثانية مع بيانات الاستخدام والذاكرة.

أعلمني إذا كنت تريد حفظ الناتج في ملف أو تحتاج إلى تنسيق مخصص.
