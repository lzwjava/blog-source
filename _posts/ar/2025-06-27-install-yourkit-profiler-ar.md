لتركيب YourKit Java Profiler على أوبونتو، اتبع هذه الخطوات استنادًا إلى المعلومات المتاحة من موقع YourKit:

1.  **تحميل البروفايلر**:
    *   قم بزيارة صفحة تحميل YourKit Java Profiler على [https://www.yourkit.com/java/profiler/download/](https://www.yourkit.com/java/profiler/download/).
    *   اختر نسخة لينكس من YourKit Java Profiler 2025.3، والتي تدعم Java 8 إلى Java 24 وهي متوافقة مع لينكس (بما في ذلك أوبونتو) على أبنية مثل arm32، arm64، ppc64le، x64، و x86. تأكد من أن نظامك يستوفي [متطلبات النظام](https://www.yourkit.com/docs/java/system-requirements/) للتأكد من التوافق.

2.  **تحميل الأرشيف**:
    *   قم بتحميل أرشيف `.zip` لنظام لينكس (مثل `YourKit-JavaProfiler-2025.3-<build>.zip`). رابط التحميل متوفر على صفحة التحميل في YourKit.

3.  **فك ضغط الأرشيف**:
    *   افتح الطرفية وانتقل إلى الدليل حيث يوجد الملف الذي تم تحميله (مثل `~/Downloads`).
    *   افك ضغط الأرشيف باستخدام الأمر التالي:
        ```bash
        unzip YourKit-JavaProfiler-2025.3-<build>.zip -d /opt/yourkit
        ```
        استبدل `<build>` برقم البناء الفعلي من الملف الذي تم تحميله. يستخرج هذا الأمر البروفايلر إلى `/opt/yourkit`. يمكنك اختيار دليل آخر إذا كنت تفضل ذلك.

4.  **تشغيل البروفايلر**:
    *   انتقل إلى الدليل الذي تم استخراج الملفات فيه:
        ```bash
        cd /opt/yourkit
        ```
    *   شغّل البروفايلر باستخدام النص البرمجي المرفق:
        ```bash
        ./bin/profiler.sh
        ```
        سيؤدي هذا إلى تشغيل واجهة مستخدم YourKit Java Profiler.

5.  **اختياري: التثبيت التلقائي بمفتاح الترخيص**:
    *   إذا كان لديك مفتاح ترخيص وتريد أتمتة عملية التثبيت، يمكنك استخدام خيارات سطر الأوامر لقبول اتفاقية الترخيص (EULA) وتطبيق مفتاح الترخيص. على سبيل المثال:
        ```bash
        ./bin/profiler.sh -accept-eula -license-key=<key>
        ```
        استبدل `<key>` بمفتاح الترخيص الفعلي الخاص بك. هذا مفيد للإعدادات الآلية أو المعتمدة على النصوص البرمجية.

6.  **التكامل مع بيئة التطوير (اختياري)**:
    *   إذا كنت تستخدم بيئة تطوير متكاملة (IDE) مثل Eclipse أو IntelliJ IDEA أو NetBeans، فإن YourKit يوفر إضافات للتكامل السلس. بالنسبة لـ Eclipse، على سبيل المثال:
        *   افتح Eclipse وانتقل إلى **Help > Install New Software**.
        *   أضف مستودع إضافة YourKit: `https://www.yourkit.com/download/yjp2025_3_for_eclipse/`.
        *   اختر إضافة YourKit Java Profiler، واتبع تعليمات التثبيت، وأعد تشغيل Eclipse إذا طُلب منك ذلك.
        *   بدلاً من ذلك، يمكنك استخدام الأرشيف غير المتصل الموجود في `<Profiler Installation Directory>/lib/eclipse-plugin/yjp2025_3_for_eclipse.zip`.
    *   بعد التثبيت، سيظهر إجراء "Profile" في شريط أدوات Eclipse، أو القائمة الرئيسية، أو القوائم السياقية لتسهيل عملية التحليل.

7.  **التحقق من التثبيت**:
    *   بمجرد تشغيل البروفايلر، ستسمح لك شاشة الترحيب بمراقبة تطبيقات Java المحلية قيد التشغيل، أو الاتصال بالتطبيقات البعيدة، أو فتح اللقطات (snapshots) الموجودة. يمكنك أيضًا تحميل تطبيق تجريبي لاستكشاف ميزات البروفايلر.

8.  **ملاحظات إضافية**:
    *   إذا واجهت مشاكل، تأكد من تثبيت Java (الإصدار 8 أو أعلى) على نظامك، حيث أن YourKit Java Profiler يتطلب JVM متوافقًا. يمكنك التحقق من إصدار Java باستخدام:
        ```bash
        java -version
        ```
    *   للتحليل عن بُعد، تأكد من تكوين الوصول عبر SSH إذا كنت تقوم بتحليل التطبيقات على خوادم بعيدة.
    *   يقدم YourKit نسخة تجريبية مجانية لمدة 15 يومًا، لذا لا يلزم وجود مفتاح ترخيص في البداية. لمشاريع المصدر المفتوح غير التجارية، يمكنك الاتصال بمبيعات YourKit للحصول على ترخيص مجاني.
    *   إذا فشل البروفايلر في البدء (على سبيل المثال، بسبب وجود مسافات في مسار التثبيت)، انقل المجلد المستخرج إلى مسار لا يحتوي على مسافات، مثل `/opt/yourkit`.
