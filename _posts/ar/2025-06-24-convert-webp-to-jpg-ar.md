لديك عدة خيارات ممتازة لتحويل WebP إلى JPG في أوبنتو، سواء من سطر الأوامر أو باستخدام أدوات واجهة المستخدم الرسومية.

## أدوات سطر الأوامر

هذه الأدوات هي عمومًا الأكثر مرونة وكفاءة لتحويل الملفات الفردية أو التحويل الدفعي.

### 1. ImageMagick

ImageMagick هي مجموعة أدوات قوية من سطر الأوامر لمعالجة الصور، وهي متاحة على نطاق واسع.

*   **التثبيت:**
    ```bash
    sudo apt update
    sudo apt install imagemagick
    ```
*   **تحويل ملف واحد:**
    ```bash
    convert input.webp output.jpg
    ```
    أو، باستخدام `magick` للإصدار ImageMagick v7+:
    ```bash
    magick input.webp output.jpg
    ```
*   **تحويل عدة ملفات (تحويل دفعي) في المجلد الحالي:**
    ```bash
    mogrify -format JPEG *.webp
    ```
    سيؤدي هذا إلى تحويل جميع ملفات `.webp` في المجلد الحالي إلى `.jpeg` (أو `.jpg`) بنفس اسم الملف. إذا كنت تريد الملفات المحولة في مجلد منفصل:
    ```bash
    mkdir output_jpgs # إنشاء مجلد للمخرجات
    mogrify -format JPEG -path output_jpgs *.webp
    ```

### 2. `dwebp` (من حزمة `libwebp`)

`dwebp` هو أداة مخصصة لفك تشفير صور WebP.

*   **التثبيت:**
    ```bash
    sudo apt update
    sudo apt install webp
    ```
*   **تحويل ملف واحد:**
    ```bash
    dwebp input.webp -o output.jpg
    ```
    بينما تحدد `.jpg` كمخرج، فإن `dwebp` غالبًا ما يخرج بصيغة PPM ويتطلب أداة أخرى مثل ImageMagick للتحويل إلى JPEG صحيح. ومع ذلك، تشير بعض المصادر إلى أن استخدام `.jpg` مباشرة كمخرج يعمل في الحالات البسيطة. إذا واجهت مشاكل، قم بالتحويل إلى PNG أولاً، ثم إلى JPG:
    ```bash
    dwebp input.webp -o output.png
    convert output.png output.jpg
    ```

### 3. `ffmpeg`

`ffmpeg` مخصص أساسًا للفيديو والصوت، ولكنه أيضًا قادر جدًا في التعامل مع تنسيقات الصور.

*   **التثبيت:**
    ```bash
    sudo apt update
    sudo apt install ffmpeg
    ```
*   **تحويل ملف واحد:**
    ```bash
    ffmpeg -i input.webp output.jpg
    ```
*   **تحويل عدة ملفات (تحويل دفعي):**
    ```bash
    for file in *.webp; do ffmpeg -i "$file" "${file%.*}.jpg"; done
    ```
    يتكرر هذا الأمر عبر جميع ملفات `.webp` في المجلد الحالي ويحولها إلى `.jpg` بنفس الاسم (مثلاً، `image.webp` تصبح `image.jpg`).

## أدوات واجهة المستخدم الرسومية

إذا كنت تفضل واجهة رسومية، إليك بضعة خيارات:

### 1. GIMP (برنامج معالجة الصور المفتوح المصدر)

GIMP هو محرر صور مفتوح المصدر وقوي يدعم WebP.

*   **التثبيت:**
    ```bash
    sudo apt update
    sudo apt install gimp
    ```
*   **كيفية التحويل:**
    1.  افتح صورة WebP في GIMP (File > Open).
    2.  اذهب إلى File > Export As.
    3.  في مربع حوار "Export Image"، اختر "JPEG image" من القائمة المنسدلة "Select File Type (By Extension)".
    4.  انقر على "Export" وضبط إعدادات الجودة إذا رغبت.

### 2. XnView MP / XnConvert

هذه برامج عارضة صور ومحولات دفعة متعددة المنصات مع دعم شامل للتنسيقات، بما في ذلك WebP.

*   **التثبيت:**
    قد تحتاج إلى تحميل حزمة `.deb` من موقعهم الرسمي وتثبيتها باستخدام `sudo dpkg -i <package_name.deb>` أو من خلال مثبت الحزم الرسومي.
*   **كيفية التحويل (بشكل عام):**
    1.  افتح XnView MP أو XnConvert.
    2.  أضف صور WebP التي تريد تحويلها.
    3.  اختر JPG كتنسيق الإخراج.
    4.  اضبط أي إعدادات (جودة، تغيير حجم، إلخ).
    5.  ابدأ عملية التحويل.

### 3. Converseen

Converseen هو محول صور دفعة مجاني يدعم WebP.

*   **التثبيت:** يمكنك عادة العثور عليه في Ubuntu Software Center أو تثبيته عبر apt إذا كان متوفرًا في مستودعاتك.
*   **كيفية التحويل:** لديه واجهة رسومية مباشرة لإضافة الصور، وتحديد تنسيق الإخراج، والتحويل.

## أي طريقة تختار؟

*   **لتحويلات الملفات الفردية السريعة أو لأتمتة المهام البرمجية:** استخدم `ffmpeg` أو `ImageMagick` من سطر الأوامر. فهي فعالة جدًا.
*   **لتحويلات عرضية حيث تريد واجهة بصرية ومزيدًا من التحكم في تحرير الصورة:** GIMP هو خيار رائع.
*   **لتحويل العديد من الصور دفعةً باستخدام واجهة رسومية وخيارات متنوعة:** XnView MP/XnConvert أو Converseen ممتازان.
