نعم، Azure Virtual Machines (VMs) تدعم بشكل كامل تشغيل أنظمة تشغيل Linux. يمكنك نشر مجموعة واسعة من التوزيعات الشهيرة مباشرة من Azure Marketplace أو تحميل صور Linux المخصصة الخاصة بك.

### أبرز النقاط:
- **التوزيعات المدعومة**: توافق Azure وتدعم توزيعات Linux الرئيسية، بما في ذلك Ubuntu وRed Hat Enterprise Linux (RHEL) وDebian وSUSE Linux Enterprise Server (SLES) وOracle Linux وCentOS Stream وRocky Linux وAlmaLinux وغيرها. يتوفر أكثر من 100 صورة Linux جاهزة للاستخدام.
- **خيارات النشر**: استخدم مدخل Azure أو واجهة سطر الأوامر (CLI) أو PowerShell أو قوالب ARM لإنشاء Linux VMs. على سبيل المثال، يمكنك تشغيل مثيل Ubuntu Server بسرعة خلال دقائق.
- **الميزات**: تتضمن Linux VMs على Azure تكاملاً كاملاً مع خدمات Azure مثل التخزين والشبكات والأمان والمراقبة. كما أنها تدعم أدوات مثل Azure Arc للإدارة الهجينة.

إذا كنت تبدأ للتو، فاطلع على دليل البدء السريع الرسمي لإنشاء Linux VM.
