يقع مسار JDK في `/opt/homebrew/Cellar/openjdk@17/17.0.14/libexec/openjdk.jdk/Contents/Home`.

```bash
mvn build
mvn verify
mvn install
mvn spring-boot:run
```

استخدم سطر الأوامر لتجميع وتشغيل الكود.
