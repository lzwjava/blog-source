## GPT-5 (OpenAI)

*   **تاريخ الإصدار**: 7 أغسطس 2025. ([OpenAI][1])
*   **ما هو**: نموذج لغوي كبير موحد مع تحسين في القدرة على التفكير، وتقليل الهلوسة، وأداء أفضل في البرمجة والرياضيات والكتابة والصحة والإدراك البصري، إلخ. يتضمن نسخة "GPT-5 pro" للمهام الأكثر تطلبًا. ([OpenAI][1])
*   **التفكير / آلية العمل**: يمتلك GPT-5 آلية داخلية ("تفكير سريع" مقابل "تفكير أعمق") ويمكن للمستخدمين اختيار "وضع تفكير GPT-5" أو استخدام مطالبات مثل "فكر مليًا في هذا" لطلب تفكير أكثر تعمقًا. ([OpenAI][1])
*   **مستويات الوصول / القيود**:
    *   جميع مستخدمي ChatGPT لديهم حق الوصول (مجاني ومدفوع). ([OpenAI][1])
    *   المستخدمون المجانيون لديهم استخدام محدود أكثر، وبعد استخدام معين قد يتم تحويلهم إلى نسخة أخف ("GPT-5 mini"). ([OpenAI][1])
    *   المستخدمون المدفوعون (Plus, Pro, Team, Enterprise, EDU) يحصلون على حدود استخدام أعلى؛ مستخدمو Pro يحصلون على "GPT-5 pro". ([OpenAI][1])

## Gemini 2.5 Pro (جوجل)

*   **الإصدار / التوفر**:
    *   أُعلن عن Gemini 2.5 Pro (تجريبي) لأول مرة في 25 مارس 2025. ([blog.google][2])
    *   أُصدر الإصدار المستقر ("المتاح بشكل عام") من Gemini 2.5 Pro في 17 يونيو 2025. ([Google Cloud][3])
*   **الإمكانيات**: إنه النموذج الأكثر تقدمًا من جوجل في عائلة Gemini 2.5. يتميز بأشياء مثل نطاق سياق كبير (1 مليون رمز)، وقدرة تفكير قوية، وبرمجة، ودعم متعدد اللغات، إلخ. ([blog.google][2])

## GPT-5 mini

*   **ما هو / متى**: GPT-5 mini هو نسخة أخف وزنًا وأسرع من GPT-5 أصبح متاحًا في GitHub Copilot (معاينة عامة) في منتصف أغسطس 2025. ([The GitHub Blog][4])
*   **أين وكيف**: إنه متاح في GitHub Copilot Chat (على github.com)، وفي VS Code، وعلى GitHub Mobile (iOS/Android). سيظهر أيضًا في أداة اختيار النموذج للمستخدمين. ([The GitHub Blog][4])
*   **طلبات الميزة / التكلفة**: إنه *لا* يستهلك "طلبات الميزة" في الخطط المدفوعة لـ GitHub Copilot. أي أن استخدام GPT-5 mini لا يستنفذ حصة "طلبات الميزة". ([The GitHub Blog][4])

## كيفية استخدام هذه النماذج في **Copilot / الميزة / Microsoft / GitHub**

*   **GitHub Copilot**:
    *   جميع خطط Copilot (بما في ذلك المجانية) لديها الآن إمكانية الوصول إلى GPT-5 mini. ([The GitHub Blog][4])
    *   GPT-5 (النسخة الكاملة) متاح فقط في خطط Copilot المدفوعة. ([The GitHub Blog][5])
    *   تتيح أداة اختيار النموذج للمستخدمين الاختيار بين GPT-5 وGPT-5 mini، إلخ، اعتمادًا على خطتهم وإعدادات المؤسسة. ([The GitHub Blog][4])

*   **Microsoft Copilot / Copilot Studio / 365 Copilot**:
    *   تدعم Microsoft استخدام GPT-5 في منتجات Copilot الخاصة بها. على سبيل المثال، يمكن لمنشئي المحتوى باستخدام Copilot Studio اختيار GPT-5، وهناك توجيه تلقائي بين المتغيرات المختلفة لـ GPT-5 (دردشة مقابل تفكير)، إلخ. ([Microsoft Developer][6])
    *   تتطلب ميزات الميزة بشكل عام اشتراكات مدفوعة. غالبًا ما يكون استخدام GPT-5، أو النماذج ذات السعة الأعلى، أو تلك ذات القدرات الأكبر في التفكير مقصورًا على المستخدمين الدافعين أو مستويات المؤسسات. ([OpenAI][1])
