القدرة على استخدام الأمر `code chat "hi"` لفتح نافذة الدردشة الخاصة بـ GitHub Copilot في Visual Studio Code (VS Code) ترتبط بالتكامل الأوسع لخاصية GitHub Copilot Chat والميزات المرتبطة بها، مثل عرض الدردشة والأوامر ذات الصلة. بناءً على المعلومات المتاحة، إليك التحليل:

### إصدار الأمر `code chat`
لم يتم ذكر الأمر `code chat` صراحةً في ملاحظات الإصدار المقدمة لـ VS Code، لكنه يرتبط ارتباطًا وثيقًا بوظيفة GitHub Copilot Chat، التي تم تعزيزها تدريجيًا عبر إصدارات VS Code المتعددة. من المرجح أن هذا الأمر هو اسم مختصر أو بديل لاستدعاء عرض Copilot Chat أو الدردشة المضمنة، والتي أصبحت بارزة مع تقديم ميزات Copilot Chat.

- **تكامل GitHub Copilot Chat**: تم تطوير ميزة Copilot Chat، بما في ذلك عرض الدردشة، بشكل كبير وتسليط الضوء عليها في الإصدارات التي بدأت حوالي أغسطس 2023 (إصدار VS Code 1.82) واستمرت في التطور عبر الإصدارات اللاحقة، بما في ذلك الإصدار 1.99 (مارس 2025). تم تقديم عرض الدردشة والأوامر ذات الصلة مثل `Chat: Open Chat` أو أوامر الدردشة المضمنة وتحسينها خلال هذه الفترة.[](https://code.visualstudio.com/updates/v1_93)[](https://code.visualstudio.com/updates/v1_86)
- **لوحة الأوامر وأوامر الدردشة**: بحلول إصدار نوفمبر 2024 (الإصدار 1.96)، كان VS Code قد قدم ميزات مثل الانتقال من عرض الدردشة إلى Copilot Edits وتحسين معالجة السياق في عرض الدردشة، مما يشير إلى دعم قوي لأوامر الدردشة. كانت القدرة على تشغيل Copilot Chat عبر الأوامر من لوحة الأوامر موجودة بالفعل، ومن المرجح أن الأمر `code chat` ظهر كجزء من هذه التحسينات.[](https://code.visualstudio.com/updates/v1_96)
- **خاص بالإصدار 1.99.2**: إصدار مارس 2025 (الإصدار 1.99) وتحديثاته (1.99.2 و 1.99.3) لا تذكر صراحةً الأمر `code chat "hi"`. ومع ذلك، فإنها تناقش التطورات في تجربة Copilot Chat، مثل أوضاع الدردشة المخصصة، ووضع الوكيل، وتحسين معالجة السياق، مما يشير إلى أن أوامر الدردشة كانت مدعومة جيدًا في هذا الوقت. الأمر `code chat`، إذا لم يكن موثقًا صراحةً، فمن المرجح أن يكون متاحًا كجزء من تكامل لوحة الأوامر مع Copilot Chat.[](https://code.visualstudio.com/updates/v1_99)

### متى تم إصدار وظيفة `code chat "hi"`؟
بينما الإصدار الدقيق للأمر `code chat "hi"` غير محدد في الملاحظات المقدمة، فمن المعقول الاستدلال على أنه كان متاحًا بحلول وقت إصدار مارس 2025 (الإصدار 1.99) أو قبل ذلك، نظرًا للحالة المتطورة لميزات Copilot Chat. من المرجح أن الأمر يستخدم البنية التحتية الحالية لفتح عرض الدردشة، والتي كانت راسخة بحلول الإصدار 1.96 (نوفمبر 2024) مع أوامر مثل `Chat: Open Chat` أو `Open Quick Chat`.[](https://code.visualstudio.com/updates/v1_96)[](https://code.visualstudio.com/updates/v1_93)

- **أقرب إصدار محتمل**: تم تطوير وظيفة Copilot Chat، بما في ذلك القدرة على فتح عرض الدردشة عبر الأوامر، بشكل كبير بحلول أغسطس 2024 (الإصدار 1.93) بميزات مثل الدردشة السريعة وتحسينات السياق. كان من الممكن تقديم الأمر `code chat` حول هذا الوقت أو بعده Shortly كجزء من تكامل لوحة الأوامر مع Copilot.[](https://code.visualstudio.com/updates/v1_93)
- **التأكيد للإصدار 1.99.2**: بحلول الإصدار 1.99.2 (مارس 2025)، كان نظام Copilot Chat ناضجًا، مع دعم للأوضاع المخصصة، ووضع الوكيل، ومتغيرات السياق مثل `#codebase`. الأمر `code chat "hi"`، الذي يفتح عرض الدردشة باستخدام مطالبة مثل "hi"، من المرجح أن يكون مدعومًا كأمر قياسي في لوحة الأوامر، نظرًا للحالة المتقدمة لوظيفة الدردشة.[](https://code.visualstudio.com/updates/v1_99)

### الخلاصة
من المرجح أن الأمر `code chat "hi"` لفتح نافذة Copilot Chat كان متاحًا بحلول إصدار مارس 2025 (الإصدار 1.99) أو قبل ذلك، ربما في وقت مبكر يعود إلى أغسطس 2024 (الإصدار 1.93)، عندما تم تعزيز ميزات Copilot Chat بشكل كبير. بالنسبة للإصدار 1.99.2 تحديدًا، فإن الأمر سيعمل almost certainly، حيث كان عرض الدردشة والأوامر ذات الصلة مدمجين بشكل جيد في هذه المرحلة. إذا كنت بحاجة إلى تأكيد دقيق، يمكنك التحقق من لوحة الأوامر في VS Code 1.99.2 عن طريق كتابة `code chat` لمعرفة ما إذا كان يظهر كأمر صالح.
