لاستخراج معلومات النظام على macOS، يمكنك استخدام عدة أدوات ومفاهيم مدمجة. إليك بعض الطرق لاستخراج أنواع مختلفة من معلومات النظام:

### 1. **مُستخدم معلومات النظام**
تقدم تطبيق معلومات النظام نظرة شاملة على الأجهزة والبرامج على ماك.

- افتح **بحث السطوع** بالضغط على `Cmd + Space` واكتب "معلومات النظام" ثم اضغط على Enter.
- أو يمكنك فتحه من **قائمة Apple** > **حول هذا ماك** > **تقرير النظام**.

### 2. **حول هذا ماك**
يقدم نظرة سريعة على مواصفات ماك.

- انقر على **قائمة Apple** في الزاوية اليسرى العليا من الشاشة.
- اختر **حول هذا ماك**. سيظهر معلومات أساسية مثل إصدار macOS، المعالج، الذاكرة، ورقم السلسلة.

### 3. **أوامر الترمينال**
يمكنك استخدام الترمينال للحصول على معلومات النظام التفصيلية باستخدام أوامر مختلفة.

- افتح **الترمينال** من `التطبيقات` > `الادوات` > `الترمينال` أو عن طريق البحث عنه في السطوع.

#### معلومات النظام الأساسية
```sh
system_profiler SPHardwareDataType
```

#### معلومات النظام التفصيلية
```sh
system_profiler
```

#### معلومات المعالج
```sh
sysctl -n machdep.cpu.brand_string
```

#### معلومات الذاكرة
```sh
sysctl hw.memsize
```

#### استخدام القرص
```sh
df -h
```

#### معلومات الشبكة
```sh
ifconfig
```

#### معلومات البطارية (للحواسيب المحمولة)
```sh
pmset -g batt
```

#### قائمة البرامج المثبته
```sh
ls /Applications
```

#### إصدار macOS
```sh
sw_vers
```

#### معلومات المستخدم
```sh
id
```

#### قائمة العمليات الجارية
```sh
ps aux
```

#### قائمة حزم Brew المثبته
```sh
brew list
```

#### قائمة حزم Pip المثبته
```sh
pip list
```

#### قائمة حزم Gem المثبته
```sh
gem list
```

### 4. **مراقب النشاط**
يقدم هذا التطبيق معلومات حية عن استخدام المعالج، الذاكرة، الطاقة، القرص، والشبكة على ماك.

- افتح **مراقب النشاط** من `التطبيقات` > `الادوات` > `مراقب النشاط` أو عن طريق البحث عنه في السطوع.

### 5. **اداة القرص**
يقدم هذا التطبيق معلومات عن أجهزة التخزين.

- افتح **اداة القرص** من `التطبيقات` > `الادوات` > `اداة القرص` أو عن طريق البحث عنه في السطوع.

### 6. **الادوات الثالثة**
هناك أيضًا تطبيقات ثالثة مثل **iStat Menus** و **CoconutBattery** التي تقدم معلومات النظام التفصيلية وقدرات المراقبة.
