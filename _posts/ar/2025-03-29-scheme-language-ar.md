سكيْم هي لهجة بسيطة من لِسْب تُعرف ببساطتها وأناقتها. سيعلمك هذا البرنامج التعليمي الأساسيات لتبدأ.

## 1. البنية الأساسية والمفاهيم

### تشغيل سكيْم
- ستحتاج إلى مُفسِّر (مترجم) لسكيْم (مثل Racket أو Guile أو Chez Scheme)
- يُكتب الكود عادةً في ملف `.scm` أو يُدخل في بيئة REPL (حلقة القراءة-التقييم-الطباعة)

### التعبيرات والأقواس
- كل شيء في سكيْم هو تعبير يُرجع قيمة
- تُستخدم الأقواس من أجل استدعاءات الدوال والتجميع
- التنسيق: `(دالة وسيط1 وسيط2 ...)`

## 5. الدوال

### تعريف الدوال
```scheme
; تعريف دالة أساسية
(define square
  (lambda (x)    ; lambda تنشئ دالة مجهولة الاسم
    (* x x)))

(square 4)     ; 16
```

### معاملات متعددة
```scheme
(define add
  (lambda (x y)
    (+ x y)))

(add 3 5)      ; 8
```

### تعريف مختصر
```scheme
; صيغة بديلة (سكر نحوي)
(define (multiply x y)
  (* x y))

(multiply 2 3) ; 6
```

## 6. الشروط

### عبارة If
```scheme
(define (is-positive? n)
  (if (> n 0)
      #t
      #f))

(is-positive? 5)   ; #t
(is-positive? -2)  ; #f
```

### Cond (شروط متعددة)
```scheme
(define (number-type n)
  (cond
    ((> n 0) "موجب")
    ((< n 0) "سالب")
    (else "صفر")))

(number-type 5)    ; "موجب"
(number-type 0)    ; "صفر"
```

## 7. القوائم

### إنشاء القوائم
```scheme
; استخدام quote
'(1 2 3)          ; قائمة أرقام

; استخدام دالة list
(list 1 2 3)      ; نفس ما سبق

; استخدام cons (بناء)
(cons 1 '(2 3))   ; نفس ما سبق
```

### عمليات على القوائم
```scheme
(car '(1 2 3))    ; 1 (العنصر الأول)
(cdr '(1 2 3))    ; (2 3) (بقية القائمة)
(null? '())       ; #t (التحقق إذا كانت فارغة)
(length '(1 2 3)) ; 3
```

## 8. التكرار العودي (الاستدعاء الذاتي)

### تكرار عودي بسيط
```scheme
(define (factorial n)
  (if (= n 0)
      1
      (* n (factorial (- n 1)))))

(factorial 5)     ; 120 (5 * 4 * 3 * 2 * 1)
```

### تكرار عودي على القوائم
```scheme
(define (sum-list lst)
  (if (null? lst)
      0
      (+ (car lst) (sum-list (cdr lst)))))

(sum-list '(1 2 3 4))  ; 10
```

## 9. دوال الرتبة الأعلى

### التعيين (Mapping)
```scheme
(map square '(1 2 3 4))    ; (1 4 9 16)
(map + '(1 2 3) '(4 5 6))  ; (5 7 9)
```

### التصفية (Filtering)
```scheme
(define (even? n) (= (remainder n 2) 0))
(filter even? '(1 2 3 4 5 6))  ; (2 4 6)
```

## 10. ربط المتغيرات محلياً (Let)

```scheme
; ربط متغيرات محلية
(define (circle-area r)
  (let ((pi 3.14159))
    (* pi (square r))))

(circle-area 2)    ; 12.56636
```

## 12. برنامج مثال

إليك برنامجاً كاملاً يجمع بين عدة مفاهيم:

```scheme
; برنامج لحساب متوسط قائمة من الأرقام
(define (average lst)
  (define (sum lst)
    (if (null? lst)
        0
        (+ (car lst) (sum (cdr lst)))))
  (define len (length lst))
  (if (= len 0)
      0
      (/ (sum lst) len)))

; اختبار البرنامج
(display (average '(1 2 3 4 5)))  ; 3
(newline)
(display (average '()))           ; 0
```

## الميزات الرئيسية لسكيْم

1. **بسيطة**: نواة لغة صغيرة مع ميزات قوية
2. **وظيفية**: تركيز على الدوال الخالصة وعدم قابلية البيانات للتغيير
3. **صيغة البادئة**: جميع العمليات تستخدم صيغة البادئة
4. **دوال من الدرجة الأولى**: يمكن تمرير الدوال كمعاملات
5. **تحسين ذيل الاستدعاء**: تكرار عودي فعال

## نصائح للتعلم

1. تدرب في بيئة REPL لرؤية النتائج فوراً
2. ابدأ بدوال بسيطة وابنِ التعقيد تدريجياً
3. فكر بطريقة عودية عند معالجة القوائم
4. استخدم المسافات البادئة المناسبة لإدارة الأقواس
5. جرب دوال الرتبة الأعلى

يغطي هذا البرنامج التعليمي المفاهيم الأساسية لسكيْم. لتعميق فهمك، جرب:
- كتابة دوال عودية أكثر تعقيداً
- استكشاف أنظمة الماكرو (موضوع متقدم)
- بناء مشاريع صغيرة مثل الآلات الحاسبة أو معالجات القوائم
- دراسة المكتبات القياسية لسكيْم (تختلف حسب التنفيذ)

هل تريد مني التفصيل في قسم معين أو تقديم المزيد من الأمثلة؟
