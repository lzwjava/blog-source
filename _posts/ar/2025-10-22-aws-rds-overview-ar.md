### قواعد البيانات العلائقية على AWS

تقدم AWS العديد من الخدمات المدارة لقواعد البيانات العلائقية، حيث تُعد **خدمة Amazon Relational Database Service (RDS)** العرض الأساسي. فهي تتعامل مع الإعداد، التصحيح، النسخ الاحتياطي، والتحجيم لمحركات قواعد البيانات الشائعة المختلفة. بالإضافة إلى ذلك، تُعد **Amazon Aurora** قاعدة بيانات علائقية عالية الأداء ومتوافقة مع MySQL وPostgreSQL ومدمجة في RDS.

إليك ملخصًا لأبرز خيارات قواعد البيانات العلائقية على AWS:

| الخدمة/المحرك | الوصف | حالات الاستخدام |
|---------------|-------------|-----------|
| **Amazon RDS for PostgreSQL** | PostgreSQL مُدارة بالكامل، تدعم ميزات متقدمة مثل JSONB والبحث النصي الكامل. | التطبيقات المعاملاتية، تطبيقات الويب، والتحليلات. |
| **Amazon RDS for MySQL** | MySQL مُدارة مع توافر عالٍ ونسخ مكررة للقراءة. | التجارة الإلكترونية، إدارة المحتوى، والتطبيقات ذات الأغراض العامة. |
| **Amazon RDS for MariaDB** | فرع من MySQL مع تحسينات في الأداء ومحركات التخزين. | تطبيقات الويب وأحمال العمل كثيفة القراءة. |
| **Amazon RDS for SQL Server** | Microsoft SQL Server مع إصدارات مثل Standard وEnterprise وWeb. | تطبيقات المؤسسات، ذكاء الأعمال، والأنظمة القائمة على Windows. |
| **Amazon RDS for Oracle** | Oracle Database مع خيارات BYOL (إحضار ترخيصك الخاص) أو الترخيص المضمن. | تطبيقات المؤسسات الحرجة. |
| **Amazon RDS for Db2** | IBM Db2 مع دعم LUW (Linux, UNIX, Windows). | أحمال العمل المعاملاتية والتحليلية واسعة النطاق. |
| **Amazon Aurora (متوافقة مع MySQL)** | قاعدة بيانات علائقية سحابية أصلية بأداء أسرع بمرة تصل إلى 5 مرات من MySQL القياسي. | OLTP عالي الإنتاجية والتطبيقات العالمية. |
| **Amazon Aurora (متوافقة مع PostgreSQL)** | متوافقة مع PostgreSQL مع خيارات بدون خادم وتخزين قابل للتوسع تلقائيًا. | التطبيقات القابلة للتوسع التي تتطلب التوافق مع ACID والتوافر العالي. |

لتخزين البيانات (أحمال العمل العلائقية المركزة على OLAP)، تقدم AWS أيضًا **Amazon Redshift**، وهي مستودع بيانات مُدار بالكامل وقابل للتوسع إلى مستوى البيتابايت ومبني على PostgreSQL.

تتكامل هذه الخدمات بسلاسة مع أدوات AWS الأخرى مثل EC2 وLambda وS3. للبدء، راجع [توثيق AWS RDS](https://aws.amazon.com/rds/) أو [نظرة عامة على خدمات قاعدة بيانات AWS](https://docs.aws.amazon.com/databases-on-aws-how-to-choose/).
