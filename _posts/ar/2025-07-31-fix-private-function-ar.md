يشير الخطأ `E0603: function run_simple is private` إلى أن الدالة `run_simple` في الملف `simple.rs` هي خاصة (تفتق إلى الكلمة المفتاحية `pub`)، لذا لا يمكن الوصول إليها من `main.rs`. في لغة Rust، تكون الدوال خاصة بشكل افتراضي وتحتاج إلى أن يتم جعلها عامة بشكل صريح باستخدام `pub` لتكون قابلة للوصول خارج الوحدة النمطية الخاصة بها.

### خطوات إصلاح الخطأ

1. **جعل الدالة `run_simple` عامة**:
   - افتح ملف `src/simple.rs`.
   - أضف الكلمة المفتاحية `pub` إلى الدالة `run_simple` لجعلها قابلة للوصول بشكل عام.
   - مثال:
     ```rust
     pub fn run_simple() {
         // جسم الدالة الحالي
         println!("Hello from run_simple!");
     }
     ```

2. **التأكد من الإعلان الصحيح للوحدة النمطية**:
   - في ملف `src/main.rs`، تأكد من أنك أعلنت الوحدة النمطية `simple` بشكل صحيح:
     ```rust
     mod simple; // يعلن عن الوحدة النمطية simple

     fn main() {
         simple::run_simple(); // يستدعي الدالة العامة
     }
     ```

3. **إعادة البناء والتشغيل**:
   - بعد جعل الدالة عامة، أعد بناء المشروع:
     ```bash
     cargo build
     ```
   - إذا نجحت عملية البناء، شغل البرنامج:
     ```bash
     cargo run
     ```

4. **التحقق من وجود مشاكل أخرى**:
   - يشير الخطأ إلى `src/simple.rs:47:1`، لذا تحقق من أن الدالة `run_simple` في ذلك السطر محددة بشكل صحيح وليس لديها مشاكل أخرى (مثل أخطاء في الصيغة أو تبعيات مفقودة).
   - إذا كان `simple.rs` يستخدم حزم خارجية، تأكد من إدراجها في `Cargo.toml` تحت `[dependencies]`.
