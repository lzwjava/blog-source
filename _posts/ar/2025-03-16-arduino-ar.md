Arduino ليس شيئًا "تلعبه" مثل لعبة — إنه أداة لإنشاء مشاريع تفاعلية. يمكنك برمجته للتحكم في الأضواء، وأجهزة الاستشعار، والمحركات، والمزيد. أدناه، سأرشدك خلال أساسيات البدء وسأقدم بعض الأمثلة المناسبة للمبتدئين.

### **كيف تبدأ مع Arduino**
1. **ما تحتاجه:**
   - لوحة Arduino (مثل Arduino Uno، وهي لوحة شائعة للمبتدئين).
   - كابل USB (عادة من النوع USB-A إلى USB-B للوحة Uno).
   - كمبيوتر مع تثبيت بيئة التطوير المتكاملة (IDE) لـ Arduino — يمكنك تنزيلها مجانًا من [arduino.cc](https://www.arduino.cc/en/software).
   - مكونات أساسية مثل مصابيح LED، ومقاومات، ولوحة تجارب (Breadboard)، وأسلاك توصيل (اختيارية ولكنها مفيدة للأمثلة).

2. **الإعداد:**
   - صل Arduino بجهاز الكمبيوتر الخاص بك via كابل USB.
   - افتح Arduino IDE، واختر لوحتك (مثل "Arduino Uno") من خلال `Tools > Board`، واختر المنفذ الصحيح من خلال `Tools > Port`.

3. **البرمجة:**
   - يستخدم Arduino نسخة مبسطة من لغة C/C++. تكتب "سكتشات" (برامج) تحتوي على وظيفتين رئيسيتين:
     - `setup()`: تعمل مرة واحدة عند بدء تشغيل Arduino.
     - `loop()`: تعمل بشكل متكرر بعد الإعداد.
   - انقل الكود إلى اللوحة باستخدام زر "Upload" في بيئة التطوير المتكاملة.

4. **ابدأ صغيرًا:**
   - ابدأ بمشاريع بسيطة لفهم كيفية عملها، ثم تدرج في التعقيد.

### **مشاريع مثاليه**

#### **1. تشغيل وإطفاء LED (مشروع Hello World لـ Arduino)**
يستخدم هذا المشروع LED المدمج في المنفذ 13 في معظم لوحات Arduino.
```cpp
void setup() {
  pinMode(13, OUTPUT); // ضبط المنفذ 13 كمخرج
}

void loop() {
  digitalWrite(13, HIGH); // تشغيل LED
  delay(1000);            // انتظر ثانية واحدة
  digitalWrite(13, LOW);  // إطفاء LED
  delay(1000);            // انتظر ثانية واحدة
}
```
- **كيفية العمل:** يومض LED كل ثانية.
- **المكونات:** لا حاجة لمكونات إضافية — فقط لوحة Arduino.

#### **2. LED يتم التحكم فيه بزر**
التحكم في LED خارجي باستخدام زر ضغط.
- **المكونات:** LED، مقاوم 220 أوم، زر ضغط، لوحة تجارب، أسلاك.
- **التوصيلات:**
   - الطرف الموجب لـ LED (الرجل الأطول) إلى المنفذ 9 عبر المقاوم، والطرف السالب إلى GND.
   - الزر: جانب واحد إلى المنفذ 2، والجانب الآخر إلى GND (استخدم المقاوم السحبي الداخلي).

```cpp
int ledPin = 9;   // LED موصل بالمنفذ 9
int buttonPin = 2; // الزر موصل بالمنفذ 2

void setup() {
  pinMode(ledPin, OUTPUT);      // ضبط منفذ LED كمخرج
  pinMode(buttonPin, INPUT_PULLUP); // ضبط منفذ الزر كمدخل مع مقاوم سحبي
}

void loop() {
  int buttonState = digitalRead(buttonPin); // قراءة حالة الزر
  if (buttonState == LOW) {     // الزر مضغوط (LOW بسبب المقاوم السحبي)
    digitalWrite(ledPin, HIGH); // تشغيل LED
  } else {
    digitalWrite(ledPin, LOW);  // إطفاء LED
  }
}
```
- **كيفية العمل:** اضغط الزر لتشغيل LED؛ أفلته لإطفائه.

#### **3. تفتيح وتعتيم LED (مثال على PWM)**
استخدم تعديل عرض النبضة (PWM) لتفتيت وتعتيم LED.
- **المكونات:** LED، مقاوم 220 أوم.
- **التوصيلات:** LED إلى المنفذ 9 (منفذ يدعم PWM)، المقاوم إلى GND.

```cpp
int ledPin = 9; // LED على المنفذ 9 (PWM)

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  for (int brightness = 0; brightness <= 255; brightness += 5) {
    analogWrite(ledPin, brightness); // ضبط شدة الإضاءة (0-255)
    delay(30);                       // انتظر قليلاً
  }
  for (int brightness = 255; brightness >= 0; brightness -= 5) {
    analogWrite(ledPin, brightness);
    delay(30);
  }
}
```
- **كيفية العمل:** يتم تفتيت LED من الإطفاء إلى أقصى سطوع، ثم العودة مرة أخرى، بشكل متكرر.
