أمر Git:

```bash
git clone --filter=blob:none <رابط-المستودع>
```

هو **استنساخ جزئي** يقوم بتنزيل **محتويات الملفات تقريبًا بدون أي محتويات** (blobs) في وقت الاستخراج. إليك بالضبط ما يفعله ولماذا هو مفيد:

### ماذا يعني `--filter=blob:none`
- `--filter=blob:none` يخبر Git باستخدام **الاستنساخ بدون blobs**.
- لا يزال Git يقوم بتنزيل:
  - جميع الحفظات (السجل)
  - جميع كائنات الشجرة (هيكل الدليل)
  - جميع المراجع/الفروع/الوسوم
- لكنه يقوم بتنزيل **صفر من محتويات الملفات** (blobs) أثناء عملية الاستنساخ الأولية.
- عندما تقوم لاحقًا بـ `git checkout` أو تبديل الفروع، يقوم Git بتنزيل الـ blobs (محتويات الملفات) للملفات المطلوبة فعليًا في شجرة العمل الخاصة بك في تلك اللحظة فقط.

### الفوائد الرئيسية
- استنساخ أولي سريع جدًا وصغير الحجم (غالبًا بضع ميجابايت فقط حتى للمستودعات الضخمة مثل نواة Linux، أو Android، أو كود مصدر Windows).
- يستخدم نطاق ترددي للشبكة ومساحة تخزين أقل بكثير في البداية.
- مثالي لـ CI/CD، أو البحث السريع، أو عندما تحتاج فقط إلى السجل (مثل `git log`، أو blame، أو diff عبر الحفظات) دون استخراج الملفات.

### مثال للمقارنة في الحجم (من العالم الحقيقي)

| المستودع          | الاستنساخ العادي | استنساخ `blob:none` | مكسب السرعة |
|-------------------|------------------|---------------------|-------------|
| torvalds/linux    | ~5+ جيجابايت     | ~300–400 ميجابايت   | أسرع بـ 10–20 مرة |
| microsoft/vscode  | ~2–3 جيجابايت   | ~200 ميجابايت       | أسرع بكثير     |

### الأوامر الشائعة المتابعة بعد استنساخ blob:none
```bash
# أول مرة تحتاج فيها إلى شجرة عمل:
git checkout <الفرع>        # يقوم بتنزيل الـ blobs المطلوبة فقط

# أو إذا أردت كل شيء لاحقًا:
git fetch --unshallow        # غير مطلوب لـ blobless
# لملء الـ blobs المفقودة بشكل دائم:
git repack -a -d             # اختياري، يجبر على تنزيل جميع الـ blobs
```

### المرشحات ذات الصلة التي قد تراها
- `--filter=blob:none` → لا توجد blobs على الإطلاق في البداية (الأكثر عدوانية)
- `--filter=tree:0`    → يقوم بتنزيل الأشجار ولكن لا توجد blobs (تأثير مشابه)
- `--filter=sparse:oid=<oid>` → للاستخراج المتناثر مع blobs جزئية

### الملخص
```bash
git clone --filter=blob:none https://github.com/example/repo.git
```
