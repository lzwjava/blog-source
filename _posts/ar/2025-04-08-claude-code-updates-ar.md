### أبرز التحديثات في Claude Code

شهد Claude Code، مساعد البرمجة المدعوم بالذكاء الاصطناعي من Anthropic، تحسينات كبيرة مؤخرًا، خاصة فيما يتعلق بالاستقلالية والتكامل وميزات واجهة المستخدم. استنادًا إلى أحدث الأخبار (29 سبتمبر 2025)، يركز التحديث الرئيسي على تمكين التشغيل الأكثر استقلالية، بما في ذلك امتداد VS Code الأصلي، وترقية واجهة الطرفية (الإصدار 2.0)، ووظيفة نقطة التفتيش الجديدة لإدارة المهام طويلة التشغيل. بالإضافة إلى ذلك، يبدو أن ميزات مثل تبديل وضع "التفكير" هي جزء من التحسينات المستمرة، مما يسمح للمستخدمين بتشغيل وإيقاف عرض خطوات استدلال Claude لتحقيق تفاعلات أكثر سلاسة [1].

#### الاستقلالية وقدرات الوكيل
- **امتداد VS Code الأصلي**: يمكّن التكامل السلس مع محرر VS Code، مما يسمح لـ Claude Code بالعمل مباشرة داخل بيئة التطوير لتحرير الأكواد وتصحيح الأخطاء بشكل أكثر استقلالية.
- **واجهة الطرفية الإصدار 2.0**: تشمل الترقيات تحسين التعامل مع الأذونات، وإدارة الذاكرة عبر المهام، وتنسيق الوكلاء الفرعيين. هذا يجعل Claude Code أفضل في موازنة تحكم المستخدم مع العمليات الآلية أثناء سير العمل المعقد [1][2].
- **نقاط التفتيش**: ميزة جديدة لحفظ التقدم في المهام الطويلة، تسمح بالإيقاف المؤقت والاستئناف دون فقدان السياق. وهذا يدعم تشغيل المهام التي تمتد على عدة أيام أو جلسات.

تبني هذه التغييرات على Claude Agent SDK من Anthropic، مما يوفر للمطورين أدوات لإنشاء وكلاء ذكاء اصطناعي مخصصة تعكس البنية التحتية الداخلية لـ Claude Code [2].

#### تغييرات أخرى ملحوظة
- **تكامل ترقية النموذج**: يستخدم Claude Code الآن Claude Sonnet 4.5 افتراضيًا، مما يقدم أداءً محسنًا في مهام البرمجة دون تغييرات في السعر. يوصى بهذا النموذج عبر جميع منتجات Claude لتحسين محاذاة النتائج وتقليل الهلوسة [2].
- **تعديلات على واجهة المستخدم**: يبدو أن ميزة "إيقاف التفكير (Tab للتبديل)" هي مفتاح لإظهار أو إخفاء عملية الاستدلال الداخلية لـ Claude، مما يمكن أن يبسط المخرجات. يتوافق هذا مع ملاحظات المستخدمين حول الشفافية وسهولة الاستخدام في جلسات البرمجة [3].
- **محاولات إصلاح المشكلات المعروفة**: يتم معالجة تقارير عن قيام Claude Code بإلحاق "2024" بعمليات البحث بسبب انقطاع بيانات التدريب، على الرغم من أن بعض المستخدمين يلاحظون استمرار وجود أخطاء. يتضمن الحل المؤقت استخدام خطاطيف مخصصة لتحديث الوعي التاريخي [3][4].

للحصول على التفاصيل الكاملة، قم بتثبيت أو التحقق من أحدث إصدار — يتم تطبيق التحديثات تلقائيًا لجميع المستخدمين [1]. إذا كنت تواجه مشكلات، تأكد من أن عميلك محدث. للتغييرات السابقة، تحقق من سجلات التغيير المجتمعية مثل ClaudeLog، على الرغم من أنها قد لا تغطي أحدث الميزات المطلقة [5].
