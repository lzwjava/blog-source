### إضافة GitHub Copilot لـ VS Code (الإصدار 1.388.0)

تم إصدار الإصدار 1.388.0 من إضافة GitHub Copilot في 24 أكتوبر 2025. يبدو أن هذا تحديث ثانوي يركز على تحسينات الاستقرار والتوافق، خاصة مع أحدث إصدار من VS Code (1.105). لم يتم تفصيل ملاحظات الإصدار المحددة علنًا في المتجر الإلكتروني أو مدونة GitHub، لكن تقارير المستخدمين تشير إلى إصلاحات لمشكلات مثل إدخال الكود غير المرغوب فيه أثناء التوليد المضمن وإكمال العلامات الإضافية عند إلغاء الاقتراحات. تندمج بسلاسة مع ميزات Copilot الحديثة، بما في ذلك أوضاع الوكيل المعززة واختيارات النموذج.

#### أبرز التحديثات في آخر 6 أشهر (مايو–أكتوبر 2025)
يتم الإعلان عادةً عن التحسينات الرئيسية لـ GitHub Copilot بالتزامن مع إصدارات VS Code الشهرية. إليك ملخصًا لأبرز التحديثات للإضافة والميزات المرتبطة خلال هذه الفترة:

-   **أكتوبر 2025 (VS Code 1.105 / الإضافة ~1.388)**:
    - تكامل OpenAI Codex متاح الآن في VS Code Insiders لمشتركي Copilot Pro+، مما يتيح تركيب كود متقدم مباشرة في المحرر.
    - واجهة "مركز التحكم" جديدة لتعيين وتوجيه ومتابعة مهام وكيل البرمجة Copilot عبر الجلسات.
    - تم توسيع عرض جلسات الوكيل لدعم GitHub Copilot CLI لإدارة الوكلاء المحليين والمعتمدين على السحابة.

-   **سبتمبر 2025 (VS Code 1.104 / الإضافة ~1.38x)**:
    - طرح نموذج GitHub Copilot-SWE التجريبي في VS Code Insiders، والمُحسّن لتحرير الكود وإعادة الهيكلة والتحولات الصغيرة. إنه يركز على المهام ويعمل في أي وضع دردشة؛ يُوصى باستخدام مطالبات مفصلة للحصول على أفضل النتائج.
    - تحسين الدردشة المضمنة لأخطاء الطرفية، مع تفسيرات وإصلاحات أفضل.

-   **أغسطس 2025 (VS Code 1.103 / الإضافة ~1.37x)**:
    - تحسين اقتراحات رسائل الالتزام مع الوعي بالسياق متعدد الأسطر والتكامل مع @workspace لتوليد هياكل المشاريع بالكامل.
    - ترقيات للدردشة المضمنة الخفيفة لتفاعلات أسرع دون فتح نوافذ كاملة.

-   **يوليو 2025 (VS Code 1.102 / الإضافة ~1.36x)**:
    - تنسيق أفضل لتحرير الملفات المتعددة عبر مطالبات فردية، مع تحليل هيكل المشروع لإجراء تغييرات متسقة.
    - إيقاف استخدام النماذج القديمة (مثل بعض إصدارات Claude وOpenAI وGemini) لصالح خيارات أحدث وأسرع مثل GPT-4.1.

-   **يونيو 2025 (VS Code 1.101 / الإضافة ~1.35x)**:
    - إدخال ملفات التعليمات والمطالبات لتخصيص سلوك Copilot باستخدام إرشادات قابلة لإعادة الاستخدام والمعرفة التنظيمية.
    - توسيع تكامل طلبات السحب (Pull Requests) في GitHub: تعيين طلبات السحب/المشكلات إلى Copilot مباشرة من نوافذ VS Code، مع استعلام جديد "Copilot on My Behalf" للمتابعة.

-   **مايو 2025 (VS Code 1.100 / الإضافة ~1.34x)**:
    - إضافة دعم Model Context Protocol (MCP) لوضع الوكيل، مما يسمح باستخدام مفاتيح API مخصصة للنماذج الخارجية.
    - تحرير أسرع بواسطة الوكيل مدعومًا بـ OpenAI GPT-4.1 و Anthropic Claude Sonnet، خاصة للملفات الكبيرة.
    - تحسينات في البحث داخل المستودع لاكتشاف الكود الدلالي والاقتراحات على مستوى قاعدة الكود.

تركز هذه التحديثات على سير عمل الوكيل، وأداء النموذج، والتكامل السلس مع نظام VS Code البيئي. للحصول على التاريخ الكامل للإضافة، تحقق من VS Code Marketplace. غالبًا ما ترتبط الإعلانات الرئيسية بدورات VS Code الشهرية.
