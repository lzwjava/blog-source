يقدم NPM (مدير حزم Node) عدة مستويات للسجلات تتحكم في مقدار التفاصيل الظاهرة عند تشغيل أوامر npm. إليك هذه المستويات من الأقل إلى الأكثر تفصيلاً:

- **silent**: لا يُظهر أي ناتج
- **error**: يُظهر الأخطاء فقط (الإعداد الافتراضي)
- **warn**: يُظهر الأخطاء والتحذيرات
- **notice**: يُظهر الأخطاء والتحذيرات والإشعارات
- **http**: يُظهر طلبات HTTP
- **timing**: يُظهر معلومات التوقيت
- **info**: يُظهر رسائل المعلومات
- **verbose**: يُظهر معلومات مفصلة
- **silly**: يُظهر كل معلومات التصحيح

يمكنك تعيين مستوى السجل باستخدام علم `--loglevel` أو متغير البيئة `npm_config_loglevel`:

```bash
npm install --loglevel verbose
```

أو يمكنك تعيينه في إعدادات npm الخاصة بك:

```bash
npm config set loglevel verbose
```

مستوى السجل الافتراضي هو "error" لمعظم الأوامر، مما يعني أنك سترى فقط رسائل الخطأ ما لم تطلب بشكل محدد ناتجًا أكثر تفصيلاً.
