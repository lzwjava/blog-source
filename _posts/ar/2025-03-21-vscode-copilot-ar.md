يعد استخدام GitHub Copilot في Visual Studio Code (VS Code) أمرًا مباشرًا بمجرد إعداده. فيما يلي دليل خطوة بخطوة لبدء استخدامك اعتبارًا من 20 مارس 2025:

---

### الخطوة 1: تثبيت VS Code
- تأكد من تثبيت Visual Studio Code. قم بتنزيله من [code.visualstudio.com](https://code.visualstudio.com/) إذا لم تكن قد قمت بذلك بالفعل.
- قم بتحديث VS Code إلى أحدث إصدار للحصول على أفضل توافق (اذهب إلى `Help > Check for Updates`).

---

### الخطوة 2: تثبيت إضافة GitHub Copilot
1. **افتح VS Code**.
2. **انتقل إلى سوق الإضافات (Extensions Marketplace)**:
   - انقر على أيقونة الإضافات في شريط النشاط على اليسار (أو اضغط على `Ctrl+Shift+X` / `Cmd+Shift+X` على جهاز Mac).
3. **ابحث عن "GitHub Copilot"**:
   - اكتب "GitHub Copilot" في شريط البحث.
   - ابحث عن الإضافة الرسمية من GitHub (سيكون لها شارة موثقة).
4. **قم بتثبيت الإضافة**:
   - انقر على زر `Install` بجوار "GitHub Copilot".
5. **اختياري: قم بتثبيت Copilot Chat (موصى به)**:
   - ابحث عن "GitHub Copilot Chat" وقم بتثبيتها أيضًا. تضيف هذه ميزات الذكاء الاصطناعي المحادثي مثل طرح الأسئلة أو إنشاء التعليمات البرمجية عبر الدردشة.

---

### الخطوة 3: تسجيل الدخول إلى GitHub Copilot
1. **المصادقة باستخدام GitHub**:
   - بعد التثبيت، ستظهر مطالبة تطلب منك تسجيل الدخول.
   - انقر على `Sign in to GitHub` في النافذة المنبثقة أو انتقل إلى أيقونة حالة Copilot (في الزاوية اليمنى السفلية من VS Code) واختر "Sign in".
2. **التفويض في المتصفح**:
   - ستفتح نافذة متصفح تطلب منك تسجيل الدخول إلى حساب GitHub الخاص بك.
   - وافق على طلب التفويض بالنقر على `Authorize Git hypoxia`.
3. **انسخ الرمز**:
   - سيوفر GitHub رمزًا لمرة واحدة. انسخه والصقه مرة أخرى في VS Code عندما تتم مطالبتك بذلك.
4. **تحقق من التفعيل**:
   - بمجرد تسجيل الدخول، يجب أن تتحول أيقونة Copilot في شريط الحالة إلى اللون الأخضر، مما يشير إلى أنها نشطة. ستشاهد أيضًا إشعارًا يؤكد وصولك.

---

### الخطوة 4: تكوين Copilot (اختياري)
- **تمكين/تعطيل الاقتراحات**:
  - انتقل إلى `File > Preferences > Settings` (أو `Ctrl+,` / `Cmd+,`).
  - ابحث عن "Copilot" لضبط الإعدادات مثل تمكين الاقتراحات المضمنة أو تعطيلها للغات محددة.
- **تحقق من الاشتراك**:
  - يتطلب Copilot اشتراكًا (10 دولارات/شهر أو 100 دولار/سنة) بعد تجربة مجانية لمدة 30 يومًا. يمكن للطلاب والمعلمين والحفاظ على المشاريع مفتوحة المصدر التقدم للحصول على وصول مجاني عبر [GitHub Education](https://education.github.com/) أو من خلال إعدادات Copilot.

---

### الخطوة 5: ابدأ باستخدام Copilot
إليك كيفية الاستفادة من Copilot في سير عمل البرمجة الخاص بك:

#### 1. **اقتراحات التعليمات البرمجية**
- **الإكمال التلقائي المضمن (Inline Autocomplete)**:
  - ابدأ في الكتابة في ملف (مثل `def calculate_sum(` في Python)، وسيقترح Copilot إكمالات بنص رمادي.
  - اضغط على `Tab` لقبول الاقتراح أو استمر في الكتابة لتجاهله.
- **اقتراحات متعددة الأسطر**:
  - اكتب تعليقًا مثل `// Function to sort an array` واضغط على Enter. قد يقترح Copilot تنفيذًا كاملاً (مثل خوارزمية فرز).
  - استخدم `Alt+]` (أو `Option+]` على جهاز Mac) للتنقل بين اقتراحات متعددة.

#### 2. **إنشاء التعليمات البرمجية من التعليقات**
- اكتب تعليقًا وصفيًا مثل:
  ```javascript
  // Fetch data from an API and handle errors
  ```
  اضغط Enter، وقد يقوم Copilot بتوليد:
  ```javascript
  async function fetchData(url) {
    try {
      const response = await fetch(url);
      if (!response.ok) throw new Error('Network response was not ok');
      return await response.json();
    } catch (error) {
      console.error('Fetch error:', error);
    }
  }
  ```
- اقبل باستخدام `Tab` أو عدل حسب الحاجة.

#### 3. **Copilot Chat (إذا تم تثبيته)**
- **افتح الدردشة**:
  - انقر على أيقونة الدردشة في الشريط الجانبي أو استخدم `Ctrl+Alt+C` (قابل للتخصيص).
- **اطرح أسئلة**:
  - اكتب شيئًا مثل "Explain how Promises work in JavaScript" أو "Write a Python script to read a CSV file."
  - سيرد Copilot في لوحة الدردشة ويمكنه إدراج التعليمات البرمجية مباشرة في المحرر الخاص بك.
- **مساعدة سياقية**:
  - ضع علامة على الكود، انقر بزر الماوس الأيمن، واختر "Ask Copilot" لشرحه أو إعادة هيكلته.

#### 4. **التصحيح (Debugging) والاختبار**
- اكتب تعليقًا مثل `// Write unit tests for this function`، ويمكن لـ Copilot إنشاء حالات اختبار باستخدام أطر عمل مثل Jest أو PyTest.

### نصائح للاستخدام الفعال
- **كن محددًا**: كلما زادت السياق الذي تقدمه (مثل التعليقات، أسماء المتغيرات)، كانت اقتراحات Copilot أفضل.
- **راجع الاقتراحات**: Copilot ليس مثاليًا — تحقق دائمًا من دقة وأمان التعليمات البرمجية التي يقترحها.
- **دعم اللغات**: يعمل بشكل أفضل مع اللغات الشائعة (Python، JavaScript، Java، إلخ) ولكنه يدعم العشرات غيرها.
- **اختصارات لوحة المفاتيح**:
  - `Alt+\` (أو `Option+\` على جهاز Mac): إظهار جميع الاقتراحات يدويًا.
  - `Ctrl+Enter`: فتح لوحة الاقتراحات.

---

### حل المشكلات
- **لا يعمل؟**:
  - تأكد من أنك قمت بتسجيل الدخول (تحقق من شريط الحالة).
  - تحقق من أن اشتراكك نشط في إعدادات حساب GitHub الخاص بك.
  - أعد تشغيل VS Code أو أعد تثبيت الإضافة إذا استمرت المشاكل.
- **لا توجد اقتراحات؟**:
  - تحقق مما إذا كان Copilot ممكّنًا للغة الحالية (يجب أن يكون `editor.inlineSuggest.enabled` مضبوطًا على true في الإعدادات).
