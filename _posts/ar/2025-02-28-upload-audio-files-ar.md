#### Spotify: غياب API عام لتحميلات

من ناحية أخرى، لا يوفر Spotify أي API عام لتحميل ملفات الصوت، سواء كانت للموسيقى أو حلقات البودكاست. تم تصميم API الويب لـ Spotify لاسترجاع البيانات الوصفية، وإدارة قوائم التشغيل، وإدارة التشغيل، وليس لتقديم المحتوى ([API الويب | Spotify for Developers](https://developer.spotify.com/documentation/web-api)). بالنسبة لصانعي البودكاست، يوفر Spotify for Creators واجهة مستخدم لتحميل حلقات، يدعم تنسيقات مثل MP3، M4A، و WAV، ولكن هذه يدوية ولا يمكن تكرارها ([نشر حلقات الصوت - Spotify](https://support.spotify.com/us/creators/article/publishing-audio-episodes/)).
