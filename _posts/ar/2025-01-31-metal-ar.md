## أدناه هو كود للاختبار أداء Metal (GPU) باستخدام PyTorch.

```python
import torch
import argparse
import time

parser = argparse.ArgumentParser(description="اختبار(torch) مع MPS أو CPU.")
parser.add_argument("--device", type=str, default="mps", choices=["mps", "cpu"], help="جهاز للاستخدام (MPS أو CPU)")
args = parser.parse_args()

if args.device == "mps":
    if torch.backends.mps.is_available():
        print("Metal متوفر")
        device = torch.device("mps")
    else:
        print("Metal غير متوفر, استخدام CPU بدلا من ذلك")
        device = torch.device("cpu")
elif args.device == "cpu":
    device = torch.device("cpu")
    print("استخدام CPU")
else:
    print("جهاز غير صالح محدد, استخدام CPU بدلا من ذلك")
    device = torch.device("cpu")

# إنشاء مصفوفة ونقلها إلى الجهاز المحدد
x = torch.randn(5000, 5000, device=device)
y = torch.randn(5000, 5000, device=device)

# أداء حاسوب أكثر تعقيدا
start_time = time.time()
result = torch.matmul(x, y)
for _ in range(10):
    result = torch.matmul(result, y)
end_time = time.time()

# طباعة النتيجة
print(result)
print(f"الوقت المستغرق: {end_time - start_time:.4f} ثواني")
```

النتائج توضح أن MPS أسرع بكثير من CPU. وقت التنفيذ على MPS هو فقط حوالي 0.2% من الوقت على CPU.

```bash
% python scripts/test_metal.py --device cpu
الوقت المستغرق: 2.8784 ثواني

% python scripts/test_metal.py --device mps
الوقت المستغرق: 0.0061 ثواني
```
