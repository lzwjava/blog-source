تم تطوير لغة البرمجة Java بشكل أساسي بواسطة **James Gosling**، الذي يُشار إليه غالبًا باسم "أبو Java"، في شركة Sun Microsystems في أوائل التسعينيات. فيما يلي نظرة موجزة على أبرز المساهمين في إنشاء Java وتطورها:

- **James Gosling**: قاد فريق تطوير Java في Sun Microsystems. قام بتصميم اللغة الأصلية وبنيتها النحوية وآلة Java الافتراضية (JVM)، حيث بدأ المشروع في عام 1991 (كان يُسمى في البداية "Oak").
- **Mike Sheridan**: عضو رئيسي في "فريق Green" الأصلي في Sun، وساهم في الرؤية والتصميم المبكر للغة لأنظمة embedded.
- **Patrick Naughton**: كان أيضًا جزءًا من فريق Green، وساعد في تشكيل الأهداف الأولية وعمل على قابلية نقل اللغة واستقلاليتها عن المنصة.
- **Bill Joy**: أحد مؤسسي Sun، وقدّم مدخلات استراتيجية وأثر على هندسة Java، لا سيما تركيزها على الشبكات والأمان.
- **Guy Steele**: ساهم في مواصفات اللغة، خصوصًا في Java Language Specification، مما ضمن الوضوح والدقة في تصميمها.
- **Arthur van Hoff**: عمل على مترجم Java وبيئة التشغيل، وساعد في جعل Java عملية للاستخدام الواسع النطاق.
- **Tim Lindholm**: ساهم في JVM وتحسينات الأداء، وهو مؤلف مشارك في Java Language Specification.
- **Frank Yellin**: لعب دورًا كبيرًا في نموذج أمان JVM وتنفيذه.

كان الإصدار الأولي لـ Java (JDK 1.0) في عام 1996 جهدًا جماعيًا في Sun Microsystems، حيث كان هؤلاء الأفراد محوريين في تصميمها وتنفيذها. بعد الإصدار، استمر مجتمع Java، بما في ذلك المساهمون في Java Community Process (JCP)، وOracle (بعد الاستحواذ على Sun في 2010)، والمشاريع مفتوحة المصدر مثل OpenJDK، في تطوير اللغة. ومن بين المساهمين البارزين لاحقًا **Brian Goetz**، الذي قاد جهود الميزات الحديثة في Java مثل lambdas ونظام الوحدات (module system).
