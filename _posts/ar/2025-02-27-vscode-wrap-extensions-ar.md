أعتقد أنك تسأل كيف يمكن تهيئة Visual Studio Code (VS Code) لتفعيل التغليف النصي فقط لمبادئ الملفات المحددة. إليك كيفية تحقيق ذلك:

1. **فتح إعدادات VS Code**:
   - اضغط على `Ctrl + ,` (أو `Cmd + ,` على macOS) لفتح الإعدادات، أو اذهب إلى `File > Preferences > Settings`.

2. **استخدام ملف إعدادات JSON**:
   - في واجهة الإعدادات، انقر على أيقونة `{}` في الزاوية العليا اليمنى لفتح ملف `settings.json`. هذا يسمح بدرجة أكبر من التحكم.

3. **تكوين التغليف النصي حسب نوع الملف**:
   - أضف تهيئة باستخدام الصيغة `"[filetype]": {}` لتحديد الإعدادات لمبادئ الملفات المحددة. على سبيل المثال، لتفعيل التغليف النصي فقط لمبادئ `.md` (Markdown) و `.txt` (Text) يمكنك إضافة شيء مثل هذا:

   ```json
   {
       "editor.wordWrap": "off", // الافتراضي: لا تغليف لجميع الملفات
       "[markdown]": {
           "editor.wordWrap": "on" // تغليف فقط لمبادئ .md
       },
       "[plaintext]": {
           "editor.wordWrap": "on" // تغليف فقط لمبادئ .txt
       }
   }
   ```

4. **شرح**:
   - `"editor.wordWrap": "off"` يحدد الافتراضي العالمي على عدم التغليف.
   - `"[markdown]"` و `"[plaintext]"` هو معرفات اللغة لمبادئ Markdown و نص عادي، على التوالي. يمكنك استبدالهما بمعرفات لغة أخرى مثل `[javascript]`, `[python]`, إلخ، بناءً على مبادئ الملفات التي تريدها.
   - `"editor.wordWrap": "on"` يفعّل التغليف لمبادئ الملفات المحددة.

5. **إيجاد معرفات اللغة**:
   - إذا كنت غير متأكد من معرف اللغة لمبادئ الملفات، افتح ملف من ذلك النوع في VS Code، انقر على نمط اللغة في الزاوية السفلى اليمنى (مثل "Plain Text" أو "Python")، واحفظ المعرف الذي يظهر في لوحة الأوامر عند اختيار اللغة.

6. **حفظ و اختبار**:
   - احفظ ملف `settings.json`، وسيعمل VS Code على تطبيق الإعدادات فورًا. افتح ملفات بمبادئ مختلفة لتأكيد السلوك.

هذا النهج يضمن أن التغليف النصي يتم تفعيله فقط لمبادئ الملفات التي تحددها، وترك جميع الملفات الأخرى غير متأثرة. أخبرني إذا كنت تحتاج إلى مساعدة بمبادئ محددة!
