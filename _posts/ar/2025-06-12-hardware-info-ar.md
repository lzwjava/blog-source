للحصول على معلومات شاملة حول أجهزة الكمبيوتر المحمول الخاص بك (الذاكرة العشوائية، وحدة المعالجة المركزية، إلخ) على نظام Ubuntu Linux، يمكنك استخدام مجموعة من أوامر الطرفية والأدوات. فيما يلي دليل خطوة بخطوة لاسترداد معلومات تفصيلية عن النظام:

---

### 1. التحقق من معلومات وحدة المعالجة المركزية (CPU)
للحصول على تفاصيل حول وحدة المعالجة المركزية (نوع المعالج، السرعة، النوى، إلخ):
- افتح الطرفية (Ctrl + Alt + T).
- شغِّل الأمر التالي:
  ```bash
  lscpu
  ```
  - **المخرجات**: تعرض بنية وحدة المعالجة المركزية، وعدد النوى، والخيوط، وساعة السرعة، وأحجام ذاكرة التخزين المؤقت، إلخ.
- لمزيد من المعلومات التفصيلية حول وحدة المعالجة المركزية، بما في ذلك اسم الطراز:
  ```bash
  cat /proc/cpuinfo
  ```
  - ابحث عن حقول مثل `model name`، `cpu MHz`، و `cpu cores`.

---

### 2. التحقق من معلومات الذاكرة العشوائية (RAM)
لعرض إجمالي الذاكرة العشوائية المستخدمة والمتبقية:
- شغِّل:
  ```bash
  free -h
  ```
  - **المخرجات**: تُظهر إجمالي الذاكرة المستخدمة والمتبقية بتنسيق مقروء (مثل GB أو MB).
- للحصول على معلومات تفصيلية عن الذاكرة (مثل النوع، السرعة، والفتحات):
  ```bash
  sudo dmidecode -t memory
  ```
  - **المخرجات**: تقدم تفاصيل حول الذاكرة الفعلية، مثل الحجم، والنوع (مثل DDR4)، والسرعة، واستخدام الفتحات. تتطلب صلاحيات root.

---

### 3. التحقق من معلومات التخزين (HDD/SSD)
للحصول على تفاصيل حول أجهزة التخزين الخاصة بك:
- شغِّل:
  ```bash
  lsblk
  ```
  - **المخرجات**: تسرد جميع أجهزة البلوك (مثل SSD، HDD) مع أحجامها ونقاط التحميل.
- للحصول على معلومات تفصيلية حول الأقراص:
  ```bash
  sudo fdisk -l
  ```
  - **المخرجات**: تُظهر أقسام القرص وأحجامها. تتطلب صلاحيات root.
- للتحقق من نوع القرص (مثل HDD أو SSD) والطراز:
  ```bash
  sudo hdparm -I /dev/sda
  ```
  - استبدل `/dev/sda` بجهاز القرص الخاص بك (يمكنك العثور عليه باستخدام `lsblk`).

---

### 4. التحقق من معلومات بطاقة الرسومات (GPU)
للعثور على تفاصيل حول بطاقة الرسومات الخاصة بك:
- شغِّل:
  ```bash
  lspci | grep -i vga
  ```
  أو
  ```bash
  lspci | grep -i nvidia
  ```
  - **المخرجات**: تعرض طراز بطاقة الرسومات (مثل Intel، NVIDIA، AMD).
- لمزيد من المعلومات التفصيلية حول بطاقة الرسومات:
  ```bash
  glxinfo | grep "OpenGL renderer"
  ```
  - يتطلب الحزمة `mesa-utils` (`sudo apt install mesa-utils` إذا لم تكن مثبتة).
- لبطاقات NVIDIA، استخدم:
  ```bash
  nvidia-smi
  ```
  - **المخرجات**: تُظهر استخدام بطاقة الرسومات، وإصدار برنامج التشغيل، وذاكرة VRAM (إذا كانت برامج تشغيل NVIDIA مثبتة).

---

### 5. نظرة عامة على النظام
للحصول على نظرة شاملة على نظامك (وحدة المعالجة المركزية، الذاكرة العشوائية، اللوحة الأم، إلخ):
- شغِّل:
  ```bash
  sudo lshw
  ```
  - **المخرجات**: تسرد معلومات تفصيلية عن الأجهزة، بما في ذلك وحدة المعالجة المركزية، والذاكرة العشوائية، والتخزين، والمزيد. استخدم `sudo lshw -short` للحصول على نسخة موجزة.
- بدلاً من ذلك، قم بتثبيت واستخدام `hardinfo` للحصول على واجهة رسومية:
  ```bash
  sudo apt install hardinfo
  hardinfo
  ```
  - **المخرجات**: تفتح واجهة مستخدم رسومية تُظهر معلومات تفصيلية عن النظام (وحدة المعالجة المركزية، الذاكرة العشوائية، التخزين، أجهزة الاستشعار، إلخ).

---

### 6. التحقق من معلومات BIOS/UEFI واللوحة الأم
للحصول على تفاصيل BIOS/UEFI واللوحة الأم:
- شغِّل:
  ```bash
  sudo dmidecode -t bios
  ```
  - **المخرجات**: تُظهر إصدار BIOS، والبائع، وتاريخ الإصدار.
- للحصول على تفاصيل اللوحة الأم:
  ```bash
  sudo dmidecode -t baseboard
  ```
  - **المخرجات**: تعرض الشركة المصنعة للوحة الأم، والطراز، ورقم التسلسلي.

---

### 7. التحقق من تفاصيل نظام التشغيل والنواة
للتحقق من إصدار Ubuntu والنواة:
- شغِّل:
  ```bash
  lsb_release -a
  ```
  - **المخرجات**: تُظهر إصدار Ubuntu وتفاصيل الإصدار.
- للحصول على معلومات النواة:
  ```bash
  uname -r
  ```
  - **المخرجات**: تعرض إصدار نواة Linux.

---

### 8. مراقبة موارد النظام في الوقت الفعلي
لمراقبة استخدام وحدة المعالجة المركزية والذاكرة العشوائية والعمليات في الوقت الفعلي:
- شغِّل:
  ```bash
  top
  ```
  أو
  ```bash
  htop
  ```
  - **ملاحظة**: قم بتثبيت `htop` إذا لم يكن موجودًا (`sudo apt install htop`). فهو يوفر واجهة أكثر سهولة للمستخدم.

---

### 9. تقرير نظام شامل باستخدام `inxi`
لأمر واحد يجمع معلومات نظام موسعة:
- قم بتثبيت `inxi`:
  ```bash
  sudo apt install inxi
  ```
- شغِّل:
  ```bash
  inxi -Fxz
  ```
  - **المخرجات**: تقدم تقريرًا مفصلاً يشمل وحدة المعالجة المركزية، والذاكرة العشوائية، وبطاقة الرسومات، والتخزين، والشبكة، والمزيد. علم `-F` يعطي تقريرًا كاملاً، و`-x` يضيف تفاصيل إضافية، و`-z` يقوم بتصفية المعلومات الحساسة.

### ملاحظات
- بعض الأوامر (مثل `dmidecode`، `fdisk`) تتطلب صلاحيات root (`sudo`).
- إذا لم يكن الأمر مثبتًا (مثل `inxi`، `htop`، `mesa-utils`)، فقم بتثبيته باستخدام `sudo apt install <package-name>`.
- للحصول على أداة رسومية، يمكن لـ `hardinfo` أو `gnome-system-monitor` (المثبت مسبقًا على Ubuntu) توفير واجهة سهلة للمستخدم.
- إذا كنت بحاجة إلى تفاصيل محددة (مثل معلومات البطارية، محولات الشبكة)، دعني أعرف، ويمكنني تقديم أوامر إضافية.

دعني أعرف إذا كنت بحاجة إلى مساعدة في تفسير المخرجات أو ترغب في التركيز على مكون معين!
