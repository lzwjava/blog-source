## 1. قراءة المدخلات من الكونسول

### 1.1 استخدام `Scanner`

تقرأ جافا المدخلات باستخدام الصنف `Scanner` الموجود في الحزمة `java.util`.

```java
import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("ادخل اسمك: ");
        String name = sc.nextLine();
        System.out.print("ادخل عمرك: ");
        int age = sc.nextInt();
        System.out.println("أهلاً " + name + "، عمرك " + age + " سنة.");
        sc.close();
    }
}
```

**نقاط رئيسية:**

* استخدم `import java.util.Scanner;` دائمًا.
* استخدم `nextInt()`، `nextDouble()`، `nextLine()` بناءً على نوع البيانات.
* أغلق الـ `Scanner` لتحرير الموارد.
* كن حذرًا: `nextLine()` تقرأ بقية السطر، لذا فإن خلطها مع `nextInt()` قد يتسبب في تخطي بعض المدخلات.

## 2. المعرّفات، المتغيرات، التعابير، التعيينات، والثوابت

### 2.1 المعرّفات

هي الأسماء التي تطلقها على المتغيرات، الدوال، أو الأصناف.
**القواعد:**

* يجب أن تبدأ بحرف، `_`، أو `$`.
* لا يمكن أن تبدأ برقم.
* حساسة لحالة الأحرف (`score` ≠ `Score`).
* لا يمكن أن تكون كلمة محجوزة (`int`، `class`، `if`، إلخ).

**أمثلة:**
`studentName`, `_total`, `$price`

### 2.2 المتغيرات

المتغير يحمل بيانات من **نوع** معين.
أمثلة على التصريح:

```java
int age = 20;
double price = 12.5;
char grade = 'A';
boolean passed = true;
```

### 2.3 جمل التعيين

عيّن قيمة باستخدام `=` (من اليمين → إلى اليسار):

```java
x = 5;
y = x + 2;
```

### 2.4 الثوابت

يتم التصريح عنها باستخدام `final`، ولا يمكن تغييرها لاحقًا:

```java
final double PI = 3.14159;
```

استخدم أسماء بالأحرف الكبيرة للثوابت.

## 3. أنواع البيانات العددية والعمليات عليها

### 3.1 الأنواع العددية الشائعة

* `byte` (عدد صحيح 8-بت)
* `short` (16-بت)
* `int` (32-بت، الأكثر شيوعًا)
* `long` (64-بت)
* `float` (عدد عشري 32-بت)
* `double` (عدد عشري 64-بت، أكثر دقة)

**مثال:**

```java
int count = 5;
double price = 19.99;
```

### 3.2 العمليات الحسابية الأساسية

`+`، `-`، `*`، `/`، `%`

أمثلة:

```java
int a = 10, b = 3;
System.out.println(a / b);  // 3 (قسمة صحيحة)
System.out.println(a % b);  // 1
System.out.println((double)a / b); // 3.3333 (تحويل نوع)
```

## 4. تحويل الأنواع (التحويل القسري)

### 4.1 التحويل التلقائي (التوسيع)

من نوع صغير → إلى نوع كبير تلقائيًا:
`int` → `long` → `float` → `double`

مثال:

```java
int i = 10;
double d = i;  // تحويل تلقائي
```

### 4.2 التحويل اليدوي (التحويل القسري)

تحويل صريح من نوع كبير → إلى نوع صغير:

```java
double d = 9.7;
int i = (int) d; // i = 9 (فقدان للجزء الكسري)
```

كن حذرًا من **فقدان الدقة**.

## 5. تقييم التعابير وأولوية العوامل

### 5.1 ترتيب العمليات

تتبع جافا ترتيبًا محددًا:

1. الأقواس `( )`
2. العوامل الأحادية `+`، `-`، `++`، `--`
3. الضرب، القسمة، باقي القسمة `* / %`
4. الجمع والطرح `+ -`
5. التعيين `=`

مثال:

```java
int x = 2 + 3 * 4;   // 14، وليس 20
int y = (2 + 3) * 4; // 20
```

### 5.2 التعابير المختلطة

إذا كان أحد المعاملات من نوع `double`، تصبح النتيجة `double`:

```java
double result = 5 / 2;     // 2.0 (قسمة صحيحة أولاً)
double result2 = 5.0 / 2;  // 2.5 (قسمة أعداد عشرية)
```

## 6. التعيين المعزز والزيادة والنقصان

### 6.1 التعيين المعزز

عوامل مختصرة:

```java
x += 3;  // نفس x = x + 3
y *= 2;  // نفس y = y * 2
```

### 6.2 الزيادة والنقصان

`++` تزيد القيمة بمقدار 1، `--` تنقص القيمة بمقدار 1.
هناك شكلان:

```java
int a = 5;
System.out.println(a++); // يطبع 5، ثم يصبح a = 6
System.out.println(++a); // يصبح a = 7، ثم يطبع 7
```

**تذكر:**

* اللاحقة (`a++`) → استخدم ثم زِد
* السابقة (`++a`) → زِد ثم استخدم

## 7. الأخطاء الشائعة التي يجب تجنبها

* استخدام `=` بدلاً من `==` في المقارنات.
* القسمة الصحيحة عندما تتوقع أعدادًا عشرية.
* نسيان تهيئة المتغير قبل استخدامه.
* فقدان البيانات في تحويل الأنواع (مثل `(int) 3.9` → 3).

## 8. أسئلة امتحانية نموذجية

**المثال 1:**

> ما ناتج الكود التالي؟
>
> ```java
> int x = 5, y = 2;
> double z = x / y;
> System.out.println(z);
> ```

**الإجابة:** `2.0` (قسمة صحيحة قبل التعيين)

**المثال 2:**

> أي عامل له أعلى أولوية؟
> أ. `+`
> ب. `*`
> ج. `=`
> د. `++`
> **الإجابة:** د

**المثال 3:**

> ما ناتج هذا الكود؟
>
> ```java
> int a = 3;
> int b = ++a + a++;
> System.out.println(b);
> ```

**الشرح:**
`++a` تجعل `a = 4`، تُضاف `4`، ثم `a++` تستخدم `4` وتزيد القيمة إلى `5`.
إذن `b = 8`.
