وكلاء Mistral AI هي أنظمة مستقلة تعمل بنماذج اللغة الكبيرة (LLMs) ومصممة لتنفيذ مهام معقدة بناءً على تعليمات عالية المستوى. بالاعتماد على معالجة اللغة الطبيعية المتقدمة، يمكن لهؤلاء الوكلاء فهم وتحقيق أهداف محددة، مما يجعلها مناسبة لمجموعة واسعة من التطبيقات مثل دعم العملاء، وتحليل البيانات، وأتمتة سير العمل، ومساعدة البرمجة. يمكنهم التخطيط، واستخدام الأدوات، واتخاذ الإجراءات، وحتى التعاون لتحقيق أهداف محددة، مما يقدم مستوى جديدًا من الأتمتة والذكاء.

## إنشاء الوكلاء

تقدم Mistral AI طريقتين أساسيتين لإنشاء الوكلاء: **منشئ الوكلاء في La Plateforme** و **واجهة برمجة تطبيقات الوكلاء (Agent API)**.

### ١. منشئ الوكلاء في La Plateforme
يقدم منشئ الوكلاء واجهة سهلة الاستخدام لإنشاء وكلاء دون الحاجة إلى معرفة تقنية متعمقة. لإنشاء وكيل:

- انتقل إلى منشئ الوكلاء على الرابط: [https://console.mistral.ai/build/agents/new](https://console.mistral.ai/build/agents/new).
- خصص الوكيل عن طريق اختيار نموذج، وضبط درجة الحرارة (temperature)، وتقديم تعليمات اختيارية.
- بمجرد التهيئة، يمكن نشر الوكيل والوصول إليه عبر واجهة برمجة التطبيقات أو Le Chat.

### ٢. واجهة برمجة تطبيقات الوكلاء (Agent API)
للمطورين، تسمح واجهة برمجة تطبيقات الوكلاء بإنشاء ودمج الوكلاء برمجيًا في سير العمل الحالية. فيما يلي أمثلة حول كيفية إنشاء واستخدام وكيل عبر واجهة برمجة التطبيقات:

#### مثال بلغة Python
```python
import os
from mistralai import Mistral

api_key = os.environ["MISTRAL_API_KEY"]
client = Mistral(api_key=api_key)

chat_response = client.agents.complete(
    agent_id="your-agent-id",
    messages=[{"role": "user", "content": "What is the best French cheese?"}],
)
print(chat_response.choices[0].message.content)
```

#### مثال بلغة JavaScript
```javascript
import { Mistral } from '@mistralai/mistralai';

const apiKey = process.env.MISTRAL_API_KEY;
const client = new Mistral({ apiKey: apiKey });

const chatResponse = await client.agents.complete({
    agentId: "your-agent-id",
    messages: [{ role: 'user', content: 'What is the best French cheese?' }],
});
console.log('Chat:', chatResponse.choices[0].message.content);
```

## تخصيص الوكلاء

يمكن تخصيص وكلاء Mistral AI لتناسب احتياجات محددة من خلال عدة خيارات:

- **اختيار النموذج**: اختر النموذج الذي يشغل الوكيل. الخيارات تشمل:
  - "Mistral Large 2" (الافتراضي، `mistral-large-2407`)
  - "Mistral Nemo" (`open-mistral-nemo`)
  - "Codestral" (`codestral-2405`)
  - النماذج المُدققة (Fine-tuned models)

- **درجة الحرارة (Temperature)**: اضبط درجة حرارة أخذ العينات (بين ٠.٠ و ١.٠) للتحكم في العشوائية لردود الوكيل. القيم الأعلى تجعل المخرجات أكثر إبداعًا، بينما القيم الأقل تجعلها أكثر تركيزًا وحتمية.

- **التعليمات**: قدم تعليمات اختيارية لفرض سلوكيات محددة عبر جميع التفاعلات. على سبيل المثال، يمكنك إنشاء وكيل يتحدث الفرنسية فقط أو يولد كود Python بدون تفسيرات.

### مثال: إنشاء وكيل ناطق بالفرنسية
لإنشاء وكيل يرد باللغة الفرنسية فقط:
- عيّن النموذج إلى "Mistral Large 2".
- استخدم تعليمات مثل: "رد دائمًا باللغة الفرنسية، بغض النظر عن لغة الإدخال."
- قدم أمثلة قليلة (few-shot) لتعزيز السلوك المطلوب.
