مستقلية الوكالات AI هي أنظمة تعمل بشكل مستقل تعمل على أساس نماذج اللغة الطبيعية الكبيرة (LLMs) مصممة لتنفيذ مهام معقدة بناءً على تعليمات عالية المستوى. باستخدام معالجة اللغة الطبيعية المتقدمة، يمكن لهذه الوكالات فهم وتنفذ أهدافًا محددة، مما يجعلها مناسبة لسلسلة واسعة من التطبيقات مثل الدعم الفني، تحليل البيانات، وتوظيف العمل، ومساعدة في البرمجة. يمكن أن تخطط، وتستخدم الأدوات، وتنفذ إجراءات، وحتى التعاون لتحقيق أهداف محددة، مما يوفر مستوى جديد من التلقائية والذكاء.

## إنشاء الوكالات

يوفر Mistral AI طريقةين رئيسيتين لإنشاء الوكالات: **La Plateforme Agent Builder** و **Agent API**.

### 1. La Plateforme Agent Builder
يوفر Builder الوكالة واجهة مستخدمية سهلة الاستخدام لإنشاء الوكالات دون معرفة تقنية شاملة. لإنشاء الوكالة:

- انتقل إلى Builder الوكالة في [https://console.mistral.ai/build/agents/new](https://console.mistral.ai/build/agents/new).
- قم بتخصيص الوكالة من خلال اختيار نموذج، تعيين درجة الحرارة، وتقديم تعليمات اختيارية.
- بعد التكوين، يمكن نشر الوكالة واستخدامها عبر API أو Le Chat.

### 2. Agent API
للمبرمجين، يسمح API الوكالة بإنشاء الوكالات وتكاملها في عمليات العمل الحالية بشكل برمجي. أدناه أمثلة على كيفية إنشاء واستخدام الوكالة عبر API:

#### مثال Python
```python
import os
from mistralai import Mistral

api_key = os.environ["MISTRAL_API_KEY"]
client = Mistral(api_key=api_key)

chat_response = client.agents.complete(
    agent_id="your-agent-id",
    messages=[{"role": "user", "content": "ما هو أفضل جبن فرنسي؟"}],
)
print(chat_response.choices[0].message.content)
```

#### مثال JavaScript
```javascript
import { Mistral } from '@mistralai/mistralai';

const apiKey = process.env.MISTRAL_API_KEY;
const client = new Mistral({ apiKey: apiKey });

const chatResponse = await client.agents.complete({
    agentId: "your-agent-id",
    messages: [{ role: 'user', content: 'ما هو أفضل جبن فرنسي؟' }],
});
console.log('Chat:', chatResponse.choices[0].message.content);
```

## تخصيص الوكالات

يمكن تخصيص الوكالات AI من Mistral لتناسب الاحتياجات المحددة من خلال عدة خيارات:

- **اختيار النموذج**: اختر النموذج الذي يعمل الوكالة. تشمل الخيارات:
  - "Mistral Large 2" (الافتراضي، `mistral-large-2407`)
  - "Mistral Nemo" (`open-mistral-nemo`)
  - "Codestral" (`codestral-2405`)
  - نماذج مخصصة

- **درجة الحرارة**: قم بتعديل درجة حرارة العينة (بين 0.0 و 1.0) للتحكم في عشوائية إجابات الوكالة. تزيد القيم الأعلى من الإبداع في الإخراج، بينما تقلل القيم الأقل من التركيز والتحديد.

- **التعليمات**: قدم تعليمات اختيارية لضمان سلوك محدد في جميع التفاعلات. على سبيل المثال، يمكنك إنشاء الوكالة التي تتحدث فقط الفرنسية أو توليد كود Python بدون شرح.

### مثال: إنشاء وكيل يتحدث الفرنسية
لإنشاء وكيل يتكلم فقط الفرنسية:
- قم بتحديد النموذج إلى "Mistral Large 2".
- استخدم تعليمات مثل: "الرد دائمًا باللغة الفرنسية، بغض النظر عن لغة المدخلات."
- قدم أمثلة قليلة لتأكيد السلوك.

## حالات الاستخدام

يمكن تطبيق الوكالات AI من Mistral على مختلف الصناعات والمهام. تشمل الحالات الاستخدام البارزة:

- **تحليل البيانات**: إنشاء الوكالات التي تحلل مجموعات البيانات، توليد تقارير أو تنفيذ الحسابات بناءً على المدخلات المستخدم.
- **توظيف العمل**: تكامل الوكالات مع الأدوات مثل البريد الإلكتروني، أنظمة إدارة العملاء، أو معالجة الدفع لتوظيف المهام المتكررة.
- **مساعدة في البرمجة**: تصميم الوكالات لتوليد الكود، تقديم اقتراحات تصحيح الأخطاء، أو إنشاء اختبارات الوحدة.

### أمثلة محددة

- **مولد كود Python**: وكيل يخرج كود Python بدون شرح، مثالي للمبرمجين الذين يحتاجون إلى كود سريع ونظيف.
