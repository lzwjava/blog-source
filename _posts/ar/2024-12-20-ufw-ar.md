إذا كنت تواجه مشكلة في الاتصال ببعض الخدمات، إليك بعض النصائح لتكوين جدار الحماية البسيط (UFW - Uncomplicated Firewall).

### السماح بحركة SSH (المنفذ 22)

إذا كان SSH محظورًا، يمكنك السماح به باستخدام:

```bash
sudo ufw allow ssh
```

### السماح بمرور حركة V2Ray (المنفذ 1080 أو 443)

بالنسبة لـ V2Ray أو الخدمات الأخرى، قم بفتح المنافذ الضرورية:

```bash
sudo ufw allow 1080/tcp
```

تم ترجمة الأمر إلى:

```bash
sudo ufw allow 1080/tcp
```

ملاحظة: الأمر نفسه باللغة الإنجليزية حيث أنه أمر في سطر الأوامر (CLI) ولا يتم ترجمته.

أو

```bash
sudo ufw allow 443/tcp
```

### التحقق من حالة UFW

لعرض قواعد جدار الحماية النشطة، استخدم:

```bash
sudo ufw status verbose
```

### إعادة تفعيل UFW

إذا كان UFW معطلاً، قم بتمكينه باستخدام:

```bash
sudo ufw enable
```

### إعادة تعيين UFW

إذا كنت بحاجة إلى إعادة تعيين إعدادات جدار الحماية UFW (Uncomplicated Firewall) إلى الإعدادات الافتراضية، يمكنك اتباع الخطوات التالية:

1. **إيقاف UFW**:
   ```bash
   sudo ufw disable
   ```

2. **إعادة تعيين القواعد**:
   ```bash
   sudo ufw reset
   ```

3. **تمكين UFW مرة أخرى**:
   ```bash
   sudo ufw enable
   ```

بعد تنفيذ هذه الخطوات، سيتم إعادة تعيين جميع القواعد والإعدادات إلى الحالة الافتراضية. تأكد من مراجعة القواعد الجديدة قبل تمكين UFW مرة أخرى لتجنب أي مشاكل في الوصول إلى الخادم.

لتبدأ من جديد، قم بإعادة تعيين UFW:

```bash
sudo ufw reset
```

### منافذ مخصصة

```bash
sudo ufw allow 1024:65535/tcp
sudo ufw allow 1024:65535/udp
```

أخبرني إذا كنت بحاجة إلى مساعدة في تكوينات UFW محددة!
