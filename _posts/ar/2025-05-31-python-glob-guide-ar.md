### استيراد Glob
لاستخدام glob في Python، قم باستيراد الوحدة النمطية من المكتبة القياسية:
```python
import glob
```

### الوظائف الأساسية
تأخذ الدالة `glob.glob()` سلسلة نمط وتُرجع قائمة بمسارات الملفات أو المجلدات المطابقة. تستخدم أحرف البدل ذات النمط الشبيه بـ Unix shell:

- `*`: يطابق أي تسلسل من الأحرف (باستثناء فواصل المسار).
- `?`: يطابق أي حرف فردي.
- `[seq]`: يطابق أي حرف في seq (على سبيل المثال، `[0-9]` للأرقام).
- `**`: يطابق المجلدات بشكل متكرر (يتطلب `recursive=True`).

تكون الأنماط نسبية للمجلد الحالي ما لم يتم تحديد مسار مطلق.

### أمثلة الاستخدام

#### 1. العثور على جميع ملفات .txt في المجلد الحالي
```python
import glob

files = glob.glob("*.txt")
print(files)  # الإخراج: ['file1.txt', 'file2.txt', ...]
```

#### 2. العثور على ملفات ذات امتدادات محددة في مجلد فرعي
```python
files = glob.glob("documents/*.pdf")
print(files)  # الإخراج: ['documents/doc1.pdf', ...]
```

#### 3. مطابقة أنماط محددة (مثل الملفات التي تبدأ بـ 'log' وتنتهي بأرقام)
```python
files = glob.glob("log[0-9].txt")
print(files)  # الإخراج: ['log1.txt', 'log2.txt', ...] إذا كانت هذه الملفات موجودة
```

#### 4. البحث المتكرر في المجلدات الفرعية
استخدم `**` مع `recursive=True` للبحث في المجلدات والمجلدات الفرعية:
```python
files = glob.glob("**/*.py", recursive=True)
print(files)  # الإخراج: ['script1.py', 'subdir/script2.py', ...]
```

### ملاحظات مهمة
- النتائج غير مرتبة افتراضيًا؛ استخدم `sorted()` إذا كنت بحاجة إلى ترتيب: `sorted(glob.glob("*.txt"))`.
- للمطابقة غير الحساسة لحالة الأحرف على أنظمة الملفات الحساسة لحالة الأحرف، فكر في الجمع مع `fnmatch` أو المعالجة في الكود، لأن glob حساس لحالة الأحرف.
- لا يتابع Glob الروابط الرمزية إلا إذا تم التعامل معها صراحةً؛ استخدم `os.walk()` للاجتيازات الأكثر تعقيدًا.
- لمطابقة واحدة في كل مرة، استخدم `glob.iglob()` التي تُرجع مكررًا لتوفير الذاكرة على النتائج الكبيرة: `for file in glob.iglob("*.txt"): print(file)`.

يغطي هذا الأساسيات؛ راجع [توثيق Python](https://docs.python.org/3/library/glob.html) لمزيد من التفاصيل.
