لتحديث Visual Studio Code (VSCode) المثبت عبر Snap على Ubuntu، اتبع الخطوات التالية:

1. **افتح طرفية**: يمكنك فعل ذلك بالضغط على `Ctrl + Alt + T`.

2. **تحقق من التحديثات**: يقوم Snap تلقائيًا بالتحقق من التحديثات في الخلفية، ولكن يمكنك تشغيل فحص تحديث يدويًا بتنفيذ الأمر:
   ```bash
   sudo snap refresh
   ```

3. **حدّث VSCode**: إذا كانت هناك نسخة أحدث من VSCode متاحة، فإن أمر `snap refresh` سيقوم بتنزيلها وتثبيتها. سترى ناتجًا يشير إلى التقدم، مثل:
   ```
   code 1.85.2 from Visual Studio Code (code✓) refreshed
   ```
   إذا لم تكن هناك تحديثات متاحة، ستظهر الرسالة:
   ```
   All snaps up to date.
   ```

4. **تحقق من التحديث**: للتأكد من النسخة المثبتة من VSCode، نفّذ الأمر:
   ```bash
   code --version
   ```
   هذا سيعرض النسخة الحالية من VSCode.

### ملاحظات:
- **التحديثات التلقائية**: عادةً ما يتم تحديث حزم Snap، بما فيها VSCode، تلقائيًا في الخلفية. لا تحتاج عادةً للتدخل يدويًا إلا إذا أردت فرض تحديث.
- **صلاحيات Root**: أمر `sudo` مطلوب لعمليات Snap على مستوى النظام.
- **تحقق من حالة Snap**: إذا أردت رؤية أي حزم Snap مثبتة وإصداراتها، استخدم:
   ```bash
   snap list
   ```
- **استكشاف الأخطاء وإصلاحها**: إذا فشل التحديث، تأكد من أن اتصالك بالإنترنت مستقر وأن خادم Snap يعمل (`sudo systemctl status snapd`). يمكنك أيضًا محاولة تحديث بيانات وصفات Snap باستخدام:
   ```bash
   sudo snap refresh --list
   ```

هذه العملية تضمن بقاء VSCode محدثًا بأحدث نسخة متاحة في متجر Snap.
