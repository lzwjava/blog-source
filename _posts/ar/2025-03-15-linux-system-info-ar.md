في نظام Ubuntu، يمكنك استرداد معلومات النظام مثل ما إذا كان النظام 32 بت أو 64 بت، وإصدار نظام التشغيل، وكمية ذاكرة الوصول العشوائي، وحجم القرص باستخدام الأوامر التالية. فيما يلي دليل خطوة بخطوة:

### 1. تحديد ما إذا كان النظام 32 بت أو 64 بت
- **الأمر**: `uname -m`
- **الشرح**: يعرض هذا الأمر اسم أجهزة الجهاز.
  - إذا كان الناتج هو `x86_64`، فإن نظامك هو **64 بت**.
  - إذا كان الناتج هو `i686` أو `i386`، فإن نظامك هو **32 بت**.
- **ناتج مثال**:
  ```
  x86_64
  ```
  يشير هذا إلى نظام 64 بت.

### 2. الحصول على إصدار نظام التشغيل (OS)
- **الأمر**: `lsb_release -a`
- **الشرح**: يوفر هذا الأمر معلومات مفصلة حول Linux Standard Base (LSB) والتوزيعة.
  - ابحث عن سطر `Description` لرؤية إصدار Ubuntu.
- **ناتج مثال**:
  ```
  No LSB modules are available.
  Distributor ID: Ubuntu
  Description:    Ubuntu 20.04.3 LTS
  Release:        20.04
  Codename:       focal
  ```
  هنا، نظام التشغيل هو **Ubuntu 20.04.3 LTS**.

### 3. الحصول على حجم ذاكرة الوصول العشوائي (RAM)
- **الأمر**: `free -h`
- **الشرح**: يظهر هذا الأمر استخدام الذاكرة بتنسيق مقروء (مثل GB أو MB).
  - انظر إلى عمود `total` تحت صف `Mem:` لإيجاد إجمالي ذاكرة الوصول العشوائي المثبتة.
- **ناتج مثال**:
  ```
              total        used        free      shared  buff/cache   available
  Mem:          15Gi       2.3Gi       9.5Gi       300Mi       3.8Gi        12Gi
  Swap:        2.0Gi         0B       2.0Gi
  ```
  في هذه الحالة، إجمالي ذاكرة الوصول العشوائي هو **15 جيجابايت**.

### 4. الحصول على حجم القرص
- **الأمر**: `lsblk -d -o NAME,SIZE`
- **الشرح**: يسرد هذا الأمر جميع أجهزة القرص (باستثناء الأقسام) مع أحجامها.
  - تضمن العلامة `-d` عرض أجهزة القرص فقط (مثل `/dev/sda`)، وليس أقسامها.
  - ابحث عن اسم القرص (مثل `sda`) وحجمه المقابل.
- **ناتج مثال**:
  ```
  NAME   SIZE
  sda    1T
  sdb    500G
  ```
  هنا، يحتوي النظام على قرصين: `/dev/sda` بحجم **1 تيرابايت** و `/dev/sdb` بحجم **500 جيجابايت**. إذا كان لديك قرص واحد فقط، فسيكون عادةً `sda`.

### ملاحظات
- تستخدم هذه الأوامر أدوات قياسية متوفرة في Ubuntu، ولا تتطلب تثبيتات إضافية.
- بالنسبة لحجم القرص، إذا كان نظامك يحتوي على أقراص متعددة، فسيقوم الناتج بسردها جميعًا. عادةً، يكون `/dev/sda` هو القرص الأساسي.
- قم بتشغيل هذه الأوامر في طرفية لرؤية النتائج الخاصة بنظامك.

باستخدام هذه الأوامر، يمكنك جمع معلومات النظام الأساسية التي تحتاجها بسرعة على جهاز Ubuntu الخاص بك!
