إليك دليلًا خطوة بخطوة لاستخدام MySQL، نظام إدارة قواعد البيانات العلائقية مفتوح المصدر الشهير:

---

### 1. **التثبيت**
- **Windows**: حمّل مُثبّت MySQL من [mysql.com/downloads](https://dev.mysql.com/downloads/installer/)
- **macOS**:
  ```bash
  brew install mysql
  brew services start mysql
  ```
- **Linux** (Debian/Ubuntu):
  ```bash
  sudo apt update
  sudo apt install mysql-server mysql-client
  sudo systemctl start mysql
  sudo mysql_secure_installation  # تشغيل إعدادات الأمان
  ```

---

### 2. **الاتصال بـ MySQL**
- **تسجيل دخول root الافتراضي**:
  ```bash
  mysql -u root -p  # أدخل كلمة المرور عند الطلب
  ```
- **الاتصال بخادم بعيد**:
  ```bash
  mysql -u username -p -h hostname -P 3306
  ```

---

### 3. **عمليات قاعدة البيانات الأساسية**
- **إنشاء مستخدم**:
  ```sql
  CREATE USER 'myuser'@'localhost' IDENTIFIED BY 'mypassword';
  ```
- **إنشاء قاعدة بيانات**:
  ```sql
  CREATE DATABASE mydb;
  GRANT ALL PRIVILEGES ON mydb.* TO 'myuser'@'localhost';
  FLUSH PRIVILEGES;
  ```
- **أوامر SQL الأساسية**:
  ```sql
  -- إنشاء جدول
  CREATE TABLE users (
      id INT AUTO_INCREMENT PRIMARY KEY,
      name VARCHAR(50),
      email VARCHAR(100) UNIQUE
  );

  -- إدراج بيانات
  INSERT INTO users (name, email) VALUES ('Bob', 'bob@example.com');

  -- استعلام عن البيانات
  SELECT * FROM users WHERE name LIKE 'B%';

  -- تحديث البيانات
  UPDATE users SET email = 'new@email.com' WHERE id = 1;

  -- حذف البيانات
  DELETE FROM users WHERE id = 2;
  ```

---

### 4. **الميزات المتقدمة**
- **الانضمامات (Joins)**:
  ```sql
  SELECT orders.id, users.name 
  FROM orders
  JOIN users ON orders.user_id = users.id;
  ```
- **الإجراءات المخزنة (Stored Procedures)**:
  ```sql
  DELIMITER //
  CREATE PROCEDURE GetUsers()
  BEGIN
    SELECT * FROM users;
  END //
  DELIMITER ;
  
  CALL GetUsers();
  ```
- **دعم JSON** (MySQL 5.7+):
  ```sql
  CREATE TABLE products (
      id INT PRIMARY KEY,
      details JSON
  );
  ```

---

### 5. **الصيانة**
- **النسخ الاحتياطي**:
  ```bash
  mysqldump -u root -p mydb > backup.sql
  ```
- **الاستعادة**:
  ```bash
  mysql -u root -p mydb < backup.sql
  ```
- **تحسين الجداول**:
  ```sql
  ANALYZE TABLE users;
  OPTIMIZE TABLE users;
  ```

---

### 6. **الأمان**
- **إنشاء مستخدم للقراءة فقط**:
  ```sql
  CREATE USER 'reader'@'localhost' IDENTIFIED BY 'readpass';
  GRANT SELECT ON mydb.* TO 'reader'@'localhost';
  FLUSH PRIVILEGES;
  ```

---

### 7. **أوامر MySQL مفيدة**
- عرض قواعد البيانات: `SHOW DATABASES;`
- استخدام قاعدة بيانات: `USE mydb;`
- عرض الجداول: `SHOW TABLES;`
- وصف جدول: `DESCRIBE users;`
- عرض المستخدمين: `SELECT user FROM mysql.user;`
- الخروج من MySQL: `EXIT;` أو `QUIT;`

---

### 8. **أدوات واجهة المستخدم الرسومية**
- **MySQL Workbench**: أداة واجهة المستخدم الرسومية الرسمية
- **phpMyAdmin**: واجهة تعتمد على الويب
- **DBeaver**: أداة قواعد بيانات متعددة المنصات
- **HeidiSQL**: عميل خفيف الوزن لنظام Windows

**الاختلافات الرئيسية عن PostgreSQL**:
1. يستخدم `AUTO_INCREMENT` بدلاً من `SERIAL`
2. أسماء الجداول غير حساسة لحالة الأحرف افتراضيًا
3. يستخدم `mysqldump` بدلاً من `pg_dump`
4. المنفذ الافتراضي: **3306** (مقابل 5432 لـ PostgreSQL)
5. نظام مصادقة وإدارة مستخدمين مختلف

**نصيحة**: احرص دائمًا على عمل نسخة احتياطية من قواعد البيانات قبل إجراء تغييرات كبيرة. استخدم المعاملات (`START TRANSACTION`/`COMMIT`) للعمليات الحرجة.
