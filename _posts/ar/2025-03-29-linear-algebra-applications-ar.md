## 9.2 مشكلات تدفق الشبكة

### 9.2.1 أساسيات تدفق الشبكة

الشبكات هي هياكل رياضية تتكون من عقد (رؤوس) متصلة بواسطة حواف. في العديد من المواقف العملية، نحتاج إلى تحديد التدفقات المثلى عبر هذه الشبكات:

- **شبكات النقل**: نقل البضائع من المصانع إلى المستودعات
- **شبكات الاتصالات**: توجيه حزم البيانات عبر الإنترنت
- **شبكات المرافق**: توزيع الكهرباء أو الماء أو الغاز

يمكن تمثيل مشكلات تدفق الشبكة بأناقة باستخدام المصفوفات:

- تمثل **مصفوفة الوقوع** A هيكل الشبكة
- يمثل المتجه x كميات التدفق على طول كل حافة
- تضبط القيود حفظ التدفق عند العقد

## 9.3 ملاءمة البيانات والمربعات الصغرى

### 9.3.1 الانحدار الخطي

عند ملاءمة خط أو منحنى لنقاط البيانات، نبحث عن دالة تقلل الخطأ بين القيم المتوقعة والفعلية:

- للانحدار الخطي، نريد إيجاد المعلمات في y = mx + b
- مع نقاط بيانات متعددة، يصبح هذا نظامًا محددًا بشكل زائد
- يقلل حل المربعات الصغرى من مجموع الأخطاء المربعة

### 9.3.2 المعادلات العادية

يمكن إيجاد الحل الأمثل باستخدام:
- A^T A x = A^T b
- حيث A هي مصفوفة التصميم، b هي متجه المخرجات
- يعطي الحل x المعلمات المثلى

- **تسريع وحدة معالجة الرسومات**: استخدام وحدات معالجة الرسومات لعمليات المصفوفات

### 9.4.3 مثال على الكود: حل نظام خطي

```python
import numpy as np

# Define a system of linear equations as a matrix
A = np.array([[3, 1, -1], 
              [1, 4, 1], 
              [2, 1, 2]])
              
b = np.array([9, 7, 8])

# Solve the system Ax = b
x = np.linalg.solve(A, b)

print("Solution:")
print(f"x = {x[0]}")
print(f"y = {x[1]}")
print(f"z = {x[2]}")
```
