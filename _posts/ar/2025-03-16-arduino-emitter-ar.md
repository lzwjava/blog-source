بناء مضخم باعث مشترك (CE) باستخدام Arduino هو مشروع مثير للاهتمام، ولكن من الجدير التوضيح بشأن الأدوار: لن يعمل Arduino نفسه كمضخم (فهو متحكم دقيق رقمي)، ولكن يمكن أن يعمل كمصدر للإشارة أو أداة قياس لدائرة مضخم CE المبنية باستخدام ترانزستور ذو وصلة ثنائية القطب (BJT). مضخم CE هو دائرة تناظرية كلاسيكية تستخدم ترانزستور NPN أو PNP لتضخيم إشارة إدخال صغيرة، مما يوفر كسب جهد معتدل وكسب تيار.

هنا، سأرشدك خلال تصميم مضخم CE أساسي قائم على ترانزستور NPN، باستخدام Arduino لتوليد إشارة الإدخال (عبر PWM) وقياس الناتج. هذا الإعداد سيوضح التضخيم مع الاستفادة من إمكانيات Arduino.

---

### **نظرة عامة على مضخم الباعث المشترك**
- **الغرض:** تضخيم إشارة تيار متردد صغيرة (مثل الصوت أو موجة جيبية).
- **الميزات الرئيسية:**
  - يعمل الترانزستور في المنطقة النشطة.
  - تُطبق إشارة الإدخال على القاعدة، ويُؤخذ الناتج من المجمع.
  - يتم تحديد كسب الجهد من خلال نسب المقاومات وخصائص الترانزستور.
- **المكونات:**
  - ترانزستور NPN (مثل 2N3904 أو BC547)
  - مقاومات (لتحيز الجهد والحمل)
  - مكثفات (لاقتران إشارات التيار المتردد)
  - Arduino (مصدر الإشارة والقياس)

### **الخطوة 1: تصميم الدائرة**

#### **المكونات المطلوبة**
- ترانزستور NPN (مثل 2N3904)
- مقاومات: R1 = 47 كيلو أوم، R2 = 10 كيلو أوم (تحيز الجهد)، RC = 1 كيلو أوم (المجمع)، RE = 220 أوم (الباعث)
- مكثفات: C1 = 10 ميكروفاراد (اقتران الإدخال)، C2 = 10 ميكروفاراد (اقتران الناتج)، CE = 100 ميكروفاراد (تحويل الباعث، اختياري لكسب أعلى)
- Arduino (مثل Uno)
- لوحة تجارب، أسلاك توصيل
- مصدر طاقة (دبوس 5V من Arduino أو 9V خارجي، مع التعديل حسب الحاجة)

#### **الرسم التخطيطي للدائرة**
```
Vcc (5V) ---- R1 ----+---- RC ---- المجمع (C)
             47kΩ     |     1kΩ          |
                      |                  |
القاعدة (B) --- C1 -----+                  |
            10µF     |                  |
Arduino PWM (دبوس 9)  R2                 |
                     10kΩ              الناتج --- C2 ---- إلى Arduino A0
                      |                  |         10µF
                      |                  |
                      +---- RE ---- الباعث (E) --- CE (اختياري) --- GND
                           220Ω                   100µF
                      |
                     GND
```
- **تحيز الجهد (R1, R2):** يحدد نقطة تشغيل الترانزستور.
- **RC:** مقاومة المجمع لإشارة الناتج.
- **RE:** مقاومة الباعث للاستقرار.
- **C1, C2:** تمنع مرور التيار المستمر، وتسمح بمرور إشارات التيار المتردد.
- **CE (اختياري):** يحول مسار RE للحصول على كسب تيار متردد أعلى.

#### **نقطة التشغيل**
- الهدف: تحيز الترانزستور في المنطقة النشطة (مثل VCE ≈ 2.5V لمصدر 5V).
- مقسم الجهد (R1, R2): \\( V_B = V_{CC} \cdot \frac{R2}{R1 + R2} = 5 \cdot \frac{10k}{47k + 10k} \approx 0.88V \\).
- \\( V_E = V_B - V_{BE} \approx 0.88 - 0.7 = 0.18V \\).
- \\( I_E = \frac{V_E}{RE} = \frac{0.18}{220} \approx 0.82 \, \text{mA} \\).
- \\( V_C = V_{CC} - I_C \cdot RC \approx 5 - 0.82 \cdot 1k \approx 4.18V \\).
- \\( V_{CE} = V_C - V_E \approx 4.18 - 0.18 = 4V \\) (جيد لمصدر 5V).

### **الخطوة 2: استخدام Arduino كمصدر للإشارة**

#### **دور Arduino**
- توليد إشارة تيار متردد صغيرة باستخدام PWM على دبوس مثل 9 (الذي يدعم PWM).
- تصفية PWM لتقريب موجة جيبية باستخدام مرشح تمرير منخرب RC بسيط (اختياري).

#### **الكود لتوليد إشارة**
```cpp
const int pwmPin = 9; // دبوس إخراج PWM

void setup() {
  pinMode(pwmPin, OUTPUT);
  // ضبط تردد PWM (اختياري، القيمة الافتراضية ~490 هرتز)
}

void loop() {
  // محاكاة موجة جيبية باستخدام PWM (مدى 0-255)
  for (int i = 0; i < 360; i += 10) {
    float sineValue = sin(radians(i)); // موجة جيبية من -1 إلى 1
    int pwmValue = 127 + 127 * sineValue; // تحجيم إلى 0-255
    analogWrite(pwmPin, pwmValue);
    delay(10); // اضبط للتحكم بالتردد (مثلاً ~100 هرتز هنا)
  }
}
```
- **الإخراج:** إشارة PWM بقيمة ~0–5V، تتمحور حول 2.5V بقيمة ذروة إلى ذروة ~2.5V.
- **C1:** يزيل إزاحة التيار المستمر، ويمرر فقط مركبة التيار المتردد (~1.25V ذروة) إلى القاعدة.

#### **المرشح الاختياري**
أضف مقاومة 1 كيلو أوم ومكثف 0.1 ميكروفاراد على التوالي من الدبوس 9 إلى GND، وأخذ الإشارة قبل C1، لتنعيم PWM إلى موجة جيبية تقريبية.

### **الخطوة 3: قياس الناتج**

#### **قياس Arduino**
- وصل ناتج المضخم (بعد C2) إلى A0.
- استخدم Arduino لقراءة الإشارة المضخمة وعرضها عبر Serial Monitor.

#### **الكود للقياس والعرض**
```cpp
const int inputPin = A0; // قياس الناتج هنا

void setup() {
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(inputPin); // 0-1023 تُقابل 0-5V
  float voltage = sensorValue * (5.0 / 1023.0);
  Serial.print("جهد الخرج (V): ");
  Serial.println(voltage);
  delay(100); // اضبط معدل أخذ العينات
}
```

#### **الكسب المتوقع**
- كسب الجهد \\( A_v = -\frac{RC}{RE} = -\frac{1k}{220} \approx -4.5 \\) (سالبة بسبب انعكاس الطور).
- الإدخال: ~1.25V ذروة (بعد الاقتران).
- الناتج: ~4.5 × 1.25 = 5.625V ذروة (ولكن سيتم القص عند 5V بسبب حد المصدر).

### **الخطوة 4: البناء والاختبار**

#### **التجميع**
1. ربط الدائرة على لوحة التجارب وفقًا للرسم التخطيطي.
2. رفع كود توليد الإشارة إلى Arduino ووصل الدبوس 9 إلى C1.
3. تشغيل الدائرة عبر 5V من Arduino (أو استخدام 9V مع تعديل المقاومات).
4. رفع كود القياس وفتح Serial Monitor (9600 باود).

#### **التحقق**
- **إشارة الإدخال:** ~1–2V ذروة إلى ذروة بعد C1 (استخدم مقياس متعدد أو راسم إشارة إذا كان متاحًا).
- **إشارة الناتج:** يجب أن تكون أكبر (مثل 4–5V ذروة إلى ذروة) ومقلوبة.
- تحقق من VCE (~2–4V) باستخدام مقياس متعدد للتأكد من عمل الترانزستور في المنطقة النشطة.
