### تثبيت Rust على أوبونتو 22.04 LTS

أفضل طريقة لتثبيت Rust هي باستخدام `rustup`، أداة التثبيت الرسمية لأدوات لغة Rust. هذه الطريقة مباشرة، تدعم التحديثات السهلة، وتعمل على نظام أوبونتو 22.04 الخاص بك (64-bit). عادةً لا توجد حاجة إلى تبعيات إضافية تتجاوز إعدادات أوبونتو الأساسية.

#### التعليمات خطوة بخطوة

1.  **حدّث قائمة الحزم لديك** (اختياري ولكنه موصى به):
    ```
    sudo apt update
    ```

2.  **ثبّت Rust باستخدام rustup**:
    قم بتشغيل الأمر التالي لتنزيل وتنفيذ سكريبت التثبيت:
    ```
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
    - سيقوم هذا بتنزيل السكريبت بشكل آمن عبر HTTPS.
    - عند المطالبة، اختر خيار التثبيت الافتراضي (اضغط على `1` ثم Enter).
    - ستقوم أداة التثبيت بإضافة الملفات التنفيذية لـ Rust إلى المسار `~/.cargo/bin` وتحديث ملف تعريف shell الخاص بك (مثل `~/.bashrc` أو `~/.profile`).

3.  **أعد تحميل إعدادات shell الخاصة بك**:
    لجعل التغييرات سارية المفعول فورًا دون إعادة تشغيل الطرفية:
    ```
    source ~/.cargo/env
    ```
    بدلاً من ذلك، يمكنك إغلاق الطرفية وإعادة فتحها.

4.  **تحقق من التثبيت**:
    تحقق من تثبيت Rust بشكل صحيح:
    ```
    rustc --version
    cargo --version
    ```
    يجب أن ترى ناتجًا مشابهًا لـ `rustc 1.82.0` (أو أحدث إصدار مستقر) ومشابهًا لـ Cargo.

#### ملاحظات إضافية
-   **إلغاء التثبيت**: إذا لزم الأمر، قم بتشغيل `rustup self uninstall`.
-   **تحديث Rust**: استخدم `rustup update` للحصول على أحدث إصدار.
-   **بيئة التطوير المتكاملة / الدعم**: للتطوير، فكّر في تثبيت `rust-analyzer` عبر محرر النصوص الخاص بك (مثل إضافة VS Code) للحصول على إكمال أفضل للكود.
-   يثبّت هذا قناة الإصدار المستقر بشكل افتراضي. للحصول على إصدار nightly أو beta، استخدم `rustup install nightly` بعد الإعداد.

إذا واجهت أي أخطاء (مثل عدم العثور على curl)، قم بتثبيته باستخدام `sudo apt install curl`.
