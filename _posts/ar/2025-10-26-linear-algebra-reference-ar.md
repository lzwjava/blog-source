مرجع سريع للمفاهيم والصيغ الأساسية. ركز على المصفوفات، المحددات، الأنظمة الخطية، الفضاءات المتجهة، التحويلات، القيم الذاتية، والأشكال التربيعية — وهي المواضيع الأساسية من الامتحانات السابقة. راجع التعريفات، الخصائص، والحسابات.

## 1. المصفوفات
- **التعريف**: \\( A = [a_{ij}] \in \mathbb{R}^{m \times n} \\).
- **العمليات**:
  - الجمع: \\( (A + B)_{ij} = a_{ij} + b_{ij} \\).
  - الضرب القياسي: \\( (cA)_{ij} = c a_{ij} \\).
  - الضرب: \\( (AB)_{ij} = \sum_k a_{ik} b_{kj} \\) (إذا كانت الأحجام متوافقة).
- **منقول المصفوفة**: \\( (A^T)_{ij} = a_{ji} \\); \\( (AB)^T = B^T A^T \\), \\( (A^T)^T = A \\).
- **المعكوس** (مربعة): \\( AA^{-1} = I \\); \\( (AB)^{-1} = B^{-1} A^{-1} \\); \\( (A^T)^{-1} = (A^{-1})^T \\).
- **الأنواع**:
  - قطرية: عناصر غير صفرية فقط على القطر.
  - مثلثية عليا/سفلى: أصفار أسفل/فوق القطر.
  - متماثلة: \\( A = A^T \\).
  - متعامدة: \\( A^T A = I \\) (أعمدة متعامدة طبيعية).

## 2. المحددات (det A)
- **الخصائص**:
  - \\( \det(AB) = \det A \cdot \det B \\); \\( \det(A^T) = \det A \\); \\( \det(cA) = c^n \det A \\).
  - تبديل صف/عمود: يضرب بـ -1.
  - إضافة مضاعف لصف/عمود: لا يغير القيمة.
  - قياس صف/عمود بـ c: يضرب بـ c.
  - \\( \det I = 1 \\); \\( \det A = 0 \\) إذا كانت مفردة (صفوف/أعمدة معتمدة خطياً).
- **الحساب**:
  - 2x2: \\( \det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc \\).
  - التوسع بالعامل المساعد (الصف i): \\( \det A = \sum_j a_{ij} C_{ij} \\)، حيث \\( C_{ij} = (-1)^{i+j} M_{ij} \\) (محدد الجزء الصغري).
  - مثلثية: حاصل ضرب العناصر القطرية.
- **المصفوفة المرافقة/المعكوس**: \\( A^{-1} = \frac{1}{\det A} \adj A \\)، حيث \\( \adj A = C^T \\) (منقول مصفوفة العوامل المساعدة).
- **قاعدة كرامر** (لـ \\( Ax = b \\)، det A ≠ 0): \\( x_i = \frac{\det A_i}{\det A} \\) (A_i تستبدل العمود i بـ b).

## 3. الأنظمة الخطية (Ax = b)
- **حذف غاوسي**: اختزال الصفوف [A | b] إلى REF/RREF.
  - REF: المحاور (الواحدات الرئيسية) بشكل سلمي لأسفل-يمين؛ أصفار أسفل المحاور.
  - التعويض العكسي للحل الفريد.
- **الحلول**:
  - فريد: رتبة A = n (رتبة عمودية كاملة)، فضاء العدم {0}.
  - لا نهائي: رتبة A = رتبة [A|b] < n (متغيرات حرة).
  - لا يوجد: رتبة A < رتبة [A|b].
- **الحل الكامل**: حل خاص + أساس فضاء العدم (حلول المعادلة المتجانسة).
- **تحليل LU** (بدون تبديل محوري): A = LU (L مثلثية سفلى وحدة، U مثلثية عليا)؛ حل Ly = b، ثم Ux = y.
- **المربعات الصغرى** (مفرطة التحديد): \\( \hat{x} = (A^T A)^{-1} A^T b \\) (إذا كانت الرتبة كاملة).

## 4. الفضاءات المتجهة والفضاءات الجزئية
- **الفضاء المتجهي**: مغلق تحت الجمع/الضرب القياسي؛ البديهيات (مثل، المتجه الصفري، المعكوسات).
- **الفضاءات الجزئية**: المدى لمتجهات؛ مغلق، يحتوي على 0.
  - فضاء الأعمدة: Col(A) = مدى(أعمدة A)؛ البعد = رتبة A.
  - فضاء الصفوف: Row(A) = Col(A^T)؛ البعد = رتبة A.
  - فضاء العدم: Nul(A) = {x | Ax = 0}؛ البعد = n - رتبة A.
  - فضاء العدم الأيسر: Nul(A^T).
- **الاستقلال الخطي**: c1 v1 + ... + ck vk = 0 ⇒ كل ci = 0.
- **الأساس**: مجموعة ممتدة مستقلة خطياً.
- **البعد**: عدد المتجهات في الأساس؛ بعد Col(A) + بعد Nul(A) = n (مبرهنة الرتبة-العدم).
- **الرتبة**: عدد الأعمدة المحورية = بعد Col(A) = بعد Row(A).

## 5. التحويلات الخطية
- **التعريف**: T: V → W خطي إذا كان T(u + v) = T u + T v، T(cu) = c T u.
- **التمثيل بالمصفوفة**: [T] بالنسبة للقواعد = A حيث T(x) = A x (الأساس القياسي).
- **النواة**: Ker T = Nul(A); الصورة: Im T = Col(A).
- **التشاكل البيني**: تقابل (مصفوفة قابلة للعكس).
- **مبرهنة الرتبة-العدم**: بعد Ker T + بعد Im T = بعد V.

## 6. القيم الذاتية والمتجهات الذاتية
- **التعريف**: A v = λ v (v ≠ 0 متجه ذاتي، λ قيمة ذاتية).
- **المعادلة المميزة**: det(A - λ I) = 0؛ الجذور λi (التعددية الجبرية).
- **المتجهات الذاتية**: حل (A - λ I) v = 0؛ التعددية الهندسية = بعد الفضاء الذاتي.
- **قابلة للقطرنة**: n متجهات ذاتية مستقلة خطياً ⇒ A = X Λ X^{-1} (Λ قطرية(λi)، X = [v1 ... vn]).
  - A متماثلة: دائماً قابلة للقطرنة؛ متجهات ذاتية متعامدة (A = Q Λ Q^T، Q متعامدة).
- **الأثر**: tr A = ∑ λi.
- **المحدد**: det A = ∏ λi.
- **مصفوفات متشابهة**: A ~ B إذا كانت A = P B P^{-1}; نفس القيم الذاتية، الأثر، المحدد.

## 7. الضرب الداخلي والأشكال التربيعية
- **الضرب الداخلي**: <u, v> = u^T v (إقليدسي)؛ ||v|| = √<v,v>.
- **التعامد**: <u,v> = 0؛ أساس متعامد طبيعي: <ei, ej> = δij.
- **غرام-شميدت**: تعامد الأساس {v1,...,vn} → {u1,...,un}.
  - u1 = v1; uk = vk - proj_{span(u1..u_{k-1})} vk; proj_w v = (<v,w>/<w,w>) w.
- **الشكل التربيعي**: q(x) = x^T A x (A متماثلة).
  - موجب معرف: q(x) > 0 من أجل x ≠ 0 (كل λi > 0).
  - قطرنة: q(x) = ∑ λi yi^2 (y = Q^T x).

## نصائح سريعة
- احسب الرتبة: اختزل الصفوف لإيجاد عدد المحاور.
- تحقق من القابلية للقطرنة: التعددية الهندسية = التعددية الجبرية لكل λ.
- للامتحانات: تدرب على اختزال الصفوف، توسيع المحددات، مسائل القيم الذاتية على مصفوفات 2x2/3x3.
- الأخطاء الشائعة: نسيان التعددية؛ الإشارة في العوامل المساعدة؛ عدم إبدالية الضرب.
