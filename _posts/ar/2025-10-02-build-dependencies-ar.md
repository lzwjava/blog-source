### تبعيات التطوير (DevDependencies)
هذه الحزم تُستخدم أثناء التطوير ولا تُضمَّن في البناء النهائي.

- **babel-core** (6.13.0): المترجم الأساسي لـ Babel لتحويل JavaScript الحديث (ES6+) إلى ES5 لتوافق أوسع مع المتصفحات.
- **babel-loader** (6.2.5): محمِّل Webpack الذي يدمج Babel لتحويل JavaScript في عمليات البناء.
- **babel-plugin-transform-runtime** (6.12.0): إضافة Babel تعيد استخدام برامج المساعدة وقت التشغيل لتقليل حجم الحزمة في الكود المحوَّل.
- **babel-preset-es2015** (6.13.2): إعداد مسبق لـ Babel لتحويل ميزات ES2015 (ES6) إلى ES5.
- **babel-runtime** (6.11.6): مكتبة وقت التشغيل التي توفر Polyfills وبرامج المساعدة للكود المحوَّل بواسطة Babel.
- **cross-env** (^1.0.8): يضبط متغيرات البيئة عبر المنصات (مثل NODE_ENV) دون اختلافات في Shell.
- **css-loader** (^0.23.1): يحمّل ويعالج ملفات CSS، ويحل الواردات والتبعيات.
- **detect-indent** (4.0.0): يكتشف نمط المسافة البادئة (مسافات/علامات جدولة) للملفات من أجل تنسيق متسق.
- **exports-loader** (^0.6.3): يجعل تصديرات الوحدة النمطية متاحة في سياقات مختلفة (مثل الوحدات غير AMD).
- **extract-text-webpack-plugin** (^1.0.1): يستخرج CSS من حزم JavaScript إلى ملفات منفصلة لأداء أفضل.
- **file-loader** (0.9.0): يتعامل مع تحميل الملفات (مثل الصور) عن طريق إصدارها إلى دليل الإخراج وإرجاع عناوين URL.
- **html-webpack-plugin** (^2.22.0): يولد ملفات HTML ويحقن الأصول المجمعة، مما يبسط إعدادات تطبيقات الصفحة الواحدة.
- **rimraf** (^2.5.4): إزالة الملفات العودية عبر المنصات (مثل rm -rf على يونكس).
- **style-loader** (^0.13.1): يحقن CSS في DOM عبر علامات النمط للتحميل الديناميكي.
- **stylus** (^0.54.5): معالج مسبق لـ CSS مع بناء جملة تعبيري، يُترجم إلى CSS.
- **stylus-loader** (^2.1.1): محمِّل Webpack لمعالجة ملفات Stylus إلى CSS.
- **url-loader** (0.5.7): يشفر الملفات الصغيرة (مثل الصور) بتنسيق Base64 مضمنًا أو يصدر الملفات الأكبر حجمًا؛ يعود إلى file-loader.
- **vue-hot-reload-api** (^1.2.0): يمكّن استبدال الوحدة النمطية الساخنة لمكونات Vue.js أثناء التطوير.
- **vue-html-loader** (^1.0.0): محمِّل Webpack لتحليل قوالب HTML في مكونات Vue ذات الملف الواحد.
- **vue-loader** (8.5.3): يحمّل ويعالج مكونات Vue ذات الملف الواحد (ملفات .vue) إلى JavaScript و CSS.
- **vue-style-loader** (^1.0.0): يتعامل مع CSS من مكونات Vue، متكاملًا مع style-loader.
- **webpack** (1.13.2): حزمة وحدات نمطية لبناء وتحسين أصول الويب مثل JS و CSS والصور.
- **webpack-dev-server** (1.14.0): خادم تطوير مع إعادة تحميل مباشرة واستبدال الوحدة النمطية الساخنة.

### التبعيات (Dependencies)
هذه حزم وقت التشغيل المضمنة في بناء التطبيق النهائي.

- **debug** (^2.2.0): أداة تصحيح مع تسجيل موسَّم ومخرجات مشروطة (تُفعَّل فقط عبر متغير البيئة DEBUG).
- **es6-promise** (^3.0.2): Polyfill لواجهة برمجة تطبيقات Promise الخاصة بـ ES6 في المتصفحات/البيئات القديمة.
- **font-awesome** (^4.6.3): مكتبة أيقونات شائعة توفر أيقونات متجهة قابلة للتطوير عبر فئات CSS.
- **github-markdown-css** (^2.4.0): CSS لتنسيق Markdown بنكهة GitHub.
- **highlight.js** (^9.6.0): أداة تمييز الصيغة لكتل الكود بلغات متعددة.
- **hls.js** (^0.7.6): مكتبة JavaScript لتشغيل فيديو HTTP Live Streaming (HLS) مع فيديو HTML5.
- **inherit** (^2.2.6): أداة مساعدة للوراثة الكلاسيكية والنموذجية في كائنات JavaScript.
- **jquery** (^3.1.0): مكتبة JavaScript سريعة وغنية بالميزات لمعالجة DOM و AJAX والأحداث.
- **json-loader** (^0.5.4): يحمّل ملفات JSON كوحدات نمطية JavaScript.
- **leancloud-realtime** (^3.2.3): SDK لخدمة المراسلة في الوقت الفعلي ومزامنة البيانات من LeanCloud.
- **marked** (^0.3.6): محلل Markdown يحول Markdown إلى HTML.
- **moment** (^2.14.1): مكتبة معالجة التاريخ/الوقت لتحليل والتحقق من صحة وتنسيق التواريخ.
- **moment-timezone** (^0.5.7): امتداد لـ Moment.js للتعامل مع المناطق الزمنية والتوقيت الصيفي.
- **qiniu-js-sdk** (0.0.2): JavaScript SDK للتفاعل مع تخزين سحابة Qiniu (تحميل/تنزيل الملفات).
- **raw-loader** (^0.5.1): محمِّل Webpack يحمّل الملفات كسلاسل نصية خام (مثل القوالب النصية).
- **rupture** (^0.6.1): امتداد استعلام الوسائط لـ Stylus، يوفر نقاط توقف للتصميم المتجاوب.
- **sprintf-js** (^1.0.3): تنفيذ sprintf لـ JavaScript لتنسيق السلاسل النصية مع العناصر النائبة.
- **vue** (^1.0.26): إطار عمل MVVM تقدمي لبناء واجهات المستخدم (الإصدار 1.x، وهو أقدم).
- **vue-resource** (^0.6.1): عميل HTTP لـ Vue.js لتبسيط طلبات AJAX/API.
- **vue-router** (^0.7.13): الموجه الرسمي لـ Vue.js، مما يمكّن التنقل في تطبيقات الصفحة الواحدة.
- **vue-spinner** (^1.0.2): مكتبة مكونات Spinner/تحميل لـ Vue.js.
- **vue-weui** (^0.3.2): أغلفة Vue.js لمكونات WeUI (مكتبة واجهة مستخدم لواجهات نمط WeChat).
- **weixin-js-sdk** (^1.2.0): WeChat JavaScript SDK لدمج ميزات ويب WeChat (مثل المشاركة، المسح).
- **weui** (^0.4.3): إطار عمل CSS لـ WeUI لبناء واجهات محمولة مستوحاة من تصميم WeChat.
- **word-color** (^1.2.0): مكتبة لتلوين الكلمات بناءً على عوامل مثل التجزئة أو اللغة، على الأرجح لتأثيرات شبيهة بخريطة الحرارة.
