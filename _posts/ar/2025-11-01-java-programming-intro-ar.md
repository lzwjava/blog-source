## أولاً: طبيعة المقرر وأهدافه
- **الطبيعة**: هذا مقرر أساسي في البرمجة يركز على البرمجة كائنية التوجه (OOP) باستخدام Java. تعتبر Java لغة مستقلة عن المنصة، آمنة، ومتعددة الاستخدامات، مما يجعلها مثالية لتطوير الشبكات والتطبيقات.
- **الأهداف**:
  - فهم مفاهيم البرمجة الأساسية وميزات Java.
  - إتقان بناء جملة Java، وأنواع البيانات، وهياكل التحكم، والطرق (Methods).
  - تطوير المهارات في إنشاء برامج بسيطة، والتعامل مع هياكل البيانات مثل المصفوفات، وتطبيق مبادئ البرمجة كائنية التوجه.
  - إعداد الطلاب لحل المشكلات العملية من خلال الترميز وتصحيح الأخطاء.

## ثانيًا: أهداف التقييم
- **المعرفة**: فهم بيئة تطوير Java، وبناء الجملة الأساسي، والمكتبات الأساسية.
- **المهارات**: كتابة برامج Java وتجميعها وتنفيذها؛ تصميم خوارزميات بسيطة؛ استخدام البرمجة كائنية التوجه لتنظيم الكود.
- **شكل الامتحان**: مغلق الكتاب، 100 نقطة، 90 دقيقة. يتضمن أسئلة متعددة الخيارات، صح/خطأ، ملء الفراغات، قراءة البرنامج، إجابات قصيرة، وأسئلة برمجة.
- **التوزيع النسبي**: المعرفة النظرية (40%)، البرمجة العملية (60%).

## ثالثًا: محتوى المقرر والمتطلبات

### الجزء الأول: نظرة عامة على Java
- مواصفات لغة Java، واجهة برمجة التطبيقات (API)، ومجموعة تطوير Java (JDK)، وأدوات بيئة التطوير المتكاملة (IDE).
- إنشاء برامج Java وتجميعها وتشغيلها.
- إرشادات نمط البرمجة.
- أخطاء البرمجة الشائعة وأساسيات تصحيح الأخطاء.

### الجزء الثاني: البرمجة الأساسية
- قراءة المدخلات من وحدة التحكم (Console).
- المعرفات، المتغيرات، التعبيرات، جمل التعيين، والثوابت.
- أنواع البيانات الرقمية والعمليات؛ تحويلات الأنواع.
- تقييم التعبيرات وأولوية العوامل.
- التعيين المعزز، وعوامل الزيادة/النقصان.

### الجزء الثالث: هياكل الاختيار
- جمل if-else والتضمين.
- جمل switch.
- التعبيرات الشرطية.
- قواعد أولوية العوامل وارتباطها.

### الجزء الرابع: الدوال الرياضية، المحارف، والسلاسل النصية
- الدوال الرياضية الشائعة (مثل فئة Math).
- أنواع بيانات المحارف والعمليات عليها.
- نوع String: الإنشاء، المعالجة، والطرق.
- تنسيق إخراج وحدة التحكم (مثل printf).

### الجزء الخامس: الحلقات التكرارية
- حلقات while.
- حلقات do-while.
- حلقات for.
- الحلقات المتداخلة.
- كلمتي break و continue.

### الجزء السادس: الطرق (Methods)
- تعريف واستدعاء الطرق.
- تمرير المعاملات بالقيمة (Pass by value).
- زيادة تحميل الطرق (Overloading).
- نطاق المتغيرات وعمرها الافتراضي.

### الجزء السابع: المصفوفات (Arrays)
- أساسيات المصفوفات (الإعلان، التهيئة).
- تمرير المصفوفات إلى الطرق.
- إرجاع المصفوفات من الطرق.
- الوسائط متغيرة الطول (Varargs).
- المصفوفات متعددة الأبعاد.

### الجزء الثامن: الكائنات والفئات (Objects and Classes)
- تعريف الفئات وإنشاء الكائنات.
- الوصول إلى الكائنات عبر متغيرات المرجع.
- استخدام الفئات المضمنة (مثل Date, Random).
- المتغيرات والطرق الساكنة (Static).
- معدلات الوصول (public, private، إلخ).
- تغليف (Encapsulation) حقول البيانات.
- تمرير الكائنات كمعاملات.
- مصفوفات الكائنات.
- نطاق متغيرات الفئة.
- مرجع 'this'.

### الجزء التاسع: الوراثة وتعدد الأشكال (Inheritance and Polymorphism)
- الفئات العليا (Superclasses) والفئات الفرعية (Subclasses).
- الكلمة الأساسية super.
- تعديل الطرق (Overriding).
- فئة Object وطريقة toString().
- تعدد الأشكال (Polymorphism) والربط الديناميكي (Dynamic binding).
- تحويل نوع الكائنات (Casting) وعامل instanceof.
- الوصول protected.
- الكلمة الأساسية final لمنع الوراثة/التعديل.

### الجزء العاشر: الفئات المجردة والواجهات (Abstract Classes and Interfaces)
- الفئات والطرق المجردة (Abstract).
- الواجهات (Interfaces): التعريف والتنفيذ.
- الواجهة Comparable (على سبيل المثال، للفرز).
