استخدام أردوينو للتحقق من قوانين كيرشوف—تحديداً قانون كيرشوف للتيار (KCL) وقانون كيرشوف للجهد (KVL)—هو طريقة رائعة لدمج نظرية الإلكترونيات مع التجريب العملي. ينص قانون كيرشوف للتيار (KCL) على أن التيار الكلي الداخل إلى نقطة تفرع يساوي التيار الكلي الخارج منها، بينما ينص قانون كيرشوف للجهد (KVL) على أن مجموع فروق الجهد حول حلقة مغلقة يساوي الجهد المُغذي. إليك كيف يمكنك تصميم دوائر بسيطة واستخدام أردوينو لقياس التيارات والجهود لتأكيد هذه القوانين.

بما أن أردوينو لا يمكنه قياس التيار مباشرة، سنستنتجه عن طريق قياس الجهد عبر المقاومات (باستخدام قانون أوم: \\( I = V/R \\))، ويمكنه قياس الجهد عبر منافذه التناظرية (نطاق 0-5 فولت). أدناه، سأوضح تجربتين—واحدة لـ KCL وأخرى لـ KVL—مع تعليمات خطوة بخطوة، وطريقة التوصيل، والكود.

### **التجربة 1: التحقق من قانون كيرشوف للتيار (KCL)**

#### **الهدف**
إثبات أن التيار الداخل إلى عقدة يساوي التيار الخارج منها.

#### **إعداد الدائرة**
- **المكونات:**
  - أردوينو (مثل Uno)
  - 3 مقاومات (مثل R1 = 330Ω، R2 = 470Ω، R3 = 680Ω)
  - لوحة تجارب (بريد بورد) وأسلاك توصيل
  - مصدر طاقة 5 فولت (من منفذ 5V في الأردوينو)
- **طريقة التوصيل:**
  - وصل 5V من الأردوينو إلى عقدة (سمها العقدة A).
  - من العقدة A، وصل R1 إلى GND (الفرع 1).
  - من العقدة A، وصل R2 إلى GND (الفرع 2، موازٍ لـ R1).
  - من العقدة A، وصل R3 إلى GND (الفرع 3، موازٍ لـ R1 و R2).
  - استخدم المنافذ التناظرية في الأردوينو لقياس الجهد عبر كل مقاومة:
    - A0 عبر R1 (مسبار عند العقدة A، والآخر عند GND).
    - A1 عبر R2.
    - A2 عبر R3.
- **ملاحظة:** نقطة GND هي النقطة المرجعية المشتركة.

#### **النظرية**
- التيار الكلي من 5V إلى العقدة A (\\( I_{in} \\)) يتفرع إلى \\( I_1 \\)، \\( I_2 \\)، و \\( I_3 \\) عبر R1، R2، و R3.
- قانون كيرشوف للتيار: \\( I_{in} = I_1 + I_2 + I_3 \\).
- قم بقياس الجهد عبر كل مقاومة، ثم احسب التيار: \\( I = V/R \\).

#### **كود الأردوينو**
```cpp
void setup() {
  Serial.begin(9600); // بدء الاتصال التسلسلي
}

void loop() {
  // قراءة قيم الجهد (0-1023 تُقابل 0-5V)
  int sensorValue1 = analogRead(A0); // الجهد عبر R1
  int sensorValue2 = analogRead(A1); // الجهد عبر R2
  int sensorValue3 = analogRead(A2); // الجهد عبر R3

  // التحويل إلى جهد (مرجع 5V، محول تناظري رقمي 10-بت)
  float V1 = sensorValue1 * (5.0 / 1023.0);
  float V2 = sensorValue2 * (5.0 / 1023.0);
  float V3 = sensorValue3 * (5.0 / 1023.0);

  // قيم المقاومات (بالأوم)
  float R1 = 330.0;
  float R2 = 470.0;
  float R3 = 680.0;

  // حساب التيارات (I = V/R)
  float I1 = V1 / R1;
  float I2 = V2 / R2;
  float I3 = V3 / R3;

  // التيار الكلي الداخل إلى العقدة (بافتراض أن جهد المصدر = 5V)
  float totalResistance = 1.0 / ((1.0/R1) + (1.0/R2) + (1.0/R3)); // التوصيل على التوازي
  float Iin = 5.0 / totalResistance;

  // عرض النتائج
  Serial.print("I1 (mA): "); Serial.println(I1 * 1000);
  Serial.print("I2 (mA): "); Serial.println(I2 * 1000);
  Serial.print("I3 (mA): "); Serial.println(I3 * 1000);
  Serial.print("Iin (mA): "); Serial.println(Iin * 1000);
  Serial.print("Sum of I1+I2+I3 (mA): "); Serial.println((I1 + I2 + I3) * 1000);
  Serial.println("---");

  delay(2000); // انتظر ثانيتين
}
```

#### **التحقق**
- افتح نافذة المراقبة التسلسلية (Ctrl+Shift+M في Arduino IDE، اضبط على 9600 باود).
- قارن \\( I_{in} \\) (المحسوب من المقاومة الكلية) مع \\( I_1 + I_2 + I_3 \\). يجب أن يكونا متساويين تقريباً، مما يتحقق من قانون كيرشوف للتيار.
- قد تنشأ فروق طفيفة بسبب تفاوت قيم المقاومات أو دقة محول الأردوينو التناظري.

### **التجربة 2: التحقق من قانون كيرشوف للجهد (KVL)**

#### **الهدف**
إظهار أن مجموع فروق الجهد حول حلقة مغلقة يساوي جهد المصدر.

#### **إعداد الدائرة**
- **المكونات:**
  - أردوينو
  - 2 مقاومات (مثل R1 = 330Ω، R2 = 470Ω)
  - لوحة تجارب (بريد بورد) وأسلاك توصيل
  - مصدر طاقة 5 فولت (من منفذ 5V في الأردوينو)
- **طريقة التوصيل:**
  - وصل 5V إلى R1.
  - وصل R1 إلى R2.
  - وصل R2 إلى GND.
  - قم بقياس الجهود:
    - A0 عبر الحلقة كاملة (من 5V إلى GND) للتأكد من جهد المصدر.
    - A1 عبر R1 (من 5V إلى نقطة التقاء R1 و R2).
    - A2 عبر R2 (من نقطة التقاء إلى GND).
- **ملاحظة:** استخدم دائرة مقسم الجهد؛ تأكد من أن الجهود لا تتجاوز 5 فولت (حد الأردوينو).

#### **النظرية**
- قانون كيرشوف للجهد: \\( V_{source} = V_{R1} + V_{R2} \\).
- قم بقياس كل انخفاض في الجهد وتحقق مما إذا كان مجموعهما يساوي جهد المصدر (5V).

#### **كود الأردوينو**
```cpp
void setup() {
  Serial.begin(9600);
}

void loop() {
  // قراءة الجهود
  int sensorValueSource = analogRead(A0); // عبر 5V إلى GND
  int sensorValueR1 = analogRead(A1);     // عبر R1
  int sensorValueR2 = analogRead(A2);     // عبر R2

  // التحويل إلى جهد
  float Vsource = sensorValueSource * (5.0 / 1023.0);
  float VR1 = sensorValueR1 * (5.0 / 1023.0);
  float VR2 = sensorValueR2 * (5.0 / 1023.0);

  // عرض النتائج
  Serial.print("Vsource (V): "); Serial.println(Vsource);
  Serial.print("VR1 (V): "); Serial.println(VR1);
  Serial.print("VR2 (V): "); Serial.println(VR2);
  Serial.print("VR1 + VR2 (V): "); Serial.println(VR1 + VR2);
  Serial.println("---");

  delay(2000);
}
```

#### **التحقق**
- في نافذة المراقبة التسلسلية، تحقق من أن \\( V_{source} \approx V_{R1} + V_{R2} \\).
- بالنسبة لـ R1 = 330Ω و R2 = 470Ω على التوالي، نتوقع \\( V_{R1} \approx 2.1V \\) و \\( V_{R2} \approx 2.9V \\) (بناءً على مقسم الجهد: \\( V_{R1} = 5 \cdot \frac{R1}{R1+R2} \\))، ومجموعهما ~5V.
