٢٣. بما أن المتجهات \\( \alpha_1, \alpha_2, \alpha_3 \\) مستقلة خطيًا، والمتجهات \\( \beta_1 = \alpha_1 + 2\alpha_2 \\)، \\( \beta_2 = -\alpha_1 + \alpha_2 - 3\alpha_3 \\)، \\( \beta_3 = 3\alpha_1 + 6\alpha_3 \\). أثبت أن المتجهات \\( \beta_1, \beta_2, \beta_3 \\) مرتبطة خطيًا.

### الحل:
لإثبات أن المتجهات \\( \beta_1, \beta_2, \beta_3 \\) مرتبطة خطيًا، نحتاج إلى إظهار وجود قيم عددية \\( c_1, c_2, c_3 \\) (ليست جميعها صفرًا) بحيث:

\\[ c_1 \beta_1 + c_2 \beta_2 + c_3 \beta_3 = 0 \\]

المعطى:
\\[ \beta_1 = \alpha_1 + 2\alpha_2 \\]
\\[ \beta_2 = -\alpha_1 + \alpha_2 - 3\alpha_3 \\]
\\[ \beta_3 = 3\alpha_1 + 6\alpha_3 \\]

نحتاج إلى إيجاد ثوابت \\( c_1, c_2, c_3 \\) بحيث:

\\[ c_1 (\alpha_1 + 2\alpha_2) + c_2 (-\alpha_1 + \alpha_2 - 3\alpha_3) + c_3 (3\alpha_1 + 6\alpha_3) = 0 \\]

يمكن إعادة كتابة هذا كالتالي:

\\[ (c_1 - c_2 + 3c_3)\alpha_1 + (2c_1 + c_2)\alpha_2 + (-3c_2 + 6c_3)\alpha_3 = 0 \\]

بما أن \\( \alpha_1, \alpha_2, \alpha_3 \\) مستقلة خطيًا، فيجب أن تكون معاملات \\( \alpha_1, \alpha_2, \alpha_3 \\) كلها مساوية للصفر:

١. \\( c_1 - c_2 + 3c_3 = 0 \\)
٢. \\( 2c_1 + c_2 = 0 \\)
٣. \\( -3c_2 + 6c_3 = 0 \\)

لنحل نظام المعادلات هذا:

من المعادلة ٢:
\\[ 2c_1 + c_2 = 0 \implies c_2 = -2c_1 \\]

من المعادلة ٣:
\\[ -3c_2 + 6c_3 = 0 \implies -3(-2c_1) + 6c_3 = 0 \implies 6c_1 + 6c_3 = 0 \implies c_3 = -c_1 \\]

عوّض \\( c_2 = -2c_1 \\) و \\( c_3 = -c_1 \\) في المعادلة ١:
\\[ c_1 - (-2c_1) + 3(-c_1) = 0 \implies c_1 + 2c_1 - 3c_1 = 0 \implies 0 = 0 \\]

هذا صحيح لأي قيمة لـ \\( c_1 \\). لنختر \\( c_1 = 1 \\):

إذن:
\\[ c_2 = -2 \\]
\\[ c_3 = -1 \\]

وبالتالي، وجدنا ثوابت \\( c_1 = 1, c_2 = -2, c_3 = -1 \\) بحيث:

\\[ 1 \cdot \beta_1 - 2 \cdot \beta_2 - 1 \cdot \beta_3 = 0 \\]

هذا يثبت أن المتجهات \\( \beta_1, \beta_2, \beta_3 \\) مرتبطة خطيًا.
