هذه جميعها حزم Python (wheels) التي يتم تثبيتها — إليك نبذة مختصرة جدًا عن كل منها:

* **safetensors**: تنسيق ملف سريع وآمن لحفظ/تحميل بيانات tensor دون مشاكل أمان pickle.
* **typer**: مكتبة حديثة لبناء تطبيقات CLI مع تلميحات أنواع Python.
* **shellingham**: أداة مساعدة لاكتشاف أي shell (مثل bash أو zsh) تعمل فيه.
* **uvicorn**: خادم ASGI فائق السرعة لاستضافة تطبيقات ويب Python (مثل FastAPI).
* **httptools**: مكتبة تحليل HTTP منخفضة المستوى، تُستخدم بواسطة خوادم مثل Uvicorn للسرعة.
* **uvloop**: حلقة أحداث asyncio فائقة السرعة مكتوبة بـ Cython، بديل مباشر لـ asyncio.
* **watchfiles**: مكتبة مراقبة الملفات التي يمكنها تشغيل إجراءات (مثل إعادة التحميل التلقائي) عند تغييرات نظام الملفات.
* **airportsdata**: جداول بحث بسيطة لرموز المطارات وأسمائها ومواقعها.
* **astor**: أدوات لتحويل شجرة التركيب المجردة (AST) لـ Python إلى شفرة مصدرية مرة أخرى (والعكس).
* **blake3**: ربط Python لخوارزمية التجزئة التشفيرية BLAKE3.
* **cloudpickle**: أداة pickle معززة لكائنات Python، تتعامل مع أنواع أكثر من pickle في المكتبة القياسية.
* **cupy\_cuda12x**: مكتبة مصفوفات GPU متوافقة مع NumPy لـ CUDA 12.x.
* **fastrlock**: تنفيذ سريع جدًا لقفل قابل لإعادة الدخول لـ Python.
* **diskcache**: مكتبة ذاكرة تخزين مؤقت مدعومة بالقرص للحفاظ على كائنات Python بكفاءة.
* **einops**: عمليات إعادة تشكيل وإعادة ترتيب tensor معبرة ويمكن قراءتها.
* **importlib\_metadata**: الوصول إلى بيانات وصف الحزمة (الإصدار، نقاط الدخول) أثناء وقت التشغيل (إصدار متوافق للأنظمة القديمة من Python).
* **zipp**: واجهة على غرار pathlib لأرشيفات ZIP، تُستخدم من قبل importlib\_metadata.
* **msgspec**: تسلسل/إلغاء تسلسل الرسائل عالي الأداء (مثل JSON، Msgpack، إلخ).
* **nest\_asyncio**: يسمح بتداخل حلقات أحداث asyncio (مثلاً لتشغيل notebooks).
* **ninja**: روابط Python لنظام البناء Ninja (بناء سريع وتدريجي).
* **partial\_json\_parser**: محلل JSON تدفقي/جزئي لـ JSON كبير أو غير صحيح التكوين.
* **py\_cpuinfo**: جمع معلومات مفصلة حول وحدة المعالجة المركزية (الموديل، الأنوية، التردد).
* **python\_json\_logger**: مُنسق JSON لوحدة التسجيل (logging) في Python.
* **pyzmq**: روابط Python لمكتبة المراسلة عالية الأداء ZeroMQ.
* **scipy**: مكتبة الحوسبة العلمية الأساسية (التحسين، التكامل، الإحصاء، إلخ).
* **outlines\_core**: أدوات مساعدة أساسية لمكتبة "outlines" (تتعامل مع هيكل المستندات).
* **xformers**: تنفيذات transformer (انتباه) فعالة ومعيارية من Meta.
* **sentencepiece**: أداة tokenizer/detokenizer للنص غير الخاضعة للإشراف (شائعة الاستخدام في معالجة اللغات الطبيعية NLP).
