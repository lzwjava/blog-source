أحاول الاتصال بجهاز في Hetzner Cloud باستخدام IPv6. لم تنجح عملية `ssh 2a01:4f8:c17:2000::/64`، لكن `ssh root@2a01:4f8:c17:2000::1` نجحت.

تم نسخ عنوان IPv6 من لوحة تحكم Hetzner Cloud.

يمكن تكوين ملف `~/.ssh/config` لتطبيق قواعد وكيل مختلفة لعناوين IPv4 و IPv6. يسمح هذا الإعداد بتحديد أمر وكيل لعناوين IPv4 مع معالجة عناوين IPv6 بشكل مختلف.

```bash
Host 192.168.1.*
    UseKeychain yes
    AddKeysToAgent yes
    IdentityFile ~/.ssh/id_rsa 
Host *.*.*.*
    UseKeychain yes
    AddKeysToAgent yes
    IdentityFile ~/.ssh/id_rsa
    ProxyCommand corkscrew localhost 7890 %h %p
Host *
    UseKeychain yes
    AddKeysToAgent yes
    IdentityFile ~/.ssh/id_rsa   
```
