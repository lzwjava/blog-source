### ما هو win10toast؟
win10toast هي مكتبة Python خفيفة الوزن تمكنك من إرسال إشعارات Toast (تنبيهات منبثقة) على نظام Windows 10 من نصوص Python الخاصة بك. وهي مفيدة لإشعارات المستخدم في تطبيقات سطح المكتب أو النصوص، على غرار إشعارات Windows الأصلية.

### المتطلبات الأساسية
- تثبيت Python 3.x.
- نظام Windows 10 (تعتمد على واجهات برمجة تطبيقات خاصة بـ Windows).
- لا توجد تبعيات إضافية بخلاف Python.

### التثبيت
قم بتثبيت المكتبة عبر pip (أداة تثبيت حزم Python):

```
pip install win10toast
```

إذا كنت تستخدم بيئة افتراضية، فقم بتنشيطها أولاً.

### الاستخدام الأساسي
1. استيراد الوحدة:
   ```python
   from win10toast import ToastNotifier
   ```

2. إنشاء كائن من `ToastNotifier` واستدعاء الدالة `show_toast` لعرض الإشعار:
   ```python
   toaster = ToastNotifier()
   toaster.show_toast("العنوان", "الرسالة", icon_path=None, duration=5)
   ```
   - **العنوان**: نص لعنوان الإشعار الرئيسي.
   - **الرسالة**: نص لمحتوى الإشعار.
   - **icon_path**: مسار اختياري لملف .ico أو .png لأيقونة الإشعار (مثال: `"path/to/icon.ico"`). اتركه فارغًا لعدم استخدام أيقونة.
   - **duration**: المدة بالثواني لظهور الإشعار (القيمة الافتراضية هي 5؛ قد يتجاوز نظام Windows هذه القيمة).

سيظهر الإشعار في مركز الإجراءات (Action Center) وكنافذة منبثقة.

### الاستخدام المتقدم
- **تخصيص الأيقونة**: استخدم ملف صورة (يُوصى بدقة 128x128 بكسل للحصول على أفضل جودة).
- **معالجة أحداث النقر**: مرر دالة رد اتصال (الدعم محدود؛ السلوك عند النقر أساسي في الغالب).
- **إشعارات متعددة**: قم بإنشاء كائن `ToastNotifier` مرة واحدة وإعادة استخدامه لإرسال إشعارات متعددة.
- **معالجة الأخطاء**: ضع الكود داخل try-except للتعامل مع مشاكل واجهات برمجة تطبيقات Windows المحتملة، على سبيل المثال، إذا كانت الإشعارات معطلة.

مثال باستخدام أيقونة ومعالجة الأخطاء:
```python
from win10toast import ToastNotifier
import os

try:
    toaster = ToastNotifier()
    icon = "C:\\path\\to\\icon.ico"  # استبدل بالمسار الفعلي
    toaster.show_toast("تنبيه!", "لقد اكتمل تشغيل النص البرمجي.", icon_path=icon, duration=10)
except Exception as e:
    print(f"فشل الإشعار: {e}")
```

### القيود والنصائح
- تتطلب الإشعارات تمكين مركز الإجراءات في Windows 10 (الإعدادات > النظام > الإشعارات والإجراءات).
- للحصول على ميزات أكثر تعقيدًا (مثل الإجراءات أو الصور)، فكر في بدائل مثل plyer أو winrt (جزء من Windows SDK الحديث).
- إذا لم تظهر الإشعارات، تحقق من إعدادات "وضع التركيز" في Windows أو حاول تشغيل النص البرمجي كمسؤول.
