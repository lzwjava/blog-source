لتركيب ملف `discord-0.0.111.deb`، يمكنك استخدام عدة طرق:

**الطريقة 1: استخدام dpkg (سطر الأوامر)**
```bash
sudo dpkg -i discord-0.0.111.deb
```

إذا واجهت مشاكل في التبعيات، قم بتشغيل:
```bash
sudo apt-get install -f
```

**الطريقة 2: استخدام apt (موصى بها)**
```bash
sudo apt install ./discord-0.0.111.deb
```

هذه الطريقة تتعامل مع التبعيات تلقائيًا بشكل أفضل من dpkg.

**الطريقة 3: استخدام gdebi (إذا كان مثبتًا)**
```bash
sudo gdebi discord-0.0.111.deb
```

**الطريقة 4: الطريقة الرسومية (GUI)**
إذا كنت تستخدم بيئة سطح مكتب، يمكنك غالبًا النقر المزدوج على ملف .deb وسيفتح في مدير الحزم الخاص بك (مثل Software Center) من أجل التثبيت.

**المتطلبات الأساسية:**
- تأكد من أنك في الدليل الذي يحتوي على ملف .deb
- ستحتاج إلى صلاحيات sudo
- إذا لم يكن الملف موجودًا لديك بعد، يمكنك تحميله من الموقع الرسمي لـ Discord

**بعد التثبيت:**
يجب أن تكون قادرًا على تشغيل Discord من قائمة التطبيقات الخاصة بك أو بتشغيل `discord` في الطرفية.

أي طريقة تفضل استخدامها، أو هل تحتاج إلى مساعدة في خطوة محددة؟
