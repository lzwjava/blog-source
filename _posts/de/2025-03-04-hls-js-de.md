### Schritte zur Verwendung von `hls.js` (v0.7.6)

#### 1. **Installieren oder Einbinden der Bibliothek**
Sie können `hls.js` in Ihr Projekt auf eine der folgenden Weisen einbinden:

- **Über npm** (wenn Sie ein Build-Tool wie Webpack verwenden):
  ```bash
  npm install hls.js@^0.7.6
  ```
  Dann importieren Sie es in Ihrem JavaScript:
  ```javascript
  var Hls = require('hls.js');
  ```

- **Über CDN** (wenn Sie es direkt in eine HTML-Datei einfügen):
  ```html

  ```
  Dies macht `Hls` als globale Variable verfügbar.

#### 2. **Einrichten Ihres HTML**
Sie benötigen ein `<video>`-Element in Ihrem HTML, um den HLS-Stream daran anzuhängen:
```html
<video id="video" controls></video>
```

#### 3. **Initialisieren von `hls.js`**
Hier ist ein grundlegendes Beispiel, wie man `hls.js` verwendet, um einen HLS-Stream (z.B. eine `.m3u8`-Datei) zu laden und abzuspielen:

```javascript
// Überprüfen Sie, ob der Browser HLS nativ unterstützt (z.B. Safari)
// Wenn nicht, verwenden Sie hls.js
var video = document.getElementById('video');
var videoSrc = 'https://example.com/playlist.m3u8'; // Ersetzen Sie dies durch Ihre HLS-URL

if (Hls.isSupported()) {
    var hls = new Hls();
    hls.loadSource(videoSrc);
    hls.attachMedia(video);
    hls.on(Hls.Events.MANIFEST_PARSED, function () {
        video.play();
    });
} else if (video.canPlayType('application/vnd.apple.mpegurl')) {
    // Für Browser mit nativem HLS-Support (z.B. Safari)
    video.src = videoSrc;
    video.addEventListener('loadedmetadata', function () {
        video.play();
    });
}
```

#### 4. **Wichtige Konzepte**
- **`Hls.isSupported()`**: Überprüft, ob der Browser `hls.js` benötigt (gibt `true` zurück, wenn Media Source Extensions unterstützt werden).
- **`loadSource(url)`**: Lädt das HLS-Manifest (`.m3u8`-Datei).
- **`attachMedia(video)`**: Hängt das Videoelement an die `hls.js`-Instanz an.
- **`Hls.Events.MANIFEST_PARSED`**: Ereignis, das ausgelöst wird, wenn das Manifest geladen ist und der Stream bereit zum Abspielen ist.

#### 5. **Beispiel mit Fehlerbehandlung**
Um es robuster zu machen:
```javascript
var video = document.getElementById('video');
var videoSrc = 'https://example.com/playlist.m3u8';

if (Hls.isSupported()) {
    var hls = new Hls();
    hls.loadSource(videoSrc);
    hls.attachMedia(video);
    hls.on(Hls.Events.MANIFEST_PARSED, function () {
        video.play();
    });
    hls.on(Hls.Events.ERROR, function (event, data) {
        console.error('HLS Fehler:', data);
        if (data.fatal) {
            switch (data.type) {
                case Hls.ErrorTypes.NETWORK_ERROR:
                    console.log('Netzwerkfehler, erneuter Versuch...');
                    hls.startLoad();
                    break;
                case Hls.ErrorTypes.MEDIA_ERROR:
                    console.log('Medienfehler, Wiederherstellung...');
                    hls.recoverMediaError();
                    break;
                default:
                    hls.destroy();
                    break;
            }
        }
    });
} else if (video.canPlayType('application/vnd.apple.mpegurl')) {
    video.src = videoSrc;
}
```
