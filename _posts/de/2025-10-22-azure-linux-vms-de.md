Ja, Azure Virtual Machines (VMs) unterstützen vollständig den Betrieb von Linux-Betriebssystemen. Sie können eine Vielzahl beliebter Distributionen direkt aus dem Azure Marketplace bereitstellen oder Ihre eigenen benutzerdefinierten Linux-Images hochladen.

### Wichtige Highlights:
- **Unterstützte Distributionen**: Azure unterstützt und befürwortet alle gängigen Linux-Varianten, darunter Ubuntu, Red Hat Enterprise Linux (RHEL), Debian, SUSE Linux Enterprise Server (SLES), Oracle Linux, CentOS Stream, Rocky Linux, AlmaLinux und mehr. Über 100 Linux-Images sind sofort einsatzbereit verfügbar.
- **Bereitstellungsoptionen**: Verwenden Sie das Azure-Portal, die CLI, PowerShell oder ARM-Vorlagen, um Linux-VMs zu erstellen. Sie können beispielsweise eine Ubuntu Server-Instanz in wenigen Minuten starten.
- **Funktionen**: Linux-VMs in Azure beinhalten eine vollständige Integration mit Azure-Diensten wie Storage, Networking, Security und Monitoring. Sie unterstützen auch Tools wie Azure Arc für Hybrid Management.

Wenn Sie gerade erst anfangen, lesen Sie den offiziellen Schnellstart-Leitfaden zum Erstellen einer Linux-VM.
