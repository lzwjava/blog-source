Der JDK-Pfad befindet sich unter `/opt/homebrew/Cellar/openjdk@17/17.0.14/libexec/openjdk.jdk/Contents/Home`.

```bash
mvn build
mvn verify
mvn install
mvn spring-boot:run
```

Kompilieren und führen Sie den Code über die Kommandozeile aus.
