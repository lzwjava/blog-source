## **1. Bildverarbeitung mit einem CNN (Convolutional Neural Network)**
CNNs werden h√§ufig f√ºr Bildklassifizierung, Objekterkennung und Feature-Extraktion verwendet.

### **Beispiel: Verwendung eines vortrainierten CNNs (ResNet)**
```python
import torch
import torchvision.models as models
import torchvision.transforms as transforms
from PIL import Image

# Ein vortrainiertes ResNet-Modell laden
model = models.resnet18(pretrained=True)
model.eval()  # In den Evaluierungsmodus versetzen

# Bildvorverarbeitung definieren
preprocess = transforms.Compose([
    transforms.Resize(256),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]),
])

# Ein Bild laden und vorverarbeiten
image = Image.open("example.jpg")  # Ersetzen Sie dies mit Ihrem Bildpfad
input_tensor = preprocess(image)
input_batch = input_tensor.unsqueeze(0)  # Batch-Dimension hinzuf√ºgen

# Auf GPU verschieben, falls verf√ºgbar
if torch.cuda.is_available():
    input_batch = input_batch.to('cuda')
    model.to('cuda')

# Features extrahieren (vor der endg√ºltigen Klassifizierungsschicht)
with torch.no_grad():
    features = model(input_batch)

print("Feature-Vektor-Form:", features.shape)  # z.B. torch.Size([1, 1000])
```
**Erkl√§rung**:
1. **ResNet18** ist eine CNN-Architektur, die auf ImageNet vortrainiert wurde.
2. Das Bild wird vorverarbeitet (skaliert, normalisiert).
3. Das Modell wandelt das Bild in einen **Feature-Vektor** um (z.B. 1000-dimensional f√ºr ResNet18).

## **2. Bildverarbeitung mit einem Vision Transformer (ViT)**
ViTs behandeln Bilder als Sequenzen von Patches und verwenden Self-Attention-Mechanismen (wie in der NLP).

### **Beispiel: Verwendung eines vortrainierten ViT (Hugging Face)**
```python
from transformers import ViTFeatureExtractor, ViTModel
from PIL import Image
import torch

# Einen vortrainierten Vision Transformer (ViT) laden
model_name = "google/vit-base-patch16-224-in21k"
feature_extractor = ViTFeatureExtractor.from_pretrained(model_name)
model = ViTModel.from_pretrained(model_name)

# Ein Bild laden
image = Image.open("example.jpg")  # Ersetzen Sie dies mit Ihrem Bildpfad

# Das Bild vorverarbeiten (in Patches umwandeln)
inputs = feature_extractor(images=image, return_tensors="pt")

# Features extrahieren (CLS-Token oder Patch-Embeddings)
with torch.no_grad():
    outputs = model(**inputs)

# Den Feature-Vektor erhalten (CLS-Token)
features = outputs.last_hidden_state[:, 0, :]  # Form: [1, 768]

print("Feature-Vektor-Form:", features.shape)  # z.B. torch.Size([1, 768])
```
**Erkl√§rung**:
1. **ViT** unterteilt das Bild in **16x16 Patches** und verarbeitet sie wie Tokens in der NLP.
2. Der `CLS-Token` (erster Token) repr√§sentiert den Feature-Vektor des gesamten Bildes.
3. Die Ausgabe ist ein **768-dimensionaler Vektor** (f√ºr `vit-base`).
