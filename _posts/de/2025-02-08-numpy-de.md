### Routinen zur Array-Erstellung

*   `np.empty`: Gibt ein neues Array mit gegebener Form und Typ zurück, ohne Einträge zu initialisieren.
*   `np.empty_like`: Gibt ein neues Array mit derselben Form und demselben Typ wie ein gegebenes Array zurück.
*   `np.eye`: Gibt ein 2-D-Array mit Einsen auf der Diagonalen und Nullen an anderer Stelle zurück.
*   `np.identity`: Gibt das Identitätsarray zurück.
*   `np.ones`: Gibt ein neues Array mit gegebener Form und Typ zurück, gefüllt mit Einsen.
*   `np.ones_like`: Gibt ein Array aus Einsen mit derselben Form und demselben Typ wie ein gegebenes Array zurück.
*   `np.zeros`: Gibt ein neues Array mit gegebener Form und Typ zurück, gefüllt mit Nullen.
*   `np.zeros_like`: Gibt ein Array aus Nullen mit derselben Form und demselben Typ wie ein gegebenes Array zurück.
*   `np.full`: Gibt ein neues Array mit gegebener Form und Typ zurück, gefüllt mit fill\_value.
*   `np.full_like`: Gibt ein vollständiges Array mit derselben Form und demselben Typ wie ein gegebenes Array zurück.
*   `np.array`: Erstellt ein Array.
*   `np.asarray`: Konvertiert die Eingabe in ein Array.
*   `np.asanyarray`: Konvertiert die Eingabe in ein ndarray, lässt aber ndarray-Unterklassen unverändert.
*   `np.ascontiguousarray`: Gibt ein zusammenhängendes Array im Speicher zurück (C-Reihenfolge).
*   `np.asfortranarray`: Gibt ein zusammenhängendes Array im Speicher zurück (Fortran-Reihenfolge).
*   `np.copy`: Gibt eine Array-Kopie des gegebenen Objekts zurück.
*   `np.frombuffer`: Interpretiert einen Puffer als ein 1-dimensionales Array.
*   `np.from_dlpack`: Erstellt ein NumPy-Array aus einem PyCapsule-Objekt (DLPack).
*   `np.fromfile`: Konstruiert ein Array aus Daten in einer Text- oder Binärdatei.
*   `np.fromfunction`: Konstruiert ein Array durch Ausführung einer Funktion über jede Koordinate.
*   `np.fromiter`: Erstellt ein neues 1-dimensionales Array aus einem iterierbaren Objekt.
*   `np.fromstring`: Ein neues 1-D-Array, initialisiert aus Textdaten in einem String.
*   `np.loadtxt`: Lädt Daten aus einer Textdatei.
*   `np.genfromtxt`: Lädt Daten aus einer Textdatei, wobei fehlende Werte wie angegeben behandelt werden.
*   `np.savetxt`: Speichert ein Array in einer Textdatei.
*   `np.save`: Speichert ein Array in einer Binärdatei im NumPy-`.npy`-Format.
*   `np.savez`: Speichert mehrere Arrays in einer einzelnen Datei im unkomprimierten `.npz`-Format.
*   `np.savez_compressed`: Speichert mehrere Arrays in einer einzelnen Datei im komprimierten `.npz`-Format.

### Routinen zur Array-Manipulation

*   `np.reshape`: Verleiht einem Array eine neue Form, ohne dessen Daten zu ändern.
*   `np.ravel`: Gibt ein zusammenhängendes, abgeflachtes Array zurück.
*   `np.transpose`: Vertauscht die Dimensionen eines Arrays.
*   `np.swapaxes`: Tauscht zwei Achsen eines Arrays aus.
*   `np.moveaxis`: Verschiebt Achsen eines Arrays an neue Positionen.
*   `np.broadcast_to`: Sendet ein Array auf eine neue Form.
*   `np.broadcast_arrays`: Sendet beliebig viele Arrays gegeneinander.
*   `np.expand_dims`: Erweitert die Form eines Arrays.
*   `np.squeeze`: Entfernt eindimensionale Einträge aus der Form eines Arrays.
*   `np.concatenate`: Verbindet eine Sequenz von Arrays entlang einer vorhandenen Achse.
*   `np.stack`: Verbindet eine Sequenz von Arrays entlang einer neuen Achse.
*   `np.hstack`: Stapelt Arrays sequenziell horizontal (spaltenweise).
*   `np.vstack`: Stapelt Arrays sequenziell vertikal (zeilenweise).
*   `np.dstack`: Stapelt Arrays sequenziell in der Tiefe (entlang der dritten Dimension).
*   `np.column_stack`: Stapelt 1-D-Arrays als Spalten in ein 2-D-Array.
*   `np.row_stack`: Stapelt Arrays sequenziell vertikal (zeilenweise).
*   `np.split`: Teilt ein Array in mehrere Teil-Arrays.
*   `np.array_split`: Teilt ein Array in mehrere Teil-Arrays.
*   `np.hsplit`: Teilt ein Array horizontal in mehrere Teil-Arrays (spaltenweise).
*   `np.vsplit`: Teilt ein Array vertikal in mehrere Teil-Arrays (zeilenweise).
*   `np.dsplit`: Teilt ein Array entlang der 3. Achse in mehrere Teil-Arrays (tiefenweise).
*   `np.tile`: Konstruiert ein Array durch Wiederholung von A gemäß der Anzahl in reps.
*   `np.repeat`: Wiederholt Elemente eines Arrays.
*   `np.delete`: Gibt ein neues Array mit gelöschten Teil-Arrays entlang einer Achse zurück.
*   `np.insert`: Fügt Werte entlang der gegebenen Achse vor den gegebenen Indizes ein.
*   `np.append`: Hängt Werte an das Ende eines Arrays an.
*   `np.resize`: Gibt ein neues Array mit der angegebenen Form zurück.
*   `np.trim_zeros`: Schneidet führende und/oder nachfolgende Nullen von einem 1-D-Array oder einer Sequenz ab.
*   `np.unique`: Findet die eindeutigen Elemente eines Arrays.

### Routinen zur Array-Indizierung

*   `np.take`: Nimmt Elemente aus einem Array entlang einer Achse.
*   `np.choose`: Konstruiert ein Array aus einem Index-Array und einer Menge von Arrays, aus denen ausgewählt wird.
*   `np.searchsorted`: Findet Indizes, an denen Elemente eingefügt werden sollten, um die Reihenfolge beizubehalten.
*   `np.count_nonzero`: Zählt die Anzahl der Nicht-Null-Werte im Array ``a``.
*   `np.nonzero`: Gibt die Indizes der Elemente zurück, die nicht Null sind.
*   `np.where`: Gibt Elemente zurück, die abhängig von der `condition` aus `x` oder `y` ausgewählt werden.
*   `np.indices`: Gibt ein Array zurück, das die Indizes eines Gitters repräsentiert.
*   `np.ix_`: Konstruiert ein offenes Gitter aus mehreren Sequenzen.
*   `np.diag_indices`: Gibt die Indizes zurück, um auf die Hauptdiagonale eines Arrays zuzugreifen.
*   `np.diag_indices_from`: Gibt die Indizes zurück, um auf die Hauptdiagonale eines n-dimensionalen Arrays zuzugreifen.
*   `np.mask_indices`: Gibt die Indizes zurück, um auf (n_arrays, n_elements) mit einer Maske zuzugreifen.
*   `np.tril_indices`: Gibt die Indizes für die untere Dreiecksmatrix eines (n, n)-Arrays zurück.
*   `np.tril_indices_from`: Gibt die Indizes für die untere Dreiecksmatrix von arr zurück.
*   `np.triu_indices`: Gibt die Indizes für die obere Dreiecksmatrix eines (n, n)-Arrays zurück.
*   `np.triu_indices_from`: Gibt die Indizes für die obere Dreiecksmatrix von arr zurück.

### Routinen für Array-Typen

*   `np.dtype`: Erstellt ein Daten-Typ-Objekt.
*   `np.can_cast`: Gibt True zurück, wenn eine sichere Umwandlung zwischen Datentypen erfolgen kann.
*   `np.result_type`: Gibt den Typ zurück, der sich aus der Anwendung der NumPy-Typ-Promotion-Regeln auf die Argumente ergibt.
*   `np.min_scalar_type`: Gibt den Datentyp mit der kleinsten Größe und der kleinsten Skalarart zurück, in den ein Array sicher umgewandelt werden kann.
*   `np.promote_types`: Gibt den Datentyp mit der kleinsten Größe und der kleinsten Skalarart zurück, der sowohl Typ1 als auch Typ2 halten kann.
*   `np.common_type`: Gibt einen Skalartyp zurück, der allen Eingabe-Arrays gemeinsam ist.
*   `np.obj2sctype`: Gibt den Skalar-Datentyp oder das NumPy-Äquivalent des Python-Typs eines Objekts zurück.

### Routinen für Array-Mengen

*   `np.unique`: Findet die eindeutigen Elemente eines Arrays.
*   `np.intersect1d`: Findet die Schnittmenge zweier Arrays.
*   `np.union1d`: Findet die Vereinigungsmenge zweier Arrays.

*   `np.setxor1d`: Findet die symmetrische Differenz zweier Arrays.
*   `np.in1d`: Testet, ob jedes Element eines Arrays auch in einem zweiten Array vorhanden ist.

*   `np.argmax`: Gibt die Indizes der Maximalwerte entlang einer Achse zurück.
*   `np.argmin`: Gibt die Indizes der Minimalwerte entlang einer Achse zurück.
*   `np.max`: Gibt das Maximum entlang einer gegebenen Achse zurück.
*   `np.min`: Gibt das Minimum entlang einer gegebenen Achse zurück.
*   `np.maximum`: Elementweises Maximum von Array-Elementen.
*   `np.minimum`: Elementweises Minimum von Array-Elementen.
*   `np.nanargmax`: Gibt die Indizes der Maximalwerte in der angegebenen Achse zurück, wobei NaNs ignoriert werden.
*   `np.nanargmin`: Gibt die Indizes der Minimalwerte in der angegebenen Achse zurück, wobei NaNs ignoriert werden.
*   `np.nanmax`: Gibt das Maximum eines Arrays oder das Maximum entlang einer Achse zurück, wobei alle NaNs ignoriert werden.
*   `np.nanmin`: Gibt das Minimum eines Arrays oder das Minimum entlang einer Achse zurück, wobei alle NaNs ignoriert werden.
*   `np.count_nonzero`: Zählt die Anzahl der Nicht-Null-Werte im Array ``a``.
*   `np.nonzero`: Gibt die Indizes der Elemente zurück, die nicht Null sind.
*   `np.where`: Gibt Elemente zurück, die abhängig von der `condition` aus `x` oder `y` ausgewählt werden.

### Array-String-Operationen

*   `np.char.add`: Gibt die elementweise String-Verkettung für zwei Arrays von str oder unicode zurück.
*   `np.char.multiply`: Gibt (a * i) zurück, also das String-Vielfache von i Kopien von a, für jedes Element.
*   `np.char.mod`: Gibt (a % i) zurück, also die String-Formatierung vor Python 2.6 (Interpolation), für jedes Element in a.
*   `np.char.capitalize`: Gibt eine Kopie von `a` zurück, bei der nur das erste Zeichen jedes Elements großgeschrieben ist.
*   `np.char.lower`: Gibt ein Array mit den in Kleinbuchstaben umgewandelten Elementen zurück.
*   `np.char.upper`: Gibt ein Array mit den in Großbuchstaben umgewandelten Elementen zurück.
*   `np.char.title`: Gibt für jedes Element in `a` eine Version des Strings zurück, in der Wörter mit Großbuchstaben beginnen und alle übrigen Zeichen Kleinbuchstaben sind.
*   `np.char.split`: Gibt für jedes Element in `a` eine Liste der Wörter im String zurück, wobei sep als Trennzeichen-String verwendet wird.
*   `np.char.splitlines`: Gibt für jedes Element in `a` eine Liste der Zeilen im Element zurück, unterbrochen an Zeilengrenzen.
*   `np.char.strip`: Gibt für jedes Element in `a` eine Kopie zurück, bei der die führenden und nachgestellten Zeichen entfernt wurden.
*   `np.char.lstrip`: Gibt für jedes Element in `a` eine Kopie zurück, bei der die führenden Zeichen entfernt wurden.
*   `np.char.rstrip`: Gibt für jedes Element in `a` eine Kopie zurück, bei der die nachgestellten Zeichen entfernt wurden.
*   `np.char.join`: Gibt einen String zurück, der die Verkettung der Strings in der Sequenz seq ist.
*   `np.char.replace`: Gibt für jedes Element in `a` eine Kopie des Strings zurück, bei der alle Vorkommen des Teilstrings old durch new ersetzt wurden.
*   `np.char.encode`: Ruft `str.encode` elementweise auf.
*   `np.char.decode`: Ruft `str.decode` elementweise auf.

*   `np.pi`: Mathematische Konstante, das Verhältnis des Umfangs eines Kreises zu seinem Durchmesser.
*   `np.e`: Mathematische Konstante, Basis der natürlichen Logarithmen.

### Array-Logikfunktionen

*   `np.all`: Testet, ob alle Array-Elemente entlang einer gegebenen Achse zu True ausgewertet werden.
*   `np.any`: Testet, ob irgendein Array-Element entlang einer gegebenen Achse zu True ausgewertet wird.
*   `np.array_equal`: True, wenn zwei Arrays die gleiche Form und Elemente haben, sonst False.
*   `np.array_equiv`: Gibt True zurück, wenn die Eingabe-Arrays formkonsistent sind und alle Elemente gleich sind.
*   `np.greater`: Gibt den Wahrheitswert von (x1 > x2) elementweise zurück.
*   `np.greater_equal`: Gibt den Wahrheitswert von (x1 >= x2) elementweise zurück.
*   `np.less`: Gibt den Wahrheitswert von (x1 < x2) elementweise zurück.
*   `np.less_equal`: Gibt den Wahrheitswert von (x1 <= x2) elementweise zurück.
*   `np.equal`: Gibt (x1 == x2) elementweise zurück.
*   `np.not_equal`: Gibt (x1 != x2) elementweise zurück.
*   `np.logical_and`: Berechnet den Wahrheitswert von x1 UND x2 elementweise.
*   `np.logical_or`: Berechnet den Wahrheitswert von x1 ODER x2 elementweise.
*   `np.logical_not`: Berechnet den Wahrheitswert von NICHT x elementweise.
*   `np.logical_xor`: Berechnet den Wahrheitswert von x1 XOR x2 elementweise.

### Array-Vergleich

*   `np.isclose`: Gibt ein boolesches Array zurück, in dem zwei Arrays innerhalb einer Toleranz gleich sind.
*   `np.allclose`: Gibt True zurück, wenn zwei Arrays elementweise innerhalb einer Toleranz gleich sind.

### Zufällige Array-Stichproben

*   `np.random.rand`: Erstellt ein Array der gegebenen Form und füllt es mit Zufallsstichproben aus einer Gleichverteilung über ``[0, 1)``.
*   `np.random.randn`: Gibt eine Stichprobe (oder Stichproben) aus der "Standardnormalverteilung" zurück.
*   `np.random.randint`: Gibt zufällige Ganzzahlen von `low` (einschließlich) bis `high` (ausschließlich) zurück.
*   `np.random.random`: Gibt zufällige Floats im halboffenen Intervall [0.0, 1.0) zurück.
*   `np.random.choice`: Erzeugt eine Zufallsstichprobe aus einem gegebenen 1-D-Array.
*   `np.random.seed`: Setzt den Seed für den Generator.

### Array-Eingabe und -Ausgabe

*   `np.save`: Speichert ein Array in einer Binärdatei im NumPy-``.npy``-Format.

*   `np.savetxt`: Speichert ein Array in einer Textdatei.
*   `np.loadtxt`: Lädt Daten aus einer Textdatei.
