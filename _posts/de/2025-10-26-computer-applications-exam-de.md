### Verständnis der Prüfung
Der Kurs 11325: Computer and Application Course Experiment (I) ist eine praktische (am Computer durchgeführte) Bewertung für das Hauptfach 610201: Computer Application Technology (Fachschulebene) an der Guangdong University of Foreign Studies (GDUFS). Es handelt sich um eine umfassende praktische Prüfung im Wert von 10 Credits, die Ihre Fähigkeit testen soll, Konzepte aus verwandten Theoriekursen durch praxisnahe computerbasierte Aufgaben anzuwenden. Sie müssen die folgenden fünf Voraussetzungsprüfungen bestehen, bevor Sie sich anmelden können:

- 电子技术基础(三) (Electronics Fundamentals III)
- 高级语言程序设计(一) (Advanced Language Programming I – typischerweise C-Sprache)
- 计算机应用技术 (Computer Application Technology)
- 微型计算机及接口技术 (Microcomputer and Interface Technology)
- 数据库及其应用 (Database and Its Application)

Die Prüfung wird wie folgt benotet (von 10 Credits):
- Electronics Fundamentals III: 2 Credits
- Advanced Language Programming I: 2 Credits
- Microcomputer and Interface Technology: 2 Credits
- Computer Application Technology: 3 Credits
- Database and Its Application (Experimentteil): 1 Credit

Sie wird an Computern durchgeführt und kombiniert praktische Aufgaben (z. B. Programmieren, Simulationen, Datenbankabfragen) mit einigen schriftlichen Elementen. Der Fokus liegt auf drei Ebenen: Auswendiglernen (grundlegende Konzepte), Verständnis (Prinzipien) und Anwendung (eigenständige Umsetzung). Erwarten Sie Aufgaben wie das Schreiben einfacher Programme, das Entwerfen grundlegender Schaltkreise, das Abfragen von Datenbanken und das Verbinden von Hardware/Software.

### Wichtige behandelte Themen
Basierend auf dem Bewertungsrahmen stammt die Prüfung aus Experimenten in den Voraussetzungsfächern. Hier eine Aufschlüsselung:

1. **Electronics Fundamentals III (2 Credits)**:
   - Grundlegende Schaltungskonzepte, Gesetze (z. B. Ohmsches Gesetz, Kirchhoffsche Gesetze) und Analysemethoden.
   - Verstärkerschaltungen, Operationsverstärker (Op-Amps) und ihre Anwendungen.
   - Digitale Schaltungen: Logikgatter, Flip-Flops und einfache kombinatorische/sequentielle Logik.
   - **Geprüfte Fähigkeiten**: Grundlegende Schaltungen simulieren oder zeichnen; Werte mit Tools wie Multisim oder ähnlicher Software berechnen.

2. **Advanced Language Programming I (2 Credits)**:
   - C-Sprache Grundlagen: Variablen, Kontrollstrukturen (Schleifen, Bedingungen), Funktionen, Arrays, Pointer.
   - Datei-I/O, String-Verarbeitung und modulare Programmierung.
   - **Geprüfte Fähigkeiten**: Einfache C-Programme schreiben und debuggen (z. B. Ausgabe von Schaltjahren von 1000–2020 oder grundlegende Datenverarbeitung). Verwendung einer IDE wie Turbo C oder Dev-C++.

3. **Microcomputer and Interface Technology (2 Credits)**:
   - Mikroprozessor-Grundlagen (z. B. 8086/8088-Architektur), Assembler-Sprache-Befehle.
   - I/O-Schnittstellen: Ports, Interrupts, DMA.
   - **Geprüfte Fähigkeiten**: Einfacher Assembler-Code für Datentransfer oder Steuerung; Simulation von Hardwareschnittstellen mit Tools wie Proteus.

4. **Computer Application Technology (3 Credits)**:
   - Office-Software: Textverarbeitung, Tabellenkalkulation (Excel), Präsentationen (PowerPoint).
   - Grundlegende Multimedia- und Internetanwendungen.
   - **Geprüfte Fähigkeiten**: Dokumente mit Formeln/Diagrammen erstellen, Aufgaben über Makros automatisieren oder einfache Webformulare erstellen.

5. **Database and Its Application (1 Credit)**:
   - Konzepte relationaler Datenbanken (Tabellen, Abfragen, Beziehungen).
   - SQL-Grundlagen oder Access-Operationen: Datenbanken erstellen, Abfragen ausführen, Berichte generieren.
   - **Geprüfte Fähigkeiten**: Einfache Datenbank entwerfen (z. B. Studentenakten), Daten einfügen/aktualisieren und Ergebnisse abfragen.

### Vorbereitung
Die Vorbereitung sollte praktische Übung betonen, da es sich um eine praktische Prüfung handelt. Planen Sie bei einem Start von den Grundlagen aus 2–3 Monate konzentriertes Lernen ein.

1. **Voraussetzungen wiederholen**:
   - Gehen Sie die Theorieunterlagen der fünf Kurse noch einmal durch. Konzentrieren Sie sich auf "Experiment"-Kapitel in den Lehrbüchern (z. B. C-Programmierübungen, Schaltungssimulationen).
   - Verwenden Sie offizielle GDUFS- oder Guangdong-Eigenstudium-Lehrbücher:
     - C Programming: 《C语言程序设计》 (C Language Programming)
     - Database: 《数据库及其应用》 (Database and Its Application)
     - Electronics: 《电子技术基础》 (Electronics Fundamentals)

2. **Praktische Fähigkeiten aufbauen**:
   - **Programmieren**: Installieren Sie kostenlose Tools wie Code::Blocks für C. Üben Sie 50–100 Beispielprogramme (z. B. Sortieralgorithmen, Dateibehandlung). Seiten wie LeetCode oder GeeksforGeeks bieten C-Herausforderungen für Anfänger.
   - **Elektronik/Mikrocomputer**: Verwenden Sie Simulationssoftware (Tinkercad, Logisim für digitale Schaltungen; Emu8086 für Assembler). Üben Sie das Verdrahten virtueller Steckbretter und das Ausführen einfacher E/A-Programme.
   - **Datenbank**: Laden Sie Microsoft Access herunter oder verwenden Sie SQLite. Bauen Sie Beispieldatenbanken (z. B. Inventarsystem) und üben Sie Abfragen.
   - **Anwendungen**: Beherrschen Sie die MS Office Suite. Erstellen Sie komplexe Excel-Tabellen mit Pivot-Tabellen und VBA-Makros.

3. **Prüfungsähnliche Aufgaben üben**:
   - Nehmen Sie sich Zeit: Simulieren Sie 2–3 stündige Sitzungen am Computer.
   - Beispielaufgaben (aus Rahmenplänen und vergangenen Beispielen):
     - Schreiben Sie ein C-Programm, um Schaltjahre in einem Bereich zu berechnen und auszugeben.
     - Entwerfen Sie eine Datenbanktabelle für Studenten, fügen Sie Datensätze hinzu und fragen Sie Durchschnittswerte ab.
     - Simulieren Sie eine grundlegende Operationsverstärkerschaltung und berechnen Sie die Ausgangsspannung.
   - Streben Sie nach Genauigkeit und Effizienz – die Prüfungen bestrafen Fehler bei der Codekompilierung oder Abfrageergebnisse.

4. **Lernressourcen**:
   - **Offizielle Rahmenpläne**: Laden Sie diese von den Seiten der GDUFS oder der Guangdong Finance University herunter (sie haben ähnliche Standards).
   - **Probeprüfungen**: Suchen Sie auf chinesischen Plattformen wie Zhihu oder Bilibili nach "11325 自考 模拟题" für Video-Anleitungen.
   - **Online-Kurse**: Kostenlose MOOCs auf Coursera/edX für C-Programmierung und SQL; YouTube-Tutorials für Elektroniksimulationen.
   - **Bücher**: 《计算机及应用课程实验指导》 (Computer and Application Experiment Guide) für schrittweise Anleitungen.

5. **Anmeldung und Logistik**:
   - Melden Sie sich über das GDUFS-Portal (jxjy.gdufs.edu.cn) während der Anmeldefenster an (z. B. Juni für das zweite Halbjahr 2025).
   - Gebühr: Etwa 120–150 RMB. Bringen Sie einen Ausweis und den Nachweis der bestandenen Voraussetzungen mit.
   - Prüfungstermin: Typischerweise Ende September für das zweite Halbjahr; prüfen Sie die Benachrichtigungen auf Terminkonflikte (z. B. mit nationalen Computertests).

6. **Tipps zum Erfolg**:
   - Täglich üben: 1 Stunde Programmieren + 1 Stunde Simulationen.
   - Systematisch debuggen: Testen Sie Code in kleinen Teilen.
   - Wenn Sie nicht weiterkommen, schließen Sie sich Foren für Eigenstudium an (z. B. Guangdong Zikao-Gruppen auf WeChat/QQ).
   - Erreichen Sie 60+ Punkte zum Bestehen; konzentrieren Sie sich auf die Anwendungsebene für bessere Noten.

Mit konsequenter Übung werden Sie das nötige Selbstvertrauen für das Format der Prüfung am Computer aufbauen. Viel Glück!
