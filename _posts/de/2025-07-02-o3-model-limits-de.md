Ja, ChatGPT Plus-Benutzer **können** das O3-Modell verwenden.

OpenAI stellt die Modelle O3, O4-mini und O4-mini-high für ChatGPT Plus-, Team- und Enterprise-Konten bereit.

Es gelten jedoch Nutzungslimits:

* **O3:** 100 Nachrichten pro Woche.
* **O4-mini:** 300 Nachrichten pro Tag.
* **O4-mini-high:** 100 Nachrichten pro Tag.

Sie können das O3-Modell über die Modellauswahl oben auf der ChatGPT-Seite auswählen. Wenn Sie Ihr Nutzungslimit für ein bestimmtes Modell erreichen, erscheint ein Pop-up, und Sie können dieses Modell nicht mehr auswählen, bis Ihr Nutzungslimit zurückgesetzt wird.

Es ist erwähnenswert, dass OpenAI auch O3-pro veröffentlicht hat, ein noch leistungsfähigeres Modell, das für hochzuverlässige Antworten auf komplexe Aufgaben entwickelt wurde. Seit kurzem ist O3-pro allgemein für ChatGPT **Pro**- und **Team**-Benutzer sowie für Entwickler über die API verfügbar, jedoch typischerweise nicht für standardmäßige ChatGPT Plus-Benutzer. Allerdings hat OpenAI die Kosten für O3 erheblich gesenkt, was bedeutet, dass Plus-Benutzer jetzt bessere Ratenlimits erhalten als zuvor.
