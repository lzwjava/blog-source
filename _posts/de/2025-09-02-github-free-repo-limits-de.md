Ein kostenloser GitHub-Benutzer kann **unbegrenzt private Repositorys** erstellen, jedoch gibt es eine Einschränkung bei **Mitarbeitern**.

* Sie können so viele private Repositorys haben, wie Sie möchten.
* Jedes private Repository kann bis zu **3 Mitarbeiter** haben (Personen, die Sie zur Zusammenarbeit einladen).

Wenn Sie mehr Mitarbeiter benötigen, müssten Sie auf einen kostenpflichtigen Plan upgraden (GitHub Pro, Team oder Enterprise).

Möchten Sie auch, dass ich erkläre, wie sich dies im Vergleich zur früheren GitHub-Richtlinie geändert hat (vor 2019, als kostenlose Benutzer überhaupt keine privaten Repositorys haben konnten)?
