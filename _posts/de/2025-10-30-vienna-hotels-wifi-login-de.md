Das WiFi in Vienna Hotels (Teil der Jinjiang International Gruppe) nutzt ein standardmäßiges Captive Portal System zur Gästauthentifizierung, wie es in chinesischen Hotels üblich ist. Die Gateway-IP (172.16.16.1) ist die interne Router-Adresse, und der Zugriff darauf (oder auf jede nicht-HTTPS-Website) leitet Sie zur Login-Seite unter portal.jinjianghotels.com.cn weiter. So funktioniert der SMS-basierte Login typischerweise – er ist unkompliziert und sicher für grundlegendes Browsen, auch wenn sich einige Nutzer um Datenschutz sorgen (es wird vom Hotel verwaltet, also verwenden Sie ein VPN, wenn Sie besorgt sind).

### Schritt-für-Schritt-Anleitung:
1. **Verbinden Sie sich mit dem WiFi-Netzwerk**: Scannen Sie auf Ihrem Gerät (Telefon, Laptop, etc.) nach dem WiFi-SSID des Hotels (oft etwas wie "Vienna_Free_WiFi" oder "Jinjiang_WiFi") und verbinden Sie sich. Normalerweise ist kein Passwort erforderlich.

2. **Das Portal aufrufen**: Öffnen Sie einen Webbrowser und versuchen Sie, eine Website zu laden (z.B. google.com oder baidu.com). Sie werden automatisch zur Jinjiang-Portalseite weitergeleitet (es kann einige Sekunden dauern; falls nicht, geben Sie manuell `http://172.16.16.1` oder `http://portal.jinjianghotels.com.cn` in die Adressleiste ein). Die Seite ist auf Chinesisch, aber Sie können Google Translate oder ähnliches zur Hilfe nehmen.

3. **Authentifizierungsmethode wählen**: Das Portal bietet einige Optionen, wie:
   - SMS-Verifizierung (短信验证 – einfachste Methode für die meisten Gäste).
   - Social Login (z.B. Weibo, QQ).
   - Manchmal eine Ein-Klick-Option, wenn Sie ein registriertes Jinjiang-Mitglied sind.

4. **Details zur SMS-Anmeldung**:
   - Wählen Sie die SMS-Option.
   - Geben Sie Ihre Handynummer ein (chinesische Nummern beginnen mit +86; internationale Nummern wie +1 für USA sollten funktionieren, können aber Roaming-Gebühren für die SMS verursachen).
   - Klicken Sie auf "Code senden" (发送验证码). Sie erhalten einen 4-6-stelligen Code per SMS vom System des Hotels (normalerweise sofort, von einer Kurzwahl wie 106XXXX).
   - Geben Sie den Code im Portal ein.
   - Klicken Sie auf "Bestätigen" oder "Verbinden" (立即连接). Wenn der Code gültig ist, erhalten Sie vollen Internetzugang für 24 Stunden (oder bis zum Check-out).

### Tipps & Problembehebung:
- **Keine Weiterleitung?** Vergessen Sie das Netzwerk, verbinden Sie sich erneut und leeren Sie den Browser-Cache/die Cookies. Auf dem Handy schalten Sie kurz den Flugmodus ein und aus.
- **Code kommt nicht an?** Überprüfen Sie das Signal (Hotel-WiFi ist vom Mobilfunknetz getrennt), warten Sie 60 Sekunden für einen erneuten Versuch, oder bitten Sie die Rezeption um Hilfe – diese hat oft einen gemeinsamen Code.
- **Probleme mit ausländischen Nummern?** Es funktioniert mit den meisten internationalen Anbietern, aber falls blockiert, versuchen Sie Social Login oder fragen Sie das Personal nach einem Gutschein für das Zimmer.
- **Geschwindigkeit/Sicherheit**: Erwarten Sie akzeptable Geschwindigkeiten nach dem Login, aber es ist ein öffentliches Netz – vermeiden Sie sensible Logins ohne VPN. Es gibt keine größeren Berichte über Betrug mit diesem Portal.
