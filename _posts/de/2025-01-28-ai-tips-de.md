* Kopieren Sie Beispielcode in Cursor und verwenden Sie Kommentare, um Kontext zu geben.

* Im Hong Kong Gebiet ist es akzeptabel, DeepSeek oder Mistral-Dienste zu verwenden, während es nicht akzeptabel ist, ChatGPT, Claude oder Gemini zu verwenden.

* Die API-Richtlinie einer Region ist ungefähr dieselbe wie ihre App-Richtlinie.

* Verwenden Sie Cursor anstatt Visual Studio Code.

* Es gibt immer noch Fälle, in denen Sie Visual Studio Code verwenden müssen, wie z.B. für Szenarien des Git Merge Editors, bei denen ich immer noch `git config --global core.editor "code --wait"` verwende.

* Ab dem Tag der Veröffentlichung von Deepseek V3 müssen wir keine AI-Tools mehr abonnieren.

* Verwenden Sie Gemini oder Grok, um Festival-Feier-Bilder mit Prompts wie "Erstelle ein glückliches Lunar Snake New Year-Bild mit eingeschlossenen Textnamen" zu erstellen.

* In einigen Fällen, selbst wenn Sie Originaltext an AI-Modelle weitergeben, um eine Tabelle zu erstellen, kann es einige Stellen im Ausgang geben, die sich vom Eingang unterscheiden. Zum Beispiel, wenn Sie das Deepseek V3-Modell in Cursor verwenden, um eine Tabelle der Pip-Liste zu erstellen, kann es Versionen wie `1.极狐0` enthalten. Hier bezieht sich `极狐` auf die chinesische GitLab-Plattform.

* Wenn Sie die Deepseek- oder Mistral-API verwenden, um Übersetzungen mit Prompts wie `Sie sind ein professioneller Übersetzer. Sie übersetzen eine Markdown-Datei für einen Jekyll-Blogpost von Englisch nach Chinesisch. {text}`, können falsche Übersetzungen auftreten. Neben dem Text, den Sie bereitstellen, enthält die Ausgabe oft eine übermäßige Übersetzung.

* Obwohl KI-Modelle in Cursor manchmal teilweise korrekten Text liefern, können wir sie akzeptieren, da wir Folgeanweisungen hinzufügen können, die die KI-Modelle veranlassen, die korrekten Teile zu regenerieren.

* Meiden Sie es, übermäßige Kontexte zu großen Sprachmodellen bereitzustellen, wenn es unwahrscheinlich ist, dass sie nützlich sind. Zum Beispiel, wenn Sie dialogische Dialogzeilen erstellen, vermeiden Sie es, 100 Punkte zu einem Thema bereitzustellen. Große Sprachmodelle enthalten bereits riesige Datenmengen.
