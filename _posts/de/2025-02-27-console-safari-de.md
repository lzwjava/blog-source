Um die Konsole in Safari anzuzeigen, befolgen Sie diese Schritte:

1. Öffnen Sie Safari auf Ihrem Mac.
2. Gehen Sie zur Menüleiste oben und klicken Sie auf **Safari** > **Einstellungen** (oder **Einstellungen** wenn Sie eine ältere Version verwenden).
3. Klicken Sie auf die Registerkarte **Erweitert**.
4. Unten überprüfen Sie das Kästchen, das **Entwickler-Menü in der Menüleiste anzeigen** sagt.
5. Schließen Sie das Einstellungen-Fenster.
6. Jetzt sehen Sie in der Menüleiste eine neue Option **Entwickler**. Klicken Sie auf **Entwickler** > **JavaScript-Konsole anzeigen** (oder **Web-Inspektor anzeigen** je nach Version).

Alternativ können Sie die Tastenkombination **Option + Befehl + C** (⌥ + ⌘ + C) verwenden, nachdem das Entwickler-Menü aktiviert wurde, um die Konsole schnell zu öffnen.

Wenn Sie ein iPhone oder iPad verwenden, zeigt Safari standardmäßig keine Konsole an, aber Sie können sie aktivieren, indem Sie Ihr Gerät über USB mit einem Mac verbinden, Safari auf dem Mac öffnen und das Entwickler-Menü verwenden, um den Browser des iOS-Geräts zu untersuchen.
