### 冪等性定義

在數學和抽象代數中，**冪等**元素或運算是指對同一輸入多次應用該運算，結果與應用一次相同。形式上，對於運算 ⊕，若元素 \\( A \\) 滿足 \\( A \oplus A = A \\)，則該元素具有冪等性。

你的範例說明了兩種常見運算：
- \\( A \cdot A = A \\)：適用於**乘法**（或布林代數中的合取/AND 運算），重複運算不會改變數值
- \\( A + A = A \\)：適用於**加法**（或布林代數中的析取/OR 運算），這在標準算術中較不直觀（因 \\( A + A = 2A \\)），但在集合（聯集：\\( A \cup A = A \\)）或布林邏輯中成立

#### 如何驗證冪等性
要檢查元素 \\( A \\) 在運算 ⊕ 下是否具有冪等性：
1. 計算 \\( A \oplus A \\)
2. 若結果等於 \\( A \\)，則具有冪等性
3. 可重複進行多次應用驗證（例如 \\( A \oplus (A \oplus A) \\)），但冪等性意味著結果會立即穩定

**範例**：
- **布林代數**：AND (·) 與 OR (+) 皆為冪等運算
  - True · True = True
  - True + True = True
- **矩陣**：若矩陣 \\( M \\) 滿足 \\( M^2 = M \\) 則具冪等性（投影矩陣）
- **函數**：若函數 \\( f \\) 滿足 \\( f(f(x)) = f(x) \\) 則具冪等性（例如絕對值函數：\\( | |x| | = |x| \\)）

此性質在計算機科學（如 HTTP 方法中的 GET 具冪等性）與代數（冪等半環）中皆為關鍵概念。
