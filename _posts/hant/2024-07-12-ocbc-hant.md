在當今的數位時代，管理財務和存取雲端服務變得比以往更加簡單。如果您是持有中國護照和中國身份證的中國公民，開立OCBC Digital銀行帳戶並使用它註冊Google Cloud Platform是一個無縫的過程。以下是一個逐步指南，幫助您完成整個流程。

<div align="center"><img src="/assets/images/ocbc/ocbc1.jpg" width="300px" /><img/></div>

### 第一步：開立OCBC銀行帳戶

#### 要求：
1. 中國護照
2. 中國身份證

#### 程序：
1. 下載OCBC Digital應用程式：
   - 前往App Store或Google Play Store，下載OCBC Digital應用程式。

2. 註冊：
   - 打開應用程式，選擇註冊新帳戶的選項。您將被要求提供個人信息，包括中國護照和身份證的詳細資料。

3. 驗證：
   - OCBC將要求您驗證身份。這可能包括拍攝護照和身份證的照片，以及進行面部識別的自拍。

4. 選擇帳戶：
   - 在註冊過程中，選擇開立全球儲蓄帳戶和對帳儲蓄帳戶。這些帳戶將幫助您有效管理財務。

5. 開立360帳戶：
   - 此外，選擇開立360帳戶。此帳戶根據您的消費和儲蓄活動提供各種福利和獎勵。當您開立360帳戶時，您還將收到一張與此帳戶相關的借記卡。

6. 完成流程：
   - 按照屏幕上的指示完成帳戶開立流程。根據您的情況，您可能需要前往分行進行最終驗證。

### 第二步：激活您的借記卡

1. 接收並激活您的借記卡：
   - 一旦您的360帳戶設置完成，您將收到一張OCBC借記卡。您可以通過OCBC Digital應用程式或在線訪問OCBC網站來激活它。

### 第三步：註冊Google Cloud Platform

1. 訪問Google Cloud Platform：
   - 前往[Google Cloud Platform網站](https://cloud.google.com/)。

2. 註冊：
   - 點擊“免費開始使用”按鈕。如果您還沒有Google帳戶，您將被提示創建一個。

3. 帳單信息：
   - 在註冊過程中，您需要提供帳單信息。在提示時輸入您的OCBC借記卡詳細資料。

4. 驗證：
   - Google將通過收取一小筆臨時金額來驗證您的借記卡，該金額將被退還。這是為了確保該卡是有效的並且有足夠的資金。

5. 完成註冊：
   - 按照剩餘的步驟完成您的Google Cloud Platform註冊。您將可以存取Google提供的各種雲端服務。

### 結論

使用中國護照和身份證開立OCBC Digital銀行帳戶是一個簡單的過程。通過設置您的全球儲蓄帳戶、對帳儲蓄帳戶和360帳戶，並獲得借記卡，您可以輕鬆管理財務。此外，使用您的OCBC借記卡註冊Google Cloud Platform為存取強大的雲端服務提供了順暢的途徑。擁抱便利，從今天開始輕鬆管理您的財務和雲端服務。
