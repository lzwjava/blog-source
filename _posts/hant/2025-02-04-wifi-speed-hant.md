##舊式Modem問題

在我父母的家中，Modem相當舊，大概有十年左右。最初的網絡設置是：

Modem -> 3米無線 -> TP-Link AX3000（無線橋接模式）-> 2米、一道牆、無線 -> 筆記型電腦

這導致下載速度很慢，Speedtest測試結果僅為10 Mbps。

改進後的設置使用了有線連接：

Modem -> 2米線纜 -> TP-Link AX3000（有線橋接模式）-> 4米無線、一道牆 -> 筆記型電腦

這將下載速度提升至最高90 Mbps。


##新型號Modem效能

在我自己的家中，Modem是新的，TP-Link路由器在無線橋接模式下表現良好。網絡設置是：

Modem -> 4米無線 -> TP-Link AX3000（無線橋接模式）-> 2米無線 -> 筆記型電腦

網絡質素良好。


##疑難排解提示

並沒有單一的解決方案來改善Wi-Fi速度。一個好方法是用線纜測試網絡的每個部分以找出瓶頸。比較使用有線連接和Wi-Fi時的網速。另外，嘗試直接用線纜連接設備，看看是否能改善效能。
