DBeaver 是一款廣受歡迎的資料庫管理工具，支援多種資料庫類型。以下是使用 DBeaver 的基本指南：

### 1. **安裝步驟**
- 從官方網站下載 DBeaver
- 根據您的作業系統遵循安裝指示

### 2. **設定資料庫連線**
- **開啟 DBeaver**：啟動應用程式
- **建立新連線**：點擊「新增資料庫連線」按鈕
- **選擇資料庫**：選取要連線的資料庫類型（例如 MySQL、PostgreSQL、SQLite）
- **設定連線**：輸入必要的連線詳細資訊，包括主機位置、連接埠、資料庫名稱、使用者名稱和密碼
- **測試連線**：點擊「測試連線」確認設定正確
- **完成**：儲存連線設定

### 3. **介面導覽**
- **資料庫導覽器**：此面板顯示資料庫結構，包括資料表、檢視表和其他物件
- **SQL 編輯器**：用於編寫和執行 SQL 查詢
- **結果面板**：顯示查詢結果

### 4. **執行查詢**
- **開啟 SQL 編輯器**：點擊 SQL 編輯器按鈕或使用快捷鍵
- **編寫查詢**：在編輯器中輸入 SQL 查詢語句
- **執行查詢**：點擊執行按鈕或使用快捷鍵運行查詢
- **檢視結果**：查詢結果將顯示在結果面板中

### 5. **資料管理**
- **編輯資料**：可直接透過資料編輯器修改資料表內容
- **匯入/匯出資料**：支援多種格式的資料匯入與匯出功能

### 6. **進階功能**
- **ER 圖表**：視覺化呈現資料庫結構
- **資料傳輸**：在不同資料庫間遷移資料
- **外掛程式**：透過擴充套件增強各類資料庫與工具的功能

### 7. **儲存與分享連線**
- **儲存連線**：可儲存資料庫連線設定供日後使用
- **分享連線**：匯出連線設定以便與他人共享

### 8. **疑難排解**
- **檢查記錄檔**：若遇到問題可查閱記錄檔中的錯誤訊息
- **更新 DBeaver**：確保使用最新版本

DBeaver 作為多功能工具，能大幅提升您的資料庫管理效率。如有任何具體問題或需要進一步協助，歡迎隨時提出！
