要判斷給定集合中哪一個是 $\mathbb{R}^2$ 的子空間，我們需要檢查它們是否滿足子空間的三個條件：

1.  **零向量在子空間中。**
2.  **子空間在向量加法下封閉。** 如果 $\mathbf{u}$ 和 $\mathbf{v}$ 在子空間中，則 $\mathbf{u} + \mathbf{v}$ 也在子空間中。
3.  **子空間在純量乘法下封閉。** 如果 $\mathbf{u}$ 在子空間中且 $c$ 是任意純量，則 $c\mathbf{u}$ 也在子空間中。

讓我們檢查每個選項：

**a) 直線 y = x + 1**

1.  **零向量：** $\mathbb{R}^2$ 中的零向量是 $\begin{pmatrix} 0 \\ 0 \end{pmatrix}$。要使該向量位於直線上，它必須滿足方程式 $y = x + 1$。代入 $x=0$ 和 $y=0$，我們得到 $0 = 0 + 1$，即 $0 = 1$。這是錯誤的，因此零向量不在直線 $y = x + 1$ 上。

由於第一個條件不滿足，直線 $y = x + 1$ 不是 $\mathbb{R}^2$ 的子空間。

**b) 直線 y = 2x**

1.  **零向量：** 要使零向量 $\begin{pmatrix} 0 \\ 0 \end{pmatrix}$ 位於直線上，它必須滿足方程式 $y = 2x$。代入 $x=0$ 和 $y=0$，我們得到 $0 = 2(0)$，即 $0 = 0$。這是正確的，因此零向量在直線 $y = 2x$ 上。

2.  **向量加法封閉性：** 令 $\mathbf{u} = \begin{pmatrix} x_1 \\ y_1 \end{pmatrix}$ 和 $\mathbf{v} = \begin{pmatrix} x_2 \\ y_2 \end{pmatrix}$ 為直線 $y = 2x$ 上的兩個向量。這意味著 $y_1 = 2x_1$ 和 $y_2 = 2x_2$。
    它們的和是 $\mathbf{u} + \mathbf{v} = \begin{pmatrix} x_1 + x_2 \\ y_1 + y_2 \end{pmatrix}$。我們需要檢查該向量是否位於直線 $y = 2x$ 上。代入分量，我們得到 $y_1 + y_2 = 2(x_1 + x_2)$。由於 $y_1 = 2x_1$ 和 $y_2 = 2x_2$，我們有 $2x_1 + 2x_2 = 2x_1 + 2x_2$，這是正確的。因此，該直線在向量加法下封閉。

3.  **純量乘法封閉性：** 令 $\mathbf{u} = \begin{pmatrix} x \\ y \end{pmatrix}$ 為直線 $y = 2x$ 上的一個向量，因此 $y = 2x$。令 $c$ 為任意純量。則 $c\mathbf{u} = \begin{pmatrix} cx \\ cy \end{pmatrix}$。我們需要檢查該向量是否位於直線 $y = 2x$ 上。代入分量，我們得到 $cy = 2(cx)$。由於 $y = 2x$，我們有 $c(2x) = 2(cx)$，即 $2cx = 2cx$。這是正確的，因此該直線在純量乘法下封閉。

由於所有三個條件都滿足，直線 $y = 2x$ 是 $\mathbb{R}^2$ 的子空間。

**c) 單位圓**

$\mathbb{R}^2$ 中的單位圓由方程式 $x^2 + y^2 = 1$ 定義。

1.  **零向量：** 零向量是 $\begin{pmatrix} 0 \\ 0 \end{pmatrix}$。代入方程式，我們得到 $0^2 + 0^2 = 1$，即 $0 = 1$。這是錯誤的，因此零向量不在單位圓上。

由於第一個條件不滿足，單位圓不是 $\mathbb{R}^2$ 的子空間。

最終答案：最終答案是 $\boxed{直線 y = 2x}$

「零向量在子空間中」這個要求是向量空間的子集被視為**子空間**必須滿足的**三個基本條件**之一。讓我們分析*為什麼*這個條件是必要的。

子空間的概念是關於在較大的向量空間中識別出那些自身行為也像向量空間的較小集合。要表現得像向量空間，這些較小的集合必須繼承某些基本性質。任何向量空間最基礎的性質之一就是**零向量**（通常表示為 $\mathbf{0}$）的存在。這個向量具有關鍵的性質：當它與空間中的任何其他向量相加時，該向量保持不變：對於空間中的所有向量 $\mathbf{v}$，$\mathbf{v} + \mathbf{0} = \mathbf{v}$。

以下是為什麼零向量必須在子空間中的原因：

1.  **繼承向量空間結構：** 子空間旨在成為較大向量空間內的一個「迷你向量空間」。如果一個子集不包含零向量，它就缺少了滿足向量空間公理所必需的關鍵元素。沒有零向量，加法反元素（另一個重要的公理）的概念就會出現問題。

2.  **純量乘法下的封閉性：** 考慮向量空間 $V$ 的一個非空子集 $W$，該子集在純量乘法下封閉。如果 $\mathbf{w}$ 是 $W$ 中的任意向量，且 $W$ 是一個子空間，那麼對於任意純量 $c$，向量 $c\mathbf{w}$ 也必須在 $W$ 中。如果我們選擇純量 $c = 0$，則 $0\mathbf{w} = \mathbf{0}$（這是向量空間的性質）。因此，如果一個子空間在純量乘法下封閉並且至少包含一個向量，它*必須*包含零向量。

3.  **確保加法反元素存在：** 雖然僅存在零向量並不能保證子空間內存在加法反元素，但它的缺失會造成根本性的問題。如果一個向量 $\mathbf{v}$ 在子空間中，並且子空間在純量乘法下封閉，那麼 $(-1)\mathbf{v} = -\mathbf{v}$ 也必須在子空間中（這是 $\mathbf{v}$ 的加法反元素）。為了使 $\mathbf{v}$ 和 $-\mathbf{v}$ 相加的結果是*在*子空間*內*的向量，該結果向量必須是零向量。如果零向量不在子集中，那麼即使在較大的向量空間中存在加法反元素，加法封閉性也可能被違反。

簡單來說，如果一個子集甚至不包含向量加法的「單位元素」（即零向量），它就不能真正作為一個在向量加法和純量乘法（這些是向量空間的定義運算）下自洽的系統運作。

因此，零向量必須在子空間中這個條件並非隨意設定的；它是子空間必須繼承其所處的較大向量空間的基本代數結構這一要求的直接結果。
