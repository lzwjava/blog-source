是的，正確無誤。Anthropic 基於終端的編碼工具 Claude Code 提供兩種主要登入方式：

- **具訂閱的 Claude 帳戶**：使用您現有的 Claude.ai 訂閱（Pro 方案起價為每月 20 美元，Max 方案為每月 100 美元），提供可預測的定價並包含在相應方案中的存取權限。
- **Anthropic Console 帳戶**：根據 API 使用量計費，需在 Anthropic Console 中設定帳單以實現隨用隨付的 token 消費。

您可以在登入時選擇方式，且該工具支援透過設定切換或限制方法（如有需要）。
