**Raspberry Pi Pico** 是一款由樹莓派基金會開發的多功能、低成本微控制器板，專為嵌入式系統、原型製作及DIY電子項目而設計。與樹莓派單板電腦（如Raspberry Pi 4）不同，Pico是微控制器，意味著它專注於硬體控制、執行輕量級程式，並與感測器、馬達等元件互動。它於2021年1月推出，標誌著樹莓派基金會進軍微控制器市場，為愛好者、教育者及專業人士提供了一個經濟實惠且功能強大的平台。

以下是對Raspberry Pi Pico的全面介紹，涵蓋其功能、規格、程式設計能力、應用場景等。

---

### **概述**
Raspberry Pi Pico 的核心是**RP2040**，這是由樹莓派基金會自訂設計的微控制器晶片。其價格約為4美元，可與Arduino及ESP32等平台競爭，但憑藉其高效能、低成本及廣泛的社群支援而脫穎而出。Pico體積小巧，僅51mm x 21mm，適合初學者和進階用戶，從簡單的LED閃爍到複雜的物聯網及機器人應用皆可勝任。

---

### **主要功能**
1. **RP2040微控制器**：
   - 雙核心**Arm Cortex-M0+**處理器，最高運行頻率達**133 MHz**（可超頻）。
   - **264 KB SRAM**及**2 MB板載QSPI快閃記憶體**，用於程式儲存。
   - 低功耗設計，具睡眠與休眠模式，適合電池供電應用。
   - 靈活的時鐘配置，可優化效能。

2. **GPIO引腳**：
   - 26個多功能**通用輸入輸出（GPIO）**引腳。
   - 支援**I2C**、**SPI**、**UART**及**PWM**介面，用於連接周邊裝置。
   - 包含2個UART、2個SPI控制器、2個I2C控制器及16個PWM通道。
   - 3個12位元類比數位轉換器（ADC），用於類比感測器輸入。
   - 8個可程式設計I/O（PIO）區塊，支援自訂協定（如WS2812 LED控制、VGA輸出）。

3. **電源與連接性**：
   - 可透過**USB micro-B**（5V）或外部電源（1.8–5.5V）供電。
   - GPIO引腳為**3.3V邏輯電平**。
   - RP2040內建**溫度感測器**。
   - USB 1.1控制器，支援裝置與主機模式（用於程式設計與除錯）。

4. **實體設計**：
   - 小巧尺寸：51mm x 21mm。
   - 40引腳DIP式佈局，具**半孔設計**，可直接焊接至PCB或使用麵包板。
   - 單面元件佈局，便於焊接。

5. **低成本**：
   - 價格約4美元，是目前最經濟實惠的微控制器之一。

### **型號變體**
自推出以來，樹莓派基金會及合作夥伴已發布多款Pico變體：
- **Raspberry Pi Pico W**（2022年）：新增**Wi-Fi**（2.4 GHz 802.11n）及**藍牙5.2**（透過Infineon CYW43439晶片），支援無線物聯網應用。價格約6美元。
- **Raspberry Pi Pico H**：預焊40引腳接頭，便於原型製作。
- **Raspberry Pi Pico WH**：結合Pico W的無線功能與預焊接頭。
- **Pico 2**（2024年）：搭載**RP2350**微控制器，為RP2040的升級版，具雙核心**Arm Cortex-M33**或**RISC-V Hazard3**（用戶可選）、520 KB SRAM、更佳功耗效率及增強安全功能（如Arm TrustZone、SHA-256加速）。

### **為Raspberry Pi Pico編程**
Pico支援多種程式語言與開發環境，適合不同層級的用戶：

1. **MicroPython**：
   - 最受初學者及快速原型開發歡迎的選擇。
   - 樹莓派基金會提供官方MicroPython韌體。
   - 支援GPIO、I2C、SPI、PWM、ADC及PIO的函式庫。
   - 透過USB進行互動式REPL（讀取-求值-輸出循環），實現即時編碼。

2. **C/C++**：
   - 使用官方**Pico SDK**（軟體開發套件）可完全控制RP2040功能。
   - 適合效能關鍵應用及底層硬體控制。
   - 支援進階功能如PIO程式設計與多核心處理。
   - 使用CMake及GCC等工具進行編譯。

3. **其他語言**：
   - **CircuitPython**：Adafruit基於MicroPython的分支，專為教育及易用性優化。
   - **Rust**：社群驅動的Rust語言支援。
   - **Arduino**：可透過Arduino IDE及官方RP2040板卡套件進行程式設計。
   - 其他語言的實驗性支援，如JavaScript（透過Espruino）及Lua。

4. **開發工具**：
   - **拖放式程式設計**：按住BOOTSEL按鈕，透過USB上傳MicroPython或CircuitPython的.uf2韌體檔案。
   - **除錯**：支援SWD（序列線除錯），可搭配Raspberry Pi Debug Probe等工具進行進階除錯。
   - 常用整合開發環境包括**Thonny**（適用Python）及**Visual Studio Code**（適用C/C++）。

### **應用場景**
Raspberry Pi Pico的靈活性使其適用於多種項目，包括：
- **原型製作與教育**：適合學習嵌入式系統、程式設計及電子學。
- **物聯網項目**：透過Pico W，可建立Wi-Fi裝置如智能家居控制器或氣象站。
- **機器人**：控制馬達、伺服器及感測器，用於機器人應用。
- **自訂介面**：使用PIO實現協定如WS2812（NeoPixel）LED控制、VGA或DVI輸出。
- **數據記錄**：連接感測器（如溫度、濕度、光線）進行環境監測。
- **穿戴式裝置與嵌入式系統**：小巧尺寸與低功耗適合穿戴科技及電池供電裝置。

### **與其他方案的比較**
- **Arduino**：Pico速度更快（雙核心、133 MHz對比Arduino Uno的16 MHz）且更便宜，GPIO更多並具PIO等進階功能。但Arduino的生態系統更龐大，有更多擴充板與函式庫。
- **ESP32**：ESP32內建Wi-Fi與藍牙，但Pico W以更低成本提供相同功能。Pico的PIO在自訂協定方面獨具優勢。
- **STM32**：Pico對初學者更易上手（如使用MicroPython），且比多數STM32開發板更經濟實惠。

---

### **限制**
- **無內建無線功能（基礎型號）**：原始Pico缺乏Wi-Fi/藍牙，但Pico W已解決此問題。
- **有限板載儲存**：2 MB快閃記憶體對多數項目足夠，但可能限制複雜應用。
- **3.3V邏輯電平**：連接5V周邊裝置時需使用邏輯位準轉換器。
- **無內建重置按鈕**：對部分用戶而言稍顯不便。

### **入門指南**
開始使用Raspberry Pi Pico的步驟：
1. **硬體設定**：透過USB將Pico連接至電腦，或使用外部電源供電。
2. **安裝韌體**：從樹莓派官網或Adafruit下載MicroPython或CircuitPython韌體，按住BOOTSEL按鈕並連接Pico以上傳韌體。
3. **程式設計**：使用Thonny進行Python開發，或設定Pico SDK進行C/C++開發。
4. **項目實作**：從簡單項目開始，如閃爍LED或讀取感測器，再進階至物聯網或機器人等應用。
