## 什麼是 OpenRouter？

**OpenRouter** 是一家以開發者為核心的 AI 基礎設施新創公司，它作為統一 API 閘道——或稱「市場平台」——讓用戶能透過單一接口存取來自多個供應商的各種大型語言模型（LLM）。它透過讓開發者使用一致的 API 界面來簡化 AI 整合流程，無需考慮底層模型供應商的差異。([Puter][1])

---

## 發展歷程概要

* 由 **Alex Atallah**（OpenSea 前技術總監暨共同創辦人）與 Louis Vichy 於 **2023 年初創立**，靈感源自 Meta 的 LLaMA 和史丹佛大學的 Alpaca 等開源 LLM 的蓬勃發展。([Puter][1])
* 核心概念？隨著各類專業模型湧現，開發者需要集中化的方式來探索、比較和切換不同模型。OpenRouter 正是為滿足此需求而生。

---

## 平台特色與使命

OpenRouter 的核心理念專注於建立多元模型、多元供應商的生態系統，具備以下基礎功能：

* **統一 API 界面**：跨所有支援模型提供單一且與 OpenAI 相容的 API——切換供應商時無需改寫程式碼。([Puter][1], [OpenRouter][2])
* **透明定價**：平台直接採用模型供應商的原始定價，不加收任何溢價。OpenRouter 僅從所有推理成本中收取 5% 費用。([Puter][1])
* **效能洞察**：提供即時數據指標，包括延遲時間、吞吐量、熱門度與成本效益。([Puter][1], [OpenRouter][2])
* **韌性與路由智能**：根據供應商可用性、成本與效能自動執行故障轉移與路由分配——維持高運作時間。([OpenRouter][2])

## 成長與市場表現

* **用戶基礎與使用增長**：截至 **2025 年中**，已有超過 **250 萬名開發者** 使用 OpenRouter；平台每年處理超過 **100 兆個 token**，開發者支出於五月時達到 **每月 800 萬美元**。([Puter][1])
* **融資情況**：

  * **種子輪（2025 年 2 月）**：由 Andreessen Horowitz 領投 1,250 萬美元

    綜合估值現約達 **5 億美元**。([Puter][1], [The Wall Street Journal][3])
* **未來展望**：營運長 Chris Clark 斷言，**AI 推理成本** 即將超越薪資支出，成為知識型企業最大的營運開銷。他預測 2025 年總推理支出將達到約 250 億美元。([Puter][1])

## 核心優勢與應用場景

OpenRouter 在以下領域表現突出：

* **開發者靈活性**：可根據成本與速度等任務限制，輕鬆測試並切換不同模型。
* **企業級工具**：提供集中化帳單管理、使用分析與組織管控（包含共享點數與 API 金鑰管理）。([OpenRouter][4], [OpenRouter][5])
* **無縫擴展**：邊緣部署與多供應商故障轉移機制，確保即使個別供應商出現問題仍能維持高可用性。([OpenRouter][6])
* **數據驅動洞察**：公開的 LLM 排行榜顯示當前趨勢模型——為開發者節省時間並協助優化選擇。([Menlo Ventures][7])
