> 給我一個快速排序算法的例子

這是一個簡單的快速排序算法示例，這是一種常用的排序算法，用於按升序排列數字列表：

```
function quicksort(array) {
    // 基本情況：0或1個元素的數組已經排序
    if (array.length <= 1) {
        return array;
    }

    // 選擇樞軸元素（這裡我們使用數組的第一個元素）
    const pivot = array[0];

    // 將數組分為兩個“子數組”——小於樞軸的元素和大於樞軸的元素
    const less = [];
    const greater = [];
    for (let i = 1; i < array.length; i++) {
        if (array[i] < pivot) {
            less.push(array[i]);
        } else {
            greater.push(array[i]);
        }
    }

    // 遞歸排序子數組
    return quicksort(less).concat(pivot, quicksort(greater));
}

// 示例用法
const array = [5, 3, 4, 1, 2];
const sortedArray = quicksort(array);  // [1, 2, 3, 4, 5]
```

在這個例子中，快速排序函數接受一個數組作為輸入，並返回一個包含相同元素但已排序的新數組。該函數通過從數組中選擇一個樞軸元素，將數組的其餘部分分為兩個“子數組”，根據元素是否小於或大於樞軸，然後遞歸地對子數組進行排序。這個過程一直持續到子數組有0或1個元素，此時數組被認為是完全排序的。
