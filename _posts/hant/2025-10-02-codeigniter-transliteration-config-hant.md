### 在 CodeIgniter 中的用途
這是一個來自 **Text Helper** 函式庫的設定檔案（在 CodeIgniter 中命名為 `foreign_characters.php`）。它用於 **音譯轉換**，這是一個將非 ASCII（帶有重音、變音符號或外語）字符轉換為最接近的 ASCII 等效字符的過程（例如，ä → ae, ñ → n）。這有助於處理像是 URL 別名、資料庫安全字串或跨平台相容性等任務，在這些情況下特殊字符可能會導致問題。

- 它是舊版 CodeIgniter（4.x 之前）的一部分；在 CI4 中，類似功能已移至輔助函式或全域函式。
- 該陣列使用正則表達式模式（例如，`/ä|æ|ǽ/` 匹配 ä、æ 或 ǽ）來將字符替換為純英文字母。
- 使用此設定的主要函式：Text Helper 中的 `convert_accented_characters()`，它會遍歷此陣列並執行 `preg_replace`。

這並非框架核心程式碼，而是一個輔助實用工具，通常可以在安裝目錄中的 `application/config/foreign_characters.php` 找到。

### 所代表的語言與字符集
該陣列涵蓋了多種書寫系統和語言的字符，以實現廣泛的音譯轉換。以下是按類別細分的說明，包括範例（來自程式碼）及其語言/來源：

- **拉丁字母（西歐）**：羅曼語族和日耳曼語族中常見的變音符號。
  - 重音符號（例如，ÀÁÂ → A, àáâ → a）用於法語、西班牙語、葡萄牙語、加泰羅尼亞語。
  - 變音符號（例如，Ä → Ae, ä → ae, Ü → Ue）用於德語、瑞典語。
  - 合字（例如，æ → ae 用於丹麥語/挪威語，œ → oe 用於法語）。
  - 特殊字符（例如，ñ → n 用於西班牙語，ł → l 用於波蘭語，þ → th 用於冰島語）。

- **希臘字母**：用於希臘字母表的西里爾式字母。
  - 例如，Γ → G, Δ → Dj, Η → I, Ν → N, Ο → O, Ρ → R, Σ → S, Τ → T（古/現代希臘語）。

- **西里爾字母（斯拉夫及東歐語言）**：基於俄語、烏克蘭語、白俄羅斯語等。
  - 基礎字符：А → A, Б → B, В → V, Г → G, Д → D, Ж → Zh, З → Z, П → P, Р → R, С → S, Т → T, Ф → F, Х → Kh, Ц → Ts, Ч → Ch, Ш → Sh, Щ → Shch, Ъ/Ь（軟/硬音符号）→ 刪除（無等效字符）, Ю → Yu, Я → Ya。
  - 額外字符：Ё/ё → Yo/yo（俄語）, Є/є → Ye/ye（烏克蘭語）, Ї/ї → Yi/yi（烏克蘭語）, Й → Y（俄語/烏克蘭語中的變體）。

- **其他符號/字符**：
