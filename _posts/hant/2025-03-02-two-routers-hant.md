要將兩台 TP-Link AX3000 (TL-XDR 3050) 路由器與您的調制解碼器配置為 **調制解碼器 → 以太網線 → 路由器1 → 無線 → 路由器2**，請按照以下步驟進行。您的調制解碼器已經有 WiFi 功能和 IP 地址 `192.168.1.1`。我們將配置路由器1 作為主路由器，通過以太網線連接到調制解碼器，並將路由器2 無線延伸網絡。

以下是如何實現這一點：

---

### **步驟 1：了解配置**
- **調制解碼器**：提供互聯網訪問並擁有自己的 WiFi（IP：`192.168.1.1`）。
- **路由器1**：將通過以太網線連接到調制解碼器，並作為您網絡的主要路由器。
- **路由器2**：將無線連接到路由器1 以擴展網絡覆蓋範圍。

您提到的幾種模式（有線 AP 桥接、無線 AP 桥接、DHCP、廣域網連接）。我們將使用 **DHCP** 讓路由器1 從調制解碼器獲取互聯網連接，並使用 **無線 AP 桥接**（或類似模式如 WDS/Repeater）讓路由器2 無線連接到路由器1。

---

### **步驟 2：設置路由器1**
1. **將路由器1 連接到調制解碼器**：
   - 取一條以太網線，將一端插入調制解碼器的 **LAN 端口**。
   - 將另一端插入路由器1 的 **WAN（互聯網）端口**。

2. **訪問路由器1 的網頁界面**：
   - 使用以太網線或連接到路由器1 的默認 WiFi 網絡（檢查路由器標籤以獲取默認 SSID 和密碼）。
   - 打開網頁瀏覽器並輸入 `http://tplinkwifi.net` 或 `192.168.0.1`（TP-Link 路由器的默認 IP）。
   - 使用默認憑證登錄（通常是 `admin` 作為用戶名和密碼），除非您已更改。

3. **配置路由器1**：
   - **互聯網連接**：
     - 轉到 **快速設置** 或 **互聯網** 設置部分。
     - 選擇 **DHCP** 模式。這允許路由器1 自動從調制解碼器獲取 IP 地址（可能在 `192.168.1.x` 范圍內）。
   - **WiFi 設置**：
     - 為路由器1 的 WiFi 設置唯一的 **SSID**（網絡名稱）和強密碼。
     - 保存這些詳細信息，因為路由器2 需要它們無線連接。
   - **LAN 設置**：
     - 確保路由器1 的 LAN IP 與調制解碼器的 IP 不同。默認情況下，它可能是 `192.168.0.1`，這是可以的，因為調制解碼器是 `192.168.1.1`。
     - 確認路由器1 上啟用了 **DHCP**。這允許路由器1 為連接到它的設備（包括路由器2）分配 IP 地址（例如 `192.168.0.x`）。
   - 保存設置並重新啟動路由器1（如果提示）。

---

### **步驟 3：將路由器2 設置為無線橋接**
1. **訪問路由器2 的網頁界面**：
   - 使用以太網線或連接到路由器2 的默認 WiFi 網絡。
   - 打開網頁瀏覽器並輸入 `http://tplinkwifi.net` 或 `192.168.0.1`。
   - 使用默認憑證（或您的自定義憑證）登錄。

2. **將路由器2 配置為無線橋接模式**：
   - 在設置中查找類似 **無線 AP 桥接**、**WDS** 或 **Repeater** 的模式（可能在 **操作模式** 或 **無線** 設置下）。
   - 選擇 **無線 AP 桥接**（或 WDS/Repeater，如果可用）。
   - **連接到路由器1 的 WiFi**：
     - 檢查可用網絡並選擇路由器1 的 SSID。
     - 输入路由器1 的 WiFi 密碼。
     - 確保路由器2 使用與路由器1 相同的無線頻道以兼容（例如，如果路由器1 在頻道 6，則將路由器2 設置為頻道 6）。
   - **LAN IP 設置**：
     - 更改路由器2 的 LAN IP 以避免與路由器1 衝突。例如，將其設置為 `192.168.0.2`（因為路由器1 可能是 `192.168.0.1`）。
     - **在路由器2 上禁用 DHCP**。路由器1 將為所有設備處理 IP 分配。
   - 保存設置並重新啟動路由器2。它應該現在無線連接到路由器1。

---

### **步驟 4：測試配置**
1. **驗證路由器2 的連接**：
   - 重新啟動後，檢查路由器2 的界面以確認它已連接到路由器1 的 WiFi。
2. **將設備連接到路由器2**：
   - 使用智能手機、筆記本電腦或其他設備連接到路由器2 的 WiFi（它可能使用與路由器1 相同的 SSID，具體取決於模式）。
   - 檢查設備是否從路由器1 获取 IP 地址（例如 `192.168.0.x`）。
   - 通過瀏覽網站測試互聯網訪問。

---

### **最終配置摘要**
- **調制解碼器**：IP `192.168.1.1`，提供互聯網和 WiFi。
- **路由器1**：
  - WAN：通過以太網線連接到調制解碼器，設置為 **DHCP**（從調制解碼器獲取 IP，例如 `192.168.1.x`）。
  - LAN：IP `192.168.0.1`，**DHCP 啟用** 以分配 IP 給設備。
  - WiFi：自定義 SSID 和密碼。
- **路由器2**：
  - 模式：**無線 AP 桥接**（或 WDS/Repeater），無線連接到路由器1 的 WiFi。
  - LAN IP：`192.168.0.2`，**DHCP 禁用**。

---

### **故障排除提示**
- **路由器2 無法無線連接**：
  - 重新檢查路由器1 的 SSID、密碼和頻道設置在路由器2 的配置中。
  - 如果 **無線 AP 桥接** 可用，請嘗試 **WDS** 或 **Repeater** 模式。參考 TP-Link AX3000 手冊以獲取支持的模式。
- **路由器2 沒有互聯網**：
  - 確保路由器1 的 DHCP 正常工作，路由器2 的 DHCP 關閉。
  - 驗證路由器2 是否成功橋接到路由器1。
- **雙重 NAT 擔憂**：
  - 由於您的調制解碼器有 WiFi 並且路由器1 作為路由器，您可能會有雙重 NAT（調制解碼器和路由器1 都分配 IP）。這對於基本使用通常是可以的，但可能會影響某些應用程序（例如遊戲）。如果需要，將調制解碼器設置為 **橋接模式**（諮詢調制解碼器的手冊），這可能會禁用其 WiFi。

通過這種配置，路由器2 將無線延伸路由器1 的網絡，並且連接到任何路由器的設備應該通過調制解碼器訪問互聯網。如果您需要進一步的幫助，請告訴我！

如果您決定為第一個路由器（路由器1）使用固定（靜態）IP 地址而不是依賴 DHCP，您需要手動配置其 WAN 設置以與調制解碼器通信。以下是您應該為 IP 地址、DNS、子網掩碼和網關配置的內容：

---

### **假設**
- 您的調制解碼器的 IP 地址為 `192.168.1.1`，並使用子網 `192.168.1.0/24`（子網掩碼 `255.255.255.0`）。
- 调制解碼器有一個 DHCP 范圍（例如 `192.168.1.2` 到 `192.168.1.100`），為連接的設備分配 IP 地址。

如果您的調制解碼器使用不同的 IP 或子網，請相應調整這些值。

---

### **路由器1 的 WAN 設置**
這些設置配置路由器1 如何連接到調制解碼器：

1. **IP 地址**
   - 在調制解碼器的子網（`192.168.1.0/24`）中選擇一個靜態 IP，但 **在調制解碼器的 DHCP 范圍之外** 以避免衝突。
   - 例如：如果調制解碼器的 DHCP 范圍是 `192.168.1.2` 到 `192.168.1.100`，您可以使用 **`192.168.1.101`**。
   - 這確保路由器1 在調制解碼器的網絡上擁有唯一的固定地址。

2. **子網掩碼**
   - 將其設置為與調制解碼器的子網掩碼匹配，通常是 **`255.255.255.0`**。
   - 這定義了網絡範圍為 `192.168.1.0` 到 `192.168.1.255`。

3. **默認網關**
   - 將其設置為調制解碼器的 IP 地址，即 **`192.168.1.1`**。
   - 網關是路由器1 用來訪問互聯網的設備（您的調制解碼器）。

4. **DNS 服務器**
   - 您有兩個選項：
     - 使用調制解碼器的 IP： **`192.168.1.1`**。這讓調制解碼器處理 DNS 請求。
     - 使用公共 DNS 服務器： **`8.8.8.8`**（Google）和 **`8.8.4.4`**（Google 次要）。這些是可靠且通常更快。
   - 建議：從 `192.168.1.1` 開始。如果您遇到 DNS 問題，切換到 `8.8.8.8` 和 `8.8.4.4`。

---

### **示例配置**
這是您的路由器1 WAN 設置可能的樣子：

- **IP 地址**：`192.168.1.101`
- **子網掩碼**：`255.255.255.0`
- **默認網關**：`192.168.1.1`
- **DNS 服務器**：`192.168.1.1`（或 `8.8.8.8` 和 `8.8.4.4`）

---

### **其他說明**
- **檢查調制解碼器設置**：登錄到您的調制解碼器（通常在 `192.168.1.1`）以確認其 IP、子網掩碼和 DHCP 范圍。根據需要調整路由器1 的靜態 IP。
- **路由器1 的 LAN 設置**：為避免衝突，將路由器1 的 LAN 網絡配置為不同的子網，例如 `192.168.0.1`，子網掩碼為 `255.255.255.0`。在路由器1 上啟用 DHCP 以分配 IP（例如 `192.168.0.2` 到 `192.168.0.254`）給連接到它的設備。
- **測試**：應用這些設置後，確保路由器1 可以訪問互聯網。如果不能，重新檢查 IP（無衝突）、網關和 DNS 值。

這種配置給路由器1 在調制解碼器的網絡上提供了一個固定 IP，確保一致的連接。如果您需要幫助配置其他路由器或具體的調制解碼器詳細信息，請告訴我！
