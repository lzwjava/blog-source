以下是對**三種基本放大器配置**——**共射極（CE）**、**共集極（CC）**（亦稱為**射極隨耦器**）與**共基極（CB）**——的全面解說，重點闡述其結構、訊號特性、優點、缺點及典型應用場景。

---

## 🔷 1. 共射極（CE）放大器

### 🔧 配置結構
- **輸入端**：施加於基極與射極之間。
- **輸出端**：從集極與射極之間取出。
- **共用端**：射極同時為輸入與輸出的共用端。

### 🔍 關鍵特性

| 屬性                  | 說明                       |
|-----------------------|----------------------------|
| **電壓增益**          | 高                         |
| **電流增益**          | 中至高                     |
| **功率增益**          | 高                         |
| **相位偏移**          | 180°（輸出訊號反相）       |
| **輸入阻抗**          | 中等                       |
| **輸出阻抗**          | 中等                       |

### ✅ 優點
- 適用於電壓與功率放大。
- 最廣泛使用的配置。

### ❌ 缺點
- 訊號反相（180°相位偏移）。
- 較不適合用於阻抗匹配。

### 🧰 應用場景
- 通用訊號放大。
- 音頻放大器。
- 放大器的中間級。

---

## 🔷 2. 共集極（CC）放大器 — *射極隨耦器*

### 🔧 配置結構
- **輸入端**：施加於基極與集極之間。
- **輸出端**：從射極與集極之間取出。
- **共用端**：集極為共用端。

### 🔍 關鍵特性

| 屬性                  | 說明                           |
|-----------------------|--------------------------------|
| **電壓增益**          | 約為 1（單位增益）             |
| **電流增益**          | 高                             |
| **功率增益**          | 中等                           |
| **相位偏移**          | 0°（無反相）                   |
| **輸入阻抗**          | 高                             |
| **輸出阻抗**          | 低                             |

### ✅ 優點
- 優異的緩衝級。
- 適合阻抗匹配（高輸入阻抗、低輸出阻抗）。
- 無訊號反相。

### ❌ 缺點
- 無電壓增益。
- 不適合需電壓放大的獨立放大器應用。

### 🧰 應用場景
- 級間緩衝。
- 驅動低阻抗負載。
- 電壓隨耦器。

---

## 🔷 3. 共基極（CB）放大器

### 🔧 配置結構
- **輸入端**：施加於射極與基極之間。
- **輸出端**：從集極與基極之間取出。
- **共用端**：基極為共用端。

### 🔍 關鍵特性

| 屬性                  | 說明                             |
|-----------------------|----------------------------------|
| **電壓增益**          | 高                               |
| **電流增益**          | 小於 1                           |
| **功率增益**          | 中等                             |
| **相位偏移**          | 0°（無反相）                     |
| **輸入阻抗**          | 極低                             |
| **輸出阻抗**          | 高                               |
| **頻率響應**          | 極寬（適用於高頻應用）           |

### ✅ 優點
- 極佳的高頻響應。
- 運作穩定。

### ❌ 缺點
- 輸入阻抗低——需仔細設計驅動級。
- 電流增益有限。

### 🧰 應用場景
- 射頻（RF）放大器。
- 高頻訊號放大。
- 阻抗匹配（適用於低阻抗源至高阻抗負載）。
