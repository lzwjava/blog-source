### Visual Studio Code 1.104 (2025年8月)

本次更新重點聚焦於 GitHub Copilot 的人工智慧功能強化、AI 工具的安全性提升，以及編輯器與終端機的生產力優化。以下是主要更新內容摘要：

#### AI 功能
- **自動模型選擇 (預覽版)**：當選擇「自動」模式時，聊天功能會根據效能與速率限制自動選取最佳模型（如 Claude Sonnet 4、GPT-5 變體）；付費用戶可享 10% 請求折扣
- **敏感檔案編輯確認**：代理模式在編輯系統檔案、dotfile 或工作區外檔案前需取得使用者授權；可透過設定自訂
- **AGENTS.md 檔案支援 (實驗性)**：自動將工作區 `AGENTS.md` 納入聊天請求的上下文環境
- **強化編碼代理協作 (實驗性)**：改進會話管理、GitHub 整合，以及從 TODO 註解或聊天進行的委派功能
- **終端機自動核准**：可選擇加入更安全的指令執行模式，對高風險操作（如 `curl` 或 `wget`）發出警告；新增核准規則
- **數學公式渲染**：KaTeX 方程式現預設在聊天回應中內嵌渲染
- **改進 #codebase 工具**：採用全新嵌入模型，實現更快速高效的語義程式碼搜尋
- **停用 AI 功能**：新增設定可全域或依工作區隱藏並停用 Copilot 聊天、自動完成與建議功能
- **Python 專屬 AI 工具 (實驗/預覽)**：AI 驅動的未註解符號懸浮摘要，以及用於記憶體內執行的「執行程式碼片段」工具

#### 安全性
- **AI 工具防護機制**：擴展敏感編輯、終端機指令與全域自動核准的確認流程，配備可自訂規則的警告功能以降低風險
- **技術文件**：新增 AI 驅動開發的安全考量指引

#### 生產力
- **聊天視窗預設值**：聊天功能現預設在次要側邊欄開啟；可自訂顯示狀態
- **終端機改進**：強化 IntelliSense（如 git 自動完成、圖示）、固定滾動區域，以及終端機編輯器/視窗中的操作功能
- **Python 環境**：增強 Pipenv 支援、從 `.env` 檔案注入環境變數，並在所有 Python 文件中提供完整 IntelliSense
- **編輯器調整**：可設定的內嵌建議延遲時間、鍵盤導覽索引標籤順序，以及 Windows 系統的視窗邊框色彩
- **GitHub Pull Requests**：可收合側邊欄、Copilot 的 TODO 操作選項，以及子模組忽略設定
- **任務與終端機支援**：改進輸入處理、錯誤顯示與 shell 整合逾時設定

#### 其他重要變更
- **無障礙功能**：針對聊天確認新增焦點指令以輔助螢幕閱讀器
- **語言支援**：移除內建 Bower.json IntelliSense（已淘汰工具，請遷移至 npm/yarn）
- **擴充功能開發**：提出驗證挑戰與次要側邊欄貢獻的 API 提案
- **淘汰項目**：如 `chat.mcp.enabled` 等設定已遷移；停止支援 Bower
