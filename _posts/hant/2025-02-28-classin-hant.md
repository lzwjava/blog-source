### ClassIn 班級應用平台的詳細使用指南

本指南提供了如何使用 ClassIn 的全面概述，這是一個專為線上和混合學習設計的教育平台，適合教師和學生。該平台由 Empower Education Online (EEO) 開發，提供互動教室、課程排程、作業管理等功能，適合 K12 學校、高等教育和企業。以下詳細說明了下載、註冊和使用平台的步驟，根據官方資源和在線用戶指南。

#### 平台概述
ClassIn 是一個多功能的教育科技解決方案，支持線上直播教室和離線智能教室，整合學習管理系統 (LMS) 和個人學習環境 (PLE)。它已被超過 60,000 個組織採用，每月教學 3000 萬節課，覆蓋 150 多個國家，服務 200 萬教師和 3000 萬學習者。該平台可在 Windows、iOS 和 Android 上使用，確保廣泛的用戶可訪問性。

#### 下載應用程式
要開始使用，用戶必須先下載 ClassIn 應用程式。應用程式可以通過以下方式訪問：
- **iOS 設備：** 從 [App Store](https://apps.apple.com/us/app/classin/id1226361488) 下載。
- **Android 設備：** 從 [Google Play](https://play.google.com/store/apps/details?id=cn.eeo.classin&hl=en_US) 下載。

下載過程簡單，用戶只需在各自的應用商店中搜索「ClassIn」並按照安裝提示操作。對於 Windows 用戶，還有 ClassIn X 和 CamIn 等額外工具，用於混合學習和相機優化，可從官方 [下載頁面](https://www.classin.com/download/) 下載。

#### 註冊和登錄
打開應用程式後，新用戶會被提示註冊。註冊過程通常包括：
- 點擊「註冊」或「註冊」。
- 输入電子郵件地址並創建密碼。
- 完成任何額外驗證，例如電子郵件確認，並填寫個人詳細信息，如姓名或學校關聯。

現有用戶可以使用註冊的電子郵件和密碼登錄。雖然具體的註冊 URL 可能會有所不同（例如，來自教程的促銷鏈接），但一般過程嵌入在應用程式介面中，確保所有用戶都能訪問。

#### 教師使用
對於教育工作者，ClassIn 提供了一套強大的工具來創建和管理線上課程。以下步驟說明了如何作為教師使用該平台：

1. **創建課程：**
   - 登錄後，導航到儀表板並選擇「創建課程」。
   - 输入課程名稱和任何額外詳細信息，例如學校帳戶名稱，並確認創建。
   - 平台支持各種教學模式，包括一對一、小組和大班，具有雙教師模式（例如，中文和外語教師）和直播流。

2. **邀請學生：**
   - 生成課程代碼並與學生分享，以便他們加入。
   - 或者，使用聯繫人列表發送邀請，接受或拒絕好友請求。

3. **進行課程：**
   - 在直播課程中使用互動工具，包括：
     - 雙向互動白板，用於實時協作。
     - 屏幕共享，用於多方演示。
     - 小組討論室，具有監控功能。
     - 虛擬舉手、投票和靈活座位等功能，以促進參與。
   - 使用高級工具，如群組聊天、媒體存儲、原生網頁瀏覽器和拖放 STEM 實驗，以增強學習。

4. **管理表現和作業：**
   - 使用作業系統分配作業並跟蹤提交。
   - 使用 44 種不同的指標（包括出勤數據、課堂活動和測試結果）來衡量表現，存儲在課程記錄庫中。
   - 使用雲存儲來存儲材料和跟蹤系統來跟蹤進行中的課程。

教師反饋強調了 ClassIn 的易用性，工具增強了互動並有效地重現了課堂體驗。例如，該平台允許在演示中自定義背景、濾鏡和貼紙，增加了參與度。

#### 學生使用
對於學生，ClassIn 提供了一個互動的學習環境，參與課程並管理課程。步驟包括：

1. **加入課程：**
   - 從教師處獲取課程代碼並將其輸入應用程式中的「加入課程」。
   - 加入後，通過儀表板訪問課程時間表和材料。

2. **參加直播課程：**
   - 使用舉手（點擊並按住手圖標以通知教師）、協作文檔和小組討論等功能參與直播課程。
   - 平台支持基於項目的、合作的和探究式學習，培養創造力、溝通和合作技能。

3. **訪問材料和作業：**
   - 通過應用程式查看和下載視頻、評分作業並提交作業。
   - 查看學習報告和記錄以供審查，增強個性化學習體驗。

學生手冊，可在 [ClassIn 學生手冊](https://www.blog.classin.com/post/classin-student-manual) 中找到，提供了這些活動的詳細說明，確保學習者易於使用。

#### 額外功能和技巧
ClassIn 提供混合學習解決方案，支持直播線上課程中最多 2000 名參與者，同時進行 50 人的音頻和視頻，適合大規模教育設定。該平台與 Moodle 集成以增強功能，並提供 API 以進行自定義集成。支持選項包括聊天、電子郵件、電話和知識庫，可通過官方網站訪問。

用戶被鼓勵探索教程和指南以獲取更深入的見解。例如，[ClassIn 教師手冊](https://www.blog.classin.com/post/classin-teacher-manual-an-ultimate-guide-to-classin) 提供逐步說明，而 YouTube 教程，如 [ClassIn 教師教程](https://www.youtube.com/watch?v=t2smgu_7SI4)，演示了實際使用。保持應用程式更新確保訪問最新功能，並檢查系統要求以確保平穩運行。
