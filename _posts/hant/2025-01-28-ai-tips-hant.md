* 將範例程式碼複製到 Cursor，並使用註釋提供背景資訊。

* 在香港地區，使用 DeepSeek 或 Mistral 服務是可以接受的，但不能使用 ChatGPT、Claude 或 Gemini。

* 一個區域的 API 政策大約與其應用程式政策相同。

* 使用 Cursor 而不是 Visual Studio Code，但在 git merge 編輯器的情況下，我仍然必須使用 `git config --global core.editor "code --wait"`。

* 從 Deepseek V3 發布的那天開始，我們不再需要訂閱任何 AI 工具。

* 使用 Gemini 或 Grok 生成節日慶祝圖片，並使用提示如 "生成包含文字名稱的快樂農曆蛇年新年圖片"。
