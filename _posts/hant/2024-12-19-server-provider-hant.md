Outline Manager 提供了多種伺服器選項，包括 Google Cloud、DigitalOcean、AWS Lightsail 等。

對於 Google Cloud 和 DigitalOcean，便利的腳本簡化了伺服器設置過程。我最初選擇這些選項是因為它們看起來比較簡單。然而，這些伺服器有時會被防火長城（GFW）封鎖。

因此，我決定再次嘗試使用 AWS Lightsail。雖然設置過程大約需要十分鐘——相比 Google Cloud 或 DigitalOcean 的大約三分鐘要長一些——但它似乎運行得更為穩定。

有些決定一開始可能讓人覺得麻煩，但隨著時間的推移，往往會變得更容易。在選擇代理伺服器提供商時——以及生活中的每一個方面——應用這種長遠的視角，最終能讓你享受到更好的體驗。
