*Java Performance: In-Depth Advice for Tuning and Programming Java 8, 11, and Beyond*（作者：Scott Oaks）是一本全面指導如何優化 Java 應用程式效能的專著。本書聚焦於 Java 8、Java 11 及後續版本中的實用調優與程式設計技巧，涵蓋核心 Java 及企業級應用場景。以下是書中重點內容概要：

### 1. **Java 效能調優導論**
   - 闡釋效能對 Java 應用程式的重要性，並系統性介紹識別與解決效能瓶頸的方法論
   - 引入效能衡量工具與方法，包括基準測試、性能剖析與監控機制

### 2. **Java 虛擬機（JVM）內部機制**
   - 解析 JVM 架構組成（堆記憶體、堆疊、元空間）及其對效能的影響
   - 探討即時編譯、類別載入機制與 JVM 的程式碼優化策略
   - 詳解如何透過 JVM 參數設定針對特定工作負載進行精細調校

### 3. **垃圾回收（GC）調優**
   - 深入分析 Java 垃圾回收機制，比較不同回收器特性（如 Serial、Parallel、CMS、G1、ZGC、Shenandoah）
   - 提供最小化 GC 停頓時間與優化記憶體使用的實戰策略，涵蓋低延遲與高吞吐量場景的調優要點
   - 探討 Java 11 後新增的 GC 功能，包括 Epsilon（無操作垃圾回收器）及 G1/ZGC 的改進

### 4. **Java 語言與 API 優化**
   - 分析 Java 語言結構的效能影響（如字串處理、集合框架、並行工具）
   - 重點解說 Java 8（lambda 表達式、流處理）與 Java 11（新 HTTP 客戶端、巢狀存取控制）的新特性對效能之影響
   - 提供高效編碼最佳實踐，包括迴圈優化、物件建立與同步機制的常見陷阱規避

### 5. **並行程式設計與多執行緒**
   - 解析 Java 並行框架體系，涵蓋 `java.util.concurrent` 套件、執行緒池與 fork/join 框架
   - 闡述如何透過減少競爭條件、提升擴展性來優化多執行緒應用，充分發揮多核處理器效能
   - 討論後續版本的新並行特性，如 VarHandles 與 CompletableFuture API 的增強功能

### 6. **效能工具與監控**
   - 評比效能分析工具（VisualVM、Java Mission Control、JProfiler）與命令行工具（`jstat`、`jmap`）
   - 解讀關鍵效能指標（CPU 使用率、記憶體消耗、執行緒活動）以診斷問題根源
   - 介紹 Java 11 後新增的飛行記錄器與進階監控功能

### 7. **微服務與雲原生 Java**
   - 探討現代 Java 應用（特別是在微服務架構與雲端環境）的效能挑戰
   - 分析容器化技術（如 Docker）與 JVM 在 Kubernetes 等編排平台的調優要點
   - 評析專為雲原生場景設計的輕量級框架（Quarkus、Micronaut）的效能特性

### 8. **Java 效能實戰應用**
   - 透過企業級真實案例展示效能調優的實踐過程
   - 涵蓋資料庫互動優化、I/O 效能調校與特定工作負載（批次處理、Web 應用）的適配策略
   - 討論程式碼優化過程中效能、可讀性與可維護性之間的取捨平衡

### 9. **建構面向未來的 Java 應用**
   - 探討如何編寫能適應 JVM 版本迭代與硬體發展的永續效能程式碼
   - 解析 Java 9 模組系統（JPMS）的效能影響
   - 提供應對未來 Java 版本與新興技術（如 GraalVM）的準備指南
