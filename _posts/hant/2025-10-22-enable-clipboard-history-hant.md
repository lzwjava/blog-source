### 在 Windows 中啟用剪貼簿歷史記錄
使用快捷鍵前，請先確保此功能已啟用（適用於 Windows 10 版本 1809 及更高版本，包括 Windows 11）：
1. 開啟 **設定**（按下 `Windows 鍵 + I`）。
2. 前往 **系統 > 剪貼簿**。
3. 開啟 **剪貼簿歷史記錄**。

這將允許 Windows 預設儲存最多 25 個近期項目（文字、HTML、圖片）達 1 小時，若同步至 Microsoft 帳戶則可儲存更長時間。

### 使用快捷鍵存取與讀取剪貼簿歷史記錄
開啟並讀取剪貼簿歷史記錄的主要鍵盤快捷鍵是 **`Windows 鍵 + V`**。以下是逐步使用說明：

1. **複製多個項目**：使用 `Ctrl + C` 複製所需文字、圖片或其他內容。Windows 會自動將其儲存至歷史記錄。

2. **開啟剪貼簿歷史記錄**：按下 `Windows 鍵 + V`。這會在螢幕右側開啟一個可搜尋的面板，顯示近期剪貼項目的縮圖或預覽（最新項目位於頂部）。

3. **導航與讀取**：
   - 使用 **方向鍵**（↑/↓）滾動清單。
   - **輸入以搜尋**：面板開啟時，可立即輸入文字來篩選項目（例如搜尋已複製文字中的關鍵字）。
   - **預覽**：將滑鼠懸停或使用方向鍵指向項目，即可預覽內容而無需貼上。

4. **從歷史記錄貼上**：
   - 使用方向鍵選擇項目後按下 **Enter**，即可在游標處貼上。
   - 或使用滑鼠點擊項目。

5. **關閉面板**：按下 `Esc` 或再次按下 `Windows 鍵 + V`。

### 額外提示
- **釘選項目**：在面板中點擊項目旁的釘選圖示，即可永久保留該項目。
- **清除歷史記錄**：右鍵點擊項目 > **刪除**，或前往 設定 > 系統 > 剪貼簿 > **清除**。
- **跨裝置同步**：在設定中啟用 **剪貼簿歷史記錄同步**，即可在其他登入相同 Microsoft 帳戶的 Windows 電腦上存取記錄。
- 若需自訂快捷鍵（例如重新映射），可使用 AutoHotkey 等第三方工具，但預設的 `Win + V` 為內建功能且穩定可靠。
