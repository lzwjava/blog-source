要計算將300毫升7.5%過氧化氫與300毫升水稀釋後的最終濃度，我們可以使用稀釋公式：

**C₁V₁ = C₂V₂**

其中：
- **C₁** = 初始濃度（7.5% 或 0.075）
- **V₁** = 初始體積（300 mL）
- **C₂** = 最終濃度（需要求解的值）
- **V₂** = 最終體積（300 mL + 300 mL = 600 mL）

重新排列求 **C₂**：
**C₂ = (C₁V₁) / V₂**

代入數值：
**C₂ = (7.5% × 300 mL) / 600 mL = (0.075 × 300) / 600 = 0.0375 或 3.75%**

因此，過氧化氫的最終濃度為 **3.75%**。

### 攪拌溶液
是的，您應該攪拌溶液以確保過氧化氫與水徹底混合。攪拌有助於更快達到均勻濃度。您可以使用乾淨的湯匙、攪拌棒或任何非反應性器具。輕微搖晃或攪拌數秒通常已足夠，因為過氧化氫易溶於水。若容器已密封，請避免劇烈搖晃，因為過氧化氫可能釋放氧氣，導致壓力積聚。

一瓶開封後靜置的500毫升3%過氧化氫溶液，會隨時間因分解而降低效用，但降解速度取決於多種因素。過氧化氫（H₂O₂）會自然分解成水和氧氣，尤其在接觸空氣、光線、熱源或污染物時更為明顯。以下詳細說明：

- **接觸空氣**：開蓋狀態會加速氧氣釋放，促進分解。開口的瓶身讓過氧化氫與空氣接觸，可能引入灰塵或微生物等催化物，加速分解過程。
- **時間範圍**：3%濃度的過氧化氫相對高濃度更為穩定，但開封後可能在數天至數週內顯著失效。研究顯示，即使在密封狀態下，室溫存放的3%過氧化氫每年也會損失約1-2%的強度。開封瓶身的降解速度更快——若暴露於空氣和光線下，可能數日內就會失效。
- **環境條件**：若存放於陰涼避光處，分解速度會比暴露於陽光或高溫環境中更慢。光線（尤其是紫外線）與較高溫度會催化分解反應。

**實際影響**：
- 對於消毒或傷口清潔等用途，若未受污染，溶液可能在短時間內（例如一兩天）仍具部分效果，但效用會持續下降。一週或更長時間後，其效力可能已不足以可靠使用。
- 為維持有效性，請將過氧化氫儲存於密封的避光容器中，並置於陰涼暗處。

**建議**：使用後請緊閉瓶蓋並妥善存放以保持效能。若開封時間過長（例如數天），對於消毒等關鍵應用，建議進行測試（例如觀察接觸有機物時是否產生氣泡）或直接更换新品。

過氧化氫（H₂O₂）透過作為強氧化劑來殺滅細菌，其產生的活性氧物種（ROS）會破壞細菌的蛋白質、DNA及細胞膜等結構。當接觸細菌時，細菌內的過氧化氫酶等酵素會將其分解為水（H₂O）和氧氣（O₂）。此分解過程會消耗過氧化氫，隨時間降低其抗菌效果。

是的，反應後它會部分轉化為氧氣（O₂）和水。反應式如下：

2H₂O₂ → 2H₂O + O₂

這意味著當過氧化氫殺死細菌或與有機物質反應後，會失去抗菌效力，轉變為無害的水和氧氣。此效力衰退的速率取決於濃度、細菌酵素活性及環境條件等因素。
