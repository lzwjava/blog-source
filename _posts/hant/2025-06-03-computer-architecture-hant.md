要判斷您目前的電腦是運行 64 位元 x86 (x64) 還是 ARM64 架構，請根據您的作業系統遵循以下指示：

**Windows：**

1.  **按下 Windows 鍵 + R** 以開啟「執行」對話方塊。
2.  **輸入 `msinfo32`** 並按下 Enter。這將開啟「系統資訊」視窗。
3.  在「系統資訊」視窗中，尋找 **「系統類型」** 項目。
    * 如果顯示 **「x64 型電腦」**，則您的電腦配備 64 位元 x86 (x64) 處理器。
    * 如果顯示 **「ARM64 型電腦」**，則您的電腦配備 64 位元 ARM (ARM64) 處理器。

**macOS：**

1.  點擊螢幕左上角的 **Apple 選單**。
2.  選擇 **「關於這部 Mac」**。
3.  在「概覽」標籤頁中，尋找 **「處理器」** 或 **「晶片」** 資訊：
    * 如果列出的是 **Intel 處理器**（例如 Intel Core i7、Intel Core i9），則您的 Mac 配備 64 位元 x86 (x64) 處理器。
    * 如果列出的是 **Apple 晶片**（例如 Apple M1、Apple M2、Apple M3），則您的 Mac 配備 ARM64 處理器。

**Linux：**

1.  開啟 **終端機** 視窗。
2.  輸入以下指令並按下 Enter：
    ```bash
    uname -m
    ```
3.  輸出將顯示架構：
    * **`x86_64`** 或 **`amd64`**：表示為 64 位元 x86 (x64) 處理器。
    * **`aarch64`** 或 **`arm64`**：表示為 64 位元 ARM (ARM64) 處理器。
    * 其他輸出可能表示 32 位元系統（現已較不常見）。

根據您為新組裝電腦列出的組件，**Intel Core i9-13900KF** 處理器是 **x64 (64 位元 x86)** 架構處理器。因此，您正在組裝的新電腦將是 **x64 型系統**。
