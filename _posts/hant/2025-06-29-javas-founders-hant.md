Java 程式語言主要由 **James Gosling** 在 1990 年代初於 Sun Microsystems 開發，他常被稱為「Java 之父」。以下是對 Java 創建與演進過程中關鍵貢獻者的簡要概述：

- **James Gosling**：在 Sun Microsystems 領導 Java 開發團隊。他設計了原始語言、語法及 Java Virtual Machine (JVM)，該專案於 1991 年啟動（最初名為「Oak」）。
- **Mike Sheridan**：作為 Sun 公司原始「Green Team」的關鍵成員，他對嵌入式系統語言的早期願景與設計作出了貢獻。
- **Patrick Naughton**：同為 Green Team 成員，他協助確立初步目標，並致力於語言的可移植性與平台獨立性。
- **Bill Joy**：作為 Sun 共同創辦人，他提供了策略性意見，並影響了 Java 的架構，特別是其在網絡與安全方面的聚焦。
- **Guy Steele**：對語言規範作出貢獻，特別是 Java Language Specification，確保其設計的清晰度與嚴謹性。
- **Arthur van Hoff**：致力於 Java 編譯器與運行環境的開發，協助 Java 實現廣泛應用。
- **Tim Lindholm**：對 JVM 及效能優化作出貢獻，並合著了 Java Language Specification。
- **Frank Yellin**：在 JVM 的安全模型與實作方面扮演重要角色。

1996 年 Java 的首次發布（JDK 1.0）是 Sun Microsystems 團隊共同努力的成果，上述個人在其設計與實作中居於核心地位。發布後，Java 社群（包括 Java Community Process (JCP) 的貢獻者）、Oracle（於 2010 年收購 Sun 後）以及 OpenJDK 等開源專案，持續推動語言的演進。後期著名貢獻者包括 **Brian Goetz**，他領導了現代 Java 功能（如 lambda 表達式與模組系統）的開發工作。
