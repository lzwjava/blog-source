是的，Ghostty 被設計為跨平台終端模擬器，這確實是其關鍵賣點之一——它旨在多個操作系統上保持一致的運行表現，透過 GPU 加速和原生 UI 整合實現高效能。然而，截至 2025 年 10 月，其完整支援目前僅限於 macOS 和 Linux，Windows 版本仍在規劃階段，尚未正式發布。

關於使用體驗：由於 Ghostty 採用原生 UI 組件（例如 macOS 上的 Cocoa、Linux 上的 GTK），在不同平台上的操作感受並非*完全*一致——這樣設計是為了讓它在每個系統上都顯得貼合原生環境，像是專為該平台打造的應用程式，而非一刀切的移植版本。話雖如此，其核心功能、特色（如分頁視窗、分割畫面、搜尋功能及進階終端協定）與效能表現，在所有支援平台上都力求高度一致且穩定可靠，盡量減少您在較不完善的模擬器中可能遇到的平台特定問題。

若 Windows 版本能如期推出（根據 GitHub 上的持續討論），它同樣會採用原生架構以實現無縫的使用體驗。
