免費的 GitHub 使用者可以建立**無限量的私人儲存庫**，但在**協作者**數量上有限制。

* 您可以擁有任意數量的私人儲存庫
* 每個私人儲存庫最多可添加**3 位協作者**（即您邀請共同作業的人員）

若需要更多協作者，您需要升級至付費方案（GitHub Pro、Team 或 Enterprise）。

您是否還需要我說明這項政策與舊版 GitHub 規定的差異（2019 年之前，免費使用者根本無法擁有私人儲存庫）？
