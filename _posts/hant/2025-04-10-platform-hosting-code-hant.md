除了 GitHub 之外，其他著名的代碼託管平台還包括：

- GitLab：完整的 DevOps 平台，提供代碼庫託管、CI/CD、監控等功能
- Bitbucket：由 Atlassian 擁有，與 Jira 及其他 Atlassian 產品良好整合
- SourceForge：最古老的代碼託管平台之一，特別受開源項目歡迎
- Azure DevOps（前稱 Visual Studio Team Services）：微軟的代碼託管與 DevOps 解決方案
- AWS CodeCommit：亞馬遜的託管 Git 服務
- Gitee：在中國廣受歡迎，常被稱為「中國版 GitHub」
- Launchpad：主要用於 Ubuntu 及相關開源項目
- CodePen：專注於前端代碼（HTML、CSS、JavaScript）
- Codeberg：非營利、社區驅動的替代方案，專注於自由及開源軟體
- Replit：結合代碼託管與線上 IDE

這些平台在功能、定價模式和目標用戶群方面各不相同，有些更側重企業用戶，有些則專注於開源社區。
