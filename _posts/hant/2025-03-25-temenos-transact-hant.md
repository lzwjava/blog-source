**Temenos Transact**（前身為 **T24 Transact**）是由 **Temenos** 提供的**核心銀行系統**，旨在支援零售銀行、企業銀行、伊斯蘭銀行及私人銀行業務。這是一個現代化、雲端原生且 API 驅動的銀行解決方案，協助金融機構自動化流程、提升客戶體驗並推動創新。

---

### **Temenos Transact 主要功能與特性**

#### **1. 客戶與帳戶管理**
- 客戶開戶與檔案管理
- KYC 與合規檢查
- 多幣種帳戶管理
- 存款與儲蓄帳戶服務

#### **2. 支付與交易**
- 即時支付（SWIFT、SEPA、RTGS、ACH、UPI 等）
- 跨境支付
- 電子錢包與 P2P 轉帳
- 支付流程協調與處理

#### **3. 貸款與信貸服務**
- 貸款發起、審批與管理
- 按揭與個人貸款處理
- 信貸風險評估與評分
- BNPL（先買後付）與微型貸款選項

#### **4. 資金與風險管理**
- 流動性與現金流管理
- 外匯與衍生產品交易
- 資產負債管理（ALM）
- 市場與信貸風險分析

#### **5. 財富與投資管理**
- 投資組合與資產管理
- 智能投顧與投資規劃
- 託管與證券交易

#### **6. 開放銀行與 API 整合**
- 用於無縫整合的 RESTful 與 SOAP API
- 第三方金融科技與監管科技支援
- 開放銀行合規（PSD2、英國開放銀行）

#### **7. 合規與監管支援**
- 反洗錢（AML）與詐騙偵測
- 巴塞爾協議 III 與國際財務報告準則合規
- 即時監管匯報

#### **8. 雲端與 SaaS 功能**
- 雲端原生與多雲端支援（AWS、Azure、Google Cloud）
- AI 驅動的銀行營運自動化
- DevOps 與 CI/CD 整合以加速部署

---

### **部署模式**
- **本地部署** – 安裝於銀行內部基礎設施
- **雲端（SaaS/PaaS）** – 託管於 Temenos Banking Cloud
- **混合模式** – 結合本地與雲端部署
