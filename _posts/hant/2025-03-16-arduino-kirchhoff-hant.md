使用 Arduino 驗證基爾霍夫定律——特別是基爾霍夫電流定律（KCL）和基爾霍夫電壓定律（KVL）——是將電子學理論與動手實驗相結合的絕佳方式。KCL 指出流入節點的總電流等於流出節點的總電流，而 KVL 則指出封閉迴路中的電壓降總和等於供應電壓。以下將介紹如何設計簡單電路，並使用 Arduino 測量電流和電壓來驗證這些定律。

由於 Arduino 無法直接測量電流，我們將通過測量電阻兩端的電壓（使用歐姆定律：\\( I = V/R \\))來推斷電流，並可透過其類比引腳（0-5V 範圍）測量電壓。下面我將概述兩個實驗——一個用於 KCL，一個用於 KVL——包含逐步說明、接線方式和程式碼。

### **實驗 1：驗證基爾霍夫電流定律（KCL）**

#### **目標**
證明流入節點的電流等於流出該節點的電流。

#### **電路設置**
- **元件：**
  - Arduino（例如 Uno）
  - 3 個電阻（例如 R1 = 330Ω，R2 = 470Ω，R3 = 680Ω）
  - 麵包板和跳線
  - 5V 電源（Arduino 的 5V 引腳）
- **接線：**
  - 將 Arduino 5V 連接到一個節點（稱為節點 A）。
  - 從節點 A，將 R1 連接到 GND（分支 1）。
  - 從節點 A，將 R2 連接到 GND（分支 2，與 R1 並聯）。
