要使用你的第二部筆記型電腦（運行 Ubuntu 並搭載 Nginx 伺服器）在顯示器上觀看電影，且不想太麻煩的話，以下是基於你現有設備的最佳方案。你想避免購買額外設備如 Fire TV Stick 或 Apple TV，雖然長 HDMI 線也能用，但你更希望無需持續操作筆電來選擇和播放影片。讓我們用現有設備建立一個簡單方便的解決方案。

既然你的電影儲存在 Ubuntu 筆電上，且已運行 Nginx 伺服器，最簡單的方法就是透過 HDMI 將筆電直接連接到顯示器，並讓播放控制變得輕鬆。以下是具體步驟：

### 解決方案：連接筆電至顯示器並使用 VLC 搭配遠端控制

#### 所需設備
- **HDMI 線**：用於連接 Ubuntu 筆電與顯示器。
- **VLC 媒體播放器**：可能已預裝在 Ubuntu 中，或可輕鬆安裝。
- **智慧型手機或其他設備**：透過家庭網路遠端控制播放。

#### 設定步驟
1. **將筆電連接至顯示器**
   - 將 HDMI 線一端插入 Ubuntu 筆電的 HDMI 埠，另一端插入顯示器的 HDMI 輸入埠。
   - 開啟顯示器並選擇正確的 HDMI 輸入源。筆電的畫面現在應顯示在顯示器上，包括視訊和音訊（若顯示器自帶揚聲器；否則請使用筆電揚聲器或連接外部揚聲器）。

2. **安裝 VLC（若尚未安裝）**
   - 在 Ubuntu 筆電上開啟終端機，執行：
     ```
     sudo apt update
     sudo apt install vlc
     ```
   - 這將確保 VLC 這款多功能媒體播放器已準備就緒。

3. **啟用 VLC 網頁介面以實現遠端控制**
   - 在 Ubuntu 筆電上開啟 VLC。
   - 前往 **工具 > 偏好設定**。
   - 在左下角點擊 **「全部」** 以顯示進階設定。
   - 導覽至 **介面 > 主介面**，勾選 **「網頁」** 以啟用 HTTP 介面。
   - 前往 **介面 > 主介面 > Lua**，在 **Lua HTTP 密碼** 欄位中設定密碼（例如 "mypassword"）。
   - 點擊 **儲存**，然後重新啟動 VLC。

4. **在 VLC 中載入電影**
   - 在 VLC 中，前往 **播放 > 播放清單**。
   - 將電影檔案從資料夾（儲存在筆電中的位置）拖放至 VLC 播放清單，或使用 **媒體 > 開啟檔案** 逐個新增。
   - 透過 **Ctrl+Y** 儲存播放清單（例如 "我的電影"），方便日後快速存取。

5. **查詢筆電的 IP 位址**
   - 在終端機中輸入：
     ```
     ip addr show
     ```
   - 尋找網路連線下的 IP 位址（例如 Wi-Fi 的 `wlan0` 下方顯示 `192.168.1.100`）。這是手機連接筆電的位址。

6. **從手機控制播放**
   - 確保手機與筆電處於同一 Wi-Fi 網路。
   - 在手機上開啟網頁瀏覽器，輸入：`http://<筆電-ip>:8080`（例如 `http://192.168.1.100:8080`）。
   - 出現提示時，使用者名稱留空，輸入你設定的密碼（例如 "mypassword"）。
   - 你將看到 VLC 的網頁介面，可用來播放、暫停、停止或從播放清單選擇下一部電影。

7. **開始觀看**
   - 在筆電上，於 VLC 中開始播放電影（雙擊播放清單中的項目）。
   - 將 VLC 切換至全螢幕模式（**檢視 > 全螢幕** 或按下 `F`）。
   - 輕鬆坐好，使用手機控制播放，無需觸碰筆電。

#### 為何此方案適合你
- **無額外花費**：使用現有的筆電、顯示器和手機，無需添購新設備。
- **極簡操作**：初始設定後，只需開啟筆電和顯示器，打開 VLC，即可用手機控制一切。
- **本地播放**：由於電影儲存在連接顯示器的筆電上，無需透過網路串流（雖然已設定 Nginx，但此處不需使用）。VLC 直接播放檔案，確保流暢效能。

#### 可選改進
- **無線控制**：若你有無線滑鼠或鍵盤，可替代手機使用，但手機在遠距離操作時更為方便。
- **自動啟動**：為進一步簡化操作，設定 VLC 在筆電啟動時自動載入播放清單：
  - 將 VLC 與你的播放清單新增至 Ubuntu 的 **啟動應用程式**（在 Ubuntu 中搜尋「啟動應用程式」，點擊「新增」，輸入：`vlc /你的播放清單路徑/playlist.m3u`）。
- **音訊檢查**：若顯示器無揚聲器，請確保音訊從筆電輸出，或透過筆電的音訊插孔連接外部揚聲器。

### 替代方案：使用 Nginx 和瀏覽器（若偏好）
若你更傾向使用 Nginx 伺服器：
- 在 Nginx 目錄中設定一個簡單的 HTML 頁面（例如 `/var/www/html/movies`），包含電影檔案的連結（例如 `<a href="movie1.mp4">電影 1</a>`）。
- 將筆電連接至顯示器，開啟瀏覽器，前往 `http://localhost/movies`。
- 點擊連結在瀏覽器中播放電影，並使用全螢幕模式。
- 然而，此方法不易實現手機控制，除非你從手機存取同一頁面並進行投射，這可能需要額外設定或硬體。

### 最終建議
VLC 方案是最佳選擇。它簡單直接，充分利用現有設備，讓你透過手機控制一切，實現「開啟顯示器即可輕鬆觀看」的體驗。完成一次設定後，你只需啟動 VLC 並用手機選擇電影，既簡單又高效！
