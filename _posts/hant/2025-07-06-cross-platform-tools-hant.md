很高興你在尋找跨平台工具，特別是考慮到 Ubuntu/Linux！許多現代應用程式的優點在於它們能夠在 Windows、macOS 和各種 Linux 發行版之間無縫運行。除了 Firefox 和 VS Code 之外，以下是一些優秀的跨平台工具和應用程式，按功能分類：

**1. 網頁瀏覽器（除了 Firefox）：**

* **Google Chrome：** 快速、安全，並擁有龐大的擴充套件庫。
* **Brave：** 專注於隱私保護，預設阻擋廣告和追蹤器，並內建 VPN。
* **Opera：** 提供獨特功能，如免費內建 VPN、廣告攔截器和側邊欄整合。

**2. 開發工具（除了 VS Code）：**

* **Sublime Text：** 以其速度和簡潔性聞名，是許多開發者的最愛。
* **Git：** 版本控制的必備工具，其命令列工具可在任何地方使用。GUI 客戶端如 **GitKraken** 或 **GitHub Desktop** 也是跨平台的。
* **Docker：** 用於容器化，讓你能夠打包應用程式及其依賴項，並在不同環境中一致地運行。
* **跨平台開發框架（用於構建應用程式）：**
    * **Flutter：**（Google）使用 Dart，非常適合從單一代碼庫構建美觀的行動、網頁和桌面應用程式。
    * **React Native：**（Meta）使用 JavaScript，用於構建原生渲染的行動應用程式。
    * **Xamarin (.NET MAUI)：**（Microsoft）使用 C#，適合與 .NET 生態系統整合。
    * **Ionic：** 使用網頁技術（HTML、CSS、JavaScript）開發混合型行動應用程式。
    * **Kotlin Multiplatform：**（JetBrains）允許共享業務邏輯代碼，同時保持平台特定的 UI。

**3. 生產力與辦公套件：**

* **LibreOffice：** 免費開源的辦公套件，是 Microsoft Office 的強大替代品，包括文書處理、試算表、簡報等。
* **Google 文件、試算表、簡報：** 主要基於網頁，但在任何瀏覽器中都能無縫工作，並提供離線功能。
* **Microsoft 365（網頁版）：** 雖然桌面應用程式主要針對 Windows/macOS，但 Word、Excel 和 PowerPoint 的網頁版在 Linux 的瀏覽器中功能齊全。
* **Obsidian：** 強大的筆記和知識庫應用程式，使用 Markdown 並允許在筆記之間進行廣泛連結。
* **Joplin：** 開源的筆記和待辦事項應用程式，支援 Markdown 並可與各種雲端服務同步。
* **Standard Notes：** 開源、加密的筆記應用程式，專注於隱私和安全性。
* **OneNote（Microsoft）：** 免費筆記應用程式，具有數位筆記本佈局，可透過網頁應用程式或非官方客戶端在 Linux 上使用。
* **Everdo：** 一款 GTD（Getting Things Done）應用程式，專注於隱私和離線優先使用。
* **Xournal++：** 非常適合手寫筆記，特別是如果你有觸控筆/平板電腦，支援 PDF 註釋和各種繪圖工具。

**4. 通訊與協作：**

* **Discord：** 在社群和團隊中很受歡迎，支援語音、視訊和文字聊天。
* **Telegram：** 安全且快速的訊息應用程式，支援大型群組聊天和檔案分享。
* **Signal：** 以其強大的端到端加密和隱私功能聞名。
* **Slack：** 廣泛使用的團隊訊息和協作應用程式。
* **Element（Matrix）：** 開源、去中心化的通訊平台，非常適合注重隱私的用戶。
* **Jami：** 免費開源的去中心化視訊會議和聊天工具。
* **Skype：** 最古老的 VoIP 和訊息應用程式之一，仍在 Linux 上受支援。
* **Beeper：** 一款通用聊天應用程式，旨在將所有聊天網絡（WhatsApp、Messenger、Telegram 等）整合到一個介面中。
* **LocalSend：** 開源、跨平台的 AirDrop 替代方案，用於在沒有網路連線的情況下進行本地檔案分享。

**5. 媒體播放器與管理：**

* **VLC Media Player：** 「什麼都能播放」的媒體播放器，支援幾乎所有音訊和視訊格式。
* **Kodi：** 強大的開源媒體中心，用於組織和串流你的媒體庫。
* **Spotify：** 流行的音樂串流服務，提供 Linux 桌面客戶端。
* **Audacity：** 強大的開源音訊編輯工具，也可作為播放器使用。
* **Plex：** 媒體伺服器，讓你能將內容串流到任何裝置，並提供適用於各種平台的客戶端應用程式。
* **SMPlayer：** 適用於 Windows 和 Linux 的免費媒體播放器，內建編解碼器，非常適合播放各種格式並恢復播放。
* **mpv：** 免費開源的命令列媒體播放器，以高品質視訊輸出和自訂性聞名。

**6. 雲端儲存與同步：**

* **Google Drive：** 提供慷慨的免費儲存空間，並透過瀏覽器與 Google 辦公工具良好整合。
* **Dropbox：** 簡單可靠的檔案儲存和分享解決方案。
* **Nextcloud：** 自託管替代方案，讓你完全控制自己的數據。
* **OneDrive：** 微軟的雲端儲存解決方案，在 Linux、Windows 和 Mac 上運行良好。

**7. 設計與圖形：**

* **GIMP：** 免費開源的 Photoshop 替代品，用於影像編輯。
* **Krita：** 專為數位繪畫和插圖設計。
* **Inkscape：** 向量圖形編輯器，非常適合標誌和插圖。
* **Darktable：** 專業級色彩校正的照片編輯應用程式。

**8. 實用工具：**

* **7-Zip：** 輕量級檔案壓縮工具，支援多種格式。
* **KeePassXC：** 安全、開源的密碼管理器，用於管理你的憑證。
* **OBS Studio：** 強大的螢幕錄製和直播串流工具。
* **Calibre：** 適用於所有平台的電子書管理工具。
* **Etcher：** 用於建立可開機 USB 隨身碟的簡單工具。

選擇時，請務必查看官方網站以獲取最新的 Linux 安裝說明，因為有些可能提供 AppImage、Flatpak、Snap 或傳統的 `.deb` 或 `.rpm` 套件。享受你的跨平台工作流程吧！
