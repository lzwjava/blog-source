主要概念與公式速查指南。重點關注矩陣、行列式、線性系統、向量空間、變換、特徵值及二次型——這些都是過往試題的核心主題。請複習定義、性質及計算方法。

## 1. 矩陣
- **定義**：\\( A = [a_{ij}] \in \mathbb{R}^{m \times n} \\)。
- **運算**：
  - 加法：\\( (A + B)_{ij} = a_{ij} + b_{ij} \\)。
  - 純量乘法：\\( (cA)_{ij} = c a_{ij} \\)。
  - 乘法：\\( (AB)_{ij} = \sum_k a_{ik} b_{kj} \\)（若尺寸相容）。
- **轉置**：\\( (A^T)_{ij} = a_{ji} \\)；\\( (AB)^T = B^T A^T \\)，\\( (A^T)^T = A \\)。
- **逆矩陣**（方陣）：\\( AA^{-1} = I \\)；\\( (AB)^{-1} = B^{-1} A^{-1} \\)；\\( (A^T)^{-1} = (A^{-1})^T \\)。
- **類型**：
  - 對角矩陣：僅對角線上有非零元素。
  - 上/下三角矩陣：對角線以下/以上為零。
  - 對稱矩陣：\\( A = A^T \\)。
  - 正交矩陣：\\( A^T A = I \\)（列向量為單位正交）。

## 2. 行列式 (det A)
- **性質**：
  - \\( \det(AB) = \det A \cdot \det B \\)；\\( \det(A^T) = \det A \\)；\\( \det(cA) = c^n \det A \\)。
  - 行/列交換：乘以 -1。
  - 加上行/列的倍數：不變。
  - 行/列乘以 c：乘以 c。
  - \\( \det I = 1 \\)；若為奇異矩陣（行/列線性相關）則 \\( \det A = 0 \\)。
- **計算**：
  - 2x2：\\( \det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc \\)。
  - 餘因子展開（第 i 行）：\\( \det A = \sum_j a_{ij} C_{ij} \\)，其中 \\( C_{ij} = (-1)^{i+j} M_{ij} \\)（餘子式行列式）。
  - 三角矩陣：對角線元素的乘積。
- **伴隨矩陣/逆矩陣**：\\( A^{-1} = \frac{1}{\det A} \adj A \\)，其中 \\( \adj A = C^T \\)（餘因子矩陣的轉置）。
- **克萊姆法則**（對於 \\( Ax = b \\)，det A ≠ 0）：\\( x_i = \frac{\det A_i}{\det A} \\)（A_i 為將第 i 列替換為 b 的矩陣）。

## 3. 線性系統 (Ax = b)
- **高斯消去法**：將增廣矩陣 [A | b] 化為階梯形式/簡化階梯形式。
  - 階梯形式：主元（首個 1）向右下階梯狀排列；主元以下為零。
  - 回代求解唯一解。
- **解的情況**：
  - 唯一解：rank A = n（滿列秩），零空間為 {0}。
  - 無限多解：rank A = rank [A|b] < n（存在自由變量）。
  - 無解：rank A < rank [A|b]。
- **完整解**：特解 + 零空間基（齊次解）。
- **LU 分解**（無行交換）：A = LU（L 為單位下三角矩陣，U 為上三角矩陣）；解 Ly = b，再解 Ux = y。
- **最小平方法**（超定系統）：\\( \hat{x} = (A^T A)^{-1} A^T b \\)（若為滿秩）。

## 4. 向量空間與子空間
- **向量空間**：對加法/純量乘法封閉；滿足公理（例如：零向量、逆元素）。
- **子空間**：向量的生成空間；封閉且包含 0。
  - 列空間：Col(A) = A 的列向量的生成空間；維度 = rank A。
  - 行空間：Row(A) = Col(A^T)；維度 = rank A。
  - 零空間：Nul(A) = {x | Ax = 0}；維度 = n - rank A。
  - 左零空間：Nul(A^T)。
- **線性獨立**：c1 v1 + ... + ck vk = 0 ⇒ 所有 ci = 0。
- **基**：線性獨立的生成集。
- **維度**：基中的向量數；dim Col(A) + dim Nul(A) = n（秩-零化度定理）。
- **秩**：主元列的數量 = dim Col(A) = dim Row(A)。

## 5. 線性變換
- **定義**：T: V → W 為線性若 T(u + v) = T u + T v，T(cu) = c T u。
- **矩陣表示**：相對於基的 [T] = A，其中 T(x) = A x（標準基）。
- **核**：Ker T = Nul(A)；像：Im T = Col(A)。
- **同構**：一對一且滿射（可逆矩陣）。
- **秩-零化度定理**：dim Ker T + dim Im T = dim V。

## 6. 特徵值與特徵向量
- **定義**：A v = λ v（v ≠ 0 為特徵向量，λ 為特徵值）。
- **特徵方程式**：det(A - λ I) = 0；根 λi（代數重數）。
- **特徵向量**：解 (A - λ I) v = 0；幾何重數 = 特徵空間的維度。
- **可對角化**：n 個線性獨立特徵向量 ⇒ A = X Λ X^{-1}（Λ 為 diag(λi)，X = [v1 ... vn]）。
  - 對稱矩陣 A：總可對角化；具正交特徵向量（A = Q Λ Q^T，Q 為正交矩陣）。
- **跡**：tr A = ∑ λi。
- **行列式**：det A = ∏ λi。
- **相似矩陣**：若 A = P B P^{-1} 則 A ~ B；具相同特徵值、跡、行列式。

## 7. 內積與二次型
- **內積**：<u, v> = u^T v（歐幾里得）；||v|| = √<v,v>。
- **正交**：<u,v> = 0；單位正交基：<ei, ej> = δij。
- **Gram-Schmidt 正交化**：將基 {v1,...,vn} 正交化為 {u1,...,un}。
  - u1 = v1；uk = vk - proj_{span(u1..u_{k-1})} vk；proj_w v = (<v,w>/<w,w>) w。
- **二次型**：q(x) = x^T A x（A 為對稱矩陣）。
  - 正定：對所有 x ≠ 0，q(x) > 0（所有 λi > 0）。
  - 對角化：q(x) = ∑ λi yi^2（y = Q^T x）。

## 速查提示
- 計算秩：行化簡求主元數。
- 檢查可對角化：每個 λ 的幾何重數 = 代數重數。
- 考試準備：練習行化簡、行列式展開、2x2/3x3 矩陣的特徵問題。
- 常見錯誤：忽略重數；餘因子符號錯誤；乘法不可交換。
