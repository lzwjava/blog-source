要在 Safari 中顯示控制台，請按照以下步驟進行：

1. 在你的 Mac 上打開 Safari。
2. 到頂部的菜單欄，點擊 **Safari** > **設定** (如果你使用的是較舊版本，則為 **偏好設定**)。
3. 點擊 **進階** 選項卡。
4. 在底部，勾選標有 **“在菜單欄中顯示開發選單”** 的方框。
5. 關閉設定視窗。
6. 現在，在菜單欄中，你會看到一個新的 **開發** 選項。點擊 **開發** > **顯示 JavaScript 控制台** (或 **顯示網頁檢查器**，視你的版本而定)。

或者，你可以在啟用開發選單後，使用鍵盤快捷鍵 **Option + Command + C** (⌥ + ⌘ + C) 迅速打開控制台。

如果你在 iPhone 或 iPad 上，Safari 並不會原生顯示控制台，但你可以通過將你的裝置通過 USB 連接到 Mac，在 Mac 上打開 Safari，並使用開發選單來檢查 iOS 裝置的瀏覽器來啟用它。
