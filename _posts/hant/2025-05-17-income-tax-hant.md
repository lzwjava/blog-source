以下是針對2025年中國個人所得稅累計預扣預繳方法的精要說明，旨在提升清晰度與理解性。

---

對於中國的工薪階層而言，個人所得稅通常由僱主採用**累計預扣法**按月計算並預扣。這種方法在計算每月應納稅額時，會考慮您當年截至當月的累計收入和扣除額，從而有助於更平穩地在全年分攤稅負。

累計預扣法並非簡單地對每月收入單獨徵稅，而是計算您*截至當月*的累計收入應繳納的總稅額，然後減去當年已在前幾個月繳納的稅款。其結果即為當月應預扣的稅額。

以下是關鍵概念和計算方式的細分：

**關鍵概念：**

*   **累計收入：** 您從當年1月1日起至當月末，從當前僱主處獲得的工資薪金總收入。
*   **累計基本減除費用：** 這是標準的免稅門檻。2025年為每月5000元人民幣。累計基本減除費用計算為5000元乘以您在當前納稅年度內為當前僱主工作的月數（截至當月）。
*   **累計專項扣除：** 強制性繳款，如社會保險（養老、醫療、失業）和住房公積金，這些是從您當年截至當月的工資中扣除的。
*   **累計專項附加扣除：** 這些是您可以就特定支出申請的可選扣除項，按當年累計計算。常見例子包括：
    *   子女教育
    *   繼續教育
    *   大病醫療
    *   住房貸款利息
    *   住房租金
    *   贍養老人
*   **累計依法確定的其他扣除：** 法律認可的其他符合資格的扣除，例如某些商業健康保險費，按當年累計計算。
*   **累計應納稅所得額：** 應稅金額，計算方式為從您的累計收入中減去所有累計扣除額。
*   **累計應預扣預繳稅額：** 使用累進稅率表（見下表）根據您的累計應納稅所得額計算出的總稅額。
*   **累計已預扣預繳稅額：** 在當年之前月份，已由您的僱主預扣並繳納給稅務機關的個人所得稅總額。

**計算公式：**

1.  **計算累計應納稅所得額：**
    累計預扣預繳應納稅所得額 = 累計收入 - 累計免稅收入 - 累計基本減除費用 - 累計專項扣除 - 累計專項附加扣除 - 累計依法確定的其他扣除

2.  **計算當月應納稅額：**
    本期應預扣預繳稅額 = (累計預扣預繳應納稅所得額 × 預扣率 - 速算扣除數) - 累計減免稅額 - 累計已預扣預繳稅額

**案例分析：計算張三2025年的個人所得稅**

我們以張三為例，他是一名月收入波動但扣除額穩定的員工：

*   **月收入：** 1月：15,000元；2月：45,000元；3月：15,000元。
*   **穩定的每月扣除額：**
    *   基本減除費用：5,000元（標準）
    *   專項扣除（五險一金）：3,000元
    *   專項附加扣除：
        *   子女教育：1,000元
        *   住房貸款利息：1,000元
        *   贍養老人（獨生子女）：2,000元
        *   *每月專項附加扣除總計：1000 + 1000 + 2000 = 4,000元*
    *   其他扣除（商業健康保險）：200元

**分月計算步驟：**

**2025年1月（第1個月）**

*   **累計收入：** 15,000元
*   **累計扣除額：**
    *   基本：5000 * 1 = 5,000
    *   專項：3000 * 1 = 3,000
    *   專項附加：4000 * 1 = 4,000
    *   其他：200 * 1 = 200
    *   *累計扣除總額：* 5000 + 3000 + 4000 + 200 = 12,200元
*   **累計應納稅所得額：** 15,000 (收入) - 12,200 (扣除額) = 2,800元
*   **累計應預扣預繳稅額：**
    *   累計應納稅所得額 (2800) 處於第1級 (< 36000)。
    *   應納稅額 = 2800 * 3% - 0 (速算扣除數) = 84元。
*   **累計已預扣預繳稅額：** 0元（這是第一個月）
*   **當月應納稅額（1月）：** 84 (累計應納稅額) - 0 (已繳稅額) = **84元**

**2025年2月（第2個月）**

*   **累計收入：** 15,000 (1月) + 45,000 (2月) = 60,000元
*   **累計扣除額：**
    *   基本：5000 * 2 = 10,000
    *   專項：3000 * 2 = 6,000
    *   專項附加：4000 * 2 = 8,000
    *   其他：200 * 2 = 400
    *   *累計扣除總額：* 10000 + 6000 + 8000 + 400 = 24,400元
*   **累計應納稅所得額：** 60,000 (收入) - 24,400 (扣除額) = 35,600元
*   **累計應預扣預繳稅額：**
    *   累計應納稅所得額 (35,600) 仍處於第1級 (< 36000)。
    *   應納稅額 = 35,600 * 3% - 0 (速算扣除數) = 1,068元。
*   **累計已預扣預繳稅額：** 84元（來自1月）
*   **當月應納稅額（2月）：** 1068 (累計應納稅額) - 84 (已繳稅額) = **984元**

**2025年3月（第3個月）**

*   **累計收入：** 60,000 (2月) + 15,000 (3月) = 75,000元
*   **累計扣除額：**
    *   基本：5000 * 3 = 15,000
    *   專項：3000 * 3 = 9,000
    *   專項附加：4000 * 3 = 12,000
    *   其他：200 * 3 = 600
    *   *累計扣除總額：* 15000 + 9000 + 12000 + 600 = 36,600元
*   **累計應納稅所得額：** 75,000 (收入) - 36,600 (扣除額) = 38,400元
*   **累計應預扣預繳稅額：**
    *   累計應納稅所得額 (38,400) 現處於第2級 (> 36000, < 144000)。
    *   應納稅額 = 38,400 * 10% - 2520 (速算扣除數) = 3840 - 2520 = 1,320元。
*   **累計已預扣預繳稅額：** 84 (1月) + 984 (2月) = 1,068元
*   **當月應納稅額（3月）：** 1320 (累計應納稅額) - 1068 (已繳稅額) = **252元**

從這個案例可以看出，即使月收入波動，每月預扣的稅額也可能因累計收入及其在全年度稅階中的變化而有所不同。累計預扣法確保在年度結束時，預扣的總稅額與基於全年總收入和扣除額計算的年度納稅義務密切匹配。
