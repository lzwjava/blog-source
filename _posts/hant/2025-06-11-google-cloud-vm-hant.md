在 Google Cloud 上尋找符合成本效益的虛擬機器（VM）時，有幾個選項和注意事項需要考慮，以滿足您每月約 5 美元的預算。以下是一些步驟和提示，協助您找到最經濟實惠的 VM 選項：

1. **共享核心虛擬機器**：Google Cloud 提供共享核心虛擬機器，這是最經濟實惠的選項。這些虛擬機器與其他客戶共享 CPU 資源，雖然效能可能較不穩定，但價格明顯更為低廉。對於資源需求較低的簡單伺服器來說，這可能是個不錯的起點。

2. **可搶佔虛擬機器**：考慮使用可搶佔虛擬機器，這類虛擬機器相比常規 VM 提供大幅折扣。當 Google Cloud 需要將資源調配給其他用途時，可能會終止這些虛擬機器，因此最適合用於容錯能力較高的工作負載。可搶佔虛擬機器最高可提供 80% 的折扣優惠。

3. **Tau 虛擬機器**：Google Cloud 的 Tau 虛擬機器專為橫向擴展工作負載設計，相比通用型虛擬機器可提供高達 42% 的性價比提升。對於某些類型的工作負載而言，這可能是更經濟高效的選擇。

4. **自訂機器類型**：您可以根據特定需求建立自訂機器類型，僅支付所需資源的費用來降低成本。這種靈活性讓您能精細調整 CPU 和記憶體配置，以符合工作負載需求。

5. **持續使用折扣**：對於在計費月份內運行時間達到一定比例的虛擬機器，Google Cloud 會自動提供持續使用折扣。折扣幅度隨使用時長增加，最高可享常規價格 30% 的優惠。

6. **免費層級**：Google Cloud 提供免費層級服務，每月包含少量運算資源。對於非常輕量級的應用程式來說可能已經足夠，並能進一步幫助降低成本。

7. **費用計算機**：使用 Google Cloud 定價計算機來估算不同配置的成本，並找到最符合需求的經濟實惠方案。

8. **現成虛擬機器**：現成虛擬機器採用動態定價模式，對大多數機器類型可提供按需價格 60-91% 的折扣。如果您的工作負載能夠容忍中斷情況，這將是非常具成本效益的選擇。

對於資源需求較低的簡單伺服器，從共享核心或可搶佔虛擬機器開始著手可能是最佳途徑。您可以後續監控效能表現，並根據需求調整配置，以在成本與效能之間取得平衡。
