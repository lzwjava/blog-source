# TabsKiller

介紹一個 Chrome 插件，當瀏覽器打開太多標籤時，會自動關閉最舊的標籤。告別亂七八糟的瀏覽器體驗，永遠！

一個神奇的 Chrome 插件，當打開網頁過多的時候，會自動關掉最老的網頁，讓瀏覽器保持清爽！

# 特性

1. 當標籤數量超過設置的限制時，自動關閉最舊的標籤。
2. 可以根據個人的喜好自訂最大標籤數量（x）。
3. 可以設置特定的 URL 模式鎖定，以確保符合該模式的標籤即使在標籤過多時也保持打開。

# 特性

1. 會自動關掉最老的網頁，當打開的網頁超過一定數量的時候。
2. 可以設置最大的標籤數量。
3. 可以設置鎖定規則，使得滿足這個規則的網頁不被關閉。

## 故事

我通常在 Chrome 中打開許多標籤。所以我按Ctrl + W 來關掉很多標籤一次。重複重複。所以我想寫一個擴展來解決我的問題。然後我找到了“Tab Wrangler”擴展，它會在 x 分鐘內不活動時關閉標籤。我從中學習並製作了一個擴展，當標籤數量超過 x 個時關閉最舊的標籤。並且鎖定一些符合特定 URL 模式的標籤。它幫助了我。我再也不需要關心標籤了。標籤從來沒有變得太多。它讓我的生活變得更輕鬆。希望你也會喜歡它。

## 安裝

請前往 Chrome 商店 https://chrome.google.com/webstore/detail/tabs-killer/hgmdeeoighmhomddlghfjcidkdcpbllf/，
或手動安裝：

首先，前往 `chrome://extensions`，然後選擇開發者模式，再加載已解壓的擴展程序。完成。

![chrome](https://cloud.githubusercontent.com/assets/5022872/10262586/ddc451b0-6a00-11e5-8b10-da16c9658221.jpg)

更多詳細信息可以在 Chrome Get Started [tutorial](https://developer.chrome.com/extensions/getstarted#unpacked) 中找到。
