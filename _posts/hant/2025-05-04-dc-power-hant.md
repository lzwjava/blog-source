## 1. 直流電源供應簡介
**直流電源供應**將交流電轉換為直流電，為微控制器、感測器與積體電路等電子裝置供電。轉換過程通常包含：
- **整流**：將交流電轉為脈動直流電
- **濾波**：平滑化脈動直流電
- **穩壓**：穩定輸出電壓或電流

直流電源供應是電子設備的關鍵，確保裝置獲得穩定低噪聲的電力。本章將重點說明**整流/濾波電路**與**電壓穩壓器**（線性與開關式）兩大核心元件。

## 2. 整流與濾波電路

整流電路負責將交流電轉換為直流電，濾波器則透過平滑化輸出降低漣波。以下詳細解析：

### a. 半波整流器
**半波整流器**是最簡易的整流電路，僅使用單一二極體。

#### 運作原理
- **輸入**：交流電壓（例如來自變壓器）
- **運作**：二極體僅在交流波形正半週期導通，阻斷負半週期
- **輸出**：脈動直流電，頻率與輸入交流電相同，僅包含正半週期（依二極體方向可能為負半週期）

#### 電路圖
```
交流源 ----> 二極體 (D1) ----> 負載 (R) ----> 接地
```
- **元件**：
  - **二極體**：例如 1N4007（通用整流二極體）
  - **負載**：電阻或電子電路

#### 特性
- **輸出電壓**：約為 \\( V_{out} = V_{in(峰值)} - V_{二極體} \\)（矽二極體 \\( V_{二極體} \approx 0.7V \\)）
- **效率**：偏低（約 40.6%），僅利用一半交流週期
- **漣波**：較高，因輸出為間歇性

#### 優點
- 結構簡單且成本低廉
- 所需元件極少

#### 缺點
- 效率不佳（浪費一半交流週期）
- 漣波較高，需大型濾波器才能獲得平滑直流電

---

### b. 全波整流器（橋式配置）
**全波整流器**同時利用交流輸入的正負半週期，產生更穩定的直流輸出。

#### 運作原理
- **配置**：採用四顆二極體組成**橋式整流器**
- **運作**：
  - 正半週期間，兩顆二極體導通，引導電流流經負載
  - 負半週期間，另兩顆二極體導通，維持相同電流方向通過負載
- **輸出**：脈動直流電，頻率為輸入交流電的兩倍

#### 電路圖
```
       交流輸入
         ------
        |      |
   D1 --|-->|--|-->|-- D2
        |      |       |
        |      R       |
        |      |       |
   D3 --|<--|--|<--|-- D4
        |      |
         ------
        接地
```
- **元件**：
  - **二極體**：四顆二極體（例如 1N4007）
  - **負載**：電阻或電路
  - **變壓器**（選配）：降低交流電壓

#### 特性
- **輸出電壓**：\\( V_{out} = V_{in(峰值)} - 2V_{二極體} \\)（兩顆二極體同時導通，矽二極體壓降約 1.4V）
- **效率**：較半波整流高（約 81.2%）
- **漣波**：較半波整流低，因每個週期產生兩次脈衝

#### 優點
- 效率更高，完整利用交流週期
- 漣波較小，所需濾波器體積更小

#### 缺點
- 結構較複雜（需四顆二極體）
- 電壓降稍高（因兩顆二極體導通）

---

### c. 濾波電路
整流器產生的脈動直流電因存在**漣波**（電壓波動）而不適用於多數電子設備。濾波器可平滑輸出，近似穩定直流電。

#### 常見濾波器：電容濾波器
**電容濾波器**是最常用的濾波方式，與負載並聯連接。

#### 運作原理
- **充電**：在整流波形峰值期間，電容充電至峰值電壓
- **放電**：當整流電壓下降時，電容透過負載放電，維持較穩定電壓
- **結果**：獲得漣波減少的平滑直流電

#### 電路圖（全波整流配電容濾波器）
```
       交流輸入
         ------
        |      |
   D1 --|-->|--|-->|-- D2
        |      |       |
        |      R       C (電容)
        |      |       |
   D3 --|<--|--|<--|-- D4
        |      |
         ------
        接地
```
- **元件**：
  - **電容**：數值取決於負載電流與漣波容忍度（例如中等負載可用 1000µF）
  - **負載**：電阻或電路

#### 漣波計算
漣波電壓 (\\( V_r \\)) 可近似計算為：
\\[ V_r \approx \frac{I_{負載}}{f \cdot C} \\]
其中：
- \\( I_{負載} \\)：負載電流（A）
- \\( f \\)：整流輸出頻率（例如 60Hz 交流電全波整流為 120Hz）
- \\( C \\)：電容值（F）

#### 範例
負載電流 100mA，電容 1000µF，頻率 120Hz：
\\[ V_r \approx \frac{0.1}{120 \cdot 1000 \times 10^{-6}} \approx 0.833V \\]
此漣波對某些應用可接受，但可透過更大電容或額外濾波（例如 LC 濾波器）進一步降低

#### 其他濾波器
- **電感濾波器**：串聯電感以抑制電流急遽變化
- **LC 濾波器**：結合電感與電容實現更佳漣波抑制
- **π型濾波器**：電容-電感-電容配置，可產生極平滑直流電

## 3. 線性穩壓器 vs 開關穩壓器

經過整流與濾波後的直流電壓，仍可能因輸入變化或負載需求而波動。**電壓穩壓器**可穩定輸出，主要分為**線性式**與**開關式**兩大類。

### a. 線性穩壓器
線性穩壓器透過將多餘功率以熱能形式消散，提供穩定輸出電壓。

#### 運作原理
- 如同可變電阻，透過調整電阻值維持恆定輸出電壓
- 要求輸入電壓必須高於預期輸出電壓（壓差電壓）

#### 範例：7805 線性穩壓器
**7805**是經典線性穩壓器，提供固定 5V 輸出。

#### 電路圖
```
輸入電壓 ----> [7805] ----> 輸出電壓 (5V)
       |         |
      C1        C2
       |         |
      接地       接地
```
- **元件**：
  - **7805 IC**：輸出 5V（搭配適當散熱片可達 1A）
  - **電容**：C1（0.33µF）與 C2（0.1µF）用於穩定運作
  - **輸入電壓**：通常 7-12V（必須 >5V + 壓差電壓，約 2V）

#### 特性
- **輸出**：固定式（如 7805 輸出 5V）或可調式（如 LM317）
- **效率**：偏低，因多餘電壓以熱能消散（\\( 效率 \approx \frac{V_{輸出}}{V_{輸入}} \\)）
- **噪聲**：低噪聲，適合精密類比電路

#### 優點
- 設計簡單，易於實作
- 輸出噪聲低，適用於音頻與精密電路
- 成本低廉

#### 缺點
- 效率不佳，特別在輸入輸出壓差大時更明顯
- 會產生熱量，高電流時需加裝散熱片
- 僅能降壓（輸出 < 輸入）

---

### b. 開關穩壓器
開關穩壓器透過高頻開關控制能量傳輸，實現高效率轉換。

#### 運作原理
- 開關元件（通常為 MOSFET）快速啟閉，控制流經電感與電容的能量
- 回授電路調整開關責任週期以維持穩定輸出

#### 開關穩壓器類型
1. **降壓型**：降低電壓（例如 12V 轉 5V）
2. **升壓型**：提升電壓（例如 5V 轉 12V）
3. **升降壓型**：可升壓或降壓（例如 9V 轉 5V 或 12V）

#### 電路圖（降壓轉換器範例）
```
輸入電壓 ----> 開關 (MOSFET) ----> 電感 ----> 輸出電壓
       |                      |
      二極體                 電容
       |                      |
      接地                    接地
```
- **元件**：
  - **MOSFET**：控制開關動作
  - **電感**：在「開啟」週期儲存能量
  - **電容**：平滑輸出
  - **二極體**：在「關閉」週期提供電感電流路徑
  - **控制器 IC**：例如 LM2596（可調降壓轉換器）

#### 特性
- **效率**：高效率（80-95%），因極少功率以熱能消散
- **噪聲**：較線性穩壓器高，因開關動作產生
- **彈性**：可升壓、降壓或兩者兼具

#### 優點
- 高效率，適合電池供電裝置
- 結構緊湊，散熱片體積更小
- 功能多元（降壓、升壓或升降壓配置）

#### 缺點
- 設計較複雜，需電感與精密佈局
- 開關噪聲可能干擾敏感電路
- 因額外元件導致成本較高

## 4. 實務範例與電路設計

### 範例 1：半波整流搭配線性穩壓器的 5V 直流電源供應
**目標**：以 9V 交流變壓器設計 5V 直流電源
**步驟**：
1. **整流**：使用 1N4007 二極體進行半波整流
2. **濾波**：加入 1000µF 電容平滑輸出
3. **穩壓**：使用 7805 穩壓器獲得穩定 5V 輸出

**電路**：
```
9V 交流 ----> 1N4007 ----> 1000µF ----> 7805 ----> 5V
                     |             |        |
                    接地           C1       C2
                                   |        |
                                   接地      接地
```
- **C1**：0.33µF（輸入穩定）
- **C2**：0.1µF（輸出穩定）

**注意事項**：
- 變壓器整流後需提供 >7V 直流電（9V 交流已足夠）
- 若負載電流超過 500mA，需為 7805 加裝散熱片

---

### 範例 2：全波整流搭配開關穩壓器的 5V 直流電源供應
**目標**：以 12V 交流變壓器設計高效能 5V 電源
**步驟**：
1. **整流**：使用橋式整流器（四顆 1N4007 二極體）
2. **濾波**：加入 2200µF 電容
3. **穩壓**：使用 LM2596 降壓轉換器

**電路**：
```
12V 交流 ----> 橋式整流器 ----> 2200µF ----> LM2596 ----> 5V
                         |                       |
                        接地                     接地
```
- **LM2596**：透過回授電阻設定為 5V 輸出
- **電容**：依 LM2596 資料手冊選擇輸入/輸出電容

**注意事項**：
- 確保電感選用符合 LM2596 資料手冊規範
- 若應用於噪聲敏感環境，需加入 EMI 濾波

## 5. 應用與注意事項

### 應用場景
- **半波整流器**：低成本、低功率裝置（例如簡易電池充電器）
- **全波整流器**：電子設備通用電源供應
- **線性穩壓器**：音頻電路、精密感測器與低功耗裝置
- **開關穩壓器**：筆記型電腦、智慧手機、LED 驅動器與電池供電系統

### 設計考量
- **負載需求**：確保電源可承受最大電流
- **效率**：高功率應用優先選用開關穩壓器
- **噪聲**：敏感電路應選用線性穩壓器或追加濾波
- **熱管理**：需配置散熱片或適當 PCB 佈局以利散熱
- **元件規格**：選擇符合電壓/電流規格的二極體、電容與穩壓器

## 6. 結論
直流電源供應是電子裝置供電的關鍵，理解其組成元件——整流器、濾波器與穩壓器——對設計可靠系統至關重要。**半波整流器**結構簡單但效率不彰，**全波整流器**則提供更佳效能。**線性穩壓器**適合低噪聲應用但能耗較高，而**開關穩壓器**能為現代高耗電裝置提供高效電能轉換。透過巧妙組合這些元件，可根據特定需求打造穩健的電源供應解決方案。

建議透過麵包板實作或 LTspice 等模擬軟體實際建置範例電路，嘗試更換不同電容值或穩壓器類型，親身觀察其對效能的影響。
