GitHub Desktop 版本 3.4.13 (arm64) 在文字差異面板中對特定語言的字體渲染出現問題。受影響的語言包括：

- 中文（簡體與繁體）
- 印地語
- 日語

然而，以下語言的渲染則正常：
- 英語
- 西班牙語
- 德語
- 阿拉伯語
- 法語
