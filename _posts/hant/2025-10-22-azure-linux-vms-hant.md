是的，Azure 虛擬機器 (VM) 完全支援運行 Linux 作業系統。您可以直接從 Azure Marketplace 部署多種熱門發行版，或上傳自訂的 Linux 映像。

### 重點摘要：
- **支援的發行版**：Azure 認可並支援主流 Linux 發行版，包括 Ubuntu、Red Hat Enterprise Linux (RHEL)、Debian、SUSE Linux Enterprise Server (SLES)、Oracle Linux、CentOS Stream、Rocky Linux、AlmaLinux 等。現成提供超過 100 種 Linux 映像。
- **部署選項**：可使用 Azure 入口網站、CLI、PowerShell 或 ARM 範本建立 Linux VM。例如，您能在數分鐘內快速啟動 Ubuntu Server 實例。
- **功能特色**：Azure 上的 Linux VM 完整整合 Azure 服務，包括儲存、網路、安全性和監控。同時支援 Azure Arc 等混合雲管理工具。

若您剛開始使用，請查閱建立 Linux VM 的官方快速入門指南。
