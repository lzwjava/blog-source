ये सभी Python पैकेज (wheels) हैं जिन्हें इंस्टॉल किया जा रहा है—यहाँ प्रत्येक का एक संक्षिप्त विवरण दिया गया है:

* **safetensors**: टेंसर डेटा को सेव/लोड करने के लिए pickle सुरक्षा समस्याओं के बिना एक तेज़, सुरक्षित फ़ाइल फॉर्मेट।
* **typer**: Python type hints के साथ CLI एप्लिकेशन बनाने के लिए एक आधुनिक लाइब्रेरी।
* **shellingham**: यह पता लगाने के लिए यूटिलिटी कि आप कौन सा शेल (bash, zsh, आदि) चला रहे हैं।
* **uvicorn**: Python वेब ऐप्स (जैसे FastAPI) होस्ट करने के लिए एक बहुत तेज़ ASGI सर्वर।
* **httptools**: लो-लेवल HTTP पार्सिंग लाइब्रेरी, गति के लिए Uvicorn जैसे सर्वर द्वारा उपयोग की जाती है।
* **uvloop**: Cython में लागू एक अति-तेज़ asyncio इवेंट लूप, asyncio के लिए ड्रॉप-इन रिप्लेसमेंट।
* **watchfiles**: फ़ाइल-वॉचिंग लाइब्रेरी जो फाइलसिस्टम परिवर्तनों पर एक्शन (जैसे ऑटो-रिलोड) ट्रिगर कर सकती है।
* **airportsdata**: एयरपोर्ट कोड, नाम और स्थानों के लिए सरल लुकअप टेबल।
* **astor**: Python ASTs को वापस सोर्स कोड में बदलने के टूल (और इसके विपरीत)।
* **blake3**: BLAKE3 क्रिप्टोग्राफ़िक हैश एल्गोरिदम के लिए Python बाइंडिंग।
* **cloudpickle**: Python ऑब्जेक्ट्स के लिए एन्हांस्ड पिकलर, stdlib pickle की तुलना में अधिक प्रकारों को हैंडल करता है।
* **cupy\_cuda12x**: CUDA 12.x के लिए NumPy-संगत GPU ऐरे लाइब्रेरी।
* **fastrlock**: Python के लिए एक बहुत तेज़ री-एंट्रेंट लॉक इम्प्लीमेंटेशन।
* **diskcache**: Python ऑब्जेक्ट्स को कुशलतापूर्वक परसिस्ट करने के लिए एक डिस्क-बैक्ड कैश लाइब्रेरी।
* **einops**: एक्सप्रेसिव, पठनीय टेंसर रीशेपिंग और रीऑर्डरिंग ऑपरेशन।
* **importlib\_metadata**: रनटाइम पर पैकेज मेटाडेटा (वर्जन, एंट्री-पॉइंट्स) तक पहुंच (पुराने Python के लिए बैकपोर्ट)।
* **zipp**: importlib\_metadata द्वारा उपयोग किए जाने वाले ZIP आर्काइव के लिए pathlib-स्टाइल इंटरफेस।
* **msgspec**: हाई-परफॉर्मेंस मैसेज सीरियलाइजेशन/डी-सीरियलाइजेशन (JSON, Msgpack, आदि)।
* **nest\_asyncio**: asyncio इवेंट लूप्स के नेस्टिंग की अनुमति देता है (जैसे नोटबुक चलाने के लिए)।
* **ninja**: Ninja बिल्ड सिस्टम (तेज़, इंक्रीमेंटल बिल्ड) के लिए Python बाइंडिंग।
* **partial\_json\_parser**: बड़े या मालफॉर्म्ड JSON के लिए स्ट्रीमिंग/आंशिक JSON पार्सर।
* **py\_cpuinfo**: CPU के बारे में विस्तृत जानकारी (मॉडल, कोर, फ्रीक्वेंसी) एकत्र करता है।
* **python\_json\_logger**: Python के लॉगिंग मॉड्यूल के लिए JSON फॉर्मेटर।
* **pyzmq**: ZeroMQ हाई-परफॉर्मेंस मैसेजिंग लाइब्रेरी के लिए Python बाइंडिंग।
* **scipy**: कोर साइंटिफिक कंप्यूटिंग लाइब्रेरी (ऑप्टिमाइजेशन, इंटीग्रेशन, स्टैटिस्टिक्स, आदि)।
* **outlines\_core**: "आउटलाइन्स" लाइब्रेरी के लिए कोर यूटिलिटीज (डॉक्यूमेंट आउटलाइन/स्ट्रक्चर को हैंडल करती है)।
* **xformers**: Meta से एफिशिएंट, मॉड्यूलर ट्रांसफॉर्मर (अटेंशन) इम्प्लीमेंटेशन।
* **sentencepiece**: अनसुपरवाइज्ड टेक्स्ट टोकनाइज़र/डीटोकनाइज़र (आमतौर पर NLP में उपयोग किया जाता है)।
