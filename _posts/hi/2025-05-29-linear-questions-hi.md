**18. मान लीजिए कि आव्यूह X और A संबंध XA = X + A को संतुष्ट करते हैं। यदि $A = \begin{pmatrix} 1 & -3 & 0 \\\ 2 & 1 & 0 \\\ 0 & 0 & 2 \end{pmatrix}$ है, तो आव्यूह X ज्ञात कीजिए।**

**उत्तर:**
$X = \begin{pmatrix} 1 & 1/2 & 0 \\\ -1/3 & 1 & 0 \\\ 0 & 0 & 2 \end{pmatrix}$

**19. k के उस मान का निर्धारण कीजिए जिसके लिए सदिश $\alpha_1 = (1, 1, k)^T$, $\alpha_2 = (1, k, 1)^T$, $\alpha_3 = (k, 1, 1)^T$ रैखिक रूप से निर्भर हैं। एक अधिकतम रैखिक रूप से स्वतंत्र समुच्चय ज्ञात कीजिए और शेष सदिश(ों) को इस समुच्चय के रैखिक संयोजन के रूप में व्यक्त कीजिए।**

**उत्तर:**
सदिश रैखिक रूप से निर्भर हैं यदि $k=1$ या $k=-2$।

स्थिति 1: $k=1$
$\alpha_1 = (1, 1, 1)^T$, $\alpha_2 = (1, 1, 1)^T$, $\alpha_3 = (1, 1, 1)^T$
एक अधिकतम रैखिक रूप से स्वतंत्र समुच्चय $\{\alpha_1\}$ है।
तब $\alpha_2 = 1 \cdot \alpha_1$ और $\alpha_3 = 1 \cdot \alpha_1$।

स्थिति 2: $k=-2$
$\alpha_1 = (1, 1, -2)^T$, $\alpha_2 = (1, -2, 1)^T$, $\alpha_3 = (-2, 1, 1)^T$
एक अधिकतम रैखिक रूप से स्वतंत्र समुच्चय $\{\alpha_1, \alpha_2\}$ है।
तब $\alpha_3 = -1 \cdot \alpha_1 - 1 \cdot \alpha_2$।

**20. रैखिक समीकरणों की निम्नलिखित प्रणाली को हल कीजिए { $x_1 - x_2 - 3x_4 = -2$ ; $x_1 + 2x_3 - 2x_4 = -1$ ; $2x_1 - 2x_2 + x_3 - 6x_4 = -5$ ; $-x_1 + 2x_2 + 3x_3 + 4x_4 = 2$ } (संगत समांगी प्रणाली के लिए एक विशेष हल और हलों का मौलिक आधार ज्ञात कीजिए)।**

**उत्तर:**
सामान्य हल है:
$x_1 = 1 + 2t$
$x_2 = 3 - t$
$x_3 = -1$
$x_4 = t$
सदिश रूप में: $X = \begin{pmatrix} 1 \\\ 3 \\\ -1 \\\ 0 \end{pmatrix} + t \begin{pmatrix} 2 \\\ -1 \\\ 0 \\\ 1 \end{pmatrix}$।

एक विशेष हल है: $X_p = \begin{pmatrix} 1 \\\ 3 \\\ -1 \\\ 0 \end{pmatrix}$ (जब $t=0$)।

समांगी प्रणाली के लिए हलों का मौलिक आधार है: $\{\begin{pmatrix} 2 \\\ -1 \\\ 0 \\\ 1 \end{pmatrix}\}$।

**21. मान लीजिए कि आव्यूह A = \begin{pmatrix} x & 0 & y \\\ 0 & 2 & 0 \\\ y & 0 & -2 \end{pmatrix} का एक अभिलक्षणिक मान -3 है, और |A| = -12 है। x और y के मान ज्ञात कीजिए।**

**उत्तर:**
$x=1$ और $y=2$, या $x=1$ और $y=-2$।

**22. मान लीजिए कि 3-चर द्विघात रूप $f(x_1, x_2, x_3) = t(x_1^2 + x_2^2 + x_3^2) + 2x_1x_2 + 2x_1x_3 - 2x_2x_3$ है। t के किस/किन मान/मानों के लिए यह द्विघात रूप धनात्मक निश्चित है, यह निर्धारित कीजिए।**

**उत्तर:**
द्विघात रूप धनात्मक निश्चित है जब $t > 2$।
