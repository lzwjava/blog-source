हाँ, यह सही है। Claude Code, Anthropic का टर्मिनल-आधारित कोडिंग टूल, दो प्राथमिक लॉगिन विधियाँ प्रदान करता है:

- **सब्सक्रिप्शन के साथ Claude अकाउंट**: यह आपके मौजूदा Claude.ai सब्सक्रिप्शन (Pro के लिए $20/माह या Max के लिए $100/माह से शुरू) का उपयोग करता है, जो पूर्वानुमेय मूल्य निर्धारण प्रदान करता है और उन प्लानों में एक्सेस शामिल होता है।
- **Anthropic Console अकाउंट**: यह API उपयोग के आधार पर बिलिंग करता है, जिसके लिए आपको पे-एज़-यू-गो टोकन खपत के लिए Anthropic Console में बिलिंग सेट अप करनी होती है।
