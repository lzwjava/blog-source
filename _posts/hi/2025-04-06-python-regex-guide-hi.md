### `re` मॉड्यूल को इम्पोर्ट करना
`re` मॉड्यूल (रेगुलर एक्सप्रेशन्स के लिए संक्षिप्त) Python की स्टैंडर्ड लाइब्रेरी का हिस्सा है। यह एक अलग "रेजेक्स पैकेज" नहीं बल्कि एक बिल्ट-इन मॉड्यूल है। इसे अपनी स्क्रिप्ट के शीर्ष पर इम्पोर्ट करें:

```python
import re
```

### बेसिक कॉन्सेप्ट्स
रेगुलर एक्सप्रेशन्स (रेजेक्स) पैटर्न होते हैं जिनका उपयोग स्ट्रिंग्स को मैच करने, खोजने और मैनिपुलेट करने के लिए किया जाता है। वे विशेष कैरेक्टर्स का उपयोग करते हैं जैसे:
- `.` (कोई भी कैरेक्टर)
- `*` (शून्य या अधिक घटनाएँ)
- `+` (एक या अधिक घटनाएँ)
- `[]` (कैरेक्टर क्लासेज, उदा., `[a-z]` लोअरकेस लेटर्स के लिए)
- `^` (स्ट्रिंग की शुरुआत), `$` (स्ट्रिंग का अंत)

उदाहरण Python के `re` डॉक्यूमेंटेशन पर आधारित हैं। कुशलता के लिए हमेशा जटिल पैटर्न को `re.compile()` के साथ कम्पाइल करें।

### मुख्य फंक्शन्स और उपयोग

#### 1. **शुरुआत में पैटर्न मैच करना**
   - `re.match(pattern, string)`: जाँचता है कि क्या पैटर्न स्ट्रिंग की शुरुआत से मेल खाता है।
   - एक मैच ऑब्जेक्ट या `None` रिटर्न करता है।

   उदाहरण:
   ```python
   import re
   result = re.match(r"Hello", "Hello world")
   print(result.group())  # आउटपुट: Hello
   ```

#### 2. **स्ट्रिंग में कहीं भी खोजना**
   - `re.search(pattern, string)`: पैटर्न की पहली घटना को कहीं भी ढूँढता है।
   - एक मैच ऑब्जेक्ट या `None` रिटर्न करता है।

   उदाहरण:
   ```python
   result = re.search(r"world", "Hello world")
   print(result.group())  # आउटपुट: world
   ```

#### 3. **सभी मैच ढूँढना**
   - `re.findall(pattern, string)`: सभी नॉन-ओवरलैपिंग मैचेस की एक लिस्ट रिटर्न करता है।

   उदाहरण:
   ```python
   matches = re.findall(r"\d+", "I have 3 cats and 42 dogs")
   print(matches)  # आउटपुट: ['3', '42']
   ```

#### 4. **स्ट्रिंग्स को स्प्लिट करना**
   - `re.split(pattern, string)`: पैटर्न की घटनाओं पर स्ट्रिं
