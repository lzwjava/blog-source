JShell जावा डेवलपमेंट किट (JDK) में शामिल एक इंटरएक्टिव REPL (रीड-ईवल-प्रिंट लूप) टूल है जो JDK 9 से उपलब्ध है। यह आपको पूरा प्रोग्राम लिखे बिना जावा कोड स्निपेट को इंटरएक्टिव तरीके से एक्ज़िक्यूट करने की सुविधा देता है। JShell के उपयोग का संक्षिप्त गाइड नीचे दिया गया है:

### 1. **JShell शुरू करना**
- **JDK इंस्टॉल करें**: सुनिश्चित करें कि आपके पास JDK 9 या बाद का वर्जन इंस्टॉल है। टर्मिनल या कमांड प्रॉम्प्ट में `java -version` चलाकर चेक कर सकते हैं।
- **JShell लॉन्च करें**:
  - टर्मिनल या कमांड प्रॉम्प्ट खोलें।
  - `jshell` टाइप कर Enter दबाएं।
  - आपको `jshell>` जैसा प्रॉम्प्ट दिखाई देगा जो बताता है कि JShell तैयार है।

### 2. **बेसिक उपयोग**
- **जावा कोड चलाना**:
  - प्रॉम्प्ट पर सीधे जावा एक्सप्रेशन, स्टेटमेंट या डिक्लेरेशन टाइप करें।
  - उदाहरण:
    ```java
    jshell> System.out.println("Hello, JShell!")
    Hello, JShell!
    ```
  - JShell इनपुट को एवल्यूएट करता है और तुरंत रिजल्ट दिखाता है।

- **वेरिएबल और एक्सप्रेशन**:
  - वेरिएबल डिक्लेयर करें या एक्सप्रेशन एवल्यूएट करें:
    ```java
    jshell> int x = 10
    x ==> 10
    jshell> x * 2
    $2 ==> 20
    ```
  - JShell एक्सप्रेशन रिजल्ट को स्वचालित रूप से अस्थायी नाम देता है (जैसे `$2`)।

- **मेथड और क्लास डिफाइन करना**:
  - आप मेथड या क्लास डिफाइन कर सकते हैं:
    ```java
    jshell> void sayHello() { System.out.println("Hello!"); }
    |  created method sayHello()
    jshell> sayHello()
    Hello!
    ```
    ```java
    jshell> class Test { int x = 5; }
    |  created class Test
    jshell> Test t = new Test()
    t ==> Test@7c417213
    jshell> t.x
    $5 ==> 5
    ```

### 3. **मुख्य कमांड**
JShell `/` से शुरू होने वाले बिल्ट-इन कमांड प्रदान करता है। कुछ आवश्यक कमांड यहां दिए गए हैं:
- **सभी कोड लिस्ट करें**: `/list` – सेशन में दर्ज किए गए सभी स्निपेट दिखाता है।
  ```java
  jshell> /list
  ```
- **कोड एडिट करें**: `/edit <id>` – दिए गए ID ( `/list` से) वाले स्निपेट के लिए GUI एडिटर खोलता है।
- **सेशन सेव करें**: `/save myfile.java` – सभी स्निपेट को एक फाइल में सेव करता है।
- **फाइल लोड करें**: `/open myfile.java` – फाइल से कोड लोड और एक्ज़िक्यूट करता है।
- **वेरिएबल देखें**: `/vars` – सभी डिक्लेयर किए गए वेरिएबल की लिस्ट दिखाता है।
  ```java
  jshell> /vars
  |    int x = 10
  ```
- **मेथड देखें**: `/methods` – सभी डिफाइन की गई मेथड की लिस्ट दिखाता है।
- **JShell से बाहर निकलें**: `/exit` – JShell सेशन बंद करता है।
- **मदद**: `/help` – सभी उपलब्ध कमांड डिस्प्ले करता है।

### 4. **पैकेज इम्पोर्ट करना**
- JShell सामान्य पैकेज (जैसे `java.util`, `java.io`) स्वचालित रूप से इम्पोर्ट करता है। अन्य का उपयोग करने के लिए, उन्हें मैन्युअली इम्पोर्ट करें:
  ```java
  jshell> import java.time.LocalDate
  jshell> LocalDate.now()
  $3 ==> 2025-06-27
  ```

### 5. **कोड एडिटिंग और करेक्शन**
- **मौजूदा कोड मॉडिफाई करें**:
  - स्निपेट का ID ढूंढने के लिए `/list` का उपयोग करें।
  - नया वर्जन टाइप करके उसे रिडिफाइन करें। JShell पुरानी डेफिनिशन को ओवरराइट कर देता है।
    ```java
    jshell> int x = 5
    x ==> 5
    jshell> int x = 10
    x ==> 10
    ```
- **स्निपेट ड्रॉप करें**: `/drop <id>` – किसी विशिष्ट स्निपेट को उसके ID द्वारा हटाता है।

### 6. **टैब कम्पलीशन**
- क्लास नाम, मेथड या कमांड को ऑटो-कम्पलीट करने के लिए `Tab` दबाएं।
- उदाहरण:
  ```java
  jshell> System.out.pr<tab>
  ```
  यह `println`, `print`, आदि सजेस्ट करेगा।

### 7. **एक्सटर्नल स्क्रिप्ट चलाना**
- जावा फाइल लोड और रन करें:
  ```java
  jshell> /open MyScript.java
  ```
- आप JShell को स्क्रिप्ट के साथ भी शुरू कर सकते हैं:
  ```bash
  jshell MyScript.java
  ```

### 8. **JShell कस्टमाइजेशन**
- **स्टार्टअप स्क्रिप्ट**: इम्पोर्ट या कॉमन कोड के साथ एक फाइल (जैसे `startup.jsh`) बनाएं, फिर उसे `/open startup.jsh` से लोड करें या डिफॉल्ट स्टार्टअप स्क्रिप्ट के रूप में सेट करें:
  ```bash
  jshell --start startup.jsh
  ```
- **फीडबैक मोड**: `/set feedback <mode>` (जैसे `normal`, `concise`, `verbose`) के साथ आउटपुट वर्बोसिटी एडजस्ट करें।

### 9. **टिप्स**
- **क्विक प्रोटोटाइपिंग**: छोटे कोड स्निपेट टेस्ट करने, APIs के साथ एक्सपेरिमेंट करने या जावा सिंटैक्स सीखने के लिए JShell का उपयोग करें।
- **एरर हैंडलिंग**: JShell क्लियर एरर मैसेज प्रदान करता है और गलतियों को तुरंत सुधारने देता है।
- **मल्टी-लाइन इनपुट**: कॉम्प्लेक्स कोड के लिए, JShell मल्टी-लाइन एडिटिंग सपोर्ट करता है। लाइन को बैकस्लैश (`\`) के साथ समाप्त करें जारी रखने के लिए:
  ```java
  jshell> int sum(int a, int b) { \
      ...> return a + b; \
      ...> }
  ```

### 10. **ट्रबलशूटिंग**
- **JShell नहीं मिल रहा**: सुनिश्चित करें कि JDK `bin` डायरेक्टरी आपके सिस्टम के PATH में है।
- **स्लो स्टार्टअप**: कुछ सिस्टम पर, JShell को इनिशियलाइज होने में थोड़ा समय लग सकता है। सुनिश्चित करें कि आपका JDK अप टू डेट है।
- **सेशन रीसेट**: सभी स्निपेट क्लियर करने और नए सिरे से शुरू करने के लिए `/reset` का उपयोग करें।

अधिक जानकारी के लिए, ऑफिशियल JDK डॉक्युमेंटेशन चेक करें या JShell में `/help` चलाएं। यदि आपके पास JShell में ट्राई करने के लिए कोई विशिष्ट टास्क या कोड है, तो मुझे बताएं, और मैं आपको आगे मार्गदर्शन कर सकता हूं!
