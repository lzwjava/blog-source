### मुख्य वेब फ्रेमवर्क और यूटिलिटीज़
- **Flask==2.0.3**: पायथन में वेब एप्लिकेशन और एपीआई बनाने के लिए हल्का WSGI वेब फ्रेमवर्क। रूटिंग, टेम्प्लेटिंग और रिक्वेस्ट हैंडलिंग प्रदान करता है।
- **click==8.0.4**: कमांड-लाइन इंटरफेस (CLI) बनाने के लिए लाइब्रेरी; अक्सर Flask के साथ स्क्रिप्ट्स या CLI टूल्स के लिए उपयोग किया जाता है।
- **gunicorn==20.1.0**: प्रोडक्शन में Flask एप्स को डिप्लॉय करने के लिए WSGI HTTP सर्वर; समवर्तीता (concurrency) के लिए एकाधिक वर्कर्स का समर्थन करता है।
- **Werkzeug==2.0.3**: व्यापक WSGI यूटिलिटी लाइब्रेरी; Flask की रिक्वेस्ट/रिस्पांस हैंडलिंग, डीबगिंग और रूटिंग को शक्ति प्रदान करती है।
- **Jinja2==3.0.3**: Flask एप्स में डायनामिक HTML/टेम्प्लेट रेंडर करने के लिए टेम्प्लेटिंग इंजन।
- **itsdangerous==2.0.1**: डेटा को सुरक्षित रूप से साइन और क्रमबद्ध करने में मदद करता है (जैसे, टोकन, कुकीज़) छेड़छाड़ को रोकने के लिए।
- **MarkupSafe==2.0.1**: XSS को रोकने के लिए Jinja2 टेम्प्लेट्स में सुरक्षित HTML आउटपुट के लिए स्ट्रिंग्स को एस्केप करता है।
- **python-dotenv==0.19.2**: कॉन्फ़िगरेशन प्रबंधन के लिए `.env` फ़ाइल से पर्यावरण चर (environment variables) को `os.environ` में लोड करता है।

### REST API और एक्सटेंशन
- **flask-restx==0.5.1**: Flask के लिए एक्सटेंशन जो Swagger/OpenAPI सपोर्ट, इनपुट/आउटपुट वैलिडेशन और RESTful API बनाने के लिए नेमस्पेसिंग जोड़ता है।
- **Flask-Cors==3.0.10**: Flask API में क्रॉस-डोमेन रिक्वेस्ट्स की अनुमति देने के लिए Cross-Origin Resource Sharing (CORS) हेडर्स को संभालता है।
- **Flask-JWT-Extended==4.4.1**: प्रमाणीकरण (authentication) के लिए JSON Web Tokens (JWT) का प्रबंधन करता है; एक्सेस/रिफ्रेश टोकन, ब्लैकलिस्टिंग और क्लेम्स का समर्थन करता है।
- **aniso8601==9.0.1**: ISO 8601 डेट/टाइम स्ट्रिंग्स को पार्स करता है; API डॉक्स/मॉडल्स में datetime को संभालने के लिए flask-restx द्वारा उपयोग किया जाता है।
- **jsonschema==4.0.0**: JSON डेटा को JSON Schema परिभाषाओं के विरुद्ध वैलिडेट करता है; API पेलोड वैलिडेशन के लिए flask-restx के साथ एकीकृत होता है।

### डेटाबेस और ORM
- **Flask-SQLAlchemy==2.5.1**: SQLAlchemy ORM को Flask के साथ एकीकृत करता है; डेटाबेस इंटरैक्शन, मॉडल और सत्रों को सरल बनाता है।
- **SQLAlchemy==1.4.46**: डेटाबेस एब्स्ट्रक्शन, क्वेरीिंग और माइग्रेशन के लिए SQL टूलकिट और ऑब्जेक्ट रिलेशनल मैपर (ORM)।
- **greenlet==2.0.1**: ग्रीन थ्रेड्स के लिए हल्के कोरोटीन्स (coroutines); SQLAlchemy द्वारा async सपोर्ट के लिए आवश्यक (हालांकि यहां उपयोग नहीं किया गया)।
- **Flask-Migrate**: Alembic का उपयोग करके डेटाबेस स्कीमा माइग्रेशन को संभालने के लिए एक्सटेंशन; Flask-SQLAlchemy के साथ एकीकृत होता है।
- **pytz==2022.6**: टाइमज़ोन परिभाषाएं और हैंडलिंग प्रदान करता है; SQLAlchemy/Flask द्वारा टाइमज़ोन-अवेयर डेटटाइम के लिए उपयोग किया जाता है।

### HTTP और नेटवर्किंग
- **requests==2.27.1**: API कॉल करने के लिए सरल HTTP क्लाइंट (जैसे, OpenAI/Anthropic जैसी बाहरी सेवाओं के लिए)।
- **certifi==2022.12.7**: requests में SSL/TLS कनेक्शन सत्यापित करने के लिए रूट सर्टिफिकेट का संग्रह।
- **charset-normalizer~=2.0.0**: टेक्स्ट में कैरेक्टर एन्कोडिंग का पता लगाता है; रिस्पांस डिकोडिंग के लिए requests द्वारा उपयोग किया जाता है।
- **idna==3.4**: URL हैंडलिंग के लिए एप्लिकेशन में इंटरनेशनलाइज्ड डोमेन नेम (IDNA) का समर्थन करता है।
- **urllib3==1.26.13**: कनेक्शन पूलिंग और SSL के साथ HTTP क्लाइंट लाइब्रेरी; requests के लिए अंतर्निहित इंजन।
