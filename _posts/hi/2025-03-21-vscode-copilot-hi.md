Visual Studio Code (VS Code) में GitHub Copilot का उपयोग करना सीधा-सादा है एक बार सेटअप हो जाने के बाद। 20 मार्च, 2025 तक, आरंभ करने के लिए नीचे एक चरण-दर-चरण मार्गदर्शिका दी गई है:

---

### चरण 1: VS Code इंस्टॉल करें
- सुनिश्चित करें कि आपके पास Visual Studio Code इंस्टॉल है। यदि नहीं है, तो इसे [code.visualstudio.com](https://code.visualstudio.com/) से डाउनलोड करें।
- इष्टतम संगतता के लिए VS Code को नवीनतम संस्करण में अपडेट करें (`Help > Check for Updates` पर जाएं)।

---

### चरण 2: GitHub Copilot एक्सटेंशन इंस्टॉल करें
1.  **VS Code खोलें**।
2.  **एक्सटेंशन मार्केटप्लेस पर जाएं**:
    - बाईं ओर एक्टिविटी बार में एक्सटेंशन आइकन पर क्लिक करें (या Mac पर `Ctrl+Shift+X` / `Cmd+Shift+X` दबाएं)।
3.  **"GitHub Copilot" खोजें**:
    - सर्च बार में "GitHub Copilot" टाइप करें।
    - GitHub द्वारा आधिकारिक एक्सटेंशन देखें (इसमें एक सत्यापित बैज होगा)।
4.  **एक्सटेंशन इंस्टॉल करें**:
    - "GitHub Copilot" के आगे `Install` बटन पर क्लिक करें।
5.  **वैकल्पिक: Copilot Chat इंस्टॉल करें (अनुशंसित)**:
    - "GitHub Copilot Chat" खोजें और इसे भी इंस्टॉल करें। यह चैट के माध्यम से प्रश्न पूछने या कोड जनरेट करने जैसी वार्तालाप AI सुविधाएँ जोड़ता है।

---

### चरण 3: GitHub Copilot में साइन इन करें
1.  **GitHub के साथ प्रमाणीकरण करें**:
    - इंस्टॉलेशन के बाद, साइन इन करने के लिए कहते हुए एक प्रॉम्प्ट दिखाई देगा।
    - पॉप-अप में `Sign in to GitHub` पर क्लिक करें या Copilot स्टेटस आइकन (VS Code के निचले-दाएं कोने में) पर जाएं और "Sign in" चुनें।
2.  **ब्राउज़र में अधिकृत करें**:
    - एक ब्राउज़र विंडो खुलेगी जो आपसे आपके GitHub अकाउंट में लॉग इन करने के लिए कहेगी।
    - `Authorize Git hypoxia` पर क्लिक करके अधिकृतीकरण अनुरोध को स्वीकार करें।
3.  **कोड कॉपी करें**:
    - GitHub एक वन-टाइम कोड प्रदान करेगा। इसे कॉपी करें और VS Code में वापस पेस्ट करें जब प्रॉम्प्ट किया जाए।
4.  **सक्रियण सत्यापित करें**:
    - एक बार साइन इन होने के बाद, स्टेटस बार में Copilot आइकन हरा हो जाना चाहिए, जो इंगित करता है कि यह सक्रिय है। आपको अपने एक्सेस की पुष्टि करने वाला एक नोटिफिकेशन भी दिखाई देगा।

---

### चरण 4: Copilot कॉन्फ़िगर करें (वैकल्पिक)
- **सुझाव सक्षम/अक्षम करें**:
  - `File > Preferences > Settings` (या `Ctrl+,` / `Cmd+,`) पर जाएं।
  - इनलाइन सुझाव सक्षम करने या विशिष्ट भाषाओं के लिए इसे अक्षम करने जैसी सेटिंग्स को समायोजित करने के लिए "Copilot" खोजें।
- **सब्सक्रिप्शन जांचें**:
  - Copilot के लिए 30-दिन के ट्रायल के बाद एक सब्सक्रिप्शन ($10/माह या $100/वर्ष) की आवश्यकता होती है। छात्र, शिक्षक और ओपन-सोर्स मेंटेनर [GitHub Education](https://education.github.com/) या Copilot सेटिंग्स के माध्यम से मुफ्त एक्सेस के लिए आवेदन कर सकते हैं।

---

### चरण 5: Copilot का उपयोग शुरू करें
अपने कोडिंग वर्कफ़्लो में Copilot का लाभ उठाने का तरीका यहां बताया गया है:

#### 1. **कोड सुझाव**
- **इनलाइन ऑटोकम्प्लीट**:
  - एक फ़ाइल में टाइप करना शुरू करें (उदाहरण के लिए, Python में `def calculate_sum(`), और Copilot ग्रे टेक्स्ट में पूरा होने के सुझाव देगा।
  - सुझाव को स्वीकार करने के लिए `Tab` दबाएं या इसे नज़रअंदाज करने के लिए टाइप करते रहें।
- **मल्टी-लाइन सुझाव**:
  - `// Function to sort an array` जैसी एक टिप्पणी लिखें और Enter दबाएं। Copilot एक संपूर्ण कार्यान्वयन (उदाहरण के लिए, एक सॉर्टिंग एल्गोरिदम) का सुझाव दे सकता है।
  - कई सुझावों के बीच चक्रित करने के लिए `Alt+]` (या Mac पर `Option+]`) का उपयोग करें।

#### 2. **टिप्पणियों से कोड जनरेशन**
- एक वर्णनात्मक टिप्पणी टाइप करें जैसे:
  ```javascript
  // Fetch data from an API and handle errors
  ```
  Enter दबाएं, और Copilot जनरेट कर सकता है:
  ```javascript
  async function fetchData(url) {
    try {
      const response = await fetch(url);
      if (!response.ok) throw new Error('Network response was not ok');
      return await response.json();
    } catch (error) {
      console.error('Fetch error:', error);
    }
  }
  ```
  - `Tab` के साथ स्वीकार करें या आवश्यकतानुसार संशोधित करें।

#### 3. **Copilot Chat (यदि इंस्टॉल है)**
- **चैट खोलें**:
  - साइडबार में चैट आइकन पर क्लिक करें या `Ctrl+Alt+C` (कस्टमाइज़ेबल) का उपयोग करें।
- **प्रश्न पूछें**:
  - "Explain how Promises work in JavaScript" या "Write a Python script to read a CSV file." जैसा कुछ टाइप करें।
  - Copilot चैट पैनल में जवाब देगा और सीधे आपके एडिटर में कोड डाल सकता है।
- **प्रासंगिक सहायता**:
  - कोड को हाइलाइट करें, राइट-क्लिक करें, और इसे समझाने या रिफैक्टर करने के लिए “Ask Copilot” चुनें।

#### 4. **डिबगिंग और टेस्टिंग**
- `// Write unit tests for this function` जैसी एक टिप्पणी लिखें, और Copilot Jest या PyTest जैसे फ्रेमवर्क का उपयोग करके टेस्ट केस जनरेट कर सकता है।

---

### प्रभावी उपयोग के लिए सुझाव
- **विशिष्ट बनें**: जितना अधिक संदर्भ आप प्रदान करेंगे (जैसे टिप्पणियाँ, वेरिएबल नाम), Copilot के सुझाव उतने ही बेहतर होंगे।
- **सुझावों की समीक्षा करें**: Copilot परफेक्ट नहीं है—सटीकता और सुरक्षा के लिए हमेशा इसके कोड की जांच करें।
- **भाषा समर्थन**: यह लोकप्रिय भाषाओं (Python, JavaScript, Java, आदि) के साथ सबसे अच्छा काम करता है लेकिन दर्जनों अन्य का समर्थन करता है।
- **कीबोर्ड शॉर्टकट**:
  - `Alt+\` (या Mac पर `Option+\`): सभी सुझावों को मैन्युअल रूप से दिखाएं।
  - `Ctrl+Enter`: सुझाव पैनल खोलें।

### समस्या निवारण
- **काम नहीं कर रहा?**:
  - सुनिश्चित करें कि आप साइन इन हैं (स्टेटस बार जांचें)।
  - सत्यापित करें कि आपका सब्सक्रिप्शन आपकी GitHub अकाउंट सेटिंग्स में सक्रिय है।
  - यदि समस्या बनी रहती है तो VS Code को रीस्टार्ट करें या एक्सटेंशन को दोबारा इंस्टॉल करें।
- **कोई सुझाव नहीं?**:
  - जांचें कि क्या Copilot वर्तमान भाषा के लिए सक्षम है (सेटिंग्स में `editor.inlineSuggest.enabled` true होना चाहिए)।
