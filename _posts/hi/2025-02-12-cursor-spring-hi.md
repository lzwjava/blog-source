JDK का पथ `/opt/homebrew/Cellar/openjdk@17/17.0.14/libexec/openjdk.jdk/Contents/Home` पर स्थित है।

```bash
mvn clean install
mvn spring-boot:run
mvn compile
mvn test
mvn package
mvn deploy
mvn verify
mvn site
mvn archetype:generate
```

कमांड लाइन का उपयोग करके कोड को संकलित और चलाएँ।
