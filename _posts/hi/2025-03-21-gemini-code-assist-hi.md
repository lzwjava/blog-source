### स्टेप 1: Gemini Code Assist एक्सटेंशन इंस्टॉल करें
- **VSCode** खोलें।
- **एक्सटेंशन्स** व्यू पर नेविगेट करें एक्टिविटी बार में एक्सटेंशन्स आइकन पर क्लिक करके या `Ctrl+Shift+X` (Windows/Linux) या `Cmd+Shift+X` (macOS) दबाकर।
- सर्च बार में, **"Gemini Code Assist"** टाइप करें।
- मार्केटप्लेस में ऑफिशियल एक्सटेंशन ढूंढें, **इंस्टॉल** पर क्लिक करें, और यदि प्रॉम्प्ट किया जाए तो VSCode को रीस्टार्ट कर दें।

### स्टेप 2: अपने Google अकाउंट से साइन इन करें
- इंस्टॉलेशन के बाद, **Gemini Code Assist आइकन** को एक्टिविटी बार (VSCode के बाईं ओर) में ढूंढें।
- आइकन पर क्लिक करके Gemini पेन खोलें।
- **"Sign in with Google"** चुनें और अपने Google अकाउंट का उपयोग करके ऑथेंटिकेशन प्रॉम्प्ट्स का पालन करें।
  - **फ्री वर्जन** (Gemini Code Assist for individuals) के लिए, एक पर्सनल Gmail अकाउंट पर्याप्त है।
  - **स्टैंडर्ड या एंटरप्राइज़ वर्जन** के लिए, आपको इसे एक Google Cloud प्रोजेक्ट से लिंक करने की आवश्यकता हो सकती है जिसमें आवश्यक APIs एनेबल हों।

### स्टेप 3: Gemini Code Assist का उपयोग शुरू करें
एक बार साइन इन हो जाने के बाद, आप इसकी फीचर्स का लाभ कई तरीकों से उठा सकते हैं:

#### a. कोड कम्प्लीशन
- जैसे ही आप एडिटर में टाइप करते हैं, Gemini स्वचालित रूप से कोड कम्प्लीशन सजेस्ट करता है।
- इन सजेशन्स को `Tab` (या किसी अन्य कॉन्फ़िगर की गई कुंजी) दबाकर एक्सेप्ट करें।

#### b. कोड जनरेशन और एक्सप्लेनेशन (चैट के माध्यम से)
- **Gemini पेन** खोलें एक्टिविटी बार में इसके आइकन पर क्लिक करके।
- एक नैचुरल लैंग्वेज प्रॉम्प्ट टाइप करें, जैसे:
  - "इस कोड को समझाएं"
  - "एक ऐरे को सॉर्ट करने के लिए एक फंक्शन जनरेट करें"
  - "इस एरर को डीबग करने में मेरी मदद करें"
- स्पेसिफिक कोड को रेफरेंस करने के लिए, अपना प्रॉम्प्ट एंटर करने से पहले एडिटर में उसे हाइलाइट कर लें।
- Gemini चैट पेन में जवाब देगा, और आप चाहें तो जनरेट किए गए किसी भी कोड को अपनी फाइल में इन्सर्ट कर सकते हैं।

#### c. कोड ट्रांसफॉर्मेशन
- क्विक पिक मेन्यू एक्सेस करें `Ctrl+I` (Windows/Linux) या `Cmd+I` (macOS) दबाकर।
- एक कमांड एंटर करें जैसे `/generate function to create a Cloud Storage bucket`।
- सजेस्ट किए गए बदलावों को एक डिफ व्यू में रिव्यू करें और आवश्यकतानुसार उन्हें एप्लाई कर लें।

#### d. इनलाइन सजेशन्स
- कोडिंग करते समय, लाइन्स या कोड ब्लॉक्स को कम्प्लीट करने के लिए Gemini इनलाइन सजेशन्स ऑफर कर सकता है।
- ये आपके टाइप करते समय दिखाई देते हैं और एक की प्रेस से एक्सेप्ट किए जा सकते हैं।
