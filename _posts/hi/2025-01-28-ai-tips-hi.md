* अपने उदाहरण कोड को Cursor में कॉपी करें और कनेक्टिंग के लिए कमेंट्स उपयोग करें।

* हॉंग कॉंग क्षेत्र में, DeepSeek या Mistral सेवाओं का उपयोग करना स्वीकार्य है, जबकि ChatGPT, Claude, या Gemini का उपयोग करना स्वीकार्य नहीं है।

* एक क्षेत्र की API नीति उसकी अप्लिकेशन नीति के बराबर है।

* Cursor का उपयोग करें बल्कि Visual Studio Code का।

* कुछ समस्याओं में, जैसे git merge editor सिनारियों में, मैं अभी भी `git config --global core.editor "code --wait"` उपयोग करता हूँ।

* Deepseek V3 की रिलीज से शुरू हो जाने तक, हम किसी भी AI उपकरण को सब्सक्राइब करने की जरूरत नहीं है।

* Gemini या Grok का उपयोग करें त्योहार मनाने के चित्र बनाने के लिए प्रोम्प्ट्स जैसे "एक खुश चंद नाग नव वर्ष चित्र बनाएं जहाँ नामें शामिल हों" का उपयोग करें।
