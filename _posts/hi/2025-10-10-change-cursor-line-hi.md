### VS Code में ब्लॉक (आयताकार) कर्सर को ठीक करना

समझ गया—ऐसा लगता है कि आपका कर्सर ब्लॉक/आयताकार स्टाइल में बदल गया है (ओवरराइट मोड या डिफ़ॉल्ट सेटिंग्स में आम), लेकिन आप आसान एडिटिंग के लिए पतली ऊर्ध्वाधर **लाइन** स्टाइल चाहते हैं। यह एक त्वरित सेटिंग्स बदलाव है। बिल्कुल इन चरणों का पालन करें:

#### सेटिंग्स UI के माध्यम से (सबसे आसान):
1. VS Code खोलें।
2. सेटिंग्स खोलने के लिए `Ctrl + ,` (Windows/Linux) या `Cmd + ,` (macOS) दबाएँ।
3. शीर्ष पर सर्च बॉक्स में, टाइप करें: **cursor style**।
4. **Editor: Cursor Style** देखें (Text Editor > Cursor के अंतर्गत)।
5. ड्रॉपडाउन पर क्लिक करें और **Line** चुनें (यह वर्तमान में "Block" कह सकता है)।
6. परिवर्तन तुरंत हो जाता है—पतली लाइन कर्सर देखने के लिए एक फ़ाइल में टाइप करके देखें।

यदि यह अभी भी अपडेट नहीं हो रहा है, तो VS Code को बंद करके फिर से खोलें, या इसे पूरी तरह से रीस्टार्ट करें।

#### settings.json के माध्यम से (सटीकता के लिए):
1. कमांड पैलेट खोलें: `Ctrl + Shift + P` (या macOS पर `Cmd + Shift + P`)।
2. टाइप करें और चुनें: **Preferences: Open User Settings (JSON)**।
3. `{}` के अंदर इस लाइन को जोड़ें या अपडेट करें (यदि खाली है तो इसे बनाएँ):
   ```json:disable-run
   "editor.cursorStyle": "line"
   ```
4. फ़ाइल सेव करें (`Ctrl + S` या `Cmd + S`)।

#### अतिरिक्त सुझाव:
- **ओवरराइट मोड?** यदि आप गलती से "ओवरराइट" मोड में हैं (जहाँ यह वर्णों को ब्लॉक करता है), तो इन्सर्ट मोड में वापस टॉगल करने के लिए अपने कीबोर्ड पर **Insert** कुंजी दबाएँ।
- **ब्लिंकिंग समस्या?** यदि लाइन कटी-फटी दिखती है, तो सेटिंग्स में **cursor blinking** खोजें और इसे **Blink** या **Smooth** पर सेट करें।
- **एक्सटेंशन हस्तक्षेप कर रहे हैं?** टेस्ट करने के लिए Vim जैसे एक्सटेंशन को अस्थायी रूप से डिसेबल करें (Extensions view > राइट-क्लिक > Disable)।

इससे आपको तुरंत वह साफ लाइन कर्सर मिल जाना चाहिए। यदि इसके बाद भी यह आयताकार है, तो अधिक लक्षित सहायता के लिए अपना OS या एक स्क्रीनशॉट साझा करें!
