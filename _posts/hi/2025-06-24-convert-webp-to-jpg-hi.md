WebP को JPG में बदलने के लिए आपके पास Ubuntu में कई बेहतरीन विकल्प हैं, चाहे कमांड लाइन से करना हो या GUI टूल्स के साथ।

## कमांड लाइन टूल्स

ये आमतौर पर सिंगल फाइलों या बैच कन्वर्जन के लिए सबसे लचीले और कुशल होते हैं।

### 1. ImageMagick

ImageMagick इमेज मैनिपुलेशन के लिए एक शक्तिशाली सूट है, और यह आसानी से उपलब्ध है।

*   **इंस्टालेशन:**
    ```bash
    sudo apt update
    sudo apt install imagemagick
    ```
*   **एक फाइल कन्वर्ट करें:**
    ```bash
    convert input.webp output.jpg
    ```
    या, ImageMagick v7+ के लिए `magick` का उपयोग करके:
    ```bash
    magick input.webp output.jpg
    ```
*   **मौजूदा डायरेक्टरी में कई फाइलें कन्वर्ट करें (बैच कन्वर्जन):**
    ```bash
    mogrify -format JPEG *.webp
    ```
    यह मौजूदा डायरेक्टरी में सभी `.webp` फाइलों को `.jpeg` (या `.jpg`) में उसी फाइलनाम के साथ कन्वर्ट कर देगा। अगर आप कन्वर्ट की गई फाइलें एक अलग डायरेक्टरी में चाहते हैं:
    ```bash
    mkdir output_jpgs # आउटपुट के लिए एक डायरेक्टरी बनाएं
    mogrify -format JPEG -path output_jpgs *.webp
    ```

### 2. `dwebp` (`libwebp` पैकेज से)

`dwebp` WebP इमेज को डिकोड करने के लिए एक समर्पित टूल है।

*   **इंस्टालेशन:**
    ```bash
    sudo apt update
    sudo apt install webp
    ```
*   **एक फाइल कन्वर्ट करें:**
    ```bash
    dwebp input.webp -o output.jpg
    ```
    हालांकि आप आउटपुट के लिए `.jpg` निर्दिष्ट करते हैं, `dwebp` अक्सर PPM फॉर्मेट में आउटपुट देता है और उसे सही JPEG में कन्वर्ट करने के लिए ImageMagick जैसे किसी अन्य टूल की आवश्यकता होती है। हालांकि, कुछ स्रोत सुझाव देते हैं कि सरल मामलों के लिए सीधे `.jpg` को आउटपुट के रूप में उपयोग करना काम करता है। अगर आपको कोई समस्या आती है, तो पहले PNG में कन्वर्ट करें, फिर JPG में:
    ```bash
    dwebp input.webp -o output.png
    convert output.png output.jpg
    ```

### 3. `ffmpeg`

`ffmpeg` मुख्य रूप से वीडियो और ऑडियो के लिए है, लेकिन यह इमेज फॉर्मेट्स के साथ भी बहुत सक्षम है।

*   **इंस्टालेशन:**
    ```bash
    sudo apt update
    sudo apt install ffmpeg
    ```
*   **एक फाइल कन्वर्ट करें:**
    ```bash
    ffmpeg -i input.webp output.jpg
    ```
*   **कई फाइलें कन्वर्ट करें (बैच कन्वर्जन):**
    ```bash
    for file in *.webp; do ffmpeg -i "$file" "${file%.*}.jpg"; done
    ```
    यह कमांड मौजूदा डायरेक्टरी में सभी `.webp` फाइलों के माध्यम से पुनरावृति करता है और उन्हें उसी नाम के साथ `.jpg` में कन्वर्ट कर देता है (उदाहरण के लिए, `image.webp`, `image.jpg` बन जाता है)।

## GUI टूल्स

अगर आप ग्राफिकल इंटरफेस पसंद करते हैं, तो यहां कुछ विकल्प दिए गए हैं:

### 1. GIMP (GNU Image Manipulation Program)

GIMP एक शक्तिशाली ओपन-सोर्स इमेज एडिटर है जो WebP को सपोर्ट करता है।

*   **इंस्टालेशन:**
    ```bash
    sudo apt update
    sudo apt install gimp
    ```
*   **कैसे कन्वर्ट करें:**
    1.  WebP इमेज को GIMP में खोलें (File > Open)।
    2.  File > Export As पर जाएं।
    3.  "Export Image" डायलॉग में, "Select File Type (By Extension)" ड्रॉपडाउन से "JPEG image" चुनें।
    4.  "Export" पर क्लिक करें और इच्छित हो तो क्वालिटी सेटिंग्स एडजस्ट करें।

### 2. XnView MP / XnConvert

ये क्रॉस-प्लेटफॉर्म इमेज व्यूअर और बैच कन्वर्टर हैं जिनमें WebP सहित व्यापक फॉर्मेट सपोर्ट है।

*   **इंस्टालेशन:**
    आपको उनकी ऑफिशियल वेबसाइट से `.deb` पैकेज डाउनलोड करके `sudo dpkg -i <package_name.deb>` का उपयोग करके या ग्राफिकल पैकेज इंस्टॉलर के माध्यम से इंस्टॉल करना पड़ सकता है।
*   **कैसे कन्वर्ट करें (सामान्य तौर पर):**
    1.  XnView MP या XnConvert खोलें।
    2.  वे WebP इमेज जोड़ें जिन्हें आप कन्वर्ट करना चाहते हैं।
    3.  आउटपुट फॉर्मेट के रूप में JPG चुनें।
    4.  कोई भी सेटिंग्स एडजस्ट करें (क्वालिटी, रीसाइज, आदि)।
    5.  कन्वर्जन शुरू करें।

### 3. Converseen

Converseen एक मुफ्त बैच इमेज कन्वर्टर है जो WebP को सपोर्ट करता है।

*   **इंस्टालेशन:** आप इसे आमतौर पर Ubuntu Software Center में पा सकते हैं या अपने रिपॉजिटरी में उपलब्ध होने पर apt के माध्यम से इंस्टॉल कर सकते हैं।
*   **कैसे कन्वर्ट करें:** इसमें इमेज जोड़ने, आउटपुट फॉर्मेट चुनने और कन्वर्ट करने के लिए एक सीधा GUI है।

## कौन सा तरीका चुनें?

*   **त्वरित, सिंगल-फाइल कन्वर्जन या ऑटोमेटेड टास्क्स स्क्रिप्टिंग के लिए:** कमांड लाइन से `ffmpeg` या `ImageMagick` का उपयोग करें। वे बहुत कुशल हैं।
*   **ऐसे कन्वर्जन के लिए जहां आप एक विजुअल इंटरफेस और इमेज एडिटिंग पर अधिक नियंत्रण चाहते हैं:** GIMP एक बढ़िया विकल्प है।
*   **GUI और विभिन्न विकल्पों के साथ कई इमेज को बैच कन्वर्ट करने के लिए:** XnView MP/XnConvert या Converseen उत्कृष्ट हैं।

कोई फर्क नहीं पड़ता कि आप कौन सा तरीका चुनते हैं, पहले आवश्यक पैकेज इंस्टॉल करना सुनिश्चित करें।
