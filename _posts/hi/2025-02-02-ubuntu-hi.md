मैंने पहले [कंप्यूटर को कैसे बनाएं](./computer-en) पर लिखा। आज, हम कुछ अलग करने की कोशिश करें। आज, मैं एक Lenovo Xiaoxing 14IAH8 लैपटॉप पर Ubuntu 24.04 इंस्टॉल कर रहा हूँ, जो Windows 10 के साथ प्री-इंस्टॉल्ड आया था।

```bash
sudo dd if=ubuntu-24.04.1-desktop-amd64.iso of=/dev/rdisk6 bs=1m
```

सेटिंग्स पर जाना और डिवाइस एन्क्रिप्शन को डिसएबल करना आवश्यक है, क्योंकि Ubuntu इंस्टॉलर के लिए यह आवश्यक है।

यदि डिवाइस एन्क्रिप्शन एनएबल्ड है और BIOS में सिक्योर बूट डिसएबल्ड है, तो Windows बिटलॉकर पासफ्रेस के लिए रिकवरी की प्रॉम्प्ट करेगा।

इसलिए, पहले BIOS में सिक्योर बूट को एनएबल करना होगा, Windows में लॉग इन करना, फिर सेटिंग्स पर जाना और डिवाइस एन्क्रिप्शन को डिसएबल करना होगा। इसके बाद, सिक्योर बूट को फिर से डिसएबल किया जा सकता है ताकि Ubuntu इंस्टॉलर शुरू किया जा सके।
