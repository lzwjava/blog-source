यह रूपरेखा चीन की राष्ट्रीय उच्च शिक्षा स्व-अध्ययन परीक्षा के मानक पाठ्यक्रम पर आधारित है, जो विशेष रूप से introductory Java प्रोग्रामिंग के लिए तैयार की गई है। यह गुआंग्डोंग प्रांत में पाठ्यक्रम के लिए मूल अवधारणाओं, संरचना और मूल्यांकन फोकस को कवर करती है। यह पाठ्यक्रम Java का उपयोग करके मूलभूत प्रोग्रामिंग कौशल पर जोर देता है, जिसमें सिंटैक्स, ऑब्जेक्ट-ओरिएंटेड सिद्धांत और बुनियादी एप्लिकेशन डेवलपमेंट शामिल हैं।

## I. पाठ्यक्रम की प्रकृति और उद्देश्य
- **प्रकृति**: यह एक मूलभूत प्रोग्रामिंग पाठ्यक्रम है जो Java का उपयोग करके ऑब्जेक्ट-ओरिएंटेड प्रोग्रामिंग (OOP) पर केंद्रित है। Java एक प्लेटफ़ॉर्म-स्वतंत्र, सुरक्षित और बहुमुखी भाषा है जो नेटवर्क और एप्लिकेशन डेवलपमेंट के लिए आदर्श है।
- **उद्देश्य**:
  - बुनियादी प्रोग्रामिंग अवधारणाओं और Java की विशेषताओं को समझें।
  - Java सिंटैक्स, डेटा प्रकार, कंट्रोल स्ट्रक्चर्स और मेथड्स में निपुणता हासिल करें।
  - सरल प्रोग्राम बनाने, arrays जैसे डेटा स्ट्रक्चर्स को हैंडल करने और OOP सिद्धांतों को लागू करने के कौशल का विकास करें।
  - कोडिंग और डिबगिंग के माध्यम से व्यावहारिक समस्या-समाधान के लिए छात्रों को तैयार करें।

## II. मूल्यांकन लक्ष्य
- **ज्ञान**: Java के डेवलपमेंट एनवायरनमेंट, बेसिक सिंटैक्स और कोर लाइब्रेरीज़ की समझ।
- **कौशल**: Java प्रोग्राम लिखना, कंपाइल करना और एक्ज़िक्यूट करना; सरल एल्गोरिदम डिजाइन करना; कोड ऑर्गनाइज़ेशन के लिए OOP का उपयोग करना।
- **परीक्षा प्रारूप**: बंद-पुस्तक, 100 अंक, 90 मिनट। इसमें बहुविकल्पी, सही/गलत, रिक्त स्थान भरना, प्रोग्राम पठन, लघु उत्तर और प्रोग्रामिंग प्रश्न शामिल हैं।
- **भारांकन**: सैद्धांतिक ज्ञान (40%), व्यावहारिक कोडिंग (60%)।

## III. पाठ्यक्रम सामग्री और आवश्यकताएँ

### प्रथम भाग: Java अवलोकन
- Java भाषा विशिष्टताएँ, API, JDK, और IDE टूल्स।
- Java प्रोग्राम बनाना, कंपाइल करना और चलाना।
- प्रोग्रामिंग स्टाइल दिशानिर्देश।
- सामान्य प्रोग्रामिंग त्रुटियाँ और डिबगिंग की मूल बातें।

### द्वितीय भाग: बेसिक प्रोग्रामिंग
- कंसोल से इनपुट पढ़ना।
- आइडेंटिफायर्स, वेरिएबल्स, एक्सप्रेशन्स, असाइनमेंट स्टेटमेंट्स और कॉन्स्टेंट्स।
- न्यूमेरिक डेटा प्रकार और ऑपरेशन्स; टाइप कनवर्जन।
- एक्सप्रेशन मूल्यांकन और ऑपरेटर प्रीसीडेंस।
- संवर्धित असाइनमेंट, इंक्रीमेंट/डिक्रीमेंट ऑपरेटर्स।

### तृतीय भाग: चयन संरचनाएँ
- if-else स्टेटमेंट्स और नेस्टिंग।
- switch स्टेटमेंट्स।
- कंडीशनल एक्सप्रेशन्स।
- ऑपरेटर प्रीसीडेंस और असोसिएटिविटी नियम।

### चतुर्थ भाग: गणितीय फलन, कैरेक्टर्स और स्ट्रिंग्स
- सामान्य गणितीय फलन (जैसे, Math क्लास)।
- कैरेक्टर डेटा प्रकार और ऑपरेशन्स।
- स्ट्रिंग प्रकार: निर्माण, मैनिपुलेशन और मेथड्स।
- कंसोल आउटपुट फॉर्मेटिंग (जैसे, printf)।

### पंचम भाग: लूप्स
- while लूप्स।
- do-while लूप्स।
- for लूप्स।
- नेस्टेड लूप्स।
- break और continue कीवर्ड्स।

### षष्ठ भाग: मेथड्स
- मेथड्स को परिभाषित करना और कॉल करना।
- वैल्यू द्वारा पैरामीटर पासिंग।
- मेथड ओवरलोडिंग।
- वेरिएबल स्कोप और लाइफटाइम।

### सप्तम भाग: ऐरेज़
- ऐरे की मूल बातें (डिक्लेरेशन, इनिशियलाइज़ेशन)।
- मेथड्स को ऐरे पास करना।
- मेथड्स से ऐरे वापस लौटाना।
- Varargs (वेरिएबल-लेंथ आर्गुमेंट्स)।
- मल्टीडायमेंशनल ऐरेज़।

### अष्टम भाग: ऑब्जेक्ट्स और क्लासेज़
- क्लासेज़ को परिभाषित करना और ऑब्जेक्ट्स बनाना।
- रेफरेंस वेरिएबल्स के माध्यम से ऑब्जेक्ट्स तक पहुँच।
- बिल्ट-इन क्लासेज़ का उपयोग (जैसे, Date, Random)।
- स्टैटिक वेरिएबल्स और मेथड्स।
- विजिबिलिटी मॉडिफायर्स (public, private, आदि)।
- डेटा फील्ड्स का एनकैप्सुलेशन।
- पैरामीटर्स के रूप में ऑब्जेक्ट्स पास करना।
- ऑब्जेक्ट्स की arrays।
- क्लास वेरिएबल स्कोप।
- 'this' रेफरेंस।

### नवम भाग: इनहेरिटेंस और पॉलीमॉर्फिज्म
- सुपरक्लासेज़ और सबक्लासेज़।
- super कीवर्ड।
- मेथड ओवरराइडिंग।
- Object क्लास और toString() मेथड।
- पॉलीमॉर्फिज्म और डायनामिक बाइंडिंग।
- ऑब्जेक्ट्स का कास्टिंग और instanceof ऑपरेटर।
- protected एक्सेस।
- इनहेरिटेंस/ओवरराइडिंग रोकने के लिए final कीवर्ड।

### दशम भाग: एब्स्ट्रैक्ट क्लासेज़ और इंटरफेसेस
- एब्स्ट्रैक्ट क्लासेज़ और मेथड्स।
