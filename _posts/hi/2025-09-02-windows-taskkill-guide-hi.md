Windows में, `taskkill` एक बिल्ट-इन कमांड-लाइन टूल है जिसका उपयोग आप **प्रोसेस ID (PID)** या **इमेज नाम (एग्ज़िक्यूटेबल नाम)** द्वारा प्रक्रियाओं को रोकने (किल) के लिए कर सकते हैं।

### बेसिक सिंटैक्स

```bat
taskkill /PID <process_id> [/F]
taskkill /IM <image_name> [/F]
```

* `/PID <process_id>` → प्रोसेस ID द्वारा किल करें।
* `/IM <image_name>` → प्रोसेस नाम द्वारा किल करें (जैसे, `notepad.exe`)।
* `/F` → प्रोसेस को जबरदस्ती समाप्त करें।
* `/T` → निर्दिष्ट प्रोसेस और सभी चाइल्ड प्रोसेस को किल करें।

---

### उदाहरण

1. **PID द्वारा प्रोसेस किल करें:**

```bat
taskkill /PID 1234
```

यह PID 1234 वाली प्रोसेस को रोकने का प्रयास करेगा।

2. **PID द्वारा प्रोसेस को फोर्स किल करें:**

```bat
taskkill /PID 1234 /F
```

3. **नाम द्वारा प्रोसेस किल करें:**

```bat
taskkill /IM notepad.exe
```

4. **नाम द्वारा फोर्स किल करें:**

```bat
taskkill /IM notepad.exe /F
```

5. **प्रोसेस और उसकी चाइल्ड प्रोसेस को किल करें:**

```bat
taskkill /IM cmd.exe /T /F
```

---

### PID ढूँढना

अगर आपको PID पता नहीं है:

```bat
tasklist
```

यह उनके PIDs के साथ सभी चल रही प्रोसेस की सूची दिखाता है। फिर आप सही वाली प्रोसेस पर `taskkill /PID` का उपयोग कर सकते हैं।

---

⚠️ **नोट्स**

* कुछ सिस्टम प्रोसेस को किल नहीं किया जा सकता।
* संरक्षित प्रोसेस को समाप्त करने के लिए आपको कमांड प्रॉम्प्ट को **एडमिनिस्ट्रेटर** के रूप में रन करने की आवश्यकता हो सकती है।

---

क्या आप चाहते हैं कि मैं आपको `tasklist` और `taskkill` को एक कमांड में कैसे जोड़ना है यह भी दिखाऊँ (जैसे, एक नाम से मेल खाने वाली सभी प्रोसेस को स्वचालित रूप से किल करना)?
