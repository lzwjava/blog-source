### पूर्वापेक्षाएँ
1. **CocoaPods Installed**: सुनिश्चित करें कि आपके सिस्टम पर CocoaPods इंस्टॉल है। यदि नहीं, इसे इंस्टॉल करने के लिए निम्नलिखित कमांड चलाएं:
   ```bash
   sudo gem install cocoapods
   ```
2. **Xcode Project**: आपको एक मौजूदा Xcode प्रोजेक्ट चाहिए जहां आप `JSBadgeView` जोड़ना चाहते हैं।

### `pod 'JSBadgeView', '1.4.1'` का उपयोग करने के चरण

1. **अपने प्रोजेक्ट डायरेक्टरी में जाएँ**:
   अपने टर्मिनल खोलें और अपने Xcode प्रोजेक्ट (`.xcodeproj` फ़ाइल) को रखने वाले डायरेक्टरी में जाएँ:
   ```bash
   cd /path/to/your/project
   ```

2. **CocoaPods को प्रारंभ करें (अगर पहले से नहीं किया गया है)**:
   अगर आपके प्रोजेक्ट में पहले से `Podfile` नहीं है, तो इसे बनाएं:
   ```bash
   pod init
   ```
   यह आपके प्रोजेक्ट डायरेक्टरी में एक `Podfile` बनाता है।

3. **Podfile को संपादित करें**:
   एक टेक्स्ट एडिटर (जैसे `nano`, `vim`, या किसी भी आईडीई) में `Podfile` खोलें और अपने टारगेट के नीचे `JSBadgeView` पॉड जोड़ें। उदाहरण के लिए:
   ```ruby
   platform :ios, '9.0' # अपने डिप्लॉयमेंट टारगेट को स्पेसिफाई करें

   target 'YourProjectName' do
     use_frameworks! # अगर आपके प्रोजेक्ट में Swift या फ्रेमवर्क का उपयोग है तो आवश्यक है
     pod 'JSBadgeView', '1.4.1'
   end
   ```
   `'YourProjectName'` को अपने Xcode टारगेट के वास्तविक नाम से बदलें।

4. **पॉड को इंस्टॉल करें**:
   `Podfile` को सेभ करें, फिर टर्मिनल में निम्नलिखित कमांड चलाएं पॉड को इंस्टॉल करने के लिए:
   ```bash
   pod install
   ```
   यह `JSBadgeView` वर्जन 1.4.1 डाउनलोड करता है और इसे आपके प्रोजेक्ट में सेटअप करता है। अगर सफल, तो आप पॉड इंस्टॉल किए गए होने की सूचना देखेंगे।

5. **वर्कस्पेस को खोलें**:
   इंस्टॉलेशन के बाद, CocoaPods एक `.xcworkspace` फ़ाइल बनाता है। इस फ़ाइल (नहीं `.xcodeproj`) को Xcode में खोलें:
   ```bash
   open YourProjectName.xcworkspace
   ```

6. **अपने कोड में JSBadgeView को आयात और उपयोग करें**:
   - अगर आप **Objective-C** का उपयोग कर रहे हैं, तो अपने फ़ाइल में हेडर को आयात करें:
     ```objective-c
     #import <JSBadgeView/JSBadgeView.h>
     ```
   - अगर आप **Swift** का उपयोग कर रहे हैं, तो अगर `use_frameworks!` आपके `Podfile` में है, तो कोई आयात आवश्यक नहीं है। सीधे क्लास का उपयोग करें।
   - उदाहरण उपयोग (Objective-C):
     ```objective-c
     JSBadgeView *badgeView = [[JSBadgeView alloc] initWithParentView:someView alignment:JSBadgeViewAlignmentTopRight];
     badgeView.badgeText = @"5"; // बैज टेक्स्ट को "5" पर सेट करें
     ```
   - उदाहरण उपयोग (Swift):
     ```swift
     let badgeView = JSBadgeView(parentView: someView, alignment: .topRight)
     badgeView.badgeText = "5"
     ```

7. **बिल्ड और चलाएं**:
   Xcode में अपने प्रोजेक्ट को बिल्ड करें (`Cmd + B`) सुनिश्चित करने के लिए कि सब कुछ सही से सेटअप है, फिर इसे एक सिमुलेटर या डिवाइस पर चलाएं।
