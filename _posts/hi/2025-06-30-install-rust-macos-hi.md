## rustup का उपयोग करके Rust इंस्टॉल करना

1.  **टर्मिनल खोलें**  
    आप टर्मिनल को **Applications > Utilities** में ढूंढ सकते हैं या स्पॉटलाइट का उपयोग करके इसकी खोज कर सकते हैं।

2.  **rustup और Rust इंस्टॉल करें**  
    rustup इंस्टॉलेशन स्क्रिप्ट को डाउनलोड और रन करने के लिए निम्नलिखित कमांड चलाएं:

    ```sh
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

    स्क्रिप्ट आगे बढ़ने के लिए प्रॉम्प्ट करेगी। डिफ़ॉल्ट इंस्टॉलेशन के लिए `1` दबाएं।

3.  **अपनी शेल को कॉन्फ़िगर करें**  
    इंस्टॉलेशन के बाद, आपको Rust को अपने सिस्टम `PATH` में जोड़ने की आवश्यकता हो सकती है। इंस्टॉलर इसके समान एक कमांड प्रदान करेगा:

    ```sh
    source $HOME/.cargo/env
    ```

    इस लाइन को अपनी शेल कॉन्फ़िगरेशन फ़ाइल (जैसे, `.zshrc`, `.bashrc`, या `.bash_profile`) में जोड़ें ताकि Rust भविष्य के सेशन में उपलब्ध रहे।

4.  **इंस्टॉलेशन सत्यापित करें**  
    जांचें कि Rust सही ढंग से इंस्टॉल हो गया है:

    ```sh
    rustc --version
    ```

    यदि इंस्टॉलेशन सफल रहा तो आपको Rust कंपाइलर का वर्जन दिखाई देगा।

## Rust इंस्टॉल करने के लिए Homebrew का उपयोग करना

वैकल्पिक रूप से, आप Rust इंस्टॉल करने के लिए [Homebrew](https://brew.sh/) का उपयोग कर सकते हैं, हालांकि आसान अपडेट और वर्जन प्रबंधन के लिए `rustup` की सामान्यतः सिफारिश की जाती है।

1.  **Homebrew इंस्टॉल करें** (यदि पहले से इंस्टॉल नहीं है):

    ```sh
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```

2.  **Homebrew के साथ Rust इंस्टॉल करें**:

    ```sh
    brew install rust
    ```

3.  **इंस्टॉलेशन सत्यापित करें**:

    ```sh
    rustc --version
    ```

## Rust कोड कंपाइल करना

1.  **एक Rust प्रोग्राम लिखें**  
    निम्नलिखित सामग्री के साथ `main.rs` नामक एक फ़ाइल बनाएं:

    ```rust
    fn main() {
         println!("Hello, World!");
    }
    ```

2.  **प्रोग्राम को कंपाइल करें**  
    टर्मिनल में, अपनी फ़ाइल वाले डायरेक्टरी में नेविगेट करें और यह कमांड चलाएं:

    ```sh
    rustc main.rs
    ```

3.  **कंपाइल किए गए प्रोग्राम को रन करें**:

    ```sh
    ./main
    ```
