### GitHub Copilot Extension for VS Code (Version 1.388.0)

GitHub Copilot एक्सटेंशन का Version 1.388.0 24 अक्टूबर, 2025 को जारी किया गया था। यह एक मामूली अपडेट प्रतीत होता है जो स्थिरता और संगतता सुधारों पर केंद्रित है, विशेष रूप से नवीनतम VS Code रिलीज़ (1.105) के साथ। मार्केटप्लेस या GitHub ब्लॉग में विशिष्ट रिलीज़ नोट्स सार्वजनिक रूप से विस्तृत नहीं हैं, लेकिन उपयोगकर्ता रिपोर्ट्स इनलाइन जनरेशन के दौरान अवांछित कोड इंसर्शन और सुझावों को रद्द करते समय अतिरिक्त टैग कम्पलीशन जैसी समस्याओं के लिए फिक्स इंगित करती हैं। यह हाल की Copilot फीचर्स, जिसमें एन्हांस्ड एजेंट मोड और मॉडल चयन शामिल हैं, के साथ निर्बाध रूप से एकीकृत होता है।

#### पिछले 6 महीनों में प्रमुख अपडेट (मई–अक्टूबर 2025)
GitHub Copilot के प्रमुख एन्हांसमेंट आमतौर पर मासिक VS Code रिलीज़ के साथ घोषित किए जाते हैं। इस अवधि के दौरान एक्सटेंशन और संबंधित फीचर्स के लिए महत्वपूर्ण अपडेट का सारांश यहां दिया गया है:

- **अक्टूबर 2025 (VS Code 1.105 / एक्सटेंशन ~1.388)**:
  - OpenAI Codex इंटीग्रेशन अब VS Code इनसाइडर्स में Copilot Pro+ सब्सक्राइबर्स के लिए उपलब्ध है, जो एडिटर में सीधे उन्नत कोड सिंथेसिस को सक्षम बनाता है।
  - सेशन्स में Copilot कोडिंग एजेंट टास्क्स को असाइन करने, स्टीयर करने और ट्रैक करने के लिए नया "मिशन कंट्रोल" इंटरफेस।
  - एजेंट सेशन्स व्यू को GitHub Copilot CLI के लिए लोकल और क्लाउड-आधारित एजेंट्स को मैनेज करने के लिए विस्तारित किया गया।

- **सितंबर 2025 (VS Code 1.104 / एक्सटेंशन ~1.38x)**:
  - VS Code इनसाइडर्स में प्रायोगिक GitHub Copilot-SWE मॉडल का रोलआउट, जो कोड एडिटिंग, रिफैक्टरिंग और छोटे ट्रांसफॉर्मेशन के लिए ऑप्टिमाइज़ किया गया है। यह टास्क-फोकस्ड है और किसी भी चैट मोड में काम करता है; सर्वोत्तम परिणामों के लिए विस्तृत प्रॉम्प्ट्स की सिफारिश की जाती है।
  - टर्मिनल एरर्स के लिए इनलाइन चैट में सुधार, बेहतर स्पष्टीकरण और फिक्स के साथ।

- **अगस्त 2025 (VS Code 1.103 / एक्सटेंशन ~1.37x)**:
  - मल्टी-लाइन कॉन्टेक्स्ट अवेयरनेस और संपूर्ण प्रोजेक्ट स्ट्रक्चर जनरेट करने के लिए @workspace के साथ इंटीग्रेशन के साथ एन्हांस्ड कमिट मैसेज सुझाव।
  - पूर्ण व्यू खोले बिना त्वरित इंटरैक्शन के लिए लाइटवेट इनलाइन चैट अपग्रेड।

- **जुलाई 2025 (VS Code 1.102 / एक्सटेंशन ~1.36x)**:
  - सिंगल प्रॉम्प्ट के माध्यम से बेहतर मल्टी-फाइल एडिट कोऑर्डिनेशन, जो सुसंगत परिवर्तनों के लिए प्रोजेक्ट स्ट्रक्चर का विश्लेषण करता है।
  - पुराने मॉडल्स (चयनित Claude, OpenAI, और Gemini वेरिएंट) को नए, तेज़ विकल्पों जैसे GPT-4.1 के पक्ष में डिप्रिकेट किया गया।

- **जून 2025 (VS Code 1.101 / एक्सटेंशन ~1.35x)**:
  - पुन: प्रयोज्य दिशानिर्देशों और संगठनात्मक ज्ञान के साथ Copilot के व्यवहार को कस्टमाइज़ करने के लिए प्रॉम्प्ट और निर्देश फाइलों का परिचय।
  - विस्तारित GitHub Pull Requests इंटीग्रेशन: VS Code व्यू से सीधे PRs/इश्यूज़ को Copilot को असाइन करें, ट्रैकिंग के लिए नए "Copilot on My Behalf" क्वेरी के साथ।

- **मई 2025 (VS Code 1.100 / एक्सटेंशन ~1.34x)**:
  - एजेंट मोड में Model Context Protocol (MCP) सपोर्ट जोड़ा गया, जो तृतीय-पक्ष मॉडल्स के लिए कस्टम API कुंजियों की अनुमति देता है।
  - OpenAI GPT-4.1 और Anthropic Claude Sonnet द्वारा संचालित तेज़ एजेंट एडिट्स, विशेष रूप से बड़ी फाइलों के लिए।
  - सिमेंटिक कोड डिस्कवरी और कोडबेस-वाइड सुझावों के लिए रिपॉजिटरी सर्च में सुधार।

ये अपडेट एजेंटिक वर्कफ़्लोज़, मॉडल प्रदर्शन और VS Code के इकोसिस्टम के साथ निर्बाध एकीकरण पर जोर देते हैं। पूर्ण एक्सटेंशन इतिहास के लिए, VS Code मार्केटप्लेस देखें। प्रमुख घोषणाएं अक्सर VS Code के मासिक चक्रों से जुड़ी होती हैं।
