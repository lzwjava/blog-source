GitHub के अलावा, कोड होस्टिंग के लिए अन्य प्रसिद्ध प्लेटफ़ॉर्म में शामिल हैं:

- GitLab: एक संपूर्ण DevOps प्लेटफ़ॉर्म जो रिपॉजिटरी होस्टिंग, CI/CD, मॉनिटरिंग और भी बहुत कुछ प्रदान करता है
- Bitbucket: Atlassian के स्वामित्व में, Jira और अन्य Atlassian उत्पादों के साथ अच्छी तरह से एकीकृत
- SourceForge: सबसे पुराने कोड होस्टिंग प्लेटफ़ॉर्मों में से एक, विशेष रूप से ओपन-सोर्स प्रोजेक्ट्स के लिए लोकप्रिय
- Azure DevOps (पूर्व में Visual Studio Team Services): Microsoft का कोड होस्टिंग और DevOps के लिए समाधान
- AWS CodeCommit: Amazon की मैनेज्ड Git सेवा
- Gitee: चीन में लोकप्रिय, अक्सर "चीन का GitHub" कहा जाता है
- Launchpad: मुख्य रूप से Ubuntu और संबंधित ओपन-सोर्स प्रोजेक्ट्स के लिए उपयोग किया जाता है
- CodePen: फ्रंट-एंड कोड (HTML, CSS, JavaScript) के लिए विशेष
- Codeberg: एक गैर-लाभकारी, समुदाय-संचालित विकल्प जो मुक्त और ओपन सोर्स सॉफ़्टवेयर पर केंद्रित है
- Replit: एक ऑनलाइन IDE के साथ कोड होस्टिंग को जोड़ता है

ये प्लेटफ़ॉर्म अपनी सुविधाओं, मूल्य निर्धारण मॉडल और लक्षित दर्शकों में भिन्न हैं, जिनमें से कुछ एंटरप्राइज़ उपयोगकर्ताओं पर और अन्य ओपन-सोर्स समुदायों पर अधिक ध्यान केंद्रित करते हैं।
