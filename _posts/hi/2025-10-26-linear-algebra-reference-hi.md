मुख्य अवधारणाओं और सूत्रों का त्वरित संदर्भ। मैट्रिक्स, सारणिक, रैखिक प्रणाली, सदिश समष्टि, रूपांतरण, आइगेनमान और द्विघात रूपों पर ध्यान दें—पिछली परीक्षाओं के मुख्य विषय। परिभाषाओं, गुणों और गणनाओं की समीक्षा करें।

## 1. मैट्रिक्स
- **परिभाषा**: \\( A = [a_{ij}] \in \mathbb{R}^{m \times n} \\).
- **संक्रियाएँ**:
  - योग: \\( (A + B)_{ij} = a_{ij} + b_{ij} \\).
  - अदिश गुणन: \\( (cA)_{ij} = c a_{ij} \\).
  - गुणन: \\( (AB)_{ij} = \sum_k a_{ik} b_{kj} \\) (यदि संगत आकार के हों)।
- **ट्रांसपोज़**: \\( (A^T)_{ij} = a_{ji} \\); \\( (AB)^T = B^T A^T \\), \\( (A^T)^T = A \\).
- **व्युत्क्रम** (वर्ग): \\( AA^{-1} = I \\); \\( (AB)^{-1} = B^{-1} A^{-1} \\); \\( (A^T)^{-1} = (A^{-1})^T \\).
- **प्रकार**:
  - विकर्ण: विकर्ण के अलावा शून्य।
  - ऊपरी/निचली त्रिकोणीय: विकर्ण के नीचे/ऊपर शून्य।
  - सममित: \\( A = A^T \\).
  - ऑर्थोगोनल: \\( A^T A = I \\) (स्तंभ ऑर्थोनॉर्मल)।

## 2. सारणिक (det A)
- **गुण**:
  - \\( \det(AB) = \det A \cdot \det B \\); \\( \det(A^T) = \det A \\); \\( \det(cA) = c^n \det A \\).
  - पंक्ति/स्तंभ अदला-बदली: -1 से गुणा करता है।
  - पंक्ति/स्तंभ का गुणज जोड़ना: कोई परिवर्तन नहीं।
  - पंक्ति/स्तंभ को c से स्केल करना: c से गुणा करता है।
  - \\( \det I = 1 \\); \\( \det A = 0 \\) यदि एकवचन (आश्रित पंक्तियाँ/स्तंभ)।
- **गणना**:
  - 2x2: \\( \det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc \\).
  - सहखंड विस्तार (पंक्ति i): \\( \det A = \sum_j a_{ij} C_{ij} \\), जहाँ \\( C_{ij} = (-1)^{i+j} M_{ij} \\) (लघु सारणिक)।
  - त्रिकोणीय: विकर्ण अवयवों का गुणनफल।
- **सहायक/व्युत्क्रम**: \\( A^{-1} = \frac{1}{\det A} \adj A \\), जहाँ \\( \adj A = C^T \\) (सहखंड का ट्रांसपोज़)।
- **क्रैमर का नियम** (\\( Ax = b \\) के लिए, det A ≠ 0): \\( x_i = \frac{\det A_i}{\det A} \\) (A_i, i-वें स्तंभ को b से प्रतिस्थापित करता है)।

## 3. रैखिक प्रणालियाँ (Ax = b)
- **गाऊसी निष्कासन**: [A | b] को REF/RREF में पंक्ति न्यूनीकृत करें।
  - REF: पिवट (अग्रणी 1) सीढ़ीनुमा नीचे-दाएँ; पिवट के नीचे शून्य।
  - अद्वितीय हल के लिए पश्च-प्रतिस्थापन।
- **हल**:
  - अद्वितीय: rank A = n (पूर्ण स्तंभ रैंक), नलसमष्टि {0}।
  - अनंत: rank A = rank [A|b] < n (मुक्त चर)।
  - कोई नहीं: rank A < rank [A|b]।
- **पूर्ण हल**: विशेष हल + नलसमष्टि आधार (समघातीय हल)।
- **LU अपघटन** (बिना पिवटिंग): A = LU (L निचला एकांक त्रिकोणीय, U ऊपरी); Ly = b, Ux = y हल करें।
- **न्यूनतम वर्ग** (अतिनिर्धारित): \\( \hat{x} = (A^T A)^{-1} A^T b \\) (यदि पूर्ण रैंक)।

## 4. सदिश समष्टि और उपसमष्टियाँ
- **सदिश समष्टि**: योग/अदिश गुणन के लिए संवृत; अभिगृहीत (जैसे, 0 सदिश, प्रतिलोम)।
- **उपसमष्टियाँ**: सदिशों का फैलाव; संवृत, 0 समाहित करती है।
  - स्तंभ समष्टि: Col(A) = A के स्तंभों का फैलाव; dim = rank A।
  - पंक्ति समष्टि: Row(A) = Col(A^T); dim = rank A।
  - नलसमष्टि: Nul(A) = {x | Ax = 0}; dim = n - rank A।
  - बायाँ नलसमष्टि: Nul(A^T)।
- **रैखिक स्वतंत्रता**: c1 v1 + ... + ck vk = 0 ⇒ सभी ci = 0।
- **आधार**: रैखिक रूप से स्वतंत्र फैलाव समुच्चय।
- **आयाम**: आधार में सदिशों की संख्या; dim Col(A) + dim Nul(A) = n (रैंक-शून्यता)।
- **रैंक**: पिवट स्तंभों की संख्या = dim Col(A) = dim Row(A)।

## 5. रैखिक रूपांतरण
- **परिभाषा**: T: V → W रैखिक यदि T(u + v) = T u + T v, T(cu) = c T u।
- **मैट्रिक्स निरूपण**: आधारों के संदर्भ में [T] = A जहाँ T(x) = A x (मानक आधार)।
- **कर्नेल**: Ker T = Nul(A); प्रतिबिम्ब: Im T = Col(A)।
- **समरूपता**: 1-1 और आच्छादक (व्युत्क्रमणीय मैट्रिक्स)।
- **रैंक-शून्यता**: dim Ker T + dim Im T = dim V।

## 6. आइगेनमान और आइगेनसदिश
- **परिभाषा**: A v = λ v (v ≠ 0 आइगेनसदिश, λ आइगेनमान)।
- **अभिलाक्षणिक समीकरण**: det(A - λ I) = 0; मूल λi (बीजगणितीय बहुलता)।
- **आइगेनसदिश**: (A - λ I) v = 0 हल करें; ज्यामितीय बहुलता = आइगेनसमष्टि का आयाम।
- **विकर्णनीय**: n रैखिक रूप से स्वतंत्र आइगेनसदिश ⇒ A = X Λ X^{-1} (Λ diag(λi), X = [v1 ... vn])।
  - सममित A: सदैव विकर्णनीय; ऑर्थोगोनल आइगेनसदिश (A = Q Λ Q^T, Q ऑर्थोगोनल)।
- **ट्रेस**: tr A = ∑ λi।
- **सारणिक**: det A = ∏ λi।
- **समरूप मैट्रिक्स**: A ~ B यदि A = P B P^{-1}; समान आइगेनमान, ट्रेस, सारणिक।

## 7. आंतरिक गुणनफल और द्विघात रूप
- **आंतरिक गुणनफल**: <u, v> = u^T v (यूक्लिडियन); ||v|| = √<v,v>।
- **ऑर्थोगोनल**: <u,v> = 0; ऑर्थोनॉर्मल आधार: <ei, ej> = δij।
- **ग्राम-श्मिट**: आधार {v1,...,vn} को ऑर्थोगोनलाइज़ करें → {u1,...,un}।
  - u1 = v1; uk = vk - proj_{span(u1..u_{k-1})} vk; proj_w v = (<v,w>/<w,w>) w।
- **द्विघात रूप**: q(x) = x^T A x (A सममित)।
  - धनात्मक निश्चित: x ≠ 0 के लिए q(x) > 0 (सभी λi > 0)।
  - विकर्णीकरण: q(x) = ∑ λi yi^2 (y = Q^T x)।

## त्वरित सुझाव
- रैंक की गणना: पिवटों की संख्या ज्ञात करने के लिए पंक्ति न्यूनीकरण करें।
- विकर्णनीयता जाँचें: प्रत्येक λ के लिए ज्यामितीय बहुलता = बीजगणितीय बहुलता।
- परीक्षाओं के लिए: 2x2/3x3 मैट्रिक्स पर पंक्ति न्यूनीकरण, सारणिक विस्तार, आइगेनसमस्याओं का अभ्यास करें।
- सामान्य त्रुटियाँ: बहुलता भूलना; सहखंडों में चिह्न; गुणन की क्रम-विनिमेयता नहीं।
