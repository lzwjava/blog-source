GitHub Desktop संस्करण 3.4.13 (arm64) में विशिष्ट भाषाओं के लिए टेक्स्ट डिफ पैनल में फ़ॉन्ट रेंडरिंग समस्याएं दिखाई देती हैं। इन समस्याओं वाली भाषाओं में शामिल हैं:

- चीनी (सरलीकृत और पारंपरिक दोनों)
- हिंदी
- जापानी

हालांकि, निम्नलिखित भाषाएं सही ढंग से रेंडर होती हैं:
- अंग्रेजी
- स्पेनिश
- जर्मन
- अरबी
- फ्रेंच
