Ubuntu पर YourKit Java Profiler इंस्टॉल करने के लिए, YourKit वेबसाइट से उपलब्ध जानकारी के आधार पर इन चरणों का पालन करें:

1.  **प्रोफाइलर डाउनलोड करें**:
    *   YourKit Java Profiler डाउनलोड पेज पर जाएं: [https://www.yourkit.com/java/profiler/download/](https://www.yourkit.com/java/profiler/download/)
    *   YourKit Java Profiler 2025.3 का Linux वर्जन चुनें, जो Java 8 से Java 24 तक सपोर्ट करता है और Linux (Ubuntu सहित) पर आर्किटेक्चर जैसे arm32, arm64, ppc64le, x64, और x86 के साथ संगत है। सुनिश्चित करें कि आपका सिस्टम संगतता के लिए [सिस्टम आवश्यकताओं](https://www.yourkit.com/docs/java/system-requirements/) को पूरा करता है।

2.  **आर्काइव डाउनलोड करें**:
    *   Linux के लिए `.zip` आर्काइव डाउनलोड करें (उदाहरण के लिए, `YourKit-JavaProfiler-2025.3-<build>.zip`)। डाउनलोड लिंक YourKit डाउनलोड पेज पर उपलब्ध है।

3.  **आर्काइव को अनज़िप करें**:
    *   एक टर्मिनल खोलें और उस डायरेक्टरी में नेविगेट करें जहां डाउनलोड की गई फ़ाइल स्थित है (उदाहरण के लिए, `~/Downloads`)।
    *   निम्नलिखित कमांड का उपयोग करके आर्काइव को अनज़िप करें:
        ```bash
        unzip YourKit-JavaProfiler-2025.3-<build>.zip -d /opt/yourkit
        ```
        `<build>` को डाउनलोड की गई फ़ाइल के वास्तविक बिल्ड नंबर से बदलें। यह कमांड प्रोफाइलर को `/opt/yourkit` में एक्सट्रैक्ट करती है। आप पसंद करने पर कोई अन्य डायरेक्टरी चुन सकते हैं।

4.  **प्रोफाइलर रन करें**:
    *   एक्सट्रैक्ट की गई डायरेक्टरी में नेविगेट करें:
        ```bash
        cd /opt/yourkit
        ```
    *   प्रदान किए गए स्क्रिप्ट का उपयोग करके प्रोफाइलर रन करें:
        ```bash
        ./bin/profiler.sh
        ```
        यह YourKit Java Profiler यूजर इंटरफेस लॉन्च करता है।

5.  **वैकल्पिक: लाइसेंस कुंजी के साथ अनअटेंडेड इंस्टॉलेशन**:
    *   यदि आपके पास लाइसेंस कुंजी है और इंस्टॉलेशन को ऑटोमेट करना चाहते हैं, तो आप EULA स्वीकार करने और लाइसेंस कुंजी लागू करने के लिए कमांड-लाइन विकल्पों का उपयोग कर सकते हैं। उदाहरण के लिए:
        ```bash
        ./bin/profiler.sh -accept-eula -license-key=<key>
        ```
        `<key>` को अपनी वास्तविक लाइसेंस कुंजी से बदलें। यह ऑटोमेशन या स्क्रिप्टेड सेटअप के लिए उपयोगी है।

6.  **डेवलपमेंट एनवायरनमेंट के साथ इंटीग्रेट करें (वैकल्पिक)**:
    *   यदि आप Eclipse, IntelliJ IDEA, या NetBeans जैसे IDE का उपयोग करते हैं, तो YourKit सीमलेस इंटीग्रेशन के लिए प्लगइन प्रदान करता है। उदाहरण के लिए, Eclipse के लिए:
        *   Eclipse खोलें और **Help > Install New Software** पर जाएं।
        *   YourKit प्लगइन रिपॉजिटरी जोड़ें: `https://www.yourkit.com/download/yjp2025_3_for_eclipse/`।
        *   YourKit Java Profiler प्लगइन चुनें, इंस्टॉलेशन प्रॉम्प्ट का पालन करें और आवश्यकता पड़ने पर Eclipse को रीस्टार्ट करें।
        *   वैकल्पिक रूप से, ऑफलाइन आर्काइव `<Profiler Installation Directory>/lib/eclipse-plugin/yjp2025_3_for_eclipse.zip` का उपयोग करें।
    *   इंस्टॉलेशन के बाद, आसान प्रोफाइलिंग के लिए "Profile" एक्शन Eclipse के टूलबार, मेनू मेनू, या कॉन्टेक्स्ट मेनू में दिखाई देगा।

7.  **इंस्टॉलेशन सत्यापित करें**:
    *   एक बार प्रोफाइलर रन होने के बाद, वेलकम स्क्रीन आपको स्थानीय रूप से चल रहे Java एप्लिकेशन को मॉनिटर करने, रिमोट एप्लिकेशन से कनेक्ट करने, या मौजूदा स्नैपशॉट खोलने की अनुमति देगी। आप प्रोफाइलर की विशेषताओं का पता लगाने के लिए एक डेमो एप्लिकेशन भी लोड कर सकते हैं।

8.  **अतिरिक्त नोट्स**:
    *   यदि आपको कोई समस्या आती है, तो सुनिश्चित करें कि आपके सिस्टम पर Java (वर्जन 8 या उच्चतर) इंस्टॉल है, क्योंकि YourKit Java Profiler के लिए एक संगत JVM की आवश्यकता होती है। आप इस कमांड से अपना Java वर्जन चेक कर सकते हैं:
        ```bash
        java -version
        ```
    *   रिमोट प्रोफाइलिंग के लिए, सुनिश्चित करें कि रिमोट सर्वर पर एप्लिकेशन प्रोफाइल करते समय SSH एक्सेस कॉन्फ़िगर किया गया है।
    *   YourKit 15-दिन का निःशुल्क ट्रायल ऑफर करता है, इसलिए शुरुआत में किसी लाइसेंस कुंजी की आवश्यकता नहीं है। गैर-वाणिज्यिक ओपन-सोर्स प्रोजेक्ट्स के लिए, आप निःशुल्क लाइसेंस के लिए YourKit सेल्स से संपर्क कर सकते हैं।
    *   यदि प्रोफाइलर शुरू होने में विफल रहता है (उदाहरण के लिए, इंस्टॉलेशन डायरेक्टरी में स्पेस के कारण), तो एक्सट्रैक्ट की गई फ़ोल्डर को बिना स्पेस वाले पाथ पर ले जाएं, जैसे `/opt/yourkit`।
