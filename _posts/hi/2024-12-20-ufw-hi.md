यदि आपको कुछ सेवाओं से कनेक्ट होने में परेशानी हो रही है, तो यहां UFW (Uncomplicated Firewall) को कॉन्फ़िगर करने के लिए कुछ सुझाव दिए गए हैं।

### SSH ट्रैफ़िक की अनुमति दें (पोर्ट 22)

यदि SSH ब्लॉक हो गया है, तो इसे अनुमति देने के लिए:

```bash
sudo ufw allow ssh
```

(यह कमांड SSH को अनुमति देने के लिए UFW (Uncomplicated Firewall) को कॉन्फ़िगर करता है। इसे हिंदी में अनुवादित करने की आवश्यकता नहीं है क्योंकि यह एक टर्मिनल कमांड है और इसे वैसे ही रहना चाहिए।)

### V2Ray ट्रैफ़िक की अनुमति दें (पोर्ट 1080 या 443)

V2Ray या अन्य सेवाओं के लिए, आवश्यक पोर्ट्स को अनुमति दें:

```bash
sudo ufw allow 1080/tcp
```

(यह कमांड TCP पोर्ट 1080 को UFW (Uncomplicated Firewall) में अनुमति देने के लिए है। इसे हिंदी में अनुवादित करने की आवश्यकता नहीं है क्योंकि यह एक टर्मिनल कमांड है और इसे मूल रूप में ही उपयोग किया जाता है।)

या

```bash
sudo ufw allow 443/tcp
```

यह कमांड `UFW` (Uncomplicated Firewall) में 443 पोर्ट पर TCP ट्रैफ़िक को अनुमति देने के लिए उपयोग की जाती है। इसे चलाने से आपके सर्वर पर HTTPS कनेक्शन (जो आमतौर पर 443 पोर्ट का उपयोग करता है) को अनुमति मिल जाएगी।

### UFW स्थिति की जाँच करें

सक्रिय फ़ायरवॉल नियम देखने के लिए, निम्नलिखित कमांड का उपयोग करें:

```bash
sudo ufw status verbose
```

### UFW को फिर से सक्षम करें

यदि UFW अक्षम है, तो इसे सक्षम करने के लिए निम्नलिखित कमांड का उपयोग करें:

```bash
sudo ufw enable
```

(यह कमांड UFW (Uncomplicated Firewall) को सक्षम करता है। इसे हिंदी में अनुवादित करने की आवश्यकता नहीं है क्योंकि यह एक टर्मिनल कमांड है और इसे मूल रूप में ही प्रयोग किया जाता है।)

### UFW को रीसेट करें

यदि आप UFW (Uncomplicated Firewall) को रीसेट करना चाहते हैं, तो निम्नलिखित कदमों का पालन करें:

1. **UFW को डिसेबल करें:**
   ```bash
   sudo ufw disable
   ```

2. **सभी नियमों को रीसेट करें:**
   ```bash
   sudo ufw reset
   ```

3. **UFW को फिर से सक्षम करें:**
   ```bash
   sudo ufw enable
   ```

इन कदमों को पूरा करने के बाद, UFW अपने डिफ़ॉल्ट सेटिंग्स पर वापस आ जाएगा और सभी पिछले नियम हटा दिए जाएंगे।

नए सिरे से शुरू करने के लिए, UFW को रीसेट करें:

```bash
sudo ufw reset
```

### कस्टम पोर्ट्स

```bash
sudo ufw allow 1024:65535/tcp
sudo ufw allow 1024:65535/udp
```

इस कमांड का उपयोग करके, आप UFW (Uncomplicated Firewall) में 1024 से 65535 तक के TCP और UDP पोर्ट्स को अनुमति दे सकते हैं। यह सुनिश्चित करता है कि इन पोर्ट्स के माध्यम से नेटवर्क ट्रैफिक को अनुमति दी जाए।

अगर आपको किसी विशिष्ट UFW कॉन्फ़िगरेशन में मदद की ज़रूरत हो, तो मुझे बताएं!
