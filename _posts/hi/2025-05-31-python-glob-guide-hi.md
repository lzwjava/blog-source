### ग्लोब आयात करना
पायथन में ग्लोब का उपयोग करने के लिए, मानक लाइब्रेरी से मॉड्यूल आयात करें:
```python
import glob
```

### मूल कार्यक्षमता
`glob.glob()` फ़ंक्शन एक पैटर्न स्ट्रिंग लेता है और मेल खाने वाली फ़ाइल या डायरेक्टरी पथों की एक सूची देता है। यह यूनिक्स शेल-स्टाइल वाइल्डकार्ड का उपयोग करता है:

- `*`: किसी भी वर्ण अनुक्रम से मेल खाता है (पथ विभाजकों को छोड़कर)।
- `?`: किसी भी एकल वर्ण से मेल खाता है।
- `[seq]`: seq में किसी भी वर्ण से मेल खाता है (उदाहरण के लिए, अंकों के लिए `[0-9]`)।
- `**`: डायरेक्टरी को पुनरावर्ती रूप से मेल करता है (इसके लिए `recursive=True` की आवश्यकता होती है)।

एक निरपेक्ष पथ निर्दिष्ट किए जाने तक, पैटर्न वर्तमान डायरेक्टरी के सापेक्ष होते हैं।

### उपयोग के उदाहरण

#### 1. वर्तमान डायरेक्टरी में सभी .txt फ़ाइलें ढूंढें
```python
import glob

files = glob.glob("*.txt")
print(files)  # आउटपुट: ['file1.txt', 'file2.txt', ...]
```

#### 2. किसी उपडायरेक्टरी में विशिष्ट एक्सटेंशन वाली फ़ाइलें ढूंढें
```python
files = glob.glob("documents/*.pdf")
print(files)  # आउटपुट: ['documents/doc1.pdf', ...]
```

#### 3. विशिष्ट पैटर्न से मेल करें (उदाहरण के लिए, 'log' से शुरू होने वाली और संख्याओं पर समाप्त होने वाली फ़ाइलें)
```python
files = glob.glob("log[0-9].txt")
print(files)  # आउटपुट: ['log1.txt', 'log2.txt', ...] यदि ऐसी फ़ाइलें मौजूद हैं
```

#### 4. उपडायरेक्टरी में पुनरावर्ती खोज
डायरेक्टरी और उपडायरेक्टरी में खोज करने के लिए `recursive=True` के साथ `**` का उपयोग करें:
```python
files = glob.glob("**/*.py", recursive=True)
print(files)  # आउटपुट: ['script1.py', 'subdir/script2.py', ...]
```

### महत्वपूर्ण नोट्स
- परिणाम डिफ़ॉल्ट रूप से क्रमबद्ध नहीं होते हैं; यदि आपको क्रम की आवश्यकता है तो `sorted()` का उपयोग करें: `sorted(glob.glob("*.txt"))`।
- केस-संवेदी फ़ाइलसिस्टम पर केस-असंवेदी मिलान के लिए, `fnmatch` के साथ संयोजन करने पर विचार करें या कोड में हैंडल करें, क्योंकि ग्लोब केस-संवेदी है।
- ग्लोब सांकेतिक लिंक का अनुसरण नहीं करता है जब तक कि स्पष्ट रूप से हैंडल न किया गया हो; अधिक जटिल ट्रैवर्सल के लिए `os.walk()` का उपयोग करें।
- एक बार में एक मिलान के लिए, `glob.iglob()` का उपयोग करें जो बड़े परिणामों पर मेमोरी बचाने के लिए एक इटरेटर देता है: `for file in glob.iglob("*.txt"): print(file)`।

यह मूल बातें कवर करता है; अधिक जानकारी के लिए [पायथन डॉक्यूमेंटेशन](https://docs.python.org/3/library/glob.html) देखें।
