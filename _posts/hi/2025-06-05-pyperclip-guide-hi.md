### Pyperclip क्या है?
Pyperclip सिस्टम क्लिपबोर्ड (कॉपी-पेस्ट फंक्शनैलिटी) तक पहुंचने और उसे मैनिपुलेट करने के लिए एक हल्का Python लाइब्रेरी है। यह क्रॉस-प्लेटफॉर्म (Windows, macOS, Linux) पर काम करता है और टेक्स्ट को क्लिपबोर्ड पर कॉपी करने तथा उससे टेक्स्ट पेस्ट करने का सपोर्ट करता है। ध्यान दें: यह केवल टेक्स्ट को हैंडल करता है; फाइलों या इमेज के लिए अन्य लाइब्रेरीज का उपयोग करें।

### इंस्टालेशन
Pyperclip को pip के माध्यम से इंस्टॉल करें:
```
pip install pyperclip
```

Linux पर, क्लिपबोर्ड एक्सेस के लिए आपको अतिरिक्त डिपेंडेंसीज (जैसे `xclip` या `xsel`) की आवश्यकता हो सकती है। उन्हें अपने पैकेज मैनेजर के साथ इंस्टॉल करें (उदाहरण के लिए, Ubuntu पर `sudo apt-get install xclip`)।

### बेसिक उपयोग
1. **लाइब्रेरी को इम्पोर्ट करें**:
   ```python
   import pyperclip
   ```

2. **टेक्स्ट को क्लिपबोर्ड पर कॉपी करें**:
   ```python
   pyperclip.copy("Hello, world!")
   ```
   यह स्ट्रिंग "Hello, world!" को क्लिपबोर्ड पर रख देता है।

3. **क्लिपबोर्ड से टेक्स्ट पेस्ट करें**:
   ```python
   text = pyperclip.paste()
   print(text)  # वर्तमान क्लिपबोर्ड कंटेंट को आउटपुट करता है
   ```
   यह क्लिपबोर्ड के टेक्स्ट को रिट्रीव और प्रिंट करता है।

### उदाहरण: यूजर इनपुट को क्लिपबोर्ड पर कॉपी करना
```python
import pyperclip

user_input = input("कॉपी करने के लिए टेक्स्ट दर्ज करें: ")
pyperclip.copy(user_input)
print("टेक्स्ट क्लिपबोर्ड पर कॉपी हो गया!")
```

### उदाहरण: पेस्ट करना और मॉडिफाई करना
```python
import pyperclip

pyperclip.copy("Original text")
modified = pyperclip.paste().upper()  # पेस्ट करें, फिर मॉडिफाई करें (जैसे, अपरकेस)
pyperclip.copy(modified)
print("मॉडिफाइड टेक्स्ट कॉपी हो गया।")
```

### अतिरिक्त टिप्स
- **एरर हैंडलिंग**: कुछ सिस्टम पर (जैसे हेडलेस सर्वर या कुछ Linux डिस्ट्रो), क्लिपबोर्ड एक्सेस फेल हो सकता है। ट्राई-एक्सेप्ट में रैप करें:
  ```python
  try:
      pyperclip.copy("text")
  except pyperclip.PyperclipException as e:
      print(f"क्लिपबोर्ड एरर: {e}")
  ```
- **उपलब्धता जांचना**: स्क्रिप्ट रन होने के तुरंत बाद `pyperclip.paste()` का उपयोग करें, क्योंकि क्लिपबोर्ड कंटेंट रन के बीच में बना रहता है।
- **लिमिटेशंस**: बाइनरी डेटा के लिए उपयुक्त नहीं; केवल टेक्स्ट। एडवांस्ड उपयोग के लिए, [ऑफिशियल डॉक्यूमेंटेशन](https://github.com/asweigart/pyperclip) देखें।
