Ubuntu पर Snap के माध्यम से इंस्टॉल किए गए Visual Studio Code (VSCode) को अपडेट करने के लिए, इन चरणों का पालन करें:

1. **टर्मिनल खोलें**: आप `Ctrl + Alt + T` दबाकर ऐसा कर सकते हैं।

2. **अपडेट के लिए जाँच करें**: Snap स्वचालित रूप से पृष्ठभूमि में अपडेट की जाँच करता है, लेकिन आप मैन्युअल रूप से अपडेट जाँच ट्रिगर कर सकते हैं इस कमांड को चलाकर:
   ```bash
   sudo snap refresh
   ```

3. **VSCode अपडेट करें**: यदि VSCode का कोई नया संस्करण उपलब्ध है, तो `snap refresh` कमांड इसे डाउनलोड और इंस्टॉल कर देगा। आप प्रगति दर्शाता आउटपुट देखेंगे, जैसे:
   ```
   code 1.85.2 from Visual Studio Code (code✓) refreshed
   ```
   यदि कोई अपडेट उपलब्ध नहीं है, तो यह कहेगा:
   ```
   All snaps up to date.
   ```

4. **अपडेट सत्यापित करें**: VSCode के इंस्टॉल किए गए संस्करण की पुष्टि करने के लिए, चलाएँ:
   ```bash
   code --version
   ```
   यह VSCode का वर्तमान संस्करण प्रदर्शित करेगा।

### नोट्स:
- **स्वचालित अपडेट**: Snap पैकेज, जिनमें VSCode शामिल है, आमतौर पर पृष्ठभूमि में स्वचालित रूप से अपडेट हो जाते हैं। आपको आमतौर पर तब तक मैन्युअल रूप से हस्तक्षेप करने की आवश्यकता नहीं होती, जब तक कि आप जबरन अपडेट करना न चाहें।
- **रूट विशेषाधिकार**: सिस्टम-वाइड Snap ऑपरेशनों के लिए `sudo` कमांड आवश्यक है।
- **Snap स्थिति जाँचें**: यदि आप यह देखना चाहते हैं कि कौन से Snap पैकेज इंस्टॉल हैं और उनके संस्करण क्या हैं, तो उपयोग करें:
   ```bash
   snap list
   ```
- **समस्या निवारण**: यदि अपडेट विफल हो जाता है, तो सुनिश्चित करें कि आपका इंटरनेट कनेक्शन स्थिर है और Snap डेमन चल रहा है (`sudo systemctl status snapd`)। आप Snap मेटाडेटा को इसके साथ रिफ्रेश करने का भी प्रयास कर सकते हैं:
   ```bash
   sudo snap refresh --list
   ```

यह प्रक्रिया सुनिश्चित करती है कि VSCode, Snap स्टोर में उपलब्ध नवीनतम संस्करण के साथ अप टू डेट रहे।
