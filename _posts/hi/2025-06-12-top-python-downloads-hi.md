- **path.py**: पाथ मैनिपुलेशन लाइब्रेरी (पुराना वर्जन)।
- **python-slugify**: स्ट्रिंग के लिए यूनिकोड स्लग लौटाता है।
- **unidecode**: यूनिकोड टेक्स्ट के ASCII ट्रांसलिटरेशन।
- **cldr**: Python के लिए CLDR यूटिलिटीज़।
- **babel**: इंटरनेशनलाइज़ेशन यूटिलिटीज़।
- **pytzdata**: टाइमज़ोन लाइब्रेरी के लिए टाइमज़ोन डेटा।
- **bottle**: छोटे एप्लिकेशन के लिए फास्ट, सरल वेब फ्रेमवर्क।
- **cherrypy**: Python के लिए मिनिमलिस्ट वेब फ्रेमवर्क।
- **tornado**: वेब फ्रेमवर्क और एसिंक्रोनस नेटवर्किंग लाइब्रेरी।
- **aiohttp**: asyncio के लिए एसिंक्रोनस HTTP क्लाइंट/सर्वर।
- **asyncpg**: async Python के लिए फास्ट PostgreSQL ड्राइवर।
- **httpx**: Python के लिए Async HTTP क्लाइंट।
- **uvicorn**: Python के लिए लाइटनिंग-फास्ट ASGI सर्वर।
- **fastapi**: आधुनिक, फास्ट वेब फ्रेमवर्क APIs के लिए।

- **pyramid**: Python के लिए वेब फ्रेमवर्क।

- **mako**: Python के लिए फास्ट टेम्पलेट लाइब्रेरी।
- **redis-py-client**: redis का डुप्लिकेट (ऊपर देखें)।

- **amqp**: Python के लिए AMQP लाइब्रेरी।
- **msgpack**: Python के लिए MessagePack सीरियलाइज़ेशन।
- **pickle5**: पिकल प्रोटोकॉल 5 का बैकपोर्ट।
- **cloudpickle**: Python के लिए एक्सटेंडेड पिकलिंग सपोर्ट।
- **dill**: Python के सभी को सीरियलाइज़ करें।
- **multiprocess**: बेहतर मल्टीप्रोसेसिंग और मल्टीथ्रेडिंग।
- **pexpect**: Python से इंटरैक्टिव प्रोग्राम को नियंत्रित करें।
- **ptyprocess**: एक स्यूडो-टर्मिनल से जुड़े इंटरैक्टिव प्रोसेस चलाएँ।
- **subprocess32**: Python 3.2 से subprocess का बैकपोर्ट।
- **threadpoolctl**: थ्रेड-ओनली कंप्यूट में समय सीमित करने के लिए Python हेल्पर्स।
