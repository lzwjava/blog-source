प्रदान किए गए MicroPython कोड को Raspberry Pi Pico पर सेट अप करने के लिए, जो एक बटन दबाने पर LED को नियंत्रित करता है, इन चरणों का पालन करें:

### हार्डवेयर आवश्यकताएँ
- **Raspberry Pi Pico** (या Pico W)
- **LED** (एक उपयुक्त करंट-लिमिटिंग रेसिस्टर के साथ, जैसे 220–330 ओम)
- **पुशबटन स्विच**
- **जम्पर वायर**
- **ब्रेडबोर्ड** (वैकल्पिक, आसान कनेक्शन के लिए)
- **USB केबल** (Pico को आपके कंप्यूटर से कनेक्ट करने के लिए)

### हार्डवेयर सेटअप
1. **LED को कनेक्ट करें**:
   - LED के एनोड (लंबा पिन) को Pico पर **GPIO 25** (भौतिक पिन 30) से कनेक्ट करें।
   - LED के कैथोड (छोटा पिन) को करंट को सीमित करने के लिए 220–330 ओम रेसिस्टर के माध्यम से एक **ग्राउंड (GND)** पिन (जैसे, भौतिक पिन 3) से कनेक्ट करें।

2. **बटन को कनेक्ट करें**:
   - पुशबटन के एक सिरे को **GPIO 14** (भौतिक पिन 19) से कनेक्ट करें।
   - बटन के दूसरे सिरे को एक **3.3V** पिन (जैसे, भौतिक पिन 36, 3V3(OUT)) से कनेक्ट करें।
   - कोड एक आंतरिक पुल-डाउन रेसिस्टर (`Pin.PULL_DOWN`) का उपयोग करता है, इसलिए किसी बाहरी पुल-डाउन रेसिस्टर की आवश्यकता नहीं है। जब बटन दबाया जाता है, तो GPIO 14 HIGH (1) रीड करेगा; जब दबाया नहीं जाता, तो यह LOW (0) रीड करेगा।

3. **कनेक्शन सत्यापित करें**:
   - सुनिश्चित करें कि सभी कनेक्शन सुरक्षित हैं। ब्रेडबोर्ड या डायरेक्ट वायरिंग का उपयोग करें, और डबल-चेक करें कि LED पोलैरिटी सही है और रेसिस्टर ठीक से लगा हुआ है।
   - पिन असाइनमेंट की पुष्टि करने के लिए Pico पिनआउट डायग्राम (ऑनलाइन उपलब्ध या Pico डेटाशीट में) देखें।

### सॉफ्टवेयर सेटअप
1. **Pico पर MicroPython इंस्टॉल करें**:
   - [आधिकारिक MicroPython वेबसाइट](https://micropython.org/download/rp2-pico/) से Raspberry Pi Pico के लिए नवीनतम MicroPython UF2 फर्मवेयर डाउनलोड करें।
   - **BOOTSEL** बटन दबाए रखते हुए Pico को USB केबल के माध्यम से अपने कंप्यूटर से कनेक्ट करें।
   - Pico एक USB ड्राइव (RPI-RP2) के रूप में दिखाई देगा। डाउनलोड किए गए `.uf2` फाइल को इस ड्राइव पर ड्रैग और ड्रॉप करें।
   - Pico MicroPython इंस्टॉल होने के साथ स्वचालित रूप से रीबूट हो जाएगा।

2. **एक डेवलपमेंट एनवायरनमेंट सेट अप करें**:
   - एक MicroPython-संगत IDE इंस्टॉल करें, जैसे **Thonny** (शुरुआती लोगों के लिए अनुशंसित):
     - [thonny.org](https://thonny.org) से Thonny डाउनलोड करें और इंस्टॉल करें।
     - Thonny में, **Tools > Options > Interpreter** पर जाएं, **MicroPython (Raspberry Pi Pico)** चुनें, और उपयुक्त पोर्ट (जैसे, Windows पर `COMx` या Linux/macOS पर `/dev/ttyACM0`) चुनें।
   - वैकल्पिक रूप से, आप `rshell`, `ampy`, या Visual Studio Code जैसे टूल्स का उपयोग कर सकते हैं।

3. **कोड अपलोड और रन करें**:
   - प्रदान किए गए कोड को `main.py` नामक एक फाइल में कॉपी करें:
     ```python
     from machine import Pin
     import time

     led = Pin(25, Pin.OUT)
     button = Pin(14, Pin.IN, Pin.PULL_DOWN)

     while True:
         if button.value():
             led.on()
         else:
             led.off()
         time.sleep(0.05)
     ```
   - Thonny में:
     - एक नई फाइल खोलें, कोड पेस्ट करें, और इसे Pico पर `main.py` के रूप में सेव करें (MicroPython बूट पर स्वचालित रूप से `main.py` रन करता है)।
     - कोड अपलोड और एक्ज़िक्यूट करने के लिए **Run** बटन क्लिक करें या **F5** दबाएं।
   - वैकल्पिक रूप से, फाइल अपलोड करने के लिए `ampy` का उपयोग करें:
     ```bash
     ampy --port /dev/ttyACM0 put main.py
     ```
     `/dev/ttyACM0` को अपने Pico के पोर्ट से बदलें।

4. **प्रोग्राम टेस्ट करें**:
   - एक बार कोड अपलोड हो जाने के बाद, बटन दबाने पर LED ऑन हो जानी चाहिए और छोड़ने पर ऑफ हो जानी चाहिए। `time.sleep(0.05)` बटन को डिबाउंस करने और CPU लोड कम करने के लिए 50ms की देरी जोड़ता है।

### समस्या निवारण
- **LED लाइट नहीं होती**:
  - LED पोलैरिटी चेक करें (एनोड GPIO 25 से, कैथोड GND से रेसिस्टर के माध्यम से)।
  - सत्यापित करें कि GPIO पिन (25) कोड और भौतिक कनेक्शन से मेल खाता है।
  - सुनिश्चित करें कि LED और रेसिस्टर कार्यात्मक हैं (मल्टीमीटर या दूसरे सर्किट से टेस्ट करें)।
- **बटन काम नहीं करता**:
  - पुष्टि करें कि GPIO 14 सही ढंग से कनेक्ट है और बटन दबाने पर 3.3V से कनेक्ट होता है।
  - चेक करें कि `Pin.PULL_DOWN` उपयुक्त है; यदि बटन अलग तरीके से वायर्ड है (जैसे, GND से), तो कोड को `Pin.PULL_UP` का उपयोग करने और लॉजिक को रिवर्स करने के लिए एडजस्ट करें (`if not button.value()`)।
- **कोड रन नहीं होता**:
  - सुनिश्चित करें कि MicroPython सही ढंग से इंस्टॉल है।
  - सत्यापित करें कि फाइल Pico पर `main.py` के रूप में सेव है।
  - अपने IDE में USB कनेक्शन और पोर्ट सेटिंग्स चेक करें।

### नोट्स
- कोड यह मानता है कि बटन दबाने पर GPIO 14 को 3.3V से कनेक्ट करता है। यदि आपका बटन GND से कनेक्ट होता है, तो कोड को `Pin.PULL_UP` का उपयोग करने और लॉजिक को रिवर्स करने के लिए मॉडिफाई करें:
  ```python
  button = Pin(14, Pin.IN, Pin.PULL_UP)
  while True:
      if not button.value():  # बटन दबा = LOW
          led.on()
      else:
          led.off()
      time.sleep(0.05)
  ```
- यदि आप बटन बाउंस (फ्लिकरिंग LED) का अनुभव करते हैं, तो `time.sleep` वैल्यू बढ़ाएं या सॉफ्टवेयर डिबाउंसिंग इम्प्लीमेंट करें।
- Pico W के लिए, सेटअप समान है, लेकिन सुनिश्चित करें कि आप सही GPIO पिन्स का उपयोग कर रहे हैं, क्योंकि पिनआउट समान है।
