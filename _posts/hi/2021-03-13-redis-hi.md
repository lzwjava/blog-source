title: '# Redis का परिचय


  Redis (Remote Dictionary Server) एक ओपन-सोर्स, इन-मेमोरी डेटा स्ट्रक्चर स्टोर है,
  जिसका उपयोग डेटाबेस, कैश, और मैसेज ब्रोकर के रूप में किया जाता है। यह विभिन्न प्रकार
  के डेटा स्ट्रक्चर्स जैसे स्ट्रिंग्स, हैश, लिस्ट, सेट, और सॉर्टेड सेट का समर्थन करता
  है। Redis अपनी उच्च प्रदर्शन और कम विलंबता के लिए जाना जाता है, जो इसे रीयल-टाइम
  एप्लिकेशन्स के लिए एक आदर्श विकल्प बनाता है।


  ## Redis की मुख्य विशेषताएं


  1. **इन-मेमोरी स्टोरेज**: Redis डेटा को मेमोरी में स्टोर करता है, जो इसे बहुत तेज़
  बनाता है।

  2. **डेटा स्ट्रक्चर्स**: Redis विभिन्न डेटा स्ट्रक्चर्स का समर्थन करता है, जैसे
  स्ट्रिंग्स, हैश, लिस्ट, सेट, और सॉर्टेड सेट।

  3. **परसिस्टेंस**: Redis डेटा को डिस्क पर सहेज सकता है, जिससे डेटा की स्थायित्व
  सुनिश्चित होती है।

  4. **रिप्लिकेशन**: Redis मास्टर-स्लेव रिप्लिकेशन का समर्थन करता है, जो डेटा की उपलब्धता
  और विश्वसनीयता को बढ़ाता है।

  5. **ट्रांजैक्शन्स**: Redis ट्रांजैक्शन्स का समर्थन करता है, जिससे एक से अधिक कमांड्स
  को एक साथ निष्पादित किया जा सकता है।


  ## Redis का उपयोग


  Redis का उपयोग विभिन्न प्रकार के एप्लिकेशन्स में किया जाता है, जैसे:


  - **कैशिंग**: Redis का उपयोग वेब एप्लिकेशन्स में कैश के रूप में किया जाता है ताकि
  डेटाबेस लोड को कम किया जा सके।

  - **सेशन स्टोरेज**: Redis का उपयोग यूजर सेशन डेटा को स्टोर करने के लिए किया जाता
  है।

  - **रियल-टाइम एनालिटिक्स**: Redis का उपयोग रियल-टाइम डेटा एनालिटिक्स के लिए किया
  जाता है।

  - **मैसेज ब्रोकर**: Redis का उपयोग मैसेज ब्रोकर के रूप में किया जाता है, जो एप्लिकेशन्स
  के बीच संदेशों का आदान-प्रदान करता है।


  ## Redis इंस्टॉलेशन


  Redis को Linux, macOS, और Windows पर इंस्टॉल किया जा सकता है। नीचे Linux पर Redis
  इंस्टॉल करने के लिए कदम दिए गए हैं:


  ```bash

  # Redis इंस्टॉल करें

  sudo apt-get update

  sudo apt-get install redis-server


  # Redis सर्वर शुरू करें

  sudo systemctl start redis-server


  # Redis सर्वर की स्थिति जांचें

  sudo systemctl status redis-server

  ```


  ## Redis कमांड्स


  Redis CLI (Command Line Interface) का उपयोग करके Redis सर्वर के साथ इंटरैक्ट किया
  जा सकता है। नीचे कुछ बुनियादी Redis कमांड्स दिए गए हैं:


  ```bash

  # Redis CLI शुरू करें

  redis-cli


  # एक key-value जोड़ी सेट करें

  SET mykey "Hello Redis"


  # एक key का मान प्राप्त करें

  GET mykey


  # एक key को डिलीट करें

  DEL mykey


  # सभी keys प्रदर्शित करें

  KEYS *

  ```


  ## निष्कर्ष


  Redis एक शक्तिशाली और बहुमुखी टूल है जो विभिन्न प्रकार के एप्लिकेशन्स के लिए उपयुक्त
  है। इसकी उच्च प्रदर्शन और विभिन्न डेटा स्ट्रक्चर्स के समर्थन के कारण, Redis आधुनिक
  एप्लिकेशन डेवलपमेंट में एक महत्वपूर्ण भूमिका निभाता है।'

Redis की आधिकारिक वेबसाइट खोलें, तो पहला वाक्य यह कहता है कि Redis एक ओपन-सोर्स, इन-मेमोरी डेटा स्ट्रक्चर स्टोर है, जिसे अक्सर डेटाबेस और कैशिंग के लिए उपयोग किया जाता है। `Redis` बहुत ही आम तौर पर उपयोग किया जाता है।

`pip` आमतौर पर `Python` इंस्टॉल करते समय पहले से ही शामिल होता है। यदि आपके कंप्यूटर में कई संस्करणों के `Python` और `Pip` हैं, तो आप `~/.bash_profile` में निम्नलिखित दो पंक्तियाँ जोड़ सकते हैं:

## Redis कैश उदाहरण

फिबोनाची अनुक्रम का `Redis` संस्करण लागू करने के लिए, हम Redis को एक कैशिंग लेयर के रूप में उपयोग कर सकते हैं। यह हमें पहले से गणना किए गए फिबोनाची मानों को स्टोर करने और उन्हें बाद में पुनः प्राप्त करने की अनुमति देता है, जिससे प्रदर्शन में सुधार होता है।

नीचे एक उदाहरण दिया गया है कि कैसे हम Redis का उपयोग करके फिबोनाची अनुक्रम को लागू कर सकते हैं:

```python
import redis
import time

# Redis कनेक्शन स्थापित करें
r = redis.Redis(host='localhost', port=6379, db=0)

def fibonacci(n):
    # यदि मान Redis में पहले से मौजूद है, तो उसे वापस लौटाएं
    if r.exists(n):
        return int(r.get(n))
    
    # बेस केस
    if n <= 1:
        return n
    
    # फिबोनाची अनुक्रम की गणना करें
    result = fibonacci(n-1) + fibonacci(n-2)
    
    # परिणाम को Redis में स्टोर करें
    r.set(n, result)
    
    return result

# उदाहरण के लिए, 10वें फिबोनाची संख्या की गणना करें
start_time = time.time()
print(fibonacci(10))
end_time = time.time()
print(f"Time taken: {end_time - start_time} seconds")
```

### व्याख्या:
1. **Redis कनेक्शन**: हम `redis.Redis` का उपयोग करके Redis सर्वर से कनेक्ट होते हैं।
2. **फिबोनाची फ़ंक्शन**: `fibonacci` फ़ंक्शन पुनरावर्ती रूप से फिबोनाची अनुक्रम की गणना करता है। यदि मान पहले से ही Redis में स्टोर है, तो उसे सीधे वापस लौटा दिया जाता है।
3. **कैशिंग**: गणना किए गए मानों को Redis में स्टोर किया जाता है ताकि भविष्य में उन्हें दोबारा गणना करने की आवश्यकता न पड़े।
4. **प्रदर्शन**: Redis का उपयोग करने से पुनरावर्ती गणना का समय काफी कम हो जाता है, खासकर बड़े `n` के लिए।

### उदाहरण आउटपुट:
```
55
Time taken: 0.0005 seconds
```

यह कोड फिबोनाची अनुक्रम की गणना करता है और Redis का उपयोग करके प्रदर्शन को अनुकूलित करता है।

```python
import redis
```

```python
r = redis.Redis(host='localhost', port=6379, db=0)
```

```python
def f(n):
    nr = r.get(n)
    if nr is not None:
        return int(nr)
    res_n = 0
    if n < 2:
        res_n = n
    else:
        res_n = f(n-1) + f(n-2)
    
    r.set(n, res_n)
    return res_n
```

```python
print(f(10))
```
