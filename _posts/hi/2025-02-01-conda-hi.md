## समस्याएँ

इस लिंक को देखें: [https://www.reddit.com/r/MachineLearning/comments/myl8yn/d_is_anaconda_worth_the_trouble/](https://www.reddit.com/r/MachineLearning/comments/myl8yn/d_is_anaconda_worth_the_trouble/).

मुझे conda के साथ समस्याएँ हुई हैं, और अन्य लोग भी समान समस्याओं के बारे में चर्चा करते हुए मिले हैं।

समय के साथ आप環境 सेटिंग्स के बारे में अनजान हो जाते हैं। तो, मुझे दूसरों को हटाना पड़ा। अब, मैं मुख्य रूप से `Python 3.10.9` का उपयोग करता हूँ और कभी-कभी `Python 3.11.11` का उपयोग करता हूँ। फिर भी, मैं `Python 3.11.11` का भी कम उपयोग करता हूँ। व्यक्तिगत रूप से मुझे बहुत से環境 की जरूरत नहीं होती। मुझे केवल एक या दो ऐसी चालान चालान को रन करने की जरूरत होती है जो मैं वर्तमान में शोध या सीख रहा हूँ।

मुझे एक सरल समाधान पसंद है। यदि मेरे दो परियोजनाओं में आपस में टकराव हो रहा है, तो मैं एक का उपयोग करना पसंद करता हूँ और दूसरा चलाने से इनकार कर देता हूँ।

उदाहरण के लिए, नीचे दर्ज किए गए समस्या को मिलने का एक वास्तव में चिढ़ है।[^1]

```bash
conda config --set proxy_servers.http $HTTP_PROXY
conda config --set proxy_servers.https $HTTP_PROXY
```

हालांकि conda के लिए एक प्रॉक्सी को सेट करने की शुरुआत में तेज़ होने का अंदेशा था, मैं अंत में उसे हटाने का फैसला लेता था।

## हटाना

```bash
rm -rf ~/anaconda3
rm -rf ~/.condarc
rm -rf ~/Library/Caches/pip
# update ~/.zprofile
# update ~/.bash_profile
```

`~/.bash_profile`:

```
# >>> conda initialize >>>
# !! इस ब्लॉक के भीतर की सामग्री 'conda init' द्वारा प्रबंधित की जाती है !!
__conda_setup="$('/Users/lzwjava/anaconda3/bin/conda' 'shell.bash' 'hook' 2> /dev/null)"
if [ $? -eq 0 ]; then
    eval "$__conda_setup"
else
    if [ -f "/Users/lzwjava/anaconda3/etc/profile.d/conda.sh" ]; then
        . "/Users/lzwjava/anaconda3/etc/profile.d/conda.sh"
    else
        export PATH="/Users/lzwjava/anaconda3/bin:$PATH"
    fi
fi
unset __conda_setup
# <<< conda initialize <<<
```

और फिर भी दिखायी दिया:

```bash
% which conda
conda () {
	local cmd="${1-__missing__}"
	case "$cmd" in
		(activate | deactivate) __conda_activate "$@" ;;
		(install | update | upgrade | remove | uninstall) __conda_exe "$@" || return
			__conda_reactivate ;;
		(*) __conda_exe "$@" ;;
	esac
}
```

iTerm2 को फिर से शुरू करने के बाद, मैं Homebrew की Python इंस्टॉलेशन पर स्विच करने का फैसला लेता था। ऐसा लग रहा था जैसे कि मेरे कंप्यूटर में कई Python इंस्टॉलेशन हैं; मुझे पहले से छह मिले हैं, जैसा नीचे दिखाया गया है।
