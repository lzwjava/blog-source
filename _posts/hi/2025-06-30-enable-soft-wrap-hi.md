Zed, कई आधुनिक टेक्स्ट एडिटर्स की तरह, लाइनों को रैप करने के लिए "सॉफ्ट रैप" की अवधारणा का उपयोग करता है। इसका मतलब है कि लाइनें एडिटर में रैप हुई दिखाई देंगी, लेकिन आपकी फ़ाइल में वास्तविक न्यूलाइन करैक्टर नहीं डाले जाते हैं।

Zed में लाइन रैपिंग को सक्षम करने का तरीका यहां बताया गया है:

1.  **Zed के सेटिंग्स खोलें:**

      * कमांड पैलेट खोलने के लिए `Command + Shift + P` (macOS) या `Ctrl + Shift + P` (Windows/Linux) दबाएं।
      * "local" टाइप करें और "zed: Open Local Settings" चुनें। इससे आपकी `settings.json` फ़ाइल खुल जाएगी।

2.  **`soft_wrap` सेटिंग जोड़ें:**

      * अपनी `settings.json` फ़ाइल में, `"soft_wrap"` सेटिंग जोड़ें या संशोधित करें। आपके पास कुछ विकल्प हैं:

          * **`"editor_width"`:** यह सबसे आम और आम तौर पर अनुशंसित विकल्प है। यह लाइनों को आपकी एडिटर विंडो की वर्तमान चौड़ाई में फिट करने के लिए रैप करेगा।
            ```json
            {
              "soft_wrap": "editor_width"
            }
            ```
          * **`"preferred_line_length"`:** यह एक निर्दिष्ट `preferred_line_length` (उदाहरण के लिए, 80 करैक्टर) के आधार पर लाइनों को रैप करेगा। आपको `preferred_line_length` को भी परिभाषित करना होगा।
            ```json
            {
              "soft_wrap": "preferred_line_length",
              "preferred_line_length": 80
            }
            ```
          * **`"bounded"`:** यह विकल्प एक अधिक उन्नत रैपिंग व्यवहार प्रदान करता है, हालांकि `editor_width` या `preferred_line_length` ज्यादातर उपयोगकर्ताओं के लिए पर्याप्त होते हैं।
          * **`"none"`:** यह लाइन रैपिंग को अक्षम कर देता है।

3.  **फ़ाइल सेव करें:**

      * `settings.json` फ़ाइल को सेव करने के लिए `Command + S` (macOS) या `Ctrl + S` (Windows/Linux) दबाएं।

Zed अब आपकी चुनी हुई सेटिंग के अनुसार लाइनों को रैप करेगा।

**महत्वपूर्ण नोट्स:**

  * **प्रोजेक्ट-लेवल सेटिंग्स:** आप अपनी प्रोजेक्ट डायरेक्टरी में एक `.zed` फोल्डर बनाकर और उसके अंदर वांछित `soft_wrap` कॉन्फ़िगरेशन के साथ एक `settings.json` फ़ाइल रखकर प्रोजेक्ट लेवल पर भी वर्ड रैप को सक्षम कर सकते हैं। यह उस विशिष्ट प्रोजेक्ट के लिए आपकी ग्लोबल सेटिंग्स को ओवरराइड कर देगा।
  * **भाषा-विशिष्ट सेटिंग्स:** आप विशिष्ट भाषाओं के लिए `soft_wrap` को कॉन्फ़िगर भी कर सकते हैं। उदाहरण के लिए, Markdown फाइलों के लिए सॉफ्ट रैप सेट करने के लिए:
    ```json
    {
      "languages": {
        "Markdown": {
          "soft_wrap": "editor_width"
        }
      }
    }
    ```
  * **"Rewrap" कमांड:** यदि आप टेक्स्ट को "हार्ड रैप" (वास्तविक न्यूलाइन डालने) के लिए एक कमांड ढूंढ रहे हैं, न कि केवल विजुअल रैपिंग के लिए, तो Zed में एक "Rewrap" कमांड भी है (अक्सर `Alt-Q` या `Alt-W` से बाउंड)। यह कमेंट्स या लंबी गद्य लाइनों को फॉर्मेट करने के लिए उपयोगी है।
