अपने macOS पर सिस्टम जानकारी प्राप्त करने के लिए आप कई बिल्ट-इन टूल्स और कमांड्स का उपयोग कर सकते हैं। यहाँ विभिन्न प्रकार की सिस्टम जानकारी प्राप्त करने के कुछ तरीके हैं:

### 1. **सिस्टम जानकारी यूटीलीटी**
सिस्टम जानकारी ऐप आपके मैक के हार्डवेयर और सॉफ्टवेयर का एक व्यापक अवलोकन प्रदान करता है।

- **स्पॉटलाइट सर्च** को `Cmd + Space` दबाकर खोलें और "सिस्टम जानकारी" टाइप करें, फिर एंटर दबाएं।
- या तो आप इसे **एप्पल मेनू** > **इस मैक के बारे में** > **सिस्टम रिपोर्ट** से खोल सकते हैं।

### 2. **इस मैक के बारे में**
यह आपके मैक के स्पेसिफिकेशन का एक तेज़ अवलोकन प्रदान करता है।

- स्क्रीन के बाएं-ऊपरी कोने में **एप्पल मेनू** पर क्लिक करें।
- **इस मैक के बारे में** चुनें। यह बुनियादी जानकारी जैसे macOS संस्करण, प्रोसेसर, मेमोरी और सीरियल नंबर दिखाएगा।

### 3. **टर्मिनल कमांड्स**
आप टर्मिनल का उपयोग करके विभिन्न कमांड्स का उपयोग करके विस्तृत सिस्टम जानकारी प्राप्त कर सकते हैं।

- `Applications` > `Utilities` > `Terminal` से **टर्मिनल** खोलें या स्पॉटलाइट में खोजें।

#### बेसिक सिस्टम जानकारी
```sh
system_profiler SPHardwareDataType
```

#### विस्तृत सिस्टम जानकारी
```sh
system_profiler
```

#### CPU जानकारी
```sh
sysctl -n machdep.cpu.brand_string
```

#### मेमोरी जानकारी
```sh
sysctl hw.memsize
```

#### डिस्क उपयोग
```sh
df -h
```

#### नेटवर्क जानकारी
```sh
ifconfig
```

#### बैटरी जानकारी (लैपटॉप के लिए)
```sh
pmset -g batt
```

#### इंस्टॉल्ड सॉफ्टवेयर की सूची
```sh
ls /Applications
```

#### macOS संस्करण
```sh
sw_vers
```

#### यूजर जानकारी
```sh
id
```

#### चल रहे प्रोसेस की सूची
```sh
ps aux
```

#### इंस्टॉल्ड ब्रू पैकेज की सूची
```sh
brew list
```

#### इंस्टॉल्ड पिप पैकेज की सूची
```sh
pip list
```

#### इंस्टॉल्ड गेम पैकेज की सूची
```sh
gem list
```

### 4. **एक्टिविटी मॉनिटर**
यह ऐप आपके मैक के CPU, मेमोरी, ऊर्जा, डिस्क और नेटवर्क उपयोग के बारे में रियल-टाइम जानकारी प्रदान करता है।

- `Applications` > `Utilities` > `Activity Monitor` से **एक्टिविटी मॉनिटर** खोलें या स्पॉटलाइट में खोजें।

### 5. **डिस्क यूटीलीटी**
यह ऐप आपके स्टोरेज डिवाइस के बारे में जानकारी प्रदान करता है।

- `Applications` > `Utilities` > `Disk Utility` से **डिस्क यूटीलीटी** खोलें या स्पॉटलाइट में खोजें।

### 6. **थर्ड-पार्टी टूल्स**
**iStat Menus** और **CoconutBattery** जैसे थर्ड-पार्टी ऐप्लिकेशन भी हैं जो विस्तृत सिस्टम जानकारी और मॉनिटरिंग क्षमताएं प्रदान करते हैं।

इन तरीकों का उपयोग करके आप अपने macOS पर सिस्टम जानकारी का एक विस्तृत श्रृंखला इकट्ठा कर सकते हैं।
