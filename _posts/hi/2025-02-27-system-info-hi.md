अपने macOS पर सिस्टम जानकारी प्राप्त करने के लिए, आप कई अंतर्निहित टूल और कमांड्स का उपयोग कर सकते हैं। यहां विभिन्न प्रकार की सिस्टम जानकारी प्राप्त करने के कुछ तरीके दिए गए हैं:

### 1. **सिस्टम इनफॉर्मेशन यूटिलिटी**
सिस्टम इनफॉर्मेशन ऐप आपके Mac के हार्डवेयर और सॉफ़्टवेयर का एक व्यापक अवलोकन प्रदान करती है।

- **स्पॉटलाइट सर्च** खोलें `Cmd + Space` दबाकर और "System Information" टाइप करें, फिर एंटर दबाएं।
- वैकल्पिक रूप से, आप इसे **Apple मेनू** > **About This Mac** > **System Report** से खोल सकते हैं।

### 2. **About This Mac**
यह आपके Mac की विशिष्टताओं का एक त्वरित अवलोकन प्रदान करता है।

- स्क्रीन के ऊपरी-बाएं कोने में **Apple मेनू** पर क्लिक करें।
- **About This Mac** चुनें। यह macOS वर्जन, प्रोसेसर, मेमोरी और सीरियल नंबर जैसी बुनियादी जानकारी दिखाएगा।

### 3. **टर्मिनल कमांड्स**
आप विभिन्न कमांड्स का उपयोग करके विस्तृत सिस्टम जानकारी प्राप्त करने के लिए टर्मिनल का उपयोग कर सकते हैं।

- `Applications` > `Utilities` > `Terminal` से **टर्मिनल** खोलें या स्पॉटलाइट में इसकी खोज करके।

#### बेसिक सिस्टम इनफॉर्मेशन
```sh
system_profiler SPHardwareDataType
```

#### डिटेल्ड सिस्टम इनफॉर्मेशन
```sh
system_profiler
```

#### CPU इनफॉर्मेशन
```sh
sysctl -n machdep.cpu.brand_string
```

#### मेमोरी इनफॉर्मेशन
```sh
sysctl hw.memsize
```

#### डिस्क यूज़ेज
```sh
df -h
```

#### नेटवर्क इनफॉर्मेशन
```sh
ifconfig
```

#### बैटरी इनफॉर्मेशन (लैपटॉप के लिए)
```sh
pmset -g batt
```

#### इंस्टॉल्ड सॉफ़्टवेयर की सूची
```sh
ls /Applications
```

#### macOS वर्जन
```sh
sw_vers
```

#### यूज़र इनफॉर्मेशन
```sh
id
```

#### चल रही प्रक्रियाओं की सूची
```sh
ps aux
```

#### इंस्टॉल्ड Brew पैकेजों की सूची
```sh
brew list
```

#### इंस्टॉल्ड Pip पैकेजों की सूची
```sh
pip list
```

#### इंस्टॉल्ड Gem पैकेजों की सूची
```sh
gem list
```

### 4. **एक्टिविटी मॉनिटर**
यह ऐप आपके Mac के CPU, मेमोरी, एनर्जी, डिस्क और नेटवर्क यूज़ेज के बारे में रीयल-टाइम जानकारी प्रदान करती है।

- `Applications` > `Utilities` > `Activity Monitor` से **एक्टिविटी मॉनिटर** खोलें या स्पॉटलाइट में इसकी खोज करके।

### 5. **डिस्क यूटिलिटी**
यह ऐप आपके स्टोरेज डिवाइस के बारे में जानकारी प्रदान करती है।

- `Applications` > `Utilities` > `Disk Utility` से **डिस्क यूटिलिटी** खोलें या स्पॉटलाइट में इसकी खोज करके।

### 6. **थर्ड-पार्टी टूल्स**
**iStat Menus** और **CoconutBattery** जैसे थर्ड-पार्टी एप्लिकेशन भी हैं जो विस्तृत सिस्टम जानकारी और मॉनिटरिंग क्षमताएं प्रदान करते हैं।

इन तरीकों का उपयोग करके, आप अपने macOS पर सिस्टम जानकारी की एक विस्तृत श्रृंखला एकत्र कर सकते हैं।
