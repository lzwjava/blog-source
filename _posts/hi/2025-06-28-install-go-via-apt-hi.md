`apt` का उपयोग करके Go को डेबियन-आधारित Linux सिस्टम (जैसे Ubuntu) पर इंस्टॉल करने के लिए, इन चरणों का पालन करें। ध्यान दें कि `apt` Go का एक पुराना वर्जन इंस्टॉल कर सकता है (जैसे Ubuntu 22.04 पर 1.18), इसलिए आपको जाँच करनी चाहिए कि क्या यह आपकी प्रोजेक्ट की आवश्यकताओं को पूरा करता है (`~/Projects/clash-core` में `go.mod` फ़ाइल में आवश्यक Go वर्जन जाँचें)।

### APT का उपयोग करके Go इंस्टॉल करने के चरण

1. **पैकेज इंडेक्स अपडेट करें**
   - सुनिश्चित करें कि आपकी पैकेज सूचियाँ अप-टू-डेट हैं:
     ```bash
     sudo apt update
     ```

2. **Go इंस्टॉल करें**
   - Go पैकेज इंस्टॉल करें:
     ```bash
     sudo apt install golang-go
     ```
   - यह Go कंपाइलर, टूल्स और स्टैंडर्ड लाइब्रेरी इंस्टॉल करता है।

3. **इंस्टॉलेशन सत्यापित करें**
   - इंस्टॉल किए गए Go वर्जन की जाँच करें:
     ```bash
     go version
     ```
   - आपको कुछ इस तरह आउटपुट दिखाई देना चाहिए:
     ```
     go version go1.18.1 linux/amd64
     ```
     (वर्जन आपके डिस्ट्रीब्यूशन की पैकेज रिपॉजिटरी पर निर्भर करता है।)

4. **GOPATH सेट अप करें (वैकल्पिक)**
   - आधुनिक Go वर्जन (1.13+) मॉड्यूल्स का उपयोग करते हैं, इसलिए `GOPATH` वैकल्पिक है, लेकिन इसे सेट करना कुछ प्रोजेक्ट्स के लिए उपयोगी हो सकता है।
   - अपनी शेल प्रोफाइल एडिट करें (जैसे, `~/.bashrc` या `~/.zshrc`):
     ```bash
     nano ~/.bashrc
     ```
   - निम्नलिखित जोड़ें:
     ```bash
     export GOPATH=$HOME/go
     export PATH=$PATH:$GOPATH/bin
     ```
   - सेव करें और शेल कॉन्फ़िगरेशन रीलोड करें:
     ```bash
     source ~/.bashrc
     ```

5. **बिल्ड दोबारा आज़माएँ**
   - अपने प्रोजेक्ट डायरेक्टरी में नेविगेट करें:
     ```bash
     cd ~/Projects/clash-core
     ```
   - `make` कमांड दोबारा चलाएँ:
     ```bash
     make
     ```

### नोट्स
- **वर्जन जाँच**: यदि आपके प्रोजेक्ट को नए Go वर्जन (जैसे 1.20 या बाद वाले) की आवश्यकता है, तो `apt` वर्जन पुराना हो सकता है। आवश्यक वर्जन `go.mod` या प्रोजेक्ट के डॉक्यूमेंटेशन में जाँचें। यदि आवश्यक हो, तो मैन्युअल रूप से [https://go.dev/dl/](https://go.dev/dl/) से tarball डाउनलोड करके एक नया वर्जन इंस्टॉल करें (जैसा कि पिछले जवाब में बताया गया है)।
- **वैकल्पिक रिपॉजिटरीज़**: `apt` के माध्यम से नए वर्जन के लिए, आप एक थर्ड-पार्टी PPA जोड़ सकते हैं (जैसे Ubuntu पर `ppa:longsleep/golang-backports`)। चलाएँ:
  ```bash
  sudo add-apt-repository ppa:longsleep/golang-backports
  sudo apt update
  sudo apt install golang-go
  ```
- **समस्या निवारण**:
  - यदि `go` नहीं मिलता है, तो सुनिश्चित करें कि `/usr/bin/go` मौजूद है (`ls /usr/bin/go`)। यदि नहीं है, तो `sudo apt install --reinstall golang-go` के साथ दोबारा इंस्टॉल करें।
  - यदि आपको डिपेंडेंसी एरर मिलते हैं, तो उन्हें ठीक करने के लिए `sudo apt install -f` चलाएँ।
  - परमिशन इश्यू के लिए, सुनिश्चित करें कि आपके पास `sudo` प्रिविलेज हैं।

### स्रोत
- Ubuntu Go इंस्टॉलेशन गाइड: [https://ubuntu.com/server/docs/programming-go](https://ubuntu.com/server/docs/programming-go)
- सामान्य apt उपयोग: [https://help.ubuntu.com/community/AptGet/Howto](https://help.ubuntu.com/community/AptGet/Howto)

यदि इंस्टॉल किया गया वर्जन बहुत पुराना है या आपको कोई समस्या आती है, तो मुझे अपना Linux डिस्ट्रीब्यूशन (जैसे Ubuntu 22.04) और आवश्यक Go वर्जन बताएँ, और मैं आपको आगे मार्गदर्शन दूंगा!
