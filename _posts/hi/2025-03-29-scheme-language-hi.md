Scheme Lisp का एक न्यूनतमवादी बोली है जो अपनी सरलता और सुंदरता के लिए जानी जाती है। यह ट्यूटोरियल आपको शुरुआत करने के लिए बुनियादी बातों को कवर करेगा।

## 1. बेसिक सिंटैक्स और कॉन्सेप्ट्स

### Scheme चलाना
- आपको एक Scheme इंटरप्रेटर (जैसे Racket, Guile, या Chez Scheme) की आवश्यकता होगी
- कोड आमतौर पर एक `.scm` फाइल में लिखा जाता है या एक REPL (रीड-ईवल-प्रिंट लूप) में दर्ज किया जाता है

### एक्सप्रेशन और कोष्ठक
- Scheme में सब कुछ एक एक्सप्रेशन है जो एक वैल्यू रिटर्न करता है
- फंक्शन कॉल और ग्रुपिंग के लिए कोष्ठकों का उपयोग किया जाता है
- फॉर्मेट: `(function arg1 arg2 ...)`

## 2. बेसिक डेटा टाइप्स

```scheme
; नंबर्स

; स्ट्रिंग्स

; सिंबल्स
'hello      ; सिंबल ( ' के साथ कोट किया गया)
```

## 5. फंक्शन्स

### फंक्शन्स डिफाइन करना
```scheme
; बेसिक फंक्शन डेफिनिशन
(define square
  (lambda (x)    ; lambda एक अनाम फंक्शन बनाता है
    (* x x)))

(square 4)     ; 16
```

### मल्टीपल पैरामीटर्स
```scheme
(define add
  (lambda (x y)
    (+ x y)))

(add 3 5)      ; 8
```

### शॉर्टहैंड डेफिनिशन
```scheme
; वैकल्पिक सिंटैक्स (सिंटैक्टिक शुगर)
(define (multiply x y)
  (* x y))

(multiply 2 3) ; 6
```

## 6. कंडीशनल्स

### If स्टेटमेंट
```scheme
(define (is-positive? n)
  (if (> n 0)
      #t
      #f))

(is-positive? 5)   ; #t
(is-positive? -2)  ; #f
```

### Cond (मल्टीपल कंडीशन्स)
```scheme
(define (number-type n)
  (cond
    ((> n 0) "positive")
    ((< n 0) "negative")
    (else "zero")))

(number-type 5)    ; "positive"
(number-type 0)    ; "zero"
```

## 7. लिस्ट्स

### लिस्ट्स बनाना
```scheme
; quote का उपयोग करना
'(1 2 3)          ; नंबर्स की लिस्ट

; list फंक्शन का उपयोग करना
(list 1 2 3)      ; ऊपर के समान

; cons (कंस्ट्रक्ट) का उपयोग करना
(cons 1 '(2 3))   ; ऊपर के समान
```

### लिस्ट ऑपरेशन्स
```scheme
(car '(1 2 3))    ; 1 (पहला एलिमेंट)
(cdr '(1 2 3))    ; (2 3) (लिस्ट का बाकी हिस्सा)
(null? '())       ; #t (चेक करें कि खाली है या नहीं)
(length '(1 2 3)) ; 3
```

## 8. रिकर्सन

### सिंपल रिकर्सन
```scheme
(define (factorial n)
  (if (= n 0)
      1
      (* n (factorial (- n 1)))))

(factorial 5)     ; 120 (5 * 4 * 3 * 2 * 1)
```

### लिस्ट रिकर्सन
```scheme
(define (sum-list lst)
  (if (null? lst)
      0
      (+ (car lst) (sum-list (cdr lst)))))

(sum-list '(1 2 3 4))  ; 10
```

## 9. हायर-ऑर्डर फंक्शन्स

### मैपिंग
```scheme
(map square '(1 2 3 4))    ; (1 4 9 16)
(map + '(1 2 3) '(4 5 6))  ; (5 7 9)
```

### फिल्टरिंग
```scheme
(define (even? n) (= (remainder n 2) 0))
(filter even? '(1 2 3 4 5 6))  ; (2 4 6)
```

## 10. लेट बाइंडिंग्स

```scheme
; लोकल वेरिएबल बाइंडिंग्स
(define (circle-area r)
  (let ((pi 3.14159))
    (* pi (square r))))

```

## 12. उदाहरण प्रोग्राम

यहां एक संपूर्ण प्रोग्राम है जो कई कॉन्सेप्ट्स को जोड़ता है:

```scheme
; नंबर्स की एक लिस्ट का औसत कैलकुलेट करने के लिए प्रोग्राम
(define (average lst)
  (define (sum lst)
    (if (null? lst)
        0
        (+ (car lst) (sum (cdr lst)))))
  (define len (length lst))
  (if (= len 0)
      0
      (/ (sum lst) len)))

; प्रोग्राम टेस्ट करें
(display (average '(1 2 3 4 5)))  ; 3
(newline)
(display (average '()))           ; 0
```

## Scheme की मुख्य विशेषताएं

1. **न्यूनतमवादी**: शक्तिशाली फीचर्स के साथ छोटा कोर भाषा
2. **फंक्शनल**: शुद्ध फंक्शन्स और अपरिवर्तनीयता पर जोर
3. **प्रीफिक्स नोटेशन**: सभी ऑपरेशन प्रीफिक्स सिंटैक्स का उपयोग करते हैं
4. **फर्स्ट-क्लास फंक्शन्स**: फंक्शन्स को आर्गुमेंट्स के रूप में पास किया जा सकता है
5. **टेल-कॉल ऑप्टिमाइजेशन**: कुशल रिकर्सन

## सीखने के टिप्स

1. तुरंत परिणाम देखने के लिए REPL में अभ्यास करें
2. सरल फंक्शन्स से शुरुआत करें और जटिलता बनाएं
3. लिस्ट प्रोसेसिंग के लिए रिकर्सिवली सोचें
4. कोष्ठकों को मैनेज करने के लिए उचित इंडेंटेशन का उपयोग करें
5. हायर-ऑर्डर फंक्शन्स के साथ प्रयोग करें

यह ट्यूटोरियल Scheme की कोर कॉन्सेप्ट्स को कवर करता है। अपनी समझ को गहरा करने के लिए, इसे आजमाएं:
- अधिक जटिल रिकर्सिव फंक्शन्स लिखना
- मैक्रो सिस्टम की खोज (एडवांस्ड टॉपिक)
- कैलकुलेटर या लिस्ट प्रोसेसर जैसे छोटे प्रोजेक्ट बनाना
- Scheme की स्टैंडर्ड लाइब्रेरीज का अध्ययन (इम्प्लीमेंटेशन के अनुसार अलग-अलग)
