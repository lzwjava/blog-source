GitHub Copilot एक्सटेंशन (VS Code के लिए) में किसी मॉडल को हटाने के लिए, ध्यान दें कि बिल्ट-इन मॉडल (जैसे डिफ़ॉल्ट GitHub Copilot मॉडल) हटाए नहीं जा सकते, क्योंकि वे एक्सटेंशन का हिस्सा हैं। हालाँकि, यदि आपकी बात किसी कस्टम मॉडल की है जिसे आपने किसी थर्ड-पार्टी प्रोवाइडर (जैसे Anthropic, OpenAI, Google Gemini, या अन्य) के माध्यम से अपनी खुद की API key का उपयोग करके जोड़ा है, तो आप संबंधित API key और प्रोवाइडर का विवरण डिलीट करके इसे प्रभावी रूप से हटा सकते हैं। यह फीचर प्रीव्यू में है और हो सकता है अभी तक Copilot Business या Enterprise उपयोगकर्ताओं के लिए उपलब्ध न हो।

यहाँ आधिकारिक डॉक्यूमेंटेशन के आधार पर चरण दिए गए हैं:

1. VS Code में Chat व्यू खोलें (यदि पहले से खुला नहीं है, तो Command Palette का उपयोग करें `Ctrl+Shift+P` या macOS पर `Cmd+Shift+P`, फिर "Copilot: Open Chat" खोजें)।
2. Chat व्यू में, मॉडल पिकर ड्रॉपडाउन (इनपुट फ़ील्ड के बगल में) पर क्लिक करें और **Manage Models** चुनें। वैकल्पिक रूप से, Command Palette खोलें और **GitHub Copilot: Manage Models** कमांड रन करें।
3. Manage Models इंटरफेस में, उस प्रोवाइडर पर होवर करें जिसके मॉडल को आप हटाना चाहते हैं और उसके विवरण को एडिट करने के लिए गियर आइकन पर क्लिक करें।
4. फील्ड्स से API key (और यदि लागू हो तो किसी भी endpoint URL) को डिलीट कर दें, फिर परिवर्तनों को सेव करें। इससे मॉडल की उपलब्धता पिकर से हट जानी चाहिए।

यदि आपको कोई समस्या आती है (जैसे, API key फ़ील्ड अपडेट नहीं होती या सही से क्लियर नहीं होती, जिसे कुछ मामलों में बग बताया गया है), तो पहले इस वर्कअराउंड को आज़माएँ:
- Chat व्यू में मॉडल पिकर से एक अलग मॉडल चुनें ताकि जिसे आप हटाना चाहते हैं उसे डिसिलेक्ट कर सकें।
- Manage Models पर वापस जाएँ, प्रोवाइडर को एडिट करें, और फिर API key को डिलीट करें।

पूर्वापेक्षाएँ और नोट्स:
- यह केवल चैट फीचर्स पर लागू होता है; इसका कोड कम्पलीशन या अन्य टूल्स पर कोई प्रभाव नहीं पड़ता।
- सुनिश्चित करें कि आपकी Copilot सब्सक्रिप्शन एक से अधिक मॉडल्स तक एक्सेस की अनुमति देती है (जैसे, Individual, Pro, या संगठनों के लिए प्रीव्यू फीचर्स सक्षम)।
- यदि मॉडल बना रहता है या आपको एरर्स का सामना करना पड़ता है, तो VS Code को रीस्टार्ट करें या एक्सटेंशन अपडेट्स के लिए चेक करें।

यदि यह आपके "मॉडल हटाने" के अर्थ से मेल नहीं खाता (उदाहरण के लिए, यदि आप पूरी एक्सटेंशन को डिसेबल करने या Copilot फीचर्स को हाइड करने का प्रयास कर रहे हैं), तो अधिक टार्गेटेड चरणों के लिए स्पष्टीकरण दें।
