किरचॉफ के नियमों—विशेष रूप से किरचॉफ का करंट लॉ (KCL) और किरचॉफ का वोल्टेज लॉ (KVL)—को सत्यापित करने के लिए Arduino का उपयोग करना इलेक्ट्रॉनिक्स सिद्धांत को प्रायोगिक अनुभव के साथ जोड़ने का एक शानदार तरीका है। KCL कहता है कि किसी जंक्शन में प्रवेश करने वाला कुल करंट, उससे निकलने वाले कुल करंट के बराबर होता है, जबकि KVL कहता है कि किसी बंद लूप के चारों ओर वोल्टेज ड्रॉप का योग सप्लाई किए गए वोल्टेज के बराबर होता है। यहां बताया गया है कि आप सरल सर्किट कैसे डिजाइन कर सकते हैं और इन नियमों की पुष्टि करने के लिए करंट और वोल्टेज को मापने के लिए Arduino का उपयोग कैसे कर सकते हैं।

चूंकि Arduino सीधे करंट को माप नहीं सकता है, इसलिए हम रेजिस्टर्स के पार वोल्टेज मापकर (ओम के नियम \\( I = V/R \\) का उपयोग करके) इसका अनुमान लगाएंगे, और यह अपने एनालॉग पिन (0-5V रेंज) के माध्यम से वोल्टेज माप सकता है। नीचे, मैं दो प्रयोगों—एक KCL के लिए और एक KVL के लिए—को चरण-दर-चरण निर्देशों, वायरिंग और कोड के साथ रेखांकित करूंगा।

### **प्रयोग 1: किरचॉफ के करंट लॉ (KCL) का सत्यापन**

#### **उद्देश्य**
यह प्रदर्शित करना कि किसी नोड में प्रवेश करने वाला करंट, उससे निकलने वाले करंट के बराबर होता है।

#### **सर्किट सेटअप**
- **घटक:**
  - Arduino (जैसे, Uno)
  - 3 रेजिस्टर (जैसे, R1 = 330Ω, R2 = 470Ω, R3 = 680Ω)
  - ब्रेडबोर्ड और जम्पर वायर
  - 5V पावर स्रोत (Arduino का 5V पिन)
- **वायरिंग:**
  - Arduino के 5V को एक नोड (इसे नोड A कहें) से कनेक्ट करें।
  - नोड A से, R1 को GND से कनेक्ट करें (ब्रांच 1)।
  - नोड A से, R2 को GND से कनेक्ट करें (ब्रांच 2, R1 के समानांतर)।
  - नोड A से, R3 को GND से कनेक्ट करें (ब्रांच 3, R1 और R2 के समानांतर)।
  - प्रत्येक रेजिस्टर के पार वोल्टेज मापने के लिए Arduino के एनालॉग पिन का उपयोग करें:
    - A0, R1 के पार (एक प्रोब नोड A पर, दूसरा GND पर)।
    - A1, R2 के पार।
    - A2, R3 के पार।
- **नोट:** GND कॉमन रेफरेंस पॉइंट है।

#### **सिद्धांत**
- 5V से नोड A तक कुल करंट (\\( I_{in} \\)), R1, R2 और R3 से होकर गुजरने वाले \\( I_1 \\), \\( I_2 \\), और \\( I_3 \\) में विभाजित हो जाता है।
- KCL: \\( I_{in} = I_1 + I_2 + I_3 \\)।
- प्रत्येक रेजिस्टर के पार वोल्टेज मापें, फिर करंट की गणना करें: \\( I = V/R \\)।

#### **Arduino कोड**
```cpp
void setup() {
  Serial.begin(9600); // सीरियल कम्युनिकेशन शुरू करें
}

void loop() {
  // वोल्टेज पढ़ें (0-1023, 0-5V पर मैप होता है)
  int sensorValue1 = analogRead(A0); // R1 के पार वोल्टेज
  int sensorValue2 = analogRead(A1); // R2 के पार वोल्टेज
  int sensorValue3 = analogRead(A2); // R3 के पार वोल्टेज

  // वोल्टेज में बदलें (5V रेफरेंस, 10-bit ADC)
  float V1 = sensorValue1 * (5.0 / 1023.0);
  float V2 = sensorValue2 * (5.0 / 1023.0);
  float V3 = sensorValue3 * (5.0 / 1023.0);

  // रेजिस्टर वैल्यू (ओम में)
  float R1 = 330.0;
  float R2 = 470.0;
  float R3 = 680.0;

  // करंट की गणना करें (I = V/R)
  float I1 = V1 / R1;
  float I2 = V2 / R2;
  float I3 = V3 / R3;

  // नोड में प्रवेश करने वाला कुल करंट (मानते हुए कि Vsource = 5V)
  float totalResistance = 1.0 / ((1.0/R1) + (1.0/R2) + (1.0/R3)); // समानांतर
  float Iin = 5.0 / totalResistance;

  // परिणाम आउटपुट करें
  Serial.print("I1 (mA): "); Serial.println(I1 * 1000);
  Serial.print("I2 (mA): "); Serial.println(I2 * 1000);
  Serial.print("I3 (mA): "); Serial.println(I3 * 1000);
  Serial.print("Iin (mA): "); Serial.println(Iin * 1000);
  Serial.print("I1+I2+I3 का योग (mA): "); Serial.println((I1 + I2 + I3) * 1000);
  Serial.println("---");

  delay(2000); // 2 सेकंड प्रतीक्षा करें
}
```

#### **सत्यापन**
- सीरियल मॉनिटर खोलें (Arduino IDE में Ctrl+Shift+M, 9600 बॉड पर सेट करें)।
- \\( I_{in} \\) (कुल प्रतिरोध से गणना की गई) की तुलना \\( I_1 + I_2 + I_3 \\) से करें। ये लगभग बराबर होने चाहिए, जो KCL की पुष्टि करेगा।
- रेजिस्टर टॉलरेंस या Arduino ADC की सटीकता के कारण छोटे अंतर हो सकते हैं।

### **प्रयोग 2: किरचॉफ के वोल्टेज लॉ (KVL) का सत्यापन**

#### **उद्देश्य**
यह दिखाना कि किसी बंद लूप के चारों ओर वोल्टेज ड्रॉप का योग सप्लाई वोल्टेज के बराबर होता है।

#### **सर्किट सेटअप**
- **घटक:**
  - Arduino
  - 2 रेजिस्टर (जैसे, R1 = 330Ω, R2 = 470Ω)
  - ब्रेडबोर्ड और जम्पर वायर
  - 5V पावर स्रोत (Arduino का 5V पिन)
- **वायरिंग:**
  - 5V को R1 से कनेक्ट करें।
  - R1 को R2 से कनेक्ट करें।
  - R2 को GND से कनेक्ट करें।
  - वोल्टेज मापें:
    - A0 पूरे लूप (5V से GND) के पार, सप्लाई वोल्टेज की पुष्टि के लिए।
    - A1, R1 के पार (5V से R1 और R2 के जंक्शन तक)।
    - A2, R2 के पार (जंक्शन से GND तक)।
- **नोट:** वोल्टेज डिवाइडर सेटअप का उपयोग करें; सुनिश्चित करें कि वोल्टेज 5V (Arduino की सीमा) से अधिक न हों।

#### **सिद्धांत**
- KVL: \\( V_{source} = V_{R1} + V_{R2} \\)।
- प्रत्येक वोल्टेज ड्रॉप को मापें और जांचें कि क्या उनका योग स्रोत वोल्टेज (5V) के बराबर है।

#### **Arduino कोड**
```cpp
void setup() {
  Serial.begin(9600);
}

void loop() {
  // वोल्टेज पढ़ें
  int sensorValueSource = analogRead(A0); // 5V से GND के पार
  int sensorValueR1 = analogRead(A1);     // R1 के पार
  int sensorValueR2 = analogRead(A2);     // R2 के पार

  // वोल्टेज में बदलें
  float Vsource = sensorValueSource * (5.0 / 1023.0);
  float VR1 = sensorValueR1 * (5.0 / 1023.0);
  float VR2 = sensorValueR2 * (5.0 / 1023.0);

  // परिणाम आउटपुट करें
  Serial.print("Vsource (V): "); Serial.println(Vsource);
  Serial.print("VR1 (V): "); Serial.println(VR1);
  Serial.print("VR2 (V): "); Serial.println(VR2);
  Serial.print("VR1 + VR2 (V): "); Serial.println(VR1 + VR2);
  Serial.println("---");

  delay(2000);
}
```

#### **सत्यापन**
- सीरियल मॉनिटर में, जांचें कि \\( V_{source} \approx V_{R1} + V_{R2} \\)।
- R1 = 330Ω और R2 = 470Ω को श्रृंखला में जोड़ने पर, \\( V_{R1} \approx 2.1V \\) और \\( V_{R2} \approx 2.9V \\) (वोल्टेज डिवाइडर के आधार पर: \\( V_{R1} = 5 \cdot \frac{R1}{R1+R2} \\)) की अपेक्षा करें, जो ~5V तक जुड़ती है।

### **सुझाव और विचारणीय बातें**
- **सटीकता:** Arduino का 10-bit ADC (0–1023) का रिज़ॉल्यूशन ~4.9mV है, इसलिए मामूली त्रुटियों की अपेक्षा करें। यदि आवश्यक हो तो क्रॉस-चेक के लिए मल्टीमीटर का उपयोग करें।
- **सुरक्षा:** Arduino को क्षति से बचाने के लिए वोल्टेज को 5V से नीचे और करंट को कम रखें।
- **स्केलिंग:** अधिक जटिल सर्किट के लिए, अधिक रेजिस्टर जोड़ें या बाहरी सेंसर (जैसे, सीधे करंट मापन के लिए INA219) का उपयोग करें।
- **डिबगिंग:** यदि परिणाम मेल नहीं खाते हैं, तो रेजिस्टर वैल्यू, कनेक्शन और GND की स्थिरता सत्यापित करें।
