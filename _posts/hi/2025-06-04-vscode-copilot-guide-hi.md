GitHub Copilot, GitHub और OpenAI द्वारा विकसित एक AI-पावर्ड कोडिंग असिस्टेंट है, जिसे Visual Studio Code (VS Code) में डेवलपर उत्पादकता बढ़ाने के लिए डिज़ाइन किया गया है। यह संदर्भ-जागरूक कोड पूर्णताएं, प्राकृतिक भाषा चैट, और उन्नत संपादन क्षमताएं प्रदान करता है। यह गाइड आपको Copilot को सेट अप करने, इसकी विशेषताओं का पता लगाने, और टिप्स और बेस्ट प्रैक्टिसेज के साथ अपने वर्कफ़्लो को ऑप्टिमाइज़ करने के माध्यम से चलती है।

## 1. GitHub Copilot का परिचय
GitHub Copilot एक AI पेयर प्रोग्रामर के रूप में कार्य करता है, जो रीयल-टाइम कोड सुझाव, कोडिंग प्रश्नों के जवाब, और दोहराए जाने वाले कार्यों को स्वचालित करने की पेशकश करता है। यह बड़े भाषा मॉडल द्वारा संचालित है जिन्हें सार्वजनिक कोड के विशाल डेटासेट पर प्रशिक्षित किया गया है, जिससे यह Python, JavaScript, TypeScript, Ruby, Go, C#, और C++ सहित कई प्रोग्रामिंग भाषाओं और फ्रेमवर्क का समर्थन करने में सक्षम है।

मुख्य विशेषताओं में शामिल हैं:
- **कोड पूर्णताएं**: जैसे ही आप टाइप करते हैं, सिंगल लाइनों से लेकर पूरे फ़ंक्शन तक कोड स्निपेट सुझाता है।
- **Copilot चैट**: कोड समझाने, स्निपेट जनरेट करने, या डीबग करने के लिए प्राकृतिक भाषा प्रश्नों की अनुमति देता है।
- **एजेंट मोड**: रीफैक्टरिंग या ऐप बनाने जैसे मल्टी-स्टेप कोडिंग टास्क को स्वचालित करता है।
- **कस्टम निर्देश**: आपकी कोडिंग शैली या प्रोजेक्ट आवश्यकताओं से मेल खाने के लिए सुझावों को अनुकूलित करता है।

## 2. VS Code में GitHub Copilot सेट अप करना

### आवश्यक शर्तें
- **VS Code**: [आधिकारिक वेबसाइट](https://code.visualstudio.com/) से Visual Studio Code डाउनलोड और इंस्टॉल करें। सुनिश्चित करें कि आपके पास एक संगत संस्करण है (कोई भी हालिया संस्करण Copilot का समर्थन करता है)।
- **GitHub अकाउंट**: आपको Copilot तक पहुंच वाले GitHub अकाउंट की आवश्यकता है। विकल्पों में शामिल हैं:
  - **Copilot फ्री**: प्रति माह सीमित पूर्णताएं और चैट इंटरैक्शन।
  - **Copilot Pro/Pro+**: उच्च सीमा और उन्नत विशेषताओं वाले पेड प्लान।
  - **ऑर्गनाइज़ेशन एक्सेस**: यदि आपकी संस्था द्वारा प्रदान किया गया है, तो एक्सेस विवरण के लिए अपने एडमिन से जांच करें।
- **इंटरनेट कनेक्शन**: सुझाव प्रदान करने के लिए Copilot को सक्रिय कनेक्शन की आवश्यकता होती है।

### इंस्टालेशन स्टेप्स
1. **VS Code खोलें**:
   अपनी मशीन पर Visual Studio Code लॉन्च करें।

2. **GitHub Copilot एक्सटेंशन इंस्टॉल करें**:
   - **एक्सटेंशन** व्यू (Ctrl+Shift+X या macOS पर Cmd+Shift+X) पर जाएं।
   - एक्सटेंशन मार्केटप्लेस में "GitHub Copilot" सर्च करें।
   - आधिकारिक GitHub Copilot एक्सटेंशन के लिए **इंस्टॉल** पर क्लिक करें। यह Copilot Chat एक्सटेंशन को भी स्वचालित रूप से इंस्टॉल कर देता है।

3. **GitHub में साइन इन करें**:
   - इंस्टालेशन के बाद, VS Code स्टेटस बार (नीचे-दायां कोना) में Copilot सेट अप करने का प्रॉम्प्ट दिखाई दे सकता है।
   - Copilot आइकन पर क्लिक करें और अपने GitHub अकाउंट से प्रमाणीकरण करने के लिए **साइन इन** चुनें।
   - यदि कोई प्रॉम्प्ट नहीं दिखाई देता है, तो कमांड पैलेट (Ctrl+Shift+P या Cmd+Shift+P) खोलें और `GitHub Copilot: Sign in` कमांड रन करें।
   - ब्राउज़र-आधारित प्रमाणीकरण प्रवाह का पालन करें, VS Code द्वारा प्रदान किए गए कोड को GitHub पर कॉपी करें।

4. **एक्टिवेशन सत्यापित करें**:
   - एक बार साइन इन करने के बाद, स्टेटस बार में Copilot आइकन हरा हो जाना चाहिए, जो सक्रिय स्थिति का संकेत देता है।
   - यदि आपके पास Copilot सब्सक्रिप्शन नहीं है, तो आपको सीमित मासिक उपयोग के साथ Copilot फ्री प्लान में नामांकित कर दिया जाएगा।

5. **वैकल्पिक: टेलीमेट्री अक्षम करें**:
   - डिफ़ॉल्ट रूप से, Copilot टेलीमेट्री डेटा एकत्र करता है। इसे अक्षम करने के लिए, **सेटिंग्स** (Ctrl+, या Cmd+,) पर जाएं, `telemetry.telemetryLevel` सर्च करें, और इसे `off` पर सेट करें। वैकल्पिक रूप से, `GitHub Copilot Settings` के तहत Copilot-विशिष्ट सेटिंग्स समायोजित करें।

> **नोट**: यदि आपकी संस्था ने Copilot Chat को अक्षम कर दिया है या सुविधाओं को प्रतिबंधित कर दिया है, तो अपने एडमिन से संपर्क करें। समस्या निवारण के लिए, [GitHub Copilot समस्या निवारण गाइड](https://docs.github.com/en/copilot/troubleshooting) देखें।[](https://code.visualstudio.com/docs/copilot/setup)

## 3. VS Code में GitHub Copilot की मुख्य विशेषताएं

### 3.1 कोड पूर्णताएं
Copilot आपके टाइप करते समय, आपके कोड और फ़ाइल संरचना के संदर्भ के आधार पर, सिंगल लाइनों से लेकर पूरे फ़ंक्शन या क्लासेज तक कोड सुझाता है।
- **यह कैसे काम करता है**:
  - एक समर्थित भाषा (जैसे JavaScript, Python, C#) में टाइप करना शुरू करें।
  - Copilot "घोस्ट टेक्स्ट" में सुझाव प्रदर्शित करता है।
  - सुझाव स्वीकार करने के लिए **Tab** दबाएं या इग्नोर करने के लिए टाइप करना जारी रखें।
  - कई सुझावों के बीच साइकिल करने के लिए **Alt+]** (अगला) या **Alt+[** (पिछला) का उपयोग करें।
- **उदाहरण**:
  ```javascript
  // किसी संख्या का फैक्टोरियल कैलकुलेट करें
  function factorial(n) {
  ```
  Copilot सुझा सकता है:
  ```javascript
  if (n === 0) return 1;
  return n * factorial(n - 1);
  }
  ```
  सुझाव स्वीकार करने के लिए **Tab** दबाएं।

- **टिप्स**:
  - Copilot को मार्गदर्शन देने के लिए वर्णनात्मक फ़ंक्शन नाम या कमेंट्स का उपयोग करें (जैसे, `// एक ऐरे को आरोही क्रम में सॉर्ट करें`)।
  - कई सुझावों के लिए, सभी विकल्प देखने के लिए कम्पलीशन पैनल (Ctrl+Enter) खोलने के लिए सुझाव पर होवर करें।

### 3.2 Copilot चैट
Copilot Chat आपको प्रश्न पूछने, कोड जनरेट करने, या डीबग करने के लिए प्राकृतिक भाषा का उपयोग करके Copilot के साथ इंटरैक्ट करने की अनुमति देता है।
- **चैट एक्सेस करना**:
  - एक्टिविटी बार से **चैट व्यू** खोलें या **Ctrl+Alt+I** (Windows/Linux) या **Cmd+Ctrl+I** (macOS) का उपयोग करें।
  - वैकल्पिक रूप से, संदर्भ-विशिष्ट प्रश्नों के लिए सीधे एडिटर में **इनलाइन चैट** (Ctrl+I या Cmd+I) का उपयोग करें।
- **उपयोग के मामले**:
  - **कोड समझाएं**: एक कोड ब्लॉक सेलेक्ट करें, इनलाइन चैट खोलें, और `explain this code` टाइप करें।
  - **कोड जनरेट करें**: चैट व्यू में `write a Python function to reverse a string` टाइप करें।
  - **डीबगिंग**: चैट में एक एरर मैसेज पेस्ट करें और फिक्स के लिए पूछें।
- **उदाहरण**:
  चैट व्यू में, टाइप करें:
  ```
  What is recursion?
  ```
  Copilot Markdown में कोड उदाहरणों सहित एक विस्तृत स्पष्टीकरण के साथ जवाब देता है।

- **स्लैश कमांड्स**:
  टास्क्स निर्दिष्ट करने के लिए `/explain`, `/doc`, `/fix`, `/tests`, या `/optimize` जैसी कमांड्स का उपयोग करें। उदाहरण के लिए:
  ```
  /explain this function
  ```
  एक सेलेक्टेड फ़ंक्शन के साथ एक विस्तृत स्पष्टीकरण जनरेट करेगा।

### 3.3 एजेंट मोड (पूर्वावलोकन)
एजेंट मोड Copilot को ऐप बनाने, कोड रीफैक्टर करने, या टेस्ट लिखने जैसे मल्टी-स्टेप कोडिंग टास्क को स्वायत्त रूप से संभालने में सक्षम बनाता है।
- **उपयोग कैसे करें**:
  - VS Code इनसाइडर्स या स्टेबल (यदि उपलब्ध हो) में **Copilot Edits व्यू** खोलें।
  - मोड ड्रॉपडाउन से **एजेंट** चुनें।
  - एक प्रॉम्प्ट दर्ज करें, जैसे, `Create a React form component with name and email fields`।
  - Copilot आपके कोडबेस का विश्लेषण करता है, संपादन सुझाता है, और टर्मिनल कमांड्स या टेस्ट्स रन कर सकता है।
- **क्षमताएं**:
  - कई फाइलों में कोड जनरेट करता है।
  - एरर्स की निगरानी करता है और मुद्दों को ठीक करने के लिए पुनरावृत्ति करता है।
  - नई लाइब्रेरीज को एकीकृत करता है या कोड को आधुनिक फ्रेमवर्क में माइग्रेट करता है।

> **नोट**: एजेंट मोड प्रायोगिक है और छोटे रिपॉजिटरी में सबसे अच्छा प्रदर्शन करता है। VS Code GitHub रेपो के माध्यम से फीडबैक साझा करें।[](https://code.visualstudio.com/blogs/2025/02/24/introducing-copilot-agent-mode)

### 3.4 कस्टम निर्देश
अपनी कोडिंग शैली या प्रोजेक्ट आवश्यकताओं के अनुरूप Copilot को अनुकूलित करें।
- **सेटअप**:
  - अपने वर्कस्पेस में एक `.github/copilot-instructions.md` फ़ाइल बनाएं।
  - Markdown में निर्देश जोड़ें, जैसे, `Use snake_case for Python variable names`।
  - **सेटिंग्स** > **GitHub** > **Copilot** > **Enable custom instructions** (VS Code 17.12 या बाद में) में कस्टम निर्देश सक्षम करें।
- **उदाहरण**:
  ```markdown
  # Copilot कस्टम निर्देश
  - JavaScript वेरिएबल्स के लिए camelCase का उपयोग करें।
  - प्रॉमिसेज के लिए .then() पर async/await को प्राथमिकता दें।
  ```
  Copilot इन प्राथमिकताओं के अनुरूप सुझाव देगा।

### 3.5 @workspace के साथ वर्कस्पेस संदर्भ
अपने संपूर्ण कोडबेस को क्वेरी करने के लिए `@workspace` कमांड का उपयोग करें।
- **उदाहरण**:
  चैट व्यू में, टाइप करें:
  ```
  @workspace Where is the database connection string configured?
  ```
  Copilot आपके वर्कस्पेस को सर्च करेगा और प्रासंगिक फाइलों को संदर्भित करेगा।

### 3.6 नेक्स्ट एडिट सुझाव (पूर्वावलोकन)
Copilot आपके हाल के परिवर्तनों के आधार पर अगले तार्किक संपादन की भविष्यवाणी करता है और सुझाव देता है।
- **यह कैसे काम करता है**:
  - जैसे ही आप कोड एडिट करते हैं, Copilot संभावित अगले संपादनों को हाइलाइट करता है।
  - सुझावों को **Tab** से स्वीकार करें या इनलाइन चैट के माध्यम से उन्हें संशोधित करें।
- **उपयोग का मामला**: पुनरावृत्ति रीफैक्टरिंग या संबंधित कोड परिवर्तनों को पूरा करने के लिए आदर्श।

## 4. Copilot उपयोग को ऑप्टिमाइज़ करने के लिए टिप्स और ट्रिक्स

### 4.1 प्रभावी प्रॉम्प्ट्स लिखें
- विशिष्ट बनें: `write a function` के बजाय, `write a Python function to sort a list of dictionaries by the 'age' key` आज़माएं।
- संदर्भ प्रदान करें: फ्रेमवर्क या लाइब्रेरी विवरण शामिल करें (जैसे, `use React hooks`)।
- कमेंट्स का उपयोग करें: पूर्णताओं को मार्गदर्शन देने के लिए `// Generate a REST API endpoint in Express` लिखें।

### 4.2 संदर्भ का लाभ उठाएं
- **फाइलें/सिंबल्स संदर्भित करें**: संदर्भ को स्कोप करने के लिए चैट प्रॉम्प्ट्स में `#filename`, `#folder`, या `#symbol` का उपयोग करें।
  ```
  /explain #src/utils.js
  ```
- **ड्रैग और ड्रॉप**: संदर्भ के लिए फाइलों या एडिटर टैब्स को चैट प्रॉम्प्ट में ड्रैग और ड्रॉप करें।
- **इमेज अटैच करें**: VS Code 17.14 पूर्वावलोकन 1 या बाद में, मुद्दों (जैसे, UI बग) को सचित्र करने के लिए स्क्रीनशॉट अटैच करें।

### 4.3 स्लैश कमांड्स का उपयोग करें
- `/doc`: किसी फ़ंक्शन के लिए डॉक्युमेंटेशन जनरेट करें।
- `/fix`: एरर्स के लिए सुझाव दें।
- `/tests`: सेलेक्टेड कोड के लिए यूनिट टेस्ट बनाएं।
- उदाहरण:
  ```
  /tests Generate Jest tests for this function
  ```

### 4.4 प्रॉम्प्ट्स सेव करें और पुन: उपयोग करें
- पुन: प्रयोज्य प्रॉम्प्ट्स को स्टोर करने के लिए `.github/prompts/` में एक `.prompt.md` फ़ाइल बनाएं।
- उदाहरण:
  ```markdown
  # React कंपोनेंट प्रॉम्प्ट
  Tailwind CSS स्टाइलिंग के साथ एक React फंक्शनल कंपोनेंट जनरेट करें। यदि प्रदान नहीं किया गया है तो कंपोनेंट नाम और प्रॉप्स के लिए पूछें।
  ```
- प्रोजेक्ट्स में इसे पुन: उपयोग करने के लिए चैट में प्रॉम्प्ट अटैच करें।

### 4.5 सही मॉडल चुनें
- Copilot कई भाषा मॉडल (जैसे, GPT-4o, Claude Sonnet) का समर्थन करता है।
- तेज़ कोडिंग या गहन तर्क के लिए चैट व्यू ड्रॉपडाउन में मॉडल चुनें।
- जटिल कार्यों के लिए, एजेंट मोड में Claude Sonnet बेहतर प्रदर्शन कर सकता है।[](https://code.visualstudio.com/blogs/2025/02/24/introducing-copilot-agent-mode)

### 4.6 अपने वर्कस्पेस को इंडेक्स करें
- तेज़, अधिक सटीक कोड खोज के लिए वर्कस्पेस इंडेक्सिंग सक्षम करें।
- GitHub रिपॉजिटरी के लिए रिमोट इंडेक्स या बड़े कोडबेस के लिए लोकल इंडेक्स का उपयोग करें।

## 5. बेस्ट प्रैक्टिसेज
- **सुझावों की समीक्षा करें**: सटीकता और अपने प्रोजेक्ट के मानकों के साथ संरेखण के लिए Copilot के सुझावों को हमेशा सत्यापित करें।
- **IntelliCode के साथ संयोजन करें**: Visual Studio में, Copilot बेहतर पूर्णताओं के लिए IntelliCode के पूरक के रूप में कार्य करता है।[](https://devblogs.microsoft.com/visualstudio/github-copilot-in-visual-studio-2022/)
- **सुरक्षा जांचें**: Copilot कमजोरियों वाले कोड का सुझाव दे सकता है। संवेदनशील प्रोजेक्ट्स में विशेष रूप से सुझावों की समीक्षा करें, और अपनी संस्था की नीतियों के साथ जांच करें।[](https://medium.com/%40codebob75/how-to-use-copilot-in-visual-studio-a-step-by-step-guide-b2a5db3b54ba)
- **सार्थक नामों का उपयोग करें**: वर्णनात्मक वेरिएबल और फ़ंक्शन नाम सुझाव गुणवत्ता में सुधार करते हैं।
- **चैट के साथ पुनरावृत्ति करें**: यदि प्रारंभिक सुझाव लक्ष्य से भटके हुए हैं तो प्रॉम्प्ट्स को परिष्कृत करें।
- **उपयोग सीमा की निगरानी करें**: Copilot फ्री के साथ, GitHub अकाउंट सेटिंग्स या VS Code में Copilot बैज के माध्यम से अपनी मासिक पूर्णताओं और चैट इंटरैक्शन को ट्रैक करें।[](https://learn.microsoft.com/en-us/visualstudio/ide/copilot-free-plan?view=vs-2022)

## 6. सामान्य समस्याओं का समस्या निवारण
- **Copilot निष्क्रिय**: सुनिश्चित करें कि आप एक GitHub अकाउंट से साइन इन हैं जिसके पास Copilot एक्सेस है। Copilot स्टेटस बार ड्रॉपडाउन के माध्यम से क्रेडेंशियल्स रीफ्रेश करें।
- **कोई सुझाव नहीं**: अपना इंटरनेट कनेक्शन जांचें या समर्थित भाषा में स्विच करें। **टूल्स** > **ऑप्शन्स** > **GitHub Copilot** के तहत सेटिंग्स समायोजित करें।
- **सीमित कार्यक्षमता**: यदि आप Copilot फ्री उपयोग सीमा तक पहुँच जाते हैं, तो आप IntelliCode सुझावों पर वापस आ जाएंगे। पेड प्लान में अपग्रेड करें या अपनी स्थिति जांचें।[](https://learn.microsoft.com/en-us/visualstudio/ide/copilot-free-plan?view=vs-2022)
- **नेटवर्क समस्याएं**: [GitHub Copilot समस्या निवारण गाइड](https://docs.github.com/en/copilot/troubleshooting) देखें।

## 7. उन्नत उपयोग के मामले
- **डेटाबेस क्वेरीज़**: SQL क्वेरीज़ जनरेट करने के लिए Copilot से पूछें (जैसे, `Write a SQL query to join two tables`)।
- **API डेवलपमेंट**: API एंडपॉइंट कोड के लिए अनुरोध करें (जैसे, `Create a Flask route to handle POST requests`)।
- **यूनिट टेस्टिंग**: Jest या Pytest जैसे फ्रेमवर्क के लिए टेस्ट जनरेट करने के लिए `/tests` का उपयोग करें।
- **रीफैक्टरिंग**: फाइलों में कोड रीफैक्टर करने के लिए एजेंट मोड का उपयोग करें (जैसे, `Migrate this jQuery code to React`)।

## 8. गोपनीयता और सुरक्षा विचार
- **डेटा उपयोग**: Copilot सुझाव जनरेट करने के लिए कोड स्निपेट्स को GitHub सर्वर पर रीयल-टाइम में प्रसारित करता है लेकिन उन्हें बनाए नहीं रखता है (Copilot for Business तुरंत स्निपेट्स को हटा देता है)।[](https://medium.com/%40codebob75/how-to-use-copilot-in-visual-studio-a-step-by-step-guide-b2a5db3b54ba)
- **सार्वजनिक कोड मिलान**: Copilot सार्वजनिक GitHub रिपॉजिटरी से मेल खाने वाले कोड का सुझाव दे सकता है। लाइसेंस विवरण देखने के लिए कोड रेफरेंसिंग सक्षम करें।[](https://learn.microsoft.com/en-us/visualstudio/ide/visual-studio-github-copilot-extension?view=vs-2022)
- **संस्थागत नीतियां**: जांचें कि क्या आपकी संस्था Copilot उपयोग की अनुमति देती है, क्योंकि कुछ चैट को अक्षम कर देती हैं या सुझावों को प्रतिबंधित करती हैं।[](https://docs.github.com/en/copilot/quickstart)
- **टेलीमेट्री**: यदि वांछित हो तो VS Code सेटिंग्स में टेलीमेट्री अक्षम करें।[](https://code.visualstudio.com/docs/copilot/setup)

## 10. निष्कर्ष
GitHub Copilot VS Code में एक शक्तिशाली टूल है जो बुद्धिमान सुझाव प्रदान करके, प्रश्नों के उत्तर देकर, और जटिल कार्यों को स्वचालित करके कोडिंग को सुव्यवस्थित करता है। इसे सही ढंग से सेट अप करके, इसकी विशेषताओं का लाभ उठाकर, और बेस्ट प्रैक्टिसेज का पालन करके, आप अपनी उत्पादकता को काफी बढ़ा सकते हैं और रचनात्मक कोडिंग कार्यों पर ध्यान केंद्रित कर सकते हैं। Copilot की क्षमताओं के साथ प्रयोग करें, इसे अपने वर्कफ़्लो के लिए अनुकूलित करें, और इस AI पेयर प्रोग्रामर का अधिकतम लाभ उठाने के लिए सुरक्षा और उपयोग सीमाओं के प्रति सजग रहें।
