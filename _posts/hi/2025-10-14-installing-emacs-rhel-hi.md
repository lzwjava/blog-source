### रेड हैट एंटरप्राइज लिनक्स पर Emacs इंस्टॉल करना

रेड हैट एंटरप्राइज लिनक्स (RHEL) मुख्य रूप से सॉफ़्टवेयर इंस्टॉल करने के लिए DNF पैकेज मैनेजर का उपयोग करता है। Emacs डिफ़ॉल्ट रिपॉजिटरीज़ में उपलब्ध है, इसलिए इंस्टॉलेशन सीधा-सादा है। नीचे आधुनिक वर्जन (RHEL 8 और बाद वाले) के लिए चरण दिए गए हैं। पुराने वर्जन (जैसे RHEL 7) के लिए, `dnf` को `yum` से बदल दें।

#### आवश्यक शर्तें
- सुनिश्चित करें कि आपके पास रूट विशेषाधिकार हैं (`sudo` का उपयोग करें)।
- यदि आधिकारिक RHEL रिपॉजिटरीज़ का उपयोग कर रहे हैं, तो आपका सिस्टम रेड हैट सब्सक्रिप्शन मैनेजमेंट के साथ पंजीकृत होना चाहिए।

#### इंस्टॉलेशन चरण
1. **अपना पैकेज कैश अपडेट करें** (सलाह दी जाती है ताकि आपको नवीनतम उपलब्ध वर्जन मिले):  
   ```
   sudo dnf update
   ```

2. **Emacs इंस्टॉल करें**:  
   ```
   sudo dnf install emacs
   ```  
   यह रिपॉजिटरीज़ में उपलब्ध नवीनतम स्थिर वर्जन इंस्टॉल कर देगा (उदाहरण के लिए, Emacs 28.x या 29.x, आपके RHEL वर्जन पर निर्भर करता है)।

3. **इंस्टॉलेशन सत्यापित करें**:  
   वर्जन जांचने के लिए निम्नलिखित चलाएँ:  
   ```
   emacs --version
   ```  
   आपको `GNU Emacs 28.2` (या इसी तरह) जैसा आउटपुट दिखाई देना चाहिए।

#### नोट्स
- यदि आप YUM का उपयोग करने वाले पुराने RHEL वर्जन (जैसे RHEL 7) पर हैं, तो कमांड `sudo yum install emacs` है।
- बिल्कुल नवीनतम Emacs वर्जन (जैसे 29+) के लिए, आपको स्रोत से कंपाइल करना पड़ सकता है या EPEL जैसी तीसरे पक्ष की रिपॉजिटरीज़ का उपयोग करना पड़ सकता है, क्योंकि बेस रिपोज़ अक्सर पीछे रह जाते हैं।
- Emacs लॉन्च करने के लिए, बस टर्मिनल में `emacs` टाइप करें।
