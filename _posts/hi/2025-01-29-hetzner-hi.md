मैं इस बार काफी उत्साहित हूँ कि इस क्लाउड प्लेटफॉर्म को प्रयोग करने के लिए।

{: .centered }
![](assets/images/hertzner/h.jpg)
*स्रोत: हेत्ज्नर*{: .caption }

एक सर्वर हेल्सिन्की में 2 AMD VCPUs, 2GB रैम, 40GB SSD, और 20TB ट्रैफिक के साथ कनफिगेशन में माहान रुपये 4.49 USD प्रति महीने का खर्च होता है।

एक IPv4 पता अतिरिक्त माहान रुपये 0.60 USD प्रति महीने का खर्च होता है, जिससे कुल माहान रुपये 5.09 USD प्रति महीने हो जाता है।

वे चारों ओर से सात स्थानों में सेवाएं प्रदान करते हैं:

- न्यूरेंबर्ग, जर्मनी
- फ़ाल्कनस्ताइन, जर्मनी
- हेल्सिन्की, फिनलैंड
- सिंगापुर, सिंगापुर
- हिल्स्बोरो, OR, यूएसए
- आशबर्न, VA, यूएसए

यह अभिनंदनीय है कि वे संख्या को लेकर प्रसिद्ध स्थानों का चयन नहीं करते। उनके स्थान वुल्ट्र या डिजिटल ओशन से अलग हैं।

फायरवॉल सेटिंग्स सुधार से सम्भव है। यह मेरा पहला बार था कि मैं इसे उपयोग कर रहा था, लेकिन मैं जल्द ही अपने प्रक्सी सर्वर के लिए सही कनफिगेशन सेट कर सका।

> sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/Jigsaw-Code/outline-server/master/src/server_manager/install_scripts/install_server.sh)"

हेत्ज्नर सर्वर हेल्सिन्की में बहुत जल्द है। स्पीडटेस्ट iOS एप का उपयोग करके, डाउनलोड स्पीड 423 Mbps, और अपलोड स्पीड 56.1 Mbps है।

शादोरॉकेट में पिंग 1175 ms है, लेकिन यह एक ही व्यापारिक समस्या नहीं है।

एक सरल पायथन स्क्रिप्ट सर्वर इंस्टांस की विवरण प्राप्त करने के लिए।

```python
from hcloud import Client
import os

# एनवायरन्ट वरियेबल से API टोकन प्राप्त करें
api_token = os.environ.get('HERTZNER_API_KEY')

यदि नहीं api_token:
    प्रिंट("समस्या: HERTZNER_API_KEY एनवायरन्ट वरियेबल नहीं सेट किया गया है।")
    एक्सिट(1)

# क्लाइंट इन्स्टांस बनाएं
client = Client(token=api_token)

# सभी सर्वरों को लिस्ट करें
servers = client.servers.get_all()

# सर्वर विवरण प्रिंट करें
सर्वर के लिए भीतर जाएं:
    प्रिंट(फ़"सर्वर आईडी: {सर्वर.आईडी}")
    प्रिंट(फ़"सर्वर नाम: {सर्वर.नाम}")
    प्रिंट(फ़"सर्वर स्टेटस: {सर्वर.स्टेटस}")
    प्रिंट(फ़"सर्वर IPv4: {सर्वर.पब्लिक_नेट.इपव4.इप}")
    प्रिंट(फ़"सर्वर IPv6: {सर्वर.पब्लिक_नेट.इपव6.इप}")
    प्रिंट(फ़"सर्वर प्रकार: {सर्वर.सर्वर_प्रकार.नाम}")
    प्रिंट(फ़"सर्वर स्थान: {सर्वर.डेटासेंटर.स्थान.नाम}")
    प्रिंट("----------------------------------")

# अगर आप एक विशिष्ट सर्वर को ID द्वारा प्राप्त करना चाहते हैं
सर्वर_आईडी = '59402674'
सर्वर = client.servers.get_by_id(सर्वर_आईडी)

प्रिंट(फ़"विशिष्ट सर्वर आईडी: {सर्वर.आईडी}")
प्रिंट(फ़"विशिष्ट सर्वर नाम: {सर्वर.नाम}")
प्रिंट(फ़"विशिष्ट सर्वर स्टेटस: {सर्वर.स्टेटस}")
प्रिंट(फ़"विशिष्ट सर्वर IPv4: {सर्वर.पब्लिक_नेट.इपव4.इप}")
प्रिंट(फ़"विशिष्ट सर्वर IPv6: {सर्वर.पब्लिक_नेट.इपव6.इप}")
प्रिंट(फ़"विशिष्ट सर्वर प्रकार: {सर्वर.सर्वर_प्रकार.नाम}")
प्रिंट(फ़"विशिष्ट सर्वर स्थान: {सर्वर.डेटासेंटर.स्थान.नाम}")
```
