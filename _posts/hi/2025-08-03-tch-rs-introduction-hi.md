### tch-rs का परिचय

tch-rs एक Rust क्रेट है जो PyTorch के C++ API के लिए बाइंडिंग्स प्रदान करता है, विशेष रूप से libtorch लाइब्रेरी के चारों ओर लपेटता है। इसका मुख्य उद्देश्य Rust डेवलपर्स को PyTorch की शक्तिशाली टेंसर गणनाओं, स्वचालित विभेदन, और मशीन लर्निंग क्षमताओं का लाभ सीधे Rust एप्लिकेशन्स में उठाने में सक्षम बनाना है। मूल C++ API को दर्शाने वाले पतले, निम्न-स्तरीय रैपर प्रदान करके, tch-rs इसके शीर्ष पर और अधिक स्वाभाविक Rust एब्स्ट्रक्शन्स के निर्माण की अनुमति देता है। इससे Rust इकोसिस्टम को छोड़े बिना मॉडल ट्रेनिंग, इन्फरेंस और टेंसर मैनिपुलेशन जैसे कार्यों को करना आसान हो जाता है।

#### मुख्य विशेषताएं
- **टेंसर ऑपरेशंस और ऑटोग्रैड**: कोर टेंसर अंकगणित, ग्रेडिएंट कम्प्यूटेशन और Adam जैसे ऑप्टिमाइज़र्स के माध्यम से मॉडल्स को प्रशिक्षित करने के लिए बैकप्रोपगेशन का समर्थन करता है।
- **न्यूरल नेटवर्क API**: न्यूरल आर्किटेक्चर बनाने और प्रशिक्षित करने के लिए टूल्स शामिल हैं, जिसमें MNIST डेटासेट पर एक सरल फीडफॉरवर्ड नेटवर्क जैसे उदाहरण शामिल हैं।
- **मॉडल लोडिंग**: safetensors फॉर्मेट का उपयोग करके पूर्व-प्रशिक्षित PyTorch मॉडल्स को आयात करने की अनुमति देता है, जो कुशल है और Python निर्भरताओं से बचाता है।
- **उदाहरण और उपयोग के मामले**: टेंसर निर्माण जैसी बुनियादी बातों, ग्रेडिएंट डिसेंट ट्रेनिंग, कस्टम न्यूरल नेट्स, और पूर्व-प्रशिक्षित मॉडल्स (जैसे, इमेज क्लासिफिकेशन के लिए ResNet18) चलाने के लिए व्यावहारिक डेमो के साथ आता है।
- **क्रॉस-प्लेटफॉर्म समर्थन**: विभिन्न सिस्टम्स पर काम करता है, जिसमें CPU या GPU (CUDA) एक्सेलेरेशन के विकल्प शामिल हैं।

यह प्रोजेक्ट सक्रिय रूप से बनाए रखा जाता है और Apache-2.0 और MIT लाइसेंस के तहत लाइसेंस प्राप्त है, जो ocaml-torch जैसे संबंधित प्रोजेक्ट्स से कुछ कोड जनरेशन प्राप्त करता है।

#### आरंभ करना
tch-rs का उपयोग करने के लिए, आपको अपने सिस्टम पर libtorch (PyTorch का C++ डिस्ट्रीब्यूशन) इंस्टॉल करना होगा। इसे Cargo के माध्यम से अपने Rust प्रोजेक्ट में जोड़ें:
```
[dependencies]
tch = "0.17.2"  # नवीनतम संस्करण की जांच करें
```
libtorch को मैन्युअल रूप से डाउनलोड करके, सिस्टम इंस्टॉलेशन का उपयोग करके, या स्वचालित फ़ेचिंग के लिए `download-libtorch` जैसी सुविधाओं को सक्षम करके सेट अप करें। GPU सपोर्ट के लिए, CUDA संस्करण निर्दिष्ट करें (उदाहरण के लिए, `TORCH_CUDA_VERSION=cu117` के माध्यम से)। `cargo run --example basics` जैसे कमांड के साथ उदाहरण चलाएं। ध्यान दें कि Windows उपयोगकर्ताओं को MSVC टूलचेन को प्राथमिकता देनी चाहिए, और स्टेटिक लिंकिंग पर्यावरण चर के माध्यम से उपलब्ध है।

अधिक जानकारी के लिए, रिपॉजिटरी में प्रदान किए गए दस्तावेज़ और उदाहरणों का अन्वेषण करें।
