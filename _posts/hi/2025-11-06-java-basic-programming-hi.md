## 1. कंसोल से इनपुट पढ़ना

### 1.1 `Scanner` का उपयोग करना

जावा `java.util` में `Scanner` क्लास के साथ इनपुट पढ़ता है।

```java
import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("अपना नाम दर्ज करें: ");
        String name = sc.nextLine();
        System.out.print("अपनी आयु दर्ज करें: ");
        int age = sc.nextInt();
        System.out.println("नमस्ते " + name + ", आपकी आयु " + age + " वर्ष है।");
        sc.close();
    }
}
```

**मुख्य बिंदु:**

* हमेशा `import java.util.Scanner;` करें।
* डेटा टाइप के आधार पर `nextInt()`, `nextDouble()`, `nextLine()` का उपयोग करें।
* संसाधनों को मुक्त करने के लिए `Scanner` को बंद करें (`close`)।
* सावधानी: `nextLine()` एक लाइन का बाकी हिस्सा पढ़ता है, इसलिए इसे `nextInt()` के साथ मिलाने पर इनपुट छूट सकता है।

## 2. आइडेंटिफायर, वेरिएबल, एक्सप्रेशन, असाइनमेंट और कॉन्स्टेंट

### 2.1 आइडेंटिफायर

वेरिएबल, मेथड या क्लासेज को दिए गए नाम।
**नियम:**

* एक अक्षर, `_`, या `$` से शुरू होना चाहिए।
* किसी संख्या से शुरू नहीं हो सकता।
* केस-सेंसिटिव (`score` ≠ `Score`)।
* कीवर्ड (`int`, `class`, `if`, आदि) नहीं हो सकता।

**उदाहरण:**
`studentName`, `_total`, `$price`

### 2.2 वेरिएबल

एक वेरिएबल किसी निश्चित **टाइप** का डेटा रखता है।
घोषणा के उदाहरण:

```java
int age = 20;
double price = 12.5;
char grade = 'A';
boolean passed = true;
```

### 2.3 असाइनमेंट स्टेटमेंट

`=` का उपयोग करके एक वैल्यू असाइन करें (दाएं → बाएं):

```java
x = 5;
y = x + 2;
```

### 2.4 कॉन्स्टेंट

`final` के साथ घोषित, बाद में नहीं बदल सकते:

```java
final double PI = 3.14159;
```

कॉन्स्टेंट के लिए अपरकेस नामों का उपयोग करें।

## 3. न्यूमेरिक डेटा टाइप और ऑपरेशन

### 3.1 सामान्य न्यूमेरिक टाइप

* `byte` (8-बिट इंटीजर)
* `short` (16-बिट)
* `int` (32-बिट, सबसे आम)
* `long` (64-बिट)
* `float` (32-बिट दशमलव)
* `double` (64-बिट दशमलव, अधिक सटीक)

**उदाहरण:**

```java
int count = 5;
double price = 19.99;
```

### 3.2 बेसिक अंकगणितीय ऑपरेटर

`+`, `-`, `*`, `/`, `%`

उदाहरण:

```java
int a = 10, b = 3;
System.out.println(a / b);  // 3 (इंटीजर डिवीजन)
System.out.println(a % b);  // 1
System.out.println((double)a / b); // 3.3333 (टाइप कनवर्जन)
```

## 4. टाइप कनवर्जन (कास्टिंग)

### 4.1 ऑटोमैटिक कनवर्जन (वाइडनिंग)

छोटा → बड़ा टाइप स्वचालित रूप से:
`int` → `long` → `float` → `double`

उदाहरण:

```java
int i = 10;
double d = i;  // ऑटोमैटिक कनवर्जन
```

### 4.2 मैनुअल कनवर्जन (कास्टिंग)

स्पष्ट रूप से बड़ा → छोटा टाइप कनवर्ट करें:

```java
double d = 9.7;
int i = (int) d; // i = 9 (भिन्न हट गया)
```

**प्रिसिजन लॉस** के साथ सावधान रहें।

## 5. एक्सप्रेशन इवैल्यूएशन और ऑपरेटर प्रीसीडेंस

### 5.1 ऑपरेशन का क्रम

जावा एक परिभाषित क्रम का पालन करता है:

1. कोष्ठक `( )`
2. यूनरी `+`, `-`, `++`, `--`
3. गुणा, भाग, मापांक `* / %`
4. जोड़ और घटाव `+ -`
5. असाइनमेंट `=`

उदाहरण:

```java
int x = 2 + 3 * 4;   // 14, 20 नहीं
int y = (2 + 3) * 4; // 20
```

### 5.2 मिश्रित एक्सप्रेशन

यदि एक ऑपरेंड `double` है, तो परिणाम `double` बन जाता है:

```java
double result = 5 / 2;     // 2.0 (पहले इंटीजर डिवीजन)
double result2 = 5.0 / 2;  // 2.5 (फ्लोटिंग-पॉइंट डिवीजन)
```

## 6. संवर्धित असाइनमेंट और इंक्रीमेंट/डिक्रीमेंट

### 6.1 संवर्धित असाइनमेंट

शॉर्टकट ऑपरेटर:

```java
x += 3;  // x = x + 3 के समान
y *= 2;  // y = y * 2 के समान
```

### 6.2 इंक्रीमेंट और डिक्रीमेंट

`++` 1 से बढ़ाता है, `--` 1 से घटाता है।
दो रूप:

```java
int a = 5;
System.out.println(a++); // 5 प्रिंट करता है, फिर a = 6
System.out.println(++a); // a = 7, फिर 7 प्रिंट करता है
```

**याद रखें:**

* पोस्टफिक्स (`a++`) → उपयोग करें फिर बढ़ाएं
* प्रीफिक्स (`++a`) → बढ़ाएं फिर उपयोग करें

## 7. टालने के लिए सामान्य गलतियाँ

* तुलना में `==` के बजाय `=` का उपयोग करना।
* इंटीजर डिवीजन जब आप दशमलव की अपेक्षा करते हैं।
* उपयोग से पहले वेरिएबल को इनिशियलाइज़ करना भूल जाना।
* टाइप कनवर्जन में डेटा खोना (जैसे, `(int) 3.9` → 3)।

## 8. विशिष्ट परीक्षा प्रश्न

**उदाहरण 1:**

> निम्नलिखित कोड का आउटपुट क्या है?
>
> ```java
> int x = 5, y = 2;
> double z = x / y;
> System.out.println(z);
> ```

**उत्तर:** `2.0` (असाइनमेंट से पहले इंटीजर डिवीजन)

**उदाहरण 2:**

> किस ऑपरेटर की प्रीसीडेंस सबसे अधिक है?
> A. `+`
> B. `*`
> C. `=`
> D. `++`
> **उत्तर:** D

**उदाहरण 3:**

> इस कोड का आउटपुट क्या है?
>
> ```java
> int a = 3;
> int b = ++a + a++;
> System.out.println(b);
> ```

**स्पष्टीकरण:**
`++a`, `a` को `4` बनाता है, `4` जोड़ता है, फिर `a++`, `4` का उपयोग करता है और `5` तक बढ़ाता है।
इसलिए `b = 8`।
