`discord-0.0.111.deb` फ़ाइल को इंस्टॉल करने के लिए, आप कई तरीकों का उपयोग कर सकते हैं:

**विधि 1: dpkg का उपयोग करना (कमांड लाइन)**
```bash
sudo dpkg -i discord-0.0.111.deb
```

यदि आपको निर्भरता संबंधी समस्याएँ आती हैं, तो चलाएँ:
```bash
sudo apt-get install -f
```

**विधि 2: apt का उपयोग करना (अनुशंसित)**
```bash
sudo apt install ./discord-0.0.111.deb
```

यह विधि dpkg की तुलना में निर्भरताओं को स्वचालित रूप से बेहतर तरीके से संभालती है।

**विधि 3: gdebi का उपयोग करना (यदि इंस्टॉल है)**
```bash
sudo gdebi discord-0.0.111.deb
```

**विधि 4: GUI विधि**
यदि आप डेस्कटॉप एनवायरनमेंट का उपयोग कर रहे हैं, तो आप अक्सर .deb फ़ाइल पर डबल-क्लिक कर सकते हैं और यह इंस्टॉलेशन के लिए आपके पैकेज मैनेजर (जैसे सॉफ़्टवेयर सेंटर) में खुल जाएगी।

**पूर्वापेक्षाएँ:**
- सुनिश्चित करें कि आप .deb फ़ाइल वाली डायरेक्टरी में हैं
- आपको sudo विशेषाधिकारों की आवश्यकता होगी
- यदि आपके पास अभी तक फ़ाइल नहीं है, तो आप इसे Discord की आधिकारिक वेबसाइट से डाउनलोड कर सकते हैं
