हाँ, Azure Virtual Machines (VMs) Linux ऑपरेटिंग सिस्टम को चलाने का पूर्ण समर्थन करती हैं। आप Azure Marketplace से सीधे लोकप्रिय डिस्ट्रीब्यूशन की एक विस्तृत श्रृंखला तैनात कर सकते हैं या अपनी स्वयं की कस्टम Linux इमेज अपलोड कर सकते हैं।

### मुख्य बिंदु:
- **समर्थित डिस्ट्रीब्यूशन**: Azure प्रमुख Linux वितरणों को समर्थन और स्वीकृति प्रदान करता है, जिनमें Ubuntu, Red Hat Enterprise Linux (RHEL), Debian, SUSE Linux Enterprise Server (SLES), Oracle Linux, CentOS Stream, Rocky Linux, AlmaLinux और भी बहुत कुछ शामिल हैं। 100 से अधिक Linux इमेज बॉक्स से बाहर उपलब्ध हैं।
- **तैनाती विकल्प**: Linux VMs बनाने के लिए Azure पोर्टल, CLI, PowerShell, या ARM टेम्पलेट्स का उपयोग करें। उदाहरण के लिए, आप मिनटों में एक Ubuntu Server इंस्टेंस शुरू कर सकते हैं।
- **विशेषताएँ**: Azure पर Linux VMs में स्टोरेज, नेटवर्किंग, सुरक्षा और मॉनिटरिंग जैसी Azure सेवाओं के साथ पूर्ण एकीकरण शामिल है। वे हाइब्रिड प्रबंधन के लिए Azure Arc जैसे टूल्स का भी समर्थन करती हैं।

यदि आप शुरुआत कर रहे हैं, तो Linux VM बनाने के लिए आधिकारिक क्विकस्टार्ट गाइड देखें।
