जब आप `curl -O` का उपयोग करके `eclipse.org` से एक Eclipse ZIP फ़ाइल डाउनलोड करने की कोशिश करते हैं, तो आप एक ऐसी समस्या का सामना कर सकते हैं जहां यह एक HTML फ़ाइल डाउनलोड करता है बजाय किराए के ZIP फ़ाइल के साथ "301 Moved Permanently" त्रुटि के साथ। यह तब होता है जब URL जो आप उपयोग कर रहे हैं, किसी और स्थान पर पुनर्निर्देशित हो रहा है, और `curl` उस पुनर्निर्देश को स्वचालित रूप से नहीं पालन कर रहा है वास्तविक फ़ाइल तक।

### समस्या समझना

- **301 Moved Permanently**: यह HTTP स्थिति कोड यह बताता है कि संसाधन (ZIP फ़ाइल) एक नए URL पर स्थायी रूप से स्थानांतरित हो गया है। सर्वर अक्सर डाउनलोड लिंक के लिए पुनर्निर्देश का उपयोग करते हैं ताकि वे मिरर साइटों या अपडेटेड स्थानों की ओर इशारा कर सकें।
- **`curl -O` व्यवहार**: `-O` विकल्प `curl` को बताता है कि डाउनलोड की गई सामग्री को दूरस्थ फ़ाइल के नाम से एक फ़ाइल में सेंधा करें। फिर भी, बिना पुनर्निर्देशों को स्पष्ट रूप से संभालने, `curl` मध्यम पुनर्निर्देश प्रतिक्रिया (अक्सर एक HTML पेज) की सामग्री को सेंधा कर सकता है बजाय किराए के ZIP फ़ाइल तक पहुंचने के लिए।

### समाधान

`curl` को पुनर्निर्देश का पालन करने और Eclipse ZIP फ़ाइल डाउनलोड करने के लिए, `-L` विकल्प `-O` के साथ उपयोग करें। `-L` फ्लैग `curl` को बताता है कि वह किसी भी पुनर्निर्देश का पालन करे जब तक कि वह अंतिम गंतव्य तक पहुंचे।

#### कमांड

```bash
curl -L -O <URL>
```

- **`-L`**: पुनर्निर्देशों का पालन करता है, जैसे कि 301 पुनर्निर्देश, नए स्थान तक।
- **`-O`**: अंतिम URL से मूल नाम से फ़ाइल को सेंधा करता है।
- **`<URL>`**: इसे एक विशिष्ट Eclipse डाउनलोड URL से बदलें, जैसे कि `https://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/2023-03/R/eclipse-java-2023-03-R-win32-x86_64.zip`.

### कदम-दर-कदम निर्देश

1. **सही URL ढूंढें**:
   - Eclipse वेबसाइट पर जाएं (जैसे कि `https://www.eclipse.org/downloads/`).
   - पसंदीदा पैकेज का चयन करें (जैसे कि Eclipse IDE for Java Developers).
   - डाउनलोड लिंक या बटन पर दाएं क्लिक करें और URL को कॉपी करें। अन्यथा, अपने ब्राउज़र के डेवलपर टूल्स (F12, नेटवर्क टैब) का उपयोग करें डाउनलोड करने पर क्लिक करने के लिए सही URL को कैप्चर करने के लिए।

2. **कमांड चलाएं**:
   - अपने टर्मिनल खोलें।
   - `-L` और `-O` विकल्पों के साथ `curl` कमांड चलाएं, URL जो आपने कॉपी किया है:
     ```bash
     curl -L -O https://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/2023-03/R/eclipse-java-2023-03-R-win32-x86_64.zip
     ```
   - यह ZIP फ़ाइल (जैसे कि `eclipse-java-2023-03-R-win32-x86_64.zip`) को आपके वर्तमान डायरेक्टरी में डाउनलोड करेगा।

### समस्या निपटाने

अगर आप अभी भी समस्याओं का सामना कर रहे हैं, तो इन अतिरिक्त कदमों का प्रयास करें:

- **पुनर्निर्देशों का जांच करें विस्तृत आउटपुट के साथ**:
  - `-v` विकल्प का उपयोग करें ताकि अनुरोध और पुनर्निर्देश के बारे में विस्तृत जानकारी देखें:
    ```bash
    curl -v -L -O <URL>
    ```
  - 301 प्रतिक्रिया में `Location` हेडर देखें ताकि पता लगाएं कि वह कहां पुनर्निर्देशित कर रहा है। अगर वह एक HTML पेज की ओर इशारा कर रहा है बजाय किराए के ZIP फ़ाइल, तो URL डाउनलोड लिंक नहीं हो सकता है।

- **URL की पुष्टि करें**:
  - Eclipse अक्सर एक `download.php` स्क्रिप्ट का उपयोग करता है जो एक मिरर पर पुनर्निर्देशित हो सकता है। सुनिश्चित करें कि आपका URL या तो ZIP फ़ाइल के सीधे लिंक है या एक पुनर्निर्देश लिंक जो फ़ाइल तक समाधान करता है। उदाहरण के लिए, एक सीधा मिरर लिंक इस प्रकार दिख सकता है:
    ```bash
    curl -L -O https://mirror.its.dal.ca/eclipse/technology/epp/downloads/release/2023-03/R/eclipse-java-2023-03-R-win32-x86_64.zip
    ```

- **एक उपयोगकर्ता एजेंट हेडर सेट करें**:
  - कुछ सर्वर क्लाइंट के आधार पर अलग व्यवहार करते हैं। अगर सर्वर `curl` को पहचानकर HTML लौटाता है, तो एक ब्राउज़र को नकल करके एक उपयोगकर्ता एजेंट जोड़ें:
    ```bash
    curl -L -O -A "Mozilla/5.0" <URL>
    ```

### यह क्यों काम करता है

Eclipse वेबसाइट डाउनलोड अनुरोधों को एक मिरर या अपडेटेड URL पर पुनर्निर्देशित कर सकती है। बिना `-L`, `curl -O` पुनर्निर्देश प्रतिक्रिया (एक HTML पेज) को सेंधा करता है बजाय किराए के ZIP फ़ाइल तक पहुंचने के लिए। `-L` जोड़ने से सुनिश्चित होता है कि `curl` 301 पुनर्निर्देश को वास्तविक ZIP फ़ाइल तक पालन करे, समस्या को हल करने के लिए।

इन कदमों का पालन करके, आप `curl` का उपयोग करके `eclipse.org` से Eclipse ZIP फ़ाइल को सफलतापूर्वक डाउनलोड कर सकते हैं।
