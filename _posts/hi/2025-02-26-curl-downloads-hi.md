जब आप `curl -O` का उपयोग करके eclipse.org से एक Eclipse ZIP फ़ाइल डाउनलोड करने का प्रयास करते हैं, तो आपको एक समस्या का सामना करना पड़ सकता है जहां यह वांछित ZIP फ़ाइल के बजाय एक HTML फ़ाइल डाउनलोड करता है, साथ ही एक "301 Moved Permanently" त्रुटि भी मिलती है। ऐसा इसलिए होता है क्योंकि आप जिस URL का उपयोग कर रहे हैं वह किसी अन्य स्थान पर रीडायरेक्ट हो रहा है, और `curl` स्वचालित रूप से उस रीडायरेक्ट को वास्तविक फ़ाइल तक फॉलो नहीं कर रहा है। इसे ठीक करने का तरीका यहां बताया गया है:

### समस्या को समझना

- **301 Moved Permanently**: यह HTTP स्टेटस कोड का मतलब है कि संसाधन (ZIP फ़ाइल) स्थायी रूप से एक नए URL पर चली गई है। सर्वर अक्सर डाउनलोड लिंक के लिए मिरर साइट्स या अपडेटेड लोकेशन की ओर इशारा करने के लिए रीडायरेक्ट का उपयोग करते हैं।
- **`curl -O` का व्यवहार**: `-O` विकल्प `curl` को डाउनलोड की गई सामग्री को रिमोट फ़ाइल के नाम पर सेव करने के लिए कहता है। हालांकि, रीडायरेक्ट को स्पष्ट रूप से हैंडल किए बिना, `curl` ZIP फ़ाइल पर जाने के बजाय इंटरमीडिएट रीडायरेक्ट प्रतिक्रिया (अक्सर एक HTML पेज) की सामग्री को सेव कर सकता है।

### समाधान

यह सुनिश्चित करने के लिए कि `curl` रीडायरेक्ट को फॉलो करे और Eclipse ZIP फ़ाइल डाउनलोड करे, `-O` के साथ `-L` विकल्प का उपयोग करें। `-L` फ्लैग `curl` को अंतिम गंतव्य तक पहुंचने तक किसी भी रीडायरेक्ट को फॉलो करने का निर्देश देता है।

#### कमांड

```bash
curl -L -O <URL>
```

- **`-L`**: 301 रीडायरेक्ट जैसे रीडायरेक्ट को नई लोकेशन तक फॉलो करता है।
- **`-O`**: फ़ाइल को अंतिम URL से मूल नाम के साथ सेव करता है।
- **`<URL>`**: इसे विशिष्ट Eclipse डाउनलोड URL से बदलें, जैसे `https://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/2023-03/R/eclipse-java-2023-03-R-win32-x86_64.zip`।

### चरण-दर-चरण निर्देश

1. **सही URL ढूंढें**:
   - Eclipse वेबसाइट पर जाएं (उदाहरण के लिए, `https://www.eclipse.org/downloads/`)।
   - वांछित पैकेज चुनें (उदाहरण के लिए, Eclipse IDE for Java Developers)।
   - डाउनलोड लिंक या बटन पर राइट-क्लिक करें और URL कॉपी करें। वैकल्पिक रूप से, डाउनलोड पर क्लिक करने पर सटीक URL कैप्चर करने के लिए अपने ब्राउज़र के डेवलपर टूल्स (F12, Network टैब) का उपयोग करें।

2. **कमांड रन करें**:
   - अपना टर्मिनल खोलें।
   - `-L` और `-O` विकल्पों के साथ `curl` कमांड निष्पादित करें, आपके द्वारा कॉपी किए गए URL का उपयोग करके:
     ```bash
     curl -L -O https://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/2023-03/R/eclipse-java-2023-03-R-win32-x86_64.zip
     ```
   - इससे ZIP फ़ाइल (उदाहरण के लिए, `eclipse-java-2023-03-R-win32-x86_64.zip`) आपके करंट डायरेक्टरी में डाउनलोड हो जानी चाहिए।

### समस्या निवारण

यदि आपको अभी भी समस्या आ रही है, तो इन अतिरिक्त चरणों को आज़माएं:

- **वर्बोस आउटपुट के साथ रीडायरेक्ट जांचें**:
  - रिक्वेस्ट और रीडायरेक्ट के बारे में विस्तृत जानकारी देखने के लिए `-v` विकल्प का उपयोग करें:
    ```bash
    curl -v -L -O <URL>
    ```
  - यह देखने के लिए कि यह कहाँ रीडायरेक्ट हो रहा है, 301 प्रतिक्रिया में `Location` हेडर देखें। यदि यह ZIP फ़ाइल के बजाय किसी HTML पेज की ओर इशारा कर रहा है, तो URL सीधा डाउनलोड लिंक नहीं हो सकता है।

- **URL सत्यापित करें**:
  - Eclipse अक्सर एक `download.php` स्क्रिप्ट का उपयोग करता है जो एक मिरर पर रीडायरेक्ट कर सकती है। सुनिश्चित करें कि आपका URL या तो सीधा ZIP फ़ाइल का लिंक है या एक रीडायरेक्टिंग लिंक है जो फ़ाइल पर रिज़ॉल्व होता है। उदाहरण के लिए, एक सीधा मिरर लिंक इस तरह दिख सकता है:
    ```bash
    curl -L -O https://mirror.its.dal.ca/eclipse/technology/epp/downloads/release/2023-03/R/eclipse-java-2023-03-R-win32-x86_64.zip
    ```

- **यूजर-एजेंट हेडर सेट करें**:
  - कुछ सर्वर क्लाइंट के आधार पर अलग तरह से व्यवहार करते हैं। यदि सर्वर HTML लौटाता है क्योंकि उसे `curl` का पता चलता है, तो एक ब्राउज़र की नकल करने के लिए यूजर-एजेंट जोड़ें:
    ```bash
    curl -L -O -A "Mozilla/5.0" <URL>
    ```

### यह क्यों काम करता है

Eclipse वेबसाइट डाउनलोड अनुरोधों को एक मिरर या अपडेटेड URL पर रीडायरेक्ट कर सकती है। `-L` के बिना, `curl -O` रीडायरेक्ट प्रतिक्रिया (एक HTML पेज) को सेव कर देता है बजाय उसे फॉलो करने के। `-L` जोड़ने से यह सुनिश्चित होता है कि `curl` वास्तविक ZIP फ़ाइल तक 301 रीडायरेक्ट को ट्रैक करता है, जिससे समस्या हल हो जाती है।

इन चरणों का पालन करके, आपको `curl` का उपयोग करके eclipse.org से Eclipse ZIP फ़ाइल सफलतापूर्वक डाउनलोड करनी चाहिए।
