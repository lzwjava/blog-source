### DevDependencies
ये पैकेज डेवलपमेंट के दौरान उपयोग किए जाते हैं, फाइनल बिल्ड में शामिल नहीं किए जाते।

- **babel-core** (6.13.0): कोर Babel कंपाइलर जो आधुनिक JavaScript (ES6+) को व्यापक ब्राउज़र संगतता के लिए ES5 में ट्रांसपाइल करता है।
- **babel-loader** (6.2.5): वेबपैक लोडर जो बिल्ड प्रक्रियाओं में JavaScript ट्रांसफॉर्मेशन के लिए Babel को एकीकृत करता है।
- **babel-plugin-transform-runtime** (6.12.0): Babel प्लगइन जो ट्रांसपाइल्ड कोड में बंडल आकार को कम करने के लिए रनटाइम हेल्पर्स का पुन: उपयोग करता है।
- **babel-preset-es2015** (6.13.2): ES2015 (ES6) फीचर्स को ES5 में कंपाइल करने के लिए Babel प्रीसेट।
- **babel-runtime** (6.11.6): रनटाइम लाइब्रेरी जो Babel-ट्रांसपाइल्ड कोड के लिए पॉलीफिल और हेल्पर्स प्रदान करती है।
- **cross-env** (^1.0.8): शेल अंतरों के बिना क्रॉस-प्लेटफॉर्म पर एनवायरनमेंट वेरिएबल सेट करता है (जैसे, NODE_ENV)।
- **css-loader** (^0.23.1): CSS फाइलों को लोड और प्रोसेस करता है, इम्पोर्ट और डिपेंडेंसी को रिजॉल्व करता है।
- **detect-indent** (4.0.0): सुसंगत फॉर्मेटिंग के लिए फाइलों की इंडेंटेशन शैली (स्पेस/टैब) का पता लगाता है।
- **exports-loader** (^0.6.3): मॉड्यूल एक्सपोर्ट को अलग-अलग संदर्भों में उपलब्ध कराता है (जैसे, गैर-AMD मॉड्यूल के लिए)।
- **extract-text-webpack-plugin** (^1.0.1): बेहतर परफॉर्मेंस के लिए JavaScript बंडल से CSS को अलग फाइलों में निकालता है।
- **file-loader** (0.9.0): फाइलों (जैसे, इमेज) को आउटपुट डायरेक्टरी में एमिट करके और URL वापस करके उन्हें हैंडल करता है।
- **html-webpack-plugin** (^2.22.0): HTML फाइलें जेनरेट करता है और बंडल्ड एसेट्स को इंजेक्ट करता है, सिंगल-पेज ऐप सेटअप को सरल बनाता है।
- **rimraf** (^2.5.4): क्रॉस-प्लेटफॉर्म रिकर्सिव फाइल रिमूवल (यूनिक्स पर rm -rf की तरह)।
- **style-loader** (^0.13.1): डायनामिक लोडिंग के लिए स्टाइल टैग के माध्यम से CSS को DOM में इंजेक्ट करता है।
- **stylus** (^0.54.5): CSS प्रीप्रोसेसर जिसमें एक्सप्रेसिव सिंटैक्स है, CSS में कंपाइल किया जाता है।
- **stylus-loader** (^2.1.1): स्टाइलस फाइलों को CSS में प्रोसेस करने के लिए वेबपैक लोडर।
- **url-loader** (0.5.7): छोटी फाइलों (जैसे, इमेज) को इनलाइन बेस64 एनकोड करता है या बड़ी फाइलों को एमिट करता है; file-loader पर फॉलबैक करता है।
- **vue-hot-reload-api** (^1.2.0): डेवलपमेंट के दौरान Vue.js कंपोनेंट्स के लिए हॉट मॉड्यूल रिप्लेसमेंट सक्षम करता है।
- **vue-html-loader** (^1.0.0): Vue सिंगल-फाइल कंपोनेंट्स में HTML टेम्पलेट को पार्स करने के लिए वेबपैक लोडर।
- **vue-loader** (8.5.3): Vue सिंगल-फाइल कंपोनेंट्स (.vue फाइलों) को JavaScript और CSS में लोड और प्रोसेस करता है।
- **vue-style-loader** (^1.0.0): Vue कंपोनेंट्स से CSS को हैंडल करता है, style-loader के साथ एकीकृत होता है।
- **webpack** (1.13.2): वेब एसेट्स जैसे JS, CSS, और इमेज बनाने और ऑप्टिमाइज़ करने के लिए मॉड्यूल बंडलर।
- **webpack-dev-server** (1.14.0): लाइव रीलोडिंग और हॉट मॉड्यूल रिप्लेसमेंट के साथ डेवलपमेंट सर्वर।

### Dependencies
ये रनटाइम पैकेज हैं जो फाइनल एप्लिकेशन बिल्ड में शामिल होते हैं।

- **debug** (^2.2.0): नेमस्पेस्ड लॉगिंग और सशर्त आउटपुट के साथ डीबग यूटिलिटी (केवल DEBUG env var के माध्यम से सक्षम)।
- **es6-promise** (^3.0.2): पुराने ब्राउज़र/एनवायरनमेंट में ES6 Promise API के लिए पॉलीफिल।
- **font-awesome** (^4.6.3): लोकप्रिय आइकन लाइब्रेरी जो CSS क्लासेस के माध्यम से स्केलेबल वेक्टर आइकन प्रदान करती है।
- **github-markdown-css** (^2.4.0): GitHub-फ्लेवर्ड मार्कडाउन को स्टाइल करने के लिए CSS।
- **highlight.js** (^9.6.0): कई भाषाओं में कोड ब्लॉक के लिए सिंटैक्स हाइलाइटर।
- **hls.js** (^0.7.6): HTML5 वीडियो के साथ HTTP लाइव स्ट्रीमिंग (HLS) वीडियो चलाने के लिए JavaScript लाइब्रेरी।
- **inherit** (^2.2.6): JavaScript ऑब्जेक्ट्स में शास्त्रीय और प्रोटोटाइपल इनहेरिटेंस के लिए यूटिलिटी।
- **jquery** (^3.1.0): DOM मैनिपुलेशन, AJAX, और इवेंट्स के लिए तेज़, फीचर-रिच JavaScript लाइब्रेरी।
- **json-loader** (^0.5.4): JSON फाइलों को JavaScript मॉड्यूल के रूप में लोड करता है।
- **leancloud-realtime** (^3.2.3): LeanCloud की रियल-टाइम मैसेजिंग और डेटा सिंक्रोनाइज़ेशन सेवा के लिए SDK।
- **marked** (^0.3.6): मार्कडाउन पार्सर जो मार्कडाउन को HTML में कन्वर्ट करता है।
- **moment** (^2.14.1): डेट पार्सिंग, वैलिडेशन, और फॉर्मेटिंग के लिए डेट/टाइम मैनिपुलेशन लाइब्रेरी।
- **moment-timezone** (^0.5.7): Moment.js का एक्सटेंशन जो टाइम ज़ोन और DST को हैंडल करता है।
- **qiniu-js-sdk** (0.0.2): Qiniu क्लाउड स्टोरेज के साथ इंटरैक्ट करने के लिए JavaScript SDK (फाइलें अपलोड/डाउनलोड करना)।
- **raw-loader** (^0.5.1): वेबपैक लोडर जो फाइलों को रॉ स्ट्रिंग के रूप में लोड करता है (जैसे, टेक्स्ट टेम्पलेट)।
- **rupture** (^0.6.1): स्टाइलस के लिए मीडिया क्वेरी एक्सटेंशन, रिस्पॉन्सिव डिज़ाइन के लिए ब्रेकपॉइंट प्रदान करता है।
- **sprintf-js** (^1.0.3): प्लेसहोल्डर के साथ स्ट्रिंग फॉर्मेटिंग के लिए JavaScript sprintf इम्प्लीमेंटेशन।
- **vue** (^1.0.26): यूजर इंटरफेस बनाने के लिए प्रोग्रेसिव MVVM फ्रेमवर्क (v1.x, जो पुराना है)।
- **vue-resource** (^0.6.1): Vue.js के लिए HTTP क्लाइंट जो AJAX/API रिक्वेस्ट को सरल बनाता है।
- **vue-router** (^0.7.13): Vue.js के लिए ऑफिशियल राउटर, सिंगल-पेज एप्लिकेशन नेविगेशन सक्षम करता है।
- **vue-spinner** (^1.0.2): Vue.js के लिए स्पिनर/लोडिंग कंपोनेंट लाइब्रेरी।
- **vue-weui** (^0.3.2): WeUI कंपोनेंट्स के लिए Vue.js रैपर (WeChat-स्टाइल इंटरफेस के लिए UI लाइब्रेरी)।
- **weixin-js-sdk** (^1.2.0): WeChat वेब फीचर्स (जैसे, शेयरिंग, स्कैनिंग) को इंटीग्रेट करने के लिए WeChat JavaScript SDK।
- **weui** (^0.4.3): WeChat के डिज़ाइन से प्रेरित मोबाइल इंटरफेस बनाने के लिए WeUI CSS फ्रेमवर्क।
- **word-color** (^1.2.0): शब्दों को रंगने के लिए लाइब्रेरी, जो हैश या भाषा जैसे कारकों पर आधारित है, संभवतः हीटमैप जैसे प्रभावों के लिए।
