### MinGW-w64 में शामिल एक्जिक्यूटेबल्स

MinGW-w64 मुख्य रूप से GCC और संबंधित टूल्स का उपयोग करके नेटिव Windows एप्लिकेशन को क्रॉस-कंपाइल करने के लिए एक मिनिमल टूलचेन है। यह एक पूर्ण यूनिक्स-जैसी कमांड सूट (उदाहरण के लिए, यह डिफ़ॉल्ट रूप से `gpg` या GnuPG शामिल **नहीं** करता है—वे MSYS2 जैसे वातावरण में अलग पैकेज के रूप में उपलब्ध हैं) के बजाय डेवलपमेंट यूटिलिटीज पर केंद्रित है। कोर इंस्टॉलेशन की `bin` डायरेक्टरी में आमतौर पर निम्नलिखित एक्जिक्यूटेबल्स होते हैं (एक स्टैंडअलोन डाउनलोड के आधार पर, जैसे कि ऑफिशियल SourceForge रिलीज़ से)। यह सूची वर्जन और बिल्ड के अनुसार थोड़ी भिन्न हो सकती है, लेकिन ये सामान्य हैं:

| एक्जिक्यूटेबल | विवरण |
|------------|-------------|
| `gcc.exe` | GNU C कंपाइलर |
| `g++.exe` | GNU C++ कंपाइलर |
| `cpp.exe` | C प्रीप्रोसेसर |
| `gfortran.exe` | GNU Fortran कंपाइलर (यदि बिल्ड में शामिल हो) |
| `ld.exe` | GNU लिंकर |
| `as.exe` | GNU असेंबलर |
| `ar.exe` | लाइब्रेरी बनाने के लिए आर्काइवर |
| `ranlib.exe` | आर्काइव लाइब्रेरी के लिए इंडेक्स जनरेट करता है |
| `nm.exe` | ऑब्जेक्ट फाइलों से सिंबल्स की सूची बनाता है |
| `objdump.exe` | ऑब्जेक्ट फाइलों से जानकारी प्रदर्शित करता है |
| `objcopy.exe` | ऑब्जेक्ट फाइलों को कॉपी और ट्रांसलेट करता है |
| `strip.exe` | एक्जिक्यूटेबल्स से सिंबल्स हटाता है |
| `size.exe` | ऑब्जेक्ट फाइलों के सेक्शन साइज की सूची बनाता है |
| `strings.exe` | फाइलों में प्रिंटेबल स्ट्रिंग्स प्रिंट करता है |
| `readelf.exe` | ELF फाइल जानकारी प्रदर्शित करता है (क्रॉस-टूल्स के लिए) |
| `dlltool.exe` | DLL इम्पोर्ट/एक्सपोर्ट फाइलें बनाता है |
| `windres.exe` | Windows रिसोर्स फाइलों को कंपाइल करता है |
| `gendef.exe` | DLL से .def फाइलें जनरेट करता है |
| `genidl.exe` | DLL से .idl फाइलें जनरेट करता है |
| `widl.exe` | Wine/Winelib के लिए IDL फाइलों को कंपाइल करता है |
| `gcc-ar.exe` | ar के लिए GCC रैपर |
| `gcc-nm.exe` | nm के लिए GCC रैपर |
| `gcc-ranlib.exe` | ranlib के लिए GCC रैपर |

ये ज्यादातर GCC और Binutils से कंपाइलर, लिंकर और बाइनरी मैनिपुलेशन टूल्स हैं। `gpg`, `make`, `bash` आदि जैसे टूल्स के साथ एक पूर्ण यूनिक्स-जैसे वातावरण के लिए, MSYS2 इंस्टॉल करने पर विचार करें (जिसमें MinGW-w64 एक कंपोनेंट के रूप में शामिल होता है) और `pacman` के माध्यम से पैकेज जोड़ें।
