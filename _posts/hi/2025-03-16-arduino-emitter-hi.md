Arduino का उपयोग करके कॉमन-एमीटर (CE) एम्पलीफायर बनाना एक दिलचस्प प्रोजेक्ट है, लेकिन भूमिकाओं को स्पष्ट करना महत्वपूर्ण है: Arduino स्वयं एम्पलीफायर का काम नहीं करेगा (यह एक डिजिटल माइक्रोकंट्रोलर है), लेकिन यह एक बाइपोलर जंक्शन ट्रांजिस्टर (BJT) से बने CE एम्पलीफायर सर्किट के लिए सिग्नल स्रोत या मापन उपकरण के रूप में कार्य कर सकता है। एक CE एम्पलीफायर एक क्लासिक एनालॉग सर्किट है जो एक छोटे इनपुट सिग्नल को एम्प्लिफाई करने के लिए NPN या PNP ट्रांजिस्टर का उपयोग करता है, जो मध्यम वोल्टेज गेन और करंट गेन प्रदान करता है।

यहाँ, मैं आपको एक बेसिक NPN-आधारित CE एम्पलीफायर को डिजाइन करने के through गाइड करूंगा, जिसमें Arduino का उपयोग इनपुट सिग्नल जनरेट करने (PWM के माध्यम से) और आउटपुट को मापने के लिए किया जाएगा। यह सेटअप एम्प्लिफिकेशन को प्रदर्शित करेगा, साथ ही Arduino की क्षमताओं का लाभ उठाएगा।

### **कॉमन-एमीटर एम्पलीफायर का अवलोकन**
- **उद्देश्य:** एक छोटे AC सिग्नल (जैसे, ऑडियो या साइन वेव) को एम्प्लिफाई करना।
- **मुख्य विशेषताएँ:**
  - ट्रांजिस्टर सक्रिय क्षेत्र (active region) में कार्य करता है।
  - इनपुट सिग्नल बेस पर लगाया जाता है, आउटपुट कलेक्टर से लिया जाता है।
  - वोल्टेज गेन रेसिस्टर अनुपात और ट्रांजिस्टर गुणों द्वारा निर्धारित होता है।
- **घटक:**
  - NPN ट्रांजिस्टर (जैसे, 2N3904 या BC547)
  - रेसिस्टर (बायसिंग और लोड के लिए)
  - कैपेसिटर (AC सिग्नल्स को कपल करने के लिए)
  - Arduino (सिग्नल स्रोत और मापन)

### **चरण 1: सर्किट डिजाइन करें**

#### **आवश्यक घटक**
- NPN ट्रांजिस्टर (जैसे, 2N3904)
- रेसिस्टर: R1 = 47kΩ, R2 = 10kΩ (बायसिंग), RC = 1kΩ (कलेक्टर), RE = 220Ω (एमीटर)
- कैपेसिटर: C1 = 10µF (इनपुट कपलिंग), C2 = 10µF (आउटपुट कपलिंग), CE = 100µF (एमीटर बायपास, उच्च गेन के लिए वैकल्पिक)
- Arduino (जैसे, Uno)
- ब्रेडबोर्ड, जम्पर वायर
- पावर सप्लाई (Arduino का 5V पिन या बाहरी 9V, आवश्यकतानुसार समायोजित)

#### **सर्किट स्कीमैटिक**
```
Vcc (5V) ---- R1 ----+---- RC ---- कलेक्टर (C)
             47kΩ     |     1kΩ          |
                      |                  |
बेस (B) --- C1 -----+                  |
            10µF     |                  |
Arduino PWM (Pin 9)  R2                 |
                     10kΩ              आउटपुट --- C2 ---- Arduino A0 से जोड़ें
                      |                  |         10µF
                      |                  |
                      +---- RE ---- एमीटर (E) --- CE (वैकल्पिक) --- GND
                           220Ω                   100µF
                      |
                     GND
```
- **बायसिंग (R1, R2):** ट्रांजिस्टर के ऑपरेटिंग पॉइंट को सेट करता है।
- **RC:** आउटपुट सिग्नल के लिए कलेक्टर रेसिस्टर।
- **RE:** स्थिरता के लिए एमीटर रेसिस्टर।
- **C1, C2:** DC को ब्लॉक करते हैं, AC सिग्नल पास करते हैं।
- **CE (वैकल्पिक):** AC गेन बढ़ाने के लिए RE को बायपास करता है।

#### **ऑपरेटिंग पॉइंट**
- लक्ष्य: ट्रांजिस्टर को सक्रिय क्षेत्र में बायस करना (जैसे, 5V सप्लाई के लिए VCE ≈ 2.5V)।
- वोल्टेज डिवाइडर (R1, R2): \\( V_B = V_{CC} \cdot \frac{R2}{R1 + R2} = 5 \cdot \frac{10k}{47k + 10k} \approx 0.88V \\)।
- \\( V_E = V_B - V_{BE} \approx 0.88 - 0.7 = 0.18V \\)।
- \\( I_E = \frac{V_E}{RE} = \frac{0.18}{220} \approx 0.82 \, \text{mA} \\)।
- \\( V_C = V_{CC} - I_C \cdot RC \approx 5 - 0.82 \cdot 1k \approx 4.18V \\)।
- \\( V_{CE} = V_C - V_E \approx 4.18 - 0.18 = 4V \\) (5V सप्लाई के लिए अच्छा है)।

### **चरण 2: Arduino को सिग्नल स्रोत के रूप में उपयोग करें**

#### **Arduino की भूमिका**
- पिन 9 (जो PWM को सपोर्ट करता है) जैसे पिन पर PWM (पल्स विड्थ मॉड्यूलेशन) का उपयोग करके एक छोटा AC सिग्नल जनरेट करें।
- PWM को एक साधारण RC लो-पास फिल्टर (वैकल्पिक) का उपयोग करके एक साइन वेव में बदलें।

#### **सिग्नल जनरेट करने के लिए कोड**
```cpp
const int pwmPin = 9; // PWM आउटपुट पिन

void setup() {
  pinMode(pwmPin, OUTPUT);
  // PWM फ्रीक्वेंसी सेट करें (वैकल्पिक, डिफॉल्ट ~490 Hz है)
}

void loop() {
  // PWM के साथ साइन वेव सिम्युलेट करें (0-255 रेंज)
  for (int i = 0; i < 360; i += 10) {
    float sineValue = sin(radians(i)); // -1 से 1 तक साइन वेव
    int pwmValue = 127 + 127 * sineValue; // 0-255 में स्केल करें
    analogWrite(pwmPin, pwmValue);
    delay(10); // फ्रीक्वेंसी के लिए समायोजित करें (जैसे, यहाँ ~100 Hz)
  }
}
```
- **आउटपुट:** ~0–5V PWM सिग्नल, 2.5V पर केंद्रित, ~2.5V पीक-टू-पीक।
- **C1:** DC ऑफसेट को हटाता है, केवल AC कंपोनेंट (~1.25V पीक) को बेस तक पहुँचाता है।

#### **वैकल्पिक फिल्टर**
पिन 9 से GND तक श्रृंखला में एक 1kΩ रेसिस्टर और 0.1µF कैपेसिटर जोड़ें, सिग्नल को C1 से पहले टैप करें, ताकि PWM को एक मोटा साइन वेव में बदला जा सके।

### **चरण 3: आउटपुट मापें**

#### **Arduino मापन**
- एम्पलीफायर आउटपुट (C2 के बाद) को A0 से कनेक्ट करें।
- आउटपुट सिग्नल को पढ़ने और Serial Monitor के माध्यम से प्रदर्शित करने के लिए Arduino का उपयोग करें।

#### **मापने और प्रदर्शित करने के लिए कोड**
```cpp
const int inputPin = A0; // आउटपुट यहाँ मापें

void setup() {
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(inputPin); // 0-1023, 0-5V पर मैप होता है
  float voltage = sensorValue * (5.0 / 1023.0);
  Serial.print("आउटपुट वोल्टेज (V): ");
  Serial.println(voltage);
  delay(100); // सैंपलिंग दर समायोजित करें
}
```

#### **अपेक्षित गेन**
- वोल्टेज गेन \\( A_v = -\frac{RC}{RE} = -\frac{1k}{220} \approx -4.5 \\) (फेज इनवर्जन के कारण नेगेटिव)।
- इनपुट: ~1.25V पीक (कपलिंग के बाद)।
- आउटपुट: ~4.5 × 1.25 = 5.625V पीक (लेकिन सप्लाई लिमिट के कारण 5V पर क्लिप हो जाएगा)।

### **चरण 4: निर्माण और परीक्षण**

#### **असेंबली**
1. सर्किट को ब्रेडबोर्ड पर स्कीमैटिक के अनुसार वायर करें।
2. सिग्नल जनरेशन कोड को Arduino पर अपलोड करें और पिन 9 को C1 से कनेक्ट करें।
3. सर्किट को Arduino के 5V (या समायोजित रेसिस्टर के साथ 9V) से पावर दें।
4. मापन कोड अपलोड करें और Serial Monitor (9600 baud) खोलें।

#### **सत्यापन**
- **इनपुट सिग्नल:** C1 के बाद ~1–2V पीक-टू-पीक (यदि उपलब्ध हो तो मल्टीमीटर या ऑसिलोस्कोप का उपयोग करें)।
- **आउटपुट सिग्नल:** बड़ा (जैसे, 4–5V पीक-टू-पीक) और इनवर्टेड होना चाहिए।
- सक्रिय क्षेत्र में ऑपरेशन सुनिश्चित करने के लिए VCE (~2–4V) को मल्टीमीटर से चेक करें।

### **सुझाव और समस्या निवारण**
- **गेन बहुत कम:** AC गेन बढ़ाने के लिए RE के पार CE (100µF) जोड़ें (यह RE के डीजनरेशन को हटा देता है)।
- **क्लिपिंग:** यदि आउटपुट 5V से अधिक हो जाता है, तो इनपुट आयाम कम करें (PWM रेंज समायोजित करें) या Vcc बढ़ाएँ (जैसे, 9V बाहरी सप्लाई)।
- **नॉइज़:** PWM वर्गाकार जैसा होता है; एक बेहतर साइन वेव (बाहरी जनरेटर या फिल्टर के माध्यम से) परिणामों में सुधार करता है।
- **ट्रांजिस्टर स्पेक्स:** सुनिश्चित करें कि 2N3904 करंट (Ic < 200mA, यहाँ सुरक्षित) को हैंडल कर सकता है।

यह सेटअप एक CE एम्पलीफायर के मूल व्यवहार - वोल्टेज एम्प्लिफिकेशन - को प्रदर्शित करता है, साथ ही सिग्नल जनरेशन और विश्लेषण के लिए Arduino को एकीकृत करता है। क्या आप गेन को ट्वीक करना, ऑडियो के लिए स्पीकर जोड़ना, या सिग्नल को परिष्कृत करना चाहते हैं? मुझे बताएं!
