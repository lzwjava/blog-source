### Google Imagen 4 Preview 0606 का उपयोग कैसे करें

1.  **Vertex AI तक पहुंचें**:
    *   **प्लेटफॉर्म**: Imagen 4 Preview 0606 Google Cloud के Vertex AI पर उपलब्ध है। शुरुआत करने के लिए आपको एक Google Cloud अकाउंट की आवश्यकता होगी।
    *   **साइन अप**: यदि आपके पास अकाउंट नहीं है, तो [cloud.google.com](https://cloud.google.com) पर साइन अप करें और एक प्रोजेक्ट सेट अप करें। बिलिंग सक्षम करना सुनिश्चित करें, क्योंकि यह एक पूर्वावलोकन ऑफरिंग है जिसमें संभावित लागत हो सकती है (मूल्य निर्धारण विवरण Vertex AI मूल्य निर्धारण पेज के Imagen सेक्शन में उपलब्ध है)।
    *   **Vertex AI पर नेविगेट करें**: एक बार लॉग इन करने के बाद, Google Cloud Console के Vertex AI सेक्शन में जाएं और Generative AI टूल्स को ढूंढें।

2.  **एनवायरनमेंट सेट अप करें**:
    *   **प्रमाणीकरण**: Google Cloud क्रेडेंशियल्स का उपयोग करके अपने अकाउंट को प्रमाणित करें। आप इस कमांड का उपयोग करके एक एक्सेस टोकन जनरेट कर सकते हैं:
        ```bash
        gcloud auth print-access-token
        ```
    *   **प्रोजेक्ट और लोकेशन**: अपना Google Cloud प्रोजेक्ट ID और लोकेशन (जैसे, `us-central1`) सेट करें। उदाहरण:
        ```bash
        export GOOGLE_CLOUD_PROJECT=your-project-id
        export GOOGLE_CLOUD_LOCATION=us-central1
        ```

3.  **Imagen 4 मॉडल का उपयोग करें**:
    *   **API एक्सेस**: Imagen 4 Preview 0606 को Vertex AI API के माध्यम से एक्सेस किया जा सकता है। मॉडल एंडपॉइंट `imagen-4.0-generate-preview-06-06` का उपयोग करें। आप cURL या Google Gen AI SDK for Python जैसे टूल्स का उपयोग करके प्रोग्रामेटिक रूप से इसके साथ इंटरैक्ट कर सकते हैं।
    *   **cURL रिक्वेस्ट उदाहरण**:
        ```bash
        curl -X POST \
        -H "Authorization: Bearer $(gcloud auth print-access-token)" \
        -H "Content-Type: application/json; charset=utf-8" \
        "https://${GOOGLE_CLOUD_LOCATION}-aiplatform.googleapis.com/v1/projects/${GOOGLE_CLOUD_PROJECT}/locations/${GOOGLE_CLOUD_LOCATION}/publishers/google/models/imagen-4.0-generate-preview-06-06:predict" \
        -d '{"instances": [{"prompt": "A cat reading a book"}], "parameters": {"sampleCount": 1}}'
        ```
        यह एक base64-एनकोडेड इमेज लौटाता है।
    *   **Python SDK उदाहरण**:
        ```python
        from google import genai
        from google.genai.types import GenerateImagesConfig
        client = genai.Client()
        image = client.models.generate_images(
            model="imagen-4.0-generate-preview-06-06",
            prompt="A dog reading a newspaper",
            config=GenerateImagesConfig(image_size="2K")
        )
        image.generated_images[0].image.save("output-image.png")
        print(f"Created output image using {len(image.generated_images[0].image.image_bytes)} bytes")
        ```
        यह एक इमेज जनरेट करता है और इसे PNG फ़ाइल के रूप में सेव करता है।

4.  **प्रभावी प्रॉम्प्ट्स तैयार करें**:
    *   **प्रॉम्प्ट स्ट्रक्चर**: बेहतरीन परिणामों के लिए, विस्तृत और विशिष्ट प्रॉम्प्ट्स का उपयोग करें। विषय, वातावरण, कलात्मक शैली (जैसे, फोटोरियलिस्टिक, एब्स्ट्रैक्ट), मूड और कम्पोज़िशनल एलिमेंट्स (जैसे, कैमरा एंगल) शामिल करें। उदाहरण: "सूर्यास्त के समय एक भविष्यवादी शहर की जीवंत ग्राफिक चित्रण, साइबरपंक शैली, नीयन लाइट्स और नाटकीय लो-एंगल परिप्रेक्ष्य के साथ।"
    *   **टिप्स**:
        *   सटीक रहें: अस्पष्ट प्रॉम्प्ट्स के परिणामस्वरूप अप्रत्याशित परिणाम आ सकते हैं।
        *   निरर्थक इनपुट (जैसे, रैंडम इमोजी) से बचें, क्योंकि ये असंगत आउटपुट उत्पन्न कर सकते हैं।
        *   आवश्यकता पड़ने पर टेक्स्ट निर्दिष्ट करें, क्योंकि Imagen 4 में टाइपोग्राफी रेंडरिंग में सुधार हुआ है।
    *   **नेगेटिव प्रॉम्प्ट्स**: आप अनचाहे एलिमेंट्स को बाहर करने के लिए एक नेगेटिव प्रॉम्प्ट का उपयोग कर सकते हैं (जैसे, यदि आप इमेज में टेक्स्ट नहीं चाहते हैं तो "no text")।

5.  **वेरिएंट एक्सप्लोर करें**:
    *   Imagen 4 Preview 0606 के वेरिएंट हैं जैसे **Fast** (10x तक तेज़, बल्क जनरेशन के लिए ऑप्टिमाइज़्ड) और **Ultra** (पेशेवर उपयोग के लिए प्रॉम्प्ट्स के साथ उच्च संरेखण)। जांचें कि क्या ये आपके Vertex AI इंटरफेस में उपलब्ध हैं और अपनी आवश्यकताओं के आधार पर चुनें (जैसे, त्वरित प्रोटोटाइपिंग के लिए Fast, उच्च-गुणवत्ता आउटपुट के लिए Ultra)।

7.  **वैकल्पिक प्लेटफॉर्म**:
    *   Imagen 4 तीसरे पक्ष के प्लेटफॉर्म जैसे Replicate, fal.ai, या AI/ML API पर भी उपलब्ध है, जो टेस्टिंग के लिए सरल इंटरफेस या सैंडबॉक्स एनवायरनमेंट प्रदान कर सकते हैं। उदाहरण के लिए:
        *   **Replicate**: "सूर्यास्त के समय एक शांत पर्वतीय परिदृश्य, हाइपररियलिस्टिक शैली" जैसे प्रॉम्प्ट के साथ Imagen 4 चलाएं। API कुंजी और उपयोग के लिए Replicate के डॉक्यूमेंटेशन की जांच करें।
        *   **fal.ai**: उनकी API का उपयोग इस तरह एक रिक्वेस्ट के साथ करें:
            ```javascript
            const result = await fal.subscribe("fal-ai/imagen4/preview", {
                input: { prompt: "A serene mountain landscape at sunset, hyperrealistic style" }
            });
            console.log(result.images[0].url);
            ```
            मूल्य निर्धारण भिन्न होता है (जैसे, Standard के लिए $0.05/इमेज, Fast के लिए $0.04, Ultra के लिए $0.06)।
    *   **Gemini ऐप या Google Workspace**: Imagen 4 को वर्कफ़्लो में सीधे इमेज जनरेशन के लिए Gemini ऐप, Google Slides, Docs, और Vids में इंटीग्रेट किया गया है। उदाहरण के लिए, Google Docs में, आप दस्तावेज़ों को चित्रित करने के लिए इमेज जनरेट कर सकते हैं।
