### Ubuntu 22.04 LTS पर Rust इंस्टॉल करना

Rust को इंस्टॉल करने का सबसे अच्छा तरीका `rustup` का उपयोग करना है, जो कि आधिकारिक टूलचेन इंस्टॉलर है। यह विधि सीधी-साधी है, आसान अपडेट्स को सपोर्ट करती है, और आपके 64-बिट Ubuntu 22.04 सिस्टम पर काम करती है। आमतौर पर एक बेसिक Ubuntu सेटअप के अलावा किसी अतिरिक्त डिपेंडेंसी की आवश्यकता नहीं होती है।

#### चरण-दर-चरण निर्देश

1. **अपनी पैकेज सूची अपडेट करें** (वैकल्पिक लेकिन अनुशंसित):
   ```
   sudo apt update
   ```

2. **rustup का उपयोग करके Rust इंस्टॉल करें**:
   इंस्टॉलर स्क्रिप्ट को डाउनलोड और एक्ज़िक्यूट करने के लिए निम्नलिखित कमांड चलाएँ:
   ```
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```
   - यह स्क्रिप्ट को HTTPS के माध्यम से सुरक्षित रूप से डाउनलोड करेगा।
   - जब संकेत दिया जाए, तो डिफ़ॉल्ट इंस्टॉलेशन विकल्प चुनें (`1` दबाएँ और एंटर करें)।
   - इंस्टॉलर Rust बाइनरीज़ को `~/.cargo/bin` में जोड़ देगा और आपके शेल प्रोफाइल (जैसे `~/.bashrc` या `~/.profile`) को अपडेट कर देगा।

3. **अपनी शेल कॉन्फ़िगरेशन रीलोड करें**:
   बिना टर्मिनल रीस्टार्ट किए तुरंत परिवर्तनों को प्रभावी बनाने के लिए:
   ```
   source ~/.cargo/env
   ```
   वैकल्पिक रूप से, अपना टर्मिनल बंद करें और फिर से खोलें।

4. **इंस्टॉलेशन सत्यापित करें**:
   जाँचें कि Rust सही ढंग से इंस्टॉल हो गया है:
   ```
   rustc --version
   cargo --version
   ```
   आपको `rustc 1.82.0` (या नवीनतम स्टेबल वर्जन) और Cargo के लिए इसी तरह का आउटपुट दिखाई देना चाहिए।

#### अतिरिक्त नोट्स
- **अनइंस्टॉल करना**: यदि आवश्यक हो, तो `rustup self uninstall` चलाएँ।
- **Rust अपडेट करना**: नवीनतम वर्जन प्राप्त करने के लिए `rustup update` का उपयोग करें।
- **IDE/सपोर्ट**: डेवलपमेंट के लिए, बेहतर कोड कम्प्लीशन के लिए अपने एडिटर (जैसे VS Code एक्सटेंशन) के माध्यम से `rust-analyzer` इंस्टॉल करने पर विचार करें।
- यह डिफ़ॉल्ट रूप से स्टेबल चैनल इंस्टॉल करता है। नाइटली या बीटा के लिए, सेटअप के बाद `rustup install nightly` का उपयोग करें।

यदि आपको कोई त्रुटि मिलती है (जैसे curl नहीं मिला), तो इसे `sudo apt install curl` से इंस्टॉल करें।
