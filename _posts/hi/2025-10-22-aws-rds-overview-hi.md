### AWS पर रिलेशनल डेटाबेस

AWS रिलेशनल डेटाबेस के लिए कई मैनेज्ड सेवाएं प्रदान करता है, जिसमें **Amazon Relational Database Service (RDS)** प्राथमिक ऑफरिंग है। यह विभिन्न लोकप्रिय डेटाबेस इंजन के लिए सेटअप, पैचिंग, बैकअप और स्केलिंग को संभालता है। इसके अतिरिक्त, **Amazon Aurora** एक हाई-परफॉर्मेंस, MySQL- और PostgreSQL-संगत रिलेशनल डेटाबेस है जो RDS के अंदर बनाया गया है।

AWS पर प्रमुख रिलेशनल डेटाबेस विकल्पों का सारांश यहाँ दिया गया है:

| सर्विस/इंजन | विवरण | उपयोग के मामले |
|---------------|-------------|-----------|
| **Amazon RDS for PostgreSQL** | पूरी तरह से मैनेज्ड PostgreSQL, जो JSONB और फुल-टेक्स्ट सर्च जैसी उन्नत सुविधाओं का समर्थन करता है। | लेन-देन वाले एप्लिकेशन, वेब ऐप्स और एनालिटिक्स। |
| **Amazon RDS for MySQL** | हाई अवेलेबिलिटी और रीड रेप्लिकास के साथ मैनेज्ड MySQL। | ई-कॉमर्स, कंटेंट मैनेजमेंट और सामान्य-उद्देश्य वाले ऐप्स। |
| **Amazon RDS for MariaDB** | बेहतर परफॉर्मेंस और स्टोरेज इंजन के साथ MySQL का फोर्क। | वेब एप्लिकेशन और रीड-हैवी वर्कलोड। |
| **Amazon RDS for SQL Server** | स्टैंडर्ड, एंटरप्राइज और वेब जैसे एडिशन के साथ Microsoft SQL Server। | एंटरप्राइज ऐप्स, BI और Windows-आधारित सिस्टम। |
| **Amazon RDS for Oracle** | BYOL (Bring Your Own License) या लाइसेंस-शामिल विकल्पों के साथ Oracle डेटाबेस। | मिशन-क्रिटिकल एंटरप्राइज एप्लिकेशन। |
| **Amazon RDS for Db2** | LUW (Linux, UNIX, Windows) के लिए सपोर्ट के साथ IBM Db2। | बड़े पैमाने के लेन-देन और विश्लेषणात्मक वर्कलोड। |
| **Amazon Aurora (MySQL-compatible)** | स्टैंडर्ड MySQL की तुलना में 5 गुना तेज परफॉर्मेंस वाला क्लाउड-नेटिव रिलेशनल DB। | हाई-थ्रूपुट OLTP और ग्लोबल ऐप्स। |
| **Amazon Aurora (PostgreSQL-compatible)** | सर्वरलेस विकल्पों और ऑटो-स्केलिंग स्टोरेज के साथ PostgreSQL-संगत। | स्केलेबल ऐप्स जिनमें ACID कंप्लायंस और हाई अवेलेबिलिटी की आवश्यकता होती है। |

डेटा वेयरहाउसिंग (OLAP-केंद्रित रिलेशनल वर्कलोड) के लिए, AWS **Amazon Redshift** भी प्रदान करता है, जो PostgreSQL पर आधारित एक पूरी तरह से मैनेज्ड, पेटाबाइट-स्केल डेटा वेयरहाउस है।

ये सेवाएं अन्य AWS टूल्स जैसे EC2, Lambda और S3 के साथ निर्बाध रूप से एकीकृत होती हैं। आरंभ करने के लिए, [AWS RDS डॉक्यूमेंटेशन](https://aws.amazon.com/rds/) या [AWS डेटाबेस सेवाएं अवलोकन](https://docs.aws.amazon.com/databases-on-aws-how-to-choose/) देखें।
