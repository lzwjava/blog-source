### मैकऑस पर सिस्टम इनफोर्मेशन प्राप्त करने के लिए कुछ आइडीयल टूल्स और कमांड्स उपलब्ध हैं। यहां कुछ विभिन्न प्रकार की सिस्टम इनफोर्मेशन प्राप्त करने के तरीके हैं:

### 1. **सिस्टम इनफोर्मेशन यूटिलिटी**
सिस्टम इनफोर्मेशन एप्लिकेशन आपके मैक की हार्डवेयर और सॉफ्टवेयर के बारे में एक पूर्णांक ओवरव्यू प्रदान करती है।

- **सपोटलाइट सर्च** को खोलें द्वारा `Cmd + Space` दबाएँ और "System Information" लिखें, फिर Enter दबाएँ।
- या तो आप इसे **एप्पल मेनु** > **About This Mac** > **System Report** से खोल सकते हैं।

### 2. **About This Mac**
यह आपके मैक की स्पेसिफिकेशन्स के बारे में एक त्वरित ओवरव्यू प्रदान करता है।

- **एप्पल मेनु** को स्क्रीन के शीर्ष बायें कोर्नर में क्लिक करें।
- **About This Mac** का चयन करें। यह बसिक जानकारी जैसे कि macOS वर्जन, प्रोसेसर, मेमोरी, और सीरियल नंबर दिखाएगा।

### 3. **टर्मिनल कमांड्स**
आप टर्मिनल का उपयोग कर विभिन्न कमांड्स के साथ विविध सिस्टम इनफोर्मेशन प्राप्त कर सकते हैं।

- **टर्मिनल** को `Applications` > `Utilities` > `Terminal` से खोलें या सपोटलाइट में खोजें।

#### बेसिक सिस्टम इनफोर्मेशन
```sh
system_profiler SPHardwareDataType
```

#### विविध सिस्टम इनफोर्मेशन
```sh
system_profiler
```

#### सिस्टम प्रोसेसर इनफोर्मेशन
```sh
sysctl -n machdep.cpu.brand_string
```

#### मेमोरी इनफोर्मेशन
```sh
sysctl hw.memsize
```

#### डिस्क उपयोग
```sh
df -h
```

#### नेटवर्क इनफोर्मेशन
```sh
ifconfig
```

#### बैटरी इनफोर्मेशन (लैपटॉप्स के लिए)
```sh
pmset -g batt
```

#### इन्स्टॉल्ड सॉफ्टवेयर की सूची
```sh
ls /Applications
```

#### macOS वर्जन
```sh
sw_vers
```

#### यूजर इनफोर्मेशन
```sh
id
```

#### चल रहे प्रोसेसेज की सूची
```sh
ps aux
```

#### इन्स्टॉल्ड ब्रू पैकेजों की सूची
```sh
brew list
```

#### इन्स्टॉल्ड पिप पैकेजों की सूची
```sh
pip list
```

#### इन्स्टॉल्ड जम पैकेजों की सूची
```sh
gem list
```

### 4. **एक्टिविटी मॉनिटर**
यह एप्लिकेशन आपके मैक की संरचना, मेमोरी, एनर्जी, डिस्क, और नेटवर्क उपयोग के बारे में रियल-टाइम जानकारी प्रदान करती है।

- **एक्टिविटी मॉनिटर** को `Applications` > `Utilities` > `Activity Monitor` से खोलें या सपोटलाइट में खोजें।

### 5. **डिस्क यूटिलिटी**
यह एप्लिकेशन आपकी स्टोरेज डिवाइस के बारे में जानकारी प्रदान करती है।

- **डिस्क यूटिलिटी** को `Applications` > `Utilities` > `Disk Utility` से खोलें या सपोटलाइट में खोजें।

### 6. **टर्ड पार्टी टूल्स**
इसके अलावा, **iStat Menus** और **CoconutBattery** जैसी टर्ड पार्टी एप्लिकेशन्स भी विभिन्न सिस्टम इनफोर्मेशन और मॉनिटरिंग क्षमताएँ प्रदान करती हैं।

इन तरीकों का उपयोग करके, आप अपने मैकऑस पर विविध सिस्टम इनफोर्मेशन प्राप्त कर सकते हैं।
