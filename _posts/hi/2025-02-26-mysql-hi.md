यहाँ MySQL, एक लोकप्रिय ओपन-सोर्स रिलेशनल डेटाबेस मैनेजमेंट सिस्टम, का उपयोग करने के लिए चरण-दर-चरण मार्गदर्शिका दी गई है:

---

### 1. **इंस्टालेशन**
- **Windows**: [mysql.com/downloads](https://dev.mysql.com/downloads/installer/) से MySQL इंस्टॉलर डाउनलोड करें
- **macOS**:
  ```bash
  brew install mysql
  brew services start mysql
  ```
- **Linux** (Debian/Ubuntu):
  ```bash
  sudo apt update
  sudo apt install mysql-server mysql-client
  sudo systemctl start mysql
  sudo mysql_secure_installation  # सुरक्षा सेटअप चलाएँ
  ```

---

### 2. **MySQL से कनेक्ट करें**
- **डिफ़ॉल्ट रूट लॉगिन**:
  ```bash
  mysql -u root -p  # संकेत मिलने पर पासवर्ड दर्ज करें
  ```
- **रिमोट सर्वर से कनेक्ट करें**:
  ```bash
  mysql -u username -p -h hostname -P 3306
  ```

---

### 3. **बेसिक डेटाबेस ऑपरेशन्स**
- **यूज़र बनाएँ**:
  ```sql
  CREATE USER 'myuser'@'localhost' IDENTIFIED BY 'mypassword';
  ```
- **डेटाबेस बनाएँ**:
  ```sql
  CREATE DATABASE mydb;
  GRANT ALL PRIVILEGES ON mydb.* TO 'myuser'@'localhost';
  FLUSH PRIVILEGES;
  ```
- **बेसिक SQL कमांड्स**:
  ```sql
  -- टेबल बनाएँ
  CREATE TABLE users (
      id INT AUTO_INCREMENT PRIMARY KEY,
      name VARCHAR(50),
      email VARCHAR(100) UNIQUE
  );

  -- डेटा डालें
  INSERT INTO users (name, email) VALUES ('Bob', 'bob@example.com');

  -- डेटा क्वेरी करें
  SELECT * FROM users WHERE name LIKE 'B%';

  -- डेटा अपडेट करें
  UPDATE users SET email = 'new@email.com' WHERE id = 1;

  -- डेटा डिलीट करें
  DELETE FROM users WHERE id = 2;
  ```

---

### 4. **एडवांस्ड फीचर्स**
- **जॉइन्स**:
  ```sql
  SELECT orders.id, users.name 
  FROM orders
  JOIN users ON orders.user_id = users.id;
  ```
- **स्टोर्ड प्रोसीजर्स**:
  ```sql
  DELIMITER //
  CREATE PROCEDURE GetUsers()
  BEGIN
    SELECT * FROM users;
  END //
  DELIMITER ;
  
  CALL GetUsers();
  ```
- **JSON सपोर्ट** (MySQL 5.7+):
  ```sql
  CREATE TABLE products (
      id INT PRIMARY KEY,
      details JSON
  );
  ```

---

### 5. **मेंटेनेंस**
- **बैकअप**:
  ```bash
  mysqldump -u root -p mydb > backup.sql
  ```
- **रिस्टोर**:
  ```bash
  mysql -u root -p mydb < backup.sql
  ```
- **टेबल्स ऑप्टिमाइज़ करें**:
  ```sql
  ANALYZE TABLE users;
  OPTIMIZE TABLE users;
  ```

---

### 6. **सिक्योरिटी**
- **रीड-ओनली यूज़र बनाएँ**:
  ```sql
  CREATE USER 'reader'@'localhost' IDENTIFIED BY 'readpass';
  GRANT SELECT ON mydb.* TO 'reader'@'localhost';
  FLUSH PRIVILEGES;
  ```

---

### 7. **उपयोगी MySQL कमांड्स**
- डेटाबेस लिस्ट करें: `SHOW DATABASES;`
- डेटाबेस यूज़ करें: `USE mydb;`
- टेबल्स लिस्ट करें: `SHOW TABLES;`
- टेबल का विवरण देखें: `DESCRIBE users;`
- यूज़र्स दिखाएँ: `SELECT user FROM mysql.user;`
- MySQL से बाहर निकलें: `EXIT;` या `QUIT;`

**PostgreSQL से मुख्य अंतर**:
1. `SERIAL` के बजाय `AUTO_INCREMENT` का उपयोग करता है
2. डिफ़ॉल्ट रूप से केस-इनसेंसिटिव टेबल नाम
3. `pg_dump` के बजाय `mysqldump` का उपयोग करता है
4. डिफ़ॉल्ट पोर्ट: **3306** (बनाम PostgreSQL का 5432)
5. अलग ऑथेंटिकेशन सिस्टम और यूज़र मैनेजमेंट

**टिप**: बड़े बदलावों से पहले हमेशा डेटाबेस का बैकअप लें। महत्वपूर्ण ऑपरेशन्स के लिए ट्रांजैक्शन (`START TRANSACTION`/`COMMIT`) का उपयोग करें।
