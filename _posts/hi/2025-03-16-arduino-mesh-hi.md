Arduino के साथ मेश एनालिसिस को सत्यापित करने में कई लूप वाला एक सर्किट बनाना, मेश एनालिसिस को सैद्धांतिक रूप से लागू करके करंट्स का अनुमान लगाना, और फिर वास्तविक करंट्स की गणना करने के लिए रेसिस्टर्स के पार वोल्टेज को मापने के लिए Arduino का उपयोग करना शामिल है। मेश एनालिसिस सर्किट थ्योरी की एक विधि है जहाँ आप प्रत्येक स्वतंत्र लूप को लूप करंट्स निर्दिष्ट करते हैं और समीकरण स्थापित करने के लिए किरचॉफ का वोल्टेज लॉ (KVL) का उपयोग करते हैं। Arduino आपके गणनाओं की पुष्टि करने के लिए वास्तविक दुनिया के व्यवहार को मापने में आपकी सहायता करेगा।

नीचे, मैं आपको एक सरल दो-मेश सर्किट को डिजाइन करने, मैन्युअल रूप से मेश एनालिसिस करने, और वोल्टेज मापकर करंट्स का अनुमान लगाने (चूंकि Arduino सीधे करंट को माप नहीं सकता, हम ओम के लॉ: \\( I = V/R \\) का उपयोग करेंगे) के लिए Arduino का उपयोग करके परिणामों को सत्यापित करने के माध्यम से मार्गदर्शन करूंगा।

### **चरण 1: एक दो-मेश सर्किट डिजाइन करें**

#### **सर्किट विवरण**
- **घटक:**
  - Arduino (जैसे, Uno)
  - 3 रेसिस्टर्स (जैसे, R1 = 330Ω, R2 = 470Ω, R3 = 680Ω)
  - ब्रेडबोर्ड और जम्पर वायर
  - पावर स्रोत (Arduino का 5V पिन)
- **वायरिंग:**
  - 5V को Node A से कनेक्ट करें।
  - Node A से, R1 को Node B से कनेक्ट करें।
  - Node B से, R2 को Node C (GND) से कनेक्ट करें।
  - Node A से, R3 को Node C (GND) से कनेक्ट करें।
- **टोपोलॉजी:**
  - मेश 1: 5V → R1 → R2 → GND (बायां लूप)।
  - मेश 2: 5V → R3 → GND (दायां लूप)।
  - R1 केवल मेश 1 में है, R3 केवल मेश 2 में है, और R2 दोनों मेश के बीच साझा है।
- **मापन बिंदु:**
  - A0: R1 के पार वोल्टेज (Node A से Node B)।
  - A1: R2 के पार वोल्टेज (Node B से Node C)।
  - A2: R3 के पार वोल्टेज (Node A से Node C)।

#### **स्कीमैटिक अवधारणा**
```
5V ---- Node A ---- R1 ---- Node B ---- R2 ---- Node C (GND)
       |                        |
       +--------- R3 -----------+
```

### **चरण 2: मेश एनालिसिस को सैद्धांतिक रूप से करें**

#### **मेश करंट्स को परिभाषित करें**
- \\( I_1 \\): मेश 1 में करंट (5V, R1, R2, GND के माध्यम से दक्षिणावर्त)।
- \\( I_2 \\): मेश 2 में करंट (5V, R3, GND के माध्यम से दक्षिणावर्त)।

#### **प्रत्येक मेश पर KVL लागू करें**
1. **मेश 1 (5V → R1 → R2 → GND):**
   - वोल्टेज स्रोत: +5V (लूप की दिशा में GND से 5V की ओर जाते हुए)।
   - R1 के पार वोल्टेज ड्रॉप: \\( -R1 \cdot I_1 \\)।
   - R2 के पार वोल्टेज ड्रॉप: \\( -R2 \cdot (I_1 - I_2) \\) (R2 के माध्यम से करंट \\( I_1 - I_2 \\) है)।
   - समीकरण: \\( 5 - R1 \cdot I_1 - R2 \cdot (I_1 - I_2) = 0 \\)।

2. **मेश 2 (5V → R3 → GND):**
   - वोल्टेज स्रोत: +5V।
   - R3 के पार वोल्टेज ड्रॉप: \\( -R3 \cdot I_2 \\)।
   - R2 के पार वोल्टेज ड्रॉप (विपरीत दिशा): \\( +R2 \cdot (I_1 - I_2) \\) (R2 के माध्यम से करंट \\( I_1 - I_2 \\) है)।
   - समीकरण: \\( 5 - R3 \cdot I_2 + R2 \cdot (I_1 - I_2) = 0 \\)।

#### **मानों को प्रतिस्थापित करें**
- R1 = 330Ω, R2 = 470Ω, R3 = 680Ω।
- मेश 1: \\( 5 - 330 I_1 - 470 (I_1 - I_2) = 0 \\)
  - सरल करें: \\( 5 - 330 I_1 - 470 I_1 + 470 I_2 = 0 \\)
  - \\( 5 - 800 I_1 + 470 I_2 = 0 \\) → (1)
- मेश 2: \\( 5 - 680 I_2 + 470 (I_1 - I_2) = 0 \\)
  - सरल करें: \\( 5 + 470 I_1 - 680 I_2 - 470 I_2 = 0 \\)
  - \\( 5 + 470 I_1 - 1150 I_2 = 0 \\) → (2)

#### **समीकरणों को हल करें**
- (1) से: \\( 5 = 800 I_1 - 470 I_2 \\) → \\( I_1 = \frac{5 + 470 I_2}{800} \\)।
- (2) में प्रतिस्थापित करें: \\( 5 + 470 \left( \frac{5 + 470 I_2}{800} \right) - 1150 I_2 = 0 \\)।
- भिन्न को हटाने के लिए 800 से गुणा करें:
  - \\( 4000 + 470 (5 + 470 I_2) - 1150 \cdot 800 I_2 = 0 \\)
  - \\( 4000 + 2350 + 220900 I_2 - 920000 I_2 = 0 \\)
  - \\( 6350 - 699100 I_2 = 0 \\)
  - \\( I_2 = \frac{6350}{699100} \approx 0.00908 \, \text{A} = 9.08 \, \text{mA} \\)।
- वापस प्रतिस्थापित करें: \\( I_1 = \frac{5 + 470 \cdot 0.00908}{800} = \frac{5 + 4.2676}{800} \approx 0.01158 \, \text{A} = 11.58 \, \text{mA} \\)।

#### **वोल्टेज की गणना करें**
- \\( V_{R1} = R1 \cdot I_1 = 330 \cdot 0.01158 \approx 3.82 \, \text{V} \\)।
- \\( V_{R2} = R2 \cdot (I_1 - I_2) = 470 \cdot (0.01158 - 0.00908) \approx 1.18 \, \text{V} \\)।
- \\( V_{R3} = R3 \cdot I_2 = 680 \cdot 0.00908 \approx 6.17 \, \text{V} \\) (लेकिन स्रोत के कारण 5V पर सीमित)।

### **चरण 3: Arduino के साथ सत्यापित करें**

#### **Arduino कोड**
```cpp
void setup() {
  Serial.begin(9600); // सीरियल कम्युनिकेशन शुरू करें
}

void loop() {
  // वोल्टेज पढ़ें (0-1023, 0-5V पर मैप होता है)
  int sensorValueR1 = analogRead(A0); // R1 के पार
  int sensorValueR2 = analogRead(A1); // R2 के पार
  int sensorValueR3 = analogRead(A2); // R3 के पार

  // वोल्टेज में बदलें
  float VR1 = sensorValueR1 * (5.0 / 1023.0);
  float VR2 = sensorValueR2 * (5.0 / 1023.0);
  float VR3 = sensorValueR3 * (5.0 / 1023.0);

  // रेसिस्टर मान
  float R1 = 330.0;
  float R2 = 470.0;
  float R3 = 680.0;

  // करंट्स की गणना करें (I = V/R)
  float I1 = VR1 / R1;              // R1 के माध्यम से मेश 1 करंट
  float I2 = VR3 / R3;              // R3 के माध्यम से मेश 2 करंट
  float IR2 = VR2 / R2;             // R2 के माध्यम से करंट (I1 - I2)

  // परिणाम आउटपुट करें
  Serial.println("मापे गए मान:");
  Serial.print("VR1 (V): "); Serial.println(VR1);
  Serial.print("VR2 (V): "); Serial.println(VR2);
  Serial.print("VR3 (V): "); Serial.println(VR3);
  Serial.print("I1 (mA): "); Serial.println(I1 * 1000);
  Serial.print("I2 (mA): "); Serial.println(I2 * 1000);
  Serial.print("I1 - I2 (mA): "); Serial.println((I1 - I2) * 1000);
  Serial.println("---");

  delay(2000); // 2 सेकंड प्रतीक्षा करें
}
```

#### **वायरिंग नोट्स**
- A0 को Node A (5V) और Node B के बीच कनेक्ट करें।
- A1 को Node B और Node C (GND) के बीच कनेक्ट करें।
- A2 को Node A (5V) और Node C (GND) के बीच कनेक्ट करें।
- सुनिश्चित करें कि सभी ग्राउंड Arduino GND के साथ कॉमन हैं।

### **चरण 4: परिणामों की तुलना करें**

#### **अपेक्षित बनाम मापित**
- **सैद्धांतिक (मेश एनालिसिस से):**
  - \\( I_1 \approx 11.58 \, \text{mA} \\)
  - \\( I_2 \approx 9.08 \, \text{mA} \\)
  - \\( I_1 - I_2 \approx 2.50 \, \text{mA} \\)
  - \\( V_{R1} \approx 3.82 \, \text{V} \\), \\( V_{R2} \approx 1.18 \, \text{V} \\), \\( V_{R3} \approx 5 \, \text{V} \\) (सीमित)।
- **मापित:** सीरियल मॉनिटर (9600 बॉड) खोलें और Arduino के आउटपुट की इन मानों से तुलना करें।

#### **सत्यापन**
- जांचें कि क्या \\( V_{R1} + V_{R2} \approx 5V \\) (मेश 1 लूप)।
- जांचें कि क्या \\( V_{R3} \approx 5V \\) (मेश 2 लूप)।
- पुष्टि करें कि \\( I_1 - I_2 \\) R2 के माध्यम से करंट से मेल खाता है।

### **सुझाव और समस्या निवारण**
- **सटीकता:** Arduino के ADC का रिज़ॉल्यूशन ~4.9mV है, और रेसिस्टर टॉलरेंस के कारण मामूली विचलन हो सकता है। सटीक सत्यापन के लिए मल्टीमीटर का उपयोग करें।
- **समायोजन:** यदि वोल्टेज 5V से अधिक हो जाता है (यहां असंभव), तो उन्हें स्केल डाउन करने के लिए वोल्टेज डिवाइडर का उपयोग करें।
- **विस्तार:** जटिलता के लिए अधिक मेश या घटक जोड़ें, लेकिन पिन सीमा (A0–A5) ध्यान में रखें।

यह सेटअप सैद्धांतिक करंट्स को मापित करंट्स के साथ संरेखित करके मेश एनालिसिस की पुष्टि करता है। यदि आपको सर्किट को परिष्कृत करने, डीबग करने, या अधिक मेश जोड़ने में सहायता की आवश्यकता हो तो मुझे बताएं!
