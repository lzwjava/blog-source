यहाँ एक बैश स्क्रिप्ट है जो कई `curl` उदाहरणों को जोड़ती है:

```bash
#!/bin/bash

# बेसिक GET रिक्वेस्ट
echo "बेसिक GET रिक्वेस्ट:"
curl http://example.com

# कस्टम हेडर्स के साथ GET
echo "\nकस्टम हेडर्स के साथ GET रिक्वेस्ट:"
curl -H "User-Agent: MyUserAgent/1.0" http://example.com

# POST रिक्वेस्ट
echo "\nPOST रिक्वेस्ट:"
curl -X POST -d "key1=value1&key2=value2" http://example.com/post

# JSON डेटा के साथ POST
echo "\nJSON डेटा के साथ POST:"
curl -X POST -H "Content-Type: application/json" -d "{\"key1\":\"value1\",\"key2\":\"value2\"}" http://example.com/post

# प्रॉक्सी का उपयोग करना
echo "\nप्रॉक्सी का उपयोग करना:"
curl -x http://proxy.example.com:8080 http://example.com

# ऑथेंटिकेशन
echo "\nबेसिक ऑथेंटिकेशन:"
curl -u username:password http://example.com/auth

# एक फ़ाइल डाउनलोड करना
echo "\nएक फ़ाइल डाउनलोड करना:"
curl -O http://example.com/file.zip

# रीडायरेक्ट्स का पालन करना
echo "\nरीडायरेक्ट्स का पालन करना:"
curl -L http://example.com/redirect

# आउटपुट को एक फ़ाइल में सेव करना
echo "\nआउटपुट को एक फ़ाइल में सेव करना:"
curl http://example.com -o output.txt

# कस्टम HTTP मेथड
echo "\nकस्टम HTTP मेथड (DELETE):"
curl -X DELETE http://example.com/delete/resource

# वर्बोज़ आउटपुट
echo "\nवर्बोज़ आउटपुट:"
curl -v http://example.com

# कनेक्शन टाइमआउट
echo "\nकनेक्शन टाइमआउट:"
curl --connect-timeout 5 http://example.com

echo "\nस्क्रिप्ट एक्ज़िक्यूशन पूरा हुआ।"
```

इस स्क्रिप्ट को `.sh` एक्सटेंशन के साथ सेव करें, जैसे `curl_examples.sh`। सुनिश्चित करें कि इसे एक्ज़िक्यूटेबल बनाएँ यदि आप यूनिक्स-जैसे वातावरण में हैं या विंडोज सबसिस्टम फॉर लिनक्स (WSL) में चला रहे हैं:

```sh
chmod +x curl_examples.sh
```

फिर, आप इसे चला सकते हैं:

```sh
./curl_examples.sh
```

**विंडोज के लिए नोट्स:**
- यदि आप इसे विंडोज Cmd या PowerShell में चला रहे हैं, तो आपको `bash` या `wsl` का उपयोग करना होगा यदि आपके पास WSL इंस्टॉल है, या सीधे एक्ज़िक्यूशन के लिए सुनिश्चित करें कि `curl` आपके PATH में है:
  - `bash curl_examples.sh` या `wsl ./curl_examples.sh`
- याद रखें, यह स्क्रिप्ट बैश सिंटैक्स का उपयोग करती है, इसलिए यह विंडोज कमांड प्रॉम्प्ट या PowerShell के साथ मूल रूप से संगत नहीं है, जब तक कि WSL या बैश वातावरण के माध्यम से न चलाया जाए। हालाँकि, `curl` कमांड्स स्वयं विंडोज वर्जन के `curl` के साथ संगत हैं।
