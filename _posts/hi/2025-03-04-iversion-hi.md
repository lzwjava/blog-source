`iVersion` पॉड को अपने iOS प्रोजेक्ट में उपयोग करने के लिए, आपको इन चरणों का पालन करना होगा:

1. **Xcode में अपना प्रोजेक्ट खोलें**: सुनिश्चित करें कि आपने अपने Xcode प्रोजेक्ट खोला है जहां आप `iVersion` पॉड जोड़ना चाहते हैं।

2. **CocoaPods को प्रारंभ करें**: अगर आपने पहले से CocoaPods को अपने प्रोजेक्ट में प्रारंभ नहीं किया है, तो टर्मिनल खोलें, अपने प्रोजेक्ट डायरेक्टरी में जाएं, और निम्नलिखित कमांड चलाएं:
   ```bash
   pod init
   ```
   यह आपके प्रोजेक्ट डायरेक्टरी में एक `Podfile` बनाएगा।

3. **Podfile को संपादित करें**: एक टेक्स्ट एडिटर या Xcode में `Podfile` खोलें। `Podfile` में `iVersion` पॉड जोड़ें, निम्नलिखित लाइन को टारगेट सेक्शन के भीतर शामिल करें:
   ```ruby
   pod 'iVersion', '1.11.4'
   ```

4. **Pod को इंस्टॉल करें**: `Podfile` को सेंधा करें और फिर टर्मिनल पर वापस जाएं। निम्नलिखित कमांड चलाएं `iVersion` पॉड को इंस्टॉल करने के लिए:
   ```bash
   pod install
   ```

5. **`.xcworkspace` फ़ाइल को खोलें**: इंस्टॉलेशन पूरा होने के बाद, अगर आपके Xcode प्रोजेक्ट खुला है, तो उसे बंद करें, और फिर नए बनाए गए `.xcworkspace` फ़ाइल को खोलें। यह फ़ाइल आमतौर पर आपके मूल `.xcodeproj` फ़ाइल के साथ ही डायरेक्टरी में स्थित होती है।

6. **iVersion को आयात करें**: अपने Swift या Objective-C फ़ाइलों में, `iVersion` मॉड्यूल को आयात करें ताकि इसका उपयोग शुरू कर सकें:
   ```swift
   import iVersion
   ```

7. **iVersion को अपने कोड में उपयोग करें**: अब आप अपने प्रोजेक्ट में `iVersion` लाइब्रेरी का उपयोग कर सकते हैं। उदाहरण के लिए, आप `iVersion` द्वारा प्रदान किए गए विधियों का उपयोग करके ऐप संस्करण या बिल्ड संख्या जांच सकते हैं।

इन चरणों का पालन करके, आप अपने iOS प्रोजेक्ट में `iVersion` पॉड को सफलतापूर्वक इंटीग्रेट और उपयोग करने में सक्षम होंगे।
