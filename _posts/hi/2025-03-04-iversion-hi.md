अपने iOS प्रोजेक्ट में `iVersion` पॉड का उपयोग करने के लिए, आपको इन चरणों का पालन करना होगा:

1. **अपना प्रोजेक्ट Xcode में खोलें**: सुनिश्चित करें कि आपका Xcode प्रोजेक्ट खुला है जहाँ आप `iVersion` पॉड जोड़ना चाहते हैं।

2. **CocoaPods इनिशियलाइज़ करें**: यदि आपने अपने प्रोजेक्ट में पहले से CocoaPods इनिशियलाइज़ नहीं किया है, तो टर्मिनल खोलें, अपने प्रोजेक्ट डायरेक्टरी में नेविगेट करें, और यह कमांड चलाएँ:
   ```bash
   pod init
   ```
   यह आपके प्रोजेक्ट डायरेक्टरी में एक `Podfile` बनाएगा।

3. **Podfile एडिट करें**: `Podfile` को टेक्स्ट एडिटर या Xcode में खोलें। टार्गेट सेक्शन के अंदर निम्नलिखित लाइन शामिल करके `iVersion` पॉड को अपने `Podfile` में जोड़ें:
   ```ruby
   pod 'iVersion', '1.11.4'
   ```

4. **पॉड इंस्टॉल करें**: `Podfile` सेव करें और टर्मिनल पर वापस जाएँ। `iVersion` पॉड को इंस्टॉल करने के लिए निम्नलिखित कमांड चलाएँ:
   ```bash
   pod install
   ```

5. **`.xcworkspace` फ़ाइल खोलें**: इंस्टॉलेशन पूरा होने के बाद, अपना Xcode प्रोजेक्ट बंद करें यदि वह खुला है, और फिर नई बनाई गई `.xcworkspace` फ़ाइल खोलें। यह फ़ाइल आमतौर पर आपकी मूल `.xcodeproj` फ़ाइल के समान डायरेक्टरी में स्थित होती है।

6. **iVersion इम्पोर्ट करें**: अपनी Swift या Objective-C फ़ाइलों में, `iVersion` मॉड्यूल को इम्पोर्ट करें ताकि आप इसका उपयोग शुरू कर सकें:
   ```swift
   import iVersion
   ```

7. **अपने कोड में iVersion का उपयोग करें**: अब आप `iVersion` लाइब्रेरी का अपने प्रोजेक्ट में उपयोग कर सकते हैं। उदाहरण के लिए, आप `iVersion` द्वारा प्रदान किए गए मेथड्स का उपयोग करके ऐप वर्जन या बिल्ड नंबर चेक कर सकते हैं।

इन चरणों का पालन करके, आपको अपने iOS प्रोजेक्ट में `iVersion` पॉड को सफलतापूर्वक एकीकृत और उपयोग करने में सक्षम होना चाहिए।
