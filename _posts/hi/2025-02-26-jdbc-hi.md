Java में JDBC (Java Database Connectivity) ड्राइवर का उपयोग करने के लिए, आपको डेटाबेस से कनेक्शन स्थापित करने, क्वेरीज़ निष्पादित करने और परिणामों को संभालने के लिए चरणों का पालन करना होगा। नीचे, मैं आपको इस प्रक्रिया से चरण-दर-चरण एक स्पष्ट और व्यावहारिक तरीके से गुजरता हूँ।

### चरण 1: अपने प्रोजेक्ट में JDBC ड्राइवर जोड़ें
सबसे पहले, आपको उस डेटाबेस के लिए उपयुक्त JDBC ड्राइवर की आवश्यकता होगी जिसके साथ आप काम कर रहे हैं (जैसे, MySQL, PostgreSQL, Oracle, आदि)। प्रत्येक डेटाबेस वेंडर अपना स्वयं का ड्राइवर प्रदान करता है।

- **Maven प्रोजेक्ट्स के लिए**:
  अपनी `pom.xml` में डिपेंडेंसी जोड़ें। उदाहरण के लिए, MySQL के लिए:
  ```xml
  <dependency>
      <groupId>mysql</groupId>
      <artifactId>mysql-connector-java</artifactId>
      <version>8.0.33</version> <!-- नवीनतम संस्करण के लिए जाँच करें -->
  </dependency>
  ```
  PostgreSQL के लिए:
  ```xml
  <dependency>
      <groupId>org.postgresql</groupId>
      <artifactId>postgresql</artifactId>
      <version>42.7.3</version> <!-- नवीनतम संस्करण के लिए जाँच करें -->
  </dependency>
  ```

- **गैर-Maven प्रोजेक्ट्स के लिए**:
  वेंडर की वेबसाइट (जैसे, mysql.com से MySQL Connector/J) से JDBC ड्राइवर JAR फ़ाइल डाउनलोड करें और इसे अपने प्रोजेक्ट के classpath में मैन्युअल रूप से जोड़ें।

### चरण 2: JDBC ड्राइवर लोड करें
Java के पुराने संस्करणों (JDBC 4.0 से पहले) में, आपको `Class.forName()` का उपयोग करके ड्राइवर को स्पष्ट रूप से लोड करना होता था। आधुनिक JDBC ड्राइवरों (JDBC 4.0+) के साथ, यह चरण वैकल्पिक है क्योंकि ड्राइवर Service Provider तंत्र के माध्यम से स्वतः पंजीकृत हो जाता है। हालाँकि, स्पष्टता के लिए, यहाँ बताया गया है कि यह कैसे काम करता है:

```java
try {
    Class.forName("com.mysql.cj.jdbc.Driver"); // MySQL उदाहरण
    // PostgreSQL के लिए: Class.forName("org.postgresql.Driver");
} catch (ClassNotFoundException e) {
    e.printStackTrace();
}
```

### चरण 3: कनेक्शन स्थापित करें
डेटाबेस से कनेक्ट होने के लिए `DriverManager` क्लास का उपयोग करें, एक कनेक्शन URL, यूज़रनेम और पासवर्ड प्रदान करके। URL का फॉर्मेट डेटाबेस पर निर्भर करता है।

```java
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class JdbcExample {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase"; // MySQL उदाहरण
        String username = "your_username";
        String password = "your_password";

        try (Connection connection = DriverManager.getConnection(url, username, password)) {
            System.out.println("Connected to the database!");
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
```

- **URL उदाहरण**:
  - MySQL: `jdbc:mysql://localhost:3306/mydatabase?useSSL=false`
  - PostgreSQL: `jdbc:postgresql://localhost:5432/mydatabase`
  - Oracle: `jdbc:oracle:thin:@localhost:1521:xe`

### चरण 4: एक स्टेटमेंट बनाएँ और क्वेरीज़ निष्पादित करें
एक बार कनेक्ट हो जाने के बाद, आप `Statement`, `PreparedStatement` (पैरामीटरयुक्त क्वेरीज़ के लिए बेहतर), या `CallableStatement` (स्टोर्ड प्रोसीजर के लिए) का उपयोग करके SQL क्वेरीज़ निष्पादित कर सकते हैं।

#### एक साधारण क्वेरी के साथ उदाहरण:
```java
import java.sql.*;

public class JdbcExample {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String username = "your_username";
        String password = "your_password";

        try (Connection connection = DriverManager.getConnection(url, username, password)) {
            Statement statement = connection.createStatement();
            ResultSet resultSet = statement.executeQuery("SELECT * FROM users");

            while (resultSet.next()) {
                System.out.println("ID: " + resultSet.getInt("id") + ", Name: " + resultSet.getString("name"));
            }
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
```

#### एक PreparedStatement के साथ उदाहरण (यूज़र इनपुट के लिए सुरक्षित):
```java
try (Connection connection = DriverManager.getConnection(url, username, password)) {
    String sql = "SELECT * FROM users WHERE id = ?";
    PreparedStatement preparedStatement = connection.prepareStatement(sql);
    preparedStatement.setInt(1, 5); // पैरामीटर मान सेट करें
    ResultSet resultSet = preparedStatement.executeQuery();

    while (resultSet.next()) {
        System.out.println("ID: " + resultSet.getInt("id") + ", Name: " + resultSet.getString("name"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```

### चरण 5: रिसोर्सेज़ बंद करें
संसाधनों को मुक्त करने के लिए हमेशा अपने `ResultSet`, `Statement`, और `Connection` ऑब्जेक्ट्स को बंद करें। try-with-resources का उपयोग (जैसा कि ऊपर दिखाया गया है) यह आपके लिए स्वचालित रूप से संभाल लेता है।

### पूर्ण कार्यशील उदाहरण (MySQL)
यहाँ एक पूर्ण उदाहरण है जो `id` (int) और `name` (varchar) कॉलम वाली `users` टेबल मानता है:

```java
import java.sql.*;

public class JdbcExample {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase?useSSL=false";
        String username = "root";
        String password = "your_password";

        try (Connection connection = DriverManager.getConnection(url, username, password)) {
            String sql = "SELECT * FROM users WHERE id = ?";
            PreparedStatement preparedStatement = connection.prepareStatement(sql);
            preparedStatement.setInt(1, 1);
            ResultSet resultSet = preparedStatement.executeQuery();

            while (resultSet.next()) {
                System.out.println("ID: " + resultSet.getInt("id") + ", Name: " + resultSet.getString("name"));
            }
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
```

### मुख्य सुझाव
- **एरर हैंडलिंग**: डेटाबेस संबंधी त्रुटियों को संभालने के लिए हमेशा `SQLException` को कैच करें।
- **डिपेंडेंसीज़**: सुनिश्चित करें कि JDBC ड्राइवर आपके डेटाबेस संस्करण से मेल खाता हो।
- **सुरक्षा**: SQL इंजेक्शन को रोकने के लिए यूज़र इनपुट वाली क्वेरीज़ के लिए `PreparedStatement` का उपयोग करें।
- **कनेक्शन पूलिंग**: प्रोडक्शन ऐप्स के लिए, `DriverManager` के बजाय कनेक्शन पूल (जैसे, HikariCP) का उपयोग करने पर विचार करें।
