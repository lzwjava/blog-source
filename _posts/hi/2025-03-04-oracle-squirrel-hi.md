Oracle JDBC ड्राइवर को Squirrel SQL क्लाइंट में जोड़ने के लिए, इन विस्तृत चरणों का पालन करें। इस प्रक्रिया में ड्राइवर की सेटिंग और आपकी Oracle डेटाबेस के लिए एक कनेक्शन एलीस का सेटअप शामिल है।

### चरण 1: आवश्यक फ़ाइल का पता लगाएं
आपको Oracle JDBC ड्राइवर फ़ाइल, विशेष रूप से `ojdbc.jar` चाहिए। यह फ़ाइल Squirrel SQL क्लाइंट को Oracle डेटाबेस से कनेक्ट करने के लिए आवश्यक क्लासेस को शामिल करती है। सुनिश्चित करें कि आपने इस फ़ाइल को डाउनलोड किया है (जैसे कि Oracle वेबसाइट या आपकी डेटाबेस एडमिनिस्ट्रेटर से) और आपका सिस्टम पर इसका स्थान जानते हैं।

### चरण 2: Squirrel SQL क्लाइंट को लॉन्च करें
अपने कंप्यूटर पर Squirrel SQL क्लाइंट एप्लिकेशन को खोलें।

### चरण 3: ड्राइवर टैब तक पहुंचें
Squirrel SQL क्लाइंट इंटरफ़ेस के बाएं हिस्से में, **ड्राइवर** टैब को खोजें और क्लिक करें। इस सेक्शन में आप एप्लिकेशन के लिए उपलब्ध JDBC ड्राइवरों को प्रबंधित कर सकते हैं।

### चरण 4: नया ड्राइवर जोड़ें
- ड्राइवर टैब में, **"+"** बटन पर क्लिक करें "ड्राइवर जोड़ें" डायलॉग बॉक्स को खोलने के लिए।

### चरण 5: ड्राइवर का नाम दें
- "ड्राइवर जोड़ें" डायलॉग में "नाम" फ़ील्ड में **Oracle Thin ड्राइवर** दर्ज करें। यह एक वर्णनात्मक नाम है Squirrel SQL क्लाइंट में Oracle ड्राइवर को पहचानने के लिए।

### चरण 6: `ojdbc.jar` फ़ाइल जोड़ें
- "ड्राइवर जोड़ें" डायलॉग में **अधिक क्लास पाथ** टैब पर स्विच करें।
- **जोड़ें** बटन पर क्लिक करें।
- सिस्टम पर `ojdbc.jar` फ़ाइल का स्थान खोजें, इसे चुने और ड्राइवर के क्लासपाथ में जोड़ने के लिए कन्फर्म करें।

### चरण 7: जावा ड्राइवर क्लास का उल्लेख करें
- "क्लास नाम" फ़ील्ड में जावा ड्राइवर क्लास दर्ज करें: **oracle.jdbc.OracleDriver**. यह Squirrel SQL क्लाइंट को बताता है कि `ojdbc.jar` फ़ाइल से Oracle डेटाबेस कनेक्शन को हैंडल करने के लिए कौन सा क्लास का उपयोग करना है।

### चरण 8: एक उदाहरण URL प्रदान करें
- आप एक उदाहरण URL फ़ॉर्मेट को ऑप्शनल रूप से सेट करने के लिए Oracle डेटाबेस से कनेक्ट करने के लिए सेट कर सकते हैं:
  - **SID के माध्यम से कनेक्ट करने के लिए**: `jdbc:oracle:thin:@HOST[:PORT]:DB`
  - **सेवा नाम के माध्यम से कनेक्ट करने के लिए**: `jdbc:oracle:thin:@//HOST[:PORT]/DB`
- `HOST`, `PORT`, और `DB` को बाद में (एलीस कन्फिगरेशन में) कनेक्शन सेटअप करते समय वास्तविक मानों से बदलें।

### चरण 9: ड्राइवर कन्फिगरेशन को सेइव करें
- ड्राइवर सेटिंग्स को सेइव करने और "ड्राइवर जोड़ें" डायलॉग को बंद करने के लिए **OK** पर क्लिक करें। "Oracle Thin ड्राइवर" अब ड्राइवर टैब में एक हरी चेकमार्क के साथ दिखाई देगा, जो यह दर्शाता है कि यह सही तरह से कन्फिगर किया गया है।

### चरण 10: अपने डेटाबेस के लिए एक एलीस बनाएं
- Squirrel SQL क्लाइंट के बाएं हिस्से में **एलीस** टैब पर स्विच करें।
- **"+"** बटन पर क्लिक करें "एलीस जोड़ें" डायलॉग बॉक्स को खोलने के लिए।

### चरण 11: एलीस को कन्फिगर करें
- "एलीस जोड़ें" डायलॉग में:
  - **नाम**: इस कनेक्शन के लिए एक नाम दर्ज करें (जैसे, "मेरा Oracle डेटाबेस")।
  - **ड्राइवर**: ड्रॉपडाउन मेनू से **Oracle Thin ड्राइवर** चुने।
  - **URL**: अपने विशिष्ट Oracle डेटाबेस के लिए कनेक्शन URL दर्ज करें:
    - SID के माध्यम से: `jdbc:oracle:thin:@HOST[:PORT]:DB`
    - सेवा नाम के माध्यम से: `jdbc:oracle:thin:@//HOST[:PORT]/DB`
    - `HOST` (जैसे कि localhost या IP पते), `PORT` (डिफॉल्ट 1521 है), और `DB` (SID या सेवा नाम) को अपने डेटाबेस के विवरणों से बदलें।
  - **यूजरनेम**: डेटाबेस यूजरनेम दर्ज करें।
  - **पासवर्ड**: डेटाबेस पासवर्ड दर्ज करें।

### चरण 12: कनेक्शन को टेस्ट करें
- "एलीस जोड़ें" डायलॉग में **टेस्ट** बटन पर क्लिक करें सेट किए गए विवरणों का उपयोग करके Squirrel SQL क्लाइंट को आपकी Oracle डेटाबेस से कनेक्ट करने की पुष्टि करने के लिए। अगर सफल, तो आप एक कन्फर्मेशन संदेश देखेंगे।

### चरण 13: एलीस को सेइव करें
- अगर टेस्ट कनेक्शन सफल है, तो **OK** पर क्लिक करें एलीस को सेइव करने के लिए। नया एलीस अब एलीस टैब में दिखाई देगा।

### चरण 14: डेटाबेस से कनेक्ट करें
- एलीस टैब में, अपने नए बनाए गए एलीस (जैसे, "मेरा Oracle डेटाबेस") को चुने।
- **कनेक्ट** बटन पर क्लिक करें अपने Oracle डेटाबेस से कनेक्शन स्थापित करने के लिए। एक बार कनेक्ट होने पर, आप Squirrel SQL क्लाइंट का उपयोग करके अपने डेटाबेस को क्वेरी करने शुरू कर सकते हैं।

### मुख्य जानकारी का सारांश
- **आवश्यक फ़ाइल**: `ojdbc.jar`
- **जावा ड्राइवर क्लास**: `oracle.jdbc.OracleDriver`
- **URL फ़ॉर्मेट्स**:
  - SID: `jdbc:oracle:thin:@HOST[:PORT]:DB`
  - सेवा नाम: `jdbc:oracle:thin:@//HOST[:PORT]/DB`

इन चरणों का पालन करके, आपने Oracle JDBC ड्राइवर को Squirrel SQL क्लाइंट में सफलतापूर्वक जोड़ा होगा और अपने Oracle डेटाबेस से कनेक्शन का सेटअप किया होगा। सुनिश्चित करें कि सभी विवरण (फ़ाइल पथ, URL, क्रेडेंशियल्स) सही हैं कनेक्शन समस्याओं से बचने के लिए।
