Oracle JDBC ड्राइवर को Squirrel SQL क्लाइंट में जोड़ने के लिए, इन विस्तृत चरणों का पालन करें। इस प्रक्रिया में ड्राइवर को कॉन्फ़िगर करना और आपके Oracle डेटाबेस के लिए एक कनेक्शन एलियास सेट करना शामिल है।

### चरण 1: आवश्यक फ़ाइल का पता लगाएँ
आपको Oracle JDBC ड्राइवर फ़ाइल, विशेष रूप से `ojdbc.jar` की आवश्यकता है। इस फ़ाइल में वे आवश्यक क्लासेज़ होती हैं जो Squirrel SQL क्लाइंट को Oracle डेटाबेस से कनेक्ट करने में सक्षम बनाती हैं। सुनिश्चित करें कि आपने इस फ़ाइल को डाउनलोड कर लिया है (उदाहरण के लिए, Oracle वेबसाइट से या अपने डेटाबेस एडमिनिस्ट्रेटर से) और आपको अपने सिस्टम पर इसका स्थान पता है।

### चरण 2: Squirrel SQL क्लाइंट लॉन्च करें
अपने कंप्यूटर पर Squirrel SQL क्लाइंट एप्लिकेशन खोलें।

### चरण 3: ड्राइवर्स टैब तक पहुँचें
Squirrel SQL क्लाइंट इंटरफेस के बाईं ओर, **Drivers** टैब को ढूंढें और उस पर क्लिक करें। यह सेक्शन आपको एप्लिकेशन के लिए उपलब्ध JDBC ड्राइवर्स को प्रबंधित करने की अनुमति देता है।

### चरण 4: एक नया ड्राइवर जोड़ें
- Drivers टैब में, **"+"** बटन पर क्लिक करें ताकि "Add Driver" डायलॉग बॉक्स खुल सके।

### चरण 5: ड्राइवर का नाम दें
- "Add Driver" डायलॉग के "Name" फ़ील्ड में, **Oracle Thin Driver** दर्ज करें। यह Squirrel SQL क्लाइंट के भीतर Oracle ड्राइवर की पहचान करने के लिए एक वर्णनात्मक नाम है।

### चरण 6: `ojdbc.jar` फ़ाइल जोड़ें
- "Add Driver" डायलॉग के अंदर **Extra Class Path** टैब पर स्विच करें।
- **Add** बटन पर क्लिक करें।
- अपने सिस्टम पर `ojdbc.jar` फ़ाइल के स्थान पर नेविगेट करें, उसे चुनें, और ड्राइवर की क्लासपथ में जोड़ने के लिए पुष्टि करें।

### चरण 7: Java ड्राइवर क्लास निर्दिष्ट करें
- "Class Name" फ़ील्ड में, Java ड्राइवर क्लास दर्ज करें: **oracle.jdbc.OracleDriver**। यह Squirrel SQL क्लाइंट को बताता है कि Oracle डेटाबेस कनेक्शन को हैंडल करने के लिए `ojdbc.jar` फ़ाइल से किस क्लास का उपयोग करना है।

### चरण 8: एक उदाहरण URL प्रदान करें
- वैकल्पिक रूप से, आप Oracle डेटाबेस से कनेक्ट होने के लिए एक उदाहरण URL फॉर्मेट निर्दिष्ट कर सकते हैं:
  - **SID के माध्यम से कनेक्ट करने के लिए**: `jdbc:oracle:thin:@HOST[:PORT]:DB`
  - **सर्विस नाम के माध्यम से कनेक्ट करने के लिए**: `jdbc:oracle:thin:@//HOST[:PORT]/DB`
- बाद में कनेक्शन सेटअप करते समय (एलियास कॉन्फ़िगरेशन में) `HOST`, `PORT`, और `DB` को वास्तविक मानों से बदलें।

### चरण 9: ड्राइवर कॉन्फ़िगरेशन सहेजें
- ड्राइवर सेटिंग्स को सहेजने और "Add Driver" डायलॉग को बंद करने के लिए **OK** पर क्लिक करें। "Oracle Thin Driver" अब Drivers टैब में एक हरे चेकमार्क के साथ दिखाई देना चाहिए, जो इंगित करता है कि यह ठीक से कॉन्फ़िगर है।

### चरण 10: अपने डेटाबेस के लिए एक एलियास बनाएँ
- Squirrel SQL क्लाइंट के बाईं ओर **Aliases** टैब पर स्विच करें।
- "Add Alias" डायलॉग बॉक्स खोलने के लिए **"+"** बटन पर क्लिक करें।

### चरण 11: एलियास कॉन्फ़िगर करें
- "Add Alias" डायलॉग में:
  - **Name**: इस कनेक्शन के लिए एक नाम दर्ज करें (उदाहरण के लिए, "My Oracle DB")।
  - **Driver**: ड्रॉपडाउन मेनू से **Oracle Thin Driver** चुनें।
  - **URL**: अपने विशिष्ट Oracle डेटाबेस के लिए कनेक्शन URL दर्ज करें:
    - SID के माध्यम से: `jdbc:oracle:thin:@HOST[:PORT]:DB`
    - सर्विस नाम के माध्यम से: `jdbc:oracle:thin:@//HOST[:PORT]/DB`
    - `HOST` (उदाहरण के लिए, localhost या IP address), `PORT` (डिफ़ॉल्ट 1521 है), और `DB` (SID या सर्विस नाम) को अपने डेटाबेस के विवरण के साथ बदलें।
  - **Username**: डेटाबेस यूज़रनेम दर्ज करें।
  - **Password**: डेटाबेस पासवर्ड दर्ज करें।

### चरण 12: कनेक्शन का परीक्षण करें
- यह सत्यापित करने के लिए कि Squirrel SQL क्लाइंट प्रदान किए गए विवरणों का उपयोग करके आपके Oracle डेटाबेस से कनेक्ट हो सकता है, "Add Alias" डायलॉग में **Test** बटन पर क्लिक करें। यदि सफल होता है, तो आपको एक पुष्टिकरण संदेश दिखाई देगा।

### चरण 13: एलियास सहेजें
- यदि टेस्ट कनेक्शन सफल होता है, तो एलियास को सहेजने के लिए **OK** पर क्लिक करें। नया एलियास अब Aliases टैब में दिखाई देगा।

### चरण 14: डेटाबेस से कनेक्ट करें
- Aliases टैब में, अपने नव निर्मित एलियास (उदाहरण के लिए, "My Oracle DB") का चयन करें।
- अपने Oracle डेटाबेस से कनेक्शन स्थापित करने के लिए **Connect** बटन पर क्लिक करें। एक बार कनेक्ट हो जाने के बाद, आप Squirrel SQL क्लाइंट का उपयोग करके अपने डेटाबेस को क्वेरी करना शुरू कर सकते हैं।

### मुख्य जानकारी का सारांश
- **आवश्यक फ़ाइल**: `ojdbc.jar`
- **Java ड्राइवर क्लास**: `oracle.jdbc.OracleDriver`
- **URL फॉर्मेट**:
  - SID: `jdbc:oracle:thin:@HOST[:PORT]:DB`
  - सर्विस नाम: `jdbc:oracle:thin:@//HOST[:PORT]/DB`

इन चरणों का पालन करके, आपने Squirrel SQL क्लाइंट में Oracle JDBC ड्राइवर को सफलतापूर्वक जोड़ दिया होगा और अपने Oracle डेटाबेस के लिए एक कनेक्शन कॉन्फ़िगर कर लिया होगा। कनेक्शन संबंधी समस्याओं से बचने के लिए सभी विवरणों (फ़ाइल पथ, URL, क्रेडेंशियल्स) के सटीक होना सुनिश्चित करें।
