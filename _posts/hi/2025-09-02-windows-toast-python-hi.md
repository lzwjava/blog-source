### win10toast क्या है?
win10toast एक हल्का Python लाइब्रेरी है जो आपको Windows 10 पर अपने Python स्क्रिप्ट्स से टोस्ट नोटिफिकेशन (पॉप-अप अलर्ट) भेजने में सक्षम बनाता है। यह डेस्कटॉप एप्लिकेशन या स्क्रिप्ट्स में यूजर नोटिफिकेशन के लिए उपयोगी है, जो मूल Windows टोस्ट के समान है।

### आवश्यक शर्तें
- Python 3.x इंस्टॉल होना चाहिए।
- Windows 10 (यह Windows-विशिष्ट APIs पर निर्भर करता है)।
- Python के अलावा कोई अतिरिक्त डिपेंडेंसी की आवश्यकता नहीं है।

### इंस्टॉलेशन
पाइप (Python पैकेज इंस्टॉलर) के माध्यम से लाइब्रेरी इंस्टॉल करें:

```
pip install win10toast
```

यदि आप वर्चुअल एनवायरनमेंट का उपयोग कर रहे हैं, तो पहले उसे एक्टिवेट करें।

### बेसिक उपयोग
1. मॉड्यूल इम्पोर्ट करें:
   ```python
   from win10toast import ToastNotifier
   ```

2. एक `ToastNotifier` इंस्टेंस बनाएं और नोटिफिकेशन दिखाने के लिए इसकी `show_toast` मेथड को कॉल करें:
   ```python
   toaster = ToastNotifier()
   toaster.show_toast("शीर्षक", "संदेश", icon_path=None, duration=5)
   ```
   - **शीर्षक**: नोटिफिकेशन हेडलाइन के लिए एक स्ट्रिंग।
   - **संदेश**: नोटिफिकेशन बॉडी के लिए एक स्ट्रिंग।
   - **icon_path**: नोटिफिकेशन आइकन के लिए .ico या .png फ़ाइल का वैकल्पिक पथ (उदाहरण के लिए, `"path/to/icon.ico"`)। कोई आइकन नहीं चाहिए तो छोड़ दें।
   - **duration**: टोस्ट के दिखाई देने का समय सेकंड में (डिफॉल्ट 5 है; Windows इसे ओवरराइड कर सकता है)।

नोटिफिकेशन एक्शन सेंटर और पॉप-अप के रूप में दिखाई देगा।

### एडवांस्ड उपयोग
- **आइकन कस्टमाइज करना**: एक इमेज फ़ाइल का उपयोग करें (बेस्ट क्वालिटी के लिए 128x128 पिक्सल की सिफारिश की जाती है)।
- **क्लिक इवेंट्स को हैंडल करना**: एक कॉलबैक फंक्शन पास करें (सीमित सपोर्ट; मुख्य रूप से ऑन-क्लिक व्यवहार बेसिक है)।
- **मल्टीपल नोटिफिकेशन**: `ToastNotifier` को एक बार इंस्टेंटिएट करें और मल्टीपल टोस्ट भेजने के लिए इसे दोबारा उपयोग करें।
- **एरर हैंडलिंग**: संभावित Windows API समस्याओं को हैंडल करने के लिए ट्राई-एक्सेप्ट में रैप करें, उदाहरण के लिए, यदि नोटिफिकेशन डिसेबल हैं।

आइकन और एरर हैंडलिंग के साथ उदाहरण:
```python
from win10toast import ToastNotifier
import os

try:
    toaster = ToastNotifier()
    icon = "C:\\path\\to\\icon.ico"  # वास्तविक पथ से बदलें
    toaster.show_toast("अलर्ट!", "आपकी स्क्रिप्ट पूरी हो गई है।", icon_path=icon, duration=10)
except Exception as e:
    print(f"नोटिफिकेशन विफल: {e}")
```

### सीमाएँ और सुझाव
- नोटिफिकेशन के लिए Windows 10 का एक्शन सेंटर एनेबल होना आवश्यक है (सेटिंग्स > सिस्टम > नोटिफिकेशन और एक्शन)।
- अधिक कॉम्प्लेक्स फीचर्स (जैसे, एक्शन या इमेज) के लिए, plyer या winrt (मॉडर्न Windows SDK का हिस्सा) जैसे विकल्पों पर विचार करें।
- यदि नोटिफिकेशन दिखाई नहीं देते हैं, तो Windows फोकस असिस्ट सेटिंग्स जांचें या स्क्रिप्ट को एडमिनिस्ट्रेटर के रूप में रन करने का प्रयास करें।
