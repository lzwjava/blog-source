YouTube वीडियो की सभी कैप्शन प्राप्त करने के लिए, अपनी आवश्यकताओं और आपके मैनुअल या स्वचालित दृष्टिकोण के आधार पर इन चरणों का पालन करें। ध्यान दें कि वीडियो के लिए कैप्शन उपलब्ध होने चाहिए (या तो स्वचालित रूप से जनरेट किए गए या क्रिएटर द्वारा अपलोड किए गए)।

### मैनुअल विधि (YouTube की अंतर्निहित ट्रांसक्रिप्ट सुविधा का उपयोग करके)
1.  **वीडियो खोलें**: वेब ब्राउज़र में YouTube पर जाएं और वांछित वीडियो पर नेविगेट करें।
2.  **कैप्शन की जांच करें**: वीडियो को रोकने के लिए क्लिक करें। प्लेयर के निचले-दाएं कोने में "CC" (क्लोज्ड कैप्शन) आइकन देखें। यदि दिखाई देता है, तो कैप्शन उपलब्ध हैं।
3.  **ट्रांसक्रिप्ट एक्सेस करें**:
    - वीडियो विवरण पर स्क्रॉल करें और "और दिखाएं" पर क्लिक करें।
    - "ट्रांसक्रिप्ट दिखाएं" (यदि उपलब्ध हो) ढूंढें और क्लिक करें। यह वीडियो के दाईं ओर टाइमस्टैम्प और टेक्स्ट के साथ एक ट्रांसक्रिप्ट पैनल खोलता है।
4.  **टाइमस्टैम्प टॉगल करें**: ट्रांसक्रिप्ट पैनल के ऊपरी-दाएं कोने में तीन लंबवत बिंदुओं पर क्लिक करें और अपनी पसंद के आधार पर टाइमस्टैम्प दिखाने या छिपाने के लिए "टाइमस्टैम्प टॉगल करें" चुनें।
5.  **ट्रांसक्रिप्ट कॉपी करें**:
    - ट्रांसक्रिप्ट के नीचे तक स्क्रॉल करें, अंतिम शब्द के बाद क्लिक करके होल्ड करें, फिर सभी टेक्स्ट को हाइलाइट करने के लिए शीर्ष तक ड्रैग करें।
    - कॉपी करने के लिए `Ctrl + C` (Windows) या `Command + C` (Mac) दबाएं।
6.  **पेस्ट और सेव करें**: एक टेक्स्ट एडिटर (जैसे, Notepad, TextEdit, या Word) खोलें, `Ctrl + V` या `Command + V` के साथ टेक्स्ट पेस्ट करें, और `.txt` फ़ाइल या अपने पसंदीदा फॉर्मेट में सेव करें।

**नोट**: यह विधि केवल YouTube वेबसाइट पर काम करती है, मोबाइल ऐप पर नहीं।[](https://www.wikihow.com/Download-YouTube-Video-Subtitles)

### कंटेंट क्रिएटर्स के लिए (अपने स्वयं के वीडियो से कैप्शन डाउनलोड करना)
यदि आप वीडियो के मालिक हैं, तो आप सीधे YouTube Studio से कैप्शन डाउनलोड कर सकते हैं:
1.  **YouTube Studio में लॉग इन करें**: [studio.youtube.com](https://studio.youtube.com) पर जाएं।
2.  **वीडियो चुनें**: बाएं मेनू में "कंटेंट" पर क्लिक करें, फिर वीडियो चुनें।
3.  **सबटाइटल एक्सेस करें**: बाएं मेनू में "सबटाइटल" पर क्लिक करें, फिर भाषा चुनें।
4.  **कैप्शन डाउनलोड करें**: सबटाइटल ट्रैक के आगे तीन-बिंदु मेनू पर क्लिक करें और "डाउनलोड" चुनें। `.srt`, `.vtt`, या `.sbv` जैसे फॉर्मेट में से चुनें।
5.  **एडिट या उपयोग करें**: आगे उपयोग के लिए डाउनलोड की गई फ़ाइल को टेक्स्ट एडिटर या सबटाइटल एडिटर (जैसे, Aegisub) में खोलें।

**नोट**: आप केवल उन वीडियो के लिए कैप्शन फ़ाइलें डाउनलोड कर सकते हैं जो आपके द्वारा मैनेज किए जाने वाले चैनलों पर हैं।[](https://ito-engineering.screenstepslive.com/s/ito_fase/a/1639680-how-do-i-download-a-caption-file-from-youtube)

### स्वचालित विधि (थर्ड-पार्टी टूल्स का उपयोग करके)
यदि आपको किसी विशिष्ट फॉर्मेट (जैसे, `.srt`) में कैप्शन की आवश्यकता है या ऐसे वीडियो के लिए जिनके आप मालिक नहीं हैं, तो किसी प्रतिष्ठित थर्ड-पार्टी टूल का उपयोग करें:
1.  **एक टूल चुनें**: लोकप्रिय विकल्पों में शामिल हैं:
    - **DownSub**: सबटाइटल डाउनलोड करने के लिए एक मुफ्त ऑनलाइन टूल।
    - **Notta**: उच्च सटीकता के साथ ट्रांसक्रिप्शन और सबटाइटल डाउनलोड प्रदान करता है।[](https://www.notta.ai/en/blog/download-subtitles-from-youtube)
    - **4K Download**: सबटाइटल एक्सट्रैक्शन के लिए एक डेस्कटॉप ऐप।[](https://verbit.ai/captioning/a-guide-to-downloading-subtitles-and-captions-from-youtube-enhancing-accessibility-and-user-experience/)
2.  **वीडियो URL कॉपी करें**: YouTube वीडियो खोलें, वीडियो के नीचे "शेयर" पर क्लिक करें, और URL कॉपी करें।
3.  **टूल का उपयोग करें**:
    - टूल के इनपुट फ़ील्ड में URL पेस्ट करें।
    - वांछित भाषा और फॉर्मेट (जैसे, `.srt`, `.txt`) चुनें।
    - "डाउनलोड" या "एक्सट्रैक्ट" पर क्लिक करें और फ़ाइल सेव करें।
4.  **सत्यापित करें**: सटीकता सुनिश्चित करने के लिए फ़ाइल खोलें, क्योंकि स्वचालित रूप से जनरेट किए गए कैप्शन में त्रुटियां हो सकती हैं।

**सावधानी**: सुरक्षा जोखिमों से बचने के लिए विश्वसनीय टूल्स का उपयोग करें। कुछ टूल्स में विज्ञापन हो सकते हैं या उन्नत सुविधाओं के लिए भुगतान की आवश्यकता हो सकती है।[](https://gotranscript.com/blog/how-to-download-subtitles-from-youtube)

### YouTube API का उपयोग करना (डेवलपर्स के लिए)
बल्क कैप्शन एक्सट्रैक्शन या ऐप एकीकरण के लिए, YouTube Data API का उपयोग करें:
1.  **API एक्सेस सेट अप करें**: [Google Cloud Console](https://console.cloud.google.com) में एक प्रोजेक्ट बनाएं, YouTube Data API v3 सक्षम करें, और एक API कुंजी प्राप्त करें।
2.  **कैप्शन ट्रैक्स की सूची बनाएं**: किसी वीडियो के लिए उपलब्ध कैप्शन ट्रैक्स को पुनः प्राप्त करने के लिए `captions.list` एंडपॉइंट का उपयोग करें। उदाहरण:
    ```
    GET https://www.googleapis.com/youtube/v3/captions?part=snippet&videoId=VIDEO_ID&key=API_KEY
    ```
3.  **कैप्शन डाउनलोड करें**: किसी विशिष्ट कैप्शन ट्रैक को प्राप्त करने के लिए `captions.download` एंडपॉइंट का उपयोग करें। उदाहरण:
    ```
    GET https://www.googleapis.com/youtube/v3/captions/CAPTION_ID?tfmt=srt&key=API_KEY
    ```
4.  **सीमाएँ**:
    - आप केवल अपने स्वयं के वीडियो के कैप्शन डाउनलोड कर सकते हैं, जब तक कि वीडियो मालिक ने उन्हें सार्वजनिक रूप से सुलभ नहीं बनाया हो।
    - API उपयोग की कोटा सीमाएँ होती हैं (लगभग 200 यूनिट प्रति कैप्शन डाउनलोड)।[](https://developers.google.com/youtube/v3/docs/captions)[](https://stackoverflow.com/questions/73863672/how-can-i-get-captions-of-a-youtube-video-and-display-it-separately)
5.  **वैकल्पिक**: कुछ डेवलपर्स वीडियो पेज सोर्स से टाइम्ड टेक्स्ट URL (जैसे, `https://www.youtube.com/api/timedtext?...`) स्क्रैप करते हैं, लेकिन यह विश्वसनीय नहीं है, YouTube की शर्तों का उल्लंघन कर सकता है, और IP ब्लॉकिंग का जोखिम होता है।[](https://stackoverflow.com/questions/73863672/how-can-i-get-captions-of-a-youtube-video-and-display-it-separately)

### अतिरिक्त सुझाव
-   **भाषा चयन**: यदि कैप्शन कई भाषाओं में उपलब्ध हैं, तो "सबटाइटल/CC" सेटिंग्स या ट्रांसक्रिप्ट के ड्रॉपडाउन मेनू से अपनी पसंदीदा भाषा चुनें।[](https://riverside.fm/blog/youtube-transcription)
-   **स्वचालित रूप से जनरेट किए गए कैप्शन**: उच्चारण, पृष्ठभूमि शोर, या जटिल ऑडियो के कारण इनमें त्रुटियां हो सकती हैं। यदि सटीकता महत्वपूर्ण है तो समीक्षा करें और संपादित करें।[](https://support.google.com/youtube/answer/6373554?hl=en)
-   **सुलभता**: कैप्शन बधिर या सुनने में अक्षम दर्शकों के लिए सुलभता बढ़ाते हैं और सामग्री को खोजने योग्य बनाकर SEO में सुधार करते हैं।[](https://www.descript.com/blog/article/transcript-of-youtube-videos)
-   **कोई कैप्शन उपलब्ध नहीं?**: यदि वीडियो में कैप्शन नहीं हैं, तो ऑडियो से ट्रांसक्रिप्ट जनरेट करने के लिए Rev या Notta जैसी ट्रांसक्रिप्शन सेवाओं का उपयोग करने पर विचार करें।[](https://www.rev.com/resources/how-to-download-youtube-subtitles-as-text-files)[](https://www.notta.ai/en/blog/download-subtitles-from-youtube)
