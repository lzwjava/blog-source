### पूर्वापेक्षाएँ
PIL (Pillow) लाइब्रेरी से `ImageGrab` का उपयोग करने के लिए, सुनिश्चित करें कि Pillow इंस्टॉल है। विंडोज़ पर, इसे pip के माध्यम से इंस्टॉल करें:
```
pip install Pillow
```
विंडोज़ के लिए आमतौर पर किसी अतिरिक्त सेटअप की आवश्यकता नहीं होती है, क्योंकि `ImageGrab` नेटिव विंडोज़ APIs का उपयोग करता है।

### बेसिक उपयोग
`ImageGrab.grab()` पूरी स्क्रीन (सभी मॉनिटर) को एक PIL Image ऑब्जेक्ट के रूप में कैप्चर करता है। स्क्रीनशॉट लेने और उसे PNG फ़ाइल के रूप में सेव करने का एक सरल उदाहरण यहाँ दिया गया है:

```python
from PIL import ImageGrab

# पूरी स्क्रीन कैप्चर करें
screenshot = ImageGrab.grab()

# इमेज को एक फ़ाइल में सेव करें
screenshot.save("screenshot.png")
```

यह आपके करंट वर्किंग डायरेक्टरी में `screenshot.png` नाम की एक फ़ाइल सेव करेगा।

### अतिरिक्त विकल्प
- **एक विशिष्ट क्षेत्र कैप्चर करें**: पिक्सेल कोऑर्डिनेट्स के टपल के रूप में एक बाउंडिंग बॉक्स (left, top, right, bottom) निर्दिष्ट करें।
  ```python
  # (100, 100) से शुरू होने वाला 300x300 पिक्सेल का क्षेत्र कैप्चर करें
  region = (100, 100, 400, 400)  # आवश्यकतानुसार कोऑर्डिनेट्स एडजस्ट करें
  screenshot = ImageGrab.grab(bbox=region)
  screenshot.save("region_screenshot.png")
  ```
- **माउस कर्सर शामिल करें** (यदि समर्थित हो; वर्जन विवरण के लिए Pillow डॉक्यूमेंटेशन चेक करें):
  ```python
  # नोट: कुछ सिस्टम पर इसमें include_layered_windows=True की आवश्यकता हो सकती है
  screenshot = ImageGrab.grab(include_layered_windows=True)
  ```
- **इमेज दिखाएँ**: इसे अपने डिफ़ॉल्ट इमेज व्यूअर में खोलने के लिए `screenshot.show()` का उपयोग करें (टेस्टिंग के लिए)।

### नोट्स
- कोऑर्डिनेट्स पिक्सेल में होते हैं, जहाँ (0, 0) प्राइमरी मॉनिटर के ऊपरी-बाएँ कोने पर होता है।
- मल्टी-मॉनिटर सेटअप के लिए, `grab()` सभी स्क्रीन को एक कंपोजिट इमेज के रूप में कैप्चर करता है।
- यदि आपको कोई समस्या आती है (जैसे, नए विंडोज़ वर्जन पर परमिशन एरर), सुनिश्चित करें कि आपकी स्क्रिप्ट के पास स्क्रीन कैप्चर की अनुमति है या इसे एडमिनिस्ट्रेटर प्रिविलेज के साथ रन करने का प्रयास करें।
- पूरी डॉक्यूमेंटेशन: [Pillow ImageGrab docs](https://pillow.readthedocs.io/en/stable/reference/ImageGrab.html). विंडोज़-विशिष्ट समस्याओं के लिए, यदि आवश्यक हो तो GitHub इश्यूज़ चेक करें।
