Ubuntu में, आप निम्नलिखित कमांड्स का उपयोग करके सिस्टम की जानकारी जैसे कि सिस्टम 32-बिट है या 64-बिट, ऑपरेटिंग सिस्टम का वर्जन, RAM की मात्रा और डिस्क का आकार प्राप्त कर सकते हैं। नीचे एक चरण-दर-चरण गाइड दी गई है:

### 1. जांचें कि सिस्टम 32-बिट है या 64-बिट
- **कमांड**: `uname -m`
- **व्याख्या**: यह कमांड मशीन के हार्डवेयर का नाम प्रदर्शित करता है।
  - यदि आउटपुट `x86_64` है, तो आपका सिस्टम **64-बिट** है।
  - यदि आउटपुट `i686` या `i386` है, तो आपका सिस्टम **32-बिट** है।
- **उदाहरण आउटपुट**:
  ```
  x86_64
  ```
  यह एक 64-बिट सिस्टम को इंगित करता है।

### 2. ऑपरेटिंग सिस्टम (OS) वर्जन प्राप्त करें
- **कमांड**: `lsb_release -a`
- **व्याख्या**: यह कमांड Linux Standard Base (LSB) और डिस्ट्रीब्यूशन के बारे में विस्तृत जानकारी प्रदान करता है।
  - Ubuntu वर्जन देखने के लिए `Description` लाइन को देखें।
- **उदाहरण आउटपुट**:
  ```
  No LSB modules are available.
  Distributor ID: Ubuntu
  Description:    Ubuntu 20.04.3 LTS
  Release:        20.04
  Codename:       focal
  ```
  यहां, OS **Ubuntu 20.04.3 LTS** है।

### 3. RAM का आकार प्राप्त करें
- **कमांड**: `free -h`
- **व्याख्या**: यह कमांड मेमोरी उपयोग को मानव-पठनीय फॉर्मेट (जैसे, GB या MB) में दिखाता है।
  - इंस्टॉल कुल RAM जानने के लिए `Mem:` पंक्ति के अंतर्गत `total` कॉलम को देखें।
- **उदाहरण आउटपुट**:
  ```
              total        used        free      shared  buff/cache   available
  Mem:          15Gi       2.3Gi       9.5Gi       300Mi       3.8Gi        12Gi
  Swap:        2.0Gi         0B       2.0Gi
  ```
  इस मामले में, कुल RAM **15 GB** है।

### 4. डिस्क का आकार प्राप्त करें
- **कमांड**: `lsblk -d -o NAME,SIZE`
- **व्याख्या**: यह कमांड सभी डिस्क डिवाइस (पार्टीशन को छोड़कर) को उनके आकार के साथ सूचीबद्ध करता है।
  - `-d` फ्लैग सुनिश्चित करता है कि केवल डिस्क डिवाइस (जैसे, `/dev/sda`) दिखाई दें, उनके पार्टीशन नहीं।
  - डिस्क का नाम (जैसे, `sda`) और उसके संबंधित आकार को देखें।
- **उदाहरण आउटपुट**:
  ```
  NAME   SIZE
  sda    1T
  sdb    500G
  ```
  यहां, सिस्टम में दो डिस्क हैं: `/dev/sda` जिसका आकार **1 टेराबाइट** है और `/dev/sdb` जिसका आकार **500 गीगाबाइट** है। यदि आपके पास केवल एक डिस्क है, तो यह आमतौर पर `sda` होगी।

### नोट्स
- ये कमांड्स Ubuntu में उपलब्ध मानक टूल्स का उपयोग करती हैं, जिनके लिए किसी अतिरिक्त इंस्टॉलेशन की आवश्यकता नहीं होती।
- डिस्क आकार के लिए, यदि आपके सिस्टम में एकाधिक डिस्क हैं, तो आउटपुट उन सभी को सूचीबद्ध करेगा। आमतौर पर, `/dev/sda` प्राथमिक डिस्क होती है।
- अपने सिस्टम के लिए विशिष्ट परिणाम देखने के लिए इन कमांड्स को एक टर्मिनल में चलाएं।

इन कमांड्स की सहायता से आप अपने Ubuntu मशीन पर आवश्यक सिस्टम जानकारी शीघ्रता से एकत्र कर सकते हैं!
