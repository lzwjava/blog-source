### अनुवाद:

23. दिया गया है कि सदिश \\( \alpha_1, \alpha_2, \alpha_3 \\) रैखिक रूप से स्वतंत्र हैं, और सदिश \\( \beta_1 = \alpha_1 + 2\alpha_2 \\), \\( \beta_2 = -\alpha_1 + \alpha_2 - 3\alpha_3 \\), \\( \beta_3 = 3\alpha_1 + 6\alpha_3 \\) हैं। सिद्ध कीजिए कि सदिश \\( \beta_1, \beta_2, \beta_3 \\) रैखिक रूप से आश्रित हैं।

### हल:
यह सिद्ध करने के लिए कि सदिश \\( \beta_1, \beta_2, \beta_3 \\) रैखिक रूप से आश्रित हैं, हमें यह दिखाना होगा कि ऐसे अदिश \\( c_1, c_2, c_3 \\) (सभी शून्य नहीं) मौजूद हैं जैसे कि:

\\[ c_1 \beta_1 + c_2 \beta_2 + c_3 \beta_3 = 0 \\]

दिया गया है:
\\[ \beta_1 = \alpha_1 + 2\alpha_2 \\]
\\[ \beta_2 = -\alpha_1 + \alpha_2 - 3\alpha_3 \\]
\\[ \beta_3 = 3\alpha_1 + 6\alpha_3 \\]

हमें स्थिरांक \\( c_1, c_2, c_3 \\) ऐसे खोजने हैं कि:

\\[ c_1 (\alpha_1 + 2\alpha_2) + c_2 (-\alpha_1 + \alpha_2 - 3\alpha_3) + c_3 (3\alpha_1 + 6\alpha_3) = 0 \\]

इसे इस प्रकार पुनः लिखा जा सकता है:

\\[ (c_1 - c_2 + 3c_3)\alpha_1 + (2c_1 + c_2)\alpha_2 + (-3c_2 + 6c_3)\alpha_3 = 0 \\]

चूंकि \\( \alpha_1, \alpha_2, \alpha_3 \\) रैखिक रूप से स्वतंत्र हैं, इसलिए \\( \alpha_1, \alpha_2, \alpha_3 \\) के गुणांक प्रत्येक शून्य होने चाहिए:

1. \\( c_1 - c_2 + 3c_3 = 0 \\)
2. \\( 2c_1 + c_2 = 0 \\)
3. \\( -3c_2 + 6c_3 = 0 \\)

आइए इस समीकरण प्रणाली को हल करें:

समीकरण 2 से:
\\[ 2c_1 + c_2 = 0 \implies c_2 = -2c_1 \\]

समीकरण 3 से:
\\[ -3c_2 + 6c_3 = 0 \implies -3(-2c_1) + 6c_3 = 0 \implies 6c_1 + 6c_3 = 0 \implies c_3 = -c_1 \\]

\\( c_2 = -2c_1 \\) और \\( c_3 = -c_1 \\) को समीकरण 1 में प्रतिस्थापित करें:
\\[ c_1 - (-2c_1) + 3(-c_1) = 0 \implies c_1 + 2c_1 - 3c_1 = 0 \implies 0 = 0 \\]

यह किसी भी \\( c_1 \\) के लिए सत्य है। आइए \\( c_1 = 1 \\) चुनें:

तब:
\\[ c_2 = -2 \\]
\\[ c_3 = -1 \\]

इस प्रकार, हमने स्थिरांक \\( c_1 = 1, c_2 = -2, c_3 = -1 \\) पाए हैं जैसे कि:

\\[ 1 \cdot \beta_1 - 2 \cdot \beta_2 - 1 \cdot \beta_3 = 0 \\]

यह सिद्ध करता है कि सदिश \\( \beta_1, \beta_2, \beta_3 \\) रैखिक रूप से आश्रित हैं।
