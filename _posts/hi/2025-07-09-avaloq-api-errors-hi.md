**समस्या निवारण:**

- निष्पादित करने वाले उपयोगकर्ता के लिए उचित अनुमतियाँ सुनिश्चित करें

### बैंकिंग संदर्भ में मानक HTTP API त्रुटियाँ

**1. 400 Bad Request**
Avaloq वातावरण में सामान्य कारण:
- अमान्य खाता संख्याएँ या क्लाइंट आईडी
- विकृत लेनदेन राशि
- ट्रेडिंग ऑर्डर में आवश्यक फ़ील्ड्स की कमी
- अमान्य तारीख स्वरूप या सीमाएँ

**समस्या निवारण:**
- यह सुनिश्चित करने के लिए URL की जांच करें कि आप उनके अनुरोधों के साथ मान्य डेटा पैरामीटर भेज रहे हैं और वे सही हेडर का उपयोग कर रहे हैं
- Avaloq की स्कीमा आवश्यकताओं के विरुद्ध सभी इनपुट पैरामीटर मान्य करें
- मुद्रा कोड और फ़ॉर्मेटिंग जांचें
- खाता स्थिति और अनुमतियाँ सत्यापित करें

**2. 401 Unauthorized**
बैंकिंग-विशिष्ट कारण:
- अमान्य API क्रेडेंशियल्स
- समाप्त प्रमाणीकरण टोकन
- विशिष्ट ऑपरेशनों के लिए अपर्याप्त उपयोगकर्ता अनुमतियाँ
- क्लाइंट संबंध प्रतिबंध

**समस्या निवारण:**
- API कुंजी और सीक्रेट वैधता सत्यापित करें
- टोकन समाप्ति समय जांचें
- पुष्टि करें कि उपयोगकर्ता के पास उचित बैंकिंग अनुमतियाँ हैं
- क्लाइंट-सलाहकार संबंध मैपिंग की समीक्षा करें

**3. 403 Forbidden**
वेल्थ मैनेजमेंट संदर्भ:
- विशिष्ट क्लाइंट खातों तक पहुँच अस्वीकृत
- ऑपरेशनों पर नियामक प्रतिबंध
- अनुपालन नियम उल्लंघन
- अधिकार क्षेत्र-आधारित सीमाएँ

**समस्या निवारण:**
- उपयोगकर्ता पहुँच अधिकार और भूमिकाओं की समीक्षा करें
- अनुपालन नियम और प्रतिबंध जांचें
- नियामक अनुमतियाँ सत्यापित करें
- क्रॉस-बॉर्डर लेनदेन की अनुमति की पुष्टि करें

**4. 404 Not Found**
बैंकिंग-विशिष्ट परिदृश्य:
- गैर-मौजूद खाता संख्याएँ
- अमान्य पोर्टफोलियो आईडी
- लापता लेनदेन संदर्भ
- हटाए गए या संग्रहीत क्लाइंट रिकॉर्ड

**समस्या निवारण:**
- अपने एंडपॉइंट को दोबारा जांचें और सुनिश्चित करें कि वह सही ढंग से लिखा गया है
- खाता अस्तित्व और स्थिति सत्यापित करें
- संग्रहीत या निष्क्रिय खातों के लिए जांचें
- उचित URL निर्माण की पुष्टि करें

**5. 500 Internal Server Error**
सिस्टम-स्तरीय मुद्दे:
- डेटाबेस कनेक्टिविटी समस्याएँ
- कोर बैंकिंग सिस्टम विफलताएँ
- एकीकरण सेवा आउटेज
- मेमोरी या प्रदर्शन संबंधी मुद्दे

**समस्या निवारण:**
- सिस्टम स्वास्थ्य डैशबोर्ड जांचें
- डेटाबेस कनेक्शन पूल की समीक्षा करें
- सिस्टम संसाधन उपयोग की निगरानी करें
- निर्भर सेवा उपलब्धता सत्यापित करें
