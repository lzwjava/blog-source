Pour afficher la console dans Safari, suivez ces étapes :

1. Ouvrez Safari sur votre Mac.
2. Allez dans la barre de menu en haut et cliquez sur **Safari** > **Préférences** (ou **Préférences** si vous utilisez une version plus ancienne).
3. Cliquez sur l'onglet **Avancé**.
4. En bas, cochez la case qui dit **"Afficher le menu Développer dans la barre de menus"**.
5. Fermez la fenêtre des Préférences.
6. Maintenant, dans la barre de menu, vous verrez une nouvelle option **Développer**. Cliquez sur **Développer** > **Afficher la console JavaScript** (ou **Afficher l'inspecteur Web** selon votre version).

Alternativement, vous pouvez utiliser le raccourci clavier **Option + Command + C** (⌥ + ⌘ + C) une fois que le menu Développer est activé pour ouvrir rapidement la console.

Si vous êtes sur un iPhone ou un iPad, Safari n'affiche pas nativement une console, mais vous pouvez l'activer en connectant votre appareil à un Mac via USB, en ouvrant Safari sur le Mac, et en utilisant le menu Développer pour inspecter le navigateur de l'appareil iOS.
