### 1. Visa de visiteur (B-1/B-2) – Entrée à court terme
- **Pourquoi cela convient** : Le plus simple si vous souhaitez juste visiter (par exemple, tourisme, conférences ou réunions d'affaires). Vous êtes déjà entré deux fois aux USA, probablement via l'ESTA (Electronic System for Travel Authorization), que les détenteurs de passeport chinois peuvent utiliser pour jusqu'à 90 jours pour des voyages d'affaires ou de plaisir sans visa. Spécifications : B-2 pour le tourisme ; valable jusqu'à 6 mois initialement.
- **Processus** : Postulez en ligne via le formulaire DS-160 via l'ambassade des États-Unis à Guangzhou. Frais : ~200 USD. Délai d'approbation : 1-2 semaines. Des liens solides avec la Chine (emploi à HSBC, famille) aident à montrer l'intention de retour.
- **Avantages/Inconvénients** : Faible barrière ; possibilité de prolonger le séjour informellement. Inconvénients : Aucune autorisation de travail ; le risque de séjour prolongé annule les futurs visas.
- **Prochaines étapes** : Vérifiez l'éligibilité sur travel.state.gov ; planifiez un entretien si nécessaire.

### 2. Options de visa de travail – Pour les professionnels comme vous
Vos 11 ans en ingénierie logicielle, 8+ ans en full-stack/mobile et vos projets GitHub pourraient vous qualifier pour les visas de travailleur qualifié. Défi : Aucun diplôme de licence formel (une exigence courante pour le H-1B), mais votre réussite au NOIP (top 300 dans un concours provincial compétitif) et votre expérience autodidacte pourraient aider à démontrer une éducation équivalente via des évaluations.

- **Visa H-1B (Profession Spécialisée)** :
  - **Pourquoi cela convient** : Conçu pour les professionnels de la technologie ; nécessite un employeur sponsor pour déposer le dossier. Vos compétences (par exemple, Java/Spring, technologies cloud) sont très demandées.
  - **Exigences** : Diplôme de licence (ou équivalent) dans le domaine ; prouve une profession spécialisée. Plafond annuel : 85 000 visas, mais seulement ~25 000 pour les nouvelles demandes—c'est une loterie. Coût : Frais de dossier ~500 $, plus traitement premium ~1 500 $.
  - **Processus** : Trouvez un sponsor américain (par exemple, des entreprises technologiques comme Apple/Google via des références à distance ou des candidatures). Postulez de mars à mai ; résultats en juin. Valable jusqu'à 3 ans, renouvelable.
  - **Astuce** : L'auto-emploi ou le freelancing ne compte pas ; besoin d'une offre d'emploi formelle. Les dépendants H-1B peuvent se joindre.
  - **Avantages/Inconvénients** : Voie vers la carte verte (EB-3/2/1). Inconvénients : Aléatoire de la loterie ; l'âge (29 ans) convient, mais de nombreux candidats retardent en raison des retards.

- **Visa O-1 (Capacité Extraordinaire)** :
  - **Pourquoi cela convient** : Vos 480 articles de blog (60K vues/an), 10+ projets open-source (500+ commits chacun), trois articles académiques et votre résolution de problèmes (1000+ algorithmes, NOIP top 300) démontrent une "distinction" en programmation/innovation. Similaire à ce que les ingénieurs logiciels comme vous utilisent souvent.
  - **Exigences** : Montrez une capacité extraordinaire via des prix, des publications, etc. Aucun plafond ; pour 1-2 ans.
  - **Processus** : L'employeur présente une pétition ; pétition personnelle si possible. Frais/délais similaires au H-1B. Mène à un pont vers la carte verte O-1.
  - **Avantages** : Pas de loterie ; potentiellement auto-financé. Inconvénients : Seuil de preuve élevé ; besoin d'une excellence continue.

- **Carte verte EB-1 (Capacité Extraordinaire)** :
  - **Pourquoi cela convient** : Idéal pour les innovateurs—votre triple couronne d'open-source, de blogging et d'auto-recherche pourrait vous qualifier selon les critères de l'USCIS (par exemple, prix/salaire élevé, publications, couverture médiatique). Le Visa Bulletin note des retards pour les Chinois (jusqu'à 10+ ans).
  - **Processus** : Auto-candidature ; aucun sponsor nécessaire. Aucun plafond annuel. Coûts ~1 100 $ - 3 500 $ ; traitement 1-2 ans plus file d'attente.
  - **Avantages** : Carte verte directe ; famille/conjoint inclus. Inconvénients : Nécessite des preuves solides ; traité au cas par cas.

### 3. Visas d'étudiant ou de formation – Si une reconversion est une option
- **Visa F-1** : Pour étudier, mais vous auriez besoin d'une admission dans une université américaine. Votre diplôme d'associé autodidacte et vos compétences pourraient transférer des crédits. Postulez via SEVIS ; concentrez-vous sur un Master/Doctorat en informatique/ML (correspond à votre année en machine learning).
- **Visa J-1 (Visiteur d'Échange)** : Pour la formation (par exemple, via des programmes d'entreprise). Nécessite un sponsor américain ; 1-2 ans maximum.
- **Adéquation** : Si vous abandonnez l'autodidacte pour un diplôme, cela ouvre des portes—de nombreux travailleurs technologiques utilisent le F-1 comme pont vers le H-1B.

- **Résidence permanente via le mariage** : Épousez un citoyen américain pour un processus rapide I-129F.
- **Travail à distance et Vigna** : Pas un visa ; possibilité de travailler pour les USA depuis l'étranger, mais la résidence permanente n'est pas officielle.
- **Coûts et délais** : Visas : 160 $ - 15 000 $ ; cartes vertes : 700 $ - 3 500 $. Traitement : 3 mois à 10+ ans.
