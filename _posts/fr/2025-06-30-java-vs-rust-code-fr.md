### La prévalence de Java
Java, publié en 1995, est l'un des langages de programmation les plus utilisés en raison de sa philosophie "écrire une fois, exécuter partout", de son écosystème mature et de son adoption massive dans les applications d'entreprise, le développement Android et les services web. Les indicateurs clés incluent :
- **Indice TIOBE (Juin 2025)** : Java se classe constamment parmi les 5 premiers langages de programmation, souvent autour de la 2ème ou 3ème place, avec une part d'environ 10 à 15 % de l'activité de programmation mondiale basée sur les données des moteurs de recherche et les offres d'emploi.
- **Sondage des développeurs Stack Overflow (2023)** : Java était utilisé par environ 30 % des développeurs professionnels, reflétant sa domination dans l'entreprise (par exemple, la banque, le commerce électronique) et le développement d'applications Android.
- **Dépôts GitHub** : Le rapport Octoverse 2023 de GitHub indiquait Java comme l'un des principaux langages, avec des millions de dépôts. La part de Java était d'environ 10 % des contributions aux dépôts publics, derrière seulement JavaScript et Python.
- **Utilisation en entreprise** : Java alimente des frameworks majeurs comme Spring et Hadoop, et il est intégré dans des milliards d'appareils Android, de backends d'entreprise et de systèmes hérités (par exemple, les remplacements de COBOL dans la finance).

Compte tenu des 30 ans d'histoire de Java et de son utilisation généralisée, le volume total de code Java est immense. On estime que des milliards de lignes de code (LdC) existent en Java, en particulier dans les systèmes d'entreprise, avec des contributions continues de l'ordre de centaines de millions de LdC chaque année dans les dépôts publics et privés.

### La prévalence de Rust
Rust, publié en 2010 avec sa première version stable en 2015, est plus récent mais a gagné du terrain pour la programmation système, les applications critiques pour les performances et les projets axés sur la sécurité. Les indicateurs clés incluent :
- **Sondage des développeurs Stack Overflow (2023)** : Rust était utilisé par environ 9 % des développeurs, mais il a été élu langage "le plus aimé" pendant des années, indiquant une forte adoption parmi les passionnés et les développeurs systèmes.
- **Dépôts GitHub** : La part de Rust dans l'Octoverse 2023 de GitHub était d'environ 2 à 3 % des contributions, nettement moins que Java mais en croissance rapide, en particulier dans les projets open source comme Servo de Mozilla, les composants Windows de Microsoft et les systèmes de bas niveau d'Android.
- **Adoption par l'industrie** : Des entreprises comme AWS, Microsoft et Google utilisent Rust pour des composants critiques pour les performances (par exemple, Firecracker d'AWS, le framework média d'Android). Cependant, son utilisation est plus niche, se concentrant sur la programmation système, l'infrastructure cloud et la blockchain.
- **Courbe d'apprentissage** : La courbe d'apprentissage abrupte de Rust et son accent sur la programmation de bas niveau limitent son utilisation dans le développement rapide d'applications par rapport à l'applicabilité plus large de Java.

La base de code de Rust est plus petite en raison de son jeune âge et de ses cas d'utilisation spécialisés. Les estimations suggèrent que la base de code totale de Rust se situe dans les dizaines de millions de LdC, avec des contributions annuelles croissantes mais représentant encore une fraction de celles de Java.
