Pour empêcher Outlook de supprimer le lien de réunion Zoom après avoir répondu à une invitation, vous devez ajuster un paramètre spécifique dans les options d'Outlook. Par défaut, Outlook est configuré pour supprimer automatiquement les emails de demande de réunion de votre boîte de réception après y avoir répondu (par exemple, accepter, refuser ou répondre "peut-être"). Étant donné que le lien de réunion Zoom est inclus dans l'email de demande de réunion original, supprimer cet email supprime l'accès facile au lien. Suivez les étapes ci-dessous pour modifier ce paramètre et conserver l'email de demande de réunion (et le lien Zoom) dans votre boîte de réception.

---

### **Étapes pour empêcher Outlook de supprimer les liens de réunion Zoom après une réponse**

1. **Ouvrir les Options d'Outlook** :
   - Dans l'application de bureau Outlook, cliquez sur l'onglet **Fichier** dans le coin supérieur gauche.
   - Dans le menu déroulant, sélectionnez **Options**. Cela ouvrira la boîte de dialogue Options d'Outlook.

2. **Accéder aux Paramètres du Courrier** :
   - Dans la boîte de dialogue Options d'Outlook, cliquez sur **Courrier** dans le menu de gauche. Cette section contient les paramètres relatifs à la gestion de vos emails par Outlook, y compris les demandes de réunion.

3. **Ajuster le Paramètre de Demande de Réunion** :
   - Faites défiler jusqu'à la section **Envoyer des messages** dans les paramètres du Courrier.
   - Recherchez l'option qui dit : **"Supprimer les demandes de réunion et les notifications de la Boîte de réception après y avoir répondu"**.
   - **Décochez** cette option. Ce faisant, vous indiquez à Outlook de conserver les emails de demande de réunion dans votre boîte de réception même après y avoir répondu.

4. **Sauvegarder les Modifications** :
   - Cliquez sur **OK** pour fermer la boîte de dialogue Options d'Outlook et enregistrer vos modifications.

---

### **Comment cela fonctionne**
Une fois que vous avez effectué cet ajustement, Outlook ne supprimera plus automatiquement les emails de demande de réunion après votre réponse. Cela signifie que l'email original contenant le lien de réunion Zoom restera dans votre boîte de réception, vous permettant d'accéder au lien chaque fois que vous en avez besoin.

---

### **Notes importantes**
- **Portée du paramètre** : Ce paramètre s'applique à toutes les demandes de réunion, pas seulement à celles pour les réunions Zoom. Il affectera la façon dont Outlook gère toutes les invitations à des réunions que vous recevez, comme celles pour Microsoft Teams ou d'autres plateformes.
- **Outlook Web Access (OWA)** : Si vous utilisez Outlook Web Access (OWA) au lieu de l'application de bureau, le paramètre pourrait se trouver à un endroit différent.
