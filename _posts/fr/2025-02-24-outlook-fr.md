Pour empêcher Outlook de supprimer le lien de réunion Zoom après avoir répondu à une invitation de réunion, vous devez ajuster un paramètre spécifique dans les options d'Outlook. Par défaut, Outlook est configuré pour supprimer automatiquement les emails de demande de réunion de votre boîte de réception après que vous y avez répondu (par exemple, accepter, décliner ou répondre de manière provisoire). Puisque le lien de réunion Zoom est inclus dans l'email de demande de réunion original, la suppression de l'email enlève l'accès facile au lien. Suivez les étapes ci-dessous pour modifier ce paramètre et conserver l'email de demande de réunion (et le lien Zoom) dans votre boîte de réception.

---

### **Étapes pour empêcher Outlook de supprimer les liens de réunion Zoom après avoir répondu**

1. **Ouvrir les options d'Outlook** :
   - Dans l'application de bureau Outlook, cliquez sur l'onglet **Fichier** dans le coin supérieur gauche.
   - Dans le menu déroulant, sélectionnez **Options**. Cela ouvrira la boîte de dialogue des options d'Outlook.

2. **Accéder aux paramètres de messagerie** :
   - Dans la boîte de dialogue des options d'Outlook, cliquez sur **Courrier** dans le menu de gauche. Cette section contient des paramètres liés à la manière dont Outlook gère vos emails, y compris les demandes de réunion.

3. **Ajuster le paramètre de demande de réunion** :
   - Faites défiler vers le bas jusqu'à la section **Envoyer des messages** dans les paramètres de messagerie.
   - Recherchez l'option qui dit **"Supprimer les demandes de réunion et les notifications de la boîte de réception après avoir répondu"**.
   - **Décochez** cette option. En faisant cela, vous indiquez à Outlook de conserver les emails de demande de réunion dans votre boîte de réception même après que vous y avez répondu.

4. **Enregistrer les modifications** :
   - Cliquez sur **OK** pour fermer la boîte de dialogue des options d'Outlook et enregistrer vos modifications.

---

### **Comment cela fonctionne**
Une fois que vous avez apporté cette modification, Outlook ne supprimera plus automatiquement les emails de demande de réunion après que vous y avez répondu. Cela signifie que l'email original contenant le lien de réunion Zoom restera dans votre boîte de réception, vous permettant d'y accéder à tout moment.

---

### **Notes importantes**
- **Portée du paramètre** : Ce paramètre s'applique à toutes les demandes de réunion, pas seulement à celles pour les réunions Zoom. Il affectera la manière dont Outlook gère toutes les invitations de réunion que vous recevez, telles que celles pour Microsoft Teams ou d'autres plateformes.
- **Outlook Web Access (OWA)** : Si vous utilisez Outlook Web Access (OWA) au lieu de l'application de bureau, le paramètre peut être situé différemment.
