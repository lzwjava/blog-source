En plus de GitHub, d'autres plateformes célèbres pour l'hébergement de code incluent :

- GitLab : Une plateforme DevOps complète qui propose l'hébergement de dépôts, CI/CD, la surveillance et plus encore
- Bitbucket : Détenu par Atlassian, s'intègre bien avec Jira et d'autres produits Atlassian
- SourceForge : L'une des plus anciennes plateformes d'hébergement de code, particulièrement populaire pour les projets open-source
- Azure DevOps (anciennement Visual Studio Team Services) : La solution de Microsoft pour l'hébergement de code et DevOps
- AWS CodeCommit : Le service Git managé d'Amazon
- Gitee : Populaire en Chine, souvent appelé le « GitHub de la Chine »
- Launchpad : Principalement utilisé pour Ubuntu et les projets open-source associés
- CodePen : Spécialisé pour le code front-end (HTML, CSS, JavaScript)
- Codeberg : Une alternative à but non lucratif et communautaire axée sur les logiciels libres et open source
- Replit : Combine l'hébergement de code avec un IDE en ligne

Ces plateformes varient dans leurs fonctionnalités, modèles de tarification et publics cibles, certaines se concentrant davantage sur les utilisateurs professionnels et d'autres sur les communautés open-source.
