- Pour activer la sortie verbeuse dans Homebrew, utilisez `export HOMEBREW_CURL_VERBOSE=1` avec `brew upgrade --verbose`.

- Pour augmenter la verbosité dans SSH, utilisez `ssh -vvv user@host`.

- Pour une sortie plus verbeuse avec `git`, utilisez `git -v` ou `git --verbose`.

- Pour obtenir une sortie plus détaillée avec `rsync`, utilisez `rsync -vv source destination`.
