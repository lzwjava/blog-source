Voici un aperçu clair des **indicatifs téléphoniques internationaux des principaux pays** et du fonctionnement habituel de la **structure de leurs numéros de téléphone** :

---

### Chine

* **Code** : `+86`
* **Structure** : Les numéros de téléphone mobile comportent 11 chiffres (par ex., `+86 138 1234 5678`). Les lignes fixes varient (indicatif régional de 2 à 3 chiffres + numéro local de 7 à 8 chiffres).

### États-Unis / Canada

* **Code** : `+1` (partagé dans le *Plan de numérotation nord-américain, NANP*)
* **Structure** : 10 chiffres après `+1` → `+1 (AAA) BBB-CCCC` (indicatif régional + numéro à 7 chiffres).

### Royaume-Uni

* **Code** : `+44`
* **Structure** : Généralement 10 chiffres après le code, par ex. `+44 7XXX XXXXXX` pour les mobiles. Le `0` initial est supprimé dans le format international (`07` → `+44 7`).

### Inde

* **Code** : `+91`
* **Structure** : Les numéros de téléphone mobile comportent 10 chiffres, par ex. `+91 98765 43210`. Les numéros de ligne fixe ont un indicatif régional de 2 à 4 chiffres + numéro local.

### Hong Kong

* **Code** : `+852`
* **Structure** : 8 chiffres, pas d'indicatif régional (par ex., `+852 5123 4567`).

### Macao

* **Code** : `+853`
* **Structure** : 8 chiffres, pas d'indicatif régional (par ex., `+853 6612 3456`).

### Japon

* **Code** : `+81`
* **Structure** : Supprimer le 0 initial → les mobiles ressemblent à `+81 90-1234-5678`.

### Corée du Sud

* **Code** : `+82`
* **Structure** : Supprimer le 0 initial → les mobiles ressemblent à `+82 10-1234-5678`.

### Singapour

* **Code** : `+65`
* **Structure** : 8 chiffres, pas d'indicatif régional (par ex., `+65 8123 4567`).

### Australie

* **Code** : `+61`
* **Structure** : Supprimer le 0 initial → les mobiles ressemblent à `+61 4XX XXX XXX`.

### Allemagne

* **Code** : `+49`
* **Structure** : Supprimer le 0 initial → les mobiles ressemblent à `+49 151 12345678`. Les indicatifs régionaux varient pour les lignes fixes.

### France

* **Code** : `+33`
* **Structure** : Supprimer le 0 initial → les mobiles ressemblent à `+33 6 12 34 56 78`.

### Italie

* **Code** : `+39`
* **Structure** : Les numéros de téléphone mobile font souvent 10 chiffres, commençant par `3`.

### Brésil

* **Code** : `+55`
* **Structure** : `(AA) NNNNN-NNNN` (indicatif régional à 2 chiffres + numéro de 8 à 9 chiffres).

### Russie

* **Code** : `+7`
* **Structure** : 10 chiffres après le code → `+7 (AAA) BBB-CC-DD`.

---

✅ **Résumé pour votre question spécifique** :

* **Royaume-Uni (giffgaff)** → `+44`, généralement `+44 7XXXXXXXXX` (mobile, 10 chiffres après `+44`).
* **Inde** → `+91`, mobile à 10 chiffres.
* **Chine** → `+86`, mobile à 11 chiffres.
* **États-Unis** → `+1`, numéro à 10 chiffres.
* **HK/Macao** → `+852` et `+853`, tous deux à 8 chiffres.
