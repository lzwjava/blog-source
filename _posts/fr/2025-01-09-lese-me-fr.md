title: Moins d'utilisation de "Je", "Moi" ou "Nous"
---

> "resume-en.txt" n'est pas autorisé à être analysé. Il doit être renommé en "resume.txt".

> "resume-en.txt" n'est pas autorisé à être analysé. Je dois le renommer en "resume.txt".

Lequel est le meilleur ? Apparemment, le premier est meilleur car il implique que ce n'est pas seulement moi, ou lui, ou elle, mais tout le monde qui doit renommer le fichier dans cette situation.

Cela suggère également que la main-d'œuvre humaine pourrait ne pas être nécessaire—cette tâche pourrait potentiellement être gérée par du code ou des machines.

La deuxième version est plus directe, exprimant les choses de manière claire. Cependant, nous devrions viser à exprimer les choses d'un point de vue objectif plutôt que de notre propre perspective. Ajuster nos expressions dans chaque situation peut être difficile.

Concernant ce qui précède, j'ai utilisé "nous" ici. Cependant, ce n'est pas seulement "nous" — le code ou la machine peut également gérer cette tâche. Par conséquent, il serait préférable de l'écrire ainsi : "Notre expression devrait être ajustée de notre point de vue à une perspective objective."

Ne vous concentrez pas sur les êtres humains. Observez simplement comment les choses devraient fonctionner objectivement.
