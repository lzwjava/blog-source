Le chemin JDK se trouve à `/opt/homebrew/Cellar/openjdk@17/17.0.14/libexec/openjdk.jdk/Contents/Home`.

```bash
mvn build
mvn verify
mvn install
mvn spring-boot:run
```

Utilisez la ligne de commande pour compiler et exécuter le code.
