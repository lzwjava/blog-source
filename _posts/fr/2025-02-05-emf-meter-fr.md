J'ai récemment entendu des rumeurs selon lesquelles les stations 5G pourraient être nocives pour les humains, ce qui m'a poussé à enquêter plus avant. Pour ce faire, j'ai acheté un multimètre EMF pour mesurer les champs électromagnétiques. EMF signifie Champ Électromagnétique.

J'ai effectué un test pour mesurer les niveaux d'EMF à différentes distances de mon routeur WiFi TP-LINK AX3000. Vous trouverez ci-dessous les résultats. [^1]

Le multimètre EMF possède trois modes de test : champ magnétique, champ électrique et radiofréquence. Pour les tests du routeur WiFi, nous avons utilisé le mode radiofréquence.

Déplacer un téléphone mobile vers le haut ou vers le bas peut générer un champ magnétique, qu'un multimètre EMF (Champ Électromagnétique) peut détecter.
