### Routines de création de tableaux

*   `np.empty` : Retourne un nouveau tableau de forme et de type donnés, sans initialiser les entrées.
*   `np.empty_like` : Retourne un nouveau tableau avec la même forme et le même type qu'un tableau donné.
*   `np.eye` : Retourne un tableau 2-D avec des uns sur la diagonale et des zéros ailleurs.
*   `np.identity` : Retourne le tableau identité.
*   `np.ones` : Retourne un nouveau tableau de forme et de type donnés, rempli de uns.
*   `np.ones_like` : Retourne un tableau de uns avec la même forme et le même type qu'un tableau donné.
*   `np.zeros` : Retourne un nouveau tableau de forme et de type donnés, rempli de zéros.
*   `np.zeros_like` : Retourne un tableau de zéros avec la même forme et le même type qu'un tableau donné.
*   `np.full` : Retourne un nouveau tableau de forme et de type donnés, rempli de fill\_value.
*   `np.full_like` : Retourne un tableau complet avec la même forme et le même type qu'un tableau donné.
*   `np.array` : Crée un tableau.
*   `np.asarray` : Convertit l'entrée en tableau.
*   `np.asanyarray` : Convertit l'entrée en ndarray, mais laisse passer les sous-classes de ndarray.
*   `np.ascontiguousarray` : Retourne un tableau contigu en mémoire (ordre C).
*   `np.asfortranarray` : Retourne un tableau contigu en mémoire (ordre Fortran).
*   `np.copy` : Retourne une copie du tableau de l'objet donné.
*   `np.frombuffer` : Interprète un tampon comme un tableau 1-dimensionnel.
*   `np.from_dlpack` : Crée un tableau NumPy à partir d'un objet PyCapsule (DLPack).
*   `np.fromfile` : Construit un tableau à partir de données dans un fichier texte ou binaire.
*   `np.fromfunction` : Construit un tableau en exécutant une fonction sur chaque coordonnée.
*   `np.fromiter` : Crée un nouveau tableau 1-dimensionnel à partir d'un objet itérable.
*   `np.fromstring` : Un nouveau tableau 1-D initialisé à partir de données texte dans une chaîne.
*   `np.loadtxt` : Charge des données à partir d'un fichier texte.
*   `np.genfromtxt` : Charge des données à partir d'un fichier texte, avec gestion des valeurs manquantes comme spécifié.
*   `np.savetxt` : Sauvegarde un tableau dans un fichier texte.
*   `np.save` : Sauvegarde un tableau dans un fichier binaire au format NumPy `.npy`.
*   `np.savez` : Sauvegarde plusieurs tableaux dans un seul fichier au format `.npz` non compressé.
*   `np.savez_compressed` : Sauvegarde plusieurs tableaux dans un seul fichier au format `.npz` compressé.

### Routines de manipulation de tableaux

*   `np.reshape` : Donne une nouvelle forme à un tableau sans modifier ses données.
*   `np.ravel` : Retourne un tableau aplati contigu.
*   `np.transpose` : Permute les dimensions d'un tableau.
*   `np.swapaxes` : Échange deux axes d'un tableau.
*   `np.moveaxis` : Déplace les axes d'un tableau vers de nouvelles positions.
*   `np.broadcast_to` : Diffuse un tableau vers une nouvelle forme.
*   `np.broadcast_arrays` : Diffuse n'importe quel nombre de tableaux les uns par rapport aux autres.
*   `np.expand_dims` : Étend la forme d'un tableau.
*   `np.squeeze` : Supprime les entrées unidimensionnelles de la forme d'un tableau.
*   `np.concatenate` : Joint une séquence de tableaux le long d'un axe existant.
*   `np.stack` : Joint une séquence de tableaux le long d'un nouvel axe.
*   `np.hstack` : Empile les tableaux en séquence horizontalement (par colonnes).
*   `np.vstack` : Empile les tableaux en séquence verticalement (par lignes).
*   `np.dstack` : Empile les tableaux en séquence en profondeur (le long de la troisième dimension).
*   `np.column_stack` : Empile des tableaux 1-D comme des colonnes dans un tableau 2-D.
*   `np.row_stack` : Empile les tableaux en séquence verticalement (par lignes).
*   `np.split` : Divise un tableau en plusieurs sous-tableaux.
*   `np.array_split` : Divise un tableau en plusieurs sous-tableaux.
*   `np.hsplit` : Divise un tableau en plusieurs sous-tableaux horizontalement (par colonnes).
*   `np.vsplit` : Divise un tableau en plusieurs sous-tableaux verticalement (par lignes).
*   `np.dsplit` : Divise un tableau en plusieurs sous-tableaux le long du 3ème axe (en profondeur).
*   `np.tile` : Construit un tableau en répétant A le nombre de fois donné par reps.
*   `np.repeat` : Répète les éléments d'un tableau.
*   `np.delete` : Retourne un nouveau tableau avec des sous-tableaux supprimés le long d'un axe.
*   `np.insert` : Insère des valeurs le long de l'axe donné avant les indices donnés.
*   `np.append` : Ajoute des valeurs à la fin d'un tableau.
*   `np.resize` : Retourne un nouveau tableau avec la forme spécifiée.
*   `np.trim_zeros` : Supprime les zéros de début et/ou de fin d'un tableau ou d'une séquence 1-D.
*   `np.unique` : Trouve les éléments uniques d'un tableau.

### Routines d'indexation de tableaux

*   `np.take` : Prend des éléments d'un tableau le long d'un axe.
*   `np.choose` : Construit un tableau à partir d'un tableau d'indices et d'un ensemble de tableaux parmi lesquels choisir.
*   `np.searchsorted` : Trouve les indices où les éléments doivent être insérés pour maintenir l'ordre.
*   `np.count_nonzero` : Compte le nombre de valeurs non nulles dans le tableau ``a``.
*   `np.nonzero` : Retourne les indices des éléments qui sont non nuls.
*   `np.where` : Retourne les éléments choisis dans `x` ou `y` en fonction de la `condition`.
*   `np.indices` : Retourne un tableau représentant les indices d'une grille.
*   `np.ix_` : Construit un maillage ouvert à partir de plusieurs séquences.
*   `np.diag_indices` : Retourne les indices pour accéder à la diagonale principale d'un tableau.
*   `np.diag_indices_from` : Retourne les indices pour accéder à la diagonale principale d'un tableau à n dimensions.
*   `np.mask_indices` : Retourne les indices pour accéder à (n_arrays, n_elements) avec un masque.
*   `np.tril_indices` : Retourne les indices pour le triangle inférieur d'un tableau (n, n).
*   `np.tril_indices_from` : Retourne les indices pour le triangle inférieur de arr.
*   `np.triu_indices` : Retourne les indices pour le triangle supérieur d'un tableau (n, n).
*   `np.triu_indices_from` : Retourne les indices pour le triangle supérieur de arr.

### Routines de type de tableau

*   `np.dtype` : Crée un objet de type de données.
*   `np.can_cast` : Retourne True si un cast entre des types de données peut se produire en toute sécurité.

*   `np.obj2sctype` : Retourne le dtype scalaire ou l'équivalent NumPy du type Python d'un objet.

### Routines d'ensembles de tableaux

*   `np.intersect1d` : Trouve l'intersection de deux tableaux.

*   `np.max` : Retourne le maximum le long d'un axe donné.
*   `np.min` : Retourne le minimum le long d'un axe donné.
*   `np.maximum` : Maximum élément par élément des éléments du tableau.
*   `np.minimum` : Minimum élément par élément des éléments du tableau.
*   `np.nanargmax` : Retourne les indices des valeurs maximales dans l'axe spécifié en ignorant les NaNs.
*   `np.nanargmin` : Retourne les indices des valeurs minimales dans l'axe spécifié en ignorant les NaNs.
*   `np.nanmax` : Retourne le maximum d'un tableau ou le maximum le long d'un axe, en ignorant les NaNs.
*   `np.nanmin` : Retourne le minimum d'un tableau ou le minimum le long d'un axe, en ignorant les NaNs.
*   `np.count_nonzero` : Compte le nombre de valeurs non nulles dans le tableau ``a``.
*   `np.nonzero` : Retourne les indices des éléments qui sont non nuls.

### Opérations sur les chaînes de tableaux

*   `np.char.add` : Retourne la concaténation de chaînes élément par élément pour deux tableaux de str ou unicode.
*   `np.char.multiply` : Retourne (a * i), c'est-à-dire la multiplication de chaîne de i copies de a, pour chaque élément.
*   `np.char.mod` : Retourne (a % i), c'est-à-dire le formatage de chaîne pré-Python 2.6 (interpolation), pour chaque élément de a.
*   `np.char.capitalize` : Retourne une copie de `a` avec seulement le premier caractère de chaque élément en majuscule.
*   `np.char.lower` : Retourne un tableau avec les éléments convertis en minuscules.
*   `np.char.upper` : Retourne un tableau avec les éléments convertis en majuscules.
*   `np.char.title` : Pour chaque élément de `a`, retourne une version en casse titre de la chaîne : les mots commencent par des caractères majuscules, tous les autres caractères sensibles à la casse sont en minuscules.
*   `np.char.split` : Pour chaque élément de `a`, retourne une liste des mots dans la chaîne, en utilisant sep comme délimiteur de chaîne.
*   `np.char.splitlines` : Pour chaque élément de `a`, retourne une liste des lignes dans l'élément, en cassant aux limites de ligne.
*   `np.char.strip` : Pour chaque élément de `a`, retourne une copie avec les caractères de début et de fin supprimés.
*   `np.char.lstrip` : Pour chaque élément de `a`, retourne une copie avec les caractères de début supprimés.
*   `np.char.rstrip` : Pour chaque élément de `a`, retourne une copie avec les caractères de fin supprimés.
*   `np.char.join` : Retourne une chaîne qui est la concaténation des chaînes dans la séquence seq.
*   `np.char.replace` : Pour chaque élément de `a`, retourne une copie de la chaîne avec toutes les occurrences de la sous-chaîne old remplacées par new.
*   `np.char.encode` : Appelle `str.encode` élément par élément.
*   `np.char.decode` : Appelle `str.decode` élément par élément.

### Constantes mathématiques sur les tableaux

*   `np.pi` : Constante mathématique, le rapport de la circonférence d'un cercle à son diamètre.
*   `np.e` : Constante mathématique, base des logarithmes naturels.

### Fonctions logiques sur les tableaux

*   `np.all` : Teste si tous les éléments du tableau le long d'un axe donné évaluent à True.
*   `np.any` : Teste si un élément du tableau le long d'un axe donné évalue à True.
*   `np.array_equal` : True si deux tableaux ont la même forme et les mêmes éléments, False sinon.
*   `np.array_equiv` : Retourne True si les tableaux d'entrée sont cohérents en forme et si tous les éléments sont égaux.
*   `np.greater` : Retourne la valeur de vérité de (x1 > x2) élément par élément.
*   `np.greater_equal` : Retourne la valeur de vérité de (x1 >= x2) élément par élément.
*   `np.less` : Retourne la valeur de vérité de (x1 < x2) élément par élément.
*   `np.less_equal` : Retourne la valeur de vérité de (x1 <= x2) élément par élément.
*   `np.equal` : Retourne (x1 == x2) élément par élément.
*   `np.not_equal` : Retourne (x1 != x2) élément par élément.
*   `np.logical_and` : Calcule la valeur de vérité de x1 ET x2 élément par élément.
*   `np.logical_or` : Calcule la valeur de vérité de x1 OU x2 élément par élément.
*   `np.logical_not` : Calcule la valeur de vérité de NON x élément par élément.
*   `np.logical_xor` : Calcule la valeur de vérité de x1 XOR x2, élément par élément.

### Comparaison de tableaux

*   `np.isclose` : Retourne un tableau booléen où deux tableaux sont égaux à une tolérance près.
*   `np.allclose` : Retourne True si deux tableaux sont élément par élément égaux à une tolérance près.

### Échantillonnage aléatoire de tableaux

*   `np.random.rand` : Crée un tableau de la forme donnée et le remplit avec des échantillons aléatoires d'une distribution uniforme sur ``[0, 1)``.
*   `np.random.randn` : Retourne un échantillon (ou des échantillons) de la distribution "normale standard".
*   `np.random.randint` : Retourne des entiers aléatoires de `low` (inclus) à `high` (exclus).
*   `np.random.random` : Retourne des flottants aléatoires dans l'intervalle semi-ouvert [0.0, 1.0).
*   `np.random.choice` : Génère un échantillon aléatoire à partir d'un tableau 1-D donné.
*   `np.random.seed` : Initialise le générateur.

### Entrée et sortie de tableaux

*   `np.save` : Sauvegarde un tableau dans un fichier binaire au format NumPy ``.npy``.
*   `np.load` : Charge des tableaux ou des objets sérialisés à partir de fichiers ``.npy``, ``.npz`` ou pickle.
*   `np.savetxt` : Sauvegarde un tableau dans un fichier texte.
*   `np.loadtxt` : Charge des données à partir d'un fichier texte.
*   `np.genfromtxt` : Charge des données à partir d'un fichier texte, avec gestion des valeurs manquantes comme spécifié.

### Transformée de Fourier sur les tableaux

*   `np.fft.fft` : Calcule la transformée de Fourier discrète unidimensionnelle.
*   `np.fft.ifft` : Calcule la transformée de Fourier discrète inverse unidimensionnelle.
*   `np.fft.fft2` : Calcule la transformée de Fourier discrète 2-D.
*   `np.fft.ifft2` : Calcule la transformée de Fourier discrète inverse 2-D.
*   `np.fft.fftn` : Calcule la transformée de Fourier discrète N-dimensionnelle.
*   `np.fft.ifftn` : Calcule la transformée de Fourier discrète inverse N-dimensionnelle.

### Redimensionnement de tableaux

*   `np.reshape` : Donne une nouvelle forme à un tableau sans modifier ses données.
*   `np.ravel` : Retourne un tableau aplati contigu.

### Jointure et division de tableaux

*   `np.concatenate` : Joint une séquence de tableaux le long d'un axe existant.
*   `np.stack` : Joint une séquence de tableaux le long d'un nouvel axe.
*   `np.split` : Divise un tableau en plusieurs sous-tableaux.

### Indexation de tableaux

*   `np.where` : Retourne les éléments choisis dans `x` ou `y` en fonction de la `condition`.
*   `np.take` : Prend des éléments d'un tableau le long d'un axe.

### Type de données des tableaux

*   `np.dtype` : Crée un objet de type de données.

### Disposition mémoire des tableaux

*   `np.ascontiguousarray` : Retourne un tableau contigu en mémoire (ordre C).

### Diffusion de tableaux

*   `np.broadcast_to` : Diffuse un tableau vers une nouvelle forme.

### Opérations sur les tableaux masqués

*   `np.ma.masked_array` : Une classe de tableau avec des valeurs potentiellement masquées.

### Fonctions polynomiales sur les tableaux

*   `np.polyfit` : Ajustement polynomial par moindres carrés.
*   `np.polyval` : Évalue un polynôme à des valeurs spécifiques.

### Fonctions de fenêtrage sur les tableaux

*   `np.hanning` : Retourne la fenêtre de Hanning.

### Support des dates/heures pour les tableaux

*   `np.datetime64` : Une représentation générique de date/heure.

### Améliorations des performances des tableaux

*   `np.vectorize` : Définit une fonction vectorisée.

### Divers sur les tableaux

*   `np.copy` : Retourne une copie du tableau de l'objet donné.
*   `np.meshgrid` : Retourne des matrices de coordonnées à partir de vecteurs de coordonnées.
