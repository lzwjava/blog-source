J'ai précédemment écrit sur [Comment construire un ordinateur](./computer-en). Essayons quelque chose de différent. Aujourd'hui, j'installe Ubuntu 24.04 sur un ordinateur portable Lenovo Xiaoxing 14IAH8 qui était préinstallé avec Windows 10.

```bash
sudo dd if=ubuntu-24.04.1-desktop-amd64.iso of=/dev/rdisk6 bs=1m
```

Il est nécessaire d'aller dans les paramètres et de désactiver le chiffrement du périphérique, car l'installateur Ubuntu le nécessite.

Si le chiffrement du périphérique est activé et que le démarrage sécurisé est désactivé dans le BIOS, Windows demandera le mot de passe BitLocker pour la récupération.

Il est donc nécessaire de commencer par activer le démarrage sécurisé dans le BIOS, de se connecter à Windows, puis d'aller dans les paramètres pour désactiver le chiffrement du périphérique. Après cela, le démarrage sécurisé peut être désactivé à nouveau pour lancer l'installateur Ubuntu.
