Oui — pour une **carte de débit HSBC Hong Kong UnionPay**, le code PIN que vous utilisez pour les transactions UnionPay en magasin est généralement **le même** que le code PIN à 6 chiffres que vous utilisez pour retirer de l’argent aux distributeurs automatiques HSBC.

En effet, les achats UnionPay dans les magasins physiques nécessitent souvent un **PIN hors ligne** (le code PIN propre à la carte), et HSBC n’émet qu’un seul code PIN pour la carte de débit, utilisé à la fois pour les **retraits aux distributeurs** et les **transactions aux points de vente**.

La seule exception serait si vous aviez expressément demandé à HSBC de définir un **« code PIN d’achat » différent** (rare, et généralement pas proposé par défaut à Hong Kong).

Si vous le souhaitez, je peux également vous expliquer comment cela fonctionne différemment pour les **paiements sans contact UnionPay QuickPass** et l’**utilisation à l’étranger**.
