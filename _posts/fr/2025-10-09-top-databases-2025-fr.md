### Principales bases de données sur le marché (2025)

Selon les derniers classements de popularité de DB-Engines (qui mesurent des facteurs tels que les mentions dans les discussions techniques, les offres d'emploi, l'activité sur les réseaux sociaux et les tendances de recherche), le marché des systèmes de gestion de base de données (SGBD) en 2025 est dominé par un mélange d'options relationnelles (basées sur SQL) et NoSQL. Oracle continue de mener, mais on observe une croissance notable des bases de données cloud-native et NoSQL comme PostgreSQL et Snowflake.

Voici un résumé des 10 principaux SGBD par score de popularité :

| Rang | Base de données     | Type                               | Points clés |
|------|---------------------|------------------------------------|-------------|
| 1    | Oracle              | Relationnel (SGBDR)               | Pilier de l'entreprise ; inchangé à la 1ère place. |
| 2    | MySQL               | Relationnel (SGBDR)               | Leader open-source ; stable à la 2ème place. |
| 3    | Microsoft SQL Server | Relationnel (SGBDR)              | Intégré à l'écosystème Microsoft ; stable à la 3ème place. |
| 4    | PostgreSQL          | Relationnel (SGBDR)               | Étoile montante pour ses fonctionnalités avancées ; maintient la 4ème place. |
| 5    | MongoDB             | NoSQL (Orienté document)          | Populaire pour les applications flexibles et évolutives ; conserve la 5ème place. |
| 6    | Snowflake           | Entrepôt de données cloud         | A grimpé de la 9ème place en 2024 grâce à l'adoption du cloud. |
| 7    | Redis               | NoSQL (Clé-valeur)                | Cache en mémoire ; est passé de la 6ème place. |
