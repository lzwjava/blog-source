### Dernière mise à jour de Windows 11
La dernière mise à jour majeure de Windows 11 est la **version 24H2 (Sun Valley 3)**, également connue sous le nom de mise à jour d'octobre 2024. Elle inclut des améliorations telles qu'une gestion des tâches améliorée, un Explorateur de fichiers repensé, de meilleures fonctionnalités de jeu (par exemple, des optimisations pour l'ordonnancement du GPU et pour des jeux comme Starfield), des capacités intégrées pour les PC Copilot+, le sondage prédictif pour des transitions fluides, et des corrections de bogues pour la stabilité et la sécurité.

Cette mise à jour a été déployée progressivement :
- Les versions préliminaires ont commencé en mai 2024 via le programme Windows Insider.
- Le package d'activation est devenu disponible le 3 septembre 2024 pour les appareils éligibles (bien que certaines fonctionnalités, comme l'IA Rappel, aient connu des retards et soient encore en cours de déploiement).
- La mise à jour complète pour le public a été achevée dans le monde entier au 1er octobre 2024.

Pour vérifier votre appareil, allez dans Paramètres > Mise à jour Windows et recherchez la "Mise à jour cumulative de 2024-10" ou une version plus récente.

### Version initiale de Windows 11
Si vous vous interrogez sur la version originale de Windows 11, elle a été dévoilée le 24 juin 2021, et la build de disponibilité générale a commencé à être déployée le 5 octobre 2021, en commençant par les appareils compatibles.

Pour plus de détails, consultez les sources officielles Microsoft Learn (learn.microsoft.com/fr-fr/windows/release-health/release-information) ou la page de l'historique des mises à jour de Windows 11 (support.microsoft.com/fr-fr/windows/historique-des-mises-%C3%A0-jour-de-windows-11-781b5c23-0797-4d25-bfe2-f2bbff5b1c45).
