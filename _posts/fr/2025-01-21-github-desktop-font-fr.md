La version 3.4.13 (arm64) de GitHub Desktop présente des problèmes de rendu des polices dans le panneau de différences de texte pour certaines langues. Les langues concernées incluent :

- Chinois (simplifié et traditionnel)
- Hindi
- Japonais

Cependant, les langues suivantes s'affichent correctement :
- Anglais
- Espagnol
- Allemand
- Arabe
- Français
