### Aperçu
Microsoft Teams a introduit de nombreuses nouvelles fonctionnalités entre juillet 2023 et juillet 2025, en se concentrant sur l'amélioration de la collaboration, l'intégration de l'IA, les réunions, les événements, les capacités téléphoniques, les appareils et la sécurité. Ces mises à jour visent à améliorer la productivité, l'inclusivité et les expériences de travail hybride. Ci-dessous, j'ai regroupé les fonctionnalités principales par catégorie, en mettant en évidence les ajouts majeurs avec de brèves descriptions et des chronologies de lancement approximatives. Les fonctionnalités sont tirées d'annonces officielles, en privilégiant celles déployées ou annoncées durant la période spécifiée.

### Chat et Collaboration
- **Émojis et réactions personnalisés** : Les utilisateurs peuvent télécharger des émojis et réactions personnalisés pour personnaliser les conversations et refléter la culture d'équipe (juillet 2024).
- **Composants Loop dans les canaux** : Co-créer et synchroniser du contenu comme des tableaux, des listes et des trackers directement dans les canaux (décembre 2023).
- **Expérience de recherche améliorée** : Recherche améliorée dans le chat/les canaux avec un bouton "rechercher dans le canal" et des filtres spécifiques au domaine (par exemple, types de fichiers ou noms) (décembre 2023).
- **Liste de chats compacte et Tout marquer comme lu** : Condenser les vues de chat et effacer les notifications en un clic (juin 2023).
- **Transfert de messages** : Transférer des messages (y compris les composants Loop et les cartes d'applications) entre les chats et les canaux, avec un contexte ajouté (janvier 2025).
- **Améliorations de Copilot** : Invites basées sur le temps (par exemple, "le mois dernier") pour les recherches dans l'historique des chats (août 2024) ; résumés de fichiers dans les chats pour des insights rapides sans ouvrir les documents (début 2025).
- **Commandes slash** : Actions rapides comme ouvrir des chats dans de nouvelles fenêtres ou ajouter des blocs de code via "/" dans la zone de composition (mai 2024).
- **Blocs de code dans Loop** : Partager et co-éditer des extraits de code dans les chats/canaux (juin 2024).
- **Cartes de canal** : Survolez les noms de canaux pour obtenir des instantanés des descriptions, membres et paramètres (août 2024).
- **Nouvelle expérience de Chat et Canaux** : Vue unifiée pour les chats/canaux, sections personnalisées, filtres ; déployée vers GCC (juin 2025).
- **Photos de profil de chat de groupe** : Images personnalisées ou émojis pour les chats de groupe (avril 2024).
- **@Nearby dans le Chat** : Se connecter avec des collègues physiquement proches (avril 2025).
- **Intégration Storyline** : Les dirigeants partagent des mises à jour dans Teams, simplifiant les communications avec les employés (début 2025).

### Réunions, Webinaires et Town Halls
- **Audio spatial** : Positionne les voix dans la vue galerie pour un audio immersif (juin 2023).
- **Avatars et réactions** : Avatars personnalisés avec de nouvelles réactions et limites personnelles dans les espaces immersifs (juillet 2024).
- **Notes collaboratives** : Co-édition en temps réel des ordres du jour/notes/tâches via Loop, synchronisation avec To Do/Planner (octobre 2023) ; désormais disponible dans l'Éducation et GCC-High/DoD (juin 2025).
- **Récapitulatif de réunion intelligent** : Résumés, notes, tâches générés par IA ; étendu aux appels/Rencontre maintenant (décembre 2024) ; support multilingue (début 2025).
- **Fonctionnalités Town Hall** : Nouveau modèle pour les grands événements (jusqu'à 20 000 participants), salle d'attente verte, RTMP-in, réactions en direct, export Q&A (octobre 2023) ; DVR (pause/rembobinage) pour le streaming (mars 2025) ; support eDiscovery (janvier 2025).
- **Améliorations des webinaires** : Inscription personnalisée, bios/thématique des présentateurs, reporting des participants (janvier 2023) ; désormais disponible dans GCC (septembre 2024).
- **Copilot dans les réunions** : S'appuie sur la transcription et le chat pour les réponses ; conversations multi-tours (avril 2024) ; contrôles pour désactiver (septembre 2024).
- **Salles de sous-commission** : Mélange des participants, déplacement automatique entre les salles (juillet 2023, janvier 2025).
- **Isolation vocale et flou de portrait** : Suppression du bruit par IA et effets d'arrière-plan (avril 2024).
- **Événements Mesh** : Modèles (par exemple, Meadow/Oasis), liens de revisite, réactions multi-salles (juillet 2024) ; audio pour le partage d'écran (janvier 2025).
- **Transcription multilingue** : Prend en charge 51 langues parlées/31 langues de traduction (début 2025).
- **Édition du nom d'affichage** : Changer les noms dans les réunions (avril 2025).

### Teams Phone et Appels
- **Copilot dans Teams Phone** : IA pour les appels non planifiés, prise de notes, coaching en temps réel (juillet 2023).
- **Teams Phone Mobile** : Étendu à la Norvège (décembre 2023).
- **Sonnerie simultanée différée** : Configure le destinataire principal avant de notifier les autres (février 2025).
- **Gestion du Direct Routing** : Télécharger/gérer les numéros de téléphone dans le centre d'administration (février 2025).
- **Améliorations de l'application Files d'attente** : Fonctions d'intrusion/chuchotement/surveillance/reprise pour les superviseurs (mars 2025).
- **Extensibilité pour les centres de contact** : Intégration avec Dynamics 365 et ISV via Azure (avril 2025).
- **Accès partagé à la ligne** : Partager les lignes téléphoniques avec des groupes sans délégation (début 2025).
- **Protection contre l'usurpation de marque** : Détecte le hameçonnage dans les appels externes (novembre 2024).

### Workplace, Places et Teams Rooms
- **Présence Workplace** : Met à jour automatiquement la localisation vers "bureau" via les connexions de bureau ou Wi-Fi ; voir les collègues à proximité (septembre 2024, début 2025).
- **Bureaux réservables** : Réservation automatique via connexion moniteur ; découvrabilité/analytiques (juin 2024, début 2024).
- **Rejoindre par proximité ultrasonore** : Détecter les salles à proximité pour un rejoindre facile (septembre 2024).
- **Reconnaissance de l'orateur** : Attribue les voix dans les transcriptions/récapitulatifs pour tous les micros (juillet 2024).
- **Vues multi-caméras** : Jusqu'à quatre flux dans les salles (janvier 2025).
- **Assistant IA dans Pro Management** : Requêtes pour des insights sur les salles/la gestion (juin 2025).
- **Redimensionnement du texte/UI** : Ajuster l'échelle d'affichage pour une meilleure lisibilité (août 2024).
- **Salle en tant que présentateur** : Rejoindre les webinaires/town halls depuis la console de salle (février 2025).

### Appareils et Matériel
- **Nouveaux appareils certifiés** : Ajouts réguliers comme le Lenovo ThinkSmart Core, Yealink MVC860, Poly G7500 (juin 2023) ; Owl Bar, Yealink Room Panel E2 (février 2025, décembre 2024).
- **Boutique d'appareils dans l'app Teams** : Découvrir/acheter des casques, téléphones, Rooms dans Teams (avril 2024).
- **Mises à jour automatiques pour les appareils Android** : Mises à jour d'applications transparentes (décembre 2023).
- **Améliorations BYOD** : Réservations par QR code, mode d'affichage partagé, enceintes intelligentes (novembre 2023).

### IA, Copilot et Agents
- **Agents dans Teams** : Ajouter des agents/bots IA dans les chats/réunions ; support mobile (mai 2025).
- **Composition Copilot** : Ajuster les messages brouillons avec des invites personnalisées (juin 2024).
- **Mémoire agentique et protocoles** : Les agents se souviennent des interactions, communiquent de pair à pair (mai 2025).
- **Copilot dans le Centre d'administration** : Insights sur les politiques, dépannage (début 2025).
- **Analytique de l'espace avec Copilot** : Interroger les données d'utilisation des salles (novembre 2024).

### Fondamentaux, Sécurité et Administration
- **Nouvelle application Teams** : Application plus rapide, plus légère avec support VDI/gouvernement (octobre 2023).
- **Optimisation VDI** : Performances améliorées, codecs, arrière-plans (juin 2024, avril 2025).
- **Connexion multi-comptes** : À travers les clouds sans changer de compte (avril 2024).
- **Barrières d'information dans les réunions** : Tenir des réunions avec des politiques conflictuelles via un modérateur (avril 2025).
- **Notifications SMS pour Bookings** : Pour tous les rendez-vous (novembre 2024).
- **Améliorations eCDN** : Zéro configuration, support GCC (novembre 2024).

### Solutions pour les travailleurs de première ligne
- **Mises à jour de l'application Shifts** : Gestion des équipes plus facile, notifications (en cours, par exemple, mai 2025 Build).
- **Application Files d'attente pour la première ligne** : Surveiller/coacher les agents (mars 2025).

Ces mises à jour reflètent l'accent de Microsoft sur la productivité pilotée par l'IA et le travail hybride. Pour le statut complet du déploiement, consultez la Feuille de route Microsoft 365.
