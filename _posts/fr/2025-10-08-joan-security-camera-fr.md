Shenzhen JOOAN Technology Co., Ltd. (également connue sous le nom de Qiao An Technology) est spécialisée dans les produits de sécurité pour maison intelligente, incluant une gamme de caméras IP sans fil qui s'intègrent à leur plateforme cloud, souvent appelée QA Cloud ou JOOAN Cloud, pour la surveillance à distance, le stockage et l'accès P2P via des applications comme Joolink ou CAM720.

### Principaux produits de caméras IP
Leurs caméras IP sont conçues pour la surveillance domestique et extérieure, avec des fonctionnalités comme la connectivité WiFi/4G, la détection de mouvement, la vision nocturne, l'audio bidirectionnel et un stockage cloud optionnel pour les clips d'événements et le streaming en direct. Les modèles populaires incluent :

- **Caméra IP Cloud JA-366 HD 720p** : Un modèle sans fil intérieur/extérieur avec une résolution de 1280x720, zoom pan-tilt (PTZ), vision nocturne jusqu'à 10m et support cloud intégré pour la visualisation à distance sur applications mobiles. Prend en charge le stockage local sur carte SD ou les forfaits cloud payants.

- **Caméra de sécurité extérieure Q3R-U** : Résolution 2K (3MP), étanchéité IP66, PTZ 360°, suivi automatique, vision nocturne en couleur et options à double objectif pour des vues panoramiques. S'intègre au cloud pour les alertes de mouvement et l'enregistrement 24h/24.

- **Caméra WiFi A2R-U** : Modèle d'intérieur à branchement avec HD 1080p, rotation à 360°, détection humaine et interphone. Compatible cloud pour une configuration facile via l'application, avec des options pour le stockage vidéo chiffré.

- **Caméra 4G solaire JA-G4** : Caméra IP extérieure alimentée par batterie/énergie solaire (5MP), ne nécessite pas de WiFi — utilise une carte SIM 4G pour un accès à distance dans les zones sans réseau. Prend en charge le stockage cloud pour la sauvegarde des séquences.

Ces caméras sont abordables (souvent entre 20 et 60 $ sur les sites de vente au détail), compatibles avec Alexa/Google Assistant pour certains modèles, et mettent l'accent sur une configuration facile via l'application sans câblage complexe. Les services cloud sont optionnels, avec des essais gratuits et des forfaits payants pour un stockage étendu (par exemple, 7 à 30 jours).

Pour plus de détails, consultez leur site officiel ou les boutiques d'applications.
