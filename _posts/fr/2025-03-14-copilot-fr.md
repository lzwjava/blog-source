Pour utiliser GitHub Copilot dans Visual Studio Code (VSCode), suivez ces étapes :

### 1. Installer l'extension Copilot
- Ouvrez VSCode sur votre ordinateur.
- Accédez à la **vue Extensions** :
  - Cliquez sur l'icône Extensions dans la Barre d'activité sur le côté gauche de la fenêtre, ou
  - Appuyez sur `Ctrl+Shift+X` (ou `Cmd+Shift+X` sur macOS).
- Dans la barre de recherche, tapez **"GitHub Copilot"**.
- Trouvez l'extension officielle **GitHub Copilot** dans les résultats et cliquez sur **Installer**.

### 2. Se connecter à GitHub
- Après avoir installé l'extension, vous devrez vous authentifier avec votre compte GitHub, car Copilot est un produit GitHub.
- Vous verrez peut-être une invite de connexion automatique après l'installation. Sinon :
  - Ouvrez la **Palette de commandes** en appuyant sur `Ctrl+Shift+P` (ou `Cmd+Shift+P` sur macOS).
  - Tapez et sélectionnez **"GitHub Copilot: Sign in"**.
- Suivez les instructions à l'écran pour vous connecter à votre compte GitHub. Cela peut impliquer d'autoriser VSCode via une fenêtre de navigateur.

### 3. Commencer à utiliser Copilot
- Une fois connecté, Copilot s'activera et commencera à vous assister pendant que vous codez.
- **Suggestions de code** : Lorsque vous tapez, Copilot suggérera des complétions de code ou des lignes entières. Pour accepter une suggestion, appuyez sur la touche `Tab`.
- **Chat Inline** : Pour une aide supplémentaire, comme poser des questions sur votre code ou demander une assistance spécifique :
  - Appuyez sur `Ctrl+I` pour ouvrir la fonctionnalité Chat Inline directement dans votre éditeur.
