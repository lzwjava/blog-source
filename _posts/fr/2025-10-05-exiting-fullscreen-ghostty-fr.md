Pour quitter le mode plein écran dans le terminal Ghostty sur Ubuntu, appuyez sur **Ctrl + Entrée**. Il s'agit du raccourci par défaut pour activer et désactiver le mode plein écran.

Si cela ne fonctionne pas (par exemple, en raison d'une configuration personnalisée), essayez ces alternatives :
- Faites un clic droit à l'intérieur de la fenêtre du terminal et recherchez une option "Quitter le mode plein écran" ou similaire dans le menu contextuel.
- Utilisez la barre de menus si elle est visible : allez dans **Affichage > Quitter le mode plein écran** (vous devrez peut-être survoler le haut de l'écran pour la faire apparaître).
- En dernier recours, fermez et rouvrez Ghostty avec la commande `ghostty` dans un autre terminal, ou tuez le processus via `pkill ghostty` et relancez-le.
