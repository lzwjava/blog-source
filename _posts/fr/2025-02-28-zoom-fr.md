### Utiliser Zoom Efficacement

Zoom est un outil puissant pour les réunions virtuelles, et l'utiliser efficacement peut améliorer la communication et la productivité. Commencez par tester vos paramètres audio et vidéo avant les réunions pour garantir la clarté. Coupez votre microphone lorsque vous ne parlez pas pour minimiser le bruit de fond, et positionnez votre caméra au niveau des yeux pour une meilleure interaction. Pour les réunions plus importantes, envisagez d'utiliser des fonctionnalités comme les sondages ou les salles de sous-commission pour maintenir l'engagement des participants.

### Utilisation Avancée et Fonctionnalités Spécifiques

Zoom offre plusieurs fonctionnalités avancées qui peuvent améliorer votre expérience de réunion :

- **Enregistrement des réunions :** Vous pouvez enregistrer les réunions localement sur votre ordinateur ou dans le cloud si vous avez un plan payant. Pour configurer, allez dans `Préférences` > `Enregistrement` et choisissez votre emplacement de stockage. Pendant la réunion, cliquez sur "Enregistrer" pour démarrer, et arrêtez lorsque vous avez terminé. Pour une meilleure qualité, activez la vidéo HD et envisagez d'enregistrer des fichiers audio séparés pour chaque participant, ce qui est idéal pour le montage ultérieur. Accédez aux enregistrements dans le dossier spécifié pour les sauvegardes locales ou via le portail web Zoom pour les enregistrements cloud.

- **Sous-titres et transcription en direct :** Les sous-titres améliorent l'accessibilité, en particulier pour les participants malentendants. En tant qu'hôte, activez les sous-titres dans `Paramètres` > `Réunion` > `Dans la réunion (Avancé)`. Pendant la réunion, cliquez sur "Sous-titres" pour démarrer, et les participants peuvent les activer ou les désactiver. Zoom propose des sous-titres automatisés via l'IA, qui peuvent contenir certaines inexactitudes, ou vous pouvez utiliser des sous-titres manuels avec un transcripteur désigné. La transcription en direct prend en charge l'anglais et peut être sauvegardée sous forme de fichier texte pour référence.

- **Inviter des participants :** Invitez d'autres personnes en démarrant une réunion et en cliquant sur "Inviter" pour copier l'URL ou envoyer par e-mail. Pour les réunions planifiées, utilisez votre calendrier pour envoyer des invitations avec des options comme les mots de passe de réunion pour la sécurité. Pendant une réunion, ajoutez plus de participants en cliquant sur "Participants" puis "Inviter". Utilisez les salles d'attente pour contrôler qui rejoint, améliorant ainsi la sécurité de la réunion.

- **Gérer les notifications :** Personnalisez les notifications pour éviter les distractions. Allez sur le portail web Zoom, cliquez sur "Paramètres" > "Notification par e-mail" pour ajuster les alertes pour les enregistrements de réunion ou les rappels. Dans l'application de bureau, sous `Préférences` > `Notifications`, définissez des heures de ne pas déranger ou désactivez les alertes dans l'application pour les messages de chat, garantissant ainsi la concentration pendant les réunions.

#### Enregistrement des Réunions dans Zoom

L'enregistrement des réunions est une fonctionnalité clé pour capturer les discussions, et Zoom propose des options locales et cloud. Pour configurer, les utilisateurs peuvent naviguer vers l'application de bureau Zoom, sélectionner `Préférences`, et aller dans l'onglet `Enregistrement`. Ici, choisissez l'emplacement de stockage pour les enregistrements locaux ou activez l'enregistrement cloud si vous avez un plan payant (par exemple, Pro, Business ou Enterprise, avec des limites de stockage cloud variables). Le processus consiste à cliquer sur "Enregistrer" pendant la réunion, qui peut être arrêté à tout moment, et les fichiers sont traités après la réunion.

Pour une utilisation avancée, envisagez d'optimiser les paramètres pour la qualité. Activez "Enregistrer un fichier audio séparé pour chaque participant" pour faciliter le post-montage, comme souligné dans un guide de FilmKraft ([Guide d'Enregistrement Zoom](https://www.filmkraft.tv/guides/record-zoom-meetings)). Cette fonctionnalité est particulièrement utile pour les interviews ou les panels, permettant des pistes audio individuelles. De plus, activez la vidéo HD dans l'onglet `Vidéo` pour une résolution potentielle de 720p, bien que la vitesse Internet impacte la qualité. Le guide recommande d'utiliser un équipement externe comme les webcams Logitech C920x Pro ou les microphones Blue Yeti pour de meilleurs résultats, et de garantir un bon éclairage (par exemple, près d'une fenêtre) et une connexion Internet stable (par exemple, ethernet plutôt que le WiFi).

L'accès aux enregistrements varie selon la méthode : les fichiers locaux sont sauvegardés dans le dossier choisi, tandis que les enregistrements cloud apparaissent dans le portail web Zoom sous l'onglet "Enregistré". Le partage peut se faire par e-mail ou en accordant l'accès aux liens cloud, facilitant la distribution du contenu aux participants absents.

| **Étape** | **Détails**                                                                                     |
|----------|-------------------------------------------------------------------------------------------------|
| 1        | **Configurer les paramètres d'enregistrement** : Téléchargez la dernière version de Zoom, allez dans Préférences > Enregistrement. Sélectionnez la destination, cochez "Enregistrer un fichier audio séparé pour chaque participant", optimisez pour un logiciel de montage vidéo tiers. |
| 2        | **Vérifier les paramètres vidéo** : Activez la HD dans l'onglet Vidéo, envisagez "Améliorer mon apparence" (optionnel, peut estomper les tons de peau, recommandé désactivé pour le montage). |
| 3        | **Paramètres audio** : Dans l'onglet Audio > Avancé, activez "Son d'origine" et "Mode musique haute fidélité" pour un meilleur audio (48Khz, 96Kbps mono/192kbps stéréo, nécessite une connexion Internet rapide). |
| 4        | **Base de l'enregistrement** : Cliquez sur "Enregistrer" pour démarrer, épinglez la vidéo d'un participant pour un enregistrement ciblé, les fichiers sont générés après la réunion. |

#### Utilisation des Sous-titres dans Zoom

Les sous-titres améliorent l'accessibilité, cruciale pour des réunions inclusives, et Zoom propose des options automatisées et manuelles. Pour activer, les hôtes se rendent sur le portail web Zoom, sélectionnent `Paramètres` > `Réunion` > `Dans la réunion (Avancé)`, et activent "Sous-titrage" et "Activer le service de transcription en direct". Pendant la réunion, cliquez sur "Sous-titres" dans la barre d'outils pour activer, et les participants peuvent les activer en sélectionnant "Afficher les sous-titres", avec des options de personnalisation (par exemple, texte blanc sur fond noir, déplaçable en bas).

Les sous-titres automatisés, alimentés par l'IA, sont pratiques mais peuvent contenir des inexactitudes, comme noté dans un guide du Disabled Students Program à l'UCSB ([Sous-titres dans Zoom](https://dsp.sa.ucsb.edu/faculty-resources/closed-captions-zoom)). Pour une meilleure précision, des sous-titres manuels peuvent être assignés à un participant ou utiliser un service tiers. La transcription en direct, prenant en charge uniquement l'anglais, peut être visualisée en entier en sélectionnant "Voir la transcription complète" et sauvegardée sous forme de fichier .txt, utile pour les comptes-rendus de réunion.

| **Activer les sous-titres** (Hôte)     | Sélectionnez Plus > Sous-titres, le bouton "Afficher les sous-titres" apparaît.                                       | Les participants peuvent activer/désactiver les sous-titres/transcription en direct.              |
| **Activer les sous-titres** (Participants) | Sélectionnez Afficher les sous-titres, confirmez/choisissez la langue, sélectionnez Enregistrer.                                  | Les sous-titres apparaissent en bas, déplaçables. Par défaut : texte blanc, fond noir. Modifiable via Paramètres des sous-titres. |
| **Désactiver les sous-titres** (Participants) | Sélectionnez Masquer les sous-titres.                                                                        | Désactive uniquement pour l'utilisateur qui l'a sélectionné.                              |
| **Utiliser la transcription en direct**     | Sélectionnez le caret à côté de Masquer/Afficher les sous-titres > Voir la transcription complète. La transcription apparaît, sauvegardez en .txt. | Prend en charge l'anglais uniquement ; une parole claire est recommandée pour la précision.             |

Cette fonctionnalité est particulièrement précieuse pour les contextes éducatifs, comme on le voit dans un guide de l'Université DePaul ([Utiliser les sous-titres dans une réunion Zoom](https://resources.depaul.edu/teaching-commons/teaching-guides/technology/zoom/Pages/captions.aspx)), mettant l'accent sur la conception universelle pour l'inclusion.

#### Inviter des Participants à une Réunion Zoom

Inviter des participants est simple, avec des options pour les réunions instantanées et planifiées. Pour inviter pendant une réunion instantanée, cliquez sur "Inviter" pour copier l'URL de la réunion ou utilisez les services de messagerie intégrés à Zoom, tels qu'Outlook ou Gmail, comme détaillé dans un guide de l'Université de Pittsburgh ([Réunions Zoom : Étiquette et Bonnes Pratiques](https://www.technology.pitt.edu/blog/zoom-tips)). Pour les réunions planifiées, utilisez le bouton "Planifier", entrez les détails, et sélectionnez les types de calendrier (par exemple, Google Calendar), en envoyant des invitations avec des mots de passe de réunion pour la sécurité.

L'utilisation avancée comprend la gestion de l'accès des participants avec des salles d'attente, en restreignant les rejoignements aux participants invités uniquement, et en assignant des rôles (par exemple, co-hôtes). Un guide de Texas Law Help ([Zoom sur Bureau : Comment ajouter ou inviter une autre personne à la réunion](https://texaslawhelp.org/article/zoom-on-desktop-how-to-add-or-invite-another-person-to-the-meeting)) note le partage du lien d'invitation et s'assurer que les participants ont le code d'accès de la réunion, améliorant la sécurité pour les réunions sensibles.

#### Gestion des Notifications dans Zoom

Les notifications peuvent être distrayantes, mais Zoom permet une personnalisation adaptée aux préférences de l'utilisateur. Les notifications par e-mail, telles que les alertes pour l'achèvement d'un enregistrement cloud, peuvent être ajustées dans le portail web Zoom sous "Paramètres" > "Notification par e-mail", comme décrit dans un guide de l'Université du Delaware ([Ajuster les Paramètres de Notification E-mail de Zoom](https://ats.udel.edu/conferencing/zoom/email-notifications/)). Les options incluent la désactivation des notifications pour les enregistrements supprimés afin de réduire l'encombrement.

Les notifications dans l'application, comme les alertes de messages de chat, peuvent être gérées dans l'application de bureau sous `Préférences` > `Notifications`. Les utilisateurs peuvent définir des heures de ne pas déranger ou désactiver des alertes spécifiques, garantissant ainsi la concentration pendant les réunions. Une discussion communautaire sur le forum de la communauté Zoom ([Notifications Bureau Zoom - COMMENT LES DÉSACTIVER](https://community.zoom.com/t5/Zoom-Meetings/Zoom-Desktop-Notifications-HOW-TO-TURN-OFF/m-p/107226)) met en lumière les frustrations des utilisateurs avec les notifications contextuelles, suggérant la personnalisation comme solution.

Cette approche détaillée garantit que les utilisateurs peuvent adapter Zoom à leurs besoins, améliorant la productivité et réduisant les distractions dans les réunions virtuelles.
