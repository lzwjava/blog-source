Les raccourcis clavier principaux pour ajuster le volume sur YouTube sont les touches **Flèche vers le haut** et **Flèche vers le bas**.

* Appuyer sur la touche **Flèche vers le haut** **augmente** le volume de 5 %.
* Appuyer sur la touche **Flèche vers le bas** **diminue** le volume de 5 %.

**Note importante :** Pour que ces raccourcis fonctionnent, assurez-vous que le lecteur vidéo YouTube a le focus. Vous devrez peut-être cliquer une fois sur la vidéo avant d'utiliser les touches fléchées pour ajuster le volume. Si le lecteur vidéo n'a pas le focus, les touches fléchées feront probablement défiler la page web vers le haut ou vers le bas.
