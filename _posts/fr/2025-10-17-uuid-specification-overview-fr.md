Le RFC 4122 est une spécification technique publiée par l'Internet Engineering Task Force (IETF) en juillet 2005. Il définit un espace de noms Uniform Resource Name (URN) pour les Universally Unique Identifiers (UUID), qui sont également connus sous le nom de Globally Unique Identifiers (GUID).

Un UUID est un nombre de 128 bits utilisé pour identifier de manière unique des informations dans les systèmes informatiques sans nécessiter une autorité centrale pour gérer les identifiants. La norme décrit cinq versions d'UUID (par exemple, la version 1 basée sur un horodatage et une adresse MAC, la version 4 basée sur des bits aléatoires), leurs méthodes de génération, leurs formats d'encodage (comme la chaîne hexadécimale courante avec des traits d'union : `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`), et des directives pour leur utilisation dans les protocoles et les applications. Cela garantit l'interopérabilité entre les différents systèmes et réseaux.

Cette spécification a été largement adoptée dans le développement de logiciels pour des tâches telles que les clés de base de données, les ID de session et les systèmes distribués.
