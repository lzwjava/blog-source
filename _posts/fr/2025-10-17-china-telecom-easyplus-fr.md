### Activation des forfaits de données quotidiens sur la carte prépayée China Telecom Easy+ (Macao)

La carte SIM prépayée China Telecom Easy+ (également appelée carte SIM China Macau Telecom) prend en charge des options d'ajout de données quotidiennes à prix abordable pour une utilisation à Macao, en Chine continentale, à Hong Kong et parfois en itinérance étendue. Elles sont activées via des commandes SMS envoyées au code court **1888**. La carte fournit un numéro de téléphone local de Macao (+853) et est conçue pour les voyageurs, avec une activation automatique lors de l'insertion (assurez-vous que votre téléphone est déverrouillé et compatible avec le réseau de China Telecom, généralement 4G/LTE).

#### Guide étape par étape pour activer les forfaits
1.  **Insérez la SIM** : Allumez votre téléphone avec la SIM insérée. Il devrait se connecter automatiquement au réseau China Telecom Macao. Sinon, sélectionnez manuellement "China Telecom" dans les paramètres réseau de votre téléphone.
2.  **Vérifiez le solde et la validité** : Composez *122# ou #321# (appuyez sur appeler/envoyer) pour afficher votre solde restant, votre date d'expiration et les bonus actifs. La carte commence généralement avec une valeur nominale (par exemple, 100 MOP) et une validité de 180 jours, prolongeable par recharge.
3.  **Activez un forfait de données quotidien** :
    - Pour le **forfait de 2 MOP/jour** (allocation de données quotidienne de base, adaptée à une utilisation légère comme la messagerie et la navigation) :
        - Envoyez un SMS avec le texte exact **602** au **1888**.
        - Déduction : 2 MOP de votre solde. Valable 24 heures à partir de l'activation.
    - Pour le **forfait de 3 MOP/jour** (données quotidiennes améliorées, mieux adaptées pour le streaming modéré ou la navigation) :
        - Envoyez un SMS avec le texte exact **903** au **1888**.
        - Déduction : 3 MOP de votre solde. Valable 24 heures à partir de l'activation.

    Vous recevrez un SMS de confirmation du 1888 une fois activé. L'utilisation des données est partagée entre Macao, la Chine continentale et Hong Kong (partenaires d'itinérance : CTM à Macao, China Mobile/Unicom en Chine continentale, CSL à HK). La vitesse peut être réduite après une utilisation élevée (par exemple, après 1 à 2 Go).

4.  **Rechargez le solde** : Si le crédit est faible, rechargez via les distributeurs automatiques, les boutiques CTM ou des applications comme CTM Buddy. La recharge minimale est généralement de 10 à 50 MOP, ce qui prolonge la validité de 180 jours.

#### Changer ou basculer entre les forfaits quotidiens (Utilisation de 3025)
Pour basculer entre les forfaits de 2 MOP et 3 MOP (ou annuler le forfait actuel) :
- D'abord, **désactivez le forfait existant** : Envoyez un SMS avec le texte **XDATA** au **1888** pour arrêter le forfait de données quotidien actuel. Cela évite les frais qui se chevauchent. Les données restantes (le cas échéant) ne sont pas remboursées mais peuvent être utilisées jusqu'à l'expiration.
- Ensuite, activez le nouveau forfait en utilisant les étapes ci-dessus (602 pour 2 MOP ou 903 pour 3 MOP).
- Note sur **3025** : Sur la base de la documentation disponible, ce code semble être un code hérité ou spécifique pour la gestion des forfaits dans certaines configurations de China Telecom Macao, potentiellement pour interroger/changer les options de forfait via USSD (composez **3025#** et suivez les invites pour sélectionner 2 MOP ou 3 MOP). Si cela ne fonctionne pas, contactez le service client en composant le 1888 (gratuit depuis la SIM) ou le 10000 (hotline China Telecom). Alternativement, utilisez l'application CTM Buddy pour changer de forfait en libre-service.

#### Conseils supplémentaires
-   **Tarifs voix et SMS** : Appels/SMS locaux à Macao : ~0,40 MOP/min et 0,25 MOP/SMS. SMS entrants gratuits. Pour les appels vers la Chine continentale/HK, préfixez avec 050 pour bénéficier de tarifs IDD réduits (0,70 MOP/min en heures creuses).
-   **Désactivation des données** : Envoyez toujours "XDATA" au 1888 si vous souhaitez désactiver les données en itinérance pour éviter des frais accidentels.
-   **Compatibilité** : Fonctionne avec la plupart des téléphones GSM/4G déverrouillés. Activez l'itinérance des données dans les paramètres.
-   **Expiration** : Les forfaits se renouvellent automatiquement chaque jour si le solde est suffisant ; surveillez via *122#.
-   En cas de problème, visitez un magasin CTM à Macao ou composez le 1888 pour obtenir de l'aide (Mandarin/Cantonais/Anglais disponibles).

Cette configuration maintient les coûts bas pour les courts séjours—par exemple, 2 MOP/jour pour l'essentiel.
