Pour demander un visa de voyage pour Singapour en tant que citoyen chinois à Guangzhou, suivez ces étapes organisées :

### 1. **Soumission via un Agent Agréé**
   - Les visas pour Singapour des citoyens chinois doivent être traités par **des agents agréés** (les demandes individuelles à l'ambassade ne sont pas acceptées).
   - **Trouver un Agent** : Consultez le site web de l'Autorité de l'Immigration et des Points de Contrôle de Singapour (ICA) ou contactez l'Ambassade de Singapour en Chine pour obtenir une liste des agents agréés à Guangzhou. Par exemple, des agences de voyage comme Ctrip (携程) ou China Travel Service (中旅), mais vérifiez leur statut d'autorisation actuel.

### 2. **Documents Requis**
   - **Passeport** : Original + copie (valide ≥6 mois, avec ≥2 pages vierges).
   - **Formulaire de Demande de Visa** : Formulaire 14A dûment rempli (téléchargeable sur le site de l'ICA) avec une photo d'identité récente (35mm x 45mm, fond blanc).
   - **Preuve d'Emploi/Études** :
     - Salarié : Lettre de l'entreprise (poste, salaire, approbation du congé).
     - Étudiant : Preuve d'inscription + lettre de congé de l'école.
     - Travailleur indépendant / Freelance : Licence commerciale ou relevés financiers.
   - **Preuve Financière** : Relevés bancaires (3-6 derniers mois).
   - **Itinéraire de Voyage** : Réservations de vol (billet aller-retour) + réservation d'hôtel.
   - **Documents Supplémentaires** :
     - Lettre d'invitation (si vous visitez de la famille/des amis) avec une copie de la carte d'identité/passeport de l'invitant à Singapour.
     - Pour les mineurs : Acte de naissance + lettre de consentement parental.

### 3. **Processus de Soumission**
   - Soumettez les documents à l'agent agréé **en personne** ou **en ligne** (si pris en charge par l'agence).
   - **Délai de Traitement** : Généralement 3-5 jours ouvrables (postulez au moins 2 semaines avant le voyage).

### 4. **Frais**
   - Frais de visa : 30 SGD (~¥150) + frais de service de l'agent (variables ; à confirmer avec l'agence).

### 5. **Réception du e-Visa**
   - Une fois approuvé, le e-visa sera envoyé par e-mail. **Imprimez une copie** à présenter pendant le voyage.

### 6. **Points Clés à Noter**
   - **Validité** : Généralement 30 jours (entrée simple/multiple ; la durée est déterminée par l'ICA).
   - **Pas d'Entrée Sans Visa** : Les citoyens chinois doivent obtenir un visa avant de voyager.
   - **Vérifier les Détails** : Assurez-vous que toutes les informations correspondent à votre passeport pour éviter tout problème à l'entrée.

### 7. **Après l'Approvation**
   - Suivez votre demande via le portail de l'agent. Pour les demandes urgentes, contactez directement l'agent.

### Ressources
   - ICA de Singapour : [https://www.ica.gov.sg](https://www.ica.gov.sg)
   - Ambassade de Singapour en Chine : Contactez via le site web pour les listes d'agents.

Postulez tôt, vérifiez attentivement vos documents et confirmez la légitimité de l'agent via les canaux officiels. Bon voyage !
