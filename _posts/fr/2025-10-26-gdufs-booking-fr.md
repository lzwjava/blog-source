### Comment réserver l'examen pratique sur ordinateur pour le cours 610201 (Technologie d'application informatique) de l'Examen Supérieur d'Éducation en Autodidacte (ESEA) à l'Université des Études Étrangères du Guangdong (GDUFS)

L'évaluation pratique (实践考核) pour le cours 610201 Technologie d'Application Informatique (niveau diplôme, maintenant mis à jour avec le code 510201) est un examen obligatoire sur ordinateur (上机考试), généralement avec le code de cours **11325 : Expérimentation du cours d'Informatique et d'Application (I)**. Il s'agit d'un test sur machine réalisé dans un laboratoire informatique. Il n'y a pas de "rendez-vous" individuel pour un créneau horaire spécifique comme chez un médecin—vous vous inscrivez pendant une période fixe, et l'université vous assigne votre place en laboratoire et votre session d'examen (généralement plusieurs sessions le jour de l'examen). L'examen a lieu à des dates désignées sur le campus Baiyun de la GDUFS (Guangzhou).

**Notes importantes :**
- **Éligibilité** : Vous devez avoir réussi au moins **2 cours théoriques (sur papier)** dans la majeure 610201 avant de vous inscrire à cet examen pratique. Vérifiez vos cours réussis dans le système ESEA du Guangdong.
- **Format de l'examen** : Combine un test écrit (le cas échéant) et des opérations sur machine. La durée est typiquement de 2 à 3 heures par session. Apportez une pièce d'identité ; aucune entrée n'est autorisée après l'heure de début.
- **Frais** : 150 RMB par personne (non remboursables ; payés en ligne lors de l'inscription).
- **Statut actuel (au 26 octobre 2025)** : L'examen du second semestre 2025 est prévu les **1er et 2 novembre 2025**. Les inscriptions sont closes depuis juin 2025. Si vous êtes inscrit, consultez votre place attribuée dès aujourd'hui (voir l'Étape 6 ci-dessous). Pour la prochaine session (premier semestre 2026), les inscriptions ouvrent en décembre 2025—vérifiez les dates exactes sur le site officiel.
- **Aucune aide tierce** : La GDUFS ne collabore pas avec des agences externes pour l'inscription ou les cours de préparation. Utilisez uniquement les canaux officiels pour éviter les arnaques.

#### Processus d'inscription étape par étape (Général ; Suivez la notification officielle pour les dates exactes)
1. **Vérifier la notification officielle** : Visitez le site web du Collège d'Éducation Continue de la GDUFS ([jxjy.gdufs.edu.cn](https://jxjy.gdufs.edu.cn/)). Allez dans "自考服务 > 自考办通知" (Services ESEA > Notifications du bureau). Cherchez la dernière "实践考核课程报考通知" (Notification d'inscription à l'évaluation pratique) pour votre semestre (par ex., "2025年下半年" pour le second semestre). Téléchargez la pièce jointe pour la liste des cours, les délais et le tableau d'éligibilité. Confirmez que le 11325 est proposé (généralement uniquement au second semestre).

2. **Préparer les documents** :
   - Identifiant et mot de passe candidat ESEA du Guangdong (inscrivez-vous sur [www.eeagd.edu.cn/selfec/](https://www.eeagd.edu.cn/selfec/) si vous êtes nouveau).
   - Preuve des cours réussis (téléchargez la transcription depuis le système ESEA).
   - Photo scannée de la pièce d'identité et autres justificatifs si requis (par ex., pour des exemptions comme un certificat NCRE Niveau 1 pour les bases).

3. **Accéder au système d'inscription** :
   - Pendant la période d'ouverture (par ex., ~10-23 juin pour le second semestre ; ~10-23 décembre pour le premier semestre), connectez-vous sur le site de la GDUFS.
   - Cliquez sur le lien vers "实践考核报考系统" (Système d'inscription à l'évaluation pratique). Utilisez vos identifiants ESEA.

4. **Remplir le formulaire** :
   - Entrez les détails personnels (nom, pièce d'identité, coordonnées—doit correspondre aux dossiers ESEA).
   - Téléversez/ajoutez la preuve d'au moins 2 cours théoriques réussis.
   - Sélectionnez votre majeure (610201 Technologie d'Application Informatique) et le cours (11325).
   - Vérifiez et confirmez—aucune modification après soumission.

5. **Payer les frais** :
   - Payez 150 RMB en ligne via WeChat/Alipay/carte bancaire (généralement dans les 1-2 jours suivant l'ouverture des inscriptions).
   - Conservez le reçu de paiement.

6. **Confirmer l'affectation et les détails de l'examen** :
   - ~1 semaine avant l'examen (par ex., le 26 octobre pour l'examen de novembre), vérifiez sur le site la "考场座位号通知" (Notification du numéro de place).
   - Téléchargez la pièce jointe ou interrogez le système avec votre identifiant candidat pour obtenir votre laboratoire/salle, votre place et l'horaire de la session (par ex., matin/après-midi le 1er nov.).
   - Arrivez 30 à 60 minutes à l'avance au campus Baiyun (adresse : No. 2 Baiyun North Avenue, Guangzhou). Respectez les règles d'entrée (par ex., code santé, pas d'appareils électroniques dans le labo).

#### Conseils de préparation à l'examen
- Reveuillez le programme/plan de l'examen sur le site de la GDUFS sous "实践考核 > 考试大纲" (téléchargez-le pour le 11325).
- Pratiquez les compétences informatiques de base (par ex., Word, Excel, bases de la programmation) conformément au plan de la majeure.
- Si vous ratez l'inscription, attendez le semestre suivant—aucune inscription sur place n'est possible.
- Résultats : Publiés ~1 mois après l'examen ; consultez-les sur les sites GDUFS/ESEA.

Si vous êtes un nouveau candidat ou avez besoin d'aide pour l'inscription globale à l'ESEA, commencez par la pré-inscription sur le site provincial. Pour les questions urgentes (par ex., vérification de la place aujourd'hui), contactez le Bureau ESEA de la GDUFS : +86-20-36206168 ou email jxjy@gdufs.edu.cn.
