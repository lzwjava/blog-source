#### Explication et Prononciation Mot par Mot
**Couplet 1 :**
- **見上げた (Miageta)**
  - **Prononciation** : mee-ah-geh-tah
  - **Signification** : "Levai les yeux" (forme passée de 見上げる, miageru, "regarder vers le haut").
  - **Décomposition** : 見 (mi, "voir") + 上げた (ageta, "levé" au passé).
- **空 (Sora)**
  - **Prononciation** : soh-rah
  - **Signification** : "Ciel".
  - **Note** : Un nom commun en japonais, évoquant souvent l'immensité ou la liberté.
- **に (Ni)**
  - **Prononciation** : nee
  - **Signification** : Particule indiquant le lieu ou la direction ("dans" ou "à").
  - **Utilisation** : Relie "le ciel" à l'action de lever les yeux.
- **浮かんでる (Ukanderu)**
  - **Prononciation** : oo-kahn-deh-roo
  - **Signification** : "Flottent" (forme informelle de 浮かんでいる, ukandeiru).
  - **Décomposition** : 浮 (ukan, "flotter") + でる (deru, auxiliaire indiquant une action en cours).
- **雲 (Kumo)**
  - **Prononciation** : koo-moh
  - **Signification** : "Nuages".
  - **Note** : Symbolise souvent l'impermanence dans la poésie japonaise.
- **が (Ga)**
  - **Prononciation** : gah
  - **Signification** : Particule marquant le sujet ("les nuages" dans ce cas).

**Traduction du Couplet 1** : "Les nuages qui flottent dans le ciel vers lequel j'ai levé les yeux."

**Refrain :**
- **未来 (Mirai)**
  - **Prononciation** : mee-rah-ee
  - **Signification** : "Futur".
  - **Décomposition** : 未 (mi, "pas encore") + 来 (rai, "venir"), suggérant quelque chose qui n'est pas encore arrivé.
- **へ (E)**
  - **Prononciation** : eh
  - **Signification** : Particule indiquant la direction ("vers").
  - **Utilisation** : S'associe à "mirai" pour signifier "vers le futur".
- **行こう (Ikou)**
  - **Prononciation** : ee-koh
  - **Signification** : "Allons" (forme volitive de 行く, iku, "aller").
  - **Note** : Suggère une invitation ou une détermination.

**Traduction du Refrain** : "Allons vers le futur, vers le futur, vers le futur."

Cette invite aide les apprenants à décomposer les paroles en parties gérables, à comprendre le vocabulaire, à pratiquer la prononciation et à saisir les nuances grammaticales.
