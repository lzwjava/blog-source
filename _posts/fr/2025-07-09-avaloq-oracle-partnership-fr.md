Oui, Avaloq entretient bien une relation significative avec la technologie Oracle. D'après les résultats de recherche, voici les principales connexions :

La solution Avaloq Core Banking fonctionne sur Oracle Exadata, qui est la plateforme de base de données haute performance d'Oracle. Avaloq a été membre Gold du Oracle PartnerNetwork (OPN) et a été nommé OPN Specialized Database Partner of the Year pour la région EMEA en 2011, ce qui indique un partenariat technique approfondi.

De plus, Avaloq a sélectionné Oracle Cloud Infrastructure (OCI) Dedicated Region pour implanter une région cloud OCI complète dans son propre centre de données, démontrant son engagement envers l'infrastructure cloud d'Oracle.

La plateforme Avaloq semble également être disponible via le marketplace cloud d'Oracle, ce qui suggère qu'elle est conçue pour s'intégrer à la pile technologique d'Oracle. Cependant, bien qu'Avaloq ait une forte intégration avec Oracle et fonctionne sur la technologie de base de données Oracle (notamment Exadata), elle n'est pas exclusivement « construite sur Oracle » - il s'agit d'une plateforme logicielle bancaire complète qui utilise Oracle comme l'une de ses technologies sous-jacentes clés, notamment pour les besoins en base de données et en infrastructure.
