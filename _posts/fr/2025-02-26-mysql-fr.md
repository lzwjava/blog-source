Voici un guide étape par étape pour utiliser MySQL, un système de gestion de bases de données relationnelles open source populaire :

---

### 1. **Installation**
- **Windows** : Téléchargez le programme d'installation de MySQL sur [mysql.com/downloads](https://dev.mysql.com/downloads/installer/)
- **macOS** :
  ```bash
  brew install mysql
  brew services start mysql
  ```
- **Linux** (Debian/Ubuntu) :
  ```bash
  sudo apt update
  sudo apt install mysql-server mysql-client
  sudo systemctl start mysql
  sudo mysql_secure_installation  # Exécuter la configuration de sécurité
  ```

---

### 2. **Connexion à MySQL**
- **Connexion root par défaut** :
  ```bash
  mysql -u root -p  # Saisir le mot de passe lorsqu'il est demandé
  ```
- **Connexion à un serveur distant** :
  ```bash
  mysql -u username -p -h hostname -P 3306
  ```

---

### 3. **Opérations de base de données de base**
- **Créer un utilisateur** :
  ```sql
  CREATE USER 'myuser'@'localhost' IDENTIFIED BY 'mypassword';
  ```
- **Créer une base de données** :
  ```sql
  CREATE DATABASE mydb;
  GRANT ALL PRIVILEGES ON mydb.* TO 'myuser'@'localhost';
  FLUSH PRIVILEGES;
  ```
- **Commandes SQL de base** :
  ```sql
  -- Créer une table
  CREATE TABLE users (
      id INT AUTO_INCREMENT PRIMARY KEY,
      name VARCHAR(50),
      email VARCHAR(100) UNIQUE
  );

  -- Insérer des données
  INSERT INTO users (name, email) VALUES ('Bob', 'bob@example.com');

  -- Interroger les données
  SELECT * FROM users WHERE name LIKE 'B%';

  -- Mettre à jour les données
  UPDATE users SET email = 'new@email.com' WHERE id = 1;

  -- Supprimer des données
  DELETE FROM users WHERE id = 2;
  ```

---

### 4. **Fonctionnalités avancées**
- **Jointures** :
  ```sql
  SELECT orders.id, users.name 
  FROM orders
  JOIN users ON orders.user_id = users.id;
  ```
- **Procédures stockées** :
  ```sql
  DELIMITER //
  CREATE PROCEDURE GetUsers()
  BEGIN
    SELECT * FROM users;
  END //
  DELIMITER ;
  
  CALL GetUsers();
  ```
- **Support JSON** (MySQL 5.7+) :
  ```sql
  CREATE TABLE products (
      id INT PRIMARY KEY,
      details JSON
  );
  ```

---

### 5. **Maintenance**
- **Sauvegarde** :
  ```bash
  mysqldump -u root -p mydb > backup.sql
  ```
- **Restauration** :
  ```bash
  mysql -u root -p mydb < backup.sql
  ```
- **Optimiser les tables** :
  ```sql
  ANALYZE TABLE users;
  OPTIMIZE TABLE users;
  ```

---

### 6. **Sécurité**
- **Créer un utilisateur en lecture seule** :
  ```sql
  CREATE USER 'reader'@'localhost' IDENTIFIED BY 'readpass';
  GRANT SELECT ON mydb.* TO 'reader'@'localhost';
  FLUSH PRIVILEGES;
  ```

---

### 7. **Commandes MySQL utiles**
- Lister les bases de données : `SHOW DATABASES;`
- Utiliser une base de données : `USE mydb;`
- Lister les tables : `SHOW TABLES;`
- Décrire une table : `DESCRIBE users;`
- Afficher les utilisateurs : `SELECT user FROM mysql.user;`
- Quitter MySQL : `EXIT;` ou `QUIT;`

---

### 8. **Outils d'interface graphique**
- **MySQL Workbench** : Outil d'interface graphique officiel

**Principales différences avec PostgreSQL** :
1. Utilise `AUTO_INCREMENT` au lieu de `SERIAL`
2. Noms de tables non sensibles à la casse par défaut
3. Utilise `mysqldump` au lieu de `pg_dump`
4. Port par défaut : **3306** (contre 5432 pour PostgreSQL)
5. Système d'authentification et gestion des utilisateurs différents

**Conseil** : Sauvegardez toujours les bases de données avant des modifications majeures. Utilisez les transactions (`START TRANSACTION`/`COMMIT`) pour les opérations critiques.
