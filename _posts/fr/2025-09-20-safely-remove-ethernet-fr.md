Sur l'image, vous tenez ce qui semble être un **connecteur RJ45 (également appelé prise Ethernet ou tête de cristal)**. Les petites languettes sur les côtés sont conçues pour verrouiller le connecteur en place lorsqu'il est inséré dans un port Ethernet ou un appareil.

**Vous pouvez appuyer sur les languettes des deux côtés et tirer doucement sur le câble.**
Cela libérera le connecteur en toute sécurité sans endommager la prise ni le port.

Si vous essayez de démonter le connecteur (par exemple, pour re-sertir les fils), vous aurez besoin d'un outil de sertissage spécial pour câbles réseau. Le simple fait d'appuyer sur les languettes ne vous permettra pas de démonter le connecteur lui-même.

N'hésitez pas si vous avez besoin d'aide supplémentaire !
