Les principaux concurrents d'Oracle varient selon les domaines de produits, mais voici les plus importants :

**Gestion de base de données :**
- Microsoft SQL Server
- IBM DB2
- MySQL (ironiquement, Oracle le possède mais il concurrence ses offres commerciales)
- PostgreSQL
- Amazon RDS et Aurora
- MongoDB

**Infrastructure cloud :**
- Amazon Web Services (AWS) - le leader du marché
- Microsoft Azure
- Google Cloud Platform
- IBM Cloud

**Logiciels d'entreprise :**
- SAP (concurrent majeur dans les ERP et applications métier)
- Microsoft (Dynamics, Office 365)
- Salesforce (CRM et applications cloud)
- Workday (gestion des RH et financière)

**Java et outils de développement :**
- Microsoft (.NET framework et Visual Studio)
- IBM (WebSphere et outils de développement)
- Diverses alternatives open-source

AWS est probablement la plus grande menace concurrentielle globale pour Oracle puisqu'ils dominent le cloud computing, tandis que SAP reste leur principal rival dans les applications d'entreprise. Microsoft est en concurrence sur plusieurs lignes de produits d'Oracle, ce qui en fait un autre concurrent majeur.

Le paysage concurrentiel a considérablement évolué vers les entreprises axées sur le cloud comme AWS et Microsoft, exerçant une pression sur le modèle traditionnel de logiciels on-premises d'Oracle.
