Le classement des langages de programmation populaires peut varier selon différentes métriques (par exemple, la demande d'emploi, le nombre de développeurs, les dépôts, etc.), mais sur la base des tendances actuelles de sources comme Stack Overflow, GitHub et l'indice TIOBE, voici quelques-uns des langages de programmation les plus populaires :

1. **Python**  
   Python continue de mener en raison de sa polyvalence, sa facilité d'utilisation et son adoption généralisée dans des domaines comme la data science, l'IA, le développement web et l'automatisation.

2. **JavaScript**  
   Colonne vertébrale du développement web, JavaScript reste très demandé, notamment pour le développement front-end (React, Angular, Vue) et back-end (Node.js).

3. **Java**  
   Java est encore largement utilisé dans les applications d'entreprise, le développement Android et les systèmes backend.

4. **C/C++**  
   Ces langages sont des incontournables en programmation système, développement de jeux, systèmes embarqués et applications nécessitant des performances critiques.

5. **C#**  
   Populaire dans le développement de jeux (Unity), les applications d'entreprise et le développement web (ASP.NET).

6. **TypeScript**  
   En tant que sur-ensemble de JavaScript, les fonctionnalités de sécurité de type de TypeScript le rendent de plus en plus populaire, surtout dans les applications web à grande échelle.

7. **PHP**  
   Bien qu'il soit en déclin dans certains domaines, PHP reste un acteur majeur du développement web, en particulier dans les systèmes hérités et les CMS comme WordPress.

8. **Ruby**  
   Ruby, notamment avec le framework Ruby on Rails, est largement utilisé dans le développement web.

9. **Swift**  
   Swift est le langage principal d'Apple pour le développement d'applications iOS et macOS et continue de gagner en popularité.

10. **Go (Golang)**  
    Go gagne du terrain pour les systèmes backend, l'infrastructure cloud et les microservices grâce à sa simplicité et ses performances.

11. **Kotlin**  
    Kotlin est le langage privilégié pour le développement Android, car il est moderne, expressif et entièrement interopérable avec Java.

12. **Rust**  
    Rust gagne en popularité pour sa sécurité mémoire et ses performances, ce qui en fait un choix privilégié pour la programmation système.

13. **R**  
    R reste populaire pour le calcul statistique, l'analyse de données et la recherche académique, en particulier en data science.

14. **SQL**  
    SQL continue d'être vital pour la gestion et l'interrogation de bases de données, et sa pertinence reste constante.

Ces classements peuvent changer rapidement, surtout avec l'émergence de nouvelles technologies, il est donc toujours bon de garder un œil sur les tendances actuelles !
