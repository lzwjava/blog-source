Voici une analyse détaillée, mot par mot, de 30 phrases japonaises courantes utilisées dans la vie quotidienne. Chaque phrase est analysée en séparant chaque mot (ou particule) individuel et en expliquant sa signification et son rôle dans la phrase. ### **Salutations et présentations**

1.  **Konnichiwa (こんにちは) - Bonjour**
    - **Konnichiwa (こんにちは)** : Un seul mot signifiant "Bonjour". Il est utilisé comme salutation générale pour la journée.

2.  **Ohayou gozaimasu (おはようございます) - Bonjour (matin)**
    - **Ohayou (おはよう)** : "Matin" (salutation informelle).
    - **Gozaimasu (ございます)** : Un suffixe poli qui ajoute de la formalité et du respect, se traduisant souvent par "il y a" ou "être" sous une forme polie.
    - Ensemble : "Bonjour" (formel).

3.  **Konbanwa (こんばんは) - Bonsoir**
    - **Konban (こんばん)** : "Ce soir".
    - **Wa (は)** : Une particule marquant le thème de la phrase ("ce soir" dans ce cas).
    - Ensemble : "Bonsoir".

4.  **Namae wa nan desu ka? (名前は何ですか？) - Comment vous appelez-vous ?**
    - **Namae (名前)** : "Nom".
    - **Wa (は)** : Marqueur de thème, indiquant que "nom" est le sujet de la question.
    - **Nan (何)** : "Quoi".
    - **Desu (です)** : Une copule signifiant "est".
    - **Ka (か)** : Une particule interrogative qui transforme la phrase en question.
    - Ensemble : "Quel est votre nom ?"

5.  **Watashi no namae wa [nom] desu (私の名前は[nom]です) - Je m'appelle [nom]**
    - **Watashi (私)** : "Je" ou "moi".
    - **No (の)** : Une particule possessive, signifiant "de" ou indiquant la possession (comme l'apostrophe 's' en anglais).
    - **Namae (名前)** : "Nom".
    - **Wa (は)** : Marqueur de thème, indiquant que "mon nom" est le sujet.
    - **[Nom]** : Espace réservé pour votre vrai nom.
    - **Desu (です)** : "Est".
    - Ensemble : "Mon nom est [nom]."

6.  **O-genki desu ka? (お元気ですか？) - Comment allez-vous ?**
    - **O (お)** : Un préfixe poli ajouté par respect.
    - **Genki (元気)** : "Santé" ou "bien-être".
    - **Desu (です)** : "Est".
    - **Ka (か)** : Particule interrogative.
    - Ensemble : "Êtes-vous en bonne santé ?"

7.  **Genki desu (元気です) - Je vais bien**
    - **Genki (元気)** : "Bien" ou "en forme".
    - **Desu (です)** : "Suis".
    - Ensemble : "Je vais bien."

8.  **Hajimemashite (初めまして) - Enchanté(e)**
    - **Hajimemashite (初めまして)** : Une expression figée signifiant "Enchanté(e)" ou "Pour la première fois". Elle est utilisée lorsque l'on rencontre quelqu'un pour la première fois.

### **Politesse et remerciements**

9.  **Arigatou gozaimasu (ありがとうございます) - Merci**
    - **Arigatou (ありがとう)** : "Merci" (informel).
    - **Gozaimasu (ございます)** : Suffixe poli ajoutant de la formalité.
    - Ensemble : "Merci" (formel).

10. **Douitashimashite (どういたしまして) - De rien**
    - **Dou (どう)** : "Comment" ou "de quelle manière".
    - **Itashimashite (いたしまして)** : Forme polie de "faire", utilisée ici comme partie d'une expression figée signifiant "Je l'ai fait" ou "ce n'est rien".
    - Ensemble : "De rien" (une réponse polie figée).

11. **Sumimasen (すみません) - Excusez-moi / Je suis désolé(e)**
    - **Sumimasen (すみません)** : Un seul mot utilisé pour s'excuser ("Je suis désolé(e)") ou attirer l'attention ("Excusez-moi").

### **Poser des questions**

12. **Doko desu ka? (どこですか？) - Où est-ce ?**
    - **Doko (どこ)** : "Où".
    - **Desu (です)** : "Est".
    - **Ka (か)** : Particule interrogative.
    - Ensemble : "Où est-ce ?"

13. **Itsu desu ka? (いつですか？) - Quand est-ce ?**
    - **Itsu (いつ)** : "Quand".
    - **Desu (です)** : "Est".
    - **Ka (か)** : Particule interrogative.
    - Ensemble : "Quand est-ce ?"

14. **Ikura desu ka? (いくらですか？) - Combien ça coûte ?**
    - **Ikura (いくら)** : "Combien".
    - **Desu (です)** : "Est".
    - **Ka (か)** : Particule interrogative.
    - Ensemble : "Combien est-ce ?"

15. **Ima nanji desu ka? (今何時ですか？) - Quelle heure est-il ?**
    - **Ima (今)** : "Maintenant".
    - **Nanji (何時)** : "Quelle heure".
    - **Desu (です)** : "Est".
    - **Ka (か)** : Particule interrogative.
    - Ensemble : "Quelle heure est-il maintenant ?"

16. **Toire wa doko desu ka? (トイレはどこですか？) - Où sont les toilettes ?**
    - **Toire (トイレ)** : "Toilettes" (emprunté à l'anglais).
    - **Wa (は)** : Marqueur de thème, indiquant que "toilettes" est le sujet.
    - **Doko (どこ)** : "Où".
    - **Desu (です)** : "Est".
    - **Ka (か)** : Particule interrogative.
    - Ensemble : "Où sont les toilettes ?"

### **Réponses et compréhension**

17. **Hai (はい) - Oui**
    - **Hai (はい)** : "Oui".

18. **Iie (いいえ) - Non**
    - **Iie (いいえ)** : "Non".

19. **Wakarimasen (わかりません) - Je ne comprends pas**
    - **Wakarimasen (わかりません)** : Forme polie négative de "wakaru" (わかる), signifiant "comprendre". Se traduit par "Je ne comprends pas."

20. **Mou ichido kudasai (もう一度ください) - Pouvez-vous répéter, s'il vous plaît ?**
    - **Mou (もう)** : "Encore".
    - **Ichido (一度)** : "Une fois".
    - **Kudasai (ください)** : "S'il vous plaît" (demande polie).
    - Ensemble : "Une fois encore, s'il vous plaît."

### **Activités quotidiennes et repas**

21. **Itadakimasu (いただきます) - Bon appétit (avant de manger)**
    - **Itadakimasu (いただきます)** : Forme polie de "itadaku" (いただく), signifiant "recevoir". Se dit avant de manger pour exprimer sa gratitude pour la nourriture.

22. **Gochisousama deshita (ごちそうさまでした) - C'était délicieux (après le repas)**
    - **Gochisousama (ごちそうさま)** : "Merci pour le repas" (expression figée).
    - **Deshita (でした)** : Passé de "desu" (です), signifiant "était".
    - Ensemble : "C'était un festin" ou "Merci pour le repas".

23. **Menu o kudasai (メニューをください) - La carte, s'il vous plaît**
    - **Menu (メニュー)** : "Menu" (emprunté à l'anglais).
    - **O (を)** : Marqueur d'objet, indiquant que "menu" est l'objet de la demande.
    - **Kudasai (ください)** : "Donnez-moi, s'il vous plaît".
    - Ensemble : "Donnez-moi le menu, s'il vous plaît."

24. **O-kaikei o onegaishimasu (お会計をお願いします) - L'addition, s'il vous plaît**
    - **O (お)** : Préfixe poli.
    - **Kaikei (会計)** : "Addition" ou "note".
    - **O (を)** : Marqueur d'objet.
    - **Onegaishimasu (お願いします)** : "S'il vous plaît" (plus formel que "kudasai").
    - Ensemble : "Je vous demande l'addition, s'il vous plaît."

### **Social et sécurité**

25. **Omedetou gozaimasu (おめでとうございます) - Félicitations**
    - **Omedetou (おめでとう)** : "Félicitations".
    - **Gozaimasu (ございます)** : Suffixe poli.
    - Ensemble : "Félicitations" (formel).

26. **Ki o tsukete (気を付けて) - Faites attention**
    - **Ki (気)** : "Esprit" ou "attention".
    - **O (を)** : Marqueur d'objet.
    - **Tsukete (付けて)** : Forme en -te de "tsukeru" (付ける), signifiant "faire attention à" ou "attacher".
    - Ensemble : "Prenez soin de vous" ou "Faites attention".

27. **Oyasumi nasai (おやすみなさい) - Bonne nuit**
    - **Oyasumi (おやすみ)** : "Repos" ou "sommeil".
    - **Nasai (なさい)** : Forme impérative polie, signifiant "faites".
    - Ensemble : "Reposez-vous bien" ou "Bonne nuit".

28. **Tasukete! (助けて！) - À l'aide !**
    - **Tasukete (助けて)** : Forme en -te de "tasukeru" (助ける), signifiant "aider". Utilisée ici comme un ordre : "Aidez-moi !"

29. **Keisatsu o yonde kudasai! (警察を呼んでください！) - Appelez la police, s'il vous plaît !**
    - **Keisatsu (警察)** : "Police".
    - **O (を)** : Marqueur d'objet.
    - **Yonde (呼んで)** : Forme en -te de "yobu" (呼ぶ), signifiant "appeler".
    - **Kudasai (ください)** : "S'il vous plaît".
    - Ensemble : "Appelez la police, s'il vous plaît !"

30. **Kyuu kyuusha o yonde kudasai! (救急車を呼んでください！) - Appelez une ambulance, s'il vous plaît !**
    - **Kyuu kyuusha (救急車)** : "Ambulance".
    - **O (を)** : Marqueur d'objet.
    - **Yonde (呼んで)** : Forme en -te de "yobu" (呼ぶ), signifiant "appeler".
    - **Kudasai (ください)** : "S'il vous plaît".
    - Ensemble : "Appelez une ambulance, s'il vous plaît !"
