Oui, les machines virtuelles Azure prennent entièrement en charge l’exécution de systèmes d’exploitation Linux. Vous pouvez déployer un large éventail de distributions populaires directement depuis la Place de marché Azure ou télécharger vos propres images Linux personnalisées.

### Points clés :
- **Distributions prises en charge** : Azure approuve et prend en charge les principales variantes de Linux, notamment Ubuntu, Red Hat Enterprise Linux (RHEL), Debian, SUSE Linux Enterprise Server (SLES), Oracle Linux, CentOS Stream, Rocky Linux, AlmaLinux, et plus encore. Plus de 100 images Linux sont disponibles immédiatement.
- **Options de déploiement** : Utilisez le portail Azure, CLI, PowerShell ou les modèles ARM pour créer des machines virtuelles Linux. Par exemple, vous pouvez rapidement déployer une instance Ubuntu Server en quelques minutes.
- **Fonctionnalités** : Les machines virtuelles Linux sur Azure incluent une intégration complète avec les services Azure comme le stockage, la mise en réseau, la sécurité et la surveillance. Elles prennent également en charge des outils comme Azure Arc pour la gestion hybride.

Si vous débutez, consultez le guide de démarrage rapide officiel pour créer une machine virtuelle Linux.
