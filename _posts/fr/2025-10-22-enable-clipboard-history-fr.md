### Activation de l'historique du presse-papiers dans Windows
Avant d'utiliser les raccourcis, assurez-vous que la fonctionnalité est activée (elle est disponible dans Windows 10 version 1809 et ultérieures, y compris Windows 11) :
1. Ouvrez **Paramètres** (appuyez sur `Touche Windows + I`).
2. Allez dans **Système > Presse-papiers**.
3. Activez **Historique du presse-papiers**.

Cela permet à Windows de stocker jusqu'à 25 éléments récents (texte, HTML, images) pendant 1 heure par défaut, ou plus longtemps s'ils sont synchronisés avec un compte Microsoft.

### Utilisation des raccourcis pour accéder et lire l'historique du presse-papiers
Le raccourci clavier principal pour ouvrir et lire votre historique du presse-papiers est **`Touche Windows + V`**. Voici comment l'utiliser étape par étape :

1. **Copier plusieurs éléments** : Utilisez `Ctrl + C` pour copier du texte, des images ou d'autres contenus selon vos besoins. Windows les enregistrera automatiquement dans l'historique.

2. **Ouvrir l'historique du presse-papiers** : Appuyez sur `Touche Windows + V`. Cela ouvre un volet consultable sur le côté droit de votre écran affichant des miniatures ou des aperçus des clips récents (le plus récent en haut).

3. **Naviguer et lire** :
   - Utilisez les **touches de direction** (↑/↓) pour faire défiler la liste.
   - **Tapez pour rechercher** : Lorsque le volet s'ouvre, vous pouvez immédiatement commencer à taper pour filtrer les éléments (par exemple, rechercher un mot-clé dans le texte copié).
   - **Aperçu** : Survolez ou naviguez avec les flèches vers un élément pour voir un aperçu sans le coller.

4. **Coller à partir de l'historique** :
   - Sélectionnez un élément avec les touches de direction et appuyez sur **Entrée** pour le coller à l'emplacement de votre curseur.
   - Ou cliquez sur l'élément si vous utilisez la souris.

5. **Fermer le volet** : Appuyez sur `Échap` ou `Touche Windows + V` à nouveau.

### Conseils supplémentaires
- **Épingler des éléments** : Dans le volet, cliquez sur l'icône d'épingle à côté d'un élément pour le conserver indéfiniment.
- **Effacer l'historique** : Clic droit sur un élément > **Supprimer**, ou allez dans Paramètres > Système > Presse-papiers > **Effacer**.
- **Synchroniser entre les appareils** : Activez **Synchronisation de l'historique du presse-papiers** dans Paramètres pour y accéder sur d'autres PC Windows connectés avec le même compte Microsoft.
- Si vous souhaitez personnaliser le raccourci (par exemple, le réaffecter), utilisez des outils tiers comme AutoHotkey, mais le raccourci par défaut `Win + V` est intégré et fiable.

Pour plus de détails, consultez le guide officiel de Microsoft : [Clipboard History for Productivity](https://www.microsoft.com/en-us/windows/tips/clipboard-history).
