Oui, les souris verticales comme la B.O.W M190U Pro incluent généralement un bouton retour/avant (souvent le bouton supérieur sur le côté gauche pour la navigation dans le navigateur).

D'après la conception de la B.O.W M190U Pro (une souris verticale ergonomique filaire de Hangshi/B.O.W, similaire à leur série MD190), les deux boutons supérieurs sur le côté gauche sont les boutons latéraux programmables. Celui du haut fonctionne par défaut comme le bouton **retour** (par exemple, pour revenir en arrière dans les navigateurs web ou les applications), et celui en dessous fonctionne comme le bouton **avant**. Ils peuvent souvent être reconfigurés via les paramètres Windows/macOS ou un logiciel tiers si nécessaire.

Le côté droit gère le clic gauche principal (index) et le clic droit (majeur), comme sur une souris traditionnelle, tandis que la molette de défilement se situe entre eux pour un accès facile avec le pouce.
