Dézipper des fichiers en Python :

```python
import zipfile
import os
import argparse

# Configurer l'analyse des arguments
parser = argparse.ArgumentParser(description="Dézipper un fichier dans le même répertoire avec le même nom.")
parser.add_argument('zip_file', help="Chemin vers le fichier zip")

# Analyser les arguments de ligne de commande
args = parser.parse_args()

# Obtenir le chemin du fichier zip et son répertoire
zip_file_path = args.zip_file
directory = os.path.dirname(zip_file_path)

# Supprimer l'extension .zip pour obtenir le nouveau nom de répertoire/dossier
folder_name = os.path.splitext(os.path.basename(zip_file_path))[0]
destination_folder = os.path.join(directory, folder_name)

# S'assurer que le dossier de destination existe
os.makedirs(destination_folder, exist_ok=True)

# Dézipper le fichier
with zipfile.ZipFile(zip_file_path, 'r') as zip_ref:
    zip_ref.extractall(destination_folder)

print(f'Contenu extrait vers {destination_folder}')

```
