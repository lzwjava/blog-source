Pour obtenir un visa de voyage australien en tant que citoyen chinois à Guangzhou, voici ce que vous devrez faire :

1. Déterminer le bon type de visa - généralement le visa de visiteur (sous-classe 600) pour le tourisme ou les visites d'affaires

2. Postuler en ligne via le site web du Département australien des Affaires intérieures ou via le Centre de demande de visa australien (AVAC) à Guangzhou :
   - Adresse : 5F, Yuexiu NEO Tower B, 238 Zhongshan Road, District de Yuexiu

3. Préparer ces documents :
   - Passeport valide
   - Formulaire de demande rempli
   - Photographies récentes format passeport
   - Justificatif de fonds suffisants (relevés bancaires)
   - Itinéraire de voyage
   - Détails de l'hébergement
   - Lettre expliquant le but de votre visite
   - Preuve de liens avec la Chine (lettre d'emploi, titre de propriété)

4. Payer les frais de demande de visa (environ 1 000 RMB)

5. Assister à un rendez-vous biométrique si requis

6. Attendre le traitement - généralement 15 à 30 jours, bien que les délais de traitement puissent varier

Il est conseillé de postuler au moins 6 à 8 semaines avant votre date de voyage prévue. Envisagez d'utiliser un agent de migration enregistré si vous avez besoin d'aide avec le processus de demande.

Le Consulat général d'Australie à Guangzhou peut fournir des informations supplémentaires :
