### APIs Provided

1. **NVIDIA CUDA:**
   - NVIDIA GPUs in the ZBook Studio 16 G11 support CUDA, a parallel computing platform and API model created by NVIDIA. CUDA allows developers to use a C-like language to code algorithms for execution on NVIDIA GPUs, significantly speeding up computations in applications like image processing, deep learning, and scientific simulations.

2. **DirectX and OpenGL:**
   - These are standard APIs for handling tasks related to multimedia, especially game programming and video. DirectX is more commonly used in Windows environments, while OpenGL is cross-platform.

3. **Vulkan:**
   - Vulkan is a low-overhead, cross-platform 3D graphics and computing API. It provides high-efficiency, cross-platform access to modern GPUs used in a wide variety of devices from PCs and consoles to mobile phones and embedded platforms.

4. **NVIDIA OptiX:**
   - OptiX is an application framework for achieving optimal ray tracing performance on the GPU. It is used for rendering high-quality images by simulating the physical behavior of light.

5. **NVIDIA TensorRT:**
   - TensorRT is an SDK for high-performance deep learning inference. It includes a deep learning inference optimizer and runtime that delivers low latency and high throughput for deep learning inference applications.
