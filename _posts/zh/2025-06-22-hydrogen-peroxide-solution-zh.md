要计算将300毫升7.5%过氧化氢与300毫升水混合后的最终浓度，可以使用稀释公式：

**C₁V₁ = C₂V₂**

其中：
- **C₁** = 初始浓度（7.5% 或 0.075）
- **V₁** = 初始体积（300 mL）
- **C₂** = 最终浓度（待求值）
- **V₂** = 最终体积（300 mL + 300 mL = 600 mL）

整理公式求 **C₂**：
**C₂ = (C₁V₁) / V₂**

代入数值：
**C₂ = (7.5% × 300 mL) / 600 mL = (0.075 × 300) / 600 = 0.0375 或 3.75%**

因此，过氧化氢的最终浓度为 **3.75%**。

### 溶液搅拌
是的，您应当搅拌溶液以确保过氧化氢与水充分混合。搅拌有助于更快实现浓度均一。可使用干净的勺子、搅拌棒或任何非反应性器具。由于过氧化氢易溶于水，通常轻柔旋晃或搅拌数秒即可。若容器密封请避免剧烈摇晃，因为过氧化氢可能释放氧气，导致压力积聚。

一瓶开封后静置的500毫升3%过氧化氢溶液会因分解随时间失效，但分解速率取决于多种因素。过氧化氢（H₂O₂）会自然分解为水和氧气，尤其在接触空气、光照、热量或污染物时更易发生。具体分析如下：

- **空气接触**：敞口放置会加速氧气释放，促进分解。敞口容器使过氧化氢与空气接触，可能引入灰尘或微生物等催化分解的杂质。
- **时间范围**：3%浓度的过氧化氢相较于高浓度溶液更稳定，但敞口存放可能在数天至数周内显著失效。研究表明，即使在密封条件下，室温存放的3%过氧化氢每年也会损失约1-2%的活性。敞口容器降解更快——若暴露于空气和光照下，可能数日内就会失效。
- **环境条件**：若置于阴凉避光处，其分解速度会慢于暴露在阳光或高温下的情况。尤其是紫外线和较高温度会催化分解过程。

**实际影响**：
- 对于消毒或伤口清洁等用途，若未被污染，溶液在短期内（如一至两天）可能仍具部分效果，但活性会持续下降。存放一周以上后，其效力可能无法满足可靠使用需求。
- 为保持有效性，建议将过氧化氢储存于密封避光容器中，并置于阴凉处。

**使用建议**：使用后请立即盖紧瓶盖并妥善储存。若长时间敞口存放（如数天），对于消毒等关键用途，建议通过测试（例如观察接触有机物是否产生气泡）或直接更换新溶液。

过氧化氢（H₂O₂）通过强氧化作用杀灭细菌，其产生的活性氧簇（ROS）会破坏细菌的蛋白质、DNA及细胞膜等结构。当接触细菌时，过氧化氢会被细菌体内的过氧化氢酶等酶类分解为水（H₂O）和氧气（O₂）。这种分解过程会随时间消耗过氧化氢，从而降低其杀菌效力。

是的，过氧化氢在反应后部分转化为氧气（O₂）和水。其反应式为：

2H₂O₂ → 2H₂O + O₂

这意味着当它杀灭细菌或与有机物反应后，会失去抗菌活性，转化为无害的水和氧气。这种活性衰减的速率取决于浓度、细菌酶活性及环境条件等因素。
