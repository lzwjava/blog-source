免费的 GitHub 用户可以创建**无限量的私有仓库**，但在**协作者**数量上存在限制。

* 您可以拥有任意数量的私有仓库
* 每个私有仓库最多可添加**3名协作者**（即您邀请共同协作的人员）

如需更多协作者名额，您需要升级至付费方案（GitHub Pro、Team 或 Enterprise）。
