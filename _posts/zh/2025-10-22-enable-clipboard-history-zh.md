### 在 Windows 中启用剪贴板历史记录
使用快捷键前，请确保已启用此功能（适用于 Windows 10 版本 1809 及更高版本，包括 Windows 11）：
1. 打开**设置**（按 `Windows 键 + I`）。
2. 进入**系统 > 剪贴板**。
3. 开启**剪贴板历史记录**。

这将允许 Windows 默认存储最多 25 个近期项目（文本、HTML、图像）1 小时，若同步至 Microsoft 账户则可保存更长时间。

### 使用快捷键访问和读取剪贴板历史记录
打开并读取剪贴板历史记录的主要键盘快捷键是 **`Windows 键 + V`**。以下是分步使用指南：

1. **复制多项内容**：使用 `Ctrl + C` 复制所需文本、图像或其他内容。Windows 会自动将其保存至历史记录。

2. **打开剪贴板历史记录**：按下 `Windows 键 + V`。此时屏幕右侧将出现可搜索面板，显示近期剪贴项目的缩略图或预览（最新项位于顶部）。

3. **导航与读取**：
   - 使用**方向键**（↑/↓）滚动列表。
   - **输入搜索**：面板打开后可直接输入文字筛选项目（例如搜索已复制文本中的关键词）。
   - **预览功能**：将鼠标悬停或用方向键选中项目即可预览内容，无需直接粘贴。

4. **从历史记录粘贴**：
   - 使用方向键选中项目后按 **Enter** 键，即可在光标处粘贴。
   - 或直接鼠标点击项目粘贴。

5. **关闭面板**：按 `Esc` 或再次按下 `Windows 键 + V`。

### 实用技巧
- **固定项目**：在面板中点击项目旁的图钉图标可永久保存。
- **清除记录**：右键点击项目选择**删除**，或前往设置 > 系统 > 剪贴板 > **清除**。
- **跨设备同步**：在设置中开启**剪贴板历史记录同步**，即可通过同一 Microsoft 账户在其他 Windows 电脑访问记录。
- 若需自定义快捷键（例如重新映射），可使用 AutoHotkey 等第三方工具，但系统内置的 `Win + V` 默认快捷键稳定可靠。

更多详细信息，请参阅微软官方指南：[提升效率的剪贴板历史记录](https://www.microsoft.com/en-us/windows/tips/clipboard-history)。
