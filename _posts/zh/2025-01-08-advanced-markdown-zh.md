## 图片类

你可以像这样在Markdown中为图片应用类：

```markdown
{: .centered }
![](assets/images/pin/pin.jpg){: .responsive }
*来源: Telegram*{: .caption }
```

这将渲染带有指定类的图片：

{: .centered }  
![](assets/images/pin/pin.jpg){: .responsive }  
*来源: Telegram*{: .caption }
