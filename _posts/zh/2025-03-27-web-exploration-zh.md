## 1. 浏览器开发者工具
### 核心技巧
- 打开 Chrome/Firefox 开发者工具（F12 或右键 > 检查）
- 使用 Elements 标签页检查组件结构
- Network 标签页监控 API 调用和网络请求
- Console 标签页查看 JavaScript 错误并进行调试
- Performance 标签页分析渲染和加载时间

## 2. 手动交互测试
### 系统化探索方法
- 点击每个按钮和交互元素
- 测试输入框时使用：
  - 有效输入
  - 无效输入（特殊字符、超长文本）
  - 边界条件输入
- 验证表单校验
- 检查错误处理
- 在不同屏幕尺寸下测试响应式设计

## 3. 状态与导航测试
### 全面覆盖要点
- 遍历所有路由/页面
- 测试浏览器前进/后退按钮
- 验证状态持久化
- 检查 URL 参数处理
- 测试深度链接功能
