GitHub Desktop 版本 3.4.13 (arm64) 在文本差异面板中针对特定语言存在字体渲染问题。受影响的语言包括：

- 中文（简体与繁体）
- 印地语
- 日语

然而，以下语言能够正确渲染：
- 英语
- 西班牙语
- 德语
- 阿拉伯语
- 法语
