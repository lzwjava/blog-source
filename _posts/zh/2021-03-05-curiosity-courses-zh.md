# 好奇心课程

学生不需要完美的老师，他们需要快乐的老师，让他们每天都能兴奋地来到学校。我一直对教育很感兴趣，也希望自己能成为一个好老师。然而，教育是如此困难，改变人是如此困难。过去几年我学到了很多东西，在学校里学到了基础知识，进入社会后学到了更多。我也想实践和改进我的教育理论，所以我推出了好奇心课程。

也许最好的教育不是学生问老师，而是老师问学生。我们一起思考，一起实验。在教育方面，我从王垠那里学到了很多。他也分享了很多教育见解，我从他身上学到了很多。他的课程办得很好，所以我也想和他一样，开设一些课程。这样我可以学习如何传授知识，如何向学生学习，如何从初学者的角度重新思考问题，以及如何用非常简单明了的方式解释知识。

## 课程计划

*   招募一些学生。
*   免费。
*   分几个大方向：编程、算法、后端、前端、iOS、Android、创业、自媒体、英语、物理、电子、日语、信息检索。
*   课时数视方向而定，可能 2-20 节课，每节课 2 小时，时长 2 周到 3 个月。
*   不限年龄和职业，尤其欢迎零基础者。
*   课后有作业。
*   视情况，学生也可以在其他时间和老师讨论。
*   同时积累学习内容，录制教学视频和讨论视频，整理讨论笔记。
*   学生可以选择感兴趣的方向。
*   前期先招募一位学生打磨课程，后续可能招募更多。
*   有意者可通过微信私信联系我。

各方向的目标如下。这些我都做到了，所以有信心尝试教学生。有些方向我也是刚开始学，这样挺好的，能更好地从初学者的角度思考，一起成长和学习。

## 课程方向

编程：

*   初步掌握编程基础知识。
*   能写出自己感兴趣的 2000 行以上代码的程序。

算法：

*   初步掌握一门编程语言。
*   了解一些算法。
*   在线刷题 100 道。

后端：

*   初步掌握一门常用的后端编程语言。
*   语言可以是 PHP、Java、Python、Ruby、NodeJS、Rust、C、C++ 或 Go。
*   能写出一个功能简单的 web 后端应用。

前端：

*   初步掌握前端知识。
*   涉及 HTML、CSS 和 Javascript。
*   能写出一个功能简单的网站或小程序。

iOS：

*   初步掌握 iOS 知识。
*   语言可以是 Objective-C 或 Swift。
*   能写出一个功能简单的 iOS 应用。

Android：

*   初步掌握 Android 知识。
*   语言可以是 Java 或 Kotlin。
*   能写出一个功能简单的 Android 应用。

创业：

*   通过技能、信息差等方式赚钱。
*   找到一个客户或售卖一些课程等。
*   通过合法劳动在工作外赚到 3000 元。

自媒体：

*   写文章、做图文、做视频。
*   通过内容输出获得 500 个关注者。

英语：

*   侧重练习输入技能。
*   学会获取英文信息。
*   能够不看字幕看美剧。
*   阅读两本英文书。
*   不看字幕看完 20 部纪录片或美剧。

物理：

*   主要学习费曼物理学讲义的知识。
*   能够较好地理解牛顿力学。
*   独立重做实验，深刻理解物理公式。
*   学好费曼物理学讲义第一卷上半册。
*   和老师一起学习。

电子：

*   学习基础的电路板实验。
*   学习 DIY 一个收音机。
*   和老师一起学习。

日语：

*   学习基础的日语知识。
*   熟练掌握 50 音图。
*   掌握 200 个基础词汇。
*   能够借助词典看懂文章。
*   和老师一起学习。

信息检索：

*   能够使用工具访问英文信息。
*   学会在电脑、手机和电视上访问英文信息。
