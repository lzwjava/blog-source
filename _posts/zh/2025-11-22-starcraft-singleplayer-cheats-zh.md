## 🕹️ 星际争霸单人模式秘籍

你肯定还记得那些作弊码！它们曾是游玩单人战役任务时常用又有趣的方式。

以下是你可能还记得的一些最著名、最实用的代码：

| 代码 | 效果 |
| :--- | :--- |
| **`show me the money`** | 获得**10,000单位晶体矿和10,000单位高能瓦斯**（建造所需的资源）。 |
| **`operation cwal`** | **加速建筑建造和单位生产**（你提到的“加速”）。 |
| **`power overwhelming`** | 让你所有单位获得**无敌状态**（常被视为“上帝”模式）。 |
| **`the great one`** | 让虫族单位获得**无敌状态**（另一个常见的变体）。 |
| **`noglues`** | 禁用“战争迷雾”，显示整个地图（但不显示敌方单位）。 |
| **`black sheep wall`** | 显示整个地图和敌方单位。 |
| **`something for nothing`** | 获得所有可用的升级。 |
