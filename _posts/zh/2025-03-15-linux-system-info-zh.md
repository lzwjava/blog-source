在 Ubuntu 系统中，您可以通过以下命令获取系统信息，包括系统架构（32位或64位）、操作系统版本、内存容量和磁盘大小。以下是详细操作指南：

### 1. 判断系统架构（32位或64位）
- **命令**：`uname -m`
- **说明**：该命令显示机器硬件名称
  - 若输出为 `x86_64`，则为 **64位** 系统
  - 若输出为 `i686` 或 `i386`，则为 **32位** 系统
- **示例输出**：
  ```
  x86_64
  ```
  表示当前为64位系统

### 2. 获取操作系统版本
- **命令**：`lsb_release -a`
- **说明**：该命令提供 Linux 标准基础（LSB）和发行版的详细信息
  - 查看 `Description` 行可获知 Ubuntu 版本
- **示例输出**：
  ```
  未安装 LSB 模块。
  发行商 ID：Ubuntu
  描述：      Ubuntu 20.04.3 LTS
  版本：      20.04
  代号：      focal
  ```
  此处显示操作系统为 **Ubuntu 20.04.3 LTS**

### 3. 获取内存容量
- **命令**：`free -h`
- **说明**：该命令以易读格式（如 GB/MB）显示内存使用情况
  - 查看 `Mem:` 行的 `total` 列即可获取总内存容量
- **示例输出**：
  ```
               总量        已用        空闲        共享    缓冲/缓存     可用
  内存：        15Gi       2.3Gi       9.5Gi       300Mi       3.8Gi        12Gi
  交换空间：    2.0Gi         0B       2.0Gi
  ```
  此例中总内存容量为 **15 GB**

### 4. 获取磁盘容量
- **命令**：`lsblk -d -o NAME,SIZE`
- **说明**：该命令列出所有磁盘设备（不包含分区）及其容量
  - `-d` 参数确保仅显示磁盘设备（如 `/dev/sda`），不显示分区
  - 查找磁盘名称（如 `sda`）及其对应容量
- **示例输出**：
  ```
  设备名   容量
  sda     1T
  sdb     500G
  ```
  此系统配备两块磁盘：容量 **1TB** 的 `/dev/sda` 和容量 **500GB** 的 `/dev/sdb`。若仅有一块磁盘，通常显示为 `sda`

### 注意事项
- 以上命令均使用 Ubuntu 系统自带工具，无需额外安装软件包
- 对于磁盘容量，若系统配备多块磁盘，命令将列出所有设备。通常 `/dev/sda` 为主磁盘
- 在终端中运行这些命令即可获取您系统的具体信息

通过以上命令组合，您可以快速获取 Ubuntu 系统的基础硬件信息！
