* 将示例代码复制到 Cursor，并使用注释提供上下文。

* 在香港地区，使用 DeepSeek 或 Mistral 服务是可以接受的，而使用 ChatGPT、Claude 或 Gemini 则不可以。

* 一个区域的 API 政策大致相当于其应用程序政策。

* 使用 Cursor 而不是 Visual Studio Code。

* 仍有一些情况需要使用 Visual Studio Code，例如在 git 合并编辑器场景中，这时我仍然使用 `git config --global core.editor "code --wait"`。

* 从 Deepseek V3 发布开始的那一天起，我们不再需要订阅任何 AI 工具。

* 使用 Gemini 或 Grok 生成节日庆祝图像，提示如：“生成带有姓名文本的幸福的农历蛇年新年图像”。

* 在某些情况下，即使向 AI 模型提供原始文本来创建表格，输出中的某些地方仍可能与输入不同。例如，使用 Cursor 中的 Deepseek V3 模型生成 pip list 表格时，可能会包含版本如 `1.极狐0`。这里的 `极狐` 指的是中国的 GitLab 平台。

* 使用 Deepseek 或 Mistral API 翻译标题时，提示如 `你是一名专业翻译员。你正在将一篇英文的 Markdown 文件翻译成中文的 Jekyll 博客文章。{text}` 可能会导致错误的翻译。除了你提供的文本之外，输出通常还包含过多的翻译。
