要在 Safari 中显示控制台，请按照以下步骤操作：

1. 在 Mac 上打开 Safari。
2. 点击顶部菜单栏中的 **Safari** > **设置**（如果使用的是较旧版本，则为 **偏好设置**）。
3. 点击 **高级** 选项卡。
4. 在底部，勾选标有 **“在菜单栏中显示开发菜单”** 的复选框。
5. 关闭设置窗口。
6. 现在，在菜单栏中，你会看到一个新的 **开发** 选项。点击 **开发** > **显示 JavaScript 控制台**（或 **显示 Web 检查器**，具体取决于你的版本）。

或者，你可以在启用开发菜单后使用快捷键 **Option + Command + C** (⌥ + ⌘ + C) 来快速打开控制台。

如果你在 iPhone 或 iPad 上，Safari 本地不显示控制台，但你可以通过将设备通过 USB 连接到 Mac，在 Mac 上打开 Safari，并使用开发菜单来检查 iOS 设备的浏览器来启用它。
