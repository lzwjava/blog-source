要在 Safari 中显示控制台，请按照以下步骤操作：

1. 在 Mac 上打开 Safari。
2. 点击顶部菜单栏中的 **Safari** > **设置**（旧版本可能显示为 **偏好设置**）。
3. 选择 **高级** 标签页。
4. 勾选底部的 **“在菜单栏中显示‘开发’菜单”** 复选框。
5. 关闭设置窗口。
6. 此时菜单栏将出现新的 **开发** 选项。点击 **开发** > **显示 JavaScript 控制台**（不同版本可能显示为 **显示 Web 检查器**）。

启用开发菜单后，您也可以直接使用快捷键 **Option + Command + C** (⌥ + ⌘ + C) 快速调出控制台。

若在 iPhone 或 iPad 上使用 Safari，系统并未内置控制台功能。但可通过 USB 连接至 Mac，在 Mac 端打开 Safari，通过开发菜单对 iOS 设备浏览器进行调试检查。
