是的，Azure 虚拟机（VM）完全支持运行 Linux 操作系统。您可以直接从 Azure 市场部署各种主流发行版，也可以上传自定义的 Linux 镜像。

### 核心亮点：
- **支持的发行版**：Azure 官方认证并支持主流 Linux 发行版，包括 Ubuntu、Red Hat Enterprise Linux (RHEL)、Debian、SUSE Linux Enterprise Server (SLES)、Oracle Linux、CentOS Stream、Rocky Linux、AlmaLinux 等。现提供超过 100 个即用型 Linux 镜像。
- **部署方式**：可通过 Azure 门户、CLI、PowerShell 或 ARM 模板创建 Linux 虚拟机。例如，您能在几分钟内快速部署 Ubuntu Server 实例。
- **功能特性**：Azure 上的 Linux 虚拟机完整集成存储、网络、安全与监控等服务，同时支持 Azure Arc 等混合管理工具。

若需快速入门，请查阅创建 Linux 虚拟机的官方快速指南。
