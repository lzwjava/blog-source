很高兴你在寻找跨平台工具，特别是考虑到 Ubuntu/Linux 环境！现代应用的魅力在于它们能够无缝运行在 Windows、macOS 和各种 Linux 发行版上。除了 Firefox 和 VS Code，以下是一些优秀的跨平台工具和应用，按功能分类：

**1. 网页浏览器（除 Firefox 外）：**

* **Google Chrome：** 快速、安全，并拥有庞大的扩展库。
* **Brave：** 默认屏蔽广告和追踪器，注重隐私，内置 VPN。
* **Opera：** 提供独特功能，如免费内置 VPN、广告拦截器和侧边栏集成。

**2. 开发工具（除 VS Code 外）：**

* **Sublime Text：** 以其速度和简洁性著称，深受许多开发者喜爱。
* **Git：** 版本控制必备工具，其命令行工具随处可用。图形界面客户端如 **GitKraken** 或 **GitHub Desktop** 也是跨平台的。
* **Docker：** 用于容器化，允许你将应用程序及其依赖项打包，在各种环境中一致运行。
* **跨平台开发框架（用于构建应用）：**
    * **Flutter：**（谷歌）使用 Dart，非常适合从单一代码库构建移动端、网页和桌面端视觉吸引力强的应用。
    * **React Native：**（Meta）使用 JavaScript，用于构建原生渲染的移动应用。
    * **Xamarin (.NET MAUI)：**（微软）使用 C#，非常适合与 .NET 生态系统集成。
    * **Ionic：** 使用 Web 技术（HTML、CSS、JavaScript）构建混合移动应用。
    * **Kotlin Multiplatform：**（JetBrains）允许共享业务逻辑代码，同时保持平台特定的 UI。

**3. 生产力与办公套件：**

* **LibreOffice：** 一款免费开源的办公套件，是 Microsoft Office 的强大替代品，包括文字处理、电子表格、演示文稿等。
* **Google 文档、表格、幻灯片：** 主要基于网页，但在任何浏览器中都能无缝工作，并提供离线功能。
* **Microsoft 365（网页版）：** 虽然桌面应用主要在 Windows/macOS 上，但 Word、Excel 和 PowerPoint 的网页版在 Linux 的浏览器中功能齐全。
* **Obsidian：** 一款强大的笔记和知识库应用，使用 Markdown 并允许在笔记之间进行广泛链接。
* **Joplin：** 一款开源笔记和待办事项应用，支持 Markdown 并与各种云服务同步。
* **Standard Notes：** 一款开源、加密的笔记应用，注重隐私和安全。
* **OneNote（微软）：** 免费的笔记应用，采用数字笔记本布局，可通过网页应用或非官方客户端在 Linux 上使用。
* **Everdo：** 一款 GTD（搞定）应用，注重隐私和离线优先使用。
* **Xournal++：** 非常适合手写笔记，尤其是如果你有触控笔/数位板，支持 PDF 注释和各种绘图工具。

**4. 通讯与协作：**

* **Discord：** 在社区和团队中很受欢迎，支持语音、视频和文字聊天。
* **Telegram：** 安全快速的即时通讯应用，支持大群组聊天和文件共享。
* **Signal：** 以其强大的端到端加密和隐私功能闻名。
* **Slack：** 广泛使用的团队消息和协作应用。
* **Element（Matrix）：** 一个开源、去中心化的通讯平台，非常适合注重隐私的用户。
* **Jami：** 一款免费开源的去中心化视频会议和聊天工具。
* **Skype：** 最古老的 VoIP 和即时通讯应用之一，在 Linux 上仍受支持。
* **Beeper：** 一款通用聊天应用，旨在将你所有的聊天网络（WhatsApp、Messenger、Telegram 等）整合到一个界面中。
* **LocalSend：** 一款开源、跨平台的 AirDrop 替代品，用于在没有互联网连接的情况下本地共享文件。

**5. 媒体播放器与管理：**

* **VLC Media Player：** "万能"媒体播放器，支持几乎所有的音频和视频格式。
* **Kodi：** 一个强大的开源媒体中心，用于组织和流式传输你的媒体库。
* **Spotify：** 流行的音乐流媒体服务，提供 Linux 桌面客户端。
* **Audacity：** 一个强大的开源音频编辑工具，也可作为播放器使用。
* **Plex：** 一个媒体服务器，允许你将内容流式传输到任何设备，并为各种平台提供客户端应用。
* **SMPlayer：** 一款适用于 Windows 和 Linux 的免费媒体播放器，内置编解码器，非常适合播放各种格式和恢复播放。
* **mpv：** 一款免费开源的命令行媒体播放器，以高质量视频输出和可定制性闻名。

**6. 云存储与同步：**

* **Google Drive：** 提供慷慨的免费存储空间，并通过浏览器与谷歌办公工具良好集成。
* **Dropbox：** 一个简单可靠的文件存储和共享解决方案。
* **Nextcloud：** 一个自托管替代方案，让你可以完全控制自己的数据。
* **OneDrive：** 微软的云存储解决方案，在 Linux、Windows 和 Mac 上运行良好。

**7. 设计与图形：**

* **GIMP：** 一个免费开源的 Photoshop 替代品，用于图像编辑。
* **Krita：** 专为数字绘画和插画设计。
* **Inkscape：** 一个矢量图形编辑器，非常适合制作徽标和插画。
* **Darktable：** 一款照片编辑应用，用于专业级色彩校正。

**8. 实用工具：**

* **7-Zip：** 一个轻量级文件归档器，支持多种格式。
* **KeePassXC：** 一个安全、开源的密码管理器，用于管理你的凭证。
* **OBS Studio：** 一个强大的屏幕录制和直播工具。
* **Calibre：** 一个适用于所有平台的电子书管理工具。
* **Etcher：** 一个简单的创建可启动 U 盘的工具。

选择时，请务必查看官方网站获取最新的 Linux 安装说明，因为有些可能提供 AppImage、Flatpak、Snap 或传统的 `.deb` 或 `.rpm` 包。享受你的跨平台工作流程吧！
