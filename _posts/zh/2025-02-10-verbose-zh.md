- 要启用 Homebrew 的详细输出，请结合使用 `export HOMEBREW_CURL_VERBOSE=1` 和 `brew upgrade --verbose`。

- 要增加 SSH 的详细程度，请使用 `ssh -vvv user@host`。

- 要获得更多 `git` 的详细输出，请使用 `git -v` 或 `git --verbose`。

- 要获得更多 `rsync` 的输出，请使用 `rsync -vv source destination`。
