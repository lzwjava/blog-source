### Visual Studio Code 1.104（2025 年 8 月版）

本次更新重点增强了 GitHub Copilot 的 AI 功能，强化了 AI 工具的安全性，并在编辑器与终端中实现了多项生产力提升。以下是主要更新内容概览：

#### AI 功能
- **自动模型选择（预览版）**：当选择“自动”模式时，聊天功能将根据性能和速率限制自动选择最佳模型（例如 Claude Sonnet 4、GPT-5 变体等）；付费用户还可享受 10% 的请求费用折扣。
- **敏感文件编辑确认**：代理模式在编辑系统文件、点文件或工作区外的任何内容前需获得用户批准；可通过设置进行自定义。
- **AGENTS.md 文件支持（实验性）**：自动将工作区中的 `AGENTS.md` 文件作为聊天请求的上下文内容。
- **增强的编程代理协作（实验性）**：优化了会话管理、GitHub 集成，并支持通过 TODO 注释或聊天进行任务委派。
- **终端自动批准**：可选择加入更安全的命令执行模式，针对高风险操作（如 `curl` 或 `wget`）显示警告；新增审批规则。
- **数学公式渲染**：聊天回复中默认内联渲染 KaTeX 公式。
- **改进的 #codebase 工具**：采用全新嵌入模型，实现更快速、更高效的语义代码搜索。
- **禁用 AI 功能**：新增设置项，可全局或按工作区隐藏并禁用 Copilot 聊天、补全和建议功能。
- **Python 专属 AI 工具（实验性/预览版）**：为未文档化的符号提供 AI 驱动的悬停摘要，并推出支持内存中执行的“运行代码片段”工具。

#### 安全性
- **AI 工具防护机制**：针对敏感编辑、终端命令和全局自动批准操作扩展了确认流程，通过警告和可配置规则降低风险。
- **文档**：新增关于 AI 驱动开发安全考量的指导说明。

#### 生产力
- **聊天视图默认设置**：聊天窗口现默认在辅助侧边栏中打开；可见性支持自定义。
- **终端改进**：增强 IntelliSense（如 git 补全、图标）、粘性滚动以及终端编辑器/窗口中的操作功能。
- **Python 环境**：优化 Pipenv 支持、支持从 `.env` 文件注入环境变量，并在所有 Python 文档中实现完整 IntelliSense。
- **编辑器优化**：可配置的内联建议延迟、键盘导航的选项卡索引，以及 Windows 系统的窗口边框色彩。
- **GitHub Pull Requests**：可折叠侧边栏、Copilot 的 TODO 操作，以及子模块忽略选项。
- **任务与终端支持**：改进输入处理、错误提示和 shell 集成超时机制。

#### 其他重要变更
- **无障碍功能**：为聊天确认对话框添加焦点命令，辅助屏幕阅读器使用。
- **语言支持**：移除内置的 Bower.json IntelliSense（该工具已弃用；请迁移至 npm/yarn）。
- **扩展开发**：新增身份验证挑战和辅助侧边栏贡献点的提案 API。
- **弃用说明**：诸如 `chat.mcp.enabled` 等设置已迁移；不再支持 Bower。

完整详情请参阅官方发布说明。
