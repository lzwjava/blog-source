## 安装

我之前写过一篇关于[如何组装电脑](./computer-en)的文章。让我们尝试一些不同的东西。今天，我正在一台预装了Windows 10的联想小新14IAH8笔记本电脑上安装Ubuntu 24.04。

```bash
sudo dd if=ubuntu-24.04.1-desktop-amd64.iso of=/dev/rdisk6 bs=1m
```

需要进入设置并禁用设备加密，因为Ubuntu安装程序需要这样做。

如果在BIOS中启用了设备加密并且禁用了安全启动，Windows将提示输入BitLocker恢复密钥。

因此，首先需要在BIOS中启用安全启动，登录Windows，然后进入设置禁用设备加密。之后，可以再次禁用安全启动以启动Ubuntu安装程序。

## 卡住

当Ubuntu启动`gdm.service`（GNOME显示管理器）并卡住时，通常表示图形用户界面（GUI）或显示管理器本身存在问题。以下是一些可以用来排除故障并可能解决问题的步骤：

1. **访问虚拟终端：**
   - 如果您的系统卡在GDM屏幕上，您可以尝试访问终端来诊断问题。
   - 按`Ctrl + Alt + F2`（或`F3`、`F4`等）切换到虚拟终端。

2. **登录：**
   - 使用您的用户名和密码登录。

3. **检查系统日志：**
   - 使用以下命令检查系统日志中是否存在任何可能指示问题原因的错误或警告：
     ```bash
     sudo journalctl -xe
     ```

4. **重启GDM服务：**
   - 尝试重启GDM服务以查看是否能解决问题：
     ```bash
     sudo systemctl restart gdm
     ```

5. **安装并配置LightDM：**
   - 如果问题仍然存在，您可以尝试安装LightDM作为替代显示管理器：
     ```bash
     sudo apt-get install lightdm
     ```
   - 安装lightdm后，系统将提示您设置默认显示管理器。

6. **重启系统：**
   - 进行更改后，重启系统以应用更改：
     ```bash
     sudo reboot
     ```
