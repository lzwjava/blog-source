## 安装

我之前写过一篇关于[如何组装电脑](./computer-en)的文章。让我们尝试一些不同的东西。今天，我正在一台预装了Windows 10的联想小新14IAH8笔记本电脑上安装Ubuntu 24.04。

```bash
sudo dd if=ubuntu-24.04.1-desktop-amd64.iso of=/dev/rdisk6 bs=1m
```

需要进入设置并禁用设备加密，因为Ubuntu安装程序需要这样做。

如果设备加密已启用且BIOS中的安全启动已禁用，Windows将提示输入BitLocker恢复密钥。

因此，需要先在BIOS中启用安全启动，登录Windows，然后进入设置禁用设备加密。之后，可以再次禁用安全启动以启动Ubuntu安装程序。

## 卡住

当Ubuntu启动`gdm.service`（GNOME显示管理器）并卡住时，通常表示图形用户界面（GUI）或显示管理器本身存在问题。以下是一些可以用来排查和解决问题的步骤：

1. **访问虚拟终端：**
   - 如果你的系统卡在GDM屏幕上，可以尝试访问终端来诊断问题。
   - 按`Ctrl + Alt + F2`（或`F3`、`F4`等）切换到虚拟终端。

2. **登录：**
   - 使用你的用户名和密码登录。

3. **检查系统日志：**
   - 使用以下命令检查系统日志中是否存在任何错误或警告，这些错误或警告可能指示问题的根本原因：
     ```bash
     sudo journalctl -xe
     ```

4. **重启GDM服务：**
   - 尝试重启GDM服务，看看是否能解决问题：
     ```bash
     sudo systemctl restart gdm
     ```

5. **重启系统：**
   - 进行更改后，重启系统以应用更改：
     ```bash
     sudo reboot
     ```
