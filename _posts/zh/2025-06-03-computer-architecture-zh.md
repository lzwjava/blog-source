要确定您当前使用的计算机是基于 64 位 x86（x64）还是 ARM64 架构，请根据操作系统按以下步骤操作：

**Windows 系统：**

1.  按下 **Win + R** 组合键打开“运行”对话框。
2.  输入 **`msinfo32`** 后按回车键，将打开“系统信息”窗口。
3.  在系统信息窗口中，查找 **“系统类型”** 条目：
    * 若显示 **“基于 x64 的电脑”**，则表示您的计算机搭载 64 位 x86（x64）处理器。
    * 若显示 **“基于 ARM64 的电脑”**，则表示您的计算机搭载 64 位 ARM（ARM64）处理器。

**macOS 系统：**

1.  点击屏幕左上角的 **苹果菜单**。
2.  选择 **“关于本机”**。
3.  在“概览”标签页中，查看 **“处理器”** 或 **“芯片”** 信息：
    * 若显示 **Intel 处理器**（如 Intel Core i7、Intel Core i9），则表示您的 Mac 搭载 64 位 x86（x64）处理器。
    * 若显示 **Apple 芯片**（如 Apple M1、Apple M2、Apple M3），则表示您的 Mac 搭载 ARM64 处理器。

**Linux 系统：**

1.  打开 **终端** 窗口。
2.  输入以下命令后按回车键：
    ```bash
    uname -m
    ```
3.  输出结果将显示系统架构：
    * **`x86_64`** 或 **`amd64`**：表示 64 位 x86（x64）处理器
    * **`aarch64`** 或 **`arm64`**：表示 64 位 ARM（ARM64）处理器
    * 其他输出结果可能表示 32 位系统（目前已较少见）
