### IBM WebSphere Application Server (WAS) 管理控制台：左侧导航窗格

IBM WebSphere Application Server (WAS) 管理控制台（又称集成解决方案控制台）中的左侧视图是一个分层导航树，用于组织配置和管理任务。它提供了快速访问管理服务器环境、应用程序和资源的关键区域。具体结构可能因 WAS 版本（例如 8.5、9.0）和版本（基础版与网络部署版）而略有不同，但核心顶级类别是一致的。

以下是导航树中主要顶级部分的列表，并简要描述了它们的主要用途。子部分可展开（使用 +/- 图标）以进行更细粒度的任务。

#### 主要顶级部分
- **应用程序**  
  用于部署、安装、更新、启动/停止和管理应用程序（例如 EAR/WAR 文件）。  
  *关键子部分*：企业应用程序、WebSphere 企业应用程序、Web 模块、共享库。  
  *常见任务*：安装新应用程序、将模块映射到服务器、配置类加载器。

- **资源**  
  配置共享资源，如数据库、消息传递和连接池，供应用程序使用。  
  *关键子部分*：JDBC（数据源/提供程序）、JMS（队列/主题）、JavaMail 会话、URL 提供程序。  
  *常见任务*：设置 JDBC 数据源、创建 JMS 连接工厂。

- **服务**  
  管理服务器范围的服务，如安全性、消息传递和通信协议。  
  *关键子部分*：安全性（全局安全性、用户/组、身份验证）、邮件提供程序、端口、ORB 服务、事务服务。  
  *常见任务*：启用 SSL、配置用户注册表、调整端口分配。

- **服务器**  
  处理服务器实例、集群和 Web 服务器定义。  
  *关键子部分*：服务器类型（WebSphere 应用程序服务器、WebSphere 代理服务器）、集群、Web 服务器。  
  *常见任务*：启动/停止服务器、配置 JVM 设置、为高可用性创建集群。

- **系统管理**  
  管理整体拓扑，包括节点、单元和控制台设置。  
  *关键子部分*：节点、单元、部署管理器、控制台首选项。  
  *常见任务*：同步节点、管理部署管理器、设置控制台主题/范围。

- **环境**  
  定义环境特定的变量和别名。  
  *关键子部分*：WebSphere 变量、主机别名。  
  *常见任务*：为路径或属性设置系统变量。

- **监控和调优**  
  监控性能并调整运行时行为。  
  *关键子部分*：性能监控基础设施 (PMI)、自定义服务、日志记录和跟踪。  
  *常见任务*：启用指标、调整线程池、查看请求指标。

- **故障排除**  
  访问诊断工具和日志。  
  *关键子部分*：日志和跟踪、第一步。  
  *常见任务*：下载日志、启用跟踪规范。

#### 附加说明
- **范围选择**：在许多页面的顶部，您可以通过范围（单元、节点、服务器）进行筛选以限制可见性。
- **引导式活动**：有时作为顶级或上下文部分出现，用于向导（例如，保护环境）。
- 访问控制台：通常通过 `https://<主机>:<端口>/ibm/console`（管理默认端口 9060，安全端口 9043）。
- 有关版本特定详细信息（例如 WAS 9.0），请参阅官方 IBM 文档，因为更新中会有细微的 UI 更改。
