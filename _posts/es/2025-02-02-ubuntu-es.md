## Instalación

Anteriormente escribí sobre [Cómo construir un ordenador](./computer-en). Intentemos algo diferente. Hoy, estoy instalando Ubuntu 24.04 en un portátil Lenovo Xiaoxing 14IAH8 que venía con Windows 10 preinstalado.

```bash
sudo dd if=ubuntu-24.04.1-desktop-amd64.iso of=/dev/rdisk6 bs=1m
```

Es necesario ir a la configuración y deshabilitar el cifrado del dispositivo, ya que el instalador de Ubuntu lo requiere.

Si el cifrado de dispositivo está habilitado y el inicio seguro está deshabilitado en la BIOS, Windows solicitará la frase de contraseña de BitLocker para la recuperación.

Por lo tanto, es necesario habilitar primero el inicio seguro en la BIOS, iniciar sesión en Windows y luego ir a la configuración para deshabilitar el cifrado del dispositivo. Después de eso, el inicio seguro se puede deshabilitar nuevamente para iniciar el instalador de Ubuntu.

## Atascado

Cuando Ubuntu inicia `gdm.service` (GNOME Display Manager) y se atasca, normalmente indica un problema con la interfaz gráfica de usuario (GUI) o con el propio administrador de visualización. Aquí hay algunos pasos que puede seguir para solucionar el problema y potencialmente resolverlo:

1. **Acceder a una terminal virtual:**
   - Si su sistema está atascado en la pantalla de GDM, puede intentar acceder a una terminal para diagnosticar el problema.
   - Presione `Ctrl + Alt + F2` (o `F3`, `F4`, etc.) para cambiar a una terminal virtual.

2. **Iniciar sesión:**
   - Inicie sesión con su nombre de usuario y contraseña.

3. **Comprobar los registros del sistema:**
   - Utilice el siguiente comando para comprobar los registros del sistema en busca de errores o advertencias que puedan indicar la causa del problema:
     ```bash
     sudo journalctl -xe
     ```

4. **Reiniciar el servicio GDM:**
   - Intente reiniciar el servicio GDM para ver si resuelve el problema:
     ```bash
     sudo systemctl restart gdm
     ```

5. **Reiniciar el sistema:**
   - Después de realizar los cambios, reinicie su sistema para aplicarlos:
     ```bash
     sudo reboot
     ```
