Previamente escribí sobre [Cómo construir una computadora](./computer-en). Intentemos algo diferente. Hoy, estoy instalando Ubuntu 24.04 en una laptop Lenovo Xiaoxing 14IAH8 que venía con Windows 10 preinstalado.

```bash
sudo dd if=ubuntu-24.04.1-desktop-amd64.iso of=/dev/rdisk6 bs=1m
```

Es necesario ir a la configuración y desactivar el cifrado de dispositivo, ya que el instalador de Ubuntu lo requiere.

Si el cifrado de dispositivo está habilitado y el inicio seguro está deshabilitado en la BIOS, Windows solicitará la frase de contraseña de BitLocker para la recuperación.

Por lo tanto, es necesario primero habilitar el inicio seguro en la BIOS, iniciar sesión en Windows y luego ir a la configuración para desactivar el cifrado de dispositivo. Después, se puede deshabilitar el inicio seguro nuevamente para comenzar el instalador de Ubuntu.
