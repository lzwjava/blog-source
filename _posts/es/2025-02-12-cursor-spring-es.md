La ruta del JDK se encuentra en `/opt/homebrew/Cellar/openjdk@17/17.0.14/libexec/openjdk.jdk/Contents/Home`.

```bash
mvn clean install
mvn spring-boot:run
mvn compile
mvn test
mvn package
mvn deploy
mvn verify
mvn site
mvn archetype:generate
```

Compile y ejecute la aplicación utilizando la línea de comandos.
