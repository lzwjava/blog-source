### Actualización más reciente de Windows 11
La actualización de características más reciente de Windows 11 es la **versión 24H2 (Sun Valley 3)**, también conocida como Actualización de octubre de 2024. Incluye mejoras como una gestión de tareas mejorada, un Explorador de archivos rediseñado, mejores funciones para juegos (por ejemplo, optimizaciones para la programación de la GPU y para juegos como Starfield), capacidades integradas de Copilot+ PC, sondeo predictivo para transiciones fluidas y correcciones de errores para una mayor estabilidad y seguridad.

Esta actualización se implementó gradualmente:
- Las compilaciones preliminares comenzaron en mayo de 2024 a través del programa Windows Insider.
- El paquete de habilitación estuvo disponible el 3 de septiembre de 2024 para dispositivos elegibles (aunque algunas funciones, como Recall AI, sufrieron retrasos y aún se están implementando).
- La versión pública completa se completó en todo el mundo el 1 de octubre de 2024.

Para comprobar su dispositivo, vaya a Configuración > Actualización de Windows y busque "Actualización acumulativa de 2024-10" o más reciente.

### Lanzamiento inicial de Windows 11
Si se refiere al lanzamiento original de Windows 11, se presentó el 24 de junio de 2021, y la compilación de disponibilidad general comenzó a implementarse el 5 de octubre de 2021, empezando por los dispositivos compatibles.

Para más detalles, consulte las fuentes oficiales de Microsoft Learn (learn.microsoft.com/es-es/windows/release-health/release-information) o la página del historial de actualizaciones de Windows 11 (support.microsoft.com/es-es/windows/windows-11-update-history-781b5c23-0797-4d25-bfe2-f2bbff5b1c45).
