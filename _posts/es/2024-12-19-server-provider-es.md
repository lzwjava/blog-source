El Outline Manager ofrece varias opciones de servidor, incluyendo Google Cloud, DigitalOcean, AWS Lightsail, entre otros.

Para Google Cloud y DigitalOcean, scripts convenientes simplifican el proceso de configuración del servidor. Comencé con estas opciones porque parecían sencillas. Sin embargo, estos servidores a veces terminan siendo bloqueados por el Gran Cortafuegos (GFW).

Como resultado, decidí probar AWS Lightsail nuevamente. Aunque toma alrededor de diez minutos configurarlo—en comparación con aproximadamente tres minutos en Google Cloud o DigitalOcean—parece funcionar de manera más confiable.

Algunas decisiones pueden parecer problemáticas al principio, pero a menudo se vuelven más fáciles con el tiempo. Al aplicar esta perspectiva a largo plazo al elegir tu proveedor de servidores proxy—y en cada aspecto de tu vida—puedes disfrutar finalmente de una experiencia mucho mejor.
