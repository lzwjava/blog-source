Aquí hay algunos prompts de uso frecuente para diversas tareas de programación, organizados por categoría. Estos prompts se emplean comúnmente al escribir código, refactorizar, cambiar nombres de variables y realizar otras actividades relacionadas, especialmente en contextos como IDEs, editores de código o al interactuar con asistentes de IA para programación.

### Escribir Código
Estos prompts se utilizan para generar nuevos fragmentos de código o implementaciones:
- "Escribe una función para [realizar tarea específica]."  
  *Ejemplo: "Escribe una función para calcular la suma de un array."*
- "Crea una clase para [propósito específico]."  
  *Ejemplo: "Crea una clase para gestionar un perfil de usuario."*
- "Genera código para [lograr cierta funcionalidad]."  
  *Ejemplo: "Genera código para obtener datos de una API."*

### Refactorización
Estos prompts se centran en mejorar la estructura, legibilidad o eficiencia del código existente sin alterar su comportamiento:
- "Refactoriza este código para mejorar la legibilidad."  
  *Ejemplo: "Refactoriza este código para usar nombres de variables más claros y un formato consistente."*
- "Simplifica esta función dividiéndola en funciones más pequeñas."  
  *Ejemplo: "Simplifica esta función extrayendo la lógica de validación en un método separado."*
- "Optimiza este código para un mejor rendimiento."  
  *Ejemplo: "Optimiza este código reduciendo los bucles anidados."*

### Cambiar Nombres de Variables
Estos prompts buscan mejorar la claridad del código actualizando los identificadores para que sean más descriptivos o consistentes:
- "Renombra la variable '[nombre_variable]' a un nombre más descriptivo."  
  *Ejemplo: "Renombra la variable 'x' a 'edadUsuario'."*
- "Cambia el nombre de la función '[nombre_función]' para que refleje mejor su propósito."  
  *Ejemplo: "Cambia el nombre de la función 'func' a 'calcularPrecioTotal'."*
- "Actualiza todos los nombres de variables para seguir [convención de nomenclatura específica]."  
  *Ejemplo: "Actualiza todos los nombres de variables para seguir la convención camelCase."*

### Depuración
Estos prompts ayudan a identificar y resolver problemas en el código:
- "Encuentra y corrige el error en este código."  
  *Ejemplo: "Encuentra y corrige el error que causa esta excepción de puntero nulo."*
- "Depura esta función para manejar [casos límite específicos]."  
  *Ejemplo: "Depura esta función para manejar arrays de entrada vacíos."*

### Añadir Comentarios/Documentación
Estos prompts mejoran la mantenibilidad del código añadiendo explicaciones o documentación formal:
- "Añade comentarios a este código para explicar su funcionalidad."  
  *Ejemplo: "Añade comentarios a este código para describir el algoritmo de ordenación utilizado."*
- "Genera documentación para esta clase/método."  
  *Ejemplo: "Genera documentación para esta clase User."*

### Traducción de Código
Estos prompts convierten código entre lenguajes de programación:
- "Traduce este código de [lenguaje origen] a [lenguaje destino]."  
  *Ejemplo: "Traduce este código de Python a Java."*
- "Convierte esta función de [lenguaje] a [otro lenguaje]."  
  *Ejemplo: "Convierte esta función de JavaScript a TypeScript."*

### Revisión de Código
Estos prompts solicitan retroalimentación o identifican posibles mejoras:
- "Revisa este código y sugiere mejoras."  
  *Ejemplo: "Revisa este código y sugiere formas de reducir la complejidad."*
- "Comprueba este código en busca de posibles problemas de seguridad."  
  *Ejemplo: "Comprueba este código en busca de vulnerabilidades de inyección SQL."*

### Generar Pruebas
Estos prompts crean casos de prueba para verificar la corrección del código:
- "Escribe pruebas unitarias para esta función."  
  *Ejemplo: "Escribe pruebas unitarias para esta función calcularDescuento."*
- "Crea un caso de prueba para verificar este método."  
  *Ejemplo: "Crea un caso de prueba para verificar que este método maneja entradas negativas."*

### Prompts Conversacionales
Estos son prompts más orientados al lenguaje natural, utilizados a menudo con asistentes de IA:
- "¿Puedes ayudarme a escribir un script para [realizar tarea]?"  
  *Ejemplo: "¿Puedes ayudarme a escribir un script para renombrar archivos en un directorio?"*
- "Necesito código que [haga cosas específicas]."  
  *Ejemplo: "Necesito código que lea un archivo CSV y ordene los datos."*
- "Muéstrame un ejemplo de [usar una característica o API particular]."  
  *Ejemplo: "Muéstrame un ejemplo de usar la Fetch API en JavaScript."*

Estos prompts proporcionan una base versátil para tareas comunes de programación. Pueden adaptarse completando detalles específicos (por ejemplo, nombres de funciones, lenguajes o restricciones) dependiendo del contexto o los requisitos de tu proyecto. Ya sea que estés escribiendo código nuevo, refinando una base de código existente o buscando asistencia de una herramienta de IA, estas plantillas deberían cubrir una amplia gama de escenarios.
