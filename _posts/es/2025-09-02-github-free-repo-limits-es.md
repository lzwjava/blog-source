Un usuario gratuito de GitHub puede crear **repositorios privados ilimitados**, pero existe una restricción en cuanto a **colaboradores**.

* Puedes tener tantos repositorios privados como desees.
* Cada repositorio privado puede tener hasta **3 colaboradores** (personas que invites a trabajar contigo).

Si necesitas más colaboradores, tendrías que actualizar a un plan de pago (GitHub Pro, Team o Enterprise).
