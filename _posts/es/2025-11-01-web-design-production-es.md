Este curso forma parte del Examen Nacional de Educación Superior por Autoaprendizaje (自学考试) en la Provincia de Guangdong, China. Es una asignatura básica profesional para la carrera de Diseño de Arte de Medios Digitales (Técnico Superior), que enfatiza las habilidades prácticas en HTML5 y CSS3 para la creación y maquetación de páginas web. El curso es altamente práctico, centrándose en la construcción de páginas web completas y estéticamente atractivas.

**Libro de Texto Especificado:**  
*HTML 5 + CSS 3 Web Design Tutorial* (HTML 5+CSS 3 网页设计教程), editado por Zhang Xingyun, Peng Jinxiu y Xing Guobo, Tsinghua University Press, edición 2021.

El plan de examen está estructurado en torno al libro de texto y evalúa el conocimiento a través de niveles de memorización, comprensión y aplicación. El alcance del examen cubre todo el contenido de este plan.

## I. Naturaleza y Objetivos del Curso

### 1. Naturaleza y Características del Curso
Este es un curso interdisciplinario que integra tecnología, humanidades y arte. Enseña conocimientos y habilidades estándar de diseño web, incluyendo tecnologías HTML5/CSS3, maquetaciones DIV+CSS y procesos de construcción de sitios web. Los estudiantes aprenden a aplicar técnicas para texto, imágenes, medios, enlaces, listas, tablas, formularios y estilos CSS para diseñar, producir y mantener páginas web.

### 2. Objetivos del Curso
- Comprender los conceptos básicos de la web y los procesos de creación.
- Dominar elementos como texto, imágenes, audio, video e hipervínculos.
- Dominar listas y tablas.
- Dominar formularios.
- Dominar estilos CSS.
- Dominar técnicas de maquetación web.

### 3. Conexiones con Cursos Relacionados
Prerrequisitos: "Fundamentos de Aplicaciones Informáticas" y "Fundamentos de Medios Digitales".

### 4. Puntos Clave y Dificultades
**Puntos Clave:** Elementos de producción web (texto, imágenes, medios, enlaces, listas, tablas, formularios).  
**Dificultades:** Control y embellecimiento con CSS (estilos, maquetaciones, posicionamiento).

## II. Objetivos de Evaluación
El plan guía el autoaprendizaje, la asistencia y el diseño del examen. Niveles de competencia:  
- **Memorización:** Recordar definiciones, sintaxis, reglas.  
- **Comprensión:** Comprender conceptos, diferencias, análisis simple.  
- **Aplicación:** Integrar HTML/CSS para diseñar páginas complejas mediante DIV+CSS.
