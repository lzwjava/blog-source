### Habilitar el historial del portapapeles en Windows
Antes de usar los accesos directos, asegúrate de que la función esté habilitada (está disponible en Windows 10 versión 1809 y posteriores, incluido Windows 11):
1. Abre **Configuración** (presiona `Tecla de Windows + I`).
2. Ve a **Sistema > Portapapeles**.
3. Activa **Historial del portapapeles**.

Esto permite a Windows almacenar hasta 25 elementos recientes (texto, HTML, imágenes) durante 1 hora por defecto, o por más tiempo si se sincronizan con una cuenta de Microsoft.

### Usar accesos directos para acceder y leer el historial del portapapeles
El acceso directo principal para abrir y leer tu historial del portapapeles es **`Tecla de Windows + V`**. Así es cómo usarlo paso a paso:

1. **Copiar múltiples elementos**: Usa `Ctrl + C` para copiar texto, imágenes u otro contenido según sea necesario. Windows los guardará automáticamente en el historial.

2. **Abrir el historial del portapapeles**: Presiona `Tecla de Windows + V`. Esto abre un panel buscable en el lado derecho de tu pantalla que muestra miniaturas o vistas previas de los clips recientes (el más reciente en la parte superior).

3. **Navegar y leer**:
   - Usa las **teclas de flecha** (↑/↓) para desplazarte por la lista.
   - **Escribe para buscar**: Al abrirse el panel, puedes empezar a escribir inmediatamente para filtrar elementos (por ejemplo, buscar una palabra clave en un texto copiado).
   - **Vista previa**: Pasa el cursor por encima o navega con las flechas hasta un elemento para ver una vista previa sin pegarlo.

4. **Pegar desde el historial**:
   - Selecciona un elemento con las teclas de flecha y presiona **Intro** para pegarlo en la posición del cursor.
   - O haz clic en el elemento si usas el ratón.

5. **Cerrar el panel**: Presiona `Esc` o `Tecla de Windows + V` nuevamente.

### Consejos adicionales
- **Anclar elementos**: En el panel, haz clic en el icono de anclar junto a un elemento para conservarlo indefinidamente.
- **Borrar el historial**: Haz clic derecho en un elemento > **Eliminar**, o ve a Configuración > Sistema > Portapapeles > **Borrar**.
- **Sincronizar entre dispositivos**: Habilita la **Sincronización del historial del portapapeles** en Configuración para acceder a él en otros PCs con Windows que tengan iniciada sesión con la misma cuenta de Microsoft.
- Si deseas personalizar el acceso directo (por ejemplo, reasignarlo), usa herramientas de terceros como AutoHotkey, pero el predeterminado `Win + V` está integrado y es confiable.

Para más detalles, consulta la guía oficial de Microsoft: [Clipboard History for Productivity](https://www.microsoft.com/en-us/windows/tips/clipboard-history).
