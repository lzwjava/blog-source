Este repositorio contiene soluciones a diversos problemas de algoritmos de plataformas como Codeforces, HackerRank, LeetCode, Lintcode, Nowcoder y UVa. Cada solución se organiza por número de problema e incluye el código fuente, archivos de entrada y, ocasionalmente, documentación adicional.

## Estructura del Repositorio

- `README.md`: Este archivo.
- `codeforces/`: Contiene soluciones a problemas de Codeforces.
- `hacker_rank/`: Contiene soluciones a problemas de SQL de HackerRank.
- `leetcode/`: Contiene soluciones a problemas de LeetCode.
- `lintcode/`: Contiene soluciones a problemas de Lintcode.
- `nowcoder/`: Contiene soluciones a problemas de Nowcoder.
- `uva/`: Contiene soluciones a problemas de UVa.

### Codeforces

Cada subcarpeta dentro del directorio `codeforces` se nombra según el número del problema y incluye lo siguiente:

- `1.in`: Archivo de entrada para el problema.
- `src/Main.java`: Archivo de código fuente en Java que contiene la solución al problema.

### HackerRank

El directorio `hacker_rank` contiene archivos SQL para diversos problemas. Cada archivo se nombra según el problema que resuelve.

### LeetCode

El directorio `leetcode` contiene scripts en Python para diferentes problemas. Los archivos se nombran según los problemas que resuelven, como `two_sum.py` o `valid_parentheses.py`.

### Lintcode

Cada subcarpeta dentro del directorio `lintcode` se nombra según el número del problema y incluye lo siguiente:

- `src/com/lintcode/Main.java`: Archivo de clase principal para la solución.
- `src/com/lintcode/Solution.java`: Archivo de clase de solución para el problema.
- `src/com/lintcode/`: Archivos de clase adicionales según sea necesario por el problema.

### Nowcoder

Cada subcarpeta dentro del directorio `nowcoder` se nombra según el problema y incluye lo siguiente:

- `Solution.java`: Archivo de clase de solución para el problema.
- `Test.java`: Archivo de clase de prueba para el problema.
- `TreeNode.java`: (Si aplica) Archivo de clase TreeNode para problemas que involucran estructuras de árbol.
