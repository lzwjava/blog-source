Shenzhen JOOAN Technology Co., Ltd. (también conocida como Qiao An Technology) se especializa en productos de seguridad para el hogar inteligente, incluyendo una gama de cámaras IP inalámbricas que se integran con su plataforma cloud, a menudo denominada QA Cloud o JOOAN Cloud, para monitorización remota, almacenamiento y acceso P2P a través de aplicaciones como Joolink o CAM720.

### Principales Productos de Cámaras IP
Sus cámaras IP están diseñadas para vigilancia doméstica y exterior, con conectividad WiFi/4G, detección de movimiento, visión nocturna, audio bidireccional y almacenamiento en la nube opcional para clips de eventos y transmisión en vivo. Los modelos populares incluyen:

- **JA-366 HD 720p Cloud IP Camera**: Un modelo inalámbrico para interior/exterior con resolución 1280x720, zoom panorámtico e inclinable (PTZ), visión nocturna de hasta 10m y soporte cloud integrado para visualización remota en aplicaciones móviles. Admite almacenamiento local con tarjeta SD o planes de cloud de pago.
  
- **Q3R-U Outdoor Security Camera**: Resolución 2K (3MP), impermeable IP66, PTZ 360°, seguimiento automático, visión nocturna en color y opciones de doble lente para vistas panorámicas. Se integra con la cloud para alertas de movimiento y grabación 24/7.

- **A2R-U WiFi Camera**: Modelo para interior que se conecta a la corriente, con HD 1080p, rotación 360°, detección humana y conversación bidireccional. Con capacidad para cloud para una configuración fácil mediante la aplicación, con opciones de almacenamiento de video cifrado.

- **JA-G4 Solar 4G Camera**: Cámara IP para exterior con batería/energía solar (5MP), no necesita WiFi—utiliza una SIM 4G para acceso remoto en áreas sin redes. Admite almacenamiento en la cloud para copia de seguridad de las grabaciones.

Estas cámaras son asequibles (a menudo entre $20 y $60 en sitios de venta al por menor), compatibles con Alexa/Google Assistant en algunos modelos, y enfatizan una configuración fácil basada en aplicaciones sin cableado complejo. Los servicios de cloud son opcionales, con pruebas gratuitas y planes de pago para almacenamiento extendido (ej. 7-30 días).

Para más detalles, consulta su sitio web oficial o las tiendas de aplicaciones.
