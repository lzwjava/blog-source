En una publicación anterior, escribí: "A veces, después de enviar documentos a Kindle por correo electrónico, informa un problema con el documento(s) enviado(s). Un problema es que el documento carece de un título, y para solucionarlo, necesitamos pasar el parámetro --metadata". Sin embargo, la condición no es realmente "a veces". Sin importar cuándo, si el documento carece de un título, el servicio actual de envío a Kindle reportará este error. Podemos usar "a veces" si el error está relacionado con el tiempo o es altamente relevante para el tiempo.
