### Rutinas de creación de arrays

*   `np.empty`: Devuelve un nuevo array de forma y tipo dados, sin inicializar las entradas.
*   `np.empty_like`: Devuelve un nuevo array con la misma forma y tipo que un array dado.
*   `np.eye`: Devuelve un array 2-D con unos en la diagonal y ceros en el resto.
*   `np.identity`: Devuelve el array identidad.
*   `np.ones`: Devuelve un nuevo array de forma y tipo dados, lleno de unos.
*   `np.ones_like`: Devuelve un array de unos con la misma forma y tipo que un array dado.
*   `np.zeros`: Devuelve un nuevo array de forma y tipo dados, lleno de ceros.
*   `np.zeros_like`: Devuelve un array de ceros con la misma forma y tipo que un array dado.
*   `np.full`: Devuelve un nuevo array de forma y tipo dados, lleno con fill\_value.
*   `np.full_like`: Devuelve un array completo con la misma forma y tipo que un array dado.
*   `np.array`: Crea un array.
*   `np.asarray`: Convierte la entrada en un array.
*   `np.asanyarray`: Convierte la entrada en un ndarray, pero deja pasar las subclases de ndarray.
*   `np.ascontiguousarray`: Devuelve un array contiguo en memoria (orden C).
*   `np.asfortranarray`: Devuelve un array contiguo en memoria (orden Fortran).
*   `np.copy`: Devuelve una copia del array del objeto dado.
*   `np.frombuffer`: Interpreta un búfer como un array unidimensional.
*   `np.from_dlpack`: Crea un array NumPy a partir de un objeto PyCapsule (DLPack).
*   `np.fromfile`: Construye un array a partir de datos en un archivo de texto o binario.
*   `np.fromfunction`: Construye un array ejecutando una función sobre cada coordenada.
*   `np.fromiter`: Crea un nuevo array unidimensional a partir de un objeto iterable.
*   `np.fromstring`: Un nuevo array 1-D inicializado a partir de datos de texto en una cadena.
*   `np.loadtxt`: Carga datos desde un archivo de texto.
*   `np.genfromtxt`: Carga datos desde un archivo de texto, manejando los valores faltantes como se especifica.
*   `np.savetxt`: Guarda un array en un archivo de texto.
*   `np.save`: Guarda un array en un archivo binario en formato NumPy `.npy`.
*   `np.savez`: Guarda varios arrays en un único archivo en formato `.npz` sin comprimir.
*   `np.savez_compressed`: Guarda varios arrays en un único archivo en formato `.npz` comprimido.

### Rutinas de manipulación de arrays

*   `np.reshape`: Da una nueva forma a un array sin cambiar sus datos.
*   `np.ravel`: Devuelve un array aplanado y contiguo.
*   `np.transpose`: Permuta las dimensiones de un array.
*   `np.swapaxes`: Intercambia dos ejes de un array.
*   `np.moveaxis`: Mueve ejes de un array a nuevas posiciones.
*   `np.broadcast_to`: Transmite (broadcast) un array a una nueva forma.
*   `np.broadcast_arrays`: Transmite (broadcast) cualquier número de arrays entre sí.
*   `np.expand_dims`: Expande la forma de un array.
*   `np.squeeze`: Elimina entradas unidimensionales de la forma de un array.
*   `np.concatenate`: Une una secuencia de arrays a lo largo de un eje existente.
*   `np.stack`: Une una secuencia de arrays a lo largo de un nuevo eje.
*   `np.hstack`: Apila arrays en secuencia horizontalmente (por columnas).
*   `np.vstack`: Apila arrays en secuencia verticalmente (por filas).
*   `np.dstack`: Apila arrays en secuencia en profundidad (a lo largo de la tercera dimensión).
*   `np.column_stack`: Apila arrays 1-D como columnas en un array 2-D.
*   `np.row_stack`: Apila arrays en secuencia verticalmente (por filas).
*   `np.split`: Divide un array en múltiples sub-arrays.
*   `np.array_split`: Divide un array en múltiples sub-arrays.
*   `np.hsplit`: Divide un array en múltiples sub-arrays horizontalmente (por columnas).
*   `np.vsplit`: Divide un array en múltiples sub-arrays verticalmente (por filas).
*   `np.dsplit`: Divide un array en múltiples sub-arrays a lo largo del tercer eje (en profundidad).
*   `np.tile`: Construye un array repitiendo A el número de veces dado por reps.
*   `np.repeat`: Repite elementos de un array.
*   `np.delete`: Devuelve un nuevo array con sub-arrays a lo largo de un eje eliminados.
*   `np.insert`: Inserta valores a lo largo del eje dado antes de los índices dados.
*   `np.append`: Añade valores al final de un array.
*   `np.resize`: Devuelve un nuevo array con la forma especificada.
*   `np.trim_zeros`: Recorta los ceros iniciales y/o finales de un array o secuencia 1-D.
*   `np.unique`: Encuentra los elementos únicos de un array.

### Operaciones de cadenas en arrays

*   `np.char.add`: Devuelve la concatenación de cadenas elemento por elemento para dos arrays de str o unicode.
*   `np.char.multiply`: Devuelve (a * i), es decir, múltiplo de cadena de i copias de a, para cada elemento.
*   `np.char.mod`: Devuelve (a % i), es decir, formato de cadena anterior a Python 2.6 (interpolación), para cada elemento en a.
*   `np.char.capitalize`: Devuelve una copia de `a` con solo el primer carácter de cada elemento en mayúsculas.
*   `np.char.lower`: Devuelve un array con los elementos convertidos a minúsculas.
*   `np.char.upper`: Devuelve un array con los elementos convertidos a mayúsculas.
*   `np.char.title`: Para cada elemento en `a`, devuelve una versión con formato de título: las palabras comienzan con caracteres en mayúsculas, todos los caracteres restantes con caso están en minúsculas.
*   `np.char.split`: Para cada elemento en `a`, devuelve una lista de las palabras en la cadena, usando sep como cadena delimitadora.
*   `np.char.splitlines`: Para cada elemento en `a`, devuelve una lista de las líneas en el elemento, rompiendo en los límites de línea.
*   `np.char.strip`: Para cada elemento en `a`, devuelve una copia con los caracteres iniciales y finales eliminados.
*   `np.char.lstrip`: Para cada elemento en `a`, devuelve una copia con los caracteres iniciales eliminados.
*   `np.char.rstrip`: Para cada elemento en `a`, devuelve una copia con los caracteres finales eliminados.
*   `np.char.join`: Devuelve una cadena que es la concatenación de las cadenas en la secuencia seq.
*   `np.char.replace`: Para cada elemento en `a`, devuelve una copia de la cadena con todas las ocurrencias de la subcadena old reemplazadas por new.
*   `np.char.encode`: Llama a `str.encode` elemento por elemento.
*   `np.char.decode`: Llama a `str.decode` elemento por elemento.

*   `np.not_equal`: Devuelve (x1 != x2) elemento por elemento.
*   `np.logical_and`: Calcula el valor de verdad de x1 AND x2 elemento por elemento.
*   `np.logical_or`: Calcula el valor de verdad de x1 OR x2 elemento por elemento.
*   `np.logical_not`: Calcula el valor de verdad de NOT x elemento por elemento.
*   `np.logical_xor`: Calcula el valor de verdad de x1 XOR x2, elemento por elemento.

### Comparación de arrays

*   `np.isclose`: Devuelve un array booleano donde dos arrays son iguales dentro de una tolerancia.

### Muestreo aleatorio de arrays

*   `np.random.random`: Devuelve flotantes aleatorios en el intervalo semiabierto [0.0, 1.0).
*   `np.random.choice`: Genera una muestra aleatoria a partir de un array 1-D dado.
*   `np.random.seed`: Inicializa el generador.
