Aquí tienes una guía paso a paso para usar MySQL, un popular sistema de gestión de bases de datos relacional de código abierto:

---

### 1. **Instalación**
- **Windows**: Descarga el MySQL Installer desde [mysql.com/downloads](https://dev.mysql.com/downloads/installer/)
- **macOS**:
  ```bash
  brew install mysql
  brew services start mysql
  ```
- **Linux** (Debian/Ubuntu):
  ```bash
  sudo apt update
  sudo apt install mysql-server mysql-client
  sudo systemctl start mysql
  sudo mysql_secure_installation  # Ejecuta la configuración de seguridad
  ```

---

### 2. **Conectarse a MySQL**
- **Inicio de sesión root por defecto**:
  ```bash
  mysql -u root -p  # Ingresa la contraseña cuando se solicite
  ```
- **Conectarse a un servidor remoto**:
  ```bash
  mysql -u username -p -h hostname -P 3306
  ```

---

### 3. **Operaciones Básicas con la Base de Datos**
- **Crear Usuario**:
  ```sql
  CREATE USER 'myuser'@'localhost' IDENTIFIED BY 'mypassword';
  ```
- **Crear Base de Datos**:
  ```sql
  CREATE DATABASE mydb;
  GRANT ALL PRIVILEGES ON mydb.* TO 'myuser'@'localhost';
  FLUSH PRIVILEGES;
  ```
- **Comandos SQL Básicos**:
  ```sql
  -- Crear tabla
  CREATE TABLE users (
      id INT AUTO_INCREMENT PRIMARY KEY,
      name VARCHAR(50),
      email VARCHAR(100) UNIQUE
  );

  -- Insertar datos
  INSERT INTO users (name, email) VALUES ('Bob', 'bob@example.com');

  -- Consultar datos
  SELECT * FROM users WHERE name LIKE 'B%';

  -- Actualizar datos
  UPDATE users SET email = 'new@email.com' WHERE id = 1;

  -- Eliminar datos
  DELETE FROM users WHERE id = 2;
  ```

---

### 4. **Características Avanzadas**
- **Joins**:
  ```sql
  SELECT orders.id, users.name 
  FROM orders
  JOIN users ON orders.user_id = users.id;
  ```
- **Procedimientos Almacenados**:
  ```sql
  DELIMITER //
  CREATE PROCEDURE GetUsers()
  BEGIN
    SELECT * FROM users;
  END //
  DELIMITER ;
  
  CALL GetUsers();
  ```
- **Soporte JSON** (MySQL 5.7+):
  ```sql
  CREATE TABLE products (
      id INT PRIMARY KEY,
      details JSON
  );
  ```

---

### 5. **Mantenimiento**
- **Copia de Seguridad (Backup)**:
  ```bash
  mysqldump -u root -p mydb > backup.sql
  ```
- **Restaurar**:
  ```bash
  mysql -u root -p mydb < backup.sql
  ```
- **Optimizar Tablas**:
  ```sql
  ANALYZE TABLE users;
  OPTIMIZE TABLE users;
  ```

---

### 6. **Seguridad**
- **Crear Usuario de Solo Lectura**:
  ```sql
  CREATE USER 'reader'@'localhost' IDENTIFIED BY 'readpass';
  GRANT SELECT ON mydb.* TO 'reader'@'localhost';
  FLUSH PRIVILEGES;
  ```

---

### 7. **Comandos Útiles de MySQL**
- Listar bases de datos: `SHOW DATABASES;`
- Usar una base de datos: `USE mydb;`
- Listar tablas: `SHOW TABLES;`
- Describir una tabla: `DESCRIBE users;`
- Mostrar usuarios: `SELECT user FROM mysql.user;`
- Salir de MySQL: `EXIT;` o `QUIT;`

**Diferencias Clave con PostgreSQL**:
1. Usa `AUTO_INCREMENT` en lugar de `SERIAL`
2. Nombres de tablas no sensibles a mayúsculas por defecto
3. Usa `mysqldump` en lugar de `pg_dump`
4. Puerto por defecto: **3306** (vs 5432 de PostgreSQL)
5. Sistema de autenticación y gestión de usuarios diferente

**Consejo**: Siempre haz copias de seguridad de las bases de datos antes de realizar cambios importantes. Usa transacciones (`START TRANSACTION`/`COMMIT`) para operaciones críticas.
