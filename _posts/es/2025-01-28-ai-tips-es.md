* Copia el código de ejemplo en Cursor y usa comentarios para proporcionar contexto.

* En la región de Hong Kong, es aceptable usar los servicios de DeepSeek o Mistral, mientras que no es aceptable usar ChatGPT, Claude o Gemini.

* La política de API de una región es aproximadamente la misma que su política de app.

* Usa Cursor en lugar de Visual Studio Code.

* Todavía hay casos en los que necesitas usar Visual Studio Code, como para escenarios de editor de fusión de git, donde aún uso `git config --global core.editor "code --wait"`.

* A partir del día del lanzamiento de Deepseek V3, no necesitamos suscribirnos a ninguna herramienta de IA.

* Usa Gemini o Grok para generar imágenes de celebración de festivales con sugerencias como "Genera una imagen feliz del Año Nuevo Lunar de la Serpiente con nombres de texto incluidos".

* En algunos casos, incluso cuando proporcionas texto original a modelos de IA para crear una tabla, algunos lugares en la salida pueden diferir de la entrada. Por ejemplo, al usar el modelo Deepseek V3 en Cursor para generar una tabla de pip list, puede incluir versiones como `1.极狐0`. Aquí, `极狐` se refiere a la plataforma GitLab china.

* Al usar las API de Deepseek o Mistral para traducir títulos con sugerencias como `You are a professional translator. You are translating a markdown file for a Jekyll blog post from English to Chinese. {text}`, puede llevar a traducciones incorrectas. Además del texto que proporcionas, la salida a menudo incluye traducción excesiva.

* Aunque a veces los modelos de IA en Cursor dan texto parcialmente correcto, podemos aceptarlos, ya que podemos agregar instrucciones de seguimiento que harán que los modelos de IA vuelvan a generar las partes correctas.
