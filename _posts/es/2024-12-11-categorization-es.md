* Todo se trata de categorización.

* Diseño, software, productos, física, revertir la miopía, startups, negocios, usuarios, estrategia, países, lenguajes de programación, variables y conceptos, todo se trata de categorización.

* La categorización es una forma de meta-pensamiento.

* Al categorizar cosas en diferentes grupos, puedes encontrar soluciones mejores o más detalladas. Es como el código de bifurcación condicional. Es como separar componentes electrónicos. Es como diferentes tipos de redes.

* Puedes comparar diferentes cosas categorizándolas en distintos aspectos.

* El número de grupos y los criterios de categorización son lo que necesitamos considerar. Es como las condiciones de un bloque de código condicional.

* Si las condiciones en sí tienen algún patrón, podemos usar una sentencia `switch`. Si las condiciones son diferentes entre sí o tienen casos especiales, podemos usar una sentencia `if`.

* El código Lambda es realmente interesante. Podemos combinar la categorización con el código Lambda. Diferentes grupos de objetos requieren diferentes procedimientos de manejo.
