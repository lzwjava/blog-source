**Cómo Configurar el Programa para el Lavavajillas Midea Q7 de 13 Cubiertos**

La configuración de los programas para el lavavajillas Midea Q7 de 13 cubiertos (modelo independiente o empotrado) normalmente se realiza utilizando los botones del panel de control o la pantalla táctil. A continuación se presentan los pasos generales para configurar un programa. Por favor, consulta el manual de usuario de tu modelo específico, ya que pueden existir ligeras variaciones:

1.  **Encender**:
    *   Asegúrate de que el lavavajillas esté conectado a una fuente de alimentación y que el suministro de agua esté abierto.
    *   Presiona el botón de encendido (generalmente etiquetado como "Power" o "ON/OFF") para iniciar la máquina.

2.  **Seleccionar un Programa de Lavado**:
    *   El panel de control ofrece múltiples opciones de programa, como "Lavado Normal", "Lavado Intensivo", "Lavado Eco", "Lavado Rápido" o "Lavado para Cristales".
    *   Utiliza el botón "Programa" o "Modo" (que puede ciclar entre las opciones o ser seleccionado en una pantalla táctil) para elegir el ciclo de lavado apropiado. Por ejemplo:
        *   **Lavado Normal**: Adecuado para la limpieza diaria de vajilla.
        *   **Lavado Intensivo**: Para vajillas muy sucias o grasientas.
        *   **Lavado Eco**: Para artículos ligeramente sucios, ahorrando agua y energía.
        *   **Lavado Rápido**: Para vajillas ligeramente sucias que necesitan un ciclo rápido.

3.  **Ajustar Configuraciones Adicionales (Opcional)**:
    *   El modelo Q7 puede ofrecer funciones adicionales como "Sanitización a Alta Temperatura", "Enjuague Extra" o "Inicio Retrasado".
    *   Presiona el botón de función correspondiente (por ejemplo, "Sanitize" o "Delay") para configurarlo. Por ejemplo:
        *   **Inicio Retrasado**: Selecciona el tiempo de retraso (típicamente 1-24 horas) usando los botones "+" o "-".
        *   **Sanitización a Alta Temperatura**: Actívalo para temperaturas de lavado más altas, ideal para esterilizar la vajilla.

4.  **Confirmar e Iniciar**:
    *   Asegúrate de que el detergente (pastilla o líquido) esté colocado en el dispensador designado.
    *   Verifica que la vajilla esté correctamente colocada (evita apilar, asegura que los brazos rociadores no estén obstruidos).
    *   Presiona el botón "Start/Pause" para comenzar el programa. La pantalla o las luces indicadoras mostrarán el estado del ciclo.

5.  **Notas**:
    *   Para cambiar un programa a mitad del ciclo, es posible que necesites presionar "Start/Pause" para pausar, y luego seleccionar un nuevo programa (algunos programas pueden no permitir cambios a mitad del ciclo).
    *   **Drenaje**: El modelo Q7 drena automáticamente después de que el ciclo se completa, por lo que no se requiere una configuración manual de drenaje. Asegúrate de que la manguera de desagüe esté correctamente instalada para evitar fugas.
