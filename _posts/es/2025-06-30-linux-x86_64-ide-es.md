Según la salida de `uname -a`, tu computadora está ejecutando un sistema Linux con una arquitectura `x86_64`. Por lo tanto, debes descargar la versión **Linux** del IDE Rust de JetBrains (CLion con el plugin Rust o IntelliJ IDEA con el plugin Rust) desde el enlace proporcionado, no la versión **Linux ARM64**, ya que ARM64 es para procesadores basados en ARM, no para x86_64.

Descarga la versión **Linux** desde: https://www.jetbrains.com/rust/download/?section=linux
