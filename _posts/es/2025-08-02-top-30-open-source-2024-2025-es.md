### Los 30 principales proyectos de código abierto en tendencia en 2024-2025

Basado en datos agregados de varias fuentes que rastrean las estrellas ganadas en GitHub, la velocidad del proyecto, las listas de tendencias y el impacto en la comunidad, aquí hay una lista compilada de los 30 principales proyectos de código abierto que ganaron tracción significativa o se lanzaron alrededor de 2024-2025. El enfoque está en proyectos con alto crecimiento en estrellas, contribuciones o relevancia en IA, herramientas para desarrolladores y otras áreas emergentes. He priorizado aquellos con un impulso notable en el período de tiempo especificado, extrayendo de innovaciones lideradas por startups, avances en IA y repositorios amigables para los contribuyentes. Los proyectos están clasificados aproximadamente por métricas de popularidad como estrellas ganadas o menciones en varias fuentes.

1.  **Ollama**
    Una herramienta de código abierto para ejecutar modelos de lenguaje grande (LLM) como Llama de Meta localmente en equipos de escritorio. Ganó 76,000 estrellas en 2024.

2.  **Zed**
    Un editor de código colaborativo de alto rendimiento para humanos e IA, diseñado para monorepositorios. Liberado como código abierto en enero de 2024, ganó más de 50,000 estrellas.

3.  **Dify (LangGenius)**
    Una plataforma de código abierto para construir aplicaciones impulsadas por LLM. Ganó 43,000 estrellas en 2024.

4.  **ComfyUI**
    Una herramienta basada en nodos para generar imágenes, videos y audio con modelos de IA generativa. Creció a 61,900 estrellas con un aumento del 195%.

5.  **OpenHands (All Hands AI)**
    Una plataforma para construir agentes de desarrollo de software con IA que pueden programar y solucionar problemas. Lanzada en marzo de 2024, ganó 39,600 estrellas.

6.  **UV (Astral)**
    Un gestor de paquetes y proyectos de Python rápido, escrito en Rust, para flujos de trabajo de desarrollador de alto rendimiento. Ganó 32,800 estrellas.

7.  **Stirling PDF**
    Una herramienta de manipulación de PDFs centrada en la privacidad y auto-alojable, con funciones como edición y conversión.

8.  **Maybe Finance**
    Software moderno de planificación financiera y gestión de patrimonio, auto-alojable y orientado a la privacidad. Ganó 35,400 estrellas.

9.  **RustDesk**
    Software de escritorio remoto de código abierto que enfatiza la seguridad y el auto-alojamiento.

10. **Puter**
    Un sistema operativo de internet extensible para construir aplicaciones web ricas en funciones. Ganó 27,000 estrellas.

11. **RAGFlow (InfiniFlow)**
    Un motor de Generación Aumentada por Recuperación basado en comprensión profunda de documentos para aplicaciones de IA. Ganó 25,600 estrellas.

12. **Shardeum**
    Una blockchain compatible con EVM que usa fragmentación dinámica de estado para escalabilidad. Ganó 22,900 estrellas.

13. **Open-Sora (HPC-AI Tech)**
    Un modelo de generación de texto a video para IA empresarial, centrado en la eficiencia. Ganó 17,300 estrellas.

14. **Firecrawl (Mendable)**
    Una API para rastrear sitios web y convertirlos en markdown limpio o datos estructurados para uso en IA. Ganó 20,100 estrellas.

15. **OpenUI (Weights and Biases)**
    Una herramienta para crear componentes de UI mediante descripciones en lenguaje natural, integrada con flujos de trabajo de ML. Ganó 19,500 estrellas.

16. **Devika (Stition.ai)**
    Un agente de IA que comprende instrucciones, investiga y parchea vulnerabilidades de código. Ganó 18,600 estrellas.

17. **Flux (Black Forest Labs)**
    Código de inferencia para ejecutar generación y edición de imágenes con modelos de IA visual avanzados.

18. **Open WebUI MCP**
    Un servidor proxy que convierte herramientas MCP en servidores compatibles con OpenAPI para integración de aplicaciones de IA.

19. **Unbody**
    Un backend modular para software nativo de IA, que maneja percepción, memoria, razonamiento y acción.

20. **OWL**
    Un framework de colaboración multi-agente que utiliza navegadores, terminales y herramientas MCP; lidera el benchmark GAIA.

21. **F/mcptools**
    Una CLI para servidores MCP para descubrir, llamar herramientas y gestionar prompts para el desarrollo de IA.

22. **self.so (Nutlope)**
    Herramienta impulsada por IA para construir sitios web personales a partir de currículums o perfiles de LinkedIn.

23. **VoiceStar**
    Síntesis de texto a voz con control de duración, con CLI y modelos pre-entrenados.

24. **Second-Me**
    Un agente de IA "gemelo digital" que refleja el conocimiento y las preferencias del usuario para tareas como la gestión de cuentas.

25. **CSM (SesameAI Labs)**
    Un modelo de conversación de voz usando la arquitectura Llama para la generación natural de audio.

26. **Letta**
    Un formato de archivo abierto para empaquetar y portar agentes de IA entre diferentes frameworks.

27. **Blender meets Claude**
    Integra Blender con Claude AI a través de MCP para el control mediante lenguaje natural de los flujos de trabajo 3D.

28. **Appwrite**
    Una plataforma de backend de código abierto para autenticación, bases de datos, funciones y almacenamiento.

29. **Storybook**
    Un taller de frontend para construir, probar y documentar componentes de UI de forma aislada.

30. **Shadcn UI**
    Componentes de UI personalizables y accesibles para React, sin necesidad de instalación.

Estos proyectos reflejan el dominio de las herramientas impulsadas por IA, los potenciadores de la productividad del desarrollador y las soluciones centradas en la privacidad en 2024-2025. Muchos son aptos para principiantes para contribuir, con comunidades activas en GitHub. Para aquellos interesados en contribuir, consulta las etiquetas de "good first issue" de cada proyecto.
