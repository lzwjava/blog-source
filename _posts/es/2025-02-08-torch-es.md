### Funciones de torch

*   `torch.Tensor`: Un array multidimensional que contiene elementos de un único tipo de dato.
*   `torch.tensor`: Construye un tensor con datos y propiedades.
*   `torch.zeros`: Devuelve un tensor lleno de ceros.
*   `torch.ones`: Devuelve un tensor lleno de unos.
*   `torch.arange`: Devuelve un tensor 1-D de valores espaciados uniformemente.
*   `torch.linspace`: Devuelve un tensor 1-D de valores espaciados uniformemente en un intervalo especificado.
*   `torch.rand`: Devuelve un tensor lleno de números aleatorios de una distribución uniforme en el intervalo [0, 1).
*   `torch.randn`: Devuelve un tensor lleno de números aleatorios de una distribución normal con media 0 y varianza 1.
*   `torch.empty`: Devuelve un tensor con datos no inicializados.
*   `torch.full`: Crea un tensor de un tamaño especificado lleno con un valor especificado.
*   `torch.eye`: Devuelve un tensor 2-D con unos en la diagonal y ceros en el resto.

### Operaciones con tensores

*   `torch.add`: Suma dos tensores elemento por elemento.
*   `torch.sub`: Resta dos tensores elemento por elemento.
*   `torch.mul`: Multiplica dos tensores elemento por elemento.
*   `torch.div`: Divide dos tensores elemento por elemento.
*   `torch.matmul`: Realiza una multiplicación de matrices.
*   `torch.pow`: Eleva cada elemento de un tensor a una potencia.
*   `torch.exp`: Calcula el exponencial de cada elemento de un tensor.
*   `torch.log`: Calcula el logaritmo natural de cada elemento de un tensor.
*   `torch.sqrt`: Calcula la raíz cuadrada de cada elemento de un tensor.
*   `torch.abs`: Calcula el valor absoluto de cada elemento de un tensor.
*   `torch.neg`: Niega cada elemento de un tensor.
*   `torch.round`: Redondea cada elemento de un tensor al entero más cercano.
*   `torch.floor`: Devuelve el valor piso de cada elemento de un tensor.
*   `torch.ceil`: Devuelve el valor techo de cada elemento de un tensor.
*   `torch.clamp`: Limita todos los elementos de la entrada al rango [mín, máx].
*   `torch.sum`: Devuelve la suma de todos los elementos del tensor de entrada.
*   `torch.mean`: Devuelve la media de todos los elementos del tensor de entrada.
*   `torch.std`: Devuelve la desviación estándar de todos los elementos del tensor de entrada.
*   `torch.var`: Devuelve la varianza de todos los elementos del tensor de entrada.
*   `torch.max`: Devuelve el valor máximo de todos los elementos del tensor de entrada.
*   `torch.min`: Devuelve el valor mínimo de todos los elementos del tensor de entrada.
*   `torch.argmax`: Devuelve el índice del valor máximo de todos los elementos del tensor de entrada.
*   `torch.argmin`: Devuelve el índice del valor mínimo de todos los elementos del tensor de entrada.
*   `torch.sort`: Ordena los elementos del tensor de entrada a lo largo de una dimensión dada.
*   `torch.topk`: Devuelve los k elementos más grandes del tensor de entrada a lo largo de una dimensión dada.
*   `torch.reshape`: Devuelve un tensor con los mismos datos y número de elementos que la entrada, pero con la forma especificada.
*   `torch.transpose`: Devuelve una vista del tensor de entrada con sus dimensiones intercambiadas.
*   `torch.squeeze`: Devuelve un tensor con todas las dimensiones de entrada de tamaño 1 eliminadas.
*   `torch.unsqueeze`: Devuelve un nuevo tensor con una dimensión de tamaño uno insertada en la posición especificada.
*   `torch.cat`: Concatena los tensores dados en la dimensión dada.
*   `torch.stack`: Concatena una secuencia de tensores a lo largo de una nueva dimensión.
*   `torch.chunk`: Divide un tensor en un número específico de fragmentos.
*   `torch.split`: Divide un tensor en fragmentos de un tamaño específico.

### Módulos de redes neuronales

*   `torch.nn.Module`: Clase base para todos los módulos de redes neuronales.
*   `torch.nn.Linear`: Aplica una transformación lineal a los datos entrantes.
*   `torch.nn.Conv2d`: Aplica una convolución 2D sobre una señal de entrada compuesta por varios planos de entrada.
*   `torch.nn.MaxPool2d`: Aplica un max pooling 2D sobre una señal de entrada.
*   `torch.nn.ReLU`: Aplica la función de unidad lineal rectificada elemento por elemento.
*   `torch.nn.Sigmoid`: Aplica la función sigmoide elemento por elemento.
*   `torch.nn.Tanh`: Aplica la función tangente hiperbólica elemento por elemento.
*   `torch.nn.BatchNorm2d`: Aplica Normalización por Lotes sobre una entrada 4D.
*   `torch.nn.Dropout`: Durante el entrenamiento, pone a cero algunos de los elementos del tensor de entrada con probabilidad p.
*   `torch.nn.Embedding`: Una tabla de búsqueda simple que almacena embeddings de un diccionario y tamaño fijos.

### Funciones de pérdida

*   `torch.nn.MSELoss`: Crea un criterio que mide el error cuadrático medio (norma L2 al cuadrado) entre cada elemento de la entrada y el objetivo.
*   `torch.nn.CrossEntropyLoss`: Este criterio calcula la pérdida de entropía cruzada entre la entrada y el objetivo.
*   `torch.nn.BCELoss`: Crea un criterio que mide la Entropía Cruzada Binaria entre el objetivo y la salida.
*   `torch.nn.L1Loss`: Crea un criterio que mide el error absoluto medio (MAE) entre cada elemento de la entrada y el objetivo.

### Optimizadores

*   `torch.optim.SGD`: Implementa el descenso de gradiente estocástico (opcionalmente con momento).
*   `torch.optim.Adam`: Implementa el algoritmo Adam.
*   `torch.optim.RMSprop`: Implementa el algoritmo RMSprop.

### Autograd

*   `torch.autograd.grad`: Calcula y devuelve la suma de los gradientes de las salidas con respecto a las entradas.

### Utilidades

*   `torch.device`: Representa el dispositivo en el que se asigna o asignará un torch.Tensor.
*   `torch.cuda.is_available`: Devuelve True si CUDA está disponible.
*   `torch.save`: Guarda un tensor en el disco.
*   `torch.load`: Carga un tensor desde el disco.

### Otros

*   `torch.no_grad`: Administrador de contexto que deshabilita el cálculo de gradientes.
*   `torch.set_grad_enabled`: Habilita o deshabilita el grad, dependiendo de su argumento.
