Para configurar tu router con una dirección manual, puerta de enlace y DNS, sigue estos pasos. Esta guía asume que estás configurando la interfaz WAN (Red de Área Amplia) del router con una configuración IP estática proporcionada por tu Proveedor de Servicios de Internet (ISP), ya que este es un escenario común para este tipo de ajustes. Además, incluye una configuración básica de la LAN (Red de Área Local) para garantizar que todo funcione sin problemas.

### Paso 1: Acceder a la Interfaz Web del Router
- **Encuentra la Dirección IP del Router**: Suele ser algo como `192.168.0.1` o `192.168.1.1`. Consulta la etiqueta en tu router o el manual de usuario para obtener la dirección exacta.
- **Abre un Navegador**: Inicia un navegador web en un dispositivo conectado al router (vía Wi-Fi o Ethernet).
- **Ingresa la Dirección IP**: Escribe la dirección IP del router en la barra de direcciones y presiona Enter. Esto debería mostrar la página de inicio de sesión del router.

### Paso 2: Iniciar Sesión
- **Ingresa las Credenciales**: Se te solicitará un nombre de usuario y una contraseña. Las credenciales predeterminadas a menudo se enumeran en el router o en el manual (por ejemplo, usuario: `admin`, contraseña: `password` o `admin`).
- **Consejo de Seguridad**: Si no has cambiado estos valores predeterminados, considera actualizarlos más tarde para proteger tu router.

### Paso 3: Navegar a la Configuración WAN o de Internet
- **Localiza la Sección**: En la interfaz del router, busca una pestaña o menú etiquetado como **Configuración WAN**, **Configuración de Internet** o similar. Aquí es donde configuras la conexión del router a internet.

### Paso 4: Seleccionar Configuración IP Estática o Manual
- **Elige la Opción**: Por defecto, muchos routers usan DHCP (asignación automática de IP). Cambia esto a **IP Estática** o **Manual** para ingresar tus propios ajustes.

### Paso 5: Ingresar la Configuración Manual
- **Introduce los Detalles**: Tu ISP debería haber proporcionado lo siguiente:
  - **Dirección IP**: Una dirección única para tu router en la red del ISP (por ejemplo, `203.0.113.5`).
  - **Máscara de Subred**: Define el rango de la red (por ejemplo, `255.255.255.0`).
  - **Puerta de Enlace**: La dirección IP de la puerta de enlace del ISP que conecta tu router a internet (por ejemplo, `203.0.113.1`).
  - **Servidores DNS**: Al menos uno, a menudo dos, direcciones de servidores DNS (por ejemplo, `8.8.8.8` y `8.8.4.4` para los DNS de Google).
- **Completa los Campos**: Ingresa estos valores exactamente como se proporcionaron en los campos correspondientes en la configuración WAN.

### Paso 6: Guardar los Cambios
- **Aplica la Configuración**: Haz clic en **Guardar**, **Aplicar** o un botón similar. Es posible que el router se reinicie para aplicar la nueva configuración. Espera a que se reconecte a internet.

### Paso 7: Configurar la LAN (Opcional pero Recomendado)
- **Revisa la Configuración LAN**: Ve a la sección **Configuración LAN** o **Configuración de Red**.
- **Establece la Dirección IP LAN**: Asigna al router una IP LAN (por ejemplo, `192.168.1.1`) con una máscara de subred (por ejemplo, `255.255.255.0`). Asegúrate de que esté en un rango diferente al de la IP WAN para evitar conflictos.
- **Configuración DHCP** (si se usa):
  - **Habilita DHCP**: Si deseas que el router asigne direcciones IP a los dispositivos automáticamente.
  - **Establece el Rango de IP**: Define un rango (por ejemplo, `192.168.1.100` a `192.168.1.200`).
  - **Puerta de Enlace**: Establece esto como la IP LAN del router (por ejemplo, `192.168.1.1`).
  - **DNS**: Usa los servidores DNS de tu ISP o alternativos como `8.8.8.8` y `8.8.4.4`.
- **Guarda los Cambios**: Aplica también esta configuración.

### Notas
- **Variación por Router**: Los nombres exactos de los menús y las opciones pueden variar según la marca y el firmware de tu router (por ejemplo, TP-Link, Netgear, ASUS). Consulta el manual de tu router si los pasos no son claros.
- **Prueba la Conexión**: Después de guardar, verifica si tu internet funciona cargando una página web en un dispositivo conectado.
- **Contacta a tu ISP**: Si no estás seguro acerca de los valores de IP, puerta de enlace o DNS, contacta a tu ISP para obtener los detalles correctos de la configuración estática.

Siguiendo estos pasos, tu router debería estar configurado con una dirección manual, puerta de enlace y DNS, garantizando una conexión de red estable y personalizada.
