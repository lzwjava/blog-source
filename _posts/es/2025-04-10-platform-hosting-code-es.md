Además de GitHub, otras plataformas famosas para alojar código incluyen:

- GitLab: Una plataforma DevOps completa que proporciona alojamiento de repositorios, CI/CD, monitorización y más
- Bitbucket: Propiedad de Atlassian, se integra bien con Jira y otros productos de Atlassian
- SourceForge: Una de las plataformas de alojamiento de código más antiguas, particularmente popular para proyectos de código abierto
- Azure DevOps (anteriormente Visual Studio Team Services): La solución de Microsoft para alojamiento de código y DevOps
- AWS CodeCommit: El servicio Git gestionado de Amazon
- Gitee: Popular en China, a menudo llamado el "GitHub de China"
- Launchpad: Utilizado principalmente para Ubuntu y proyectos de código abierto relacionados
- CodePen: Especializado en código front-end (HTML, CSS, JavaScript)
- Codeberg: Una alternativa sin fines de lucro impulsada por la comunidad, enfocada en software libre y de código abierto
- Replit: Combina el alojamiento de código con un IDE en línea

Estas plataformas varían en sus características, modelos de precios y audiencias objetivo, con algunas enfocándose más en usuarios empresariales y otras en comunidades de código abierto.
