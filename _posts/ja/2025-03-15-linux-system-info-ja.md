Ubuntuでは、以下のコマンドを使用して、システムが32ビットか64ビットか、OSのバージョン、RAMの容量、ディスクサイズなどのシステム情報を取得できます。以下にステップバイステップのガイドを示します：

### 1. システムが32ビットか64ビットかを確認する
- **コマンド**: `uname -m`
- **説明**: このコマンドはマシンのハードウェア名を表示します。
  - 出力が `x86_64` の場合、システムは **64ビット** です。
  - 出力が `i686` または `i386` の場合、システムは **32ビット** です。
- **出力例**:
  ```
  x86_64
  ```
  これは64ビットシステムを示しています。

### 2. オペレーティングシステム（OS）のバージョンを取得する
- **コマンド**: `lsb_release -a`
- **説明**: このコマンドは、Linux Standard Base（LSB）とディストリビューションに関する詳細情報を提供します。
  - Ubuntuのバージョンを確認するには、`Description` の行を参照してください。
- **出力例**:
  ```
  No LSB modules are available.
  Distributor ID: Ubuntu
  Description:    Ubuntu 20.04.3 LTS
  Release:        20.04
  Codename:       focal
  ```
  ここでは、OSは **Ubuntu 20.04.3 LTS** です。

### 3. RAMのサイズを取得する
- **コマンド**: `free -h`
- **説明**: このコマンドは、メモリ使用量を人間が読みやすい形式（例：GBまたはMB）で表示します。
  - インストールされている合計RAMを確認するには、`Mem:` 行の `total` 列を参照してください。
- **出力例**:
  ```
              total        used        free      shared  buff/cache   available
  Mem:          15Gi       2.3Gi       9.5Gi       300Mi       3.8Gi        12Gi
  Swap:        2.0Gi         0B       2.0Gi
  ```
  この場合、合計RAMは **15 GB** です。

### 4. ディスクサイズを取得する
- **コマンド**: `lsblk -d -o NAME,SIZE`
- **説明**: このコマンドは、すべてのディスクデバイス（パーティションを除く）とそのサイズを一覧表示します。
  - `-d` フラグは、パーティションではなくディスクデバイス（例: `/dev/sda`）のみを表示することを保証します。
  - ディスク名（例: `sda`）とそれに対応するサイズを参照してください。
- **出力例**:
  ```
  NAME   SIZE
  sda    1T
  sdb    500G
  ```
  ここでは、システムには2つのディスクがあります：サイズ **1 テラバイト** の `/dev/sda` と **500 ギガバイト** の `/dev/sdb` です。ディスクが1つしかない場合、通常は `sda` になります。

### 注意点
- これらのコマンドは、追加のインストールを必要とせず、Ubuntuで利用可能な標準ツールを使用しています。
- ディスクサイズについては、システムに複数のディスクがある場合、出力にはすべてがリストされます。通常、`/dev/sda` がプライマリディスクです。
- これらのコマンドをターミナルで実行すると、あなたのシステムに固有の結果を確認できます。

これらのコマンドを使用すると、Ubuntuマシンで必要な基本的なシステム情報を素早く収集できます！
