## 9.2 ネットワークフロー問題

### 9.2.1 ネットワークフローの基礎

ネットワークは、ノード（頂点）がエッジで接続された数学的構造です。多くの実用的な状況では、これらのネットワークを通じた最適なフローを決定する必要があります：

- **交通ネットワーク**：工場から倉庫への商品の移動
- **通信ネットワーク**：インターネットを通じたデータパケットのルーティング
- **ユーティリティネットワーク**：電気、水、ガスの配分

ネットワークフロー問題は、行列を使用して優雅に表現できます：

- **接続行列** A はネットワーク構造を表します
- ベクトル x は各エッジに沿った流量を表します
- 制約はノードでの流量保存を保証します

### 9.2.2 最大フロー最小カット定理

ネットワーク理論における最も重要な結果の一つは、最大フローと最小カットを結びつけます：

1. ネットワークを通る最大フローは、最小カットの容量に等しい
2. この双対性は線形代数を使用して表現でき、以下のような技術で解決できます：
   - Ford-Fulkerson アルゴリズム
   - 線形計画法の定式化

## 9.3 データフィッティングと最小二乗法

### 9.3.1 線形回帰

データポイントに直線や曲線をフィッティングする場合、予測値と実際の値の間の誤差を最小化する関数を求めています：

- 線形回帰では、y = mx + b のパラメータを見つけたい
- 複数のデータポイントでは、これは過剰決定系になります
- 最小二乗解は、二乗誤差の和を最小化します

### 9.3.2 正規方程式

最適解は以下を使用して見つけることができます：
- A^T A x = A^T b
- ここで A は計画行列、b は出力ベクトルです
- 解 x は最適なパラメータを与えます

### 9.4.3 コード例：線形システムの解法

```python
import numpy as np

# 線形方程式系を行列として定義
A = np.array([[3, 1, -1], 
              [1, 4, 1], 
              [2, 1, 2]])
              
b = np.array([9, 7, 8])

# 連立方程式 Ax = b を解く
x = np.linalg.solve(A, b)

print("解:")
print(f"x = {x[0]}")
print(f"y = {x[1]}")
print(f"z = {x[2]}")
```
