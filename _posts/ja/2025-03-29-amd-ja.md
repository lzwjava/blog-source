Advanced Micro Devices, Inc. (AMD) は、マイクロプロセッサ、グラフィックスプロセッシングユニット (GPU)、その他のコンピューティング技術で知られるグローバルな半導体企業です。1969年に設立され、AMD はインテルや NVIDIA などの巨人と競争しながら、コンピューティング産業を形成する上で極めて重要な役割を果たしてきました。以下は、AMD の起源、マイルストーン、課題、進化を辿る同社の歴史に関する包括的な紹介です。

---

### 創業と初期の年代 (1969–1980年代)
AMD は、1969年5月1日、Jerry Sanders III と、Edwin Turney、Jack Gifford を含む他の7人のフェアチャイルドセミコンダクターの元幹部によって設立されました。カリフォルニア州サニーベールに本社を置き、AMD は他社が設計した集積回路をライセンスの下で製造するセカンドソースメーカーとして始まりました。これは、初期の半導体産業においてサプライチェーンの安定を確保するための一般的な慣行でした。

-   **初期の焦点**: AMD は当初、ロジックチップとメモリデバイスを製造し、インテル、テキサス・インスツルメンツ、モトローラなどの企業と競争しました。その最初の製品は、1970年にリリースされた4ビットシフトレジスタの Am9300 でした。
-   **セカンドソース契約**: 1970年代、AMD はインテルの 8080 マイクロプロセッサを含むプロセッサの主要なセカンドソース供給業者となりました。この関係は、AMD がマイクロプロセッサ製造の専門知識を構築するのに役立ちました。
-   **株式公開**: AMD は1972年に株式公開し、事業拡大のための資金を調達しました。
-   **インテルとの法廷闘争**: 1970年代後半までに、AMD の成長する野心はインテルとの緊張を招きました。1976年、AMD は x86 互換チップを生産するためのクロスライセンス契約をインテルと結び、将来の直接競合相手としての地盤を築きました。

1980年代、AMD は製品ポートフォリオと製造能力を拡大しました:
-   **マイクロプロセッサ**: AMD は、ミニコンピュータや産業用アプリケーションで使用された Am2900 シリーズのビットスライスプロセッサなど、独自設計の生産を開始しました。
-   **x86 プロセッサ**: インテルとの契約の下、AMD は初期のパーソナルコンピュータ (PC) を駆動した 8086 や 80286 などの x86 プロセッサを製造しました。
-   **課題**: 1980年代の PC 市場の成長は競争を激化させました。AMD は、インテルが 80386 のような新しい設計への AMD のアクセスを制限しようとしたため、x86 ライセンス契約をめぐるインテルとの法的紛争に直面しました。

### 競合他社としての台頭 (1990年代)
1990年代は、インテルとの競争に後押しされ、AMD がセカンドソース供給業者からマイクロプロセッサ市場の主要な革新者へと変貌を遂げた時代でした。

-   **法的勝利**: 1991年、AMD はインテルに対する画期的な法廷闘争に勝利し、x86 互換プロセッサを生産する権利を確保しました。これにより、AMD は独自のマイクロアーキテクチャを開発できるようになりました。
-   **K5 および K6 プロセッサ**: 1996年、AMD は独自初の x86 プロセッサである K5 をリリースし、続いて1997年に K6 をリリースしました。K6 は商業的成功を収め、インテルの Pentium プロセッサよりも低価格で競争力のある性能を提供しました。
-   **NexGen の買収**: 1996年、AMD は x86 互換プロセッサを開発していた企業 NexGen を買収しました。NexGen の技術は、後の AMD の設計に影響を与えました。
-   **Athlon の突破口**: 1999年、AMD は Athlon (K7) プロセッサを発表しました。これはゲームチェンジャーであり、多くのベンチマークでインテルの Pentium III を性能で上回りました。Athlon は、高性能 PC 市場において AMD が本格的な競合他社であることを確立しました。

この期間中、AMD はフラッシュメモリやその他の半導体製品にも進出し、サムスンやマイクロンなどの企業と競争するために製品ポートフォリオを多様化しました。

### 課題と回復 (2000年代)
2000年代前半は、重要な成果があったものの、財務的苦境と激しい競争に影を落とされた混在した時期でした。

-   **Opteron と Athlon 64 (2003年)**: AMD は x86 命令セットを64ビットに拡張する x86-64 アーキテクチャを導入しました。Opteron (サーバー向け) と Athlon 64 (デスクトップ向け) は、メインストリームコンピューティング向け初の64ビットプロセッサであり、AMD にインテルに対する技術的優位性をもたらしました。これらのチップは、特にデータセンターで広く採用されました。
-   **ATI Technologies の買収 (2006年)**: AMD は、主要な GPU メーカーである ATI を 54億ドルで買収しました。この動きは、AMD のポートフォリオをグラフィックスカード (Radeon) とチップセットにまで拡大し、統合コンピューティングソリューションにおいて NVIDIA およびインテルと競争する態勢を整えました。
-   **財務的苦境**: 技術的成功にもかかわらず、インテルの積極的な価格設定、市場支配、および ATI 買収に伴う AMD の多額の負債により、AMD は財務的課題に直面しました。2000年代後半までに、AMD は赤字と市場シェアの減少に陥りました。
-   **Barcelona の欠陥**: 2007年に発売された Phenom および Opteron 「Barcelona」プロセッサは、重大なバグ (TLB エラッタ) により汚点が残り、サーバー市場における AMD の評判を損ないました。
-   **製造部門の分離**: 2008年、AMD は製造部門を GlobalFoundries として分離しました。これはアブダビの Advanced Technology Investment Company が支援する別会社です。これにより、AMD はファブレス半導体企業となり、チップ生産を GlobalFoundries や TSMC などの第三者ファウンドリに依存することになりました。

### Ryzen と EPYC によるルネサンス (2010年代)
2010年代は、新しいリーダーシップの下での顕著な業績回復と革新的な製品によって特徴づけられる、AMD にとって変革の10年でした。

-   **Bulldozer の失望 (2011年)**: 高いコア数を目指して設計された AMD の Bulldozer アーキテクチャは、シングルスレッドタスクで性能が振るわず、インテルの Core シリーズに対抗できませんでした。これにより、市場シェアのさらなる喪失と財務的圧迫を招きました。
-   **リーダーシップの変更**: 2014年、Dr. Lisa Su が AMD の CEO に就任し、エンジニアリングの卓越性と戦略的再構築に焦点を当てました。
-   **Zen アーキテクチャと Ryzen (2017年)**: 2017年に Ryzen プロセッサで発売された AMD の Zen アーキテクチャは、大成功を収めました。Ryzen は、インテルの製品と比較して競争力のある性能、高いコア数、優れた価値を提供し、デスクトップおよびノートパソコン市場で AMD の市場シェアを大幅に回復させました。
-   **サーバー向け EPYC**: また2017年、AMD はデータセンター向けに EPYC プロセッサを導入し、インテルの Xeon 支配に挑戦しました。EPYC の高いコア数、効率性、競争力のある価格設定は、AWS、Microsoft Azure、Google Cloud などの主要クラウドプロバイダーを獲得しました。
-   **Radeon および Instinct GPU**: AMD は、ゲーミングおよびプロフェッショナル市場向けの Radeon GPU の開発を継続しました。また、AI とハイパフォーマンスコンピューティング (HPC) 向けに Instinct アクセラレーターを立ち上げ、NVIDIA のデータセンター支配を標的としました。
-   **7nm プロセスのリーダーシップ**: 2019年、AMD は Ryzen 3000 および EPYC Rome プロセッサに TSMC の 7nm プロセスを採用し、10nm 移行に苦戦していたインテルを追い抜きました。これにより、AMD は性能と効率性で優位に立ちました。

この10年の終わりまでに、AMD の株価は急騰し、CPU および GPU 市場で重要な地盤を回復しました。Lisa Su のリーダーシップは広く称賛され、AMD を半導体産業のリーダーとして位置づけました。

### 現代と継続的な成長 (2020年代–現在)
2020年代において、AMD は CPU、GPU、および AI や HPC のような新興技術における強力な地位を固めました。

-   **Ryzen 5000 および 7000 シリーズ**: AMD は Zen アーキテクチャの反復を継続し、Zen 3 (2020年) および Zen 4 (2022年) が Ryzen 5000 および 7000 シリーズプロセッサを駆動しました。これらのチップは、ゲーミング、生産性、電力効率において AMD の優位性を維持しました。
-   **3D V-Cache 技術**: 2022年、AMD は Ryzen 7 5800X3D および後のプロセッサに 3D V-Cache を導入し、追加のキャッシュメモリを積層してゲーミング性能を向上させました。これはコンシューマー CPU では初めてのことでした。
-   **EPYC Genoa および Bergamo**: AMD の第4世代 EPYC プロセッサ (Zen 4, 2022–2023) は、コア数を 96 (Genoa) および 128 (Bergamo) にまで押し上げ、クラウドおよびエンタープライズ市場を支配しました。
-   **GPU の進歩**: AMD の RDNA 3 アーキテクチャ (Radeon RX 7000 シリーズ, 2022年) はゲーミング性能を向上させましたが、NVIDIA の RTX 40 シリーズからの激しい競争に直面しました。AMD はまた、AI ワークロード向けに Instinct MI300 シリーズを拡大し、NVIDIA の H100 および A100 GPU と競合しました。
-   **Xilinx の買収 (2022年)**: AMD は、フィールドプログラマブルゲートアレイ (FPGA) のリーダーである Xilinx を 490億ドルで買収しました。これは、組み込みシステム、自動車、データセンター市場における AMD のポートフォリオを強化しました。
-   **AI と HPC への焦点**: AMD は、Instinct MI300X のような製品や、AI インフラストラクチャにおけるマイクロソフトなどの企業とのパートナーシップを通じて、AI と HPC を標的とすることを強化しています。AMD のチップは、2025年現在世界最速のスーパーコンピュータ Frontier などを駆動します。
-   **チップレットのリーダーシップ**: プロセッサがより小さなモジュール式チップレットから構築される AMD のチップレットベース設計は、コストと性能の利点を提供し、業界標準となりました。インテル他社もその後、同様のアプローチを採用しています。

### AMDの歴史における主要なテーマ
1.  **インテルおよび NVIDIA との競争**: AMD の歴史は、CPU におけるインテル、GPU における NVIDIA との競争によって定義されています。多くの場合アンダードッグでありながら、AMD の革新 (Athlon, Ryzen, EPYC) は市場を繰り返し混乱させてきました。
2.  **技術的革新**: x86-64 からチップレット、3D V-Cache まで、AMD は産業を形成する画期的な技術を導入してきました。
3.  **財務的変動**: AMD は、特に2000年代と2010年代前半に財務的不安定の時期に直面しましたが、GlobalFoundries への製造部門分離や Lisa Su のリーダーシップといった戦略的決定により業績を回復させました。
4.  **多様化**: ATI や Xilinx のような買収を通じて、AMD は CPU を超えて GPU、FPGA、AI アクセラレーターに事業を拡大し、多様な市場での成長に向けた態勢を整えました。
5.  **レジリエンス**: 法廷闘争、技術的挫折、市場の課題にもかかわらず、AMD の復活する能力は、同社を技術分野で尊敬される名前にしました。

### 現在のAMD (2025年)
2025年5月2日現在、AMD は時価総額 2000億ドルを超える半導体リーダーです。その製品は、PC、サーバー、ゲーム機 (例: PlayStation 5, Xbox Series X/S)、スーパーコンピュータ、AI システムを駆動します。Lisa Su のリーダーシップの下、AMD は Zen 5 プロセッサ、RDNA 4 GPU、AI に焦点を当てたソリューションで革新を続けています。同社のチップレット、AI、戦略的買収への焦点は、急速に進化する技術環境において継続的な成長への態勢を整えています。

AMD の、セカンドソース供給業者からグローバルな革新者への旅路は、産業の巨人に挑戦するその適応性、技術力、決意を反映しています。その歴史は、技術進歩を推進する上で競争が重要であることの証左です。
