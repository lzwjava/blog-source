VSCodeでマウスカーソルが十字/プラス記号に変わる場合、通常はVSCodeが特殊モードになっているか、設定に問題があることを示しています。以下に一般的な原因と解決策を示します：

## 一般的な原因：

1. **列選択モード**: 誤って列/ブロック選択モードを有効にした可能性があります
   - `Alt + Shift + I` または `Ctrl + Shift + Alt + 矢印キー` を押してこのモードを終了してみてください

2. **拡張機能の競合**: 一部の拡張機能がカーソルの動作を変更することがあります
   - 拡張機能を一時的に無効にしてみてください：`Ctrl + Shift + P` → "Developer: Reload Window"
   - またはセーフモードでVSCodeを起動：`code --disable-extensions`

3. **ズーム/選択ツール**: ズームまたは選択ツールを有効にした可能性があります
   - `Escape` キーを数回押してください
   - 最近の操作を元に戻すために `Ctrl + Z` を試してみてください

4. **ターミナルのフォーカス問題**: ターミナルにフォーカスがあると、カーソルの動作が変わる可能性があります
   - エディターエリアをクリックしてフォーカスを戻してください
   - `Ctrl + 1` を押してエディターにフォーカスを移してみてください

## クイックフィックス：

1. **VSCodeの再起動**: シンプルですが効果的なことが多いです
   - `Ctrl + Shift + P` → "Developer: Reload Window"

2. **マウス設定の確認**:
   - ファイル → 基本設定 → 設定 に移動
   - "cursor" を検索してカーソル関連の設定を確認してください

3. **ワークスペースのリセット**:
   - VSCodeを閉じる
   - プロジェクト内の `.vscode` フォルダを削除（存在する場合）
   - プロジェクトを再オープン

4. **システム全体の問題確認**:
   - 他のアプリケーションでもカーソル問題が発生するかテストしてください
   - もしそうなら、GNOME/システム設定の問題かもしれません

まず `Escape` を数回押してみてから、VSCodeを再起動してください。これで通常は十字カーソルの問題は解決します。
