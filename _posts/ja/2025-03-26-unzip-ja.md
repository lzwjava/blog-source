Pythonでのファイル解凍：

```python
import zipfile
import os
import argparse

# 引数解析の設定
parser = argparse.ArgumentParser(description="ZIPファイルを同じ名前のディレクトリに解凍します。")
parser.add_argument('zip_file', help="ZIPファイルへのパス")

# コマンドライン引数の解析
args = parser.parse_args()

# ZIPファイルのパスとそのディレクトリを取得
zip_file_path = args.zip_file
directory = os.path.dirname(zip_file_path)

# .zip拡張子を除去して新しいディレクトリ/フォルダ名を取得
folder_name = os.path.splitext(os.path.basename(zip_file_path))[0]
destination_folder = os.path.join(directory, folder_name)

# 出力先フォルダの存在を確認
os.makedirs(destination_folder, exist_ok=True)

# ファイルの解凍
with zipfile.ZipFile(zip_file_path, 'r') as zip_ref:
    zip_ref.extractall(destination_folder)

print(f'内容は {destination_folder} に展開されました')

```
