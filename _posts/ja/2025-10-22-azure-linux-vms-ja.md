はい、Azure Virtual Machines (VM) は Linux オペレーティングシステムの実行を完全にサポートしています。Azure Marketplace から人気のさまざまなディストリビューションを直接デプロイしたり、独自のカスタム Linux イメージをアップロードしたりできます。

### 主な特徴:
- **サポートされているディストリビューション**: Azure は、Ubuntu、Red Hat Enterprise Linux (RHEL)、Debian、SUSE Linux Enterprise Server (SLES)、Oracle Linux、CentOS Stream、Rocky Linux、AlmaLinux など、主要な Linux フレーバーを承認およびサポートしています。100 以上の Linux イメージがすぐに利用できます。
- **デプロイオプション**: Azure portal、CLI、PowerShell、または ARM テンプレートを使用して Linux VM を作成できます。例えば、Ubuntu Server インスタンスを数分で迅速に起動できます。
- **機能**: Azure 上の Linux VM には、ストレージ、ネットワーキング、セキュリティ、監視などの Azure サービスとの完全な統合が含まれます。また、ハイブリッド管理のための Azure Arc などのツールもサポートしています。

初めての場合は、Linux VM 作成の公式クイックスタートガイドを確認してください。
