ビット操作は、ビット演算子を使用して問題を効率的に解決します。単一の数値を見つける方法と、数値が2の累乗かどうかを確認する方法について見ていきましょう。

## 1. 単一の数値: XORの魔法
すべての他の数値が2回現れる配列において、1回だけ現れる数値をXORを使用して見つけます。

### Java実装
```java
public class SingleNumber {
    public static int singleNumber(int[] nums) {
        int result = 0;
        for (int num : nums) result ^= num;
        return result;
    }

    public static void main(String[] args) {
        int[] nums = {2, 2, 1, 1, 4};
        System.out.println("Single number: " + singleNumber(nums));
    }
}
```
**出力:** `Single number: 4`

## 2. 2の累乗: ビットチェック
数値が2の累乗である場合、その2進数表現には1のビットが正確に1つだけ含まれます。

### Java実装
```java
public class PowerOfTwo {
    public static boolean isPowerOfTwo(int n) {
        return n > 0 && (n & (n - 1)) == 0;
    }

    public static void main(String[] args) {
        int n = 16;
        System.out.println(n + " is power of 2: " + isPowerOfTwo(n));
        n = 18;
        System.out.println(n + " is power of 2: " + isPowerOfTwo(n));
    }
}
```
**出力:**
```
16 is power of 2: true
18 is power of 2: false
```
