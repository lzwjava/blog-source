Windows ネットワーキングのトラブルシューティングや情報収集、管理に役立つ、コマンドプロンプト（CMD）で実行できる便利なコマンドをご紹介します。`getmac`のような定番コマンドを中心に、他にも使えるコマンドをいくつか説明します。技術に詳しくなくても大丈夫です。コマンドに慣れれば、とても簡単に使えるようになります。

まず、これらのコマンドを使うには、コマンドプロンプトを開きます: `Windows キー + R` を押し、`cmd` と入力して Enter キーを押します。簡単ですね。

### 1. `getmac` - MAC アドレスを確認する
`getmac` コマンドは、ネットワークアダプター（イーサネットや Wi-Fi など）のメディアアクセス制御（MAC）アドレス、つまりネットワーク上でのデバイスの一意の識別子を表示します。使い方は以下の通りです。

- `getmac` と入力して Enter キーを押します。
- 各ネットワークアダプターの MAC アドレスの一覧が表示されます。
- `-v` スイッチを追加すると (`getmac -v`)、詳細モードとなり、アダプター名やトランスポートの種類（例: イーサネットやワイヤレス）などの追加情報が得られます。

表示例は以下のようになります:
```
物理アドレス          トランスポート名
=================== ==========================================================
00-14-22-01-23-45   \Device\Tcpip_{12345678-ABCD-1234-EF56-7890ABCDEF12}
```
「物理アドレス」があなたの MAC アドレスです。ネットワークのトラブルシューティングや、ルーターでの MAC アドレスフィルタリング設定に便利です。

### 2. `ipconfig` - IP 構成を確認する
これはネットワーク情報の定番コマンドです:
- `ipconfig` と入力して Enter キーを押すと、IP アドレス、サブネットマスク、デフォルトゲートウェイなどの基本情報が表示されます。
- `ipconfig /all` を使用すると、DNS サーバー、DHCP ステータス、そしてもちろん MAC アドレスも含む完全な詳細が表示されます。

デバイスが正しく接続されているか、IP アドレスの競合が発生していないかを確認するのに最適です。

### 3. `ping` - 接続をテストする
別のデバイスや Web サイトに到達できるか確認したいですか？
- `ping [アドレス]` と入力します（例: `ping google.com` または `ping 8.8.8.8`）。
- いくつかのパケットを送信し、それらが返ってきたかどうか、そしてどのくらい時間がかかったか（ミリ秒単位）を教えてくれます。

「要求がタイムアウトしました」と表示された場合は、ファイアウォール、ダウンしたサーバー、または自身のネットワークの何かが接続を妨げている可能性があります。

### 4. `tracert` - ルートをトレースする
"trace route" の略で、データが目的地までにたどる経路を表示します:
- `tracert [アドレス]` と入力します（例: `tracert google.com`）。
- 途中の各ホップ（ルーターやサーバー）とその応答時間を一覧表示します。

接続が遅くなる、または失敗する場所を特定するのに便利です。

### 5. `netstat` - ネットワークアクティビティを確認する
これは、現在のネットワーク上の状況を確認するためのコマンドです:
- `netstat -a` と入力すると、すべてのアクティブな接続と待機中のポートが表示されます。
- `-n` を追加すると (`netstat -an`)、ホスト名の代わりに IP アドレスが表示され、より高速で生の情報が得られます。

これは、あなたのコンピューターが何と通信しているかを覗き見るようなもので、不審なアクティビティを発見するのに役立ちます。

### 6. `nslookup` - DNS を調査する
Web サイトが読み込まれない場合、このコマンドでドメインネームシステム（DNS）解決を確認します:
- `nslookup [ドメイン名]` と入力します（例: `nslookup google.com`）。
- そのドメインに関連付けられた IP アドレスを返し、DNS が機能しているかどうかを示します。

### よりスムーズに使うためのヒント
- コマンドがより高い権限を必要とする場合は、管理者として CMD を実行します（コマンドプロンプトを右クリック > 「管理者として実行」）。
- コマンドを `|` (パイプ) で組み合わせてフィルタリングできます。例: `getmac | find "00-14"` で特定の MAC アドレスを検索。
- わからなくなったら、`[コマンド] /?` (例: `ipconfig /?`) と入力すると、そのコマンドのヘルプが表示されます。

これらは、Windows ネットワークを調査するための基本的なツールです。`getmac` のように特定のコマンドに焦点を当てたいですか、それとも何か特定のことを試してみたいですか？あなたが目指していることを教えてください！
