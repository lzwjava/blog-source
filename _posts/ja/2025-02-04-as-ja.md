## 実験

このコードは「Hello, World!」プログラムを作成します。システムコールを使用してメッセージをコンソールに書き込み、その後終了します。`.text`セクションには実行可能な命令が格納され、`_start`から始まります。最初に`sys_write`コール（標準出力への表示）を設定し、次に`sys_exit`コール（正常終了）を設定します。`.data`セクションにはメッセージ自体が格納され、その長さが計算されます。

```bash
.global _start
.text

_start:
    // "Hello, World!"を標準出力に書き込み
    mov x8, #64             // writeのシステムコール番号 (sys_write)
    mov x0, #1              // ファイル記述子1 (stdout)
    ldr x1, =msg            // メッセージのアドレス
    mov x2, #14             // メッセージの長さ
    svc #0                  // システムコールを呼び出し

    // プログラムを終了
    mov x8, #93             // exitのシステムコール番号 (sys_exit)
    mov x0, #0              // 終了コード0
    svc #0                  // システムコールを呼び出し

.data
msg:
    .asciz "Hello, World!\n"
```
