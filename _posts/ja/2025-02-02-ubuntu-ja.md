## インストール

以前[自作PCの作り方](./computer-en)について書きました。今回は少し趣向を変えてみます。今日は、Windows 10がプリインストールされたLenovo Xiaoxing 14IAH8ラップトップにUbuntu 24.04をインストールします。

```bash
sudo dd if=ubuntu-24.04.1-desktop-amd64.iso of=/dev/rdisk6 bs=1m
```

Ubuntuインストーラを使用するには、設定でデバイス暗号化を無効にする必要があります。

デバイス暗号化が有効で、BIOSでセキュアブートが無効になっている場合、WindowsはBitLocker回復用パスフレーズの入力を求めます。

そのため、最初にBIOSでセキュアブートを有効にし、Windowsにログインしてから、設定でデバイス暗号化を無効にする必要があります。その後、セキュアブートを再度無効にしてUbuntuインストーラを起動できます。

## 動作停止

Ubuntuが`gdm.service`（GNOMEディスプレイマネージャ）を起動して動作が停止する場合は、通常、グラフィカルユーザーインターフェース（GUI）またはディスプレイマネージャ自体に問題があることを示しています。問題のトラブルシューティングと解決策をいくつかご紹介します。

1. **仮想ターミナルへのアクセス：**
   - システムがGDM画面で停止している場合は、ターミナルにアクセスして問題を診断してみてください。
   - `Ctrl + Alt + F2`（または`F3`、`F4`など）を押して、仮想ターミナルに切り替えます。

2. **ログイン：**
   - ユーザー名とパスワードでログインします。

3. **システムログの確認：**
   - 次のコマンドを使用して、問題の原因を示す可能性のあるエラーや警告がないかシステムログを確認します。
     ```bash
     sudo journalctl -xe
     ```

4. **GDMサービスの再起動：**
   - 問題が解決するかどうかを確認するために、GDMサービスを再起動してみてください。
     ```bash
     sudo systemctl restart gdm
     ```

5. **システムの再起動：**
   - 変更を加えた後、システムを再起動して適用します。
     ```bash
     sudo reboot
     ```
