## 🗄️ 4. データベース — SQL と NoSQL

**目標:** データの保存、クエリ、最適化を学ぶ。

**学習内容 (SQL):**

*   リレーショナルモデル
*   テーブル、インデックス、キー (主キー、外部キー)
*   CRUDクエリ
*   結合と副問い合わせ
*   トランザクション (ACID)
*   正規化と非正規化
*   クエリの最適化 (EXPLAIN, インデックス)

**学習内容 (NoSQL):**

*   ドキュメントDB (MongoDB)
*   キーバリューDB (Redis)
*   一貫性、可用性、分断耐性の違い (CAP定理)

**実践:**

*   JDBCまたはJPAを使用してMySQL/PostgreSQLに接続するJavaアプリを構築する
*   キャッシングのためにRedisにデータを保存する

## ⚡ 5. キャッシングとRedis

**目標:** キャッシングレイヤーとその使用タイミング、方法を理解する。

**学習内容:**

*   キャッシングがパフォーマンスを向上させる理由
*   Redisのデータ型 (文字列、ハッシュ、セット、ソート済みセット)
*   有効期限とエビクションポリシー
*   分散キャッシュ vs ローカルキャッシュ
*   一般的なパターン (cache-aside, write-through, write-behind)
*   セッションストレージとレート制限のユースケース

**実践:**

*   SpringとRedisを使用してJava RESTアプリにキャッシングを実装する

## 🧱 6. Spring Framework / Spring Boot

**目標:** エンタープライズJavaバックエンド開発を習得する。

**学習内容:**

*   Spring Core: ビーン、コンテキスト、依存性注入
*   Spring Boot: 自動設定、スターター、 `application.properties`
*   Spring MVC: コントローラー、RequestMapping、バリデーション
*   Spring Data JPA: リポジトリ、エンティティ、ORM (Hibernate)
*   Spring Security: 認証、認可
*   Spring AOP: 横断的関心事
*   Spring Actuator: ヘルスチェックとメトリクス

**実践:**

*   CRUD REST APIを構築する (例: ユーザー管理)
*   JWTベースのログインを追加する
*   Swagger/OpenAPIドキュメントを追加する
*   Dockerでコンテナ化する
