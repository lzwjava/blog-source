与えられた集合のうち、どれが $\mathbb{R}^2$ の部分空間であるかを判断するには、それらが部分空間の3つの条件を満たすかどうかを確認する必要があります：

1.  **ゼロベクトルが部分空間に含まれる。**
2.  **部分空間はベクトル加法について閉じている。** $\mathbf{u}$ と $\mathbf{v}$ が部分空間に含まれるならば、$\mathbf{u} + \mathbf{v}$ も部分空間に含まれる。
3.  **部分空間はスカラー倍について閉じている。** $\mathbf{u}$ が部分空間に含まれ、$c$ が任意のスカラーであるならば、$c\mathbf{u}$ も部分空間に含まれる。

それぞれの選択肢を検討してみましょう：

**a) 直線 y = x + 1**

1.  **ゼロベクトル：** $\mathbb{R}^2$ におけるゼロベクトルは $\begin{pmatrix} 0 \\ 0 \end{pmatrix}$ です。このベクトルが直線上にあるためには、方程式 $y = x + 1$ を満たさなければなりません。$x=0$ と $y=0$ を代入すると、$0 = 0 + 1$、すなわち $0 = 1$ が得られます。これは偽です。したがって、ゼロベクトルは直線 $y = x + 1$ 上にはありません。

最初の条件が満たされないため、直線 $y = x + 1$ は $\mathbb{R}^2$ の部分空間ではありません。

**b) 直線 y = 2x**

1.  **ゼロベクトル：** ゼロベクトル $\begin{pmatrix} 0 \\ 0 \end{pmatrix}$ が直線上にあるためには、方程式 $y = 2x$ を満たさなければなりません。$x=0$ と $y=0$ を代入すると、$0 = 2(0)$、すなわち $0 = 0$ が得られます。これは真です。したがって、ゼロベクトルは直線 $y = 2x$ 上にあります。

2.  **ベクトル加法についての閉性：** $\mathbf{u} = \begin{pmatrix} x_1 \\ y_1 \end{pmatrix}$ と $\mathbf{v} = \begin{pmatrix} x_2 \\ y_2 \end{pmatrix}$ を直線 $y = 2x$ 上の2つのベクトルとします。これは $y_1 = 2x_1$ および $y_2 = 2x_2$ を意味します。
    それらの和は $\mathbf{u} + \mathbf{v} = \begin{pmatrix} x_1 + x_2 \\ y_1 + y_2 \end{pmatrix}$ です。このベクトルが直線 $y = 2x$ 上にあるかどうかを確認する必要があります。成分を代入すると、$y_1 + y_2 = 2(x_1 + x_2)$ が得られます。$y_1 = 2x_1$ および $y_2 = 2x_2$ なので、$2x_1 + 2x_2 = 2x_1 + 2x_2$ となり、これは真です。したがって、この直線はベクトル加法について閉じています。

3.  **スカラー倍についての閉性：** $\mathbf{u} = \begin{pmatrix} x \\ y \end{pmatrix}$ を直線 $y = 2x$ 上のベクトルとします。すなわち $y = 2x$ です。$c$ を任意のスカラーとします。すると、$c\mathbf{u} = \begin{pmatrix} cx \\ cy \end{pmatrix}$ となります。このベクトルが直線 $y = 2x$ 上にあるかどうかを確認する必要があります。成分を代入すると、$cy = 2(cx)$ が得られます。$y = 2x$ なので、$c(2x) = 2(cx)$、すなわち $2cx = 2cx$ となります。これは真です。したがって、この直線はスカラー倍について閉じています。

3つの条件すべてが満たされているため、直線 $y = 2x$ は $\mathbb{R}^2$ の部分空間です。

**c) 単位円**

$\mathbb{R}^2$ における単位円は、方程式 $x^2 + y^2 = 1$ で定義されます。

1.  **ゼロベクトル：** ゼロベクトルは $\begin{pmatrix} 0 \\ 0 \end{pmatrix}$ です。これを方程式に代入すると、$0^2 + 0^2 = 1$、すなわち $0 = 1$ が得られます。これは偽です。したがって、ゼロベクトルは単位円上にはありません。

最初の条件が満たされないため、単位円は $\mathbb{R}^2$ の部分空間ではありません。

「ゼロベクトルが部分空間に含まれる」という要件は、ある集合が**部分空間**と見なされるためにベクトル空間の部分集合が満たさなければならない**3つの基本条件**の1つです。この条件がなぜ必要であるかを詳しく見てみましょう。

部分空間の概念は、より大きなベクトル空間内にある、それ自身がベクトル空間のように振る舞う小さな集合を特定することに関わっています。ベクトル空間のように振る舞うためには、これらの小さな集合は特定の本質的な性質を継承しなければなりません。あらゆるベクトル空間の最も基本的な性質の1つは、**ゼロベクトル** (しばしば $\mathbf{0}$ と表記される) が存在することです。このベクトルは、空間内の他の任意のベクトルに加えられても、そのベクトルを変化させないという重要な性質を持ちます：空間内のすべてのベクトル $\mathbf{v}$ に対して $\mathbf{v} + \mathbf{0} = \mathbf{v}$ が成り立ちます。

以下に、なぜ部分空間にゼロベクトルが含まれなければならないかの理由を示します：

1.  **ベクトル空間構造の継承：** 部分空間は、より大きな空間内の「ミニベクトル空間」となることを意図しています。部分集合がゼロベクトルを含まない場合、それはベクトル空間の公理を満たすために必要な重要な要素を欠いていることになります。ゼロベクトルがなければ、加法逆元 (もう1つの重要な公理) の概念に問題が生じます。

2.  **スカラー倍についての閉性：** スカラー倍について閉じているベクトル空間 $V$ の空でない部分集合 $W$ を考えます。$\mathbf{w}$ が $W$ 内の任意のベクトルであり、$W$ が部分空間であるならば、任意のスカラー $c$ に対して、ベクトル $c\mathbf{w}$ も $W$ に含まれなければなりません。スカラー $c = 0$ を選択すると、$0\mathbf{w} = \mathbf{0}$ となります (これはベクトル空間の性質です)。したがって、部分空間がスカラー倍について閉じており、少なくとも1つのベクトルを含むならば、それは*必ず*ゼロベクトルを含まなければなりません。

3.  **加法逆元の保証：** ゼロベクトルの存在だけでは、部分空間内での加法逆元の存在を保証するものではありませんが、その不在は根本的な問題を引き起こします。ベクトル $\mathbf{v}$ が部分空間に含まれ、部分空間がスカラー倍について閉じているならば、$(-1)\mathbf{v} = -\mathbf{v}$ も部分空間に含まれなければなりません (これは $\mathbf{v}$ の加法逆元です)。$\mathbf{v}$ と $-\mathbf{v}$ の加算結果が部分空間*内*のベクトルとなるためには、その結果のベクトルはゼロベクトルでなければなりません。ゼロベクトルが部分集合に含まれていない場合、加法逆元がより大きなベクトル空間内に存在していたとしても、加法についての閉性が損なわれる可能性があります。

簡単に言えば、部分集合がベクトル加法の「単位元」(すなわちゼロベクトル) すら含んでいないならば、それはベクトル空間を定義する演算であるベクトル加法とスカラー倍の下で、自己完結した系として機能することはできません。

したがって、ゼロベクトルが部分空間に含まれなければならないという条件は恣意的なものではなく、部分空間がそれが存在するより大きなベクトル空間の本質的な代数的構造を継承しなければならないという要件から直接導かれる結果です。
