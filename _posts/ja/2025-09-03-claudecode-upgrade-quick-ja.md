Anthropicのコマンドラインコーディングエージェントである**Claude Code**をアップグレードしたい場合の方法をご紹介します。最新の学習モードやカスタムサブエージェントなどの機能をスムーズに利用するための手順です。

### Claude Codeのアップグレード方法

1. **現在のバージョンを確認**
   以下のコマンドを実行:

   ```
   claude --version
   ```

   これで現在実行中のバージョンを確認できます。

2. **npm経由でのアップデート**
   最も簡単なアップグレード方法:

   ```
   npm install -g @anthropic-ai/claude-code@latest
   ```

   これによりnpmから最新リリースがダウンロードされます。
   Redditユーザーによると、この方法でバグが修正され応答性が向上したとの報告があります。

3. **ネイティブインストーラーの使用（Beta）**
   スタンドアロンバイナリを希望する場合は:

   ```
   curl -fsSL https://claude.ai/install.sh | bash -s latest
   ```

   （WindowsではPowerShell版を使用）

4. **アップデートの確認**
   インストール後、ターミナルを再起動し、再度`claude --version`を実行して新しいバージョンがアクティブになっていることを確認します。

5. **インストール問題のトラブルシューティング**
   問題が発生した場合は、まずアンインストールを試みてください:

   ```
   npm uninstall -g @anthropic-ai/claude-code
   npm install -g @anthropic-ai/claude-code
   ```

   この手順でアップデートの問題が解決されたというユーザー報告があります。

### 最近のアップデートでの新機能

#### 学習モード

Anthropicは最近、Claude Code内に**「説明型」**と**「学習型」**という2つの新しい学習スタイルを追加しました。これらは単なるソリューションの提供ではなく、生成されたコードの背後にある理由を理解するのに役立ちます。有効にするには:

1. Claude Codeがアップデートされていることを確認
2. Claude Codeセッション内で`/output-styles`を実行
3. **デフォルト**、**説明型**、**学習型**のスタイルから選択
   この機能は**2025年8月14日**にリリースされました。

#### カスタムサブエージェント

Claude Codeは現在**サブエージェント**をサポートしています。これは特定のタスク（例：バグ修正、リファクタリング、ドキュメント作成）に特化した専門AIアシスタントです。各エージェントに対して異なるシステムプロンプト、ツール権限、コンテキストウィンドウを設定できます。
設定方法:

1. `/agents`を実行
2. 「新規エージェント作成」を選択（ユーザーレベルまたはプロジェクトレベル）
3. 動作、許可されたツールを定義して保存
   サブエージェントはワークフローをより整理され、焦点が明確で、コンテキストを意識したものに保つのに役立ちます。
