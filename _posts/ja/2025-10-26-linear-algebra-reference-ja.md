主要な概念と公式のクイックリファレンス。過去の試験から、行列、行列式、線形システム、ベクトル空間、変換、固有値、二次形式などの核心トピックに焦点を当てる。定義、性質、計算方法を確認する。

## 1. 行列
- **定義**: \\( A = [a_{ij}] \in \mathbb{R}^{m \times n} \\)。
- **演算**:
  - 加法: \\( (A + B)_{ij} = a_{ij} + b_{ij} \\)。
  - スカラー倍: \\( (cA)_{ij} = c a_{ij} \\)。
  - 乗法: \\( (AB)_{ij} = \sum_k a_{ik} b_{kj} \\) (サイズが適合する場合)。
- **転置**: \\( (A^T)_{ij} = a_{ji} \\); \\( (AB)^T = B^T A^T \\), \\( (A^T)^T = A \\)。
- **逆行列** (正方行列): \\( AA^{-1} = I \\); \\( (AB)^{-1} = B^{-1} A^{-1} \\); \\( (A^T)^{-1} = (A^{-1})^T \\)。
- **種類**:
  - 対角行列: 対角線上のみ非ゼロ。
  - 上/下三角行列: 対角線の下/上にゼロ。
  - 対称行列: \\( A = A^T \\)。
  - 直交行列: \\( A^T A = I \\) (列が正規直交)。

## 2. 行列式 (det A)
- **性質**:
  - \\( \det(AB) = \det A \cdot \det B \\); \\( \det(A^T) = \det A \\); \\( \det(cA) = c^n \det A \\)。
  - 行/列の交換: -1倍。
  - 行/列の定数倍を加算: 変化なし。
  - 行/列をc倍: c倍。
  - \\( \det I = 1 \\); \\( \det A = 0 \\) 特異の場合 (線形従属な行/列)。
- **計算**:
  - 2x2: \\( \det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc \\)。
  - 余因子展開 (行 i): \\( \det A = \sum_j a_{ij} C_{ij} \\)、ここで \\( C_{ij} = (-1)^{i+j} M_{ij} \\) (小行列式)。
  - 三角行列: 対角要素の積。
- **余因子行列/逆行列**: \\( A^{-1} = \frac{1}{\det A} \adj A \\)、ここで \\( \adj A = C^T \\) (余因子行列の転置)。
- **クラメルの公式** (\\( Ax = b \\)、det A ≠ 0 の場合): \\( x_i = \frac{\det A_i}{\det A} \\) (A_i は i 列を b で置換)。

## 3. 線形システム (Ax = b)
- **ガウスの消去法**: [A | b] を行基本変形して REF/RREF にする。
  - REF: ピボット (先頭の1) が右下がりの階段状; ピボットの下はゼロ。
  - 後退代入で唯一解を求める。
- **解**:
  - 唯一解: rank A = n (列フルランク)、零空間 {0}。
  - 無限解: rank A = rank [A|b] < n (自由変数あり)。
  - 解なし: rank A < rank [A|b]。
- **完全解**: 特殊解 + 零空間の基底 (同次解)。
- **LU分解** (ピボット選択なし): A = LU (L は単位下三角、U は上三角); Ly = b, Ux = y を解く。
- **最小二乗法** (過決定系): \\( \hat{x} = (A^T A)^{-1} A^T b \\) (フルランクの場合)。

## 4. ベクトル空間と部分空間
- **ベクトル空間**: 加法/スカラー倍について閉じている; 公理 (例: 零ベクトル、逆元)。
- **部分空間**: ベクトルの張る空間; 閉じていて、0を含む。
  - 列空間: Col(A) = Aの列ベクトルの張る空間; 次元 = rank A。
  - 行空間: Row(A) = Col(A^T); 次元 = rank A。
  - 零空間: Nul(A) = {x | Ax = 0}; 次元 = n - rank A。
  - 左零空間: Nul(A^T)。
- **線形独立性**: c1 v1 + ... + ck vk = 0 ⇒ すべての ci = 0。
- **基底**: 線形独立な生成系。
- **次元**: 基底のベクトル数; dim Col(A) + dim Nul(A) = n (階数・退化次数の定理)。
- **階数**: ピボット列の数 = dim Col(A) = dim Row(A)。

## 5. 線形変換
- **定義**: T: V → W が線形 ⇔ T(u + v) = T u + T v, T(cu) = c T u。
- **行列表示**: 基底に関する [T] = A、ここで T(x) = A x (標準基底)。
- **核**: Ker T = Nul(A); 像: Im T = Col(A)。
- **同型写像**: 単射かつ全射 (可逆行列)。
- **階数・退化次数定理**: dim Ker T + dim Im T = dim V。

## 6. 固有値と固有ベクトル
- **定義**: A v = λ v (v ≠ 0 固有ベクトル、λ 固有値)。
- **固有方程式**: det(A - λ I) = 0; 根 λi (代数的重複度)。
- **固有ベクトル**: (A - λ I) v = 0 を解く; 幾何学的重複度 = 固有空間の次元。
- **対角化可能**: n 個の線形独立な固有ベクトル ⇒ A = X Λ X^{-1} (Λ = diag(λi), X = [v1 ... vn])。
  - 対称行列 A: 常に対角化可能; 直交する固有ベクトル (A = Q Λ Q^T, Q は直交行列)。
- **トレース**: tr A = ∑ λi。
- **行列式**: det A = ∏ λi。
- **相似な行列**: A ~ B ⇔ A = P B P^{-1}; 同じ固有値、トレース、行列式。

## 7. 内積と二次形式
- **内積**: <u, v> = u^T v (ユークリッド); ||v|| = √<v,v>。
- **直交**: <u,v> = 0; 正規直交基底: <ei, ej> = δij。
- **グラム・シュミットの直交化法**: 基底 {v1,...,vn} → {u1,...,un} を直交化。
  - u1 = v1; uk = vk - proj_{span(u1..u_{k-1})} vk; proj_w v = (<v,w>/<w,w>) w。
- **二次形式**: q(x) = x^T A x (A は対称)。
  - 正定値: x ≠ 0 に対して q(x) > 0 (すべての λi > 0)。
  - 対角化: q(x) = ∑ λi yi^2 (y = Q^T x)。

## クイックヒント
- 階数の計算: 行基本変形してピボットの数を求める。
- 対角化可能性の確認: 各 λ に対して幾何学的重複度 = 代数的重複度。
- 試験対策: 2x2/3x3行列での行基本変形、行列式の展開、固有値問題の演習。
- よくある誤り: 重複度の見落とし; 余因子の符号; 非可換な乗法。
