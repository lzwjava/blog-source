### Visual Studio Code 1.104 (2025年8月)

このリリースは、GitHub Copilot の AI 機能強化、AI ツールのセキュリティ向上、そしてエディタとターミナル全体の生産性向上に重点を置いています。主な更新内容は以下の通りです：

#### AI 機能
- **Auto Model Selection (プレビュー)**: 「Auto」が選択されると、パフォーマンスとレート制限に基づいて最適なモデル（例：Claude Sonnet 4、GPT-5 バリアント）を自動的に選択；有料ユーザーは10%のリクエスト割引が適用されます。
- **機密ファイルの編集確認**: Agent モードでは、システムファイル、ドットファイル、ワークスペース外のファイルを編集する前にユーザーの承認が必要；設定でカスタマイズ可能。
- **AGENTS.md ファイルのサポート (実験的)**: ワークスペースの `AGENTS.md` をチャットリクエストのコンテキストとして自動的に組み込みます。
- **コーディングエージェントの連携強化 (実験的)**: セッション管理、GitHub 統合、TODO コメントやチャットからの委任機能が改善されました。
- **ターミナル自動承認**: 安全なコマンド実行のためにオプトインで利用可能。`curl` や `wget` などのリスクのあるアクションに対して警告；新しい承認ルールを追加。
- **数式レンダリング**: KaTeX 数式がチャット応答でデフォルトでインライン表示されるようになりました。
- **#codebase ツールの改善**: 新しい埋め込みモデルを使用し、より高速で効率的なセマンティックコード検索を実現。
- **AI 機能の無効化**: Copilot チャット、補完、提案をグローバルまたはワークスペースごとに非表示および無効化する新しい設定。
- **Python 向け AI ツール (実験的/プレビュー)**: ドキュメント未記載のシンボルに対する AI 駆動のホバー要約と、メモリ内実行のための「Run Code Snippet」ツール。

#### セキュリティ
- **AI ツールの保護機能**: 機密編集、ターミナルコマンド、グローバル自動承認に対する確認ダイアログを拡充。リスク軽減のための警告と設定可能なルールを追加。
- **ドキュメント**: AI 駆動開発におけるセキュリティ考慮事項に関する新しいガイダンス。

#### 生産性
- **チャットビューのデフォルト**: チャットがデフォルトでセカンダリサイドバーに開く；表示設定をカスタマイズ可能。
- **ターミナルの改善**: IntelliSense の強化（例：git 補完、アイコン）、スティッキースクロール、ターミナルエディタ/ウィンドウでのアクションを追加。
- **Python 環境**: Pipenv サポートの強化、`.env` ファイルからの環境変数注入、すべての Python ドキュメントでの完全な IntelliSense を実現。
- **エディタの調整**: 設定可能なインライン提案の遅延、キーボードナビゲーションのためのタブインデックス、Windows でのウィンドウ境界線の色を追加。
- **GitHub Pull Requests**: 折りたたみ可能なサイドバー、Copilot のための TODO アクション、サブモジュール無視オプションを追加。
- **タスクとターミナルサポート**: 入力処理、エラー表示、シェル統合のタイムアウトが改善されました。

#### その他の主な変更点
- **アクセシビリティ**: スクリーンリーダーを支援するためのチャット確認ダイアログへのフォーカスコマンドを追加。
- **言語**: 組み込みの Bower.json IntelliSense を削除（非推奨ツール；npm/yarn への移行を推奨）。
- **拡張機能のオーサリング**: 認証チャレンジとセカンダリサイドバーへのコントリビューションのための提案 API を追加。
- **非推奨化**: `chat.mcp.enabled` などの設定が移行；Bower サポートを終了。

詳細については、公式のリリースノートをご確認ください。
