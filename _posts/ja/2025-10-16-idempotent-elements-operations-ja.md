### 冪等性の定義

数学および抽象代数学において、**冪等**（idempotent）な要素または演算とは、同じ入力に対して複数回適用した場合に、1回だけ適用した場合と同じ結果が得られるものを指します。形式的には、演算 ⊕ について、要素 \\( A \\) が冪等であるとは、\\( A \oplus A = A \\) が成り立つことです。

例では、2つの一般的な演算を用いてこれを説明しています：
- \\( A \cdot A = A \\): これは**乗法**（またはブール代数における論理積/AND）で成り立ち、演算を繰り返しても値は変わりません。
- \\( A + A = A \\): これは**加法**（またはブール代数における論理和/OR）で成り立ち、標準的な算術（ここでは \\( A + A = 2A \\)）では直感的に理解しにくいですが、集合（和集合: \\( A \cup A = A \\)）やブール論理などの構造では真となります。

#### 冪等性の検証方法
要素 \\( A \\) が演算 ⊕ のもとで冪等であるか確認するには：
1. \\( A \oplus A \\) を計算します。
2. 結果が \\( A \\) と等しければ、それは冪等です。
3. 複数回の適用（例: \\( A \oplus (A \oplus A) \\)）についても繰り返しますが、冪等性はこれが即座に安定することを意味します。

**例**:
- **ブール代数**: AND (·) と OR (+) は冪等な演算です。
  - True · True = True
  - True + True = True
- **行列**: 行列 \\( M \\) が \\( M^2 = M \\) を満たす場合、冪等です（射影行列など）。
- **関数**: 関数 \\( f \\) が \\( f(f(x)) = f(x) \\) を満たす場合、冪等です（例: 絶対値関数: \\( | |x| | = |x| \\)）。

この性質は、計算機科学（例: GETのようなHTTPメソッドは冪等）や代数学（冪等半環）において重要です。
