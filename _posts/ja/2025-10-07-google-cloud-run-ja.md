Cloud Runは、GCPの完全マネージドなサーバーレスコンピュートプラットフォームで、ステートレスなコンテナをGoogleのスケーラブルなインフラストラクチャ上で直接実行できます。基盤となるサーバーやプロビジョニング、スケーリングを気にせずにアプリケーションをデプロイしたい開発者向けに設計されています。Webアプリ、API、マイクロサービス、イベント駆動型ワークロードを構築する場合でも、Cloud Runがインフラを処理するため、コードに集中できます。

### 主な機能
- **サーバーレス実行**: コンテナ化されたコード（あらゆる言語やランタイムをサポート）をデプロイし、受信リクエストやトラフィックに基づいてインスタンス数を0から数千まで自動的にスケールします。
- **従量課金制**: 消費したリソース（リクエストごと、またはインスタンスの実行時間ごと）に対してのみ課金されるため、変動するワークロードに費用対効果が高くなります。
- **組み込み統合**: データベースのCloud SQL、ファイル用のCloud Storage、メッセージング用のPub/Subなど、他のGCPサービスとシームレスに連携します。また、プライベートネットワーキングのためのVPCもサポートしています。
- **デプロイオプション**:
  - Artifact RegistryまたはDocker Hubから事前にビルドされたコンテナイメージをプッシュ。
  - Cloud Buildを使用してソースコードから直接デプロイ（Node.js、Python、Java、Go、.NET、Rubyなどの言語をサポート）。
  - よりシンプルなFunction-as-a-ServiceスタイルのデプロイにはCloud Run Functionsを利用。
- **セキュリティとネットワーキング**: サービスは公開（認証不要）または非公開（認証必須）に設定可能で、HTTPSエンドポイントとカスタムドメインをサポートします。
- **追加モード**: リクエスト駆動型サービスに加えて、バッチタスク（スケジュールされたスクリプトやデータ処理など）向けのJobs、長時間実行される非HTTPワークロード向けのWorker Poolsを提供します。

始めるには、GCPコンソール、gcloud CLI、またはCI/CDパイプラインを介してデプロイできます。例えば、シンプルな「Hello World」コンテナを数分でビルドしてデプロイできます。

### Cloud Run管理コンソール
GCPコンソールのCloud Runセクションは、デプロイメントを管理するための直感的なダッシュボードを提供します。共有いただいたServicesビューに基づく内訳は以下の通りです：

- **概要**: メインの「Cloud Run > Services」ページでは、デプロイされたすべてのサービスが表形式で一覧表示されます。初めての場合は、「完全マネージドプラットフォームでアプリを実行」のような役立つ推奨バナーからクイックスタートを促されます。

- **テーブル列**（共有スニペットに基づく）:
  - **名前**: 各サービスの一意の識別子（例: "my-api"）。
  - **デプロイメントタイプ**: デプロイ方法を示します（例: イメージの場合は「Container」、コードベースのデプロイの場合は「Source」）。
  - **Req/sec**: トラフィック負荷を監視するためのリアルタイムの1秒あたりのリクエスト数メトリクス。
  - **リージョン**: サービスが実行されるGCPリージョン（例: us-central1）。レイテンシとコンプライアンスに影響します。
  - **認証**: サービスが公開（未認証を許可）か、IAM認証を必要とするかを示します。
  - **Ingress**: トラフィックルーティングを制御します（例: 内部/外部の場合は「All」、内部のみに制限など）。
  - **最終デプロイ**: 最新の更新のタイムスタンプ。
  - **デプロイ担当者**: デプロイを実行したユーザーまたはサービスアカウント。

- **フィルターとアクション**: テーブルの上部で、これらの列のいずれかでフィルタリングしてリストを絞り込むことができます。「サービスを作成」ボタンを使用して新しいサービスをデプロイするか、サービス名をクリックしてCPU/メモリ制限、環境変数、スケーリング設定、トラフィック分割のためのリビジョンなどの設定を編集します。

このコンソールを使用すると、ブラウザから離れることなくパフォーマンスを監視し、ログを表示し、デプロイメントを反復的に改善できます。実際に試すには、APIを有効にした後、[console.cloud.google.com/run](https://console.cloud.google.com/run) に移動してください。
