GNOME Terminal では、「すべて選択」オプション（通常 `編集 > すべて選択` または `Ctrl+Shift+A` でアクセス）は画面上に表示されているテキストのみを選択します。**すべてのテキスト**（表示範囲を超えたスクロールバックバッファ内のコンテンツを含む）を選択するには、以下のいずれかの方法を使用できます：

### 方法 1: スクロールバックバッファを増やして手動で選択
1. **スクロールバックが有効になっていることを確認**：
   - GNOME Terminal で `編集 > プロファイルの設定` に移動します。
   - 「スクロール」タブで、スクロールバックバッファが大きな数値または「無制限」に設定されていることを確認します。
2. **手動選択**：
   - マウスまたは `Shift+Page Up` を使用してターミナル出力の先頭までスクロールします。
   - 先頭から最後までクリックしてドラッグし、すべてのテキストを手動で選択します。
   - または、`Shift+Home` でバッファの先頭にジャンプし、クリックしてドラッグするか `Shift+End` を使用して最後まで選択します。

### 方法 2: コマンドを使用してすべての出力をキャプチャ
ターミナル出力全体（スクロールバックを含む）をキャプチャしたい場合は、コマンドを使用して出力をリダイレクトまたはコピーできます：
1. **出力をファイルにリダイレクト**：
   - 出力を生成するコマンドがわかっている場合は、リダイレクト付きで再実行します：
     ```bash
     command > output.txt
     ```
     これにより、すべての出力が `output.txt` に保存され、後で開いて選択できます。
2. **`script` を使用してターミナルセッションを記録**：
   - 以下のコマンドで記録セッションを開始します：
     ```bash
     script output.log
     ```
   - コマンドを実行した後、`exit` と入力して記録を停止します。
   - `output.log` を開いてすべてのテキストを選択します。

### 方法 3: ターミナルコマンド経由でスクロールバックバッファをコピー
1. コマンド実行中に `tee` コマンドまたは類似のコマンドを使用して出力をキャプチャします：
   ```bash
   command | tee output.txt
   ```
   これにより、出力がターミナルに表示されると同時に `output.txt` に保存されます。
2. ファイルをテキストエディタで開き、すべてのテキストを簡単に選択します。

### 方法 4: 高度な制御に `tmux` または `screen` を使用
頻繁にターミナル出力全体をキャプチャする必要がある場合：
1. **`tmux` をインストールして使用**：
   - `tmux` をインストールします：
     ```bash
     sudo apt install tmux  # Debian/Ubuntu の場合
     ```
   - `tmux` セッションを開始します：
     ```bash
     tmux
     ```
   - コマンドを実行します。
   - `Ctrl+b, [` でスクロールモードに入り、`Ctrl+a, Ctrl+e` を押してバッファ内のすべてのテキストを選択します。
   - （コピーを有効にするように `tmux` を設定した後）`Ctrl+b, ]` でコピーします。
2. **tmux バッファを保存**：
   - バッファ全体をファイルに保存します：
     ```bash
     tmux capture-pane -S - -E - -b 0 -J > output.txt
     ```
   - `output.txt` を開いてすべてのテキストを選択します。

### 制限事項
- GNOME Terminal 自体には、GUI 経由での「スクロールバック全体を選択」機能は組み込まれていません。
- スクロールバックバッファのサイズが無制限に設定されていない場合、利用可能なテキスト量が制限される可能性があります。
- 非常に大きな出力の場合、`tmux`、`screen` の使用やファイルへのリダイレクトがより信頼性の高い方法です。
