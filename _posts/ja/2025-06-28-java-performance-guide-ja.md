### 1. **Javaパフォーマンスチューニング入門**
   - 本書は、Javaアプリケーションにおけるパフォーマンスの重要性を強調し、パフォーマンスのボトルネックを特定して解決する体系的なアプローチを概説します。
   - ベンチマーキング、プロファイリング、モニタリングなど、パフォーマンスを測定するためのツールと方法論を紹介します。

### 2. **Java仮想マシン（JVM）の内部構造**
   - ヒープ、スタック、メタスペースなどのJVMのアーキテクチャと、それらがパフォーマンスに与える影響について説明します。
   - Just-In-Time（JIT）コンパイル、クラスローディング、およびJVMがコード実行を最適化する方法について論じます。
   - 特定のワークロードに対してパフォーマンスを微調整するためのJVMフラグと設定についてカバーします。

### 3. **ガベージコレクション（GC）チューニング**
   - シリアル、パラレル、CMS、G1、ZGC、Shenandoahなど、Javaにおけるガベージコレクションのメカニズムを詳細に解説します。

   - Java 11以降で導入されたEpsilon（ノーOP GC）やG1、ZGCの改善など、新しいGC機能について探ります。

### 4. **Java言語とAPIの最適化**
   - 文字列、コレクション、並行性ユーティリティなどのJava言語構成要素がパフォーマンスに与える影響について論じます。
