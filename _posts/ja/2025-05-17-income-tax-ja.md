中国における給与所得者の個人所得税は、通常、雇用主が**累進課税方式**を用いて毎月計算し、源泉徴収します。この方法では、各月の税額を計算する際に、その年のはじめから現在までの累計所得と控除額を考慮するため、年間を通じて税負担がよりスムーズに分散されます。

各月の所得を単独で課税するのではなく、累進方式では*現在月までの*所得に対して発生する可能性のある総税額を計算し、そこからその年の前月までに既に支払われた税額を差し引きます。その結果が、当月に源泉徴収される税額となります。

以下に、主要な概念と計算方法の詳細を示します。

**主要な概念:**

*   **累計所得:** その年の1月1日から現在月末までに現在の雇用主から受け取った給与の総支給額。
*   **累計的基本控除:** これは標準的な非課税基準額です。2025年は月額5,000元です。累計的基本控除は、5,000元に、その年の現雇用主での勤務月数（現在月まで）を乗じて計算します。
*   **累計特別控除:** 社会保険（年金、医療、失業）や住宅積立金など、給与から天引きされる必須の拠出金の累計額。
*   **累計特別追加控除:** 特定の支出に対して申告できる任意の控除の累計額。一般的な例は以下の通りです：
    *   子女教育
    *   継続教育
    *   大病医療費
    *   住宅ローン利子
    *   家賃
    *   老人扶養
*   **累計その他法令により定められた控除:** 特定の商業健康保険料など、法律で認められたその他の控除対象の累計額。
*   **累計課税所得:** 累計所得からすべての累計控除額を差し引いて計算された、課税対象額。
*   **累計应纳税額:** 累進税率表（下記参照）を使用して累計課税所得に対して計算された総税額。
*   **累計源泉徴収済・前払済税額:** その年の前月までに雇用主が既に源泉徴収し、税務当局に納付した個人所得税の総額。

**計算式:**

1.  **累計課税所得の計算:**
    累計課税所得 = 累計所得 - 累計非課税所得 - 累計的基本控除 - 累計特別控除 - 累計特別追加控除 - 累計その他法令により定められた控除

2.  **当月の納付税額の計算:**
    当月の源泉徴収・前払い税額 = (累計源泉徴収・前払い課税所得 × 適用累進税率 - 対応する控除額) - 累計税額控除・免除 - 累計源泉徴収済・前払済税額

**個人所得税率表（給与所得の累進源泉徴収適用）**

この表は、*累計課税所得*に基づいて*累計应纳税額*を計算するために使用されます。

| レベル | 累計源泉徴収・前払い課税所得                          | 前払税率 | 控除額       |
| :----- | :----------------------------------------- | :------------------- | :-------------- |
| 1      | 36,000元以下                               | 3%                   | 0               |
| 2      | 36,000元超～144,000元以下                  | 10%                  | 2,520           |
| 3      | 144,000元超～300,000元以下                 | 20%                  | 16,920          |
| 4      | 300,000元超～420,000元以下                 | 25%                  | 31,920          |
| 5      | 420,000元超～660,000元以下                 | 30%                  | 52,920          |
| 6      | 660,000元超～960,000元以下                 | 35%                  | 85,920          |
| 7      | 960,000元超                               | 45%                  | 181,920         |

*注：適用される税率と控除額は、*累計課税所得*がどの所得レベルに該当するかによって決定されます。*

**事例研究: 2025年の張三の個人所得税計算**

月収が変動し、控除額が一定の従業員、張三の例を追ってみましょう。

*   **月収:** 1月: 15,000元; 2月: 45,000元; 3月: 15,000元。
*   **毎月の一定控除額:**
    *   基本控除: 5,000元 (標準)
    *   特別控除 (五保険一金): 3,000元
    *   特別追加控除:
        *   子女教育: 1,000元
        *   住宅ローン利子: 1,000元
        *   老人扶養 (一人っ子): 2,000元
        *   *月間特別追加控除合計: 1000 + 1000 + 2000 = 4,000元*
    *   その他控除 (商業健康保険): 200元

**月別計算ステップ:**

**2025年1月 (第1月)**

*   **累計所得:** 15,000元
*   **累計控除額:**
    *   基本: 5000 * 1 = 5,000
    *   特別: 3000 * 1 = 3,000
    *   特別追加: 4000 * 1 = 4,000
    *   その他: 200 * 1 = 200
    *   *累計控除額合計:* 5000 + 3000 + 4000 + 200 = 12,200元
*   **累計課税所得:** 15,000 (所得) - 12,200 (控除) = 2,800元
*   **累計应纳税額:**
    *   累計課税所得 (2800) はレベル1 (36,000未満) です。
    *   应纳税額 = 2800 * 3% - 0 (控除額) = 84元。
*   **累計源泉徴収済税額:** 0元 (最初の月)
*   **当月の税額 (1月):** 84 (累計应纳税額) - 0 (既納税額) = **84元**

**2025年2月 (第2月)**

*   **累計所得:** 15,000 (1月) + 45,000 (2月) = 60,000元
*   **累計控除額:**
    *   基本: 5000 * 2 = 10,000
    *   特別: 3000 * 2 = 6,000
    *   特別追加: 4000 * 2 = 8,000
    *   その他: 200 * 2 = 400
    *   *累計控除額合計:* 10000 + 6000 + 8000 + 400 = 24,400元
*   **累計課税所得:** 60,000 (所得) - 24,400 (控除) = 35,600元
*   **累計应纳税額:**
    *   累計課税所得 (35,600) は依然としてレベル1 (36,000未満) です。
    *   应纳税額 = 35,600 * 3% - 0 (控除額) = 1,068元。
*   **累計源泉徴収済税額:** 84元 (1月分)
*   **当月の税額 (2月):** 1068 (累計应纳税額) - 84 (既納税額) = **984元**

**2025年3月 (第3月)**

*   **累計所得:** 60,000 (2月) + 15,000 (3月) = 75,000元
*   **累計控除額:**
    *   基本: 5000 * 3 = 15,000
    *   特別: 3000 * 3 = 9,000
    *   特別追加: 4000 * 3 = 12,000
    *   その他: 200 * 3 = 600
    *   *累計控除額合計:* 15000 + 9000 + 12000 + 600 = 36,600元
*   **累計課税所得:** 75,000 (所得) - 36,600 (控除) = 38,400元
*   **累計应纳税額:**
    *   累計課税所得 (38,400) はレベル2 (36,000超、144,000未満) です。
    *   应纳税額 = 38,400 * 10% - 2520 (控除額) = 3840 - 2520 = 1,320元。
*   **累計源泉徴収済税額:** 84 (1月) + 984 (2月) = 1,068元
*   **当月の税額 (3月):** 1320 (累計应纳税額) - 1068 (既納税額) = **252元**

この事例からわかるように、月々の源泉徴収税額は、累計所得とそれが年間を通じてどの税区分に該当するかによって変動します。これは月収が変動する場合でも同様です。累進方式により、年末までに源泉徴収された総税額は、年間総所得と控除額に基づく年間の納税義務額にほぼ一致することが保証されます。
