macOSでシステム情報を取得するには、いくつかの組み込みツールやコマンドを使用できます。以下に、さまざまな種類のシステム情報を取得する方法を紹介します：

### 1. **システム情報ユーティリティ**
システム情報アプリは、Macのハードウェアとソフトウェアの包括的な概要を提供します。

- `Cmd + Space`を押して**Spotlight検索**を開き、「システム情報」と入力してEnterキーを押します。
- または、**アップルメニュー** > **このMacについて** > **システムレポート**から開くこともできます。

### 2. **このMacについて**
これにより、Macの仕様を簡単に確認できます。

- 画面左上の**アップルメニュー**をクリックします。
- **このMacについて**を選択します。これにより、macOSのバージョン、プロセッサ、メモリ、シリアル番号などの基本情報が表示されます。

### 3. **ターミナルコマンド**
ターミナルを使用して、さまざまなコマンドで詳細なシステム情報を取得できます。

- `アプリケーション` > `ユーティリティ` > `ターミナル`から、またはSpotlightで検索して**ターミナル**を開きます。

#### 基本システム情報
```sh
system_profiler SPHardwareDataType
```

#### 詳細システム情報
```sh
system_profiler
```

#### CPU情報
```sh
sysctl -n machdep.cpu.brand_string
```

#### メモリ情報
```sh
sysctl hw.memsize
```

#### ディスク使用量
```sh
df -h
```

#### ネットワーク情報
```sh
ifconfig
```

#### バッテリー情報（ノートブックの場合）
```sh
pmset -g batt
```

#### インストール済みソフトウェアのリスト
```sh
ls /Applications
```

#### macOSバージョン
```sh
sw_vers
```

#### ユーザー情報
```sh
id
```

#### 実行中のプロセスのリスト
```sh
ps aux
```

#### インストール済みBrewパッケージのリスト
```sh
brew list
```

#### インストール済みPipパッケージのリスト
```sh
pip list
```

#### インストール済みGemパッケージのリスト
```sh
gem list
```

### 4. **アクティビティモニタ**
このアプリは、MacのCPU、メモリ、エネルギー、ディスク、ネットワークの使用状況に関するリアルタイム情報を提供します。

- `アプリケーション` > `ユーティリティ` > `アクティビティモニタ`から、またはSpotlightで検索して**アクティビティモニタ**を開きます。

### 5. **ディスクユーティリティ**
このアプリは、ストレージデバイスに関する情報を提供します。

- `アプリケーション` > `ユーティリティ` > `ディスクユーティリティ`から、またはSpotlightで検索して**ディスクユーティリティ**を開きます。

### 6. **サードパーティ製ツール**
**iStat Menus**や**CoconutBattery**などのサードパーティ製アプリケーションもあり、詳細なシステム情報と監視機能を提供します。

これらの方法を使用することで、macOS上で広範なシステム情報を収集できます。
