以下に、**3つの基本増幅回路構成**—**エミッタ接地 (CE)**、**コレクタ接地 (CC)** (別名 **エミッタフォロワ**)、**ベース接地 (CB)**—について、その構造、信号特性、利点、欠点、および典型的な応用に焦点を当てた包括的な説明を示します。

---

## 🔷 1. エミッタ接地 (CE) 増幅回路

### 🔧 構成
- **入力**: ベースとエミッタ間に印加。
- **出力**: コレクタとエミッタ間から取得。
- **共通端子**: エミッタが入力と出力の両方に共通。

### 🔍 主な特性

| 特性                | 説明                       |
|-------------------------|------------------------------------|
| **電圧利得**        | 高                               |
| **電流利得**        | 中程度から高                   |
| **電力利得**          | 高                               |
| **位相シフト**         | 180° (出力が反転)             |
| **入力インピーダンス**     | 中程度                           |
| **出力インピーダンス**    | 中程度                           |

### ✅ 利点
- 電圧および電力増幅に優れる。
- 最も広く使用される構成。

### ❌ 欠点
- 信号を反転させる (180° の位相シフト)。
- インピーダンス整合にはあまり適さない。

### 🧰 応用
- 汎用信号増幅。
- オーディオ増幅器。
- 増幅器の中間段。

---

## 🔷 2. コレクタ接地 (CC) 増幅回路 — *エミッタフォロワ*

### 🔧 構成
- **入力**: ベースとコレクタ間に印加。
- **出力**: エミッタとコレクタ間から取得。
- **共通端子**: コレクタが共通。

### 🔍 主な特性

| 特性                | 説明                           |
|-------------------------|----------------------------------------|
| **電圧利得**        | 約1 (ユニティゲイン)           |
| **電流利得**        | 高                                   |
| **電力利得**          | 中程度                               |
| **位相シフト**         | 0° (反転なし)                      |
| **入力インピーダンス**     | 高                                   |
| **出力インピーダンス**    | 低                                    |

### ✅ 利点
- 優れたバッファ段。
- インピーダンス整合に優れる (高入力インピーダンス、低出力インピーダンス)。
- 信号反転なし。

### ❌ 欠点
- 電圧利得なし。
- 電圧増幅が必要な単体の増幅器には不向き。

### 🧰 応用
- 段間のバッファ。
- 低インピーダンス負荷の駆動。
- 電圧フォロワ。

---

## 🔷 3. ベース接地 (CB) 増幅回路

### 🔧 構成
- **入力**: エミッタとベース間に印加。
- **出力**: コレクタとベース間から取得。
- **共通端子**: ベースが共通。

### 🔍 主な特性

| 特性                | 説明                             |
|-------------------------|------------------------------------------|
| **電圧利得**        | 高                                     |
| **電流利得**        | 1未満                              |
| **電力利得**          | 中程度                                 |
| **位相シフト**         | 0° (反転なし)                        |
| **入力インピーダンス**     | 非常に低                                 |
| **出力インピーダンス**    | 高                                     |
| **周波数応答**  | 非常に広い (高周波使用に優れる) |

### ✅ 利点
- 優れた高周波応答。
- 安定した動作。

### ❌ 欠点
- 入力インピーダンスが低い — 駆動段の注意深い設計が必要。
- 電流利得が限られる。

### 🧰 応用
- RF (無線周波数) 増幅器。
- 高周波信号増幅。
- インピーダンス整合 (低インピーダンス源から高インピーダンス負荷へ)。
