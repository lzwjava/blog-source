DBS Tech Indiaは、テランガーナ州ハイデラバードに位置し、アジアを代表する金融サービスグループであるDBSバンクのグローバル技術ハブとして機能しています。2016年にDBS Asia Hub 2として設立され、2022年に世界クラスのエンジニアリング組織へと変貌を遂げたことを反映し、DBS Tech Indiaにブランド名を変更しました。デジタルイノベーションを推進し、最先端の銀行ソリューションを提供することに焦点を当てています。このハブは、DBSバンクの戦略的技術イニシアチブにおいて極めて重要な役割を果たしており、先進的なテクノロジーとソフトウェアを活用したプラットフォームを通じて銀行業務に革命を起こし、顧客が「より豊かに生き、銀行利用を減らす」ことを可能にすることを目指しています。[](https://www.dbs.com/dbstechindia/decode.html)[](https://www.thehindubusinessline.com/info-tech/dbs-asia-hub-2-becomes-dbs-tech/article65686356.ece)

**設立**

DBS Tech Indiaは、2016年にDBS Asia Hub 2として、シンガポール以外で初となる同バンクのオフショア技術開発センターとして設立されました。アジア全域におけるDBSの技術能力を強化し、そのデジタルバンキング戦略を支援するために、70.5クロールの初期払込資本金を投じた完全子会社として創設されました。ハイデラバードは、技術革新のための強力なエコシステムと豊富な人材プールへのアクセスが可能であることから選ばれました。[](https://economictimes.indiatimes.com/industry/banking/finance/banking/dbs-bank-to-set-up-offshore-tech-hub-in-hyderabad/articleshow/52295899.cms)[](https://www.dbs.com/dbstechindia/decode.html)

**進化**

設立以来、DBS Tech Indiaはコスト重視の技術センターから、価値主導のエンジニアリングハブへと進化を遂げてきました。その進化における主なマイルストーンは以下の通りです：

- **2018年**: コストセンターからバリューセンターへの急速な転換が評価され、Zinnov Centre Transformation Awardを受賞。200を超えるライブApplication Program Interfaces (API)を構築し、50のライブパートナーとの強力なエコシステムを育成。[](https://www.dbs.com/newsroom/DBS_BANKS_INDIA_TECHNOLOGY_DEVELOPMENT_CENTRE_WINS_TRANSFORMATION_AWARD)
- **2020年**: 技術中心のワークスペースでイノベーションを促進するために設計された、190,000平方フィートに及ぶSkyviewビル内のモダンオフィスに移転。このオフィスは、DBSのシンガポール事業の拡張として、Asia Pacific Property Awards 2020 for Office Interiorを受賞。[](https://officesnapshots.com/2020/07/09/dbs-bank-offices-hyderabad/)
- **2022年**: DBS Asia Hub 2からDBS Tech Indiaへとブランド名を変更し、プロダクトマインドセットを持つ技術会社となるという戦理的焦点を明確化。DBSのデジタルトランスフォーメーション目標を支援するために人材開発を重視し、新卒者600名を採用。[](https://www.thehindubusinessline.com/info-tech/dbs-asia-hub-2-becomes-dbs-tech/article65686356.ece)[](https://hr.economictimes.indiatimes.com/news/industry/dbs-tech-india-onboards-600-fresh-graduates-at-its-hyderabad-office/93112836)
- **継続的なイノベーション**: センターは、世界クラスのエンジニアリングチームの構築、新興技術の実験、顧客体験を向上させるデジタルソリューション（例えば、顧客のスマートフォンに銀行機能全体を提供するなど）の提供に焦点を移しています。[](https://hr.economictimes.indiatimes.com/news/industry/dbs-tech-india-onboards-600-fresh-graduates-at-its-hyderabad-office/93112836)[](https://www.dbs.com/dbstechindia/index.html?linkId=100000137387232)

**現在の規模**

2022年現在、DBS Tech Indiaは、ハイデラバードの2つの施設（ハイテクシティのSkyviewとフィナンシャルディストリクトのWaverock）で4,000名以上の技術専門家を雇用しており、総収容人数は3,600席です。従業員数は大幅に増加し、COVID-19パンデミック前の1,200名から、2022年7月までに3,600名へと3倍に増え、さらに同年に新卒者600名が採用されました。このハブは、DBSのグローバルな技術およびデジタルバンキングの取り組みを支援するために、人材プールの拡大を続けています。[](https://hr.economictimes.indiatimes.com/news/industry/dbs-tech-india-onboards-600-fresh-graduates-at-its-hyderabad-office/93112836)[](https://timesofindia.indiatimes.com/city/hyderabad/dbs-hyd-grows-3x-during-pandemic/articleshow/93124375.cms)

DBS Tech Indiaは、技術人材の育成、スタートアップのような文化の醸成、ハイデラバードの技術エコシステムの活用を通じて、技術主導の金融サービスリーダーであり続けるというDBSバンクのビジョンを推進することにコミットし続けています。[](https://www.dbs.com/dbstechindia/career.html)[](https://hr.economictimes.indiatimes.com/news/industry/dbs-tech-india-onboards-600-fresh-graduates-at-its-hyderabad-office/93112836)
